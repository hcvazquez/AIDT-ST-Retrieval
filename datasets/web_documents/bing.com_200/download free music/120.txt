          <h1 class="entry-title entry-title-single" itemprop="headline">PHP Download File Script – Free Must Have Class</h1>
         </header>
         <div class="entry-content" itemprop="text">
          <p>PHP File download class will be helpful for developers and users to provide facility of “File Download” by hiding real file paths. Chip File Download Class which is written in PHP is very flexible and easy to use. The script triggers the download dialogue so that visitors can choose to download the file instead of opening it in the browser.</p>
          <div class="chip-life-child-tabs">
           <ul>
            <li><a href="#tab-description">Description</a></li>
            <li><a href="#tab-installation">Installation</a></li>
            <li><a href="#tab-demo">Demo</a></li>
            <li><a href="#tab-download">Download</a></li>
            <li><a href="#tab-changelog">Changelog</a></li>
           </ul>
           <div id="tab-description">
            <p>This is very useful PHP download script. You can use PHP file download script to avoid direct files downloads (hide real file paths), and to limit downloads to only some set of file types. Get free Demos of Framework and Zend PHP.</p>
            <p>It is an easy to use force file download script, useful for sites that have different files and would like force people to download the files instead of having them run remotely off their web server. This can be applied to mp3’s, video clips, archives, docs, images, pdf files and more.</p>
            <h2>Features</h2>
            <ul>
             <li>Provide any file type for download</li>
             <li>Flexible control over files type, which are being offered to download</li>
             <li>A set of predefined mime type settings for most common file extensions</li>
             <li>Built-in feature for detecting mime type automatically, if Mimetype or Fileinfo extension installed on the server</li>
             <li>Hot-linking protection</li>
             <li>Displays download dialogue</li>
             <li>Download folder protection</li>
             <li>Directory(s) containing files, within “download_path” support</li>
            </ul>
            <h2>Requirements</h2>
            <ul>
             <li>PHP5 or higher</li>
            </ul>
            <h2>Extension Requirements – Optional<strong><br> </strong></h2>
            <ul>
             <li>Mimetype Extension</li>
             <li>Fileinfo Extension</li>
            </ul>
           </div>
           <div id="tab-installation">
            <p>I have already given a full sample code in the download archive but i will also explain the usage of class online as simple as possible. I hope you will be able to use it within a little time.</p>
            <h2>Include <q>class.chip_download.php</q></h2>
            <p>You have to include download file script in your file at the suitable position you think. I will recommend to use all code of this tutorial at the top of your page to avoid any <q>output</q> warning.</p>
            <pre class="brush: php; title: ; notranslate" title="">require_once("class.chip_download.php");
</pre>
            <h2>Variable: $download_path</h2>
            <p>Set <q>$download_path</q> variable. You should provide absolute path to the download directory. Windows example can be “E:/wamp/www/tutorialchip/download/”, where as Linux example can be “/home/tutorialchip/public_html/download/”.</p>
            <h2>Variable: $file<strong><br> </strong></h2>
            <p>Set <q>$file</q> variable. You should provide file name in this variable. Example can be “nature.zip”, “nature.jpg” etc.</p>
            <h2>Array: $args<strong><br> </strong></h2>
            <ul>
             <li><strong>download_path (String – Absolute Path):</strong> It will hold value of $download_path.</li>
             <li><strong>file (String – Valid File Name):</strong> It will hold value of $file.</li>
             <li><strong>extension_check (boolean TRUE | FALSE):</strong> It is TRUE by default, It will give you a flexible control to offer particular files to download with this PHP download script. You can set it to FALSE, if you donot want to control over file types and willing to download all types of files.</li>
             <li><strong>referrer_check (boolean TRUE | FALSE):</strong> It is FALSE by default, It will provide you a flexible control over hot-linking. You have to set “referrer” after making it TRUE. It will allow download files from a server which will match “referrer” value only.</li>
             <li><strong>referrer (String):</strong> You have to set its value after setting “referrer_check” TRUE. An example of its value can be “tutorialchip.com” so download will only be offered from tutorialchip server request only.</li>
            </ul>
            <h2>Class Instance</h2>
            <p>It is time to instance PHP download file script class now.</p>
            <pre class="brush: php; title: ; notranslate" title="">$download_path = "E:/wamp/www/tutorialchip/download/";
$file = $_REQUEST['f'];

$args = array(
		'download_path'		=&gt;	$download_path,
		'file'				=&gt;	$file,
		'extension_check'	=&gt;	TRUE,
		'referrer_check'	=&gt;	FALSE,
		'referrer'			=&gt;	NULL,
		);
$download = new chip_download( $args );
</pre>
            <h2>Pre Download Hook</h2>
            <p>PHP download class will do its job, and provide you a Hook to execute your code or changes before throwing download stream to the users. You can check available options of Pre Download Hook by executing following code.</p>
            <pre class="brush: php; title: ; notranslate" title="">$download_hook = $download-&gt;get_download_hook();
//$download-&gt;chip_print($download_hook);
//exit;
</pre>
            <p>You can remove comments of last two lines to stop the execution of code and check the available options. A sample output of successful $download_hook array can be,</p>
            <pre class="brush: php; title: ; notranslate" title="">Array
(
    [file_path] =&gt; E:/wamp/www/tutorialchip/download/nature.jpg
    [file] =&gt; nature.jpg
    [file_name] =&gt; nature
    [file_extension] =&gt; jpg
    [file_size] =&gt; 97854
    [file_mime_type] =&gt; image/jpeg; charset=binary
    [download] =&gt; 1
    [message] =&gt; File is ready to download
)
</pre>
            <p>A sample output of non successful $download_hook array can be,</p>
            <pre class="brush: php; title: ; notranslate" title="">Array
(
    [file_path] =&gt; E:/wamp/www/tutorialchip/downloads/nature.jpg
    [download] =&gt;
    [message] =&gt; Invalid File or File Path.
)
</pre>
            <p>Download key of the $download_hook array will be TRUE in case of successful execution and FALSE in case of any error.</p>
            <h2>Download Initialization</h2>
            <p>It is the last step of PHP download file script settings. You can make the best use of Download key of the $download_hook array now.</p>
            <pre class="brush: php; title: ; notranslate" title="">if( $download_hook['download'] == TRUE ) {

	/* You can write your logic before proceeding to download */

	/* Let's download file */
	$download-&gt;get_download();

}
</pre>
            <h2>Complete Tutorial – PHP File Download Example</h2>
            <p>I have already explained the usage of PHP file download class in steps. Now it is time to see the whole tutorial at once.</p>
            <pre class="brush: php; title: ; notranslate" title="">&lt;?php
/*
|-----------------
| Chip Download Class
|------------------
*/

require_once("class.chip_download.php");

/*
|-----------------
| Class Instance
|------------------
*/

$download_path = "E:/wamp/www/tutorialchip/download/";
$file = $_REQUEST['f'];

$args = array(
		'download_path'		=&gt;	$download_path,
		'file'				=&gt;	$file,
		'extension_check'	=&gt;	TRUE,
		'referrer_check'	=&gt;	FALSE,
		'referrer'			=&gt;	NULL,
		);
$download = new chip_download( $args );

/*
|-----------------
| Pre Download Hook
|------------------
*/

$download_hook = $download-&gt;get_download_hook();
//$download-&gt;chip_print($download_hook);
//exit;

/*
|-----------------
| Download
|------------------
*/

if( $download_hook['download'] == TRUE ) {

	/* You can write your logic before proceeding to download */

	/* Let's download file */
	$download-&gt;get_download();

}

?&gt;
</pre>
