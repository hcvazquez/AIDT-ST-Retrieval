                <li><span class="title" title="Eclipse Java development tools (binary runtime and user documentation).">Eclipse Java Development Tools</span></li>
                <li><span class="title" title="m2e is a Maven integration in Eclipse. Launching Maven from Eclipse, dependency management and search.">Maven Integration for Eclipse</span></li>
                <li><span class="title" title="Provides the Task List for task and time management.">Mylyn Task List</span></li>
