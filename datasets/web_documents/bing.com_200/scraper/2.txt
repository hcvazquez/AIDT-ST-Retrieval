       </svg></a>wscraper</h1> 
     <p class="package-description-redundant">wscraper.js is a web scraper agent written in node.js and based on <a href="https://github.com/MatthewMueller/cheerio">cheerio.js</a> a fast, flexible, and lean implementation of core jQuery; It is built on top of <a href="https://github.com/mikeal/request">request.js</a> and inspired by <a href="https://github.com/indexzero/http-agent">http-agent.js</a>;</p> 
     <h2><a id="user-content-usage" class="deep-link" href="#usage">
       <svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" width="16">
        <path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
       </svg></a>Usage</h2> 
     <p>There are two ways to use wscraper: http agent mode and local mode.</p> 
     <h3><a id="user-content-http-agent-mode" class="deep-link" href="#http-agent-mode">
       <svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" width="16">

       </svg></a>HTTP Agent mode</h3> 
     <p>In HTTP Agent mode, pass it a host, a list of URLs to visit and a scraping JS script. For each URLs, the agent makes a request, gets the response, runs the scraping script and returns the result of the scraping. Valid usage is:</p>
