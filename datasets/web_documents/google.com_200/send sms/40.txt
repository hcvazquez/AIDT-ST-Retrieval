           <h1 name="f695" id="f695" class="graf graf--h3 graf-after--figure graf--title">Easy SMS integration with&nbsp;Node.js</h1>
           <p name="3694" id="3694" class="graf graf--p graf-after--h3"><strong class="markup--strong markup--p-strong">Being a technology driven company, we understand how important it is to provide easy integration with the latest and greatest code libraries. That’s why we’re happy to announce a new addition to our list: the MessageBird Node.js package!</strong></p>
           <p name="e9cb" id="e9cb" class="graf graf--p graf-after--p">Over the summer we were playing around with Node.js at our internal hackathon and thought that it’d be cool if we had a MessageBird package. Turns out, there already was one! We reached out to the author, Franklin van de Meent, and asked him if he’d like to collaborate in building the next version. He accepted, and together we built version 2 of the MessageBird package, which we are releasing today!</p>
           <p name="9e0b" id="9e0b" class="graf graf--p graf-after--p">If you haven’t heard of <a href="https://nodejs.org/" data-href="https://nodejs.org/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Node.js</a> before, it’s a Javascript-driven runtime environment which has become very popular in recent years due to its low barrier to entry; getting something up and running is very quick and straightforward. It’s perfect for building powerful and realtime applications: Paypal uses it extensively for their payments interface, Github’s Atom editor is powered by it and the new WordPress runs on it.</p>
           <p name="af2b" id="af2b" class="graf graf--p graf-after--p">Let me show you how quickly you can build something on Node.js by building a <strong class="markup--strong markup--p-strong">Hello, World!</strong> application, using the MessageBird package.</p>
           <h4 name="2c7e" id="2c7e" class="graf graf--h4 graf-after--p">Building a Hello, world! application</h4>
           <p name="4d5a" id="4d5a" class="graf graf--p graf-after--h4">Let’s dive straight in. The goal is to have the application send you an SMS containing “Hello, world!” when you run it. Note that, in order to run the code below, you’ll need to have <a href="https://nodejs.org/" data-href="https://nodejs.org/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Node.js</a> and <a href="https://www.npmjs.com/" data-href="https://www.npmjs.com/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">npm</a> (Node Package Manager) installed, and you’ll also need to<a href="https://www.messagebird.com/app/en/sign-up" data-href="https://www.messagebird.com/app/en/sign-up" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">sign up for a MessageBird account</a>.</p>
           <p name="69a8" id="69a8" class="graf graf--p graf-after--p">The first thing you’ll want to do is include the MessageBird package in your project. You can do that easily by running npm install messagebird in your terminal.</p>
           <p name="4db2" id="4db2" class="graf graf--p graf-after--p">The next step is to write the application. Create a filed called index.js in your project folder and copy the code below, replacing both the access key and the phone number with your own values (you can find and create an access key in your <a href="https://www.messagebird.com/app/en/settings/developers/access" data-href="https://www.messagebird.com/app/en/settings/developers/access" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">dashboard</a>).</p>

           <p name="fc24" id="fc24" class="graf graf--p graf-after--figure">What’s happening in the code? Then, we create an array of parameters to send to the API, and lastly, we send the request, making sure we catch any error that might occur.</p>
           <p name="ce1f" id="ce1f" class="graf graf--p graf-after--p">Execute the code by running node index.js and if all is well, you should receive an SMS saying “Hello, world!”. Success ?!</p>
