         <h1 class="entry-title"> C Program to Implement Quick Sort Using Randomization </h1> 
         <div class="entry-meta">
           Posted on 
          <a href="http://www.sanfoundry.com/c-program-implement-quick-sort-using-randomization/" title="4:36 pm" rel="bookmark"> <time class="entry-date updated" datetime="2014-03-19T16:36:07+00:00">March 19, 2014</time> </a>
          <span class="byline"> by <span class="author vcard"> <a class="url fn n" href="http://www.sanfoundry.com/author/cguy3/" title="View all posts by cguy3" rel="author">cguy3</a> </span> </span> 
         </div>
         <!-- .entry-meta --> 
        </header>
        <!-- .entry-header --> 
        <div class="entry-content"> 
         <div style="text-align:justify">
           This C program sorts a given array of integer numbers using randomized Quick sort technique.
          <p></p> 
          <p>It is comparison sort which works on divide and conquer technique. It is in-place partitioning algorithm and one of the fastest sorting techniques available till date. Average Time Complexity of this algorithm is O(nlog(n)).</p> 
          <p>Here is the source code of the C program to sort integers using randomized Quick sort technique. The C program is successfully compiled and run on a Linux system. The program output is also shown below. </p>
         </div>

              <div class="hk1_style">
