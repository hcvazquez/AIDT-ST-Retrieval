               <br> In this post, we are going to learn about speech to text conversion using java speech API.
               <br> We will be discussing the following topics :
               <br> 
               <br> 
               <ul style="text-align: left;"> 
                <li>Brief about Speech Recognition.</li> 
                <li>Setup Speech Recognition engine</li> 
                <li>Verifying the Speech Recognition engine installation</li> 
                <li>Sample code</li> 
                <li>Conclusion</li> 
               </ul> 
               <div> 
                <div> 
                 <div class="separator" style="clear: both; text-align: center;"> 
                 </div> 
                 <br> 
                 <img src="https://images-blogger-opensocial.googleusercontent.com/gadgets/proxy?url=http%3A%2F%2F4.bp.blogspot.com%2F-mTLmJ1JW0o0%2FU77Vk1X_XUI%2FAAAAAAAAA0Y%2FNM0_CIcxKXU%2Fs1600%2Fidea.png&amp;container=blogger&amp;gadget=a&amp;rewriteMime=image%2F*">
                </div> 
                <div>
                  Speech Recognition is the process of converting spoken input to digital output, such as text.
                </div> 
                <div>
                  Speech recognition systems provide computers with the ability to listen to user speech and determine what is said.
                </div> 
                <div> 
                 <br>
                </div> 
                <div>
                  The Speech Recognition process can be divided into these four steps:
                </div> 
                <div> 
                 <ol style="text-align: left;"> 
                  <li>Speech is converted to digital signals.</li> 
                  <li>Actual speech sounds are extracted from the sounds (based on energy of the sounds).</li> 
                  <li>The extracted sounds are put together into 'speech frames.'</li> 
                  <li>The speech frames are compared with words from the grammar file to determine the spoken word.</li> 
                 </ol> 
                 <div> 
                  <blockquote class="tr_bq" style="text-align: left;">
                    We are going to use a third party java speech recognizer engine 
                   <a href="http://www.cloudgarden.com/JSAPI/index.html" target="_blank">TalkingJava SDK</a>&nbsp;which&nbsp;is a full implementation of Sun's 
                   <a href="http://www.oracle.com/technetwork/java/index.html" target="_blank">Java Speech API</a> providing Text-To-Speech and Speech-Recognition engines.
                  </blockquote> 
                  <br> 
                  <blockquote class="tr_bq"> 
                   <span style="color: red;">TalkingJava SDK website has been shutdown. I have shared the old SDK You can download from here. <a href="https://drive.google.com/file/d/0B5T8zoos6HTeS18xX1J1ODRoVFU/view?usp=sharing" target="_blank">Click Here</a></span>&nbsp;
                  </blockquote> 
                 </div> 
                </div> 
               </div> 
               <div> 
                <div class="separator" style="clear: both; text-align: center;">

                <div class="separator" style="clear: both; text-align: center;">

                <br> 
                <img src="https://images-blogger-opensocial.googleusercontent.com/gadgets/proxy?url=http%3A%2F%2F4.bp.blogspot.com%2F-Jr_tV4-AJnQ%2FU7-42aiRq5I%2FAAAAAAAAA1k%2FbfG8-vUNjxE%2Fs1600%2Fsetup.png&amp;container=blogger&amp;gadget=a&amp;rewriteMime=image%2F*">
                <br> 
                <b>Four easy step to set up &nbsp;TalkingJava SDK</b>
                <br> 
                <div> 
                 <ol style="text-align: left;"> 
                  <li><span style="text-align: center;">Download TalkingJava SDK Installer file : <a href="https://drive.google.com/file/d/0B5T8zoos6HTeS18xX1J1ODRoVFU/view?usp=sharing" target="_blank">Download</a><span id="goog_294971819"></span><span id="goog_294971820"></span><a href="https://www.blogger.com/"></a></span></li> 
                  <li>Click on&nbsp;<i>TalkingJavaSDK-1xx.jar</i>&nbsp;and follow the instruction to install SDK.</li> 
                  <li>Unpack &nbsp;<i>TalkingJavaSDK-1xx.jar </i>file. Then unpack&nbsp;<i>packet.jar</i>&nbsp;(available inside TalkingJavaSDK-1xx directory).</li> 
                  <li>Copy cgjsapi.jar and cgjsapi1xx.dll files available inside packet directory to your Java\jdk1.x.x.x\jre\lib\ext\ &nbsp;direcoty.</li> 
                  <li>Include your <i>packet</i> directory path into your CLASSPATH environment variable.</li> 
                 </ol> 
                </div> 
                <blockquote class="tr_bq" style="text-align: center;">
                  You can also refer to the installation guide here -
                 <a href="http://www.cloudgarden.com/JSAPI/docs/installation.html" target="_blank">cloudgarden installation</a>
                </blockquote> 
               </div> 
               <div> 
                <br> 
                <img src="https://images-blogger-opensocial.googleusercontent.com/gadgets/proxy?url=http%3A%2F%2F3.bp.blogspot.com%2F-RKxmceYhwXM%2FU7-8IDDzieI%2FAAAAAAAAA1w%2FCUYz-ZgdNDY%2Fs1600%2FVerify.png&amp;container=blogger&amp;gadget=a&amp;rewriteMime=image%2F*">
                <br> 
                <div class="separator" style="clear: both; text-align: center;"> 
                </div> We are going to verify whether the speech recognition engine has been installed successfully or not using a simple java program.
               </div> 
               <div> 
                <b>TestRecognizerConfig.java</b>
               </div> 
               <pre class="brush:java">package com.sarf.talkingjava;
package com.sarf.talkingjava;

import java.util.Locale;
import javax.speech.Central;
import javax.speech.EngineList;
import javax.speech.recognition.RecognizerModeDesc;

public class TestRecognizerConfig {
  public static void main(String[] args) {
  try
    {
     Central.registerEngineCentral 
      ("com.cloudgarden.speech.CGEngineCentral");
     RecognizerModeDesc desc =  
       new RecognizerModeDesc(Locale.US,Boolean.TRUE);
     EngineList el = Central.availableRecognizers(desc);
     if(el.size() &lt; 1){
        System.out.println("Recognition Engine is not available");
        System.exit(1);
        }else{
        System.out.println("Recognition Engine is available");
        System.exit(1);
        }
     }catch(Exception exception)
     {
      exception.printStackTrace();
      }
   }
}
</pre> Run this class file and check the output. If output is&nbsp;Recognition Engine is available&nbsp;then you have installed recognition engine successfully and you are good to go.

               <br> 
               <img src="https://images-blogger-opensocial.googleusercontent.com/gadgets/proxy?url=http%3A%2F%2F1.bp.blogspot.com%2F-EqAAIfpfDY8%2FU77Vj_iSxcI%2FAAAAAAAAA0I%2FAtjqSGOZHsU%2Fs1600%2FhereWeGo.png&amp;container=blogger&amp;gadget=a&amp;rewriteMime=image%2F*">
               <br> 
               <div class="separator" style="clear: both; text-align: center;"> 
               </div> 
               <b>SpeechToTextConverter.java</b>
               <br> 
               <pre class="brush:java">package com.sarf.talkingjava;

import javax.speech.Central;
import javax.speech.recognition.*;
import java.io.FileReader;
import java.util.Locale;

public class SpeechToTextConverter extends ResultAdapter {
  static Recognizer recognizer;
  public void resultAccepted(ResultEvent resultEvent) {
    Result result = (Result)(resultEvent.getSource());
    ResultToken resultToken[] = result.getBestTokens();
    for (int nIndex = 0; nIndex &lt; resultToken.length; nIndex++){
        System.out.print(resultToken[nIndex].getSpokenText() + " ");
    }
    try {      
         // Deallocate the recognizer
         recognizer.forceFinalize(true);          
         recognizer.deallocate();
        }catch (Exception exception) {
         exception.printStackTrace();
        }
       System.exit(0);
    }

    public static void main(String args[]) {
    try {
          Central.registerEngineCentral 
             ("com.cloudgarden.speech.CGEngineCentral");
          RecognizerModeDesc desc = 
              new RecognizerModeDesc(Locale.US,Boolean.TRUE);
         // Create a recognizer that supports US English.
         recognizer = Central.createRecognizer(desc);
         // Start up the recognizer
         recognizer.allocate();
         // Load the grammar from a file, and enable it
         FileReader fileReader =  
             new FileReader("D:\\my_grammar.grammar");
         RuleGrammar grammar = recognizer.loadJSGF(fileReader);
         grammar.setEnabled(true);
         // Add the listener to get results
         recognizer.addResultListener(new SpeechToTextConverter());
         // Commit the grammar
         recognizer.commitChanges();
         recognizer.waitEngineState(Recognizer.LISTENING);
         // Request focus and start listening
         recognizer.requestFocus();
         recognizer.resume();
         recognizer.waitEngineState(Recognizer.FOCUS_ON);
         recognizer.forceFinalize(true);               
         recognizer.waitEngineState(Recognizer.DEALLOCATED);
        } catch (Exception e) {
          e.printStackTrace();
          System.exit(0);
         }
   }
}
</pre> 
               <b>A Little Grammer</b>
               <br> 
               <blockquote class="tr_bq">
                 The JSpeech Grammar Format (JSGF) is a platform-independent, vendor-independent textual representation of grammars for use in speech recognition. Grammars are used by speech recognizers to determine what the recognizer should listen for, and so describe the utterances a user may say. JSGF adopts the style and conventions of the JavaTM Programming Language in addition to use of traditional grammar notations. To know more about 
                <a href="http://www.w3.org/TR/jsgf/" target="_blank">JSGF</a>.&nbsp;
               </blockquote> 
               <b>Here is the content of &nbsp;<i>my_grammar.grammar</i> file.</b>
               <br> 
               <br> 
               <i>#JSGF V1.0;</i>
               <br> 
               <i><br></i> 
               <i>grammar com.sarf.talkingjava.example;</i>
               <br> 
               <i><br></i> 
               <i>public &lt;startExample&gt; = (please | My name is sarf |&nbsp;</i>
               <br> 
               <i>What is your Name | Open Firefox | Open notepad |Open grammar |</i>
               <br> 
               <i>Please to meet you ) *;</i>
               <br> 
               <i>public &lt;endExample&gt; = [thanks | thank you | thank you very much];</i>
               <br>
