        <p>At the moment there's no way to specify the title of a gauge chart as you can with other Google Charts. In the example above, simple HTML is used to display the title.</p> 
        <p>Also, the <code>animation.startup</code> option available for many other Google Charts is not available for the Gauge Chart. If you'd like a startup animation, draw the chart initially with values set to zero, and then draw again with the value you'd like it to animate to.</p> 
        <h2><a name="Loading" id="Loading"></a>Loading</h2> 
        <p> The <code>google.charts.load</code> package name is <code>"gauge"</code>.</p> 
        <pre class="prettyprint">
  google.charts.load('current', {packages: ['gauge']});
</pre> 
        <p> The visualization's class name is <code>google.visualization.Gauge</code>. </p> 
        <pre class="prettyprint">
  var visualization = new google.visualization.Gauge(container);
</pre> 
        <h2><a name="Data_Format" id="Data_Format"></a>Data Format</h2> 
        <p> Each numeric value is displayed as a gauge. Two alternative data formats are supported: </p> 
        <ol> 
         <li> Two columns. The first column should be a string, and contain the gauge label. The second column should be a number, and contain the gauge value. </li> 
         <li> Any number of numeric columns. The label of each gauge is the column's label. </li> 
        </ol>
