                          <div class="jive-rendered-content">
                           <p>I recommend that you get an IDE if you don't have one.&nbsp; I recommend WebStorm, but it's only free for a month, then you'll have to pay.&nbsp; Some alternatives are Visual Studio, Aptana, search this forum for others</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>next you'll go on your command line and navigate to your project directory, mine is C:\projects\tk106\server\src</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>the next step will walk you through creating your package.json file, it will ask you want you want to call your application file, it defaults to index.js, I call mine server.js, so anywhere you see server.js below, replace it with the name you give your file</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>type 'npm init'</p>
                           <p>This will help you create your package.json file which is the package manager for your file.&nbsp; It will ask you a few questions, then create the file.</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>I believe you said you installed tk102 already, if so, skip this step:</p>
                           <p>&nbsp;&nbsp;&nbsp;&nbsp; Once the file is created, type 'npm install tk102 --save'</p>
                           <p>&nbsp;&nbsp;&nbsp;&nbsp; the --save option will be sure to add the dependency into your package.json file</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>My personal preference is to install nodemon as well, so that my server auto restarts as I make changes</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>I do that by typing 'npm install nodemon --save-dev'</p>
                           <p>This will add nodeman as a dependency to your development environment inside your package.json file</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>if you installed nodemon do the following:</p>
                           <p>&nbsp;&nbsp;&nbsp; Now open your package.json file</p>
                           <p>&nbsp;&nbsp;&nbsp; remove the line "main": "server.js",</p>
                           <p>&nbsp;&nbsp;&nbsp; add this line in the scripts object above test</p>
                           <p>&nbsp;&nbsp;&nbsp; "start": "NODE_ENV=local ./node_modules/.bin/nodemon server.js",</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>Depending on what you stated your default node file would be, create that file.&nbsp; I created a file named server.js</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>Open your server.js file and follow the example from the tk106 repo:</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>type the following:</p>
                           <p>------------------------------------------------------------------------------------</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>'use strict';</p>
                           <p>var server = require('tk102');</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>server.createServer({</p>
                           <p>&nbsp;&nbsp;&nbsp; port: 1337</p>
                           <p>});</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>server.on( 'listening', function( listen ) {</p>
                           <p>&nbsp;&nbsp;&nbsp; console.log('listening on port: ' + listen.port);</p>
                           <p>});</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>server.on( 'data', function( raw ) {</p>
                           <p>&nbsp;&nbsp;&nbsp; console.log( 'Incoming data: '+ raw )</p>
                           <p>});</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>server.on( 'track', function( gps ) {</p>
                           <p>&nbsp;&nbsp;&nbsp; console.log( gps )</p>
                           <p>});</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>------------------------------------------------------------------------------------</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>at this point, you should be able connect your device to your server.&nbsp; if things don't work, then you'll have to dig into the tk102 server source code and update it. it's located in node_modules\tk102\tk102.js</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>if you attach the manual, I may be able to help further as any errors will probably be in the parser.&nbsp; Hopefully you get something in the console.log( 'Incoming data: '+ raw ) statement</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>let me know how that goes.</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>sometimes you may have to stop node from running, in that case I run this on my mac</p>
                           <p>lsof -n -i :1337</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>COMMAND&nbsp; PID&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; USER&nbsp;&nbsp; FD&nbsp;&nbsp; TYPE&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; DEVICE SIZE/OFF NODE NAME</p>
                           <p>node&nbsp;&nbsp;&nbsp; 1887 alexbostic&nbsp;&nbsp; 14u&nbsp; IPv4 0x7e6cfdb8acf18bf5&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 0t0&nbsp; TCP *:hbci (LISTEN)</p>
                           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                           <p>sudo kill -9 1887</p>
                           <p>lsof -n -i :1337</p>
                           <p>npm start</p>
