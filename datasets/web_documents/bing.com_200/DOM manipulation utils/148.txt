        <h1>How to read and parse CSV file in Java</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="http://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2013-04-25T00:07:10+00:00">April 25, 2013</time> | Updated : July 18, 2016 | Viewed : 980,416 times <span>+5,728 pv/w</span></p> 
        </div> 
        <!-- /.post-meta --> 
        <div class="post-content"> 
         <div class="pic"> 
          <img src="http://www.mkyong.com/wp-content/uploads/2013/04/country-csv-example.png" alt="country-csv-example" width="592" height="206" class="aligncenter size-full wp-image-14018" srcset="http://www.mkyong.com/wp-content/uploads/2013/04/country-csv-example.png 592w, http://www.mkyong.com/wp-content/uploads/2013/04/country-csv-example-300x104.png 300w" sizes="(max-width: 592px) 100vw, 592px"> 
         </div> 
         <p>A Comma-Separated Values (CSV) file is just a normal plain-text file, store data in column by column, and split it by a separator (e.g normally it is a comma “,”). </p> 
         <p>For example :</p> 
         <pre>
1,US,United States
2,MY,Malaysia
3,AU,Australia
</pre> 
         <p>or </p> 
         <pre>
"1","US","United States"
"2","MY","Malaysia"
"3","AU","Australia"
</pre> 
         <div class="note"> 
          <strong>Note</strong>
          <br> Read this 
          <a href="https://tools.ietf.org/html/rfc4180" target="_blank">RFC4180</a> document for Comma-Separated Values (CSV) format. 
         </div> 
         <p>In a CSV file, normally there are two issues:</p> 
         <p>1. The field containing separator, for example, separator is a comma, and the field containing commas:</p> 
         <pre>
"aaa","b,bb","ccc"
</pre> 
         <p>2. The double-quotes are used to enclose fields, and the field containing double-quotes. To solve it, a double-quote appearing inside a field must be escaped by preceding it with another double quote (<a href="https://tools.ietf.org/html/rfc4180" target="_blank">RFC4180</a>)</p> 
         <pre>
"aaa","b""bb","ccc"
</pre> 
         <p>In this tutorial, we show you three examples to read, parse and print out the values from a CSV file.</p> 
         <ol> 
          <li>Simple solution to parse a simple formatted CSV file.</li> 
          <li>Advance solution to parse a weird formatted CSV file (field containing separator or double-quotes)</li> 
          <li>Third party solution, OpenCSV example.</li> 
         </ol> 
         <h2>1. Simple Solution</h2> 
         <p>If you are sure the CSV files doesn’t contain “separator or double-quotes”, just use the standard <code>split()</code> to parse the CSV file.</p> 
         <p>1.1 Review a simple CSV file</p> 
         <div class="filename">
          /Users/mkyong/csv/country.csv
         </div> 
         <pre><code class="language-csv">"1.0.0.0","1.0.0.255","16777216","16777471","AU","Australia"
"1.0.1.0","1.0.3.255","16777472","16778239","CN","China"
"1.0.4.0","1.0.7.255","16778240","16779263","AU","Australia"
"1.0.8.0","1.0.15.255","16779264","16781311","CN","China"
"1.0.16.0","1.0.31.255","16781312","16785407","JP","Japan"
"1.0.32.0","1.0.63.255","16785408","16793599","CN","China"
"1.0.64.0","1.0.127.255","16793600","16809983","JP","Japan"
"1.0.128.0","1.0.255.255","16809984","16842751","TH","Thailand"
</code></pre> 
         <p>1.2 No magic, just read above text file, and splits it by a comma separator.</p> 
         <div class="filename">
          CSVReader.csv
         </div> 
         <pre><code class="language-java">
package com.mkyong.csv;

import java.io.BufferedReader;
import java.io.FileNotFoundException;
import java.io.FileReader;
import java.io.IOException;

public class CSVReader {

    public static void main(String[] args) {

        String csvFile = "/Users/mkyong/csv/country.csv";
        BufferedReader br = null;
        String line = "";
        String cvsSplitBy = ",";

        try {

            br = new BufferedReader(new FileReader(csvFile));
            while ((line = br.readLine()) != null) {

                // use comma as separator
                String[] country = line.split(cvsSplitBy);

                System.out.println("Country [code= " + country[4] + " , name=" + country[5] + "]");

            }

        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (br != null) {
                try {
                    br.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

    }

}
</code></pre> 
         <p>1.3 For JDK 7 and above, uses try-resources.</p> 
         <div class="filename">
          CSVReader.csv
         </div> 
         <pre><code class="language-java">

         <h2>2. Advance Solution</h2> 
         <p>This solution will solve the field containing “separator or double-quotes” issue, and also support the custom separator and custom enclosed field. Review the following CSV parsing example and also the JUnit test cases to understand how it works.</p> 
         <div class="note"> 
          <strong>Note</strong>
          <br> Again, if a double-quote appearing inside a field must be escaped by preceding it with another double quote, for example 
          <p></p> 
          <pre>
"aaa","b""bb","ccc"
</pre> 
         </div> 
         <p>2.1 Review another CSV file</p> 
         <div class="filename">
          /Users/mkyong/csv/country2.csv
         </div> 
         <pre><code class="language-csv">10,AU,Australia
11,AU,Aus""tralia
"12","AU","Australia"
"13","AU","Aus""tralia"
"14","AU","Aus,tralia"
</code></pre> 
         <p>2.2 The below example is inspired by this article – “<a href="https://agiletribe.wordpress.com/2012/11/23/the-only-class-you-need-for-csv-files/" target="_blank">Only Class You Need for CSV Files</a>” (with some fixed to support other features, read “fixed comments”) and also this third party <a href="http://opencsv.sourceforge.net/" target="_blank">OpenCSV</a> library.</p> 
         <div class="filename">
          CSVUtils.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.utils;

import java.io.File;
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class CSVUtils {

    private static final char DEFAULT_SEPARATOR = ',';
    private static final char DEFAULT_QUOTE = '"';

    public static void main(String[] args) throws Exception {

        String csvFile = "/Users/mkyong/csv/country2.csv";

        Scanner scanner = new Scanner(new File(csvFile));
        while (scanner.hasNext()) {
            List&lt;String&gt; line = parseLine(scanner.nextLine());
            System.out.println("Country [id= " + line.get(0) + ", code= " + line.get(1) + " , name=" + line.get(2) + "]");
        }
        scanner.close();

    }

    public static List&lt;String&gt; parseLine(String cvsLine) {
        return parseLine(cvsLine, DEFAULT_SEPARATOR, DEFAULT_QUOTE);
    }

    public static List&lt;String&gt; parseLine(String cvsLine, char separators) {
        return parseLine(cvsLine, separators, DEFAULT_QUOTE);
    }

    public static List&lt;String&gt; parseLine(String cvsLine, char separators, char customQuote) {

        List&lt;String&gt; result = new ArrayList&lt;&gt;();

        //if empty, return!
        if (cvsLine == null &amp;&amp; cvsLine.isEmpty()) {
            return result;
        }

        if (customQuote == ' ') {
            customQuote = DEFAULT_QUOTE;
        }

        if (separators == ' ') {
            separators = DEFAULT_SEPARATOR;
        }

        StringBuffer curVal = new StringBuffer();
        boolean inQuotes = false;
        boolean startCollectChar = false;
        boolean doubleQuotesInColumn = false;

        char[] chars = cvsLine.toCharArray();

        for (char ch : chars) {

            if (inQuotes) {
                startCollectChar = true;
                if (ch == customQuote) {
                    inQuotes = false;
                    doubleQuotesInColumn = false;
                } else {

                    //Fixed : allow "" in custom quote enclosed
                    if (ch == '\"') {
                        if (!doubleQuotesInColumn) {
                            curVal.append(ch);
                            doubleQuotesInColumn = true;
                        }
                    } else {
                        curVal.append(ch);
                    }

                }
            } else {
                if (ch == customQuote) {

                    inQuotes = true;

                    //Fixed : allow "" in empty quote enclosed
                    if (chars[0] != '"' &amp;&amp; customQuote == '\"') {
                        curVal.append('"');
                    }

                    //double quotes in column will hit this!
                    if (startCollectChar) {
                        curVal.append('"');
                    }

                } else if (ch == separators) {

                    result.add(curVal.toString());

                    curVal = new StringBuffer();
                    startCollectChar = false;

                } else if (ch == '\r') {
                    //ignore LF characters
                    continue;
                } else if (ch == '\n') {
                    //the end, break!
                    break;
                } else {
                    curVal.append(ch);
                }
            }

        }

        result.add(curVal.toString());

        return result;
    }

}

         <p>3.2 OpenCSV example to parse above CSV file.</p> 
         <div class="filename">
          CSVReaderExample.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.csv;

import com.opencsv.CSVReader;

import java.io.FileReader;
import java.io.IOException;

public class CSVReaderExample {

    public static void main(String[] args) {

        String csvFile = "/Users/mkyong/csv/country3.csv";

        CSVReader reader = null;
        try {
            reader = new CSVReader(new FileReader(csvFile));
            String[] line;
            while ((line = reader.readNext()) != null) {
                System.out.println("Country [id= " + line[0] + ", code= " + line[1] + " , name=" + line[2] + "]");
            }
        } catch (IOException e) {
            e.printStackTrace();
        }


    }

}
