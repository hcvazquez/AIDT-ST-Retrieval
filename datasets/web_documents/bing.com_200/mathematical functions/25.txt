        <h2>Introduction</h2> 
        <p>The <b>java.math.BigDecimal</b> class provides operations for arithmetic, scale manipulation, rounding, comparison, hashing, and format conversion.</p> 
        <p>The toString() method provides a canonical representation of a BigDecimal. It gives the user complete control over rounding behavior.</p> 
        <p>Two types of operations are provided for manipulating the scale of a BigDecimal ?</p> 
        <ul class="list"> 
         <li>scaling/rounding operations</li> 
         <li>decimal point motion operations</li> 
        </ul> 
        <p>This class and its iterator implement all of the optional methods of the <b>Comparable</b> interfaces.</p> 
        <h2>Class declaration</h2> 
        <p>Following is the declaration for <b>java.math.BigDecimal</b> class ?</p> 
        <pre class="result notranslate">
public class BigDecimal
   extends Number
      implements Comparable&lt;BigDecimal&gt;
</pre> 
        <h2>Field</h2> 
        <p>Following are the fields for <b>java.math.BigDecimal</b> class ?</p> 
        <ul class="list"> 
         <li><p><b>static BigDecimal ONE</b> ? The value 1, with a scale of 0.</p></li> 
         <li><p><b>static int ROUND_CEILING</b> ? Rounding mode to round towards positive infinity.</p></li> 
         <li><p><b>static int ROUND_DOWN</b> ? Rounding mode to round towards zero.</p></li> 
         <li><p><b>static int ROUND_FLOOR</b> ? Rounding mode to round towards negative infinity.</p></li> 
         <li><p><b>static int ROUND_HALF_DOWN</b> ? Rounding mode to round towards "nearest neighbor" unless both neighbors are equidistant, in which case round down.</p></li> 
         <li><p><b>static int ROUND_HALF_EVEN</b> ? Rounding mode to round towards the "nearest neighbor" unless both neighbors are equidistant, in which case, round towards the even neighbor.</p></li> 
         <li><p><b>static int ROUND_HALF_UP</b> ? Rounding mode to round towards "nearest neighbor" unless both neighbors are equidistant, in which case round up.</p></li> 
         <li><p><b>static int ROUND_UNNECESSARY</b> ? Rounding mode to assert that the requested operation has an exact result, hence no rounding is necessary.</p></li> 
         <li><p><b>static int ROUND_UP</b> ? Rounding mode to round away from zero.</p></li> 
         <li><p><b>static BigDecimal TEN</b> ? The value 10, with a scale of 0.</p></li> 
         <li><p><b>static BigDecimal ZERO</b> ? The value 0, with a scale of 0.</p></li> 
        </ul>
