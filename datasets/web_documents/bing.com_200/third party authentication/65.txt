           <h1 name="5fa5" id="5fa5" class="graf graf--h3 graf-after--figure graf--title">Notes: Passport and&nbsp;OAuth</h1>
           <p name="11ea" id="11ea" class="graf graf--p graf-after--h3">What is OAuth? What is Passport? How do I use them together?</p>
           <p name="2040" id="2040" class="graf graf--p graf-after--p"><em class="markup--em markup--p-em">Disclaimer: This post is not meant to be a full tutorial on its own, but rather a supplement to future tutorials.</em></p>
           <blockquote name="ec28" id="ec28" class="graf graf--blockquote graf-after--p">
            <strong class="markup--strong markup--blockquote-strong">(&nbsp;! )</strong>

           <h4 name="2847" id="2847" class="graf graf--h4 graf-after--blockquote">Prerequisite knowledge</h4>
           <p name="6d38" id="6d38" class="graf graf--p graf-after--h4">You need to know some JavaScript, Node, and Express.</p>
           <h4 name="1b08" id="1b08" class="graf graf--h4 graf-after--p">What is&nbsp;OAuth?</h4>
           <p name="157a" id="157a" class="graf graf--p graf-after--h4">OAuth stands for Open Authorization, it’s for using third-party providers (i.e Facebook, Google, GitHub, etc) to handle authentication and authorization. You have seen it before, think about all of those social sign in buttons; “sign in with Google”, “sign in with Facebook”…</p>

           <h4 name="7ad6" id="7ad6" class="graf graf--h4 graf-after--figure">What is Passport?</h4>
           <p name="37bf" id="37bf" class="graf graf--p graf-after--h4">Passport.js is a JavaScript library / Express middleware for handling authentication. It is <strong class="markup--strong markup--p-strong">not</strong> limited to using only OAuth. It uses different “strategies” for handling authentication. You can use over <a href="http://passportjs.org/" data-href="http://passportjs.org/" class="markup--anchor markup--p-anchor" rel="noopener nofollow" target="_blank">300+ strategies for authentication</a>. A lot of it can seem like magic and the overwhelming number of strategies can make Passport difficult to understand for people learning it.</p>

           <h4 name="4ad1" id="4ad1" class="graf graf--h4 graf-after--p">When to use&nbsp;OAuth?</h4>
           <p name="005a" id="005a" class="graf graf--p graf-after--h4">Whenever you want to allow your users to easily sign in with other accounts. It’s very popular and makes it simple for your users.</p>
