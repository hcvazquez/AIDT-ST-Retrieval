          This is class can be used as a tool for optical character recognition.
          <br> 
          <br> It can recognize text in monochrome graphical images after a training phase. The training phase is necessary to let the class build recognition data structures from images that have known characters.
          <br> 
          <br> The training data structures are used during the recognition process to attempt to identify text in real images using the corner algorithm.
