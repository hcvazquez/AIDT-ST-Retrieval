                This is a very basic functionality to 
                <b>validate the email address</b>. In this post, I will show you how to 
                <b>validate</b> the 
                <b>email address</b> using 
                <b>jQuery</b>. To validate the email address, I have created a separate function which based on email address, returns true or false. Email address validation is done using regular expression.
                <br> 
                <br> Earlier I had posted about 
                <a href="http://jquerybyexample.blogspot.com/2011/10/validate-date-format-using-jquery.html"><b><span style="color: #cc0000;"><u>Validate Date format using jQuery</u></span></b></a>, 
                <a href="http://jquerybyexample.blogspot.com/2011/12/validate-date-using-jquery.html"><b><span style="color: #cc0000;"><u>Validate Date using jQuery</u></span></b></a> and 
                <a href="http://jquerybyexample.blogspot.com/2012/05/validate-phone-numbers-using-jquery.html"><b><span style="color: #cc0000;"><u>Validate Phone numbers using jQuery</u></span></b></a>. And in this post, find jQuery way to Validate email address.
                <br> 
                <br> The validateEmail() functions (created below) will accept a parameter which is nothing but the email address and using regex it validates the email address. If it is correct then it returns true, otherwise false.
                <br> 
                <pre class="brush:javascript">function validateEmail(sEmail) {
    var filter = /^([\w-\.]+)@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.)|(([\w-]+\.)+))([a-zA-Z]{2,4}|[0-9]{1,3})(\]?)$/;
    if (filter.test(sEmail)) {
        return true;
    }
    else {
        return false;
    }
}?
</pre>One just need to make a call to this function to validate the email address. For demo, I have used on click event of button. But It can be used on blur event of textbox or any another event.
                <br> 
                <pre class="brush:javascript">$(document).ready(function() {
   $('#btnValidate').click(function() {
        var sEmail = $('#txtEmail').val();
        if ($.trim(sEmail).length == 0) {
            alert('Please enter valid email address');
            e.preventDefault();
        }
        if (validateEmail(sEmail)) {
            alert('Email is valid');
        }
        else {
            alert('Invalid Email Address');
            e.preventDefault();
        }
    });
});
</pre>
