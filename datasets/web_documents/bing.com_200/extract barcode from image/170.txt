         <h1 class="page-title"><span class="field field--name-title field--type-string field--label-hidden">Java BufferedImage - how to get the RGB value of each image pixel</span> </h1> 
        </div> 
        <div id="block-aadotcom-content" class="block block-system block-system-main-block"> 
         <article data-history-node-id="347" role="article" about="/blog/post/java/getting-rgb-values-for-each-pixel-in-image-using-java-bufferedi" class="node node--type-blog node--promoted node--view-mode-full"> 
          <div class="submitted_by">
           By Alvin Alexander. Last updated: June 3 2016
          </div> 
          <div class="node__content"> 
           <div class="clearfix text-formatted field field--name-body field--type-text-with-summary field--label-hidden field__item">
            <p>I'm just getting started with image parsing in Java, but right away&nbsp;I need to do something that’s relatively hard:&nbsp;find a small image inside of a larger image. I’ve run into some initial problems so I’m writing small tests as I go along.</p> 
            <p>In this article I’ll share the results of a first test I’ve written to walk through all the pixels in an image (a Java BufferedImage) and print out their RGB (technically ARGB) values.</p> 
            <h2>Java Image/BufferedImage example</h2> 
            <p>Here’s my initial Java <a href="http://docs.oracle.com/javase/7/docs/api/java/awt/image/BufferedImage.html">BufferedImage</a> example code. I&nbsp;think everything is straightforward, with the possible the exception of the bitwise operator stuff where I convert a Java <code>int</code> into the RGB/ARGB values the <code>int</code> represents. If you haven’t used code like that before it can be a bit surprising.</p> 
            <p>As final comment, notice how easy the Java <a href="http://docs.oracle.com/javase/7/docs/api/javax/imageio/ImageIO.html">ImageIO</a> class makes it to open a JPG file. That’s very nice:</p> 
            <pre class="brush: java">package com.devdaily.imagetests;

import java.awt.Component;
import java.awt.image.BufferedImage;
import java.io.IOException;
import javax.imageio.ImageIO;

public class JavaWalkBufferedImageTest1 extends Component {

  public static void main(String[] foo) {
    new JavaWalkBufferedImageTest1();
  }

  public void printPixelARGB(int pixel) {
    int alpha = (pixel &gt;&gt; 24) &amp; 0xff;
    int red = (pixel &gt;&gt; 16) &amp; 0xff;
    int green = (pixel &gt;&gt; 8) &amp; 0xff;
    int blue = (pixel) &amp; 0xff;
    System.out.println("argb: " + alpha + ", " + red + ", " + green + ", " + blue);
  }

  private void marchThroughImage(BufferedImage image) {
    int w = image.getWidth();
    int h = image.getHeight();
    System.out.println("width, height: " + w + ", " + h);

    for (int i = 0; i &lt; h; i++) {
      for (int j = 0; j &lt; w; j++) {
        System.out.println("x,y: " + j + ", " + i);
        int pixel = image.getRGB(j, i);
        printPixelARGB(pixel);
        System.out.println("");
      }
    }
  }

  public JavaWalkBufferedImageTest1() {
    try {
      // get the BufferedImage, using the ImageIO&nbsp;class
      BufferedImage image = 
        ImageIO.read(this.getClass().getResource("WhiteSpot.jpg"));
      marchThroughImage(image);
    } catch (IOException e) {
      System.err.println(e.getMessage());
    }
  }

}
</pre>

Here's the initial output from my program and sample JPG&nbsp;image file:</p>
