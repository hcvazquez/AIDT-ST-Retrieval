        <p>"I'M GONNA POP SOME TAGS, ONLY GOT TWENTY DOLLARS IN MY POCKET. I'M A HUNTING, LOOKING FOR A COME UP, THIS IS F......" Parties are a blast when they are YOUR party. They aren't so much fun when the sound is booming from across the street. While I'm SURE you love your loud neighbors and want to tell them in person to turn it down, it's cold outside and even your warmest flannel zebra jammies can't keep the chill out. Why don't we make a button the texts our neighbors when they are too loud?<strong> Perhaps a text from your handy Intel Edison?</strong><br><br>This instructable is all about how to send SMS messages (ie texts) using node.js.</p>

        <div class="photoset-item photoset-image">
         <a class="photoset-link" data-fancybox-group="photoset-gallery-S5BRBVTI3H57326" href="/file/F9TYZJ8I3H57B7Z/" data-fancybox-href="https://cdn.instructables.com/F9T/YZJ8/I3H57B7Z/F9TYZJ8I3H57B7Z.MEDIUM.jpg" title="Set Up Your Edison"><img alt="Set Up Your Edison" data-image-id="F9TYZJ8I3H57B7Z" data-notes="[]" src="https://cdn.instructables.com/F9T/YZJ8/I3H57B7Z/F9TYZJ8I3H57B7Z.MEDIUM.jpg" data-orig-height="140" data-orig-width="360" data-pin-no-hover="true"></a>

       <div class="step-body">
        <p>Before you go any further, you've flashed your Edison with the newest image and configured the wifi, right? No? Well you had better get on it: <a href="https://communities.intel.com/docs/DOC-23147" rel="nofollow">https://communities.intel.com/docs/DOC-23147</a></p>
        <p>We are going to need some things to make this work.</p>
        <p>The first is the node packages from Twilio. Twilio is a company in San Francisco that has an API that allows folks to make calls and texts via THE CLOUD! (all hail the cloud). <strong>You will need to make a free account at <a href="www.Twilio.com" rel="nofollow">www.Twilio.com</a> in order to complete this instructable</strong>. Open up your terminal and connect to your Edison. Type the following and press enter:</p>
        <pre>&gt;&gt; npm install twilio</pre>
        <p>Cool. Now you can text. Now we need to get a package that allows us to run our app even when our terminal has been closed. Note that the "-g" flag is extremely important for our needs.</p>
        <pre>&gt;&gt; npm install forever -g</pre>
        <p>Awesome. Now we need a text editor. There is Vim/Vi on the Edison, but it doesn't always like to work. Here's how to install Nano (which is also much easier to use for beginners):</p>
        <pre>&gt;&gt; wget &lt;a href="http://www.nano-editor.org/dist/v2.2/nano-2.2.6.tar.gz" rel="nofollow"&gt;http://www.nano-editor.org/dist/v2.2/nano-2.2.6.tar.gz&lt;/a&gt; &amp;&amp; tar xvf nano-2.2.6.tar.gz &amp;&amp; cd nano-2.2.6 &amp;&amp; ./configure &amp;&amp; make &amp;&amp; make install
&gt;&gt; make clean 
&gt;&gt; rm -r nano*</pre>
        <p>We need a folder to put our project into as well. Note that if you followed my email tutorial, you don't have to enter in the second line here.</p>
        <pre>&gt;&gt; cd<br>&gt;&gt; makedir mySandbox<br>&gt;&gt; makedir partyAlarm</pre>
        <p>I like to reboot to clear up ram that has been used (and improperly disposed of) during installs, so:</p>
        <pre>&gt;&gt; reboot
</pre>
        <p>Cool beans. Now we are ready to write some code and make some circuits!</p>

       <div id="photoset-SDQUJJSI3H57B7B" data-item-id="SDQUJJSI3H57B7B" data-item-url="https://www.instructables.com/id/Send-Texts-with-Intel-Edison/step3/Simple-Circuits-and-Start-Up/" class="photoset">
        <div class="photoset-item photoset-image">
         <a class="photoset-link" data-fancybox-group="photoset-gallery-SDQUJJSI3H57B7B" href="/file/F87EJSQI3H57B9B/" data-fancybox-href="https://cdn.instructables.com/F87/EJSQ/I3H57B9B/F87EJSQI3H57B9B.MEDIUM.jpg" title="Simple Circuits and Start Up."><img alt="Simple Circuits and Start Up." data-image-id="F87EJSQI3H57B9B" data-notes="[]" src="https://cdn.instructables.com/F87/EJSQ/I3H57B9B/F87EJSQI3H57B9B.MEDIUM.jpg" data-orig-height="315" data-orig-width="329" data-pin-no-hover="true"></a>
        </div>
        <a class="photoset-showmore">Show All Items<br><i class="icon"></i></a>
       </div>
       <div class="step-body">
        <p>Since this is an instructable about texting with your Edison, we'll keep the circuit simple - a button with a 10K pulldown resistor. If you aren't familiar with how to wire that, search "Arduino button circuit" on Google.<br><br>Now that you've written your code and wired your button, we need to start our code running! Forever.js is a neat little package that will run your code until the end of time, restarting it over and over again until Godzilla knocks over the power lines. Unlike my email tutorial, you may notice that this code doesn't automatically loop - in order for the SMS message to be sent, the node.js app needs to exit. Once it exits, forever.js will restart it and keep going. In your Edison's terminal, type the following and press enter:</p>
        <pre>&gt;&gt; forever start --minUptime 1 partyAlarm.js</pre>
        <p>Ah man, you added some fancy stuff outside of this instructable, screwed up and your friend is getting text messages every half second. You need to stop the madness immediately! How do you do this? In the terminal, type the following and press enter: </p>
        <pre>&gt;&gt; forever list </pre>
        <p> This will tell you what jobs are running and what the job numbers are. The job number will typically be "0" if you aren't running anything else with forever.js. So, we will stop job 0. Type the following and press enter:</p>
        <pre>&gt;&gt; forever stop 0 </pre>
        <p> GREAT! You are done! You now have a weapon of mass annoyance to bug your neighbors! Sweet!</p>

             <p>As I mentioned throughout this instructable, I am using Node.js here. You can write code in any language you want. Ones like Python, Node.Js and C++ are the best supported, but anything that can read and write files will work.<br><br>Also, Putty doesn't really "do" anything when it comes to programming. Think of it like a telephone that you are using to call a friend for help - the only thing Putty does is relay a message between you and your Edison in a format called SSH. What you see on the screen is a message from the Edison and what you type into Putty is messaged over to the Edison. Putty (or any other SSH client) is not a programming software, just a messenger.</p>
