      <div class="white-row"> 
       <h4>Lecture: JavaScript Math Round Function</h4> 
       <button class="fa fa-print" style="font-size:48px;color:#1980B6" onclick="pdfgenerate()"></button> 
      </div> 
      <p> </p>
      <p> We can <strong>round decimal numbers</strong> to the nearest integer using the <code>math.round()</code> function.</p> 
      <div class="well"> 
       <code> &lt;!DOCTYPE html&gt;<br> &lt;html&gt;<br> &lt;body&gt;<br> <br>&lt;p id="demo"&gt;&lt;/p&gt;<br> <br> &lt;script&gt;<br> <br> document.getElementById("demo").innerHTML = Math.round(15.4);<br> <br> &lt;/script&gt;<br> <br> &lt;/body&gt;<br> &lt;/html&gt; </code> 
      </div> 
      <p>In this example, we use the <code>math.round</code> function to round the value <em>15.4</em> to the nearest integer. Since the value after the decimal is below .5, the number will be rounded to 15. </p> 
      <p>We can test this in our web browser.</p> 
      <p>Now if we update the value to 15.6, the number will be rounded to 16. </p> 
      <p>To specify whether the value should be rounded up or down, we can use the <code>math.ceil</code> or <code>math.floor</code> function.</p> 
      <p><code>math.ceil</code> will round the value up and <code>math.floor</code> will round the value down. </p> 
      <p><code>document.getElementById("demo").innerHTML = Math.ceil(15.6);</code></p> 
      <p></p>
