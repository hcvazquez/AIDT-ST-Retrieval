      <p><b>The Yahoo! User Interface Library</b> (<b>YUI</b>) is a discontinued open-source <a href="/wiki/JavaScript_library" title="JavaScript library">JavaScript library</a> for building richly interactive <a href="/wiki/Web_application" title="Web application">web applications</a> using techniques such as <a href="/wiki/Ajax_(programming)" title="Ajax (programming)">Ajax</a>, <a href="/wiki/Dynamic_HTML" title="Dynamic HTML">DHTML</a>, and <a href="/wiki/Document_Object_Model" title="Document Object Model">DOM</a> scripting. YUI includes several core <a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">CSS</a> resources. It is available under a <a href="/wiki/BSD_License" class="mw-redirect" title="BSD License">BSD License</a>.<sup id="cite_ref-developer_yahoo_yui_3-0" class="reference"><a href="#cite_note-developer_yahoo_yui-3">[3]</a></sup> Development on YUI began in 2005 and Yahoo! properties such as My Yahoo! and the Yahoo! front page began using YUI in the summer of that year. YUI was released for public use in February 2006.<sup id="cite_ref-initialRelease_1-1" class="reference"><a href="#cite_note-initialRelease-1">[1]</a></sup> It was actively developed by a core team of Yahoo! engineers.</p> 
      <p>In September 2009, Yahoo! released YUI 3, a new version of YUI rebuilt from the ground up to modernize the library and incorporate lessons learned from YUI 2. Among the enhancements are a <a href="/wiki/CSS_selector" class="mw-redirect" title="CSS selector">CSS selector</a> driven engine, like <a href="/wiki/JQuery" title="JQuery">jQuery</a>, for retrieving <a href="/wiki/Document_Object_Model" title="Document Object Model">DOM</a> elements, a greater emphasis on granularity of modules, a smaller seed file that loads other modules when necessary, and a variety of syntactic changes intended to make writing code faster and easier.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup></p> 
      <p>The YUI Library project at Yahoo! was founded by <a href="/w/index.php?title=Thomas_Sha&amp;action=edit&amp;redlink=1" class="new" title="Thomas Sha (page does not exist)">Thomas Sha</a> and sponsored internally by Yahoo! co-founder <a href="/wiki/Jerry_Yang_(entrepreneur)" class="mw-redirect" title="Jerry Yang (entrepreneur)">Jerry Yang</a>; its principal architects have been Sha, Adam Moore, and Matt Sweeney. The library's developers maintain the YUIBlog; the YUI community discusses the library and implementations in its community forum.</p> 
      <p>On August 29, 2014, it was announced that active development of YUI by Yahoo! would end, citing the evolution of the JavaScript standards, steadily decreasing interest in large JavaScript libraries by developers, and the proliferation of server-side solutions. Future development will be limited to maintenance releases addressing issues which are "absolutely critical to Yahoo properties."<sup id="cite_ref-yui-ended_5-0" class="reference"><a href="#cite_note-yui-ended-5">[5]</a></sup></p>

      <h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>The YUI Library is fully documented on its website; detailed API documentation accompanies the library download. It has six types of components: YUI core, utilities, UI controls, CSS components, developer tools, and build tools.</p> 
      <h3><span class="mw-headline" id="Core">Core</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=2" title="Edit section: Core">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>The YUI Core is a light (31KB minified) set of tools for event management and DOM manipulation.</p> 
      <dl> 
       <dt>
        YUI Global Object
       </dt> 
       <dd>
        The YUI Global Object contains language utilities, a script loader, and other baseline infrastructure for YUI.
       </dd> 
       <dt>
        Dom Collection
       </dt> 
       <dd>
        Helps with common 
        <a href="/wiki/Document_Object_Model" title="Document Object Model">DOM</a> scripting tasks, including element positioning and 
        <a href="/wiki/Cascading_Style_Sheets" title="Cascading Style Sheets">CSS</a> style management.
       </dd> 
       <dt>
        Event Utility
       </dt> 
       <dd>
        Provides developers with easy and safe access to browser 
        <a href="/wiki/Event-driven_programming" title="Event-driven programming">events</a> (such as mouse clicks and key presses). It also provides the Custom Event object for publishing and subscribing to custom events.
       </dd> 
      </dl> 
      <h3><span class="mw-headline" id="Utilities">Utilities</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=3" title="Edit section: Utilities">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <dl> 
       <dt>
        Animation
       </dt> 
       <dd>
        Helps create "effects" by animating the position, size, opacity or other characteristics of page elements.
       </dd> 
       <dt>
        Browser History Manager
       </dt> 
       <dd>
        Helps web applications use the browser's back button and bookmarking functionality.
       </dd> 
       <dt>
        Connection Manager
       </dt> 
       <dd>
        Helps manage 
        <a href="/wiki/XMLHttpRequest" title="XMLHttpRequest">XMLHttpRequest</a> transactions in a 
        <a href="/wiki/Cross-browser" title="Cross-browser">cross-browser</a> fashion. It has integrated support for 
        <a href="/wiki/Form_(web)" class="mw-redirect" title="Form (web)">form</a> posts, 
        <a href="/wiki/Error_handling" class="mw-redirect" title="Error handling">error handling</a>, 
        <a href="/wiki/Callback_(computer_science)" class="mw-redirect" title="Callback (computer science)">callbacks</a> and file uploading.
       </dd> 
       <dt>
        Cookie
       </dt> 
       <dd>
        Allows you to manage 
        <a href="/wiki/HTTP_Cookie" class="mw-redirect" title="HTTP Cookie">browser cookies</a> and subcookies through a simple API.
       </dd> 
       <dt>
        DataSource
       </dt> 
       <dd>
        Provides a common configurable interface for other components to interact with different types of data, from simple JavaScript arrays to online servers over XHR.
       </dd> 
       <dt>
        Drag and drop
       </dt> 
       <dd>
        The YUI Drag and Drop Utility makes it easy to make elements "
        <a href="/wiki/Drag_and_drop" title="Drag and drop">draggable</a>" and to create drop targets that respond to drag events.
       </dd> 
       <dt>
        Element
       </dt> 
       <dd>
        Provides a wrapper for 
        <a href="/wiki/HTML_element" title="HTML element">HTML elements</a> in the DOM and makes simpler common tasks such as adding listeners, manipulating the DOM, and setting and getting attributes.
       </dd> 
       <dt>
        Get
       </dt> 
       <dd>
        The Get Utility supports the asynchronous loading of data and scripts through script nodes and the dynamic loading of external CSS files.
       </dd> 
       <dt>
        ImageLoader
       </dt> 
       <dd>
        YUI's ImageLoader allows you to defer the loading of images that are not visible in the viewport at the time the page loads. This can result in big performance boosts.
       </dd> 
       <dt>
        JSON
       </dt> 
       <dd>
        The JSON Utility provides methods for validation of incoming 
        <a href="/wiki/JSON" title="JSON">JSON</a> data to verify that it is safe and methods to convert JavaScript data to a JSON-formatted string. These methods are based on 
        <a href="/wiki/Douglas_Crockford" title="Douglas Crockford">Douglas Crockford's</a> work at JSON.org.
       </dd> 
       <dt>
        Resize
       </dt> 
       <dd>
        Allows you to make any block-level HTML element resizable.
       </dd> 
       <dt>
        Selector
       </dt> 
       <dd>
        The YUI Selector Utility allows you to grab references to HTML elements via CSS3 selector syntax.
       </dd> 
       <dt>
        YUI Loader
       </dt> 
       <dd>
        YUI Loader is a client-side loader engine that can dynamically load any YUI component (and dependencies) on the fly.
       </dd> 
      </dl> 
      <h3><span class="mw-headline" id="Controls">Controls</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=4" title="Edit section: Controls">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <dl> 
       <dt>
        AutoComplete
       </dt> 
       <dd>
        Provides 
        <a href="/wiki/Autocomplete" title="Autocomplete">autocomplete</a> feature (suggestion lists and type-ahead functionality) for user interactions involving text-entry. It supports a variety of data-source formats. It also supports server-side data-sources via XMLHttpRequest.
       </dd> 
       <dt>
        Button
       </dt> 
       <dd>
        Enables the creation of rich, graphical buttons that function like traditional HTML form buttons.
       </dd> 
       <dt>
        Calendar
       </dt> 
       <dd>
        A graphical, dynamic control used for date selection.
       </dd> 
       <dt>
        Charts
       </dt> 
       <dd>
        The Charts Control retrieves data via the DataSource Utility and displays the data in a variety of common chart formats (line, bar, pie, etc.).
       </dd> 
       <dt>
        Color Picker
       </dt> 
       <dd>
        The Color Picker Control provides a rich visual interface for color selection.
       </dd> 
       <dt>
        Container
       </dt> 
       <dd>
        Supports a variety of DHTML windowing patterns including 
        <a href="/wiki/Tooltip" title="Tooltip">Tooltip</a>, Panel, Dialog, SimpleDialog, Module and Overlay.
       </dd> 
       <dt>
        DataTable
       </dt> 
       <dd>
        Simple yet powerful API to display screen-reader accessible tabular data on a web page. Notable features include sortable columns, pagination, scrolling, row selection, resizable columns, and inline editing.
       </dd> 
       <dt>
        ImageCropper
       </dt> 
       <dd>
        ImageCropper provides the UI and interactive elements for a client-side image cropper.
       </dd> 
       <dt>
        Layout Manager
       </dt> 
       <dd>
        Allows you to create cross-browser, pixel perfect layouts with little effort by providing a fixed layout containing, top, bottom, left, right and center layout units.
       </dd> 
       <dt>
        Menu
       </dt> 
       <dd>
        Provides an easy API for creating fly-out 
        <a href="/wiki/Menu_(computing)" title="Menu (computing)">menus</a>, menu bars, and context menus.
       </dd> 
       <dt>
        Rich Text Editor
       </dt> 
       <dd>
        The 
        <a href="/wiki/YUI_Rich_Text_Editor" title="YUI Rich Text Editor">YUI Rich Text Editor</a> is a sophisticated client-side text-processor that is modular and highly configurable, suitable for any open-ended text-entry situation.
       </dd> 
       <dt>
        Slider
       </dt> 
       <dd>
        Provides a generic slider element that enables the user to choose within a finite range of values on one or two axes.
       </dd> 
       <dt>
        TabView
       </dt> 
       <dd>
        Provides navigable tabbed views of content; supports dynamic loading of tab content via XMLHttpRequest.
       </dd> 
       <dt>
        TreeView
       </dt> 
       <dd>
        Produces a content tree whose nodes can be expanded and contracted.
       </dd> 
       <dt>
        Uploader
       </dt> 
       <dd>
        Allows for multi-file file upload with feedback on upload progress.
       </dd> 
      </dl> 
      <h3><span class="mw-headline" id="CSS_resources">CSS resources</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=5" title="Edit section: CSS resources">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <dl> 
       <dt>
        CSS Base
       </dt> 
       <dd>
        Use Base after Reset to provide a consistent, cross-browser replacement for the standard browser CSS rules to which web developers are accustomed.
       </dd> 
       <dt>
        CSS Grids
       </dt> 
       <dd>
        Seven basic page wireframes with subsection components to support over 1000 different page layouts.
       </dd> 
       <dt>
        CSS Fonts
       </dt> 
       <dd>
        Standardized cross-browser font families and size rendering.
       </dd> 
       <dt>
        CSS Reset
       </dt> 
       <dd>
        CSS declarations remove margins and standardize cross-browser rendering on common elements.
       </dd> 
      </dl> 
      <h3><span class="mw-headline" id="Developer_tools">Developer tools</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=6" title="Edit section: Developer tools">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <dl> 
       <dt>
        Logger
       </dt> 
       <dd>
        Provides a quick and easy way to write 
        <a href="/wiki/Data_logging" class="mw-redirect" title="Data logging">log</a> messages to an on-screen console, the 
        <a href="/wiki/Firebug_(web_development)" class="mw-redirect" title="Firebug (web development)">Firebug</a> extension for 
        <a href="/wiki/Mozilla_Firefox" class="mw-redirect" title="Mozilla Firefox">Mozilla Firefox</a>, or the 
        <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a> JavaScript console.
       </dd> 
       <dt>
        Profiler
       </dt> 
       <dd>
        A cross-browser, non-visual code profiler for JavaScript.
       </dd> 
       <dt>
        ProfilerViewer
       </dt> 
       <dd>
        Used in combination with Profiler to provide rich visualizations of your profiling data&nbsp;— both graphically (using the Charts Control) and in tabular format (using DataTable).
       </dd> 
       <dt>
        YUI Test
       </dt> 
       <dd>
        YUI Test is a testing framework for browser-based JavaScript solutions. Using YUI Test, you can easily add unit testing to your JavaScript solutions. While not a direct port from any specific xUnit framework, YUI Test does derive some characteristics from nUnit and JUnit.
       </dd> 
      </dl> 
      <h3><span class="mw-headline" id="Build_tools">Build tools</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=YUI_Library&amp;action=edit&amp;section=7" title="Edit section: Build tools">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <dl> 
       <dt>
        YUI Compressor
       </dt> 
       <dd>
        YUI Compressor is a tool that minifies 
        <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a> and CSS safely.
       </dd> 
       <dt>
        YUIDoc
       </dt> 
       <dd>
        YUIDoc is a tool written in 
        <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a> that generates searchable API documentation of JavaScript code. It is typically used as part of a build process. YUIDoc is comment-driven and is compatible with a variety of coding styles and programming languages.
       </dd> 
      </dl>
