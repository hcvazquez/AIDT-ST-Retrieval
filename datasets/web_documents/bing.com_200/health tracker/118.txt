      <div role="note" class="hatnote navigation-not-searchable">
       This article is about software free to be modified and distributed. For examples of software free in the monetary sense, see 
       <a href="/wiki/List_of_freeware" title="List of freeware">List of freeware</a>.
      </div> 
      <p>This is a list of <b><a href="/wiki/Free_and_open-source" class="mw-redirect" title="Free and open-source">free and open-source</a> <a href="/wiki/Software_package_(installation)" class="mw-redirect" title="Software package (installation)">software packages</a></b>, <a href="/wiki/Computer_software" class="mw-redirect" title="Computer software">computer software</a> licensed under <a href="/wiki/Free_software_license" title="Free software license">free software licenses</a> and <a href="/wiki/Open-source_license" title="Open-source license">open-source licenses</a>. Software that fits the <a href="/wiki/Free_Software_Definition" class="mw-redirect" title="Free Software Definition">Free Software Definition</a> may be more appropriately called <a href="/wiki/Free_software" title="Free software">free software</a>; the <a href="/wiki/GNU_project" class="mw-redirect" title="GNU project">GNU project</a> in particular objects to their works being referred to as <a href="/wiki/Open-source" class="mw-redirect" title="Open-source">open-source</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> For more information about the philosophical background for open-source software, see <a href="/wiki/Free_software_movement" title="Free software movement">free software movement</a> and <a href="/wiki/Open_Source_Initiative" title="Open Source Initiative">Open Source Initiative</a>. However, nearly all software meeting the <a href="/wiki/Free_Software_Definition" class="mw-redirect" title="Free Software Definition">Free Software Definition</a> also meets the <a href="/wiki/Open_Source_Definition" class="mw-redirect" title="Open Source Definition">Open Source Definition</a> and vice versa. A small fraction of the software that meets either definition is listed here.</p> 
      <p>Some of the open-source applications are also the basis of commercial products, shown in the <a href="/wiki/List_of_commercial_open-source_applications_and_services" title="List of commercial open-source applications and services">List of commercial open-source applications and services</a>.</p> 
      <p></p>
