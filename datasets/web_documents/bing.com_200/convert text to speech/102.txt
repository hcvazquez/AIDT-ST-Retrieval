                 <h1 class="post-title entry-title" itemprop="name"> Csharp Example: Text to Speech 
                  <div class="post-details">
                   <a href="">04:13</a> by 
                   <a href="https://plus.google.com/110224953378058043623">Anjan Kant</a>
                  </div> </h1> 
                </header> 
                <div class="post-content"> 
                 <div dir="ltr" style="text-align: left;" trbidi="on"> 
                  <div dir="ltr" style="text-align: left;" trbidi="on">

                     <div dir="ltr" style="text-align: left;" trbidi="on"> 
                      <h2 style="text-align: left;"> Introduction</h2> Today, I am providing a working example of text to speech using .Net framework 4.5 with System.Text namespace. Text to speech is used widely in large scale for different purposes.
                      <br> I am explaining all steps how to convert 
                      <b>Text to Speech using C#</b>, .Net framework.
                      <br> 
                      <br> 
                      <b>Step 1</b>. First of all, we have to add reference of System.Speech in our Text Speech application.
                      <br> 
                      <br> 
                      <div class="separator" style="clear: both; text-align: center;"> 
                       <a href="http://3.bp.blogspot.com/-m9fz6NMF_pc/VgZ3l2wjYzI/AAAAAAAABlQ/jaSbWIiZ9gk/s1600/System.Speech.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="System.Speech" border="0" height="266" src="http://3.bp.blogspot.com/-m9fz6NMF_pc/VgZ3l2wjYzI/AAAAAAAABlQ/jaSbWIiZ9gk/s400/System.Speech.png" title="System.Speech" width="400"></a>
                      </div> 
                      <br> 
                      <b>Step 2</b>. Now import namespace using 
                      <b>System.Speech.Synthesis</b> in our form.
                     </div> 
                     <pre class="prettyprint">using System.Speech.Synthesis;&nbsp;</pre> 
                     <b>Step&nbsp;3</b>. Now create object of 
                     <b>SpeechSynthesizer</b>
                    </div> 
                    <pre class="prettyprint">SpeechSynthesizer speech = new SpeechSynthesizer();&nbsp;</pre> 
                    <b>Step&nbsp;4</b>. Next one, write this line code speech.SpeakAsync(textBox1.Text) on button click.
                   </div> 
                   <pre class="prettyprint">speech.SpeakAsync(textBox1.Text);&nbsp;</pre> 
                   <b>Step&nbsp;5</b>. Run application using 
                   <b>F5</b> button to view Text speech interface. 
                   <br> 
                   <b>Step&nbsp;6</b>. Enter sample text in input box to read through our application.
                   <br> 
                   <b>Step&nbsp;7</b>. Finally you can see how gracefully reading our text using our C# application.
                   <br> 
                   <div class="separator" style="clear: both; text-align: center;"> 
                    <a href="http://1.bp.blogspot.com/-G_0i5WtWodA/VgZ3z1dTDVI/AAAAAAAABlY/rqqORaE0Lw4/s1600/Start_Speech.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="Start Speech" border="0" height="246" src="http://1.bp.blogspot.com/-G_0i5WtWodA/VgZ3z1dTDVI/AAAAAAAABlY/rqqORaE0Lw4/s400/Start_Speech.png" title="Start Speech" width="400"></a>
                   </div> 
                   <br> 
                   <h2 style="text-align: left;"> Summary</h2> Explained above, all steps 
                   <b>Text to Speech using C#</b> .Net framework 4.5 with working sample. I have attached working sample at the bottom of this article.
