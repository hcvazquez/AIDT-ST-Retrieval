        <h1 class="entry-title" itemprop="headline">How to Convert String to Date and Date to String in Java</h1>
        <p class="entry-meta"><time class="entry-modified-time" itemprop="dateModified" datetime="2017-09-04T02:59:18+00:00">September 4, 2017</time> by <span class="entry-author" itemprop="author" itemscope itemtype="https://schema.org/Person"><a href="https://www.journaldev.com/author/pankaj" class="entry-author-link" itemprop="url" rel="author"><span class="entry-author-name" itemprop="name">Pankaj</span></a></span> <span class="entry-comments-link"><a href="https://www.journaldev.com/692/how-to-convert-string-to-date-and-date-to-string-in-java#comments">1 Comment</a></span></p>
       </header>

        <div style="float:none;margin:10px 0 10px 0;text-align:center;">
         <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-5143693538742787" data-ad-slot="7174295726" data-ad-format="auto"></ins>

        <p>String is one of the most widely used Object in Java and if you are working in web services or web applications with form, you get date in the form of String object and in the backend we convert it from String to Date object.</p>
        <p>Similarly while showing date information on any web page, we convert Date to String in required format.</p>
        <p><a href="http://docs.oracle.com/javase/7/docs/api/java/text/DateFormat.html">j<strong><em>ava.text.DateFormat</em></strong></a> is the <a href="https://www.journaldev.com/1582/abstract-class-in-java">abstract class</a> for Date/Time formatting and <strong><em><a href="http://docs.oracle.com/javase/7/docs/api/java/text/SimpleDateFormat.html">java.text.SimpleDateFormat</a></em></strong> is the concrete class we use to <strong>convert String to Date</strong> and to <strong>convert Date to String</strong> in different formats.</p>
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">
         <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-5143693538742787" data-ad-slot="5697562523" data-ad-format="auto"></ins> 
         <script>(adsbygoogle=window.adsbygoogle||[]).push({});</script> 
        </div>
        <p>Let’s see how to <em>convert String to Date</em> and <em>convert Date to String</em> with a simple example:</p>
        <pre class="brush: java; title: DateUtils.java; notranslate" title="DateUtils.java">
package com.journaldev.util;

import java.text.DateFormat;
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;
import java.util.Locale;

/**
 * Java program example to convert String to Date and to convert Date to String
 * @author pankaj
 *
 */
public class DateUtils {
    public static void main(String[] args) {
    	//initialize SimpleDateFormat object
        DateFormat sdf = new SimpleDateFormat("dd/MM/yyyy");
 
        try {
        	//Convert String to Date
            Date today = sdf.parse("14/11/2012");
            System.out.println("Date is:"+today.toString());
            
            //using locale
            sdf = new SimpleDateFormat("dd-MMM-yyyy", Locale.CANADA_FRENCH);
            today = new Date();
            System.out.println("Date is:"+today.toString());
            
            //Convert Date to String
            System.out.println(sdf.format(today));
        } catch (ParseException e) {
            e.printStackTrace();
        }
    }
}
</pre>
        <p>From example it’s clear that we use <em>parse(String str)</em> method to convert String to Date object. For converting Date to String, we use <em>format(Date date)</em> method. Note that both of these methods implementation is provided in DateFormat class and SimpleDateFormat inherits them through <a href="https://www.journaldev.com/644/inheritance-java-example">java inheritance</a>.</p>
        <p>DateFormat class supports TimeZone and Locale specific conversions also.</p>
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">
         <div data-type="ad" data-publisher="journaldev.com" data-format="300x250" data-zone="jd_post_mid_300x250"></div>
        </div>

14-nov.-2012
</pre>

        <p>Let’s extend our program a bit to support multiple String formats while parsing to Date. This situation can arise when you have a web page or xml field that supports multiple format for passing date as string.</p>
        <pre class="brush: java; title: StringToDateUtil.java; notranslate" title="StringToDateUtil.java">
package com.journaldev.util;

import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.ArrayList;
import java.util.Date;
import java.util.List;

public class StringToDateUtil {
	
	private List&lt;SimpleDateFormat&gt; sdfList;

	//set the List of Format's you want to use
	public StringToDateUtil(List&lt;SimpleDateFormat&gt; sdfList) throws Exception {
		if(sdfList == null) throw new Exception ("sdfList can't be null");
		this.sdfList = sdfList;
	}

	public Date stringToDate(String str) throws Exception{
		if(str == null) return null;
		Date date = null;
		// parse the input String with list of SimpleDateFormats we have
		for(SimpleDateFormat sdf : sdfList){
			try{
				date = sdf.parse(str);
			}catch(ParseException pe){
				//do nothing, we need to try other format
			}
			//check if parsed successfully?
			if(date !=null) break;
		}
		//return date if parsed successfully or else throw exception
		if(date !=null) return date;
		throw new Exception("invalid format for String:"+str);
	}
	
	public static void main(String args[]) throws Exception{
		List&lt;SimpleDateFormat&gt; formatList = new ArrayList&lt;&gt;();
		formatList.add(new SimpleDateFormat("dd MMM yyyy"));
		formatList.add(new SimpleDateFormat("M/dd/yyyy"));
		formatList.add(new SimpleDateFormat("M/dd/yyyy hh:mm:ss a"));
		StringToDateUtil sdUtil = new StringToDateUtil(formatList);
		// Lets format some String to Date
		String [] arr = {"10 NOV 2012", "10/14/2012","10/14/2012 10:45:30", "ABC", null};
		for(String str : arr){
			try{
				System.out.println(str + " Date Object = " + sdUtil.stringToDate(str));
			}catch(Exception e){
				System.out.println(str + " is not a valid date");
			}
		}
	}

}
</pre>
        <p>In the above class, we are passing all the format’s of Date string we are expecting and then <em>stringToDate(String str)</em> method use those to parse given String.</p>
        <p>Here is the output of the above program.</p>
        <pre class="brush: bash; title: ; notranslate" title="">
10 NOV 2012 Date Object = Sat Nov 10 00:00:00 PST 2012
10/14/2012 Date Object = Sun Oct 14 00:00:00 PDT 2012
10/14/2012 10:45:30 Date Object = Sun Oct 14 00:00:00 PDT 2012
ABC is not a valid date
null Date Object = null
</pre>
        <p>Note that you should create static object for list of formats in your application rather than initializing it all the time, you might also want to create <a href="https://www.journaldev.com/171/thread-safety-in-java-singleton-classes-with-example-code">Singleton</a> class for this purpose.</p>
        <p>Also note that DateFormat classes are not thread safe, so if you want thread safety, you need to create a wrapper class for SimpleDateFormat and implement synchronized format and parse methods that internally invokes DateFormat methods.</p>
        <p><strong>Update</strong>: Java 8 new Date Time API provides easy and standard approach for handling parsing and date formatting, you should check it out at <a href="https://www.journaldev.com/2800/java-8-date-localdate-localdatetime-instant" title="Java 8 Date Time API Example Tutorial – LocalDate, Instant, LocalDateTime, Parse and Format">Java 8 Date API Tutorial</a>.</p>
