              A converter to Convert user's Text or any .txt File into Speech or .wav AudioFile

             <h2>Introduction</h2> 
             <p>This is an application to convert the user's Text or Text file of <em>.txt</em> format to audio file of <em>.wav</em> format. Here the application identifies the words and punctuation marks from the text file and breaks it accordingly with the letters and phonemes concerned. The user will get the audio output of whatever text he/she has written in the richtextbox or whatever text file he/she has introduced in the textbox concerned.</p> 
             <h2>Background</h2> 
             <p>I am trying to develop a Friend (an application) who will understand your words, remember it and will also talk as a friend. This Text to Speech Converter is another module of the Application, through which the application understands the user's words and reapplies them accordingly (Goal).</p> 
             <h2>Using the Code</h2> 
             <pre lang="cs">        <span class="code-comment">/*</span><span class="code-comment">********************************************************/</span>
    <span class="code-comment">//</span><span class="code-comment">This is a converter to convert your .txt file or your Text </span>
    <span class="code-comment">//</span><span class="code-comment">to voice or .wav file.</span>
    <span class="code-comment">/*</span><span class="code-comment">********************************************************/</span>
&lt;code&gt;           
<span class="code-keyword">using</span> System;                      <span class="code-comment">//</span><span class="code-comment">Adding System namespace through keyword using.</span>
<span class="code-keyword">using</span> System.Collections.Generic;  <span class="code-comment">//</span><span class="code-comment">Adding namespace Collections.Generic of System </span>
<span class="code-keyword">using</span> System.ComponentModel;       <span class="code-comment">//</span><span class="code-comment">Adding namespace ComponentModel of System</span>
<span class="code-keyword">using</span> System.Data;                 <span class="code-comment">//</span><span class="code-comment">Adding namespace Data</span>
<span class="code-keyword">using</span> System.Drawing;              <span class="code-comment">//</span><span class="code-comment">Adding namespace Drawing</span>
<span class="code-keyword">using</span> System.Text;                 <span class="code-comment">//</span><span class="code-comment">Adding namespace Text</span>
<span class="code-keyword">using</span> System.Windows.Forms;        <span class="code-comment">//</span><span class="code-comment">Adding namespace Windows Forms</span>
<span class="code-keyword">using</span> SpeechLib;                   <span class="code-comment">//</span><span class="code-comment">Adding namespace/dll file SpeechLib</span></pre> 
             <p>Here in the above code the most importance difference you will notice is the namespace <code>SpeechLib</code>. This is the Dynamic Link Library of SAPI5.1, i.e. Speech Application programming Interphase5.1. </p> 
             <pre lang="cs"><span class="code-keyword">namespace</span> WindowsApplication1            <span class="code-comment">//</span><span class="code-comment">Namespace WindowApplication1 Started</span>
{
    <span class="code-keyword">public</span> <span class="code-keyword">partial</span> <span class="code-keyword">class</span> Form1 : Form    <span class="code-comment">//</span><span class="code-comment">Class Form1 is extended to Class Form (system)</span>
        {
        <span class="code-keyword">public</span> Form1()                   <span class="code-comment">//</span><span class="code-comment">Constructor for Form1</span>
            {
            InitializeComponent();       <span class="code-comment">//</span><span class="code-comment">Method to initialize the Form1 Component.</span>
            }

             <h2>What Does the Code Do?</h2> 
             <p>This code belongs to the main form as "<em>Form1.cs</em>", here in User Choice group section the user will have two selection checkboxes - one for <code>richTextbox </code>and another to browse the text file from the system. Through these checkboxes, the user can choose the required way to convert his text to speech/audio file or user can browse his/her text file to Convert into the audio file. When user Clicks the "speak" button, then text from the richtextbox or the text from the <em>.txt</em> file passes to the <code>speak </code>method of <code>spVoice </code>class contained in the <em>Speechlib.dll</em> file. As through the <code>SaveFile </code>button, the procedure is the same as the speak method converts the text into audio format of <em>.wav</em> and a <code>saveDialogbox </code>opens for the preferred location to save. User can also set the volume and speaking rate as from the "Inc" and "Dec" button. Here the volume and rate is an integral variable which initializes the volume and rate of the Volume and Rate of the <code>SpVoice </code>class of <em>Speechlib.dll</em> file.</p> 
             <h2>How Do I Use It?</h2> 
             <p>Mark the <code>checkbox </code>that user needs. If user is going through the <code>richtextbox</code>, then the user has to write the text there in <code>richtextbox </code>and click the speak button or if user had marked the <code>TextFilePath </code>checkbox, then he/she will have to browse the <code>textfile </code>through browser button and then click the speak button. To save the audio file, the user will get a <code>saveDialogbox </code>through <code>saveFile </code>button and he/she can save the audio file generated at any location with a new name.</p>
