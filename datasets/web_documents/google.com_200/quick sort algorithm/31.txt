        <h2>What do the sounds mean?</h2> 
        <p>This javascript program runs each sorting algoritm on the given array of pitches (as MIDI values). Each time a pitch is read from the array, the pitch is played softly, and each time a pitch is written to the array, the pitch is played loudly. Swaps and comparisons are played as intervals (with both pitches sounding simultaneously.)</p> 
        <h2>Why did you make this?</h2> 
        <p>I built this for an electronic composition called <a href="http://www.caseyrule.com/music/works/sorted-affairs/">Sorted Affairs</a>. The entire composition process took many hours spread over several months.</p> 
        <h2>How does it work?</h2> 
        <p>This program was written in Javascript using the <a href="http://jquery.com/" target="_blank">jQuery</a>, <a href="http://raphaeljs.com/" target="_blank">Raphaël</a>, and <a href="http://mudcu.be/midi-js/" target="_blank">MIDI.js</a> libraries for animation and sound. In order to accomodate the timing necessary for animation and sound, each algoritm had to be altered extensively using 'setTimout' to allow for delays between reads and writes to the array.</p> 
        <p> Here is an example with one of the simplest algorithms, Bubble sort. This is the original Bubble sort algorithm: </p> 
        <pre class="prettyprint">
function bubbleSort( array ) {
  var swapped;
  do {
    swapped = false;
    for ( var i=0; i &lt; array.length-1; i++ ) {
      if ( compare( array, i, i + 1 ) &gt; 0 ) {
        swap( array, i, i+1 );
        swapped = true;
      }
    }
  } while ( swapped );
}
</pre> 
        <p> ...and this is the function rewritten to allow delays for animation and sound: </p> 
        <pre class="prettyprint">
function bubbleSortWithSound ( array, callback ) {
  ( function bubbleSortWhileLoop() {
    swapped = false;
    ( function bubbleSortForLoop( i ) {
      setTimeout( function() {
        var test = compare( array, i, i + 1 ) &gt; 0;
        setTimeout( function() {
          if( test ){
            swap( array, i, i+1 );
            swapped = true;
          }
          if ( ++i &lt; array.length - 1 ) bubbleSortForLoop( i );
          else {
            if ( swapped ) bubbleSortWhileLoop();
            else if ( callback ) callback();
          }
        }, test ? delay : 0 );
      }, delay );
    } )( 0 );
  } )();
}
</pre> 
        <p> This isn't too bad with something like Bubble sort, as there are only two loops and no complex recursion. However, this process can become pretty tedious with a more complicated algorithm. Take Merge sort for example, which I found easiest to implement with a sort of make-shift call stack. This is what the time-delayed Merge sort algoritm ended up looking like: </p> 
        <pre class="prettyprint" style="font-size: .7em;">
function mergeSortWithSound ( array, callback ) {
	var stack = [];
	
	function executeFromStack() {
		if ( stack.length &gt; 0 ) {
			var func = stack.pop();
			func[0].apply( null, func[1] );
		} else {
			if ( callback ) callback();	
		}
	}
	
	function mergeSortRecursive( left, right ){
		if ( left &lt; right ) {
			var middle = Math.floor( ( left + right ) / 2);
						
			stack.push( [ merge, [ left, middle, right ] ] );
			stack.push( [ mergeSortRecursive, [ middle + 1, right ] ] );
			stack.push( [ mergeSortRecursive, [ left, middle ] ] );
		}
		executeFromStack();
	}
	
	function merge( left, middle, right ) {
		var l = left;
		var r = middle + 1;
		( function mergeWhileLoop() {
			setTimeout( function () {
				if ( compare( array, l , r) &lt;= 0 ) {
					l++;
					if ( l &lt;= middle &amp;&amp; r &lt;= right ) mergeWhileLoop();
					else executeFromStack();
				} else {
					setTimeout( function () {
						var tmp = read( array, r );
						( function mergeForLoop( j ) {
							setTimeout( function() {
								var tmp2 = read( array, j-1 );
								setTimeout( function() {
									write( array, j, tmp2 );
									if ( --j &gt; l ) { 
										mergeForLoop( j );
									} else {
										setTimeout( function() {
											write( array, j, tmp );
											middle++;
											r++
											if ( l &lt;= middle &amp;&amp; r &lt;= right )
												mergeWhileLoop();
											else executeFromStack();
										}, delay);
									}
								}, delay );
							}, delay );
						} )( r );
					}, delay );
				}
			}, delay );
		} )();
	}
	
	mergeSortRecursive( 0 , array.length - 1 );
}
</pre> 
        <p> ...but enough technical stuff. Just have fun with it! I would personally recommend Quicksort on the "Dark" pitch set, or, if you've got some time, Stooge sort on the "Shifting" pitch set. </p>
