         <h2>Validate Form on Submit using Google reCaptcha Validation Example in Laravel</h2>
         <p>A Qoutes about using captcha is "Tough on bots and Easy on humans".You are going to use Google recaptcha so you must know the benefits of using Google recaptcha.</p>
         <p>Nomally any captcha protect your websites from spamming.</p>
         <p>Google day by day made changes in their library as in past you have to validate via awful distorted text which you have to put in the text box to verify that you are not a robot but now there is a single click to confirm that yes you are not a robot.</p>
         <p>reCaptcha is a totally free service that make you websites away from being spam so it means reCaptcha is built only for security purpose.</p>
         <p>I am going to use NoCaptcha reCaptcha which is improved version of Captcha announced by Google.</p>
         <p>We all know that when Google announce improved version then definitely they will put some great benefits.</p>
         <p>In Laravel, There are so many packages to use Google NoCaptcha reCaptcha.</p>
         <p>Almost every websites using captcha for security reason either in contact form or in register form.</p>
         <p>Let's start with anhskohbo/no-captcha packages that is very popular now a days. As I told you there are many packages are available like <code>buzz/laravel-google-captcha</code> and <code>google/recaptcha</code> you can use any one of them. </p>
         <strong class="step">Step1: anhskohbo/no-captcha package installation</strong>
         <p>I will recommend Composer that is widely used as a dependecy manager for PHP packages. To add this dependency using the command, run following command from your project directory.</p>
         <pre>composer require anhskohbo/no-captcha</pre>
         <p>Now add the service provider to the <code>providers</code> array in following path <strong class="filepath">config/app.php</strong></p>
         <strong>config/app.php</strong>
         <pre>Anhskohbo\NoCaptcha\NoCaptchaServiceProvider::class</pre>

NOCAPTCHA_SITEKEY=[site-key] - 6LeIxAcTAAAAAJcZVRqyHh71UMIEGNQ_MXjiZKhI</pre>
