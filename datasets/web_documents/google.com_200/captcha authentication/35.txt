      <br> 
      <p>I think we should use captcha code on our registration form because captcha code prevent spams, bots etc. Most of the application we need to use captcha varification because it very important for security reason. There are several library to generate captcha image in Laravel. In this example i use BotDetect package library for generate captcha image and using their validation. BotDetect is very popular library and very customize library. they provide several ui design for captcha code and also how much character do you want?, you can set all the thing using this package.</p>
      <p></p>
      <p>In this post i give you very simple and from scratch of generate captcha code image as you can also see bellow preview. After complete this example you can found ui design like bellow preview, you have to just follow few step and find result:</p>
      <p><strong>Preview:</strong></p>
      <p><img width="100%" src="/upload/Laravel-captcha-code.png" style="border:2px solid"></p>
      <p></p>
      <p><strong class="step">Step 1: Installation</strong></p>
      <p>In first step we will install captcha-com/laravel-captcha package for generate captcha code image. this package through we can generate generate captcha code image for our project. so first fire bellow command in your cmd or terminal:</p>
      <p></p>
      <pre>composer require captcha-com/laravel-captcha:"4.*"</pre>
      <p></p>
      <p></p>
      <p>Now we need to add provider path and alias path in config/app.php file so open that file and add bellow code.</p>
      <p><strong>config/app.php</strong></p>
      <p></p>
      <pre><p></p><p>return [</p><p>	......</p><p>	$provides =&gt; [</p><p>		......</p><p>		......,</p><p>		LaravelCaptcha\Providers\LaravelCaptchaServiceProvider::class</p><p>	],</p><p>	.....</p><p>]</p><p></p></pre>
      <p></p>
      <p></p>
      <p>Now we will run bellow command that way it will generate app/captcha.php file for configration and we can change and customize easily.</p>
      <p></p>
      <pre>php artisan vendor:publish</pre>
      <p></p>
      <p></p>
      <p><strong class="step">Step 2: Add Route</strong></p>
      <p>In second step we will add new two route for creating small example that way we can undestand very well. so first add bellow route in your routes.php file.</p>
      <p><strong>app/Http/routes.php</strong></p>
      <p></p>
      <pre class="prettyprint lang-php"><p></p><p>Route::get('web-register', 'Auth\AuthController@webRegister');</p><p>Route::post('web-register', 'Auth\AuthController@webRegisterPost');</p><p></p></pre>
      <p></p>
      <p></p>
      <p><strong class="step">Step 3: Add Controller Method</strong></p>
      <p>In this step we will add webRegister() and webRegisterPost() method in AuthController Controller file for our example. If you want to copy whole controller file then also you can just copy and paste.</p>
      <p><strong>app/Http/Controllers/Auth/AuthController.php</strong></p>
      <p></p>
      <pre class="prettyprint lang-php"><p></p><p>namespace App\Http\Controllers\Auth;</p><p><br></p><p>use Validator;</p><p>use App\Http\Controllers\Controller;</p><p>use Illuminate\Foundation\Auth\ThrottlesLogins;</p><p>use Illuminate\Foundation\Auth\AuthenticatesAndRegistersUsers;</p><p>use Auth;</p><p>use Illuminate\Http\Request;</p><p><br></p><p>class AuthController extends Controller</p><p>{</p><p><br></p><p>    use AuthenticatesAndRegistersUsers, ThrottlesLogins;</p><p><br></p><p>    protected $redirectTo = '/';</p><p><br></p><p>    /**</p><p>     * Create a new authentication controller instance.</p><p>     *</p><p>     * @return void</p><p>     */</p><p>    public function __construct()</p><p>    {</p><p>        $this-&gt;middleware('guest', ['except' =&gt; 'logout']);</p><p>    }</p><p><br></p><p>    /**</p><p>     * Get a validator for an incoming registration request.</p><p>     *</p><p>     * @param  array  $data</p><p>     * @return \Illuminate\Contracts\Validation\Validator</p><p>     */</p><p>    protected function validator(array $data)</p><p>    {</p><p>        return Validator::make($data, [</p><p>            'name' =&gt; 'required|max:255',</p><p>            'email' =&gt; 'required|email|max:255|unique:users',</p><p>            'password' =&gt; 'required|confirmed|min:6',</p><p>        ]);</p><p>    }</p><p><br></p><p>    /**</p><p>     * Create a new user instance after a valid registration.</p><p>     *</p><p>     * @param  array  $data</p><p>     * @return User</p><p>     */</p><p>    protected function create(array $data)</p><p>    {</p><p>        return User::create([</p><p>            'name' =&gt; $data['name'],</p><p>            'email' =&gt; $data['email'],</p><p>            'password' =&gt; bcrypt($data['password']),</p><p>        ]);</p><p>    }</p><p><br></p><p>    public function webRegister()</p><p>    {</p><p>        return view('webRegister');</p><p>    }</p><p><br></p><p>    public function webRegisterPost(Request $request)</p><p>    {</p><p>        $this-&gt;validate($request, [</p><p>            'name' =&gt; 'required',</p><p>            'email' =&gt; 'required|email',</p><p>            'password' =&gt; 'required|same:password_confirmation',</p><p>            'password_confirmation' =&gt; 'required',</p><p>            'CaptchaCode' =&gt; 'required|valid_captcha'</p><p>        ]);</p><p></p><p>        print('write your other code here.');</p><p>    }</p><p>}</p><p></p></pre>
      <p></p>
      <p></p>
      <p><strong class="step">Step 4: Add Blade file</strong></p>
      <p> <ins class="adsbygoogle" style="display:block; text-align:center;" data-ad-format="fluid" data-ad-layout="in-article" data-ad-client="ca-pub-9165454495231653" data-ad-slot="9016598873"></ins> <script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script></p>
      <p>This is a last step and you have to just create new blade file webRegister.blade.php and put bellow code on that file.</p>
      <p><strong>resources/views/webRegister.blade.php</strong></p>
      <p></p>
