         <p>In this short and concise&nbsp;course you will <strong class="redactor-inline-converted">learn everything&nbsp;to get started with web scraping</strong> using Java.&nbsp;</p> 
         <p>You will <strong class="redactor-inline-converted">learn the concepts behind web scraping</strong> that you can apply to <strong class="redactor-inline-converted">practically any web page</strong> (static AND dynamic /&nbsp;AJAX).</p> 
         <p><br></p> 
         <p><strong class="redactor-inline-converted">Course structure</strong></p> 
         <p>We start with an <strong class="redactor-inline-converted">overview</strong> of what web scraping is and what you can do with it.&nbsp;</p> 
         <p>Then we <strong class="redactor-inline-converted">explain the difference in scraping static pages vs dynamic / AJAX pages</strong>. You learn how to classify a website in one of the two categories and then apply the right concept in order to scrape the data you want.</p> 
         <p>Now you will learn how to <strong class="redactor-inline-converted">export</strong> the scraped data either as <strong class="redactor-inline-converted">CSV or JSON</strong>. These are some popular formats that can be used for further processing.</p> 
         <p>Unfortunately many websites try to block scrapers or sometimes you just do not want to be detected. In the section&nbsp;<em>going undercover</em> <strong class="redactor-inline-converted">you will learn how to stay undetected and avoid getting blocked</strong>.</p> 
         <p>At the end of the course you can download the <strong class="redactor-inline-converted">full source code</strong> of all the lectures and we discuss an <strong class="redactor-inline-converted">outlook to some advanced topics</strong> (private proxies, cloud deployment, multi threading ...). Those advanced topics are covered in a follow up course I am going to teach.</p>
