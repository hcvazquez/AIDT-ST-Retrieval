          <p><strong><code>Math</code></strong> is a built-in object that has properties and methods for mathematical constants and functions. Not a function object.</p> 
          <h2 id="Description">Description</h2> 
          <p>Unlike the other global objects, <code>Math</code> is not a constructor. All properties and methods of <code>Math</code> are static. You refer to the constant pi as <code>Math.PI</code> and you call the sine function as <code>Math.sin(x)</code>, where <code>x</code> is the method's argument. Constants are defined with the full precision of real numbers in JavaScript.</p> 
          <h2 id="Properties">Properties</h2> 
          <dl> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/E" title="The Math.E property represents the base of natural logarithms, e, approximately 2.718."><code>Math.E</code></a>
           </dt> 
           <dd>
            Euler's constant and the base of natural logarithms, approximately 2.718.
           </dd> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/LN2" title="The Math.LN2 property represents the natural logarithm of 2, approximately 0.693:"><code>Math.LN2</code></a>
           </dt> 
           <dd>
            Natural logarithm of 2, approximately 0.693.
           </dd> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/LN10" title="The Math.LN10 property represents the natural logarithm of 10, approximately 2.302:"><code>Math.LN10</code></a>
           </dt> 
           <dd>
            Natural logarithm of 10, approximately 2.303.
           </dd> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/LOG2E" title="The Math.LOG2E property represents the base 2 logarithm of e, approximately 1.442:"><code>Math.LOG2E</code></a>
           </dt> 
           <dd>
            Base 2 logarithm of E, approximately 1.443.
           </dd> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/LOG10E" title="The Math.LOG10E property represents the base 10 logarithm of e, approximately 0.434:"><code>Math.LOG10E</code></a>
           </dt> 
           <dd>
            Base 10 logarithm of E, approximately 0.434.
           </dd> 
           <dt>
            <a href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/PI" title="The Math.PI property represents the ratio of the circumference of a circle to its diameter, approximately 3.14159:"><code>Math.PI</code></a>
           </dt> 
           <dd>
            Ratio of the circumference of a circle to its diameter, approximately 3.14159.
           </dd> 
           <dt>

          <h2 id="Extending_the_Math_object">Extending the <code>Math</code> object</h2> 
          <p>As most of built-in objects in JavaScript the <code>Math</code> object can be extended with custom properties and methods. To extend the <code>Math</code> object, you do not use 'prototype'. Instead you directly extend <code>Math</code>:</p> 
          <pre>Math.propName = propValue;
Math.methodName = methodRef;</pre> 
          <p>For instance, the following example adds a method to the <code>Math</code> object for calculating the <em>greatest common divisor</em> of a list of arguments.</p> 
          <pre class="brush: js">/* Variadic function -- Returns the greatest common divisor of a list of arguments */
Math.gcd = function () {
&nbsp; var d = Math.min.apply(Math, arguments);
&nbsp; for (let n = arguments.length, i = 0; d &gt; 1 &amp;&amp; n &gt; i; arguments[i] % d === 0 ? i++ : (d--, i = 0));
&nbsp; return d;
}</pre> 
          <p>Try it:</p> 
          <pre class="brush: js">console.log(Math.gcd(20, 30, 15, 70, 40)); // `5`</pre>
