             <h3>What is Captcha Image?</h3> 
             <p>Source: <a href="http://myvsdotnetstuff.blogspot.com/2008/11/captcha-image-using-c-in-aspnet.html" title="My DotNet Stuff">DotNet Stuff</a></p> 
             <p><img alt="169371/done.JPG" src="/KB/Blogs/169371/done.JPG" style="width: 304px; height: 138px"></p> 
             <p>If both match, then displays the above message:</p> 
             <p><img alt="169371/notvalid.JPG" src="/KB/Blogs/169371/notvalid.JPG" style="width: 316px; height: 138px"></p> 
             <p>If both don't match, then display the above message.</p> 
             <p>In the web, whenever you register on any site, normally they ask at the end of page for entering Image code/Challenge Image/Image verification code, etc. This type of image is called Captcha Image.</p> 
             <h3>What is the Use of Captcha?</h3> 
             <p><em>(Source: <a href="http://en.wikipedia.org/wiki/Captcha">http://en.wikipedia.org/wiki/Captcha</a>)</em></p> 
             <p>Captcha is a type of <a href="http://en.wikipedia.org/wiki/Challenge-response_authentication" title="Challenge-response authentication">challenge-response</a> test used in <a href="http://en.wikipedia.org/wiki/Computing" title="Computing">computing</a> to ensure that the response is not generated by a computer. The process usually involves one computer (a <a href="http://en.wikipedia.org/wiki/Server_(computing)" title="Server (computing)">server</a>) asking a user to complete a simple test which the computer is able to generate and grade. Because other computers are unable to solve the CAPTCHA, any user entering a correct solution is presumed to be human.</p> 
             <p>A best technique to restrict automatic form submissions when developing a web page is to add some kind of verification. As per my knowledge, image verification is the best way, it’s also called Captcha image. In this process, dynamically we are creating an image with random <code>string</code>, either <code>string</code>, number or alphanumeric and displaying it on the web page. After that, at the time of form submission, the user is asked to enter the same code in the text box as shown in that image. Once the form is submitted, then we can compare both auto generated value and entered by user because there is no other good way to protect your webworms from spammers (auto submission).</p> 
             <h3>How Can We Do That?</h3> 
             <p>Here is the detailed explanation with code snippets.</p> 
             <p>This is the <em>default.aspx</em> (partially):</p>

</span><span class="code-attribute">	</span><span class="code-attribute">ForeColor</span><span class="code-keyword">="</span><span class="code-keyword">Red"</span><span class="code-attribute"> </span><span class="code-attribute">Text</span><span class="code-keyword">="</span><span class="code-keyword">"</span><span class="code-keyword">&gt;</span><span class="code-keyword">&lt;</span><span class="code-leadattribute">/asp:Label</span><span class="code-keyword">&gt;</span>

             <h3>CImage.aspx Page</h3> 
             <p>Remove all the code except the first line.</p> 
             <p>The main code is here which is calling a method to generate random number <code>string</code> and passes it to the class file which will generate image with the same <code>string</code> and return it as a JPEG format.</p> 
             <p>Here are the details of the code:</p> 
             <pre lang="cs"><span class="code-keyword">using</span> System.Drawing.Imaging;
<span class="code-keyword">public</span> <span class="code-keyword">partial</span> <span class="code-keyword">class</span> CImage : System.Web.UI.Page

        <span class="code-comment">//</span><span class="code-comment"> Create a CAPTCHA image using the text stored in the Session object.</span>
        RandomImage ci = <span class="code-keyword">new</span> RandomImage(<span class="code-keyword">this</span>.Session
			[<span class="code-string">"</span><span class="code-string">CaptchaImageText"</span>].ToString(), <span class="code-digit">300</span>, <span class="code-digit">75</span>);

    }
   
    <span class="code-comment">//</span><span class="code-comment"> Function to generate random string with Random class.</span>
    <span class="code-keyword">private</span> <span class="code-keyword">string</span> GenerateRandomCode()
    {
        Random r = <span class="code-keyword">new</span> Random();
        <span class="code-keyword">string</span> s = <span class="code-string">"</span><span class="code-string">"</span>;
        <span class="code-keyword">for</span> (<span class="code-keyword">int</span> j = <span class="code-digit">0</span>; j <span class="code-keyword">&lt;</span> <span class="code-digit">5</span>;j++)
        {
            <span class="code-keyword">int</span> i = r.Next(<span class="code-digit">3</span>);
            <span class="code-keyword">int</span> ch;
            <span class="code-keyword">switch</span> (i)
            {
                <span class="code-keyword">case</span> <span class="code-digit">1</span>:
                    ch = r.Next(<span class="code-digit">0</span>, <span class="code-digit">9</span>);
                    s = s + ch.ToString();
                    <span class="code-keyword">break</span>;
                <span class="code-keyword">case</span> <span class="code-digit">2</span>:
                    ch = r.Next(<span class="code-digit">65</span>, <span class="code-digit">90</span>);
                    s = s + Convert.ToChar(ch).ToString();
                    <span class="code-keyword">break</span>;
                <span class="code-keyword">case</span> <span class="code-digit">3</span>:
                    ch = r.Next(<span class="code-digit">97</span>, <span class="code-digit">122</span>);
                    s = s + Convert.ToChar(ch).ToString();
                    <span class="code-keyword">break</span>;
                <span class="code-keyword">default</span>:
                    ch = r.Next(<span class="code-digit">97</span>, <span class="code-digit">122</span>);
                    s = s + Convert.ToChar(ch).ToString();
                    <span class="code-keyword">break</span>;
            }
            r.NextDouble();
            r.Next(<span class="code-digit">100</span>, <span class="code-digit">1999</span>);
        }
        <span class="code-keyword">return</span> s;
    }
