        <div class="post-body__content">
         <p>Apps with interfaces that make use of voice have a unique appeal. They tend to make their users feel that they are using something futuristic. Since its early days, Android has had very robust text-to-speech (TTS) functionality. This year, Google has added a lot of high-quality voices to its TTS engine and that's&nbsp;all the more reason for developers to use it in their apps.</p>
         <p>In this tutorial, you'll learn how to create a simple appwith a minimalist user interfacethat can receive text messages and read them to the user.</p>
         <h2>Prerequisites</h2>
         <p>Ensure that you have the Eclipse ADT Bundle set up. You can download it at the <a href="http://developer.android.com/sdk/index.html" rel="external" target="_blank">Android Developer website</a>. For best results, you are also going to need a real Android device and some friends who can send you text messages.</p>
         <h2>1. Create a New Project<br> </h2>
         <p>Start Eclipse, and create a new Android application. Call this application&nbsp;<strong>SMSReader</strong>. If you think you are going to publish this app to Google Play to share it with your friends, then make sure you use a unique package name. Set the <strong>Minimum Required SDK</strong> to <strong>Android 2.2</strong>&nbsp;and set the <strong>Target SDK</strong> to <strong>Android 4.4</strong>.</p>
         <figure class="post_image">
          <img alt="New Android Application window" src="https://cms-assets.tutsplus.com/uploads/users/362/posts/22524/image/s1.png">
         </figure>
         <p>This app will have one <code class="inline">Activity</code>. Select <strong>Create Activity</strong>&nbsp;and choose <strong>Empty Activity</strong>.<br></p>
         <figure class="post_image">
          <img alt="Create an Empty new Activity" src="https://cms-assets.tutsplus.com/uploads/users/362/posts/22524/image/s3.png">
         </figure>
         <p>Name it&nbsp;<strong>MainActivity</strong> and click <strong>Finish</strong>.</p>
         <figure class="post_image">
          <img alt="Name the Activity" src="https://cms-assets.tutsplus.com/uploads/users/362/posts/22524/image/s4.png">
         </figure>
         <h2> <span class="sectionnum">2.</span> Edit the Manifest</h2>
         <p>This app needs three permissions:</p>
         <ul> 
          <li> <strong>RECEIVE_SMS</strong> to know that the device has received an SMS</li> 
          <li> <strong>READ_SMS</strong> to read that SMS</li> 
          <li> <strong>READ_CONTACTS</strong> to map the phone number of the sender to a name (if possible)<br> </li> 
         </ul>
         <p>Add the following lines to your <strong>AndroidManifest.xml</strong>.</p>
         <pre class="brush: xml noskimlinks noskimwords">&lt;uses-permission android:name="android.permission.READ_SMS"/&gt;
&lt;uses-permission android:name="android.permission.RECEIVE_SMS"/&gt;
&lt;uses-permission android:name="android.permission.READ_CONTACTS"/&gt;</pre>
         <p>This app is going to have only one screen orientation, portrait. Therefore, edit the <code class="inline">activity</code> tag and add the following attribute to it:</p>
         <pre class="brush: xml noskimlinks noskimwords">android:screenOrientation="portrait"</pre>
         <p>The manifest is now complete.</p>
         <h2> <span class="sectionnum">3.</span> Edit strings.xml</h2>
         <p>It is not absolutely necessary, but storing all strings that the application uses in the <strong>res/values/strings.xml</strong> file is a good practice. Edit this file so that it has the following contents:</p>
         <pre class="brush: xml noskimlinks noskimwords">&lt;?xml version="1.0" encoding="utf-8"?&gt;
&lt;resources&gt;
    &lt;string name="app_name"&gt;SMSReader&lt;/string&gt;
    &lt;string name="sms_label"&gt;Latest SMS&lt;/string&gt;
    &lt;string name="none"&gt;None&lt;/string&gt;
    &lt;string name="speech_toggle_on"&gt;START SPEAKING&lt;/string&gt;
	&lt;string name="speech_toggle_off"&gt;STOP SPEAKING&lt;/string&gt;
	
	&lt;string name="start_speaking"&gt;Okay! I will read your messages out loud for you now.&lt;/string&gt;
	&lt;string name="stop_speaking"&gt;Okay! I will stay silent now.&lt;/string&gt;
&lt;/resources&gt;</pre>
         <p>Most of these strings are used in the next step.<br></p>
         <h2> <span class="sectionnum">4.</span> Edit the Layout</h2>
         <p>Edit <strong>res/layout/activity_main.xml</strong> to add the following:</p>
         <ul> 
          <li>a&nbsp;<code class="inline">TextView</code> to display the name of the person who sent the latest SMS<br> </li> 
          <li>a&nbsp;<code class="inline">TextView</code> to display the contents of the latest SMS</li> 
          <li>a&nbsp;<code class="inline">ToggleButton</code> to toggle speech output on and off</li> 
         </ul>
         <p>After adding code for positioning and styling these elements, your file should have the following contents:</p>
         <pre class="brush: xml noskimlinks noskimwords">&lt;RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context="${packageName}.${activityClass}"
    android:background="#99CC00"
    &gt;

    &lt;TextView
        android:id="@+id/sms_sender"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentTop="true"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="20dp"
        android:text="@string/sms_label"
        android:textColor="#ffffff"
        android:textAppearance="?android:attr/textAppearanceSmall" /&gt;

    &lt;TextView
        android:id="@+id/sms_text"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_below="@+id/sms_sender"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="10dp"
        android:text="@string/none"
        android:textAppearance="?android:attr/textAppearanceLarge" 
        android:textColor="#ffffff"
        /&gt;

    &lt;ToggleButton
        android:id="@+id/speechToggle"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerHorizontal="true"
        android:layout_centerVertical="true"
        android:textOff="@string/speech_toggle_on"
        android:textOn="@string/speech_toggle_off"                 
        /&gt;
    
&lt;/RelativeLayout&gt;</pre>
         <p>The layout for our application is now complete.</p>
         <h2> <span class="sectionnum">5.</span> Create a Helper Class</h2>
         <p>We're now going to create a helper class for the TTS engine.&nbsp;Create a new Java class and call it <strong>Speaker.java</strong>. This class is used to avoid calling the TTS API directly from the <code class="inline">Activity</code>.</p>
         <p>This class implements the&nbsp;<code class="inline">OnInitListener</code>&nbsp;interface so that it knows when the TTS engine is ready. We store this ready state in a boolean variable named <code class="inline">ready</code>. We use another boolean variable named <code class="inline">allowed</code>&nbsp;whose value is <code class="inline">true</code> only if the user has allowed the TTS engine to speak. We also add methods to get and set the value of this variable. At this point, <strong>Speaker.java</strong> should have the following contents:</p>
         <pre class="brush: java noskimlinks noskimwords">public class Speaker implements OnInitListener {

    private TextToSpeech tts;
	
	private boolean ready = false;
	
	private boolean allowed = false;
	
	public Speaker(Context context){
		tts = new TextToSpeech(context, this);		
	}	
	
	public boolean isAllowed(){
		return allowed;
	}
	
	public void allow(boolean allowed){
		this.allowed = allowed;
	}
}</pre>
         <p>The <code class="inline">OnInitListener</code> interface has only one method,&nbsp;<code class="inline">onInit</code>. This method is called when the TTS engine has been&nbsp;initialized. The <code class="inline">status</code> parameter lets us know if the initialization was successful. Once we know that the initialization was successful, we set the language of the TTS engine. This is important to produce speech that is comprehensible. Add the following code:</p>
         <pre class="brush: java noskimlinks noskimwords">@Override
public void onInit(int status) {
	if(status == TextToSpeech.SUCCESS){
		// Change this to match your
		// locale
		tts.setLanguage(Locale.US);
		ready = true;
	}else{
		ready = false;
	}
}</pre>
         <p>Next, we add a method named <code class="inline">speak</code>, which uses the engine to read out any text that is passed to it. Before doing so, it checks if both the <code class="inline">allowed</code> and the <code class="inline">ready</code> values are <code class="inline">true</code>. The speech it generates is placed in the notification stream.</p>
         <pre class="brush: java noskimlinks noskimwords">public void speak(String text){
	
	// Speak only if the TTS is ready
	// and the user has allowed speech
	
	if(ready &amp;&amp; allowed) {
		HashMap&lt;String, String&gt; hash = new HashMap&lt;String,String&gt;();
		hash.put(TextToSpeech.Engine.KEY_PARAM_STREAM, 
				String.valueOf(AudioManager.STREAM_NOTIFICATION));
		tts.speak(text, TextToSpeech.QUEUE_ADD, hash);
	}
}</pre>
         <p>We then add a method that plays silence for a specified duration. Using this method, we can add pauses to the speech to make it sound a little clearer. Add the following code to the implementation:</p>
         <pre class="brush: java noskimlinks noskimwords">public void pause(int duration){
	tts.playSilence(duration, TextToSpeech.QUEUE_ADD, null);
}</pre>
         <p>Finally, add a method to free up resources when the TTS engine is no longer needed.</p>
         <pre class="brush: java noskimlinks noskimwords">// Free up resources
public void destroy(){
	tts.shutdown();
}</pre>
         <h2> <span class="sectionnum">6.</span> Edit the Activity Class</h2>
         <p>Edit <strong>MainActivity.java</strong>&nbsp;and declare all the views that we mentioned in the layout.&nbsp;Declare two integers,&nbsp;<code class="inline">LONG_DURATION</code> and <code class="inline">SHORT_DURATION</code>. These are merely values that are passed to the <code class="inline">Speaker</code>'s <code class="inline">pause</code> method.</p>
         <p>Also declare a <code class="inline">CHECK_CODE</code> integer. It's value isn't important.&nbsp;It is passed to the <code class="inline">startActivityforResult</code> method and then used to identify the result.</p>
         <p>Lastly, declare a <code class="inline">Speaker</code> object and a <code class="inline">BroadcastReceiver</code> object.<br></p>
         <p>At this point, your class should look like this:<br></p>
         <pre class="brush: java noskimlinks noskimwords">public class MainActivity extends Activity {    
    
	private final int CHECK_CODE = 0x1;
	private final int LONG_DURATION = 5000;
	private final int SHORT_DURATION = 1200;
	
	private Speaker speaker;	
	
	private ToggleButton toggle;
	private OnCheckedChangeListener toggleListener;
	
	private TextView smsText;
	private TextView smsSender;
	
	private BroadcastReceiver smsReceiver;
}</pre>
         <p>Add a method to check if a TTS engine is installed on the device. The check is performed by making use of the result of another <code class="inline">Activity</code>.</p>
         <pre class="brush: java noskimlinks noskimwords">private void checkTTS(){
	Intent check = new Intent();
	check.setAction(TextToSpeech.Engine.ACTION_CHECK_TTS_DATA);
	startActivityForResult(check, CHECK_CODE);
}</pre>
         <p>When the result of <code class="inline">startActivityForResult</code> arrives, the <code class="inline">onActivityResult</code> method is called. Therefore, we need to override it. In this method, if the result is positive, we initialize the <code class="inline">Speaker</code> object. If no TTS engine is installed, we redirect the user to install it.</p>
         <pre class="brush: java noskimlinks noskimwords">@Override
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
	if(requestCode == CHECK_CODE){
		if(resultCode == TextToSpeech.Engine.CHECK_VOICE_DATA_PASS){
			speaker = new Speaker(this);
		}else {
            Intent install = new Intent();
            install.setAction(TextToSpeech.Engine.ACTION_INSTALL_TTS_DATA);
            startActivity(install);
        }
	}
}</pre>
         <p>It is now time to create our <code class="inline">BroadcastReceiver</code> to deal with the messages that the device is receiving. Whenever there are new messages, its <code class="inline">onReceive</code> method is called. We parse the messages, which&nbsp;arrive as byte arrays, using the <code class="inline">SmsMessage</code> class. Once the message is parsed, we use methods like <code class="inline">getDisplayMessageBody</code> and <code class="inline">getOriginatingAddress</code> to extract meaningful information out of it.</p>
         <p>With this information, we generate the text that the TTS engine should read out. We pause for <code class="inline">LONG_DURATION</code> before reading out a new SMS and for <code class="inline">SHORT_DURATION</code> between the utterances of the SMS sender's name and the SMS's&nbsp;body.<br></p>
         <p>Add the following code to the implementation:</p>
         <pre class="brush: java noskimlinks noskimwords">private void initializeSMSReceiver(){
	smsReceiver = new BroadcastReceiver(){
		@Override
		public void onReceive(Context context, Intent intent) {
			
			Bundle bundle = intent.getExtras();
			if(bundle!=null){
				Object[] pdus = (Object[])bundle.get("pdus");
				for(int i=0;i&lt;pdus.length;i++){
					byte[] pdu = (byte[])pdus[i];
					SmsMessage message = SmsMessage.createFromPdu(pdu);
					String text = message.getDisplayMessageBody();
					String sender = getContactName(message.getOriginatingAddress());
					speaker.pause(LONG_DURATION);
					speaker.speak("You have a new message from" + sender + "!");
					speaker.pause(SHORT_DURATION);
					speaker.speak(text);
					smsSender.setText("Message from " + sender);
					smsText.setText(text);
				}
			}
			
		}			
	};		
}</pre>
         <p>We can only extract the phone number of the sender from the message. To map this number to a contact's name, we have to make use of the user's contacts. The following method queries the contacts data. If the phone number is not available in the user's contacts, then it simply returns the string <code class="inline">unknown number</code>:</p>
         <pre class="brush: java noskimlinks noskimwords">private String getContactName(String phone){
	Uri uri = Uri.withAppendedPath(PhoneLookup.CONTENT_FILTER_URI, Uri.encode(phone));
	String projection[] = new String[]{ContactsContract.Data.DISPLAY_NAME};
	Cursor cursor = getContentResolver().query(uri, projection, null, null, null);				
	if(cursor.moveToFirst()){
		return cursor.getString(0);
	}else {
		return "unknown number";
	}
}</pre>
         <p>Before the <code class="inline">BroadcastReceiver</code> can be used, it has to be registered. In the following method, we create an <code class="inline">IntentFilter</code> for incoming text messages and then register our <code class="inline">smsReceiver</code> for it:</p>
         <pre class="brush: java noskimlinks noskimwords">private void registerSMSReceiver() {	
	IntentFilter intentFilter = new IntentFilter("android.provider.Telephony.SMS_RECEIVED");
	registerReceiver(smsReceiver, intentFilter);
}</pre>
         <p>Next, we create the <code class="inline">onCreate</code> method. Here's where we initialize all the objects we declared. We initialize the <code class="inline">toggleListener</code> to set the value of <code class="inline">allowed</code> in the <code class="inline">Speaker</code> class.</p>
         <p>After these initializations, we call the <code class="inline">checkTTS</code>, <code class="inline">initializeSMSReceiver</code>, and <code class="inline">registerSMSReceiver</code> methods.<br></p>
         <pre class="brush: java noskimlinks noskimwords">@Override
protected void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_main);			
	
	toggle = (ToggleButton)findViewById(R.id.speechToggle);	
	smsText = (TextView)findViewById(R.id.sms_text);
	smsSender = (TextView)findViewById(R.id.sms_sender);

	toggleListener = new OnCheckedChangeListener() {			
		@Override
		public void onCheckedChanged(CompoundButton view, boolean isChecked) {
			if(isChecked){
				speaker.allow(true);
				speaker.speak(getString(R.string.start_speaking));
			}else{
				speaker.speak(getString(R.string.stop_speaking));
				speaker.allow(false);					
			}
		}
	};		
	toggle.setOnCheckedChangeListener(toggleListener);
	
	checkTTS();
	initializeSMSReceiver();
	registerSMSReceiver();
}	</pre>
         <p>Finally, in the <code class="inline">onDestroy</code> method of the activity, we unregister our receiver and shut down the TTS engine to free up resources.</p>
         <pre class="brush: java noskimlinks noskimwords">@Override
protected void onDestroy() {	
	super.onDestroy();
	unregisterReceiver(smsReceiver);
	speaker.destroy();
}</pre>
         <h2> <span class="sectionnum">7.</span> Run and Test</h2>
         <p>The app is now ready to be tested. Compile and run it on a physical Android device. Tap the toggle button to enable voice and send yourself an SMS from another phone or ask one of your friends to do so. You should soon be able to hear your phone read out the SMS for you.</p>
         <p>Here is a sample of the speech generated by the TTS engine:<br></p>
         <audio src="https://cms-assets.tutsplus.com/uploads/users/362/posts/22524/audio/translate_tts1.mp3" controls="controls"></audio>
         <h2>Conclusion</h2>
         <p>In this tutorial, you have learnt to not only use the text-to-speech API, but also to use broadcast receivers&nbsp;and to make sense out of raw SMS data. You can now go on to further customize this app to you needs.</p>
