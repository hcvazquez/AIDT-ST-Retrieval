          <p style="text-align: justify;"><img class="alignright wp-image-864 size-medium" title="Cameleon (mwanasimba, Flickr Creative Commons)" src="https://myjeeva.com/content/blog/cameleon-mwanasimba-at-flickr-300x225.jpg" alt="Convert Image to String and String to Image in Java" width="300" height="225">The goal here is to convert/transform Image into Base64 String and convert/transform Base64 String back to Image without&nbsp;compromising a Image data.&nbsp; I will be using <a title="https://commons.apache.org/codec/apidocs/overview-summary.html" href="https://commons.apache.org/codec/apidocs/overview-summary.html" target="_blank">Apache Commons Codec</a> library to achieve the ‘Convert Image to String and String to Image in Java’. This article was triggered by <a title="http://www.linkedin.com/groups/How-convert-binary-string-image-52795.S.129450809?qid=f0683e5e-e6c6-4231-a0ac-4b58aaeee012&amp;goback=.gmp_52795" href="https://www.linkedin.com/groups/How-convert-binary-string-image-52795.S.129450809?qid=f0683e5e-e6c6-4231-a0ac-4b58aaeee012&amp;goback=.gmp_52795" target="_blank">Linkedin Discussion</a>.</p>
          <p><strong>Class:</strong> <a title="https://commons.apache.org/codec/apidocs/org/apache/commons/codec/binary/Base64.html" href="https://commons.apache.org/codec/apidocs/org/apache/commons/codec/binary/Base64.html" target="_blank"><strong>org.apache.commons.codec.binary.Base64</strong></a></p>
          <p><strong>Methods:</strong> Base64 class has overloaded methods</p>
          <ul>
           <li> <span id="crayon-59becb3fc8df5822114032" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">Base64.encodeBase64URLSafeString</span></span>&nbsp;– accepts byte array and converts into Base64 String</li>
           <li> <span id="crayon-59becb3fc8e07486536811" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">Base64.decodeBase64</span></span>&nbsp;– accepts Base64 String and converts into byte array</li>
          </ul>
          <hr class="myjeeva_divider">
          <h2><span style="color: #333399;">How to do: step by step?</span></h2>
          <h3>Step 1</h3>
          <p>Preparing a <span id="crayon-59becb3fc8e0b235881174" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">byte array</span></span>&nbsp;from image.&nbsp; Typically achieved in following ways:</p>
          <ul>
           <li>During File upload – at server side application will be receiving a file data as byte stream</li>
           <li>Reading a image file from File System as byte stream</li>
          </ul>
          <h3>Step 2</h3>
          <p style="text-align: justify;">Once we have byte array of Image file, apply below method to convert byte array into <span id="crayon-59becb3fc8e0d089642984" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">Base64</span></span>&nbsp;string.&nbsp; Here even though string is not used in URL param, consider using <span id="crayon-59becb3fc8e10317358963" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">URL safe Base 64</span></span>&nbsp;string.&nbsp; The url-safe variation emits <span id="crayon-59becb3fc8e13824077611" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4;">- and _</span></span>&nbsp;instead of <span id="crayon-59becb3fc8e15737862565" class="crayon-syntax crayon-syntax-inline  crayon-theme-myjeeva-sublime-text crayon-theme-myjeeva-sublime-text-inline crayon-font-consolas" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important;"><span class="crayon-pre crayon-code" style="font-size: 13px !important; line-height: 15px !important;font-size: 13px !important; -moz-tab-size:4; package com.myjeeva.image;

import java.io.File;
import java.io.FileInputStream;
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.IOException;

import org.apache.commons.codec.binary.Base64;

/**
 * @desc Image manipulation - Conversion
 * 
 * @filename ImageManipulation.java
 * @author Jeevanandam M. (jeeva@myjeeva.com)
 * @copyright myjeeva.com
 */
public class ImageManipulation {

	public static void main(String[] args) {

		File file = new File("/Users/jeeva/Pictures/wallpapers/water-drop.jpg");

		try {			
			// Reading a Image file from file system
			FileInputStream imageInFile = new FileInputStream(file);
			byte imageData[] = new byte[(int) file.length()];
			imageInFile.read(imageData);

			// Converting Image byte array into Base64 String
			String imageDataString = encodeImage(imageData);

			// Converting a Base64 String into Image byte array
			byte[] imageByteArray = decodeImage(imageDataString);

			// Write a image byte array into file system
			FileOutputStream imageOutFile = new FileOutputStream(
					"/Users/jeeva/Pictures/wallpapers/water-drop-after-convert.jpg");

			imageOutFile.write(imageByteArray);

			imageInFile.close();
			imageOutFile.close();

			System.out.println("Image Successfully Manipulated!");
		} catch (FileNotFoundException e) {
			System.out.println("Image not found" + e);
		} catch (IOException ioe) {
			System.out.println("Exception while reading the Image " + ioe);
		}
	}

	/**
	 * Encodes the byte array into base64 string
	 *
	 * @param imageByteArray - byte array
	 * @return String a {@link java.lang.String}
	 */
	public static String encodeImage(byte[] imageByteArray) {
		return Base64.encodeBase64URLSafeString(imageByteArray);
	}

	/**
	 * Decodes the base64 string into byte array
	 *
	 * @param imageDataString - a {@link java.lang.String}
	 * @return byte array
	 */
	public static byte[] decodeImage(String imageDataString) {
		return Base64.decodeBase64(imageDataString);
	}
}</textarea>
