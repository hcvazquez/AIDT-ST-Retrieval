                        <td valign="top" width="257"> <p><span class="style12">The DBMS_WORKLOAD_CAPTURE package configures the Workload Capture system and produces the workload capture data. Replay of this capture is implemented by way of the DBMS_WORKLOAD_REPLAY</span> &nbsp;package.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top"> <p>DBMS_WORKLOAD_REPLAY </p></td>
                        <td valign="top" width="257"> <p>The DBMS_WORKLOAD_REPLAY <font class="contentText"> <span class="style12">&nbsp;package provides an interface to replay a workload capture that was originally created by way of the DBMS_WORKLOAD_CAPTURE package. Typically, the DBMS_WORKLOAD_CAPTURE package will be used in the production system to capture a production workload, and the DBMS_WORKLOAD_REPLAY package will be subsequently used in a test system to replay the captured production workload for testing purposes.</span></font></p><font class="contentText"> </font></td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XA</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The DBMS_XA package contains the XA/Open interface for applications to call XA interface in PL/SQL. Using this package, application developers can switch or share transactions across SQL*Plus sessions or processes using PL/SQL.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XDB_ADMIN</p> </td>  
                        <td valign="top" width="257" class="style12"> <p class="style12">The DBMS_XDB_ADMIN package provides an interface to manage the XMLIndex on the XML DB repository.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XDBRESOURCE</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The DBMS_XDBRESOURCE package provides the interface to operate on the resource's metadata and contents.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XEVENT</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The DBMS_XEVENT package provides event-related types and supporting subprograms.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XMLINDEX</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The DBMS_XMLINDEX package provides an interface to implement asynchronous indexing. </p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">DBMS_XMLTRANSLATIONS</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The DBMS_XMLTRANSLATIONS package provides an interface to perform translations so that strings can be searched or displayed in various languages.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_CSW_PROCESS</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_CSW_PROCESS package contains subprograms for various processing operations related to support for Catalog Services for the Web (CSW).</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_GEOR_ADMIN</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_GEOR_ADMIN package contains subprograms for administrative operations related to GeoRaster.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_OLS</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_OLS package contains subprograms for Spatial OpenLS support.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_PC_PKG</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_PC_PKG package contains subprograms to support the use of point clouds in Spatial.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_TIN_PKG</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_TIN_PKG package contains subprograms to support the use of triangulated irregular networks (TINs) in Spatial.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_WFS_LOCK</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_WFS_LOCK package contains subprograms for WFS support for registering and unregistering feature tables. Registering a feature table enables the table for WFS transaction locking; unregistering a feature table disables the table for WFS transaction locking.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SDO_WFS_PROCESS</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SDO_WFS_PROCESS package contains subprograms for various processing operations related to support for Web Feature Services.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SEM_APIS</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SEM_APIS package contains subprograms for working with the Resource Description Framework (RDF) and Web Ontology Language (OWL) in an Oracle database.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">SEM_PERF</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The SEM_PERF package contains subprograms for examining and enhancing the performance of the Resource Description Framework (RDF) and Web Ontology Language (OWL) support in an Oracle database.</p> </td> 
                       </tr> 
                       <tr> 
                        <td valign="top" style="width: 263px" class="style12"> <p class="style12">UTL_SPADV</p> </td> 
                        <td valign="top" width="257"> <p class="style12">The UTL_SPADV package, one of a set of Oracle Streams packages, provides subprograms to collect and analyze statistics for the Oracle Streams components in a distributed database environment. This package uses the Oracle Streams Performance Advisor to gather statistics.</p> </td> 
                       </tr>
