                   Hello, I'm from singapore also. and I know a friend who has access to free sms gateway if you are interested. However, I feel that sending sms through your mobile device using your hand phone is a better alternative. In fact, I just finished testing a few programs to ans another similiar question. I'll just copy it from the other thread.
                  <br> 
                  <br> First, you need the java.comm api for windows, you can get it here.
                  <br> 
                  <a href="http://www.jsmsengine.org/" target="_blank" rel="nofollow">http://www.jsmsengine.org/</a>
                  <br> 
                  <br> jsms engine also has some examples on how to use their API, which I think makes it easier to write your own programs.
                  <br> 
                  <br> Here's a nice gui program that sends and receives SMS using just the java.comm api.
                  <br> 
                  <a href="http://www.wrankl.de/SMST/SMST.html" target="_blank" rel="nofollow">http://www.wrankl.de/SMST/SMST.html</a>
                  <br> 
                  <br> Also, if you are using infrared port you need to map it to a virtual com port, you can use this driver
                  <br> 
                  <a href="http://www.ircomm2k.de/" target="_blank" rel="nofollow">http://www.ircomm2k.de/</a>
                  <br> 
                  <br> If you have any problems with those program please let me know.
                  <br> good luck.

                   Sorry, it is no longer free as my friend doesn't work at the telco there any more.
                  <br> 
                  <br> However, there are 2 options avaliable to you
                  <br> 
                  <br> 1) setup your own SMS gateway. This is an open source sms gateway, you'll need to get a work station running Linux to set up the gateway, then you can use your program to send text messages via HTML to the gateway, and the gateway in turn will send the sms to your mobile device via WAP.
                  <br> 
                  <br> 
                  <a href="http://www.kannel.org/" target="_blank" rel="nofollow">http://www.kannel.org/</a>
                  <br> 
                  <br> 2) Another alternative, is to subscribe to sites which provide internet sms for a fee. Such as
                  <br> 
                  <br> 
                  <a href="http://www.ipipi.com/" target="_blank" rel="nofollow">http://www.ipipi.com/</a>
                  <br> 
                  <br> So you write a client to send an email, via Java mail API to their servers, and in turn an SMS will be sent to your mobile phone. While this will cost you some money, it is much faster to get going. 
                  <br> 
                  <br> Good luck
