       <p>In this section let's look into the implementation of basic sorting algorithms in java.<br> In today's world of computer science, sorting is a basic operation which is unavoidable. Whatever may be the application you work on, sorting of data will be the preliminary operation that one would be required to do. Here we will stick only to integer sorting.<br> The following java program is self explanatory with necessary comments. It contains a set of methods implementing various sorting algorithms to sort integer values in ascending order.<br> The algorithms implemented here are bubble sort, selection sort, insertion sort, quick sort and heap sort. OK! Which is the best sorting algorithm?<br> The answer depends on the data set used. If the data set is less (say 10 values) then bubble sort (or selection sort) will work fine. In other case if there is a huge data set to be sorted (say 1000 values) then quick or heap sort will do best as the complexity is 0(nlogn). </p>

        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">if</span> 
        <span style="color: #009900;">(</span>finalArr
        <span style="color: #009900;">[</span>lowerLimit
        <span style="color: #009900;">]</span> 
        <span style="color: #339933;">&gt;</span> insElement
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">if</span> 
        <span style="color: #009900;">(</span>lowerLimit == 
        <span style="color: #cc66cc;">0</span>
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; insertElement
        <span style="color: #009900;">(</span>finalArr, insElement, lowerLimit, len + 
        <span style="color: #cc66cc;">1</span>
        <span style="color: #009900;">)</span>
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">else</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; insertElement
        <span style="color: #009900;">(</span>finalArr, insElement, lowerLimit - 
        <span style="color: #cc66cc;">1</span>, len + 
        <span style="color: #cc66cc;">1</span>
        <span style="color: #009900;">)</span>
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">else</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; insertElement
        <span style="color: #009900;">(</span>finalArr, insElement, lowerLimit + 
        <span style="color: #cc66cc;">1</span>, len + 
        <span style="color: #cc66cc;">1</span>
        <span style="color: #009900;">)</span>
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> 
        <br> &nbsp; &nbsp; 
        <span style="color: #000000; font-weight: bold;">public</span> 
        <span style="color: #006600; font-weight: bold;">void</span> insertElement
        <span style="color: #009900;">(</span>
        <span style="color: #006600; font-weight: bold;">int</span>
        <span style="color: #009900;">[</span>
        <span style="color: #009900;">]</span> finalArr, 
        <span style="color: #006600; font-weight: bold;">int</span> insElement, 
        <span style="color: #006600; font-weight: bold;">int</span> pos, 
        <span style="color: #006600; font-weight: bold;">int</span> len
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #006600; font-weight: bold;">int</span> addElem = insElement
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">for</span> 
        <span style="color: #009900;">(</span>
        <span style="color: #006600; font-weight: bold;">int</span> i = pos
        <span style="color: #339933;">;</span> i 
        <span style="color: #339933;">&lt;</span> len
        <span style="color: #339933;">;</span> i++
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #006600; font-weight: bold;">int</span> temp = finalArr
        <span style="color: #009900;">[</span>i
        <span style="color: #009900;">]</span>
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; finalArr
        <span style="color: #009900;">[</span>i
        <span style="color: #009900;">]</span> = addElem
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; addElem = temp
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> 
        <br> &nbsp; &nbsp; 
        <span style="color: #000000; font-weight: bold;">public</span> 
        <span style="color: #006600; font-weight: bold;">void</span> printArr
        <span style="color: #009900;">(</span>
        <span style="color: #006600; font-weight: bold;">int</span>
        <span style="color: #009900;">[</span>
        <span style="color: #009900;">]</span> a
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #000000; &nbsp;font-weight: bold;">for</span> 
        <span style="color: #009900;">(</span>
        <span style="color: #006600; font-weight: bold;">int</span> i = 
        <span style="color: #cc66cc;">0</span>
        <span style="color: #339933;">;</span> i 
        <span style="color: #339933;">&lt;</span> a.
        <span style="color: #006633;">length</span>
        <span style="color: #339933;">;</span> i++
        <span style="color: #009900;">)</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">{</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
        <a href="http://java.sun.com/j2se/1%2E5%2E0/docs/api/java/lang/System.html"><span style="color: #003399; font-weight: bold;">System</span></a>.
        <span style="color: #006633;">out</span>.
        <span style="color: #006633;">print</span>
        <span style="color: #009900;">(</span>a
        <span style="color: #009900;">[</span>i
        <span style="color: #009900;">]</span> + 
        <span style="color: #0000ff;">" "</span>
        <span style="color: #009900;">)</span>
        <span style="color: #339933;">;</span>
        <br> &nbsp; &nbsp; &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> &nbsp; &nbsp; 
        <span style="color: #009900;">}</span>
        <br> 
        <span style="color: #009900;">}</span>
       </div>
