             <div id="ctl00_description" class="summary">
              Simply send SMS using Java with GSM Modem or HSDPA Dongle
             </div>
             <span id="ctl00_thumbnailUrl" class="date" content="https://www.codeproject.com/script/Articles/Images/tip100x80.png"></span> 
            </div> 
            <div id="contentdiv" class="text"> 
             <p><strong>Hello Everyone!</strong></p> 
             <p>Today I am going to tell you about how to simply send SMS by using your HSDPA dongle or GSM modem. Cool! Isn't it? yes It's not that difficult either. I have tried so many ways to accomplish this. After so many attempts I got the way to do it right. So I will tell you how I have done that.</p>
             <div class="separator" style="text-align: center; color: rgb(78, 78, 78); font-family: &quot;Segoe UI&quot;, Calibri, &quot;Myriad Pro&quot;, Myriad, &quot;Trebuchet MS&quot;, Helvetica, Arial, sans-serif; font-size: small; background-color: rgb(255, 255, 255)">
              <img width="400" height="170" src="https://1.bp.blogspot.com/-gzhqJ2en_Bw/T-WVpkDgjyI/AAAAAAAAA08/2a2i3TphR4c/s400/imagddes.jpeg" border="0">
             </div> 
             <br>
             <div class="separator" style="text-align: center; color: rgb(78, 78, 78); font-family: &quot;Segoe UI&quot;, Calibri, &quot;Myriad Pro&quot;, Myriad, &quot;Trebuchet MS&quot;, Helvetica, Arial, sans-serif; font-size: small; background-color: rgb(255, 255, 255)"></div>
             <div class="separator" style="text-align: center; color: rgb(78, 78, 78); font-family: &quot;Segoe UI&quot;, Calibri, &quot;Myriad Pro&quot;, Myriad, &quot;Trebuchet MS&quot;, Helvetica, Arial, sans-serif; font-size: small; background-color: rgb(255, 255, 255)"></div>
             <p> <br>Basically this was created by using&nbsp;<a href="http://www.smslib.org/">SMSLib</a>&nbsp;which is an open source library for Java. After studying the API I have created a simple API wrapper for convenience. I have hosted the open source code in Github with a sample project named&nbsp;<a href="https://github.com/harshadura/SMS.Dura.Wrapper">SMS.Dura.Wrapper</a>. For this sample project I have made and used a prebuilt Jar library named&nbsp;<strong>smsdura-1.0.jar</strong>&nbsp;which works as the wrapper. So in your project you have to simply import that smsdura-1.0.jar, just configure the modem, put the message, TP, and send. Now SMS from Java? It' s very easy! I will tell you how to do that. May be it will add more value for your project as well.</p>
             <p>I will guide you through the rest of the process. Okay let's start! </p>
             <p>1. First of all you have to&nbsp;<a href="https://github.com/harshadura/SMS.Dura.Wrapper/zipball/master">Download&nbsp;</a>this sample project.</p> 
             <p>2. Now we have to copy few files in the &lt;Extras folder&gt; to your Java Classpath. [Normally JDKDIR can be found in <em>&gt; C:\Program Files\Java</em>] Go to that path and do it as said. Detailed instructions are listed below.</p> 
             <pre lang="text">---------------------------&nbsp;&nbsp;&nbsp;&nbsp;
    External Configurations for the JVM on the targeted machine. (Strictly Recommended!)
    [These files can be found in &lt;extras&gt; folder of the Project path.)
---------------------------
Java Comm Installation &lt;comm dir&gt;
-----------------------------------
    File comm.jar should go under JDKDIR/jre/lib/ext/&nbsp;
    File javax.comm.properties should go under JDKDIR/jre/lib/
    Library files (i.e. win32com.dll for Win32 or the .so Linux library files) should go under JDKDIR/jre/bin/
    If you have a separate JRE directory, do the same copies for the JREDIR directory!
RxTx Installation &lt;rxtx dir&gt;
-----------------------------------
    File RXTXcomm.jar should go under JDKDIR/jre/lib/ext/
    The necessary library (e.g.. for Linux 32bit, the librxtxSerial.so) should go under JDKDIR/jre/bin/
    If you have a separate JRE directory, do the same copies for the JREDIR directory!</pre> 
             <p>Huh! All the hard work has been done. be cool now <img align="top" alt="Smile | :)" src="https://www.codeproject.com/script/Forums/Images/smiley_smile.gif"> </p> 
             <p>3. Okay now simply open the project from Netbeans IDE or whatever IDE you are using.</p> 
             <p>4. Importing the Required Dependencies. I have imported all the Jars for this project. So by default you have to do nothing with importing. But in your project you have to import few Jar files you can found in <em>&lt;lib&gt;</em> folder. You have to import all the Jars in that folder.</p> 
             <p>5. Okay done! Now take a look at this code.<br> </p>
             <pre lang="java"><span class="code-keyword">package</span> logic;

<span class="code-keyword">import</span> com.harshadura.gsm.smsdura.GsmModem;

<span class="code-keyword">public</span> <span class="code-keyword">class</span> TestSMS {

    <span class="code-keyword">private</span> <span class="code-keyword">static</span> <span class="code-sdkkeyword">String</span> port = <span class="code-string">"</span><span class="code-string">COM3"</span>; <span class="code-comment">//</span><span class="code-comment">Modem Port.</span>

Okay then you can simply notice theirs a Name called port. In front of that there is the port number. Now you know the port number. Insert that into the code.</p>
