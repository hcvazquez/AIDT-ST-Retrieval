       <h2 id="p201372" class="post-title">Object SMS - send and receive SMS from GSM card SIM with JAVA Communication API</h2> 
       <p class="post-info">By FRANCK ANDRIANO on Sunday 27 January 2008, 14:01 - <a href="http://blog.jservlet.com/category/serial-comm">serial comm</a> - <a href="http://blog.jservlet.com/post/2008/01/27/Object-SMS-send-and-receive-SMS-from-GSM-card-SIM-with-JAVA-Communication-API">Permalink</a> </p> 
       <ul class="post-tags"> 
        <li><a href="http://blog.jservlet.com/tag/GSM%20SIM%20card">GSM SIM card</a></li> 
        <li><a href="http://blog.jservlet.com/tag/list%20read%20delete%20sms">list read delete sms</a></li> 
        <li><a href="http://blog.jservlet.com/tag/list%20read%20phonebook">list read phonebook</a></li> 
        <li><a href="http://blog.jservlet.com/tag/serial%20comm%20api">serial comm api</a></li> 
       </ul> 
       <div class="post-content">
        <p>Send and receive SMS (Short Message Service), read PhoneBook from GSM card SIM, handle communication with Serial Port (RS-232 hardware) with Java Communication Api :</p> - Send SMS and Receive SMS / encode and decode binary PDU format
        <br> - List, read, write and delete SMS in specific memory&nbsp; (SIM or Phone)
        <br> - List, read, write and delete PhoneBook in specific memory (SIM or Phone)
        <br> 
        <p>Accept commands 3GPP 27.007 and recommandations 27.005 GSM.</p> 
        <p><strong>Needs :</strong></p> Download and install in your classpath Java Comm API&nbsp; :
        <br> 
        <a hreflang="en" href="http://java.sun.com/products/javacomm/">http://java.sun.com/products/javacomm/</a> (Sun decided to not continue on this way...)
        <br> or same spec (LGPL license) :
        <br> 
        <a hreflang="en" href="http://www.rxtx.org/">http://www.rxtx.org/</a>
        <br> or here :
        <br> 
        <a href="/public/serialcomm/javaSerial.zip">javaSerial</a>
        <br> 
        <br> Install javax.comm.properties file in classpath of your application :
        <br> 
        <br> #
        <br> # Drivers loaded by the Java Communications API standard extension
        <br> # at initialization time
        <br> #
        <br> # Format:
        <br> #&nbsp;&nbsp;&nbsp; Each line must contain ONE driver definition only
        <br> #&nbsp;&nbsp;&nbsp; Each line must be of the form:
        <br> #&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; driver=&lt;ClassName&gt;
        <br> #&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; No spaces or tabs in the line.
        <br> #&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ClassName must implement the interface javax.comm.CommDriver
        <br> #&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; example: driver=Win32Serial
        <br> #&nbsp;&nbsp;&nbsp;
        <br> #
        <br> # The hash(#) character indicates comment till end of line.
        <br> #
        <br> # Windows Serial Driver
        <br> Driver=com.sun.comm.Win32Driver
        <br> # end file
        <br> 
        <br> Install win32com.dll for Win32 only in your JRE (for me \j2sdk1.4.x_xx\jre\bin\)
        <br> See for linux : 
        <a hreflang="en" href="http://wass.homelinux.net/howtos/Comm_How-To.shtml">http://wass.homelinux.net/howtos/Comm_How-To.shtml</a>
        <br> 
        <br> Now we can communicate with SIM via the Serial Port of GSM Phone :
        <br> - Use virtual Serial Port BlueTooth Device of you GSM, use a Bluetooth Adapter to communicate with your GSM.
        <br> (Pairing communication between PC to GSM before use this program...)
        <br> - Or use IRDA with virtual COM port : 
        <a hreflang="en" href="http://www.ircomm2k.de/">http://www.ircomm2k.de/</a>
        <br> - Or use a cable...
        <br> 
        <p><strong><br> Example :</strong></p>

&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;String[] portArray = sms.listPorts();<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;System.out.println("Number of ports detected: " + portArray.length);<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;for (int i = 0; i &lt; portArray.length; i++)<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;{<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;System.out.println("OPEN serial Port : " + portArray[i]);<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;}<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp; smsc = 33689004000<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;// SFR &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp; smsc = 33609001390<br> <br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;//sms.SendMessage("3361xxxxxxx", "33660003000", "Testing Network connection");<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;//sms.SendMessage("xxxxx", "33660003000", "CONTACT"); // call sms+ french service<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;// store sms on SIM memory! (SM)<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;//sms.WriteTextMessage("336xxxxxxxx", "33660003000", "coucou");<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;// store sms on SIM memory! &nbsp;//sms.DeleteSMS(i);<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;}<br> &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;else System.out.println("\nProblem SMS received !!!");<br> &nbsp;&nbsp;
