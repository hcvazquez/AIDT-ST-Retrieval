      <h1><a href="https://blog.appery.io/2014/03/dialing-number-and-sending-sms-via-javascript/">Dialing number and sending SMS via JavaScript</a></h1>
      <p><span>Posted: March 27th, 2014</span><span>Author: <a href="https://blog.appery.io/author/dmitry/" title="Posts by Dmitry Zaytsev" class="author url fn" rel="author">Dmitry Zaytsev</a></span><span class="hidden-xs">Filed under: <a href="https://blog.appery.io/category/javascript/" rel="category tag">JavaScript</a></span><span class="hidden-xs">Tags: <a href="https://blog.appery.io/tag/javascript/" rel="tag">JavaScript</a></span></p> 
     </header>
     <article>
      <p>Being able to launch the native dialer app or SMS app from the app you are building might be a very useful feature. For instance, the app you are building might display a list of restaurants in the area with a phone number. You want to be able to click the number and call the restaurant.&nbsp;This functionality can be easily done with calling the tel: or sms: and passing the needed values. Here is the simple Appery.io app that contains just a two buttons:</p>
      <p><a href="https://blog.appery.io/wp-content/uploads/2014/03/buttons.jpg"><img class="alignnone size-full wp-image-4679" style="border: 1px solid black;" src="https://blog.appery.io/wp-content/uploads/2014/03/buttons.jpg" alt="buttons" width="292" height="102"></a></p>
      <p>Here is the JavaScript code for the Send SMS button:</p>
      <p><code>window.location.href = "sms:+375292771265?body=Hello from Appery.io!";</code></p>
      <p>Such code will open the SMS typing window with predefined text and phone number as following:</p>
      <p><a href="https://blog.appery.io/wp-content/uploads/2014/03/2014-03-12-14.42.341.png"><img class="alignnone size-full wp-image-4681" src="https://blog.appery.io/wp-content/uploads/2014/03/2014-03-12-14.42.341.png" alt="2014-03-12 14.42.34" width="294" height="500"></a></p>
      <p>Second button has the following JavaScript code:</p>
      <p><code>window.location.href = "tel:+375292771265";</code></p>
      <p>Clicking on that button will open the dialer with pre-populated phone number:</p>
      <p><a href="https://blog.appery.io/wp-content/uploads/2014/03/2014-03-12-14.42.15.png"><img class="alignnone size-full wp-image-4682" src="https://blog.appery.io/wp-content/uploads/2014/03/2014-03-12-14.42.15.png" alt="2014-03-12 14.42.15" width="292" height="500"></a></p>
      <p>It is not possible to programmatically call or send SMS without opening the appropriate window due to the security reasons.</p>
      <p>iOS has some nuances when dialer or SMS&nbsp;window should be opened. Try to use such code for iOS platform:</p>
      <p><code>window.open('tel:+375292771265', '_system');</code></p>
      <p>Or for SMS:<br> <code>window.open('sms:+375292771265?body=Hello from Appery.io!', '_system');</code></p>
      <p>Please also note that these protocols (tel: and sms:) might work differently depending on OS/browser versions. On Android, you might get different functionally depending what app is selected for sending SMS messages.</p>
      <p>Instead of using the native SMS app, a number of services such as AT&amp;T and Twilio provide APIs to send SMS messages. Appery.io has <a href="https://appery.io/plugins/" target="_blank">plugins</a> for AT&amp;T and Twilio SMS API.</p>
