      <h1 id="page-title" class="title">The 10 Best JavaScript Libraries for SVG Animation</h1> 
      <div class="region region-content"> 
       <div id="block-system-main" class="block block-system"> 
        <div class="content"> 
         <article> 
          <div class="article-author bar">
           by : 
           <a href="/users/noeticsunil" title="View user profile." class="username">noeticsunil</a> Last Updated : Aug 02, 2017
          </div> 
          <div class="center-content-article"> 
           <div class="field field-name-field-image field-type-image field-label-hidden">
            <div class="field-items">
             <div class="field-item even">
              <img src="http://noeticforce.com/sites/default/files/styles/article-detail/public/articles/javascript-libraries-for-svg-animation.jpg?itok=tTvMJH-h" width="710" height="360" alt="javascript libraries for svg animation">
             </div>
            </div>
           </div>
           <div class="field field-name-body field-type-text-with-summary field-label-hidden">
            <div class="field-items">
             <div class="field-item even">
              <p>SVG literally means Scalable Vector Graphic. SVG graphics are resolution independent which means SVG can scale to any size without losing the quality, even on retina displays. SVG is the format for new age graphics and stores information of the graphic or image in XML format.</p> 
              <p>Usage of SVG for animtions and graphics is so handy and in demand that there are many dedicated JavaScript libraries to easily manipulate SVG graphics and creating animations. Of course, there are many other ways to create animated graphics but SVG brings in real powers in the era or responsive web design.&nbsp;</p> 
              <p>Recommended Course (currently at discount) -&nbsp;<a href="http://bit.ly/2f95E4c" rel="nofollow" target="_blank">Learn SVG Animation - With HTML, CSS &amp; JavaScript</a></p> 
              <p>So, what are the top JavaScript libraries for creating SVG animation? Let us look at few of the JavaScript libraries that are best in the business.</p> 
              <h3><a href="http://julian.com/research/velocity/" target="_blank">Velocity</a></h3> 
              <p>Velocity is one of my personal favourites when it comes to bringing life to graphics. Velocity is built for speed and renders animations real fast. It is real easy to get started with Velocity if you are already familiar with animations in jQuery. It uses the same API that is used for jQuery’s $.animate ().</p> 
              <p>The best thing about velocity library is its full support for SVG element animation which includes SVG specific properties like x, rx, stroke-width etc. Color animation is also supported for fill, stroke and stopColor properties of SVG.</p> 
              <p><img alt="VelociyJS for SVG Animation" src="/sites/default/files/velocityjs.JPG" style="width:100%"></p> 
              <p>You can read more about Velocity at - <a href="http://julian.com/research/velocity/" target="_blank">julian.com</a>.</p> 
              <h3><a href="http://svgjs.com/" target="_blank">SVG.JS</a></h3> 
              <p>Are you looking for a lightweight library? SVG.JS is the one for you, the size of this library is around 11k only gzipped and that is all. You get all the powers to play around with scalable vector graphics a.k.a. SVG with this 11k size packet in your pocket.</p> 
              <p>SVG.JS is readable and uncluttered, supports animations on size, position,, transformations and color for sure. It supports binding events to elements, opacity masks, clipping paths, text paths and dynamic gradients to name a few.</p> 
              <p>SVG.JS is here to stay for long for the simple reason that it is modular which makes it a breeze to create extension and plugins to extend the base functionality. It is well documented too.</p> 
              <p><img alt="SVGJS for SVG Animation" src="/sites/default/files/svgjs.JPG" style="width:100%"></p> 
              <p>You can know more about svg.js at - <a href="http://svgjs.com/" target="_blank">svgjs.com</a>.</p> 
              <h3><a href="http://connoratherton.com/walkway" target="_blank">Walkway</a></h3> 
              <p>Looking for something very basic and specific for path drawing, line and polyline elements, walkway is the library for you. It does just the minimum, what does it well. It is a very small library created by Connor Atherton inspired by polygon ps4 review line animation.&nbsp;</p> 
              <p><img alt="Walkway for SVG Animation" src="/sites/default/files/walkwayjs.JPG" style="width:100%"></p> 
              <p>You can know more about Walkway.js at -&nbsp;<a href="http://connoratherton.com/walkway" target="_blank">connoratherton.com/walkway</a>.</p> 
              <h3><a href="http://raphaeljs.com/" target="_blank">RaphaelJS</a></h3> 
              <p>Why use RaphaelJS? You can use RaphaelJS for creating vector graphics as well as manipulating or animating them. It follows W3C SVG standards which mean that the graphic object crated with this library is a DOM object too and can be manipulated, of course, with JavaScript. Browser support is excellent, you can guess that from the fact that it works even in IE 6.0.</p> 
              <p><img alt="RaphaelJS for SVG Animation" src="/sites/default/files/raphaeljs.JPG" style="width:100%"></p> 
              <p>You can know more about Raphael.js at -&nbsp;<a href="http://raphaeljs.com/" target="_blank">raphaeljs.com</a>.</p> 
              <h3><a href="http://snapsvg.io/" target="_blank">Snap.Svg</a></h3> 
              <p>If you want to leverage the latest of SVG features including masking, clipping, full gradients, groups and patterns. Snap .svg is the library to manipulate all of these. &nbsp;It is for forward looking designers and targets modern web browsers. Snap provides simple JavaScript API for creating animations and making content more engaging.</p> 
              <p>Snap can be used for creating SVG and animating SVG created using graphic design tools like Sketch, Illustrator or Inkscape.</p> 
              <p>Snap is open source, free to use and available under Apache2 license, you can clone it from GitHub. All credit goes to Dmitry Baranovskiy for creating this library, the guy who also authored RaphaelJS.</p> 
              <p><img alt="Snap for SVG Animation" src="/sites/default/files/snapsvg.JPG" style="width:100%"></p> 
              <p>You can know more about Snap.svg at -&nbsp;<a href="http://snapsvg.io/" target="_blank">snapsvg.io</a>.</p> 
              <h3><a href="https://bonsaijs.org/" target="_blank">Bonsai</a></h3> 
              <p>Bonsai is a complete toolset for graphics manipulation and animations. It is one of the most powerful libraries available as of today. It comes with an SVG rendered which essentially means you can create SVG with Bonsai as well as use graphics API for manipulation and animations. The key features of Bonsai that make it real powerful are –</p> 
              <ul>
               <li>Runner and renderer are separated at the core</li> 
               <li>Supports path morphing, assets including images, fonts, audio and video</li> 
               <li>Keyframe&nbsp; and time based animations</li> 
              </ul>
              <p>There is an online editor as well that gives you playground to try your hands with bonsai. It is a real fun and power working with this library.</p> 
              <p><img alt="BonsaiJS for SVG Animation" src="/sites/default/files/bonsaijs.JPG" style="width:100%"></p> 
              <p>You can know more about bonsai.js at -&nbsp;<a href="https://bonsaijs.org/" target="_blank">bonsaijs.org</a>.</p> 
              <h3><a href="http://lazylinepainter.info/" target="_blank">Lazy Line Painter</a></h3> 
              <p>Lazy line painter is a small jQuery plugin for SVG path animation. This is what it does and for sure does with style.&nbsp; Creating path animations is as easy as exporting the line art from illustrator as .svg , use the “SVG to Lazy Converter” and define the path durations in the code. &nbsp;That is it, three step recipe for creating SVG path animation.</p> 
              <p><img alt="Lazy Line Painter for SVG Animation" src="/sites/default/files/lazylinepainterjs.JPG" style="width:100%"></p> 
              <p>You can know more about lazylinepainter.js at -&nbsp;<a href="http://lazylinepainter.info/" target="_blank">lazylinepainter.info</a>.</p> 
              <h3><a href="https://maxwellito.github.io/vivus/" target="_blank">Vivus</a></h3> 
              <p>Draw graphics with elegance by using Vivus library, <em><strong>the big daddy of SVG animation when it comes to line drawing</strong></em>.</p> 
              <p>Vivus library has no external dependencies and can run standalone. Library comes with out of the box animation styles as well as gives you an option to draw your SVG the way you want it to be.&nbsp;</p> 
              <p><img alt="VivusJS for SVG Animation" src="/sites/default/files/vivusjs.JPG" style="width:100%"></p> 
              <p>You can read more about Vivus at - <a href="https://maxwellito.github.io/vivus/" target="_blank">maxwellito.github.io/vivus</a>.</p> 
              <h3><a href="http://kimmobrunfeldt.github.io/progressbar.js/" target="_blank">ProgressBar.js</a></h3> 
              <p>This small JavaScript library is aimed at creating animated SVG progress bars. It comes packed with basic shapes like line, circle and square etc., but that is just the basic. One can create graphical progress bars with illustrator or other vector graphic editors and create animated SVG progress bars using this library.</p> 
              <p><img alt="Progressbarjs for SVG Animation" src="/sites/default/files/progressbarjs.JPG" style="width:100%"></p> 
              <p>You can read more about Progressbar.js at -&nbsp;<a href="http://kimmobrunfeldt.github.io/progressbar.js/" target="_blank">kimmobrunfeldt.github.io/progressbar.js</a>.</p> 
              <h3><a href="https://jonobr1.github.io/two.js/" target="_blank">Two.JS</a></h3> 
              <p>Two.JS is the library for playing around with SVG, Canvas and webgl and focuses completely on two dimensional shapes. This library relies on the scenegraph and operations like rotation, translation, scale etc. can be applied easily on the drawn objects. Two.js comes packed with SVG interpreter that ensures that vector graphics created in commercial software’s like Adobe Illustrator etc. can be imported in Two.js scene and manipulated further.&nbsp;</p> 
              <p><img alt="TwoJS for SVG Animation" src="/sites/default/files/twojs.JPG" style="width:100%"></p> 
              <p>You can read more about Two.js at -&nbsp;<a href="https://jonobr1.github.io/two.js/" target="_blank">jonobr1.github.io/two.js</a>.</p> 
              <h3><strong>Conclusion</strong></h3> 
              <p>SVG use is on the rise and it goes really well with the needs of modern web design, where we need small sized files for performance and scalable graphics that fit devices of all sizes without losing on the quality. Vector graphic animations are great fit for fonts, path drawings, line drawing, dashboards, simple games, logos and maps. JavaScript SVG animation libraries mentioned in this article help do the job with ease, share your experience working with vector graphic animations via comments.</p> 
              <p>Do check out this course&nbsp;<a href="http://bit.ly/2f95E4c" rel="nofollow" target="_blank">Learn SVG Animation - With HTML, CSS &amp; JavaScript</a></p>
