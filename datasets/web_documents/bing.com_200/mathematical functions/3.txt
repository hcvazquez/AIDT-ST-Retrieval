     <h2 class="refsect1">abs</h2> 
     <p class="titleinrefsubsect"><a id="sthref1544"></a>Syntax</p> 
     <img width="183" height="27" src="img/abs.png" alt="Surrounding text describes abs.png." title="Surrounding text describes abs.png.">
     <br> 
     <p class="titleinrefsubsect"><a id="sthref1545"></a>Purpose</p> 
     <p><a id="sthref1546"></a><a id="sthref1547"></a><code>abs</code> returns the absolute value of the input <code>integer</code> argument as an <code>integer</code>.</p> 
     <p>For more information, see <code><a href="http://java.sun.com/javase/6/docs/api/java/lang/Math.html#abs(int)">http://java.sun.com/javase/6/docs/api/java/lang/Math.html#abs(int)</a></code>.</p> 
     <p class="titleinrefsubsect"><a id="sthref1548"></a>Examples</p> 
     <p>Consider the query <code>q66</code> in <a href="#CBHGEJCJ">Example 12-1</a>. Given the data stream <code>SFunc</code> with schema <code>(c1 integer, c2 double, c3 bigint)</code> in <a href="#CBHHJEAA">Example 12-2</a>, the query returns the stream in <a href="#CBHBCGHB">Example 12-3</a>.</p> 
     <div class="example"> 
      <p class="titleinexample"><a id="CBHGEJCJ"></a><a id="CQLLR955"></a>Example 12-1 abs Function Query</p> 
      <pre>
&lt;query id="q66"&gt;&lt;![CDATA[ 
    select abs(c1) from SFunc
]]&gt;&lt;/query&gt;
</pre>
     </div>

     <h2 class="refsect1">IEEERemainder</h2> 
     <p class="titleinrefsubsect"><a id="sthref1624"></a>Syntax</p> 
     <img width="340" height="27" src="img/ieeeremainder.png" alt="Surrounding text describes ieeeremainder.png." title="Surrounding text describes ieeeremainder.png.">
     <br> 
     <p class="titleinrefsubsect"><a id="sthref1625"></a>Purpose</p> 
     <p><a id="sthref1626"></a><a id="sthref1627"></a><code>IEEERemainder</code> computes the remainder operation on two <code>double</code> arguments as prescribed by the IEEE 754 standard and returns the result as a <code>double</code>.</p> 
     <p>This function takes the following arguments:</p> 
     <ul> 
      <li> <p><code>double1</code>: the dividend.</p> </li> 
      <li> <p><code>double2</code>: the divisor.</p> </li> 
     </ul> 
     <p>The remainder value is mathematically equal to <code>f1 - f2  n</code>, where <code>n</code> is the mathematical integer closest to the exact mathematical value of the quotient <code>f1/f2</code>, and if two mathematical integers are equally close to <code>f1/f2</code>, then <code>n</code> is the integer that is even. If the remainder is zero, its sign is the same as the sign of the first argument.</p> 
     <p>For more information, see <code><a href="http://java.sun.com/javase/6/docs/api/java/lang/Math.html#IEEEremainder(double,%20double)">http://java.sun.com/javase/6/docs/api/java/lang/Math.html#IEEEremainder(double,%20double)</a></code>.</p>

     <p class="titleinrefsubsect"><a id="sthref1700"></a>Purpose</p> 
     <p><a id="sthref1701"></a><a id="sthref1702"></a><code>todegrees</code> converts a <code>double</code> angle measured in radians to an approximately equivalent angle measured in degrees as a <code>double</code>.</p>
