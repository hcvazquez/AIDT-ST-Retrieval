      <h4> Download our network clients here. The WiKID server handles Radius, LDAP and TACACS+ through the WiKIDAdmin web interface. This code is for custom web applications that might require two-factor authentication. In addition to logins, the API can be used to create applications to manage users - for self-service or customer service applications. Users can be registered, enabled, disabled deleted etc. </h4> 
      <div itemprop="articleBody"> 
       <p>For complete documentation, see our <a href="/support/api-docs/introduction/">API manual</a>. It has lists of all the transactions.</p> 
       <p>The best option is to see the API in action in the <a href="/support/troubleshooting-faq/how-can-i-test-if-the-server-is-working-correctly/">example.jsp</a> page on your WiKID server itself.&nbsp;</p>

To run the example on a WiKID server, use this command: java -cp wClient-4.2.0.jar:/opt/WiKID/lib/jdom.jar:/opt/WiKID/lib/xstream-1.2.jar:/opt/WiKID/lib/log4j-1.2.15.jar com.wikidsystems.client.Example 127.0.0.1 8388 your_cert.p12 yourpassphrase /opt/WiKID/private/CACertStore changeit</p>
