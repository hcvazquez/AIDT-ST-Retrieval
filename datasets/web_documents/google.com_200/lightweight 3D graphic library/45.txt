      <p>It's often said that data is the new world currency, and the web is the exchange bureau through which it's traded. As consumers, we're positively swimming in data; it's everywhere from labels on food <a href="http://www.creativebloq.com/branding/packaging-design-resources-4131480">packaging design</a> to World Health Organisation reports. As a result, for the designer it's becoming increasingly difficult to present data in a way that stands out from the mass of competing data streams.</p>
      <ul>
       <li><a href="https://creative.adobe.com/plans?sdid=KSODC"><ins>Get Adobe Creative Cloud</ins></a></li>
      </ul>
      <p>One of the best ways to get your message across is to use a visualization to quickly draw attention to the key messages, and by presenting data visually it's also possible to uncover surprising patterns and observations that wouldn't be apparent from looking at stats alone. And nowadays, there's plenty of <a href="http://www.creativebloq.com/graphic-design/free-graphic-design-software-8134039"><ins>free graphic design software</ins></a> to help you do just that.</p>
      <p>As author, data journalist and information designer David McCandless said in his <a href="http://www.ted.com/talks/david_mccandless_the_beauty_of_data_visualization.html"><ins>TED talk</ins></a>: "By visualizing information, we turn it into a landscape that you can explore with your eyes, a sort of information map. And when you're lost in information, an information map is kind of useful."</p>
      <p>There are many different ways of telling a story, but everything starts with an idea. So to help you get started we've rounded up some of the most awesome data visualization tools available on the web.</p>
      <h3 id="01-a-href-https-plot-ly-target-blank-plotly-a">01. <a href="https://plot.ly/" target="_blank">Plotly</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/NkTtfRkvJJj4AxKgfKADuU-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/NkTtfRkvJJj4AxKgfKADuU-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/NkTtfRkvJJj4AxKgfKADuU-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/NkTtfRkvJJj4AxKgfKADuU.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/NkTtfRkvJJj4AxKgfKADuU.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Plotly can produce some very slick graphics
        </div>
       </figcaption>
      </figure>
      <p>Make charts, presentations and dashboards with this flexible software. You can perform your analysis using JavaScript, Puthon, R, Matlab, Jupyter or Excel, and there are several options for importing data. The visualisation library and online chart creation tool allow you to make great-looking graphics.&nbsp;</p>
      <h3 id="02-a-href-http-send-getapp-com-aff-c-offer-id-1939-amp-aff-id-1371-rel-nofollow-target-blank-datahero-a">02. <a href="http://send.getapp.com/aff_c?offer_id=1939&amp;aff_id=1371" rel="nofollow" target="_blank">DataHero</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/TkBAsTPjMnNqTi8b36ryLX-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/TkBAsTPjMnNqTi8b36ryLX-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/TkBAsTPjMnNqTi8b36ryLX-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/TkBAsTPjMnNqTi8b36ryLX.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/TkBAsTPjMnNqTi8b36ryLX.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Anyone can use this to make sense of data from multiple services
        </div>
       </figcaption>
      </figure>
      <p>DataHero enables you to pull together data from cloud services and create charts and dashboards. No technical abilities are required, so this is a great tool for your whole team to use.&nbsp;</p>
      <h3 id="03-a-target-blank-href-http-www-chartjs-org-chart-js-a">03. <a target="_blank" href="http://www.chartjs.org/">Chart.js</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Chart js" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/16e93ec6bfcd9737a03586cfdbaf2aa9-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/16e93ec6bfcd9737a03586cfdbaf2aa9-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/16e93ec6bfcd9737a03586cfdbaf2aa9-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/16e93ec6bfcd9737a03586cfdbaf2aa9.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/16e93ec6bfcd9737a03586cfdbaf2aa9.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Chart.js is perfectly suited to smaller projects
        </div>
       </figcaption>
      </figure>
      <p>Although armed with only six chart types, open source library Chart.js is the perfect data visualization tool for hobbies and small projects. Using HTML 5 canvas elements to render charts, Chart.js creates responsive, flat designs, and is quickly becoming one of the most poplar open-source charting libraries.</p>
      <h3 id="04-a-href-http-send-getapp-com-aff-c-offer-id-635-amp-aff-id-1371-rel-nofollow-target-blank-tableau-a">04. <a href="http://send.getapp.com/aff_c?offer_id=635&amp;aff_id=1371" rel="nofollow" target="_blank">Tableau</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Tableau" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/696756f6604933b9342458175e951587-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/696756f6604933b9342458175e951587-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/696756f6604933b9342458175e951587-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/696756f6604933b9342458175e951587.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/696756f6604933b9342458175e951587.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Create and share data in real time with Tableau
        </div>
       </figcaption>
      </figure>
      <p>Packed with graphs, charts, maps and more Tableau public is a popular data visualization tool that's also completely free. Users can easily drag and drop data into the system and watch it update in real-time, plus they can collaborate with other team members for quick project turnaround.</p>
      <h3 id="05-a-target-blank-href-http-raw-densitydesign-org-raw-a">05. <a target="_blank" href="http://raw.densitydesign.org/">Raw</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Raw" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/3dd17bc36c22b29739d3b8667a3920f7-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/3dd17bc36c22b29739d3b8667a3920f7-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/3dd17bc36c22b29739d3b8667a3920f7-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/3dd17bc36c22b29739d3b8667a3920f7.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/3dd17bc36c22b29739d3b8667a3920f7.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Raw is an open web app with a simple interface
        </div>
       </figcaption>
      </figure>
      <p>Open, customizable, and free to download and modify, Raw lets users create vector-based data visualizations. Data can be safely uploaded from apps to computers, plus it can be exported as an SVG or PNG and embedded in your webpage.</p>
      <h3 id="06-a-target-blank-href-http-dygraphs-com-dygraphs-a">06. <a target="_blank" href="http://dygraphs.com/">Dygraphs</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Dygraphs" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/a2d654410ff9a786470cf4ce226b67a6-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/a2d654410ff9a786470cf4ce226b67a6-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/a2d654410ff9a786470cf4ce226b67a6-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/a2d654410ff9a786470cf4ce226b67a6.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/a2d654410ff9a786470cf4ce226b67a6.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Help visitors explore dense data sets with JavaScript library Dygraphs
        </div>
       </figcaption>
      </figure>
      <p>Dygraphs is a fast, flexible open source JavaScript charting library that allows users to explore and interpret dense data sets. It's highly customizable, works in all major browsers, and you can even pinch to zoom on mobile and tablet devices.</p>
      <h3 id="07-a-target-blank-href-http-www-zingchart-com-zingchart-a">07. <a target="_blank" href="http://www.zingchart.com">ZingChart</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization ZingChart" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/e6cfb6d04e20685aaad01a673e168238-320-80.png" data-srcset="http://cdn.mos.cms.futurecdn.net/e6cfb6d04e20685aaad01a673e168238-320-80.png 320w, http://cdn.mos.cms.futurecdn.net/e6cfb6d04e20685aaad01a673e168238-650-80.png 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/e6cfb6d04e20685aaad01a673e168238.png" data-pin-media="http://cdn.mos.cms.futurecdn.net/e6cfb6d04e20685aaad01a673e168238.png"></p>
       <figcaption>
        <div class="caption-text">
         ZingChart lets you create HTML5 Canvas charts and more
        </div>
       </figcaption>
      </figure>
      <p>ZingChart is a JavaScript charting library and feature-rich API set that lets you build interactive Flash or HTML5 charts. It offer over 100 chart types to fit your data.</p>
      <h3 id="08-a-target-blank-href-http-www-instantatlas-com-instantatlas-a">08. <a target="_blank" href="http://www.instantatlas.com">InstantAtlas</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization InstantAtlas" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/9bbc0ea19ff5860f7e24978904f9b890-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/9bbc0ea19ff5860f7e24978904f9b890-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/9bbc0ea19ff5860f7e24978904f9b890-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/9bbc0ea19ff5860f7e24978904f9b890.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/9bbc0ea19ff5860f7e24978904f9b890.jpg"></p>
       <figcaption>
        <div class="caption-text">
         InstantAtlas enables you to create highly engaging visualisations around map data
        </div>
       </figcaption>
      </figure>
      <p>If you're looking for a data viz tool with mapping, InstantAtlas is worth checking out. This tool enables you to create highly-interactive dynamic and profile reports that combine statistics and map data to create engaging data visualizations.</p>
      <h3 id="09-a-target-blank-href-http-www-simile-widgets-org-timeline-timeline-a">09. <a target="_blank" href="http://www.simile-widgets.org/timeline/">Timeline</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Timeline" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/591b56c912a5768f6ba695de47f96262-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/591b56c912a5768f6ba695de47f96262-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/591b56c912a5768f6ba695de47f96262-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/591b56c912a5768f6ba695de47f96262.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/591b56c912a5768f6ba695de47f96262.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Timeline creates beautiful interactive visualizations
        </div>
       </figcaption>
      </figure>
      <p>Timeline is a fantastic widget which renders a beautiful interactive timeline that responds to the user's mouse, making it easy to create advanced timelines that convey a lot of information in a compressed space.</p>
      <p>Each element can be clicked to reveal more in-depth information, making this a great way to give a big-picture view while still providing full detail.</p>
      <h3 id="10-a-target-blank-href-http-www-simile-widgets-org-exhibit-exhibit-a">10. <a target="_blank" href="http://www.simile-widgets.org/exhibit/">Exhibit</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Exhibit" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/1dfb3adeede26fb6070f701405a986cf-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/1dfb3adeede26fb6070f701405a986cf-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/1dfb3adeede26fb6070f701405a986cf-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/1dfb3adeede26fb6070f701405a986cf.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/1dfb3adeede26fb6070f701405a986cf.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Exhibit makes data visualization a doddle
        </div>
       </figcaption>
      </figure>
      <p>Developed by MIT, and fully open-source, Exhibit makes it easy to create interactive maps, and other data-based visualizations that are orientated towards teaching or static/historical based data sets, such as flags pinned to countries, or birth-places of famous people.</p>
      <h3 id="11-a-target-blank-href-http-modestmaps-com-modest-maps-a">11. <a target="_blank" href="http://modestmaps.com">Modest Maps</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Modest Maps" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/37cfe19d4a30132798a798b004e459b9-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/37cfe19d4a30132798a798b004e459b9-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/37cfe19d4a30132798a798b004e459b9-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/37cfe19d4a30132798a798b004e459b9.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/37cfe19d4a30132798a798b004e459b9.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Integrate and develop interactive maps within your site with this cool tool
        </div>
       </figcaption>
      </figure>
      <p>Modest Maps is a lightweight, simple mapping tool for web designers that makes it easy to integrate and develop interactive maps within your site, using them as a data visualization tool.</p>
      <p>The API is easy to get to grips with, and offers a useful number of hooks for adding your own interaction code, making it a good choice for designers looking to fully customise their user's experience to match their website or web app. The basic library can also be extended with additional plugins, adding to its core functionality and offering some very useful data integration options.</p>
      <h3 id="12-a-target-blank-href-http-leafletjs-com-leaflet-a">12. <a target="_blank" href="http://leafletjs.com">Leaflet</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Leaflet" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/0908fff1b52afb970af21e33df75e851-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/0908fff1b52afb970af21e33df75e851-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/0908fff1b52afb970af21e33df75e851-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/0908fff1b52afb970af21e33df75e851.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/0908fff1b52afb970af21e33df75e851.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Use OpenStreetMap data and integrate data visualisation in an HTML5/CSS3 wrapper
        </div>
       </figcaption>
      </figure>
      <p>Another mapping tool, Leaflet makes it easy to use OpenStreetMap data and integrate fully interactive data visualisation in an HTML5/CSS3 wrapper.</p>
      <p>The core library itself is very small, but there are a wide range of plugins available that extend the functionality with specialist functionality such as animated markers, masks and heatmaps. Perfect for any project where you need to show data overlaid on a geographical projection (including <a href="http://en.wikipedia.org/wiki/Map_projection" target="_blank">unusual projections</a>!).</p>
      <h3 id="13-a-target-blank-href-http-www-wolframalpha-com-wolframalpha-a">13. <a target="_blank" href="http://www.wolframalpha.com">WolframAlpha</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Wolfram Alpha" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/1e00c028b66f5d659e6d41002c0f9c0a-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/1e00c028b66f5d659e6d41002c0f9c0a-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/1e00c028b66f5d659e6d41002c0f9c0a-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/1e00c028b66f5d659e6d41002c0f9c0a.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/1e00c028b66f5d659e6d41002c0f9c0a.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Wolfram Alpha is excellent at creating charts
        </div>
       </figcaption>
      </figure>
      <p>Billed as a "computational knowledge engine", the Google rival WolframAlpha is really good at intelligently displaying charts in response to data queries without the need for any configuration. If you're using publically available data, this offers a simple widget builder to make it really simple to get visualizations on your site.</p>
      <h3 id="14-a-target-blank-href-http-create-visual-ly-visual-ly-a">14. <a target="_blank" href="http://create.visual.ly/">Visual.ly</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Visual ly" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/690ec4b34b365c42bd806a632bffbc5c-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/690ec4b34b365c42bd806a632bffbc5c-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/690ec4b34b365c42bd806a632bffbc5c-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/690ec4b34b365c42bd806a632bffbc5c.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/690ec4b34b365c42bd806a632bffbc5c.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Visual.ly makes data visualisation as simple as can be
        </div>
       </figcaption>
      </figure>
      <p>Visual.ly is a combined gallery and infographic generation tool. It offers a simple toolset for building stunning data representations, as well as a platform to share your creations. This goes beyond pure data visualisation, but if you want to create something that stands on its own, it's a fantastic resource and an info-junkie's dream come true!</p>
      <h3 id="15-a-target-blank-href-https-www-visualizefree-com-visualize-free-a">15. <a target="_blank" href="https://www.visualizefree.com/">Visualize Free</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Visualize Free" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/c4a75e2dcc5f9f7b18a259606d9b35e5-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/c4a75e2dcc5f9f7b18a259606d9b35e5-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/c4a75e2dcc5f9f7b18a259606d9b35e5-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/c4a75e2dcc5f9f7b18a259606d9b35e5.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/c4a75e2dcc5f9f7b18a259606d9b35e5.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Make visualizations for free!
        </div>
       </figcaption>
      </figure>
      <p>Visualize Free is a hosted tool that allows you to use publicly available datasets, or upload your own, and build interactive visualizations to illustrate the data. The visualizations go well beyond simple charts, and the service is completely free plus while development work requires Flash, output can be done through HTML5.</p>
      <h3 id="16-a-target-blank-href-http-www-betterworldflux-com-better-world-flux-a">16. <a target="_blank" href="http://www.betterworldflux.com/">Better World Flux</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization Better World Flux" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/d22e7f8182a26937e3ff28b06f6691b1-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/d22e7f8182a26937e3ff28b06f6691b1-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/d22e7f8182a26937e3ff28b06f6691b1-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/d22e7f8182a26937e3ff28b06f6691b1.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/d22e7f8182a26937e3ff28b06f6691b1.jpg"></p>
       <figcaption>
        <div class="caption-text">
         Making the ugly beautiful - that's Better World Flux
        </div>
       </figcaption>
      </figure>
      <p>Orientated towards making positive change to the world, Better World Flux has some lovely visualizations of some pretty depressing data. It would be very useful, for example, if you were writing an article about world poverty, child undernourishment or access to clean water. This tool doesn't allow you to upload your own data, but does offer a rich interactive output.</p>
      <h3 id="17-a-target-blank-href-http-www-fusioncharts-com-fusioncharts-a">17. <a target="_blank" href="http://www.fusioncharts.com/">FusionCharts</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization FusionCharts Suite XT" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/a3d9f52b4ca4e6e643179b485e9c0d3b-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/a3d9f52b4ca4e6e643179b485e9c0d3b-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/a3d9f52b4ca4e6e643179b485e9c0d3b-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/a3d9f52b4ca4e6e643179b485e9c0d3b.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/a3d9f52b4ca4e6e643179b485e9c0d3b.jpg"></p>
       <figcaption>
        <div class="caption-text">
         A comprehensive JavaScript/HTML5 charting solution for your data visualization needs
        </div>
       </figcaption>
      </figure>
      <p>FusionCharts Suite XT brings you 90+ charts and gauges, 965 data-driven maps, and ready-made business dashboards and demos. FusionCharts comes with extensive JavaScript API that makes it easy to integrate it with any AJAX application or JavaScript framework. These charts, maps and dashboards are highly interactive, customizable and work across all devices and platforms. They also have <a href="http://www.fusioncharts.com/javascript-charting-comparison/" target="_blank">a comparison of the top JavaScript charting libraries</a> which is worth checking out.</p>
      <h3 id="18-a-target-blank-href-http-www-jqplot-com-jqplot-a">18. <a target="_blank" href="http://www.jqplot.com">jqPlot</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization jQPlot" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/884d94ebfca3521c73f7310870c6c190-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/884d94ebfca3521c73f7310870c6c190-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/884d94ebfca3521c73f7310870c6c190-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/884d94ebfca3521c73f7310870c6c190.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/884d94ebfca3521c73f7310870c6c190.jpg"></p>
       <figcaption>
        <div class="caption-text">
         jqPlot is a nice solution for line and point charts
        </div>
       </figcaption>
      </figure>
      <p>Another jQuery plugin, jqPlot is a nice solution for line and point charts. It comes with a few nice additional features such as the ability to generate trend lines automatically, and interactive points that can be adjusted by the website visitor, updating the dataset accordingly.</p>
      <h3 id="19-a-target-blank-href-http-d3js-org-d3-js-a">19. <a target="_blank" href="http://d3js.org/">D3.js</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization D3 js" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/deaececf0675db491f64e5f692de1c3c-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/deaececf0675db491f64e5f692de1c3c-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/deaececf0675db491f64e5f692de1c3c-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/deaececf0675db491f64e5f692de1c3c.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/deaececf0675db491f64e5f692de1c3c.jpg"></p>
       <figcaption>
        <div class="caption-text">
         You can render some amazing diagrams with D3
        </div>
       </figcaption>
      </figure>
      <p>D3.js is a JavaScript library that uses HTML, SVG, and CSS to render some amazing diagrams and charts from a variety of data sources. This library, more than most, is capable of some seriously advanced visualizations with complex data sets. It's open source, and uses web standards so is very accessible. It also includes some fantastic user interaction support.</p>
      <h3 id="20-a-target-blank-href-http-thejit-org-javascript-infovis-toolkit-a">20. <a target="_blank" href="http://thejit.org/">JavaScript InfoVis Toolkit</a></h3>
      <figure>
       <p class="bordeaux-image-check"><img src="http://www.creativebloq.com/media/img/missing-image.svg" alt="Data visualization JavaScript InfoVis Toolkit" class=" lazy-image lazy-image-loading lazyload optional-image" onerror="this.parentNode.replaceChild(window.missingImage(),this)" sizes="auto" data-normal="http://www.creativebloq.com/media/img/missing-image.svg" data-src="http://cdn.mos.cms.futurecdn.net/8146d72f02305186d7630d5ee5388c35-320-80.jpg" data-srcset="http://cdn.mos.cms.futurecdn.net/8146d72f02305186d7630d5ee5388c35-320-80.jpg 320w, http://cdn.mos.cms.futurecdn.net/8146d72f02305186d7630d5ee5388c35-650-80.jpg 650w" data-sizes="auto" data-original-mos="http://cdn.mos.cms.futurecdn.net/8146d72f02305186d7630d5ee5388c35.jpg" data-pin-media="http://cdn.mos.cms.futurecdn.net/8146d72f02305186d7630d5ee5388c35.jpg"></p>
       <figcaption>
        <div class="caption-text">
         JavaScript InfoVis Toolkit includes a handy modular structure
        </div>
       </figcaption>
      </figure>
      <p>A fantastic library written by Nicolas Belmonte, the JavaScript InfoVis Toolkit includes a modular structure, allowing you to only force visitors to download what's absolutely necessary to display your chosen data visualizations. This library has a number of unique styles and swish animation effects, and is free to use (although donations are encouraged).</p>
