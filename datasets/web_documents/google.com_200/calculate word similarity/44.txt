      <h1>fuzzywuzzy 0.15.1</h1> 
      <div id="download-button"> 
       <a class="button green" style="float:right;" href="#downloads">Downloads ?</a> 
      </div> 
      <p style="font-style: italic">Fuzzy string matching in python</p> 
      <a href="https://travis-ci.org/seatgeek/fuzzywuzzy" rel="nofollow"><img src="https://travis-ci.org/seatgeek/fuzzywuzzy.svg?branch=master"></a> 
      <div id="fuzzywuzzy"> 
       <h2>FuzzyWuzzy</h2> 
       <p>Fuzzy string matching like a boss. It uses <a href="https://en.wikipedia.org/wiki/Levenshtein_distance" rel="nofollow">Levenshtein Distance</a> to calculate the differences between sequences in a simple-to-use package.</p> 
      </div> 
      <div id="requirements"> 
       <h2>Requirements</h2> 
       <ul> 
        <li>Python 2.4 or higher</li> 
        <li>difflib</li> 
        <li><a href="https://github.com/ztane/python-Levenshtein/" rel="nofollow">python-Levenshtein</a> (optional, provides a 4-10x speedup in String Matching, though may result in <a href="https://github.com/seatgeek/fuzzywuzzy/issues/128" rel="nofollow">differing results for certain cases</a>)</li> 
       </ul> 
      </div> 
      <div id="installation"> 
       <h2>Installation</h2> 
       <p>Using PIP via PyPI</p> 
       <pre>pip install fuzzywuzzy
</pre> 
       <p>or the following to install <cite>python-Levenshtein</cite> too</p> 
       <pre>pip install fuzzywuzzy<span class="o">[</span>speedup<span class="o">]</span>
</pre> 
       <p>Using PIP via Github</p> 
       <pre>pip install git+git://github.com/seatgeek/fuzzywuzzy.git@0.15.1#egg<span class="o">=</span>fuzzywuzzy
</pre> 
       <p>Adding to your <tt>requirements.txt</tt> file (run <tt>pip install <span class="pre">-r</span> requirements.txt</tt> afterwards)</p> 
       <pre>git+ssh://git@github.com/seatgeek/fuzzywuzzy.git@0.15.1#egg<span class="o">=</span>fuzzywuzzy
</pre> 
       <p>Manually via GIT</p> 
       <pre>git clone git://github.com/seatgeek/fuzzywuzzy.git fuzzywuzzy
<span class="nb">cd</span> fuzzywuzzy
python setup.py install
</pre> 
      </div> 
      <div id="usage"> 
       <h2>Usage</h2> 
       <pre><span class="o">&gt;&gt;&gt;</span> <span class="kn">from</span> <span class="nn">fuzzywuzzy</span> <span class="kn">import</span> <span class="n">fuzz</span>
<span class="o">&gt;&gt;&gt;</span> <span class="kn">from</span> <span class="nn">fuzzywuzzy</span> <span class="kn">import</span> <span class="n">process</span>
</pre> 
       <div id="simple-ratio"> 
        <h3>Simple Ratio</h3> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">ratio</span><span class="p">(</span><span class="s2">"this is a test"</span><span class="p">,</span> <span class="s2">"this is a test!"</span><span class="p">)</span>
    <span class="mi">97</span>
</pre> 
       </div> 
       <div id="partial-ratio"> 
        <h3>Partial Ratio</h3> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">partial_ratio</span><span class="p">(</span><span class="s2">"this is a test"</span><span class="p">,</span> <span class="s2">"this is a test!"</span><span class="p">)</span>
    <span class="mi">100</span>
</pre> 
       </div> 
       <div id="token-sort-ratio"> 
        <h3>Token Sort Ratio</h3> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">ratio</span><span class="p">(</span><span class="s2">"fuzzy wuzzy was a bear"</span><span class="p">,</span> <span class="s2">"wuzzy fuzzy was a bear"</span><span class="p">)</span>
    <span class="mi">91</span>
<span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">token_sort_ratio</span><span class="p">(</span><span class="s2">"fuzzy wuzzy was a bear"</span><span class="p">,</span> <span class="s2">"wuzzy fuzzy was a bear"</span><span class="p">)</span>
    <span class="mi">100</span>
</pre> 
       </div> 
       <div id="token-set-ratio"> 
        <h3>Token Set Ratio</h3> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">token_sort_ratio</span><span class="p">(</span><span class="s2">"fuzzy was a bear"</span><span class="p">,</span> <span class="s2">"fuzzy fuzzy was a bear"</span><span class="p">)</span>
    <span class="mi">84</span>
<span class="o">&gt;&gt;&gt;</span> <span class="n">fuzz</span><span class="o">.</span><span class="n">token_set_ratio</span><span class="p">(</span><span class="s2">"fuzzy was a bear"</span><span class="p">,</span> <span class="s2">"fuzzy fuzzy was a bear"</span><span class="p">)</span>
    <span class="mi">100</span>
</pre> 
       </div> 
       <div id="process"> 
        <h3>Process</h3> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">choices</span> <span class="o">=</span> <span class="p">[</span><span class="s2">"Atlanta Falcons"</span><span class="p">,</span> <span class="s2">"New York Jets"</span><span class="p">,</span> <span class="s2">"New York Giants"</span><span class="p">,</span> <span class="s2">"Dallas Cowboys"</span><span class="p">]</span>
<span class="o">&gt;&gt;&gt;</span> <span class="n">process</span><span class="o">.</span><span class="n">extract</span><span class="p">(</span><span class="s2">"new york jets"</span><span class="p">,</span> <span class="n">choices</span><span class="p">,</span> <span class="n">limit</span><span class="o">=</span><span class="mi">2</span><span class="p">)</span>
    <span class="p">[(</span><span class="s1">'New York Jets'</span><span class="p">,</span> <span class="mi">100</span><span class="p">),</span> <span class="p">(</span><span class="s1">'New York Giants'</span><span class="p">,</span> <span class="mi">78</span><span class="p">)]</span>
<span class="o">&gt;&gt;&gt;</span> <span class="n">process</span><span class="o">.</span><span class="n">extractOne</span><span class="p">(</span><span class="s2">"cowboys"</span><span class="p">,</span> <span class="n">choices</span><span class="p">)</span>
    <span class="p">(</span><span class="s2">"Dallas Cowboys"</span><span class="p">,</span> <span class="mi">90</span><span class="p">)</span>
</pre> 
        <p>You can also pass additional parameters to <tt>extractOne</tt> method to make it use a specific scorer. A typical use case is to match file paths:</p> 
        <pre><span class="o">&gt;&gt;&gt;</span> <span class="n">process</span><span class="o">.</span><span class="n">extractOne</span><span class="p">(</span><span class="s2">"System of a down - Hypnotize - Heroin"</span><span class="p">,</span> <span class="n">songs</span><span class="p">)</span>
    <span class="p">(</span><span class="s1">'/music/library/good/System of a Down/2005 - Hypnotize/01 - Attack.mp3'</span><span class="p">,</span> <span class="mi">86</span><span class="p">)</span>
<span class="o">&gt;&gt;&gt;</span> <span class="n">process</span><span class="o">.</span><span class="n">extractOne</span><span class="p">(</span><span class="s2">"System of a down - Hypnotize - Heroin"</span><span class="p">,</span> <span class="n">songs</span><span class="p">,</span> <span class="n">scorer</span><span class="o">=</span><span class="n">fuzz</span><span class="o">.</span><span class="n">token_sort_ratio</span><span class="p">)</span>
    <span class="p">(</span><span class="s2">"/music/library/good/System of a Down/2005 - Hypnotize/10 - She's Like Heroin.mp3"</span><span class="p">,</span> <span class="mi">61</span><span class="p">)</span>
</pre>
