               <h3>50+ JavaScript &amp; HTML5 Frameworks and Related Tools</h3> 
               <p style="font-size: 10px;">Posted on Mar 08, 2012</p> 
               <p>This is the <em>first draft</em> of a document I am using and handing out along with my upcoming presentations at <a href="http://360flex.com">360Flex</a> and <a href="http://cfobjective.com">cf.Objective()</a>. It is a list of every JavaScript (and, to a lesser degree HTML5) framework I have come across over the past 6 months. I have organized them according to categories, given the URL, license details, source availability and a brief summary.</p> 
               <p>My goal in posting this now, other than hopefully people find it useful, is to ask for feedback from the community. What frameworks am I missing? Are there any mistakes? Would you choose different categories? What additional information would be useful?</p> 
               <p>Some ideas I have for this is to put it on GitHub so that anyone can submit an edit. Perhaps even convert it to some data format like JSON that can be used to create many means of searching or building off of it. Any thoughts on those ideas? For now please contribute via the comments.</p> 
               <p><em>Update Mar 12: Added Pure, CoreChain, Dojo Mobile, ClojureScript, GWT, DHTMLX, DHTMLX Touch, Qooxdoo</em></p> 
               <h4 dir="ltr">Full-stack Frameworks/Toolkits</h4> 
               <p><a href="http://jquery.com/">jQuery</a><br> License: MIT and GPL licenses<br> GitHub: Yes<br> Summary: A framework that simplifies DOM traversing, event handling, animating and Ajax interactions for rapid web development. jQueryUI library includes UI components.</p> 
               <p><a href="http://dojotoolkit.org/">Dojo Toolkit</a><br> License: BSD license or the Academic Free License version 2.1<br> GitHub: No (source hosted on SVN)<br> Summary: A full stack framework and component set for building web-based and mobile applications that includes DOM manipulation, Ajax, animations, effects, and widgets.</p> 
               <p><a href="http://www.sencha.com/products/extjs/">ExtJS</a> <br> License: GNU GPL license v3 and paid Commercial licenses available<br> GitHub: No<br> Summary: A full stack framework and extensive UI component set for building web-based rich internet applications.</p> 
               <p><a href="http://twitter.github.com/bootstrap/">Twitter Bootstrap</a> <br> License: Apache v2<br> Github: Yes<br> Summary: A &nbsp;toolkit for kickstarting HTML, CSS and JavaScript for websites and apps including base styles for typography, forms, buttons, tables, grids, navigation, alerts, and more..</p> 
               <p><a href="http://yuilibrary.com/">YUI</a> <br> License: BSD<br> GitHub: Yes<br> Summary: A lightweight and modular JavaScript and CSS framework for building interactive web applications.</p> 
               <p><a href="http://mootools.net/">MooTools</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A modular, object-oriented framework that re-creates pseudo-classical inheritance with Classes that allow for extensible and reusable behaviors.</p> 
               <p><a href="http://www.kendoui.com/">Kendo</a> <br> License: GPLv3 and paid Commercial licenses available<br> GitHub: No<br> Dependencies: jQuery<br> Summary: A full stack framework for building JavaScript-based web and mobile applicatings including support for templates, themes and UI widgets.</p> 
               <p><a href="http://prototypejs.org/">Prototype</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A framework including a toolkit for class-driven development and an Ajax library.</p> 
               <p><a href="http://sproutcore.com/">SproutCore</a> <br> License: MIT<br> Github: Yes<br> Summary: An MVC application framework for data-centric applications that includes data bindings.</p> 
               <p><a href="http://emberjs.com/">Ember.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: Framework that tries to eliminate boilerplate and provide a standard architecture. Includes bindings, computed properties and auto-updating templates. Formerly SproutCore 2.0.</p> 
               <p><a href="https://github.com/fohr/blossom">Blossom</a> <br> License: GPL v3<br> Github: Yes<br> Summary: Another SproutCore fork, it re-architects the view layer to be HTML/CSS-independent while providing a high-performance animation subsystem that leverages hardware acceleration. </p> 
               <p><a href="http://code.google.com/closure/">Google Closure</a> <br> License: Apache v2<br> Github: No (Source available on Google Code)<br> Summary: Developed by Google engineers, it includes features like a JavaScript optimizer, a library of UI widgets and controls, a templating engine and extensions to CSS.</p> 
               <p><a href="http://cappuccino.org/">Cappuccino</a> <br> License: LGPL<br> Github: Yes<br> Summary: Created with Objective-J, which is modelled after Objective-C but on JavaScript, it abstracts away HTML, CSS and the DOM and implements APIs from GNUstep and Cocoa.</p> 
               <p><a href="http://towerjs.org/">Tower.js</a> <br> License: MIT<br> Github: Yes<br> Summary: A full stack web framework for Node.js and the Browser built on top of Node's Connect and Express and modeled after Ruby on Rails.</p> 
               <p>&nbsp;</p> 
               <p><a href="http://dhtmlx.com/index.shtml">DHTMLX</a><br>License: Standard Edition is GNU GPL v2, Commercial &amp; Enterprise versions available.<br>Github: No<br>Summary: Includes collections of user interface components, allowing you to create robust web interfaces, manage client-server communications with Ajax, and implement server-side logic.</p> 
               <p>&nbsp;</p> 
               <p><a href="http://www.corechain.com/index.html">CoreChain</a><br>License: MIT and GPL<br>Github: No<br>Summary:&nbsp;An architectural framework and UI toolkit. (Documentation in Korean)</p> 
               <p>&nbsp;</p> 
               <p><a href="http://qooxdoo.org/about">qooxdoo</a><br>License: &nbsp;GNU LGPL and the Eclipse Public License (EPL)<br>Github: Yes<br>Summary: A JavaScript framework for creating apps for a wide range of platforms (mobile, website, desktop, server) that are object-oriented and utilize a comprehensive tool chain.</p> 
               <p>&nbsp;</p> 
               <h4>Architectural Frameworks</h4> 
               <p><a href="http://knockoutjs.com/">Knockout.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A lightweight framework for building JavaScript UI’s with declarative bindings using the Model-View-View Model (MVVM) pattern.</p> 
               <p><a href="http://documentcloud.github.com/backbone/">Backbone.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: An architectural framework provides things like models key-value binding and custom events, views with declarative event handling and a RESTful JSON interface. </p> 
               <p><a href="http://angularjs.org/">AngularJS</a> <br> License: MIT<br> GitHub: Yes<br> Summary: An MVC and dependency injection framework that includes support for declarative UI templates and two-way binding.</p> 
               <p><a href="http://spinejs.com/">Spine</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A lightweight MVC framework built with, but not requiring, CoffeeScript that includes features like asynchronous server communication, a full model layer and ORM.</p> 
               <p><a href="http://agilityjs.com/">Agility.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: An MVC library that works with a single object type that contains a full model-view-controller stack.</p> 
               <p><a href="http://sammyjs.org/">Sammy.js</a> <br> License: MIT<br> MIT: Yes<br> Summary: A very lightweight framework for applications made up primarily of routes and events that is designed to be modular, with adapters and plugins.</p> 
               <p><a href="http://javascriptmvc.com/">JavaScriptMVC</a> <br> License: MIT<br> Github: Yes<br> Dependencies: jQuery<br> Summary: Comprised of 4 sub projects including the DocumentJS documentation engine, FuncUnit testing framework, jQueryMX MVC extensions and StealJS code manager.</p> 
               <p><a href="http://batmanjs.org/">Batman.js</a> <br> License: MIT<br> Github: Yes<br> Summary: A framework for building applications with CoffeeScript or JavaScript including a system of view bindings and observable properties</p> 
               <p><a href="http://somajs.github.com/somajs/">Soma.js</a> <br> License : Mozilla Public License (MPL) Version 1.1<br> Github: Yes<br> Summary: MVC framework focused on loosely-coupled apps that implements the Observer and Command patterns via the native JavaScript event system. Includes a MooTools version.</p> 
               <h4 dir="ltr">Supplementary Libaries</h4> 
               <p> <a href="http://amplifyjs.com/">AmplifyJS</a> <br> License: MIT or GPLv2<br> GitHub: Yes<br> Dependencies: jQuery<br> Summary: A set of components to simplify data handling and AJAX request management by providing a unified API for various data sources.</p> 
               <p><a href="http://documentcloud.github.com/underscore/">Underscore</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A library that provides functions and helpers for functional programming support including function binding, JavaScript templates and more.</p> 
               <p><a href="https://github.com/janl/mustache.js">Mustache.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: An implementation of the Mustache logic-less template system in JavaScript.</p> 
               <p><a href="http://handlebarsjs.com/">Handlebars.js</a> <br> License: MIT<br> Github: Yes<br> Summary: A library that works in conjunction with Mustache.js to let you build semantic templates.</p> 
               <p><a href="http://requirejs.org/">RequireJS</a> <br> License: BSD and MIT<br> GitHub: Yes<br> Summary: A JavaScript file and module loader optimized for the browser but works with server-side solutions like Node and Rhino.</p> 
               <p><a href="http://microjs.com/">Microjs </a> <br> License: depends on micro-framework or micro-library<br> GitHub: Yes<br> Summary: A large collection of micro-frameworks each with a very limited and specific intent.</p> 
               <p><a href="http://ukijs.org/">UKI</a> <br> License: MIT<br> Github: Yes<br> Summary: A simple JavaScript user interface toolkit for desktop-like web applications that comes with a rich view-component library.</p> 
               <p><a href="http://beebole.com/pure/">Pure</a><br>License: MIT<br>Github: Yes<br>Summary:&nbsp;A templating tool to generate HTML from JSON data that can work standalone or with dojo, DomAssistant, jQuery, Mootools, Prototype, Sizzle and Sly.</p> 
               <h4 dir="ltr">Graphics/Animation</h4> 
               <p> <a href="http://easeljs.com/">EaselJS</a> <br> License: MIT<br> GitHub: Yes<br> Summary: A library to simplify working with the html5 canvas element including a full, hierarchical display list and a core interaction model.</p> 
               <p><a href="http://paperjs.org/">Paper.js</a> <br> License: MIT<br> GitHub: Yes<br> Summary: An open source vector graphics scripting framework that runs on top of the HTML5 Canvas.</p> 
               <p><a href="http://raphaeljs.com/index.html">Raphael</a> <br> License: MIT<br> Github: Yes<br> Summary: A small JavaScript library to simplify working with vector graphics via SVG and VML.</p> 
               <h4 dir="ltr">Mobile Specific</h4> 
               <p> <a href="http://jquerymobile.com/">jQuery Mobile</a> <br> License: MIT and GPLv2<br> GitHub: Yes<br> Dependencies: jQuery<br> Summary: A HTML5-base UI system for all popular mobile device platforms, built on a jQuery and jQuery UI foundation.</p> 
               <p><a href="http://www.sencha.com/products/touch/">Sencha Touch</a> <br> License: GNU GPL license v3 and free Commercial licenses available<br> GitHub: No<br> Summary: A mobile web app development framework built with HTML5, CSS3, and Javascript and including components like audio and video, a localStorage proxy and styling layer.</p> 
               <p><a href="http://zeptojs.com/">Zepto </a> <br> License: MIT<br> GitHub: Yes<br> Summary: Primarily focused on mobile, this is &nbsp;a minimalist JavaScript framework with a jQuery-compatible syntax.</p> 
               <p><a href="http://the-m-project.org/">The M Project</a> <br> License: MIT &amp; GPL<br> Github: Yes<br> Dependencies: jQuery, jQuery Mobile<br> Summary: An MVC framework for mobile app development including features like offline support, local storage, positioning, a touch-optimized UI, i18n support and native packing via Espresso.</p> 
               <p><a href="http://www.jqmobi.com/">jQ.Mobi</a> <br> License: MIT<br> Github: Yes<br> Summary: The jQuery framework rewritten from the ground up to be optimized for mobile browsers on iOS and Android.</p> 
               <p><a href="http://spinejs.com/mobile/index">Spine Mobile</a> <br> License: MIT<br> Github: Yes<br> Dependencies: Spine<br> Summary: Built on top of Spine that, it strives for a native look-and-feel and includes features like specialized controllers and panel layout, hardware accelerated transitions and touch events. </p> 
               <p><a href="https://github.com/gobhi/gbone.js">gbone.js</a> <br> License: MIT<br> Github: Yes<br> Dependencies: Backbone.js, Underscore.js, Zepto or jQuery<br> Summary: Inspired by Spine Mobile but written for Backbone and adding design patterns like observer and cleanup mixins based on Backbone.js on Rails ebook. </p> 
               <p><a href="http://jqtouch.com/">jQTouch</a> <br> License: MIT<br> Github: Yes<br> Summary: A plugin for Zepto or jQuery designed for mobile web development on the iPhone, <br> Android and iPod Touch. </p> 
               <p><a href="http://enyojs.com/">Enyo</a> <br> License: Apache v2<br> Github: Yes<br> Summary: An object-oriented JavaScript framework built for mobile first, though it supports desktop browsers, and powering WebOS. </p> 
               <p><a href="http://www.chocolatechip-ui.com/">ChocolateChip UI (ChUI)</a> <br> License: MIT<br> Github: Yes<br> Summary: A framework for mobile web application on the WebKit platform that provides layouts and widgets with built-in functionality. Has been ported to jQuery and Zepto.</p> 
               <p><a href="http://xuijs.com/">xui.js</a> <br> License: MIT<br> Github: Yes<br> Summary: An extremely small DOM library for authoring html5 mobile web applications. Originally written for PhoneGap by members of the PhoneGap team.</p> 
               <p><a href="http://joapp.com/">Jo</a> <br> License: MIT<br> Github: Yes<br> Summary: A framework for mobile HTML5 web or native apps that includes widgets, lists and scrolling goodness using JavaScript and CSS3. Works well with PhoneGap.</p> 
               <p><a href="http://dojotoolkit.org/features/mobile">Dojo Mobile</a><br>License: BSD license or the Academic Free License version 2.1<br>GitHub: No (source hosted on SVN)<br>Summary: An HTML5 and JavaScript framework that focuses on a native look and feel on modern webkit-enabled mobile devices such as iPhone, iPod Touch, iPad, Android and RIM.</p> 
               <p>&nbsp;</p> 
               <p><a href="http://dhtmlx.com/touch/">DHTMLX Touch</a><br>License: GNU GPL and free commercial Licenses<br>Github: No<br>Summary: A full framework and UI widgets for building HTML5-based mobile web apps.</p> 
               <p>&nbsp;</p> 
               <h4 dir="ltr">Server-side JS</h4> 
               <p> <a href="http://nodejs.org/">Node.js</a> <br> License: MIT (included libraries vary)<br> GitHub: Yes<br> Summary: &nbsp;Platform built on Chrome's JavaScript runtime for building network applications using an event-driven, non-blocking I/O model for data-intensive real-time applications.</p> 
               <p><a href="http://www.mozilla.org/rhino/">Rhino</a> <br> License: MPL 1.1/GPL 2.0 license<br> Github: Yes<br> Summary: &nbsp;An implementation of JavaScript written entirely in Java typically for embedding into Java applications to provide scripting to end users.</p> 
               <h4 dir="ltr">Testing</h4> 
               <p> <a href="http://visionmedia.github.com/mocha/">Mocha</a> <br> License: MIT<br> Github: Yes<br> Summary: A JavaScript test framework running on node and the browser for asynchronous testing.</p> 
               <p><a href="http://docs.jquery.com/QUnit">QUnit</a> <br> License: MIT<br> Github: Yes<br> Summary: QUnit is a JavaScript test suite used by the jQuery project to test its code and plugins but capable of testing any generic JavaScript code.</p> 
               <p><a href="https://github.com/marstall/shim/">Shim</a> <br> License: unknown<br> Github: Yes<br> Dependencies: Node.js<br> Summary: A cross-browser and cross-device compatibility testing framework.</p> 
               <p><a href="https://github.com/douglascrockford/JSDev">JSDev</a> <br> License: Not listed<br> Github: Yes<br> Summary: A filter that is useful for testing as it activates selected comments, making them executable. </p> 
               <p><a href="http://www.phantomjs.org/">PhantomJS</a> <br> License: MIT<br> Github: Yes<br> Summary: Designed for headless website testing, Shim is a headless WebKit with JavaScript API including native support for DOM handling, CSS selector, JSON, Canvas, and SVG.</p> 
               <p><a href="https://github.com/WebReflection/wru">wru</a> <br> License: MIT<br> Github: Yes<br> Summary: A general purpose test framework compatible with web environment, node.js, Rhino, and now PhantomJS.</p> 
               <p><a href="http://sinonjs.org/">SinonJS</a> <br> License: Simplified BSD<br> Github: Yes<br> Summary: Standalone test spies, stubs and mocks for JavaScript with no dependencies and works with any unit testing framework.</p> 
               <p><a href="http://pivotal.github.com/jasmine/">Jasmine</a> <br> License: MIT<br> Github: Yes<br> Summary: A behavior-driven development framework for testing JavaScript code that does not depend on any other JavaScript frameworks.</p> 
               <h4 dir="ltr">Gaming</h4> 
               <p> <a href="http://impactjs.com/">Impact</a> <br> License: Paid commercial license<br> GitHub: No<br> Summary: Impact is a JavaScript Game Engine that allows you to develop stunning HTML5 Games for desktop and mobile browsers.</p> 
               <p><a href="https://github.com/mrdoob/three.js">ThreeJS</a> <br> License: MIT<br> GitHub: Yes<br> Summary: &nbsp;A lightweight 3D library designed for a very low level of complexity and providing canvas, svg and WebGL renderers.</p> 
               <p><a href="http://createjs.com/">CreateJS</a> <br> License: MIT<br> Github: Yes<br> Summary: A whole suite of libraries, including EaselJs, TweenJS, SoundJS, PreloadJS and Zoe, designed for creating games and animations.</p> 
               <h4 dir="ltr">Languages That Compile to JavaScript</h4> 
               <p> <a href="http://coffeescript.org/">CoffeeScript</a> <br> License: MIT<br> GitHub: Yes<br> Dependencies: Node.js<br> Summary: Billed as an attempt to expose the good parts of JavaScript in a simple way.</p> 
               <p><a href="http://www.dartlang.org/">Google Dart</a> <br> License: BSD<br> Github: No (Source available on Google Code)<br> Summary: Intended to solve JavaScript's problems and offer better performance, including &nbsp;the ability "to be more easily tooled for large-scale projects" and better security features.</p> 
               <p><a href="http://brunch.io/">Brunch</a> <br> License: MIT<br> GitHub: Yes<br> Dependencies: CoffeeScript, Backbone.js, Underscore.js, Stitch, Eco, Stylus &amp; jQuery.<br> Summary: Billed as "a lightweight approach to building HTML5 applications with emphasis on elegance and simplicity."</p> 
               <p><a href="https://github.com/clojure/clojurescript">ClojureScript</a><br>License: Eclipse Public License 1.0<br>Github: Yes<br>Summary: A compiler for Clojure that targets JavaScript and is designed to emit JavaScript code which is compatible with the advanced compilation mode of the Google Closure compiler.</p> 
               <p><a href="http://code.google.com/webtoolkit/">Google Web Toolkit</a>&nbsp;(GWT)<br>License: Creative Commons Attribution 3.0 License<br>Github: No<br>Summary:&nbsp;The GWT SDK contains the Java API libraries, compiler, and development server that let you write client-side applications in Java and deploy them as JavaScript.</p>
