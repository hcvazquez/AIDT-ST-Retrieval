          <h1><span role="img" title="Document" class="jive-icon-big jive-icon-document"></span>How does Adobe Export PDF recognize text and support different languages?</h1> 
          <span class="j-version"> <span class="jive-icon-glyph icon-stack"></span> Version 3 </span> 
          <div class="j-byline font-color-meta">
            Created by 
           <a href="/people/Allta%20Media" data-externalid="" data-username="Allta%20Media" data-avatarid="23762" id="jive-2686751070202701173338" data-userid="268675" data-presence="null" class="jiveTT-hover-user jive-username-link">Allta Media</a> on Mar 21, 2012 6:19 AM. Last modified by

          <div class="jive-rendered-content">
           <p><span style="font-family: arial,helvetica,sans-serif; font-weight: bold; font-size: 10pt;">Does Adobe Export PDF support OCR (Optical Character Recognition)?</span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Yes. Adobe ExportPDF supports optical character recognition or OCR, when you convert a PDF file to Word (.doc and .docx), Excel (.xlsx), or RTF (rich text format).&nbsp; OCR is the conversion of images of text (scanned text) into editable characters, so that you can search, correct and copy the text.</span></p>
           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
           <p><span style="font-size: 10pt;"><span style="font-weight: bold;">When should I use OCR in Adobe </span><span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;">?</span></span></p>
           <p><span style="font-size: 10pt;">If you open your PDF file in Adobe Reader, can you select the text with your mouse?&nbsp; If so, you likely don't need to apply OCR to your file when converting it with ExportPDF.</span></p>
           <p style="min-height: 8pt; padding: 0px;"><span style="font-family: Myriad Pro; font-size: 9pt;"> </span>&nbsp;</p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"><span style="font-weight: bold;">How do I set up Adobe </span><span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;"> to perform OCR when converting a PDF file</span><span style="font-weight: bold;">?</span></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">You don't need to do anything! OCR is automatically enabled. You will usually see better results&nbsp; by leaving this (see step 2 below) enabled. </span></p>
           <ol style="list-style-type: 'decimal';">
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Log into Adobe Export PDF and choose a file type from the Export PDF&nbsp; file to: drop down menu.</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Make sure the Check box Recognize scanned text in: is selected (this is the default setting)</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Choose a language from the drop-down menu located under "Recognize scanned text in:"</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Select the PDF file that you want to convert, and click Open.</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Click Save to save the exported file.</span></li>
           </ol>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"><br></span></p>
           <p><span style="font-size: 10pt; font-family: arial,helvetica,sans-serif; font-weight: bold;">What elements in a PDF file are interpreted by the <span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;"> OCR engine?</span></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">When OCR is enabled, Adobe ExportPDF performs OCR on PDF files that contain:</span></p>
           <ul>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Images</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Vector art</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Hidden text</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Text that cannot be interpreted due to incorrectly encoded text in the source application</span></li>
           </ul>
           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
           <p><span style="font-size: 10pt; font-family: arial,helvetica,sans-serif; font-weight: bold;">Can OCR be turned on/off in <span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;">?</span></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Yes. If you do not want Adobe ExportPDF to convert images of text to editable/renderable text, consider disabling OCR.&nbsp; When OCR is disabled, Adobe ExportPDF can process the PDF document more quickly and the resulting Office file will have embedded images instead of editable.renderable text.&nbsp; Likewise, if the PDF you are converting contains a mix of formatted text and graphics, and you want to leave the graphics as they are, disable OCR.&nbsp; Finally, you can disable OCR if you find that Adobe ExportPDF does not interpret text correctly (because it was encoded incorrectly). The screen capture below shows how to enable/disable OCR by toggling the checkbox labeled "Recognize scanned text in:".&nbsp; </span></p>
           <p><a href="https://forums.adobe.com/servlet/JiveServlet/showImage/102-1812-3-178263/ToggleOCR.png"><img alt="ToggleOCR.png" class="jive-image-thumbnail jive-image" height="282" src="https://forums.adobe.com/servlet/JiveServlet/downloadImage/102-1812-3-178263/620-282/ToggleOCR.png" style="" width="620"></a></p>
           <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
           <p><span style="font-size: 10pt; font-family: arial,helvetica,sans-serif; font-weight: bold;">What languages are supported for OCR by Adobe <span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;">?</span></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Adobe ExportPDF supports the following languages for OCR:</span></p>
           <ul>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">English (US)</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">English (UK)</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">German</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Spanish</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">French</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Italian</span></li>
            <li><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">Japanese</span></li>
           </ul>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"> </span><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"><a href="https://forums.adobe.com/servlet/JiveServlet/showImage/102-1812-3-178264/Language.png"><img alt="Language.png" class="jive-image-thumbnail jive-image" height="368" src="https://forums.adobe.com/servlet/JiveServlet/downloadImage/102-1812-3-178264/620-368/Language.png" style="" width="620"></a></span></p>
           <p style="min-height: 8pt; padding: 0px;"><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"> </span>&nbsp;</p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">By default, OCR uses the language selected in the My Information dialog box (see screen capture below).&nbsp;&nbsp; <br></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"> <a href="https://forums.adobe.com/servlet/JiveServlet/showImage/102-1812-3-178265/Myinformation.png"><img alt="Myinformation.png" class="jive-image" height="369" src="https://forums.adobe.com/servlet/JiveServlet/downloadImage/102-1812-3-178265/Myinformation.png" style="" width="524"></a></span></p>
           <p style="min-height: 8pt; padding: 0px;"><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"> </span>&nbsp;</p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">The OCR engine uses the selected language to interpret the scanned text.&nbsp; Selecting the correct language improves the accuracy of the conversion, as the OCR engine user language-specific dictionaries for conversion.&nbsp; For non-Latin languages like Japanese, the OCR engine cannot interpret and convert the text unless you've selected the appropriate language.</span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"><br></span></p>
           <p style="min-height: 8pt; padding: 0px;"><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"> </span>&nbsp;</p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;"><span style="font-weight: bold;">What if the language in my PDF is not supported for OCR by Adobe </span><span style="font-weight: bold;">ExportPDF</span><span style="font-weight: bold;">?</span></span></p>
           <p><span style="font-family: arial,helvetica,sans-serif; font-size: 10pt;">If OCR is enabled, but the language in the PDF is not supported for OCR by Adobe ExportPDF, the Office file that results from the conversion will contain incorrectly interpreted text.</span></p>
