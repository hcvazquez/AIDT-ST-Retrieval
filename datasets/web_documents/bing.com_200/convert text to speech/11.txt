                    <h3 class="post-title entry-title" itemprop="name"> How Convert Text to Speech using Java API </h3> 
                    <div class="post-header"> 
                     <div class="post-header-line-1"></div> 
                    </div> 
                    <div class="post-body entry-content" id="post-body-4427026043469441187" itemprop="description articleBody"> 
                     <div dir="ltr" style="text-align: left;" trbidi="on"> 
                      <b>Hello Friends,</b>
                      <br> Today I tell you how to develop a Text to Speech Recognition&nbsp;&nbsp; Programme&nbsp; using Java.
                      <br> 
                      <br> 
                      <b>Text to Speech</b>
                      <br> 
                      <br> This is my first post in java speech API series intended to give a brief idea about Java Speech APIs. 
                      <br> 
                      <div class="separator" style="clear: both; text-align: center;"> 
                      </div> 
                      <br> 
                      <h2 style="text-align: left;"> <b>A Brief Idea</b></h2> 
                      <div style="font-family: Times,&quot;Times New Roman&quot;,serif;"> 
                       <span style="color: #444444; font-family: Arial, Helvetica, sans-serif; font-weight: bold;"></span> 
                       <span style="color: #444444; font-family: Arial, Helvetica, sans-serif; font-weight: bold;">The Java Speech API</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">The <b>Java Speech API</b> is a specification that describes a standard set of classes and interfaces for integrating speech technologies into Java software. It is important to keep in mind that Java Speech is only a specification -no implementation is included. Thus third-parties provide the implementations.&nbsp;</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"> </span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Java Speech supports two kinds of speech technologies: </span>
                       <b><span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b>speech recognition</b></span></b>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b> </b>and </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b>speech</b> <b>synthesis</b></span>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">. Speech recognition converts speech to text. Special input devices called </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;">recognizers</span>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"> make speech recognition possible. In contrast, speech synthesis converts text to speech. Special output devices called </span>
                       <b><i><span style="background-color: transparent; font-size: x-small; font-weight: normal; text-decoration: none; vertical-align: baseline;">synthesizers</span></i></b>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b><i> </i></b>make speech synthesis possible.</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">The Java Speech API outlines standards and guidelines as to how speech applications can be built to inter-operate with each other and on all Java compatible platforms. As such, it provides the API and functionality to build the base for speech applications.</span>
                      </div> 
                      <h3 style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: bold; text-decoration: none; vertical-align: baseline;">Features</span></h3> 
                      <ul style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Converts speech to text</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Converts text and delivers them in various formats of speech</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Supports events based on the Java event queue</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Easy to implement API interoperates with multiple Java-based applications like applets and Swing applications</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Interacts seamlessly with the AWT event queue</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Supports annotations using JSML to improve pronunciation and naturalness in speech</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Supports grammar definitions using JSGF</span></li> 
                       <li style="background-color: transparent; font-style: normal; font-weight: normal; list-style-type: disc; text-decoration: none; vertical-align: baseline;"><span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Ability to adapt to the language of the speaker</span></li> 
                      </ul> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span id="internal-source-marker_0.9346039390881463" style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">The <b>javax.speech</b> package defines the common functionality of recognizers, synthesizers, and other </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;">speech engines</span>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">. The package javax.speech.recognition extends this basic functionality for recognizers. Similarly, javax.speech.synthesis extends this basic functionality for synthesizers.</span>
                      </div> 
                      <div class="separator" style="clear: both; font-family: Arial,Helvetica,sans-serif; text-align: center;"> 
                      </div> 
                      <div style="font-family: Arial,Helvetica,sans-serif;"> 
                      </div> 
                      <div class="separator" style="clear: both; font-family: Arial,Helvetica,sans-serif; text-align: center;"> 
                      </div> 
                      <div style="color: black; font-family: Arial,Helvetica,sans-serif;"> 
                      </div> 
                      <div style="color: black; font-family: Arial,Helvetica,sans-serif;"> 
                       <span id="internal-source-marker_0.9346039390881463" style="background-color: transparent; color: black; font-size: 12pt; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span id="internal-source-marker_0.9346039390881463" style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">We are going to use an open source implementation of java </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;">speech synthesis</span>
                       <span id="internal-source-marker_0.9346039390881463" style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"> called&nbsp;</span>
                       <span style="font-size: x-small;"><a href="http://freetts.sourceforge.net/docs/index.php"><span style="background-color: transparent; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"> </span><span style="background-color: transparent; font-style: normal; font-weight: normal; text-decoration: underline; vertical-align: baseline;">FreeTTS</span></a> .&nbsp;</span>
                       <span style="font-size: x-small; vertical-align: baseline;">FreeTTS was built by the</span>
                       <span style="font-size: x-small;"><a href="http://research.sun.com/research/speech/index.html"><span style="text-decoration: none; vertical-align: baseline;"> </span><span style="vertical-align: baseline;">Speech Integration Group</span></a></span>
                       <span style="font-size: x-small; vertical-align: baseline;"> of</span>
                       <span style="font-size: x-small;"><a href="http://research.sun.com/"><span style="text-decoration: none; vertical-align: baseline;"> </span><span style="vertical-align: baseline;">Sun Microsystems Laboratories</span></a></span>
                       <span style="font-size: x-small; vertical-align: baseline;">.</span>
                      </div> 
                      <div style="color: black; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="font-size: x-small;"><br> </span>
                      </div> 
                      <div style="color: black; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="color: #444444;">we are going to explore about the steps to configure <i>FreeTTS </i>in local windows machine.</span>
                       <br> 
                       <span style="color: #444444;"><br></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span id="internal-source-marker_0.9346039390881463" style="font-size: x-small; vertical-align: baseline;"><b>Step 1</b> : Download freeTTS -</span>
                       <span style="font-size: x-small;"><a href="http://freetts.sourceforge.net/docs/index.php#download_and_install"><span style="text-decoration: none; vertical-align: baseline;"> </span><span style="vertical-align: baseline;">Click here</span></a></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b>Step 2</b> : Unzip the freeTTS binary package and check inside the \lib directory for jsapi.exe . </span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b>Step 3</b> : Run jsapi.exe. It will create a jar file namely </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;">jsapi.jar</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b>Step 4</b> : Place all the jar files available inside </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: italic; font-weight: normal; text-decoration: none; vertical-align: baseline;">lib </span>
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">directory to your classpath.</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Or copy all jar files to your &nbsp;Java\jdk1.x.x.x\jre\lib\ext directory.</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><b><u>Source Code Of Text to Speech Program&nbsp;</u></b></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><br></span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;">Here I take a class name TextSpeech under package name is tts.</span>
                      </div> 
                      <div style="color: #444444; font-family: Arial,Helvetica,sans-serif;"> 
                       <span style="background-color: transparent; font-size: x-small; font-style: normal; font-weight: normal; text-decoration: none; vertical-align: baseline;"><br></span>
