      <h1>Javascript Regular Expressions: Form Validation</h1> 
      <hr style="border: 1px dashed #006600;"> 
      <h2 class="heading" style="text-align:center;"> Using Regular Expressions for Form Validations</h2> 
      <hr style="border: 1px dashed #006600;"> 
      <h2 class="code-editor">Javascript Regular Expressions: Form Validation </h2> 
      <p>Everyone must have filled an online form at some stage, a form usually asks for information related to name, phone no, address, credit-card no etc. </p> 
      <p>Incase you didn't provide information in the format specified by the form field or leave it empty, the message will appear and form cannot be submitted until you get it right.</p> 
      <p>This is done using a Javascript program on the client side, the Javascript program uses a <b>regular expression</b> pattern to test the input of each form field.</p> 
      <hr> 
      <h2 class="code-editor">Javascript Form Validation: Checking for Empty fields</h2> 
      <p>Usually, in a form some fields are optional while some fields are mandatory, hence a javascript program is used to check if the input field is empty or null. </p> 
      <p>The demo below , uses a function to check if value in the form field is empty or null.</p>

      <h2 class="code-editor">JavaScript Form Validaiton: Checking for Zip Codes</h2> 
      <p>A simple six digit zipcode can be checked using regular expression which matches exactly six digits : <b>/^d{6}$/</b></p> 
      <p>Another way can be <b>/^[0-9][0-9][0-9][0-9][0-9][0-9]$/</b> , for zipcode with dash in between it can be <b>/^\d{6}-?\d{5}$</b> .</p> 
      <h2 class="heading">Example: JavaScript Form Validation: <span style="color:#fa4b2a;">Checking for Zip Codes</span> </h2> 
      <textarea id="code112632" name="code112632">
 &lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
  &lt;title&gt;JavaScript Form Validation: Valid Zip Code   &lt;/title&gt;
  &lt;style&gt;body{font-family:Verdana;}&lt;/style&gt;
&lt;/head&gt;
  
&lt;body&gt;
&lt;script&gt;
  function check_Zip(zipCode){
    var regex = /^\d{5}$/;
    if(regex.test(zipCode.value) == false){
   alert("The lenght of zip code must be atleast five digits");
   zipCode.focus();
   return false;
    }
    if(zipCode.value == " "){
      alert("Zip code Field cannot be left empty");
      zipCode.focus();
      return false;
    }
    return true;
  }
&lt;/script&gt;

      <h2 class="code-editor">Javascript Form Validation: Alphabetic Data Input</h2> 
      <p>There may be some form fields whose values must be strictly alphabetic characters, eg: Name, Country, State.</p> 
      <p>In the below demo the regular expression looks for one or more uppercase or lowercase letters within the character class <b><i>[A-Za-z]</i></b>, followed with an end of a line anchor <b><i>$</i></b> </p> 
      <h2 class="heading">Example: JavaScript Form Validation - <span style="color:#fa4b2a;">Alphabetic Data Input</span>.</h2> 
      <textarea id="code11268" name="code11268">
 &lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
  &lt;title&gt;JavaScript Form Validation: Valid Zip Code &lt;/title&gt;
  &lt;style&gt;body{font-family:Verdana;}&lt;/style&gt;
&lt;/head&gt;
  
&lt;body&gt;
&lt;script&gt;
  function check_Alpha(letters){
    var regex = /^[a-zA-Z]+$/;
    if(regex.test(letters.yourname.value) == false){
   alert("Name must be in alphabets only");
   letters.yourname.focus();
   return false;
    }
    if(letters.yourname.value == " "){
      alert("Name Field cannot be left empty");
      letters.yourname.focus();
      return false;
    }
    return true;
  }
&lt;/script&gt;
  &lt;form name="alphavalidate" action="form_process.php"
  method="post" onSubmit="return check_Alpha(this)" &gt;

      <h2 class="code-editor">JavaScript Form Validation: Check E-Mail Address</h2> 
      <p> An email address has a format of type : <b><i>An @ sign</i></b> , <b><i>a dot between address and domian</i></b> and <b><i>At least six characters</i></b></p> 
      <p>The regular expression for e-mail validation is : <span class="verge"> / ^ ( ( [ \ - \ w ] + ) \ . ? ) + @ ( ( [ \ - \ w ] + ) \ . ? ) + \ . [ a - zA -Z ] { 2 , 4 } $ /</span></p> 
      <p><b><i> ^ </i></b> : Go to beginning of the line.</p> 
      <p><b><i>([\-\w]+)\.?</i></b> : To check username consists of one or more dashes or word characters grouped by parentheses, followed by a literal period.</p> 
      <p><b><i>(([\-\w]+)\.?)+</i></b> : username can consists of more than one set of word characters seperated by a single dots.</p> 
      <p><b><i>@ </i></b> : To check for <b>@</b> symbol required in the email address.</p>
      <p><b><i>(([\-\w]+)\.?)+</i></b> : username can consists of more than one set of word characters seperated by a single dots.</p> 
      <p><b><i>(([\-\w]+)\.?)+</i></b> : The email server name is similar to user name as well.</p> 
      <p><b><i>[a-zA-Z]{2,4}</i></b> : The mail server name is followed by the domain name, which is usually of 2 to 4 characters.Eg: <b>.com</b>, <b>.in</b> , <b>.info</b> etc.</p> 
      <p><b><i>$</i></b> : To see that no extra character is added to the end of the email address.</p>

    var today = new Date();
    var currentmonth = (today.getMonth() + 1);
    var currentyear = today.getFullYear();
    var y_expire = parseInt(myform.y_expire.value);
    var length_year = myform.y_expire.value.length;
    var month_expire = parseInt(myform.month_expire.value);
    if( length_year == 2){
