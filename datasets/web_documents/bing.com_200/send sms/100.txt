               <h2>You Asked </h2> 
               <div class="ate-Question-youAsked">
                 Hi TOM, 
                <br> 
                <br>We are running an application on Oracle 8.1.5 on solaris and oas 4.081. We are using basic oracle users for authentication. We want to develop a new functionality in our system by which we will be able to send messages on the user's mobile phones. Currently, whenever a new activity happens for any user we send a mail to the user asking him to log on to the site and find out the details. In the similar way can we send text messages on the user's mobile phone along with the emails? 
                <br> 
                <br> 
                <br>Thanks, 
                <br> 
                <br>ashish 
                <br> 
                <br> 
               </div> 
               <h2>and we said...</h2> 
               <div class="ate-Question-weSaid">
                 depends on the phone service. 
                <br> 
                <br>I know someone with a pager that we can send a text message to via EMAIL or a URL. Every night we have a job that queries tomorrows schedule -- formats a text message and uses UTL_HTTP to run a URL on the pagers website that sends him a page. 
                <br> 
                <br>If your phone service provider has a similar setup -- then yes, either sending an email or running a URL can be used.. You'd have to find out if thats available. 
                <br> 
                <br>short of that -- if they have an api, a C or Java external procedure can do virtually anything you want.

                <h3> <a id="61484681767189" name="61484681767189"></a> Send SMS From Oracle to Mobile Device </h3> 
                <p> April 11, 2006 - 1:46 pm UTC </p> 
                <p> Reviewer: Muhammad Waseem Haroon from Karachi, Pakistan </p> 
               </div> 
               <div class="ate-Question-barActions"> 
                <a href="asktom.search?tag=can-i-send-messages-on-mobile-phone-from-oracle-database#61484681767189" class="t-Button t-Button--noLabel t-Button--noUI t-Button--icon t-Button--small t-Button--pillStart" title="Bookmark" aria-label="Bookmark"> <span class="t-Icon fa fa-bookmark" aria-hidden="true"></span> </a>

              <div class="ate-Question-review">
                Tom, 
               <br>I hope that you will fine. Tom, Here I am sending My Suggestion &amp; Experience to Send SMS From Oracle to Mobile Device. 
               <br> 
               <br>There is a Way to send Message From Oracle Database Server to Mobile Device. 
               <br> 
               <br>One Way is to Send Email using Oracle SMTP Package - [UTL_SMTP]. 
               <br>---------------------------------------------------------------- 
               <br>Mostly Cellular Service Providers give the Facility to their Post-Paid Users to Recieve Email on Post-Paid Connections.{Mobile_No@Service_Provider_Domain.com} is the E-Mail Address for Post-Paid Connection Holder. 
               <br>e.g. +923003338669@mobilinkgsm.com. 
               <br> 
               <br>In this Way, Install/Configure Oracle Database 8.x/9.x/10.x and Create a Package. 
               <br>&lt;/code&gt; 
               <pre><a href="http://www.oracle.com/technology/sample_code/tech/pl_sql/htdocs/maildemo_sql.txt">http://www.oracle.com/technology/sample_code/tech/pl_sql/htdocs/maildemo_sql.txt</a></pre> 
               <pre>

BEGIN 
     demo_mail.mail(
                    sender =&gt; 'Me &lt;me@some-company.com&gt;',
                    recipients =&gt; 'Someone &lt;Mobile_No@Celluler_Service_Provider_Domain.com&gt;' ,                       subject    =&gt; 'SMS From Oracle Database Server',                       
                    message =&gt; 'Hi! This is a test Message.'
                    );
END;
/

Second Way is to Send SMS Using GSM Modem/Mobile Phone &amp; its Supporting Software/Utility:
-----------------------------------------------------------------------------------------

There are many software/utilities/Tools which can do this, but here i am using a free project.
In this Way, Install/Configure:

* Red Hat Linux Operating System
* Oracle Database 8i/9.x/10.x
* Gnokii - For Nokia Mobile Phone
                       
To Download Gnokii:</pre> 
               <pre><a href="http://gnokii.org/download/gnokii/0.6.x/gnokii-0.6.2.tar.gz">http://gnokii.org/download/gnokii/0.6.x/gnokii-0.6.2.tar.gz</a></pre> 
               <pre>To Install Gnokii:</pre> 
               <pre><a href="http://urtica.linuxnews.pl/~pkot/gnokii/gnokii.html#ch1">http://urtica.linuxnews.pl/~pkot/gnokii/gnokii.html#ch1</a></pre> 
               <pre>Gnokii Sample File:</pre> 
               <pre><a href="http://cvs.savannah.nongnu.org/viewcvs/gnokii/gnokii/Docs/sample/gnokiirc?rev=1.22">http://cvs.savannah.nongnu.org/viewcvs/gnokii/gnokii/Docs/sample/gnokiirc?rev=1.22</a></pre> 
               <pre>Gnokii User Guide:</pre> 
               <pre><a href="http://wiki.gnokii.org/index.php/User%27s_Guide">http://wiki.gnokii.org/index.php/User%27s_Guide</a></pre> 
               <pre>              
* Nokia Mobile Phone With Data Cable</pre> 
               <pre><a href="http://www.gnokii.org/cables.shtml">http://www.gnokii.org/cables.shtml</a></pre> 
               <pre>
Try to Send SMS Using the Command.

Waseem$ echo "This is a test SMS using Gnokii-Linux..." | gnokii --sendsms +923003338669

Note : Message Length = 160 Characters.

It Will Take 30 - 40 Seconds and will send the SMS to the desired Number.

To Test &amp; Configure it in Oracle Database, you have to Create a Java Class.
</pre> 
               <pre><a href="http://www.oracle-base.com/articles/8i/ShellCommandsFromPLSQL.php">http://www.oracle-base.com/articles/8i/ShellCommandsFromPLSQL.php</a></pre> &lt;code&gt;
               <br> 
               <br> 
               <br>SET SERVEROUTPUT ON SIZE 1000000 
               <br>CALL DBMS_JAVA.SET_OUTPUT(1000000); 
               <br>BEGIN
