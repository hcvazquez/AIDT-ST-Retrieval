         <p>Scrapes the data in the provided element.</p> 
         <h4><a id="user-content-params-1" class="anchor" href="#params-1" aria-hidden="true">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>Params</h4> 
         <ul> 
          <li> <p><strong>Cheerio</strong> <code>$</code>: The input element.</p> </li> 
          <li> <p><strong>Object</strong> <code>opts</code>: An object containing the scraping information. If you want to scrape a list, you have to use the <code>listItem</code> selector:</p> 
           <ul> 
            <li><code>listItem</code> (String): The list item selector.</li> 
            <li><code>data</code> (Object): The fields to include in the list objects: 
             <ul> 
              <li><code>&lt;fieldName&gt;</code> (Object|String): The selector or an object containing: 
               <ul> 
                <li><code>selector</code> (String): The selector.</li> 
                <li><code>convert</code> (Function): An optional function to change the value.</li> 
                <li><code>how</code> (Function|String): A function or function name to access the value.</li> 
                <li><code>attr</code> (String): If provided, the value will be taken based on the attribute name.</li> 
                <li><code>trim</code> (Boolean): If <code>false</code>, the value will <em>not</em> be trimmed (default: <code>true</code>).</li> 
                <li><code>closest</code> (String): If provided, returns the first ancestor of the given element.</li> 
                <li><code>eq</code> (Number): If provided, it will select the <em>nth</em> element.</li> 
                <li><code>listItem</code> (Object): An object, keeping the recursive schema of the <code>listItem</code> object. This can be used to create nested lists.</li> 
               </ul> </li> 
             </ul> </li> 
           </ul> <p><strong>Example</strong>:</p> 
           <div class="highlight highlight-source-js">
            <pre>{
   articles<span class="pl-k">:</span> {
       listItem<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.article<span class="pl-pds">"</span></span>
     , data<span class="pl-k">:</span> {
           createdAt<span class="pl-k">:</span> {
               selector<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.date<span class="pl-pds">"</span></span>
             , <span class="pl-en">convert</span><span class="pl-k">:</span> <span class="pl-smi">x</span> <span class="pl-k">=&gt;</span> <span class="pl-k">new</span> <span class="pl-en">Date</span>(x)
           }
         , title<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>a.article-title<span class="pl-pds">"</span></span>
         , tags<span class="pl-k">:</span> {
               listItem<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.tags &gt; span<span class="pl-pds">"</span></span>
           }
         , content<span class="pl-k">:</span> {
               selector<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.article-content<span class="pl-pds">"</span></span>
             , how<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>html<span class="pl-pds">"</span></span>
           }
         , traverseOtherNode<span class="pl-k">:</span> {
               selector<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.upperNode<span class="pl-pds">"</span></span>
             , closest<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>div<span class="pl-pds">"</span></span>
             , <span class="pl-en">convert</span><span class="pl-k">:</span> <span class="pl-smi">x</span> <span class="pl-k">=&gt;</span> <span class="pl-smi">x</span>.<span class="pl-c1">length</span>
           }
       }
   }
}</pre>
           </div> <p>If you want to collect specific data from the page, just use the same schema used for the <code>data</code> field.</p> <p><strong>Example</strong>:</p> 
           <div class="highlight highlight-source-js">
            <pre>{
     title<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.header h1<span class="pl-pds">"</span></span>
   , desc<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.header h2<span class="pl-pds">"</span></span>
   , avatar<span class="pl-k">:</span> {
         selector<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>.header img<span class="pl-pds">"</span></span>
       , attr<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>src<span class="pl-pds">"</span></span>
     }
}</pre>
           </div> </li>
