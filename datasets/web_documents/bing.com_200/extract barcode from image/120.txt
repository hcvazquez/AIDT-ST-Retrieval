       <div class="section"> 
        <h1>Oracle / PLSQL: <span>EXTRACT Function</span></h1> 
        <p>This Oracle tutorial explains how to use the Oracle/PLSQL <strong>EXTRACT function</strong> with syntax and examples.</p> 
       </div> 
       <div class="section"> 
        <h2>Description</h2> 
        <p>The Oracle/PLSQL EXTRACT function extracts a value from a date or interval value.</p> 
       </div> 
       <div class="section"> 
        <h2>Syntax</h2> 
        <p>The syntax for the EXTRACT function in Oracle/PLSQL is:</p> 
        <pre class="notranslate">EXTRACT (
{ YEAR | MONTH | DAY | HOUR | MINUTE | SECOND }
| { TIMEZONE_HOUR | TIMEZONE_MINUTE }
| { TIMEZONE_REGION | TIMEZONE_ABBR }
FROM { date_value | interval_value } )</pre> 
       </div> 
       <div class="section"> 
        <h2>Note</h2> 
        <ul> 
         <li>You can only extract YEAR, MONTH, and DAY from a DATE.</li> 
         <li>You can only extract TIMEZONE_HOUR and TIMEZONE_MINUTE from a timestamp with a time zone datatype.</li> 
        </ul> 
       </div> 
       <div class="section"> 
        <h2>Returns</h2> 
        <p>The EXTRACT function returns a numeric value when the following parameters are provided: YEAR, MONTH, DAY, HOUR, MINUTE, SECOND, TIMEZONE_HOUR, TIMEZONE_MINUTE, TIMEZONE_REGION, TIMEZONE_MINUTE.<br> The EXTRACT function returns a VARCHAR2 when TIMEZONE_REGION or TIMEZONE_ABBR parameters are provided (because the time zone name or abbreviation information is returned).</p> 
       </div> 
       <div class="section"> 
        <h2>Applies To</h2> 
        <p>The EXTRACT function can be used in the following versions of Oracle/PLSQL:</p> 
        <ul> 
         <li>Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i</li> 
        </ul> 
       </div> 
       <div class="section"> 
        <h2>Example</h2> 
        <p>Let's look at some Oracle EXTRACT function examples and explore how to use the EXTRACT function in Oracle/PLSQL.</p> 
        <p>For example:</p> 
        <pre class="notranslate">EXTRACT(YEAR FROM DATE '2003-08-22')
<em>Result:</em> 2003

EXTRACT(MONTH FROM DATE '2003-08-22')
<em>Result:</em> 8

EXTRACT(DAY FROM DATE '2003-08-22')
<em>Result:</em> 22</pre> 
       </div>
