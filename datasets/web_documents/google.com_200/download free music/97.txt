     <h1>Tween<span>Max</span></h1>
     <img width="100%" src="https://greensock.com/wp-content/themes/greensock/images/divider.png">
     <div class="clear"></div>
     <div class="content">
      <p>TweenMax extends TweenLite, adding many useful (but non-essential) features like &nbsp;<code>repeat()</code>, <code>repeatDelay()</code>, <code>yoyo()</code>, <code>updateTo()</code>, and more. TweenMax was built for convenience, providing a single JavaScript file that contains everything you will commonly need for animating DOM elements:</p> 
      <ul class="compact"> 
       <li>TweenLite</li> 
       <li>TweenMax</li> 
       <li>TimelineLite</li> 
       <li>TimelineMax</li> 
       <li>CSSPlugin</li> 
       <li>AttrPlugin</li> 
       <li>RoundPropsPlugin</li> 
       <li>DirectionalRotationPlugin</li> 
       <li>BezierPlugin</li> 
       <li>EasePack</li> 
      </ul> 
      <p>All that for less than ~33kb (gzipped). Load one file (TweenMax.min.js) and you're good to go!</p> 
      <p>Since TweenMax extends TweenLite, it can do&nbsp;<b>ANYTHING</b>&nbsp;TweenLite can do plus more. You can mix and match TweenLite and TweenMax in your project as you please, but if file size is a concern it is best to stick with TweenLite unless you need a particular TweenMax-only feature.</p> 
      <p>Like TweenLite, a TweenMax instance handles tweening one or more properties of&nbsp;<b>any object</b>&nbsp;(or array of objects) over time.</p> 
      <h2>TweenMax's unique special properties</h2> 
      <p>TweenMax's syntax is identical to TweenLite's. Notice how the TweenMax tween below uses the special properties: <code>repeat</code>, <code>repeatDelay</code>, <code>yoyo</code> and the <code>onRepeat</code> event callback.</p> 
      <div class="codepen" data-height="300" data-theme-id="6453" data-slug-hash="uyfDH" data-default-tab="js"> 
       <pre><code>//basic illustration of TweenMax's repeat, repeatDelay, yoyo and onRepeat
var box = document.getElementById("greenBox"),
    count = 0,
    tween;

tween = TweenMax.to(box, 2, {left:"740px", repeat:10, yoyo:true, onRepeat:onRepeat, repeatDelay:0.5, ease:Linear.easeNone});

function onRepeat() {
  count++;
  box.innerHTML = count;
  TweenLite.set(box, {backgroundColor:"hsl(" + Math.random() * 255 + ", 90%, 60%)"});
}							
</code></pre> 
       <p>See the Pen <a href="http://codepen.io/GreenSock/pen/uyfDH/">TweenMax basic repeat and onRepeat</a> by GreenSock (<a href="http://codepen.io/GreenSock">@GreenSock</a>) on <a href="http://codepen.io">CodePen</a>.</p> 
      </div> 
      <h2>Staggered animations</h2> 
      <p>TweenMax makes it easy to create staggered animations on multiple objects. The animations can overlap, run in direct sequence or have gaps between their start times. TweenMax's three stagger methods: <code><a class="" href="/docs/#/HTML5/GSAP/TweenMax/staggerTo/" target="_self">staggerTo()</a></code>, <code><a class="" href="/docs/#/HTML5/GSAP/TweenMax/staggerFrom/" target="_self">staggerFrom()</a></code> and <code><a class="" href="/docs/#/HTML5/GSAP/TweenMax/staggerFromTo/" target="_self">staggerFromTo()</a></code> are literal one-line wonders.</p> 
      <p class="codepen" data-height="290" data-theme-id="3984" data-slug-hash="HxzEl" data-default-tab="result">See the Pen <a href="http://codepen.io/GreenSock/pen/HxzEl/">TweenMax.staggerTo()</a> by GreenSock (<a href="http://codepen.io/GreenSock">@GreenSock</a>) on <a href="http://codepen.io">CodePen</a>.</p> 
      <p><script src="//codepen.io/assets/embed/ei.js" async></script><br> <script src="//codepen.io/assets/embed/ei.js" async></script></p> 
      <h2>Additional Methods</h2> 
      <p>TweenMax inherits a ton of methods from TweenLite and has quite a few of its own.</p>

  margin-right:80px;
}

  font-weight:bold;
  list-style:none;
  margin-left:-20px;
  font-size:20px;
  margin-bottom:20px; 
}</style></p>
      <ul class="chart compact"> 
       <li>TweenLite and TweenMax Methods</li> 
       <li>delay()</li> 
       <li>delayedCall()</li> 
       <li>duration()</li> 
       <li>eventCallback</li> 
       <li>from()</li> 
       <li>fromTo()</li> 
       <li>getTweensOf()</li> 
       <li>invalidate()</li> 
       <li>isActive()</li> 
       <li>kill()</li> 
       <li>killDelayedCallsTo()</li> 
       <li>killTweensOf()</li> 
       <li>pause()</li> 
       <li>paused()</li> 
       <li>play()</li> 
       <li>progress()</li> 
       <li>restart()</li> 
       <li>resume()</li> 
       <li>reverse()</li> 
       <li>reversed()</li> 
       <li>seek()</li> 
       <li>set()</li> 
       <li>startTime()</li> 
       <li>time()</li> 
       <li>timeScale()</li>

       <li>totalProgress()</li> 
       <li>totalTime()</li> 
      </ul>
