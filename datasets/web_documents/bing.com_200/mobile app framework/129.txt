      <span id="iframes-in-apps" class="anchor"></span> 
      <a href="#iframes-in-apps" class="doc-anchor-link"> <h3 class="doc-anchor">Iframes in Apps</h3> </a> 
      <p>You can embed websites in your apps with iframes. The Apps framework provides a set of APIs that allow you to post and receive messages from both your website and your app.</p> 
      <span id="getting-started" class="anchor"></span> 
      <a href="#getting-started" class="doc-anchor-link"> <h4 class="doc-anchor">Getting Started</h4> </a> 
      <p>Suppose you want to embed the website "https://dashboard.myapp.com" in a Zendesk app. To embed the site, make the following changes to your website and your app.</p> 
      <span id="website" class="anchor"></span> 
      <a href="#website" class="doc-anchor-link"> <h5 class="doc-anchor">Website</h5> </a> 
      <p>Start by adding the following code to your website:</p> 
      <div class="highlight">
       <pre><span class="nt">&lt;script </span><span class="na">type=</span><span class="s">"text/javascript"</span> <span class="na">src=</span><span class="s">"https://assets.zendesk.com/apps/sdk/1.0/zaf_sdk.js"</span><span class="nt">&gt;&lt;/script&gt;</span>
<span class="nt">&lt;script&gt;</span>
  <span class="kd">var</span> <span class="nx">app</span> <span class="o">=</span> <span class="nb">window</span><span class="p">.</span><span class="nx">ZAFClient</span><span class="p">.</span><span class="nx">init</span><span class="p">();</span>

  <span class="nx">app</span><span class="p">.</span><span class="nx">postMessage</span><span class="p">(</span><span class="s1">'hello'</span><span class="p">,</span> <span class="p">{</span> <span class="nx">foo</span><span class="o">:</span> <span class="kc">true</span> <span class="p">});</span> <span class="c1">// post the message 'hello' to the Zendesk app</span>

  <span class="c1">// listen to the 'app.registered' event, which is fired by the framework once the iframe is registered with the app</span>
  <span class="nx">app</span><span class="p">.</span><span class="nx">on</span><span class="p">(</span><span class="s1">'app.registered'</span><span class="p">,</span> <span class="kd">function</span><span class="p">(</span><span class="nx">data</span><span class="p">)</span> <span class="p">{</span>
    <span class="c1">// go nuts</span>
  <span class="p">});</span>

  <span class="nx">app</span><span class="p">.</span><span class="nx">on</span><span class="p">(</span><span class="s1">'helloIframe'</span><span class="p">,</span> <span class="kd">function</span><span class="p">(</span><span class="nx">data</span><span class="p">)</span> <span class="p">{</span> <span class="c1">// listen to the 'helloIframe' message sent from the app</span>
    <span class="k">if</span> <span class="p">(</span><span class="nx">data</span><span class="p">.</span><span class="nx">bar</span><span class="p">)</span> <span class="p">{</span>
      <span class="c1">// app says hello</span>
    <span class="p">}</span>
  <span class="p">});</span>
<span class="nt">&lt;/script&gt;</span>
</pre>
      </div> 
      <p>The code imports the Zendesk App Framework (ZAF) SDK. The ZAF SDK is an open-source JavaScript library that simplifies cross-frame communication between an external website and the Zendesk app containing it. You can view and contribute to the source code on <a href="https://github.com/zendesk/zendesk_app_framework_sdk">GitHub</a>.</p> 
      <p>The <code>src</code> attribute in the <code>&lt;script&gt;</code> element must point to a copy of <a href="https://assets.zendesk.com/apps/sdk/1.0/zaf_sdk.js">zaf_sdk.js</a> or the minified version <a href="https://assets.zendesk.com/apps/sdk/1.0/zaf_sdk.min.js">zaf_sdk.min.js</a>. To benefit from automatic updates and caching, we recommend you always link to our CDN rather than including your own copy.</p> 
      <p>After importing the SDK, you call <code>ZAFClient.init()</code>, which returns a <a href="#zafclient-api">ZAF SDK client</a> object. The client allows you to post and receive framework events on your external website. See the <a href="#client-object">Client Object</a> reference below.</p> 
      <span id="app" class="anchor"></span> 
      <a href="#app" class="doc-anchor-link"> <h5 class="doc-anchor">App</h5> </a> 
      <p>Switch to your app and use the <code>iframe</code> template helper to load the website in a template. Example:</p> 
      <span id="my_template.hdbs" class="anchor"></span> 
      <a href="#my_template.hdbs" class="doc-anchor-link"> <h6 class="doc-anchor">my_template.hdbs</h6> </a> 
      <div class="highlight">
       <pre>{{iframe "https://dashboard.myapp.com"}}
</pre>
      </div> 
      <p>As soon as you switch to the template, your app can start receiving <code>iframe</code> events and posting messages back to the iframe. Example:</p> 
      <span id="app.js" class="anchor"></span> 
      <a href="#app.js" class="doc-anchor-link"> <h6 class="doc-anchor">app.js</h6> </a> 
      <div class="highlight">
       <pre><span class="nx">events</span><span class="o">:</span> <span class="p">{</span>
  <span class="s1">'app.created'</span><span class="o">:</span>  <span class="s1">'init'</span><span class="p">,</span>
  <span class="s1">'iframe.hello'</span><span class="o">:</span> <span class="s1">'handleHello'</span>
<span class="p">},</span>

<span class="nx">init</span><span class="o">:</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span>
  <span class="k">this</span><span class="p">.</span><span class="nx">switchTo</span><span class="p">(</span><span class="s1">'my_template'</span><span class="p">);</span>
<span class="p">},</span>

<span class="nx">handleHello</span><span class="o">:</span> <span class="kd">function</span><span class="p">(</span><span class="nx">data</span><span class="p">)</span> <span class="p">{</span>
  <span class="k">if</span> <span class="p">(</span><span class="nx">data</span><span class="p">.</span><span class="nx">foo</span><span class="p">)</span> <span class="p">{</span>
    <span class="k">this</span><span class="p">.</span><span class="nx">postMessage</span><span class="p">(</span><span class="s1">'helloIframe'</span><span class="p">,</span> <span class="p">{</span> <span class="nx">bar</span><span class="o">:</span> <span class="kc">true</span> <span class="p">});</span>
  <span class="p">}</span>
<span class="p">}</span>
</pre>
      </div> 
      <p>To learn more about <code>iframe</code> events and the <code>postMessage</code> API, see the <a href="./events">Events</a> reference.</p> 
      <span id="zafclient-api" class="anchor"></span> 
      <a href="#zafclient-api" class="doc-anchor-link"> <h4 class="doc-anchor">ZAFClient API</h4> </a> 
      <p>When you include the ZAF SDK on your website, you get access to the <code>ZAFClient</code> object.</p> 
      <span id="zafclient.initcallbackcontext" class="anchor"></span> 
      <a href="#zafclient.initcallbackcontext" class="doc-anchor-link"> <h5 class="doc-anchor">ZAFClient.init([callback(context)])</h5> </a> 
      <p>Returns a <a href="#client-object"><code>client</code></a> object.</p> 
      <span id="arguments" class="anchor"></span> 
      <a href="#arguments" class="doc-anchor-link"> <h6 class="doc-anchor">Arguments</h6> </a> 
      <ul> 
       <li><code>callback(context)</code> (optional) a function called as soon as communication with the Zendesk app is established. The callback receives a context object with data related to the Zendesk app, including <code>currentAccount</code>, <code>currentUser</code>, and <code>location</code></li> 
      </ul> 
      <p>Example:</p> 
      <div class="highlight">
       <pre><span class="kd">var</span> <span class="nx">client</span> <span class="o">=</span> <span class="nx">ZAFClient</span><span class="p">.</span><span class="nx">init</span><span class="p">(</span><span class="kd">function</span><span class="p">(</span><span class="nx">context</span><span class="p">)</span> <span class="p">{</span>
  <span class="kd">var</span> <span class="nx">currentUser</span> <span class="o">=</span> <span class="nx">context</span><span class="p">.</span><span class="nx">currentUser</span><span class="p">;</span>
  <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="s1">'Hi '</span> <span class="o">+</span> <span class="nx">currentUser</span><span class="p">.</span><span class="nx">name</span><span class="p">);</span>
<span class="p">});</span>
</pre>
      </div>
      <span id="client-object" class="anchor"></span> 
      <a href="#client-object" class="doc-anchor-link"> <h4 class="doc-anchor">Client Object</h4> </a> 
      <span id="client.postmessagename-data" class="anchor"></span> 
      <a href="#client.postmessagename-data" class="doc-anchor-link"> <h5 class="doc-anchor">client.postMessage(name, [data])</h5> </a> 
      <p>Allows you to send message events to the Zendesk app.</p> 
      <span id="arguments" class="anchor"></span> 
      <a href="#arguments" class="doc-anchor-link"> <h6 class="doc-anchor">Arguments</h6> </a> 
      <ul>

      <a href="#client.onname-handler-context" class="doc-anchor-link"> <h5 class="doc-anchor">client.on(name, handler, [context])</h5> </a> 
      <p>Allows you to add handlers to a framework event. You can add as many handler as you wish. They will be executed in the order they were added.</p> 
      <span id="arguments" class="anchor"></span> 
      <a href="#arguments" class="doc-anchor-link"> <h6 class="doc-anchor">Arguments</h6> </a> 
      <ul> 
       <li><code>name</code> the name of the framework event you want to listen to. This can be <a href="./events.html#framework-events">framework</a>, <a href="./events.html#request-events">request</a>, or <a href="./events.html#custom-events">custom</a> events. Your iframe can listen to any events your app receives, apart from DOM events. You don't need to register these events in the app first</li> 
       <li><code>handler</code> a function to be called when this event fires. You can expect to receive the same event object your app would receive, parsed as JSON</li> 
       <li><code>context</code> (optional) the value of <code>this</code> within your handler</li> 
      </ul>

       <pre><span class="kd">var</span> <span class="nx">client</span> <span class="o">=</span> <span class="nx">ZAFClient</span><span class="p">.</span><span class="nx">init</span><span class="p">();</span>
