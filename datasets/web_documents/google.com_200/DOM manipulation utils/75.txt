    <h1>Projects</h1> 
    <p>Below is a list of projects, both academic and professional, that I've worked on.</p> 
    <h2><a href="http://www.patentsview.org" target="_blank">PatentsView</a></h2> 
    <p>An interactive visualization and exploration tool for more than 5 million US patents since 1975, developed for the <a href="http://www.uspto.gov" target="_blank">US Patents and Trademark Office</a>. The application was recently <a href="https://www.whitehouse.gov/blog/2016/02/05/open-data-empowering-americans-make-data-driven-decisions" target="_blank">featured by the White House</a> as an example of excellence in promoting transparency in government data.</p> 
    <p>Filtering and sorting options provide visitors with the ability to search for patent information by patent, inventor, assignee, class, location, and date, and view results by patent, inventor, assignee, or class in either a list view or a map view. Within the search results, links to detail views give summary information for individual entities in the data.</p> 
    <p>Click <a href="http://www.patentsview.org" target="_blank">here</a> to launch the website. I developed this application while working for <a href="http://www.periscopic.com" target="_blank">Periscopic</a>. Please click <a href="http://www.periscopic.com/our-work/patentsview-open-government-in-action" target="_blank">here</a> to read more about the project on their website.</p> 
    <h3>JavaScript libraries used:</h3> 
    <ul> 
     <li>jQuery, jQueryUI (DOM manipulation, AJAX)</li> 
     <li>Mocha, Chai, Sinon (unit testing)</li> 
     <li>Handlebars (HTML templating)</li> 
     <li>Backbone (overall application structure, event hub)</li> 
     <li>Ramda (data manipulation in a functional way)</li> 
     <li>Underscore (utilities)</li> 
     <li>RequireJS (module and bundling system)</li> 
     <li>Mapbox, Leaflet (mapping)</li> 
     <li>MomentJS (date/time conversion and formatting)</li> 
    </ul> 
    <h3>Other tools used</h3> 
    <ul> 
     <li>Compass/SASS (CSS preprocessor)</li> 
     <li>Grunt (build tool)</li> 
     <li>Python (API testing, data validation)</li> 
     <li>JSHint (JavaScript linting)</li> 
    </ul> 
    <h2><a href="http://salmonexplorer.ca" target="_blank">Pacific Salmon Explorer</a></h2> 
    <p>A data exploration tool that offers the most comprehensive look at the health and status of wild salmon in British Columbia, developed for the <a href="https://www.psf.ca" target="_blank">Pacific Salmon Foundation</a>.</p> 
    <p>The watersheds of British Columbia support five species of Pacific salmon (sockeye, pink, Chinook, coho, and chum) and provide spawning and rearing habitat for over 400 genetically and geographically distinct populations of wild salmon. The tool provides access to the core biological data for the 55 distinct salmon populations found in the Skeena watershed, as well as key pressures on their freshwater and estuarine habitats.</p> 
    <p>Click <a href="http://salmonexplorer.ca" target="_blank">here</a> to launch the website. I developed this application while working for <a href="http://www.periscopic.com" target="_blank">Periscopic</a>. Please click <a href="http://www.periscopic.com/our-work/a-data-driven-look-at-salmon-habitat-and-populations" target="_blank">here</a> to read more about the project on their website.</p> 
    <h3>Main JavaScript libraries used:</h3> 
    <ul> 
     <li>jQuery (DOM manipulation, AJAX)</li> 
     <li>Backbone, Backbone-Relational, Marionette (MVC framework)</li> 
     <li>Flux (mixed in with Backbone for state management)</li> 
     <li>Underscore/LoDash (utilities, HTML templating)</li> 
     <li>RequireJS (module system and bundling)</li> 
     <li>PdfKit (client-side PDF generation)</li> 
     <li>EaselJS (canvas rendering for charts)</li> 
    </ul> 
    <h3>Other tools:</h3> 
    <ul> 
     <li>Compass/SASS (CSS preprocessor)</li> 
     <li>Grunt (build tool)</li> 
     <li>ESLint (JavaScript linting)</li> 
    </ul> 
    <h2><a href="http://educatorsurvey.tnk12.gov" target="_blank">Tennessee Educator Survey</a></h2> 
    <p>A visualization of the results of the annual Tennessee Educator Survey for sharing with educators, the broader education community, and the general public. Developed for the <a href="https://www.tn.gov/education" target="_blank">Tennessee Department of Education</a>.</p> 
    <p>The Tennessee Educator Survey is distributed annually to teachers and administrators throughout the state. It aims to offer a pulse check on major state initiatives and to provide actionable feedback for policymakers at the state, district, and school levels.</p> 
    <p>Click <a href="http://educatorsurvey.tnk12.gov" target="_blank">here</a> to launch the website. I developed this application while working for <a href="http://www.periscopic.com" target="_blank">Periscopic</a>. Please click <a href="http://www.periscopic.com/our-work/the-2015-tennessee-educator-survey-results" target="_blank">here</a> to read more about the project on their website.</p> 
    <h3>Main JavaScript libraries used:</h3> 
    <ul> 
     <li>jQuery (DOM manipulation, AJAX)</li> 
     <li>Backbone/Marionette (application structure, view management)</li> 
     <li>Ramda (data transformation, functional programming)</li> 
     <li>Handlebars (templating)</li> 
     <li>d3 (chart drawing)</li> 
     <li>EaselJS (canvas drawing)</li> 
     <li>Mocha, Chai, Sinon (unit testing)</li> 
     <li>RequireJS (module system)</li> 
     <li>Underscore (utilities)</li> 
    </ul> 
    <h3>Other tools used:</h3>

     <li>Grunt (build tool)</li> 
     <li>Compass/SASS (CSS preprocessor)</li> 
     <li>ESLint (JavaScript linting)</li> 
    </ul>

    <p>Interactive teacher survey and visualization of survey results for the <a href="http://www.gatesfoundation.org" target="_blank">Bill &amp; Melinda Gates Foundation</a>.</p> 
    <p>Click <a href="http://survey.periscopic.com" target="_blank">here</a> to launch the website. I developed this application while working for <a href="http://www.periscopic.com" target="_blank">Periscopic</a>.</p> 
    <h3>JavaScript libraries used:</h3> 
    <ul> 
     <li>React (view layer)</li> 
     <li>Backbone, Backbone.Radio (model layer, eventing system)</li> 
     <li>jQuery (AJAX, DOM manipulation)</li> 
     <li>LoDash (utilities)</li> 
    </ul> 
    <h3>Other tools used:</h3> 
    <ul>

     <li>Grunt (build tool)</li>

    <h2><a href="http://nodeassets.nbcnews.com/cdnassets/projects/periscopicquiz2/index.html" target="_blank">In Plain Sight Quiz</a></h2> 
    <p>An interactive quiz, developed for <a href="http://www.nbcnews.com" target="_blank">NBC News</a>, that asks users to rank themselves on a class scale of 1 through 10 and then compares those self-rankings to the opinions of other Americans. Results are based on extensive U.S. polling data on income, race, geography, family, health and well-being compiled by the <a href="http://gss.norc.org" target="_blank">General Social Survey</a>.</p> 
    <p>NBC’s <em>In Plain Sight</em> is a two-year project exploring poverty and inequality in the US that wants to know how Americans see their place in the new economic order — and what class means to them now.</p> 
    <p>Click <a href="http://nodeassets.nbcnews.com/cdnassets/projects/periscopicquiz2/index.html" target="_blank">here</a> to launch the website. I developed this application while working for <a href="http://www.periscopic.com" target="_blank">Periscopic</a>. Please click <a href="http://www.periscopic.com/our-work/class-in-america-identities-blur" target="_blank">here</a> to read more about the project on their website.</p> 
    <h3>Main JavaScript libraries used:</h3> 
    <ul> 
     <li>jQuery (DOM manipulation)</li> 
     <li>UnderscoreJS (templating, functional programming, utilities)</li> 
     <li>BackboneJS (application structure, eventing system)</li> 
     <li>EaselJS (canvas rendering and animation)</li> 
     <li>RequireJS (module system)</li> 
    </ul> 
    <h3>Other tools used:</h3> 
    <ul> 
     <li>Compass (CSS preprocessor)</li>

    <h2><a href="https://www.huy-nguyen.com/reed-thesis">Undergraduate Thesis</a></h2> 
    <p>Calculation of the imaginary-time Green’s function (correlation function) of the Hubbard model using the constrained path Monte Carlo method. <a href="https://www.huy-nguyen.com/reed-thesis">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/cpmc-lab">CPMC-Lab</a></h2> 
    <p>A Matlab package (including an interactive Graphical User Interface) to calculate the ground-state energy of the Hubbard model using the constrained path Monte Carlo (CPMC) method. <a href="https://www.huy-nguyen.com/cpmc-lab">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/dmc">DMC</a></h2> 
    <p>A Java package that performs diffusion Monte Carlo (DMC) calculations with an interactive Graphical User Interface using Java’s Swing. <a href="https://www.huy-nguyen.com/dmc">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/heh">Hartree-Fock Study of Helium Hydride</a></h2> 
    <p>An innovative functional and rule-based implementation in Mathematica to compute the equilibrium bond length of HeH+ cation using Hartree-Fock. <a href="https://www.huy-nguyen.com/heh">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/ssc">Avalon Satellite System</a></h2> 
    <p>An award-winning Pol-InSAR (Polarimetric-Interferometric Synthetic Aperture Radar) satellite constellation for maritime security and piracy detection, consisting of three satellites flying in formation. <a href="https://www.huy-nguyen.com/ssc">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/ssef">Integrated Microfluidic and Optical Sorting Device</a></h2> 
    <p>An award-winning Lab-on-a-chip system to separate yeast cells of different sizes using optical force. <a href="https://www.huy-nguyen.com/ssef">Click here to learn more</a>.</p> 
    <h2><a href="https://www.huy-nguyen.com/sspc">The Truth Behind Free Markets</a></h2> 
    <p>An award-winning project that statistically analyzes Adam Smith’s classical liberal view that “basic institutions that protect the liberty of individuals to pursue their own economic interests result in greater prosperity for the larger society.” <a href="https://www.huy-nguyen.com/sspc">Click here to learn more</a>.</p>
