         <h1 id="NWDSDownloadLinks-DeveloperStudioDownloadLinks">Developer Studio Download Links</h1>
         <p><strong>700</strong></p>
         <p><a class="external-link" href="https://support.sap.com/patches" rel="nofollow">https://support.sap.com/patches</a> -&gt; Software Downloads -&gt; By Category -&gt; SAP NETWEAVER AND COMPLEMENTARY PRODUCTS -&gt; SAP NETWEAVER -&gt; <br>SAP NETWEAVER 7.0 -&gt; SUPPORT PACKAGES AND PATCHES -&gt; ENTRY BY COMPONENT -&gt; DEVELOPER STUDIO -&gt; NW DEVELOPER STUDIO 7.00</p>
         <p><strong>701</strong></p>
         <p><a class="external-link" href="https://support.sap.com/patches" rel="nofollow">https://support.sap.com/patches</a> -&gt; Software Downloads -&gt; By Category -&gt; SAP NETWEAVER AND COMPLEMENTARY PRODUCTS -&gt; SAP NETWEAVER -&gt; <br>SAP EHP1 FOR SAP NETWEAVER 7.0 -&gt; SUPPORT PACKAGES AND PATCHES -&gt; ENTRY BY COMPONENT -&gt; DEVELOPER STUDIO -&gt; NW DEVELOPER STUDIO 7.01</p>
         <div class="confluence-information-macro confluence-information-macro-information conf-macro output-block" data-hasbody="true" data-macro-name="info">
          <span class="aui-icon aui-icon-small aui-iconfont-info confluence-information-macro-icon"> </span>
          <div class="confluence-information-macro-body">
           <p><em>Additional remarks to 700 and 701 NWDS releases:</em></p>
           <p>&nbsp;</p>
           <div>
            <p>The downladed SAR files can be extracted using SAPCAR -xvf &lt;JIDEXXXXX.SAR&gt;. After this you can run the exe installer from the<br>extracted SAR file. SAPCAR is available at&nbsp;<a class="external-link" href="http://service.sap.com/patches" rel="nofollow">http://service.sap.com/patches</a>&nbsp;--&gt; Browse our Download Catalog --&gt; Additional Components --&gt; SAPCAR</p>
           </div>
          </div>
         </div>
         <p>&nbsp;</p>
         <hr>
         <p>&nbsp;</p>
         <p>&nbsp;</p>
         <p><strong>710 CE</strong> <br class="atl-forced-newline"><a class="external-link" href="https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/ce/" rel="nofollow">https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/ce/</a></p>
         <p><strong>711 CE</strong> <br class="atl-forced-newline"><a class="external-link" href="https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/711/" rel="nofollow">https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/711/</a></p>
         <p><strong>720 CE</strong> <br class="atl-forced-newline"><a class="external-link" href="https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/720/" rel="nofollow">https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/720/</a></p>
         <p><strong>730</strong> <br class="atl-forced-newline"><a class="external-link" href="https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/730/" rel="nofollow">https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/730/</a></p>
         <p><strong>731</strong> <br class="atl-forced-newline"><a class="external-link" href="https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/731/" rel="nofollow">https://nwds.sap.com/swdc/downloads/updates/netweaver/nwds/nw/731/</a></p>
         <p><strong>740</strong> <br class="atl-forced-newline">There is no dedicated SAP NW Developer Studio 7.40, the SAP NW Developer Studio 7.31 is also to be used for SAP&nbsp;NW 7.40.<br>See also note&nbsp;<a href="https://service.sap.com/sap/support/notes/1791485">#1791485 - NWDS 7.3 EHP1 as development environment for NetWeaver 7.4</a></p>
         <div class="confluence-information-macro confluence-information-macro-information conf-macro output-block" data-hasbody="true" data-macro-name="info">
          <span class="aui-icon aui-icon-small aui-iconfont-info confluence-information-macro-icon"> </span>
          <div class="confluence-information-macro-body">
           <p><em>Additional remarks to SAP NW Developer Studio releases higher equal to 710 and lower than 750:</em></p>
           <p>These links are expecting&nbsp;<strong>S-User authentication</strong>. After download and installation, the Update Site installation has to be executed. You do this from<br>the installed Developer Studio's menu (Help – Software Updates — Find and Install...). This is required to have a complete installation, else Developer Studio will be just an "empty" Developer Studio without SAP content.</p>
           <p><strong>Caution:</strong>&nbsp;Even if you install SAP NW Developer Studio onto a 64bit OS, the JDK has to be 32bit.</p>
          </div>
         </div>
         <p>&nbsp;</p>
         <hr>
         <p><strong><br></strong></p>
         <p><strong>750<br></strong>There is no more update site solution. The new 750 release can be downloaded from&nbsp;<br><a class="external-link" href="https://launchpad.support.sap.com/#/softwarecenter" rel="nofollow">https://launchpad.support.sap.com/#/softwarecenter</a>&nbsp;-&gt;&nbsp;By Alphabetical Index (A-Z) -&gt; N -&gt;&nbsp;SAP NETWEAVER -&gt;&nbsp;SAP NETWEAVER 7.5 -&gt; Click on the blue "SUPPORT PACKAGES AND PATCHES" button at the top right corner -&gt;&nbsp;DEVELOPER STUDIO -&gt;&nbsp;NW DEVELOPER STUDIO 7.50</p>
         <p>&nbsp;</p>
         <div class="confluence-information-macro confluence-information-macro-information conf-macro output-block" data-hasbody="true" data-macro-name="info">
          <span class="aui-icon aui-icon-small aui-iconfont-info confluence-information-macro-icon"> </span>
          <div class="confluence-information-macro-body">
           <p><em>Additional remarks to SAP NW Developer Studio 750:</em></p>
           <ul>
            <li><span>64bit SAP JVM 8 is required for NWDS 7.50. (Oracle JDK is not supported in NW 7.50)</span></li>
            <li>NWDS can be downloaded directly from Support Portal, not from update site</li>
           </ul>
          </div>
