       <h1 class="entry-title">C Quicksort Algorithm</h1>
       <div data-type="ad" data-publisher="dm.zentut.site" data-zone="ron" data-format="300x250"></div>
       <div class="entry-content">
        <p><strong>Summary</strong>: in this tutorial, you will learn how to implement the quicksort algorithm in C.</p>
        <h2>Introduction to quicksort algorithm</h2>
        <p>The quicksort algorithm sorts an unordered list based on the <em>divide and conquer</em> strategy. It divides the unordered list into two sub-lists: low elements sub-list and high elements sub-list, and then recursively sort these sub-lists.</p>
        <p>The following describes the quicksort algorithm steps:</p>
        <ol>
         <li>Pick an element from the list, which is called a <strong>pivot</strong>.</li>
         <li>Reorder the list with a rule that all elements which are less than the <strong>pivot</strong> come before the pivot, whereas all elements that are higher than the list come after the pivot. After partitioning the list, the pivot is in its position.</li>
         <li>With the two sub-lists, apply the above steps recursively.</li>
        </ol>
        <p>The complexity of the quicksort algorithm in the worst case is&nbsp;O(<i>n</i><sup>2</sup>), in both best and average cases are&nbsp;O(<i>n</i>&nbsp;log&nbsp;<i>n</i>) where n is the number of unsorted elements.</p>
        <p>The following picture illustrates how the &nbsp;quicksort algorithm sort a list of integers:</p>
