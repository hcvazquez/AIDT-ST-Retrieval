       <p>Java programs communicate with the database and manipulate its data with the help of the JDBC API. The JDBC driver enables the Java application to connect to a database. JDBC is almost always used with relational databases, also it can be used with any other table based data source. We do not have to worry about the availability of a driver, as major RDBMS (Relational Database&nbsp; Management System) providers provide them free. Apart from that there are many third-party JDBC drivers available.</p>

       <h2><strong>JDBC Programming Steps</strong></h2> 
       <p>Every Java code in JDBC Programming goes through the following six steps in one way or the other. These steps give an idea about what order to follow during coding and a basic insight into their significance.</p> 
       <div style="margin-left: 20px;"> 
        <div class="colLeft dimRelatedArticles_Ads"> 
         <div class="section relatedArticles"> 
          <div class="sectionHeader"> 
           <h3>Related Articles</h3> 
          </div> 
          <div class="sectionContent"> 
           <ul> 
            <li><a href="/java/data/jdbc-application-design-consideration.html">JDBC Application Design Consideration</a></li> 
            <li><a href="/java/data/manipulating-a-database-with-jdbc.html">Manipulating a Database with JDBC</a></li> 
           </ul>

        <p>1. <strong>Importing <em>java.sql</em> Package</strong></p> 
        <p>Almost all the classes and interfaces used in JDBC programming are compiled in the <em>java.sql</em> package. As a result it is our primary requirement to import the package as follows.</p> 
        <pre><strong>import</strong> java.sql.*;</pre> 
        <p>2. <strong>Load and Register JDBC Driver</strong></p> 
        <p>The most common and easiest way to load the driver is by using the <em>Class.forName()</em> method.</p> 
        <pre>Class.<em>forName</em>("com.mysql.jdbc.Driver");</pre> 
        <p>This method takes the complete package name of the driver as its argument. Once the driver is loaded, it will call the <em>DriverManager.registerDriver()</em> method to register itself. Registering a driver implies that the currently registered driver is added to a list of available <em>Driver</em> objects maintained by the <em>DriverManager</em>. The driver manager serves the request from the application using one of the lists of available <em>Driver</em> objects.</p> 
        <p>3. <strong>Establishing Connection</strong></p> 
        <p>The standard method to establish a connection to a database is through the method call <em>DriverManager.getConnection()</em>. The arguments accepted by this method are: a string representation of the database URL, the user name to log in to the database and the password.</p> 
        <pre>DriverManager.<em>getConnection</em>("jdbc:mysql://localhost/hr","user1","pass");</pre> 
        <p>4. <strong>Creating a Statement</strong></p> 
        <p>We need to create a <em>Statement</em> object to execute a static SQL query. Static SQL statements can be updates, inserts, queries and even DDL SQL statements. <em>Statement</em> objects are created&nbsp; with the help of the <em>Connection</em> object's&nbsp; <em>createStatement()</em> method as follows:</p> 
        <pre>Statement statement = connection.createStatement();</pre> 
        <p>5. <strong>Execute SQL Statement and Retrieve Result</strong></p> 
        <p>We can use the <em>executeQuery()</em> method of the <em>Statement</em> object to fire the query to the database. This method takes an SQL query string as an argument and returns the result as a <em>ResultSet</em> object. The <em>ResultSet</em> object contains both the data returned by the query and methods for retrieving the data.</p> 
        <pre>ResultSet resultSet=statement.executeQuery("SELECT
* FROM employees");</pre> 
        <p>The get methods of the <em>ResultSet</em> object can be used to retrieve each of the fields in the record fetched from the database into java variables.</p> 
        <pre><strong>while</strong>(resultSet.next()){
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(resultSet.getString(“emp_id”));
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(resultSet.getString(“first_name”));
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; System.out.println(resultSet.getString(“last_name”));
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ...
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }</pre> 
        <p>6. <strong>Close Connection</strong></p> 
        <p>It is highly recommended that an application should close the <em>Connection</em> object and <em>Statement</em> objects explicitly, because, earlier opened connections can cause trouble for the database and open connections are also prone to security threats. Simply add following statements:</p> 
        <pre>statement.close();
connection.close();</pre> 
       </div> 
       <h2>Putting it Together</h2> 
       <p>Let us create an application to demonstrate the CRUD operation to manipulate the database records. CRUD stands for <strong>Create</strong>, <strong>Read</strong>, <strong>Update </strong>and <strong>Delete</strong>. We shall fetch the database record with a <strong>Read</strong> operation, create a new record and save it into the database with a <strong>Create</strong> operation, modify the existing record with an <strong>Update</strong> operation and remove a record from the database with a <strong>Delete</strong> operation. Observe that the code below is self-explanatory and almost repetitive with a few significant lines actually invoking the change in the operation. To get a deeper grasp on the Java SQL library functions, refer to the Java API documentation.</p> 
       <pre>//import statements...
&nbsp;
public class Main {
&nbsp;
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; private static final String JDBC_DRIVER = "com.mysql.jdbc.Driver";
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; private static final String DATABASE_URL = "jdbc:mysql://localhost/hr";
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; private static final String USERNAME = "admin";
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; private static final String PASSWORD = "secret";
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 
&nbsp;
/* This operation creates the table in the database which otherwise have to be created through SQL DDL. This operation is given for convenience and does not belong to the CRUD operation we are talking of. Nonetheless novice programmer may find it useful as of how to create a table through Java code...*/
&nbsp;
