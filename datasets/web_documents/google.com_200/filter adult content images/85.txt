              <p>There is a JavaScript library called <strong><a href="http://www.patrick-wied.at/static/nudejs/" rel="nofollow noreferrer">nude.js</a></strong> which is for this, although I have never used it. <a href="http://www.patrick-wied.at/static/nudejs/demo/" rel="nofollow noreferrer">Here is a demo</a> of it in use.</p> 
              <p>There is also <strong><a href="http://www.dansdata.com/pornsweeper.htm" rel="nofollow noreferrer">PORNsweeper</a></strong>.</p> 
              <p>Another option is to "outsource" the moderation work using something like <strong><a href="https://www.mturk.com/mturk/welcome" rel="nofollow noreferrer">Amazon Mechanical Turk</a></strong>, which is a crowdsourced platform which "<em>enables computer programs to co-ordinate the use of human intelligence to perform tasks which computers are unable to do</em>". So you would basically pay a small amount per moderation item and have an outsourced actual human to moderate the content for you.</p> 
              <p>The only other solution I can think of is to make the images user moderated, where users can flag inappropriate posts/images for moderation, and if nobody wants to manually moderate them they can simply be removed after a certain number of flags.</p> 
              <p>Here are a few other interesting links on the topic:</p> 
              <ul>

              <p>If you are looking for an API-based solution, you may want to check out <a href="https://sightengine.com" rel="nofollow noreferrer">Sightengine.com</a></p> 
              <p>It's an automated solution to detect things like adult content, violence, celebrities etc in images and videos.</p> 
              <p>Here is an example in PHP, using the SDK:</p> 
              <pre><code>&lt;?php
$client = new SightengineClient('YourApplicationID', 'YourAPIKey');

$output = $client&gt;check('nudity')&gt;image('https://sightengine.com/assets/img/examples/example2.jpg');
</code></pre> 
              <p>The output will then return the classification:</p> 
              <p><code>{ "status": "success", "request": { "id": "req_VjyxevVQYXQZ1HMbnwtn", "timestamp": 1471762434.0244, "operations": 1 }, "nudity": { "raw": 0.000757, "partial": 0.000763, "safe": 0.999243 }, "media": { "id": "med_KWmB2GQZ29N4MVpVdq5K", "uri": "https://sightengine.com/assets/img/examples/example2.jpg" } } </code></p> 
              <p>Have a look at the documentation for more details: <a href="https://sightengine.com/docs/#nudity-detection" rel="nofollow noreferrer">https://sightengine.com/docs/#nudity-detection</a> (disclaimer: I work there)</p>
