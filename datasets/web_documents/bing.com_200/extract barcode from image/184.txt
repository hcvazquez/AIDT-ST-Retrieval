          <span class="titleThree">FAQ</span> 
          <b>What kinds of compression does the Java TIFF library support?</b> 
          <br>
          <!--? placeHolderImage(6); ?-->
          <br> 
          <img src="/legacy/res/img//icon_answer.gif" align="top"> Asprise TIFF library supports all popular compression algorithms, namely: 
          <ul> 
           <li><b>TIFF_COMPRESSION_DEFLATE </b> - Zip in TIFF compression (lossless) </li>
           <li><b>TIFF_COMPRESSION_GROUP3_1D </b> - CCITT run length encoding </li>
           <li><b>TIFF_COMPRESSION_GROUP3_2D </b> - CCITT T.4 compression for bilevel only </li>
           <li><b>TIFF_COMPRESSION_GROUP4 </b> - CCITT T.6 compression for bilevel only </li>
           <li><b>TIFF_COMPRESSION_NONE </b> - No compression </li>
           <li><b>TIFF_COMPRESSION_PACKBITS </b> - byte-oriented 'packbits' compression </li>
          </ul>
