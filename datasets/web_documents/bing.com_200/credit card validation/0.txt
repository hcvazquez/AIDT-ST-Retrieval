              <p>You can use this snippet to validate 16 digits card numbers with <a href="http://en.wikipedia.org/wiki/Luhn_algorithm">Luhn algorithm</a>:</p> 
              <pre><code>function validateCardNumber(number) {
    var regex = new RegExp("^[0-9]{16}$");
    if (!regex.test(number))
        return false;

    return luhnCheck(number);
}

function luhnCheck(val) {
    var sum = 0;
    for (var i = 0; i &lt; val.length; i++) {
        var intVal = parseInt(val.substr(i, 1));
        if (i % 2 == 0) {
            intVal *= 2;
            if (intVal &gt; 9) {
                intVal = 1 + (intVal % 10);
            }
        }
        sum += intVal;
    }
    return (sum % 10) == 0;
}
</code></pre>

            <td class="votecell"> 
             <div class="vote"> 
              <input type="hidden" name="_id_" value="30868032"> 
              <a class="vote-up-off" title="This answer is useful">up vote</a> 
              <span itemprop="upvoteCount" class="vote-count-post ">3</span> 
              <a class="vote-down-off" title="This answer is not useful">down vote</a>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>Luhn algorithm (also known as <a href="https://en.wikipedia.org/wiki/Luhn_algorithm" rel="nofollow">Luhn formula</a>) is useful to validate a variety of identification numbers (e.g. <strong>credit card numbers</strong>, <strong>IMEI</strong>).</p> 
              <p>I omit the explanation of the algorithm because it has already been exposed by others but if you need <strong>the fastest Javascript implementation</strong>, you can see it <a href="https://github.com/leodido/luhn.js/blob/master/src.js" rel="nofollow">here</a>. </p> 
              <p>Put simply ...</p> 
              <pre><code>function luhn(array) {
  return function (number) {
    let len = number ? number.length : 0,
      bit = 1,
      sum = 0;

    while (len--) {
      sum += !(bit ^= 1) ? parseInt(number[len], 10) : array[number[len]];
    }
    return sum % 10 === 0 &amp;&amp; sum &gt; 0;
  };
}([0, 2, 4, 6, 8, 1, 3, 5, 7, 9]);
</code></pre> 
              <p>Note that linked source is in <strong>ES6</strong> language (also known as JavaScript 2015), but is transpiled in <strong>ES5</strong> (see <a href="https://github.com/leodido/luhn.js/blob/master/index.js" rel="nofollow">index.js</a>) and it is fully unit tested.</p> 
              <p>Furthermore it is usable both in <strong>browsers</strong> and/or <strong>node.js</strong>.</p> 
              <p>Benchmarks and other implementation are on <a href="http://jsperf.com/credit-card-validator/20" rel="nofollow">jsperf</a> to verify its high performances.</p> 
              <p>Now, if you simply want to use it grab the code from the linked <a href="https://github.com/leodido/luhn.js" rel="nofollow">repository</a>.</p> 
              <p>Otherwise install it via <a href="http://bower.io/" rel="nofollow">bower</a> ...</p> 
              <pre><code>bower install luhn-alg
</code></pre> 
              <p>Or via <a href="http://npmjs.com" rel="nofollow">npm</a> ...</p> 
              <pre><code>npm install luhn-alg
</code></pre> 
              <hr> 
              <p>Disclaimer: I am the author of the <code>luhn-alg</code> package.</p>

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p><a href="http://en.wikipedia.org/wiki/Luhn_algorithm" rel="nofollow"><strong>Luhn formula</strong></a> is the most popular algorithm in credit card validation. And don't be so afraid of the word <code>algorithm</code> that you're looking for a library. It's incredibly easy to understand. From Wikipedia description, this algorithm can be divide in 3 steps:</p> 
              <blockquote> 
               <ul> 
                <li>From the rightmost digit, which is the check digit, moving left, double the value of every second digit; if the product of this doubling operation is greater than 9 (e.g., 8 × 2 = 16), then sum the digits of the products (e.g., 16: 1 + 6 = 7, 18: 1 + 8 = 9).</li> 
                <li>Take the sum of all the digits.</li> 
                <li>If the total modulo 10 is equal to 0 (if the total ends in zero) then the number is valid according to the Luhn formula; else it is not valid.</li> 
               </ul> 
              </blockquote> 
              <p>Here is my working draft.</p> 
              <pre><code>function luhn(anum){
    anum = anum+'';
    var sum = 0,
        max = anum.length - 1;
    //From the rightmost digit, which is the check digit, moving left
    for(var j=max;j&gt;=0;j--){
        var digit = parseInt(anum[j]);
        //Take the sum of all the digits
        if((max - j) &amp; 1){
            //double the value of every second digit
            var add = digit * 2;
            //if the product of this doubling operation is greater than 9 ,
            //then sum the digits of the products
            sum += add &lt; 10 ? add : 1 + add % 10;
        }else{
            sum += digit;
        }
    }
    //If the total modulo 10 is equal to 0 (if the total ends in zero)
    //then the number is valid according to the Luhn formula;else it is not valid.
    return sum % 10 === 0;
}

luhn(79927398713) -&gt; true
</code></pre>
