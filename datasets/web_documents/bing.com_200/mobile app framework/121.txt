           <p>The popularity of mobile applications is on a perpetual flux of advancing and expanding, as their usage has become increasingly prevalent across mobile phone users.<span id="more-2293"></span> The mobile app explosion has incited a rapid expansion into other sectors, such as <strong>factory automation, GPS and location-based services, banking, order-tracking, and ticket purchases</strong> have made app development more of a challenge; which in turn has led to the creation of a wide range of reviews, recommendations, and sources, including blogs, magazines, and dedicated online app-discovery services.</p> 
           <p>In this article I have gathered <strong>20 Mobile Frameworks</strong> for development that would help developers to create innovative, useful and entertaining apps for mobile lovers. I hope mobile developers will find this list handy and according to their needs. If you know any other mobile frameworks which you think is good enough to serve developers purposes, do let us know by commenting in comment section, we would love to hear your feedback. Enjoy !</p> 
           <p>If you like the article you might be interested in our other articles on <a target="_blank" href="http://codegeekz.com/best-html5-frameworks/">Best HTML5 Frameworks </a> and <a target="_blank" href="http://codegeekz.com/css-frameworks-for-accelerated-development/"> Best CSS Frameworks for Developers</a></p> 
           <h2>1. Fries</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/Fries1.jpg"><img class="alignnone size-full wp-image-2294" alt="Fries" src="http://codegeekz.com/wp-content/uploads/Fries1.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/Fries1.jpg 625w, https://codegeekz.com/wp-content/uploads/Fries1-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Fries is a solid HTML-CSS-JS framework for creating Android-like UIsboth for real-world usage and prototyping. The framework has all the major components like forms, action bars, lists, buttons, spinners or tabs. It is also optimized for PhoneGap and can be easily converted to a native app.</p> 
           <span class="button-wrap"><a href="http://jaunesarmiento.me/fries/" class="button medium" target="_blank">Source</a></span> 
           <h2>2. Appium</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/appium2.jpg"><img class="alignnone size-full wp-image-2295" alt="appium" src="http://codegeekz.com/wp-content/uploads/appium2.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/appium2.jpg 625w, https://codegeekz.com/wp-content/uploads/appium2-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Appium is an open source framework which helps automating mobile app testing from any language and any test framework, with full access to back-end APIs and DBs from test code. It works for both iOS + Android apps and tests can be written with Java, Objective-C, JavaScript, PHP, Python, Ruby, C#, Clojure, or Perl. The framework is Mac OS X only and requires Nodejs to run.</p> 
           <span class="button-wrap"><a href="http://appium.io/" class="button medium" target="_blank">Source</a></span> 
           <h2>3. Junior</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/junior1.jpg"><img class="alignnone size-full wp-image-2296" alt="junior" src="http://codegeekz.com/wp-content/uploads/junior1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/junior1.jpg 625w, https://codegeekz.com/wp-content/uploads/junior1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Junior is a HTML5 front-end framework for creating mobile apps that look and behave like native. It uses CSS3 transitions for a slick performance, supports swipable carousels and includes various UI components (from Ratchet). The framework uses Zepto (which is nice for jQuery fans as it has a jQuery-like syntax) and has integration with backbone.js views + routers. Junior is pretty easy to use and has the documents + examples to get you started.</p> 
           <span class="button-wrap"><a href="http://justspamjustin.github.com/junior/" class="button medium" target="_blank">Source</a></span> 
           <h2>4. Enyo</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/enyo.jpg"><img class="alignnone size-full wp-image-2297" alt="enyo" src="http://codegeekz.com/wp-content/uploads/enyo.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/enyo.jpg 625w, https://codegeekz.com/wp-content/uploads/enyo-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Enyo, the JavaScript framework which was created to power the apps on HP’s TouchPad, has recently reached v2 and now offers cross-platform compatibility rather than only webOS (as v1 did). The new Enyo supports both desktop + mobile, works in all major browsers and comes with a broad set of cross-platform UI widgets + a powerful layout library for building apps.</p> 
           <span class="button-wrap"><a href="http://enyojs.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>5. Sidetap</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/sidetap.jpg"><img class="alignnone size-full wp-image-2298" alt="sidetap" src="http://codegeekz.com/wp-content/uploads/sidetap.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/sidetap.jpg 625w, https://codegeekz.com/wp-content/uploads/sidetap-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Sidetap is a simple and lightweight (2kb minified, gzipped) framework for creating mobile web apps. The framework actually focuses on providing a flexible side-navigationsimilar to the one we know from Facebook’s mobile app, and, once the navigation part is solved, it is pretty easy to build a simple mobile web app. It works by telling Sidetap “the content element to be animated”. It takes care of clearing the stage using whatever animation is appropriate for the situation.</p> 
           <span class="button-wrap"><a href="http://sidetap.it/" class="button medium" target="_blank">Source</a></span> 
           <h2>6. Mobello</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/mobello.jpg"><img class="alignnone size-full wp-image-2299" alt="mobello" src="http://codegeekz.com/wp-content/uploads/mobello.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/mobello.jpg 625w, https://codegeekz.com/wp-content/uploads/mobello-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Mobello is an open source JavaScript UI framework that simplifies developing mobile web apps. The framework aims to provide a native-like experience in mobile, it is optimized for touch events and has 20+ widely-used UI components. It also has an IDE, named Mobello Studio that focuses on building mobile apps with HTML5, CS3 and the Mobello framework. Mobello Studio has a drag ‘n’ drop enabled visual design editor, a JavaScript debugger and built-in mobile device emulator.</p> 
           <span class="button-wrap"><a href="http://mobello.github.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>7. Moobile</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/moobile.jpg"><img class="alignnone size-full wp-image-2300" alt="moobile" src="http://codegeekz.com/wp-content/uploads/moobile.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/moobile.jpg 625w, https://codegeekz.com/wp-content/uploads/moobile-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Moobile is an exciting and fresh project; a mobile web application framework built on MooTools. Currently, it is focused on providing a iOS-like experience on WebKit browsers and comes with many controls like bar, button, image, list, activity indicator, slider and more. It also has support for transitions like fade, slide, cubic or cover and can display native-like alerts. Interfaces created with Moobile are fluid, they will work well both in iPhone + iPad and look good when orientation changes.</p> 
           <span class="button-wrap"><a href="http://www.moobilejs.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>8. Spine Mobile</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/spine-mobile.jpg"><img class="alignnone size-full wp-image-2301" alt="spine mobile" src="http://codegeekz.com/wp-content/uploads/spine-mobile.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/spine-mobile.jpg 625w, https://codegeekz.com/wp-content/uploads/spine-mobile-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Spine Mobile is a JavaScript framework, built on top of SpineJS, for building mobile web apps that look and feel native. The framework comes with specialized controllers, panel layout, hardware accelerated transitions and touch events.</p> 
           <span class="button-wrap"><a href="http://spinejs.com/mobile" class="button medium" target="_blank">Source</a></span> 
           <h2>9. Zoey</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/Zoey2.jpg"><img class="alignnone size-full wp-image-2302" alt="Zoey" src="http://codegeekz.com/wp-content/uploads/Zoey2.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/Zoey2.jpg 625w, https://codegeekz.com/wp-content/uploads/Zoey2-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> It is a HTML5-CSS3-powered framework for creating mobile apps, built on top of Zepto.js and weights only 6kb (gzipped). Zoey has the UI components like navigation, lists, buttons, control groups, forms and grids. The framework is compatible with iOS + Android and comes with an app skeleton that covers all the features.</p> 
           <span class="button-wrap"><a href="http://stanangeloff.github.com/zoey/" class="button medium" target="_blank">Source</a></span> 
           <h2>10. iUI</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/iui.jpg"><img class="alignnone size-full wp-image-2303" alt="iui" src="http://codegeekz.com/wp-content/uploads/iui.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/iui.jpg 625w, https://codegeekz.com/wp-content/uploads/iui-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> iUI is a mobile web framework that has been around for a long time (since 2007) compared to many other mobile resource. The framework includes a JavaScript library, CSS and images for developing touch-enabled web apps. User interfaces to be created are iPhone-like yet compatible with most smartphones and tablets and has support for menus, forms, lists, image galleries and more. Also, there are built-in extensions for caching, getting system information, HTML5 videos or theme switching.</p> 
           <span class="button-wrap"><a href="http://www.iui-js.org/" class="button medium" target="_blank">Source</a></span> 
           <h2>11. Lungo.js</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/lungo1.jpg"><img class="alignnone size-full wp-image-2304" alt="lungo" src="http://codegeekz.com/wp-content/uploads/lungo1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/lungo1.jpg 625w, https://codegeekz.com/wp-content/uploads/lungo1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Lungo.js a framework, powered with HTML5, CSS3 and JavaScript, for creating mobile web apps. The applications created with it works in all of the popular platforms (iOS, Android, Blackberry and WebOs). It has support for touch events like tap, double-tap or swipe and doesn’t use images (including the icons) as everything is vectorized.</p> 
           <span class="button-wrap"><a href="http://lungo.tapquo.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>12. Wink Toolkit</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/wink.jpg"><img class="alignnone size-full wp-image-2305" alt="wink" src="http://codegeekz.com/wp-content/uploads/wink.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/wink.jpg 625w, https://codegeekz.com/wp-content/uploads/wink-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Wink Toolkit is a JavaScript framework for creating mobile web applications with less-hassle. It comes in multiple parts where the “core” (which includes HTTP requests utilities, event management system or DOM extensions) is a must to use and every other feature is optional. Such a structure helps keeping the filesize minimum which is pretty important when it is mobile.</p> 
           <span class="button-wrap"><a href="http://www.winktoolkit.org/" class="button medium" target="_blank">Source</a></span> 
           <h2>13. The M Project</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/the-m-project.jpg"><img class="alignnone size-full wp-image-2306" alt="the-m-project" src="http://codegeekz.com/wp-content/uploads/the-m-project.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/the-m-project.jpg 625w, https://codegeekz.com/wp-content/uploads/the-m-project-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> The-M-Project is an exciting HTML5 JavaScript framework for building cross-browser mobile web applications (iOS, Android, Palm webOS, BlackBerry). It makes use of jQuery on the JavaScript part and contains all ui + core files with features like offline support, internationalization and more. The-M-Project is not alone, it requires nodeJS and comes with a build tool called: Espresso! which makes it easy to structure your code, build and run it on a built-in server.</p> 
           <span class="button-wrap"><a href="http://www.the-m-project.net/" class="button medium" target="_blank">Source</a></span> 
           <h2>14. DHTMLX Touch</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/dhtmlxtouch.jpg"><img class="alignnone size-full wp-image-2307" alt="dhtmlxtouch" src="http://codegeekz.com/wp-content/uploads/dhtmlxtouch.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/dhtmlxtouch.jpg 625w, https://codegeekz.com/wp-content/uploads/dhtmlxtouch-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> DHTMLX Touch is a JavaScript library which heavily makes use of HTML5,for building mobile web applications. It’s not just a set of UI widgets, but a complete framework that allows you to create cross-platform web applications for mobile and touch-screen devices. The framework is compatible with multiple devices and applications built with it will work smoothly on iPad, iPhone and Android.</p> 
           <span class="button-wrap"><a href="http://www.dhtmlx.com/touch/" class="button medium" target="_blank">Source</a></span> 
           <h2>15. Zepto.js</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/zepto.jpg"><img class="alignnone size-full wp-image-2308" alt="zepto" src="http://codegeekz.com/wp-content/uploads/zepto.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/zepto.jpg 625w, https://codegeekz.com/wp-content/uploads/zepto-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Zepto.js is a lightweight (2kb) JavaScript framework for mobile WebKit browsers, with a jQuery-compatible syntax. It aims to help overcoming the most drudge work with a nice API so users can concentrate on getting things done. Besides the most common functions that can be expected from a JS framework, it has support for touch events and detection of the environment.</p> 
           <span class="button-wrap"><a href="http://zeptojs.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>16. jQuery Mobile</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/jquerymobile.jpg"><img class="alignnone size-full wp-image-2309" alt="jquerymobile" src="http://codegeekz.com/wp-content/uploads/jquerymobile.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/jquerymobile.jpg 625w, https://codegeekz.com/wp-content/uploads/jquerymobile-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> jQuery project has just published the expected jQuery Mobile; the touch-optimized web framework for smartphones &amp; tablets. It is built on top of jQuery and provides a unified user interface system that’llperform the same in all popular mobile device platforms. The framework is built around the principle of progressive enhancement and uses the HTML5 and CSS3 features but degrades nicely for browsers that don’t support them.</p> 
           <span class="button-wrap"><a href="http://jquerymobile.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>17. Jo</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/jo1.jpg"><img class="alignnone size-full wp-image-2310" alt="jo" src="http://codegeekz.com/wp-content/uploads/jo1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/jo1.jpg 625w, https://codegeekz.com/wp-content/uploads/jo1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> Jo is an open source mobile application framework that is based on HTML5. It enables you to create for multiple platforms like webOS, iOS, Android, Symbian, Safari, Chrome and Dashboard Widgets. Jo is also compatible with PhoneGap and makes possible to create native apps besides web app.</p> 
           <span class="button-wrap"><a href="http://www.joapp.com/" class="button medium" target="_blank">Source</a></span> 
           <h2>18. Sencha Touch</h2> 
           <p><a href="http://codegeekz.com/wp-content/uploads/sencha-touch1.jpg"><img class="alignnone size-full wp-image-2311" alt="sencha-touch" src="http://codegeekz.com/wp-content/uploads/sencha-touch1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/sencha-touch1.jpg 625w, https://codegeekz.com/wp-content/uploads/sencha-touch1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></a><br> It is an HTML5 Mobile App. Framework, which allows you to develop web applications that look and feel native on Apple iOS and Google Android touchscreen devices. It makes use of HTML5 for delivering audio/video or localStorage and CSS3for maximum styling like rounded corners, background gradients, and shadows.</p>
