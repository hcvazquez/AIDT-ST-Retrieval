          <br> 
          <br> It may take a text string or an HTML excerpt and sends a request to Google Translate API Web servers to return an MP3 audio file with text converted into speech.
          <br> 
          <br> The class may also generate HTML with JavaScript to play the supplied text speech audio in a Web page.
