            }

                var date = new Date();
                date.setDate(date.getDate() + expiryInDays);
                expires = '; expires=' + date.toUTCString();
            }

            domain = '; domain=' + domain;
            document.cookie = name + '=' + value + domain + expires + '; path=/';
        }

           <p class="title">"Encourages Good JavaScript Programming Practices"</p> 
           <p class="author"> April 22, 2010 &nbsp;|&nbsp; By <a href="https://www.cnet.com/profiles/adesigninteractive/" class="noluke"> adesigninteractive</a> </p> 
           <p class="review-author-mobile"> 2010-04-22 01:38:14 &nbsp;|&nbsp; By adesigninteractive </p> 
           <p class="version"> &nbsp;|&nbsp; Version: Free JavaScript Editor 4.7 </p> 
          </header> 
          <p class="sectionHeading">Pros</p>
          <p class="sectionText">Better coding by validation, and example. Free JavaScript Editor, by Yaldex, integrates a local version of JSLint such that the user might learn to improve javascript coding practices. It's a great tool, at no cost.</p> 
          <p class="sectionHeading">Cons</p>
          <p class="sectionText">NAME GAME: If abbreviated as "FJSE", the bells-n-whistles shareware edition "First JavaScript Editor", is easily confused with "Free JavaScript Editor". Be careful not to confuse the "Free" with the "First", both by Yaldex. Confusing name convention</p> 
          <p class="sectionHeading">Summary</p>
          <p class="sectionText">One might question: Why a so-called "JavaScript Editor", when it would seem that most any editor would make a fine "javascript" editor? I asked myself the same question-- an intriguing question, and likely why many users have tried it. The answer; what I believe separates FJSE from other text editors is its inclusion of an advanced JS Debugger, and the introduction of JSLint, a free js app available at jslint.com. <br>Yaldex "Free JavaScript Editor" (as well as the oft confused, "First JavaScript Editor", a shareware by the same developer) integrates a local version of JSLint such that the user might learn to improve javascript coding practices. It's a great tool, at no cost.</p>
