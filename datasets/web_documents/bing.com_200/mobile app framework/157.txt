        <h1 class="disp-none">Getting Started with Intel’s App Framework</h1> 
        <p>First impression, “<em>A Comprehensive Hybrid Mobile App development kit!</em>“.</p> 
        <p>The guys at Intel have done&nbsp;an awesome job in packing everything we need to build a Hybrid Mobile app in most convenient workflow possible. In this post, we will take a look at the App framework, its components, theme’s and the invaluable XDK to build Hybrid mobile apps.</p> 
        <h2>App Framework</h2> 
        <blockquote>
         <p>App Framework is comprised of three parts</p> 
         <p><strong>App Framework</strong> – a query selector library<br> <strong>UI</strong> – a UI/UX library for smartphone and tablets<br> <strong>Plugins</strong> – plugins built on top of App Framework</p> 
         <p><strong>App Framework</strong> was built from the ground up to excel on mobile devices by being light and fast. It takes advantage of new browser features and supports valid W3C CSS selectors. User generated performance tests at jsperf.net have shown it to be faster than every other JavaScript* library available.</p> 
         <p><strong>App Framework UI</strong>, also known as AFUI, is the only mobile framework that treats Android*, iOS*, Blackberry* and Win8* the same. In other words, AFUI does not degrade features or performance in order to resolve conflicts or bugs. For example, most frameworks consider Android* to be “broken” and ignore specific versions of that operating system or take shortcuts that perform poorly. App Framework solves these key issues to provide a clean and responsive HTML5 user interface.</p> 
         <p><strong>Plugins</strong> are separate JavaScript libraries created to augment apps written using App Framework App Framework. Plugins provide useful features or functionality that may be outside the scope of basic applications. App Framework UI harnesses a few of the core plugins like af.scroller.js and af.css3animate.js.</p>
        </blockquote>

        <p>As you can see, you will find everything you need to develop an&nbsp;app using Intel’s app f/w. The menu bar on the left hand side and the tabs at the bottom segregate the complete app components into styles, structure and plugins.</p> 
        <p>You can play around with the features and check out how easy it is to use the f/w.</p> 
        <p>Now that we have a decent idea about the app f/w, let us build a “Hello World” app. There are 3&nbsp;ways you can do it</p> 
        <ol> 
         <li>Create a new folder for your app, copy paste the related code from Kitchen sink folder and build the app, by authoring HTML/CSS and JS.</li> 
         <li>Using the web&nbsp;based App Starter</li> 
         <li>Downloading the Intel XDK “studio”.</li> 
        </ol> 
        <p>If you are planning to use the browser interface, you can click on “App Starter” on the home page</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-11.28.11-am.png" data-rel="lightbox-3" title=""><img class="size-medium wp-image-1867 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-11.28.11-am.png?resize=451%2C215" alt="Screen Shot 2014-08-23 at 11.28.11 am" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1867 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-11.28.11-am.png?resize=451%2C215" alt="Screen Shot 2014-08-23 at 11.28.11 am" data-recalc-dims="1">
          </noscript></a>Then you can login/create a new account into the “Web” XDK. Once you are in, you can either create a new app, or upload your own app. I have dragged and dropped a few things and built the app below</p> 
        <p><a href="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.33.58-pm.png" data-rel="lightbox-4" title=""><img class="size-medium wp-image-1869 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.33.58-pm.png?resize=451%2C202" alt="Screen Shot 2014-08-23 at 12.33.58 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1869 aligncenter" src="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.33.58-pm.png?resize=451%2C202" alt="Screen Shot 2014-08-23 at 12.33.58 pm" data-recalc-dims="1">
          </noscript></a></p> 
        <p>You can click on the preview icon above the layout canvas and you can see a live preview of the app.&nbsp;Once you are happy with the design, you can download the zip. <em>You can find links below that shows how to use the App Starter.&nbsp;</em></p> 
        <p>With the zip downloaded, you can use the <a href="https://appcenter.html5tools-software.intel.com" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://appcenter.html5tools-software.intel.com', 'Intel’s App Center');" title="Intel's App Center" target="_blank">Intel’s App Center</a> to manage your app on the web. Once you are logged in, you can click on “Start a new app”. Then you can select “Create your own from scratch” as shown below and click next</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.07-pm.png" data-rel="lightbox-5" title=""><img class="size-medium wp-image-1872 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.07-pm.png?resize=451%2C269" alt="Screen Shot 2014-08-23 at 12.45.07 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1872 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.07-pm.png?resize=451%2C269" alt="Screen Shot 2014-08-23 at 12.45.07 pm" data-recalc-dims="1">
          </noscript></a>In the next screen, you can select the project bundle we just downloaded and upload it here</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.55-pm.png" data-rel="lightbox-6" title=""><img class="size-medium wp-image-1873 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.55-pm.png?resize=451%2C268" alt="Screen Shot 2014-08-23 at 12.45.55 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1873 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.45.55-pm.png?resize=451%2C268" alt="Screen Shot 2014-08-23 at 12.45.55 pm" data-recalc-dims="1">
          </noscript></a>Once the upload is successful, You can click next till you see the home screen again with the project added. Here you will have a &nbsp;bazillion options to manage your app</p> 
        <p><a href="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.47.58-pm.png" data-rel="lightbox-7" title=""><img class="size-medium wp-image-1874 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.47.58-pm.png?resize=451%2C293" alt="Screen Shot 2014-08-23 at 12.47.58 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1874 aligncenter" src="http://i2.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.47.58-pm.png?resize=451%2C293" alt="Screen Shot 2014-08-23 at 12.47.58 pm" data-recalc-dims="1">
          </noscript></a></p> 
        <p>You can make changes to your code and upload the bundle back and issue builds.</p> 
        <p>The second method is to download XDK on your machine and I would recommend this approach. Navigate to Intel XDK home page and you can <a href="http://xdk-software.intel.com/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://xdk-software.intel.com/', 'download');" title="Download XDK" target="_blank">download</a> the XDK for your OS.</p> 
        <p>Once you have downloaded, installed and launched the app, you will be asked to login. Once you are logged in, you will be shown a page, where you can create a new project</p> 
        <p><a href="http://i1.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.56.28-pm.png" data-rel="lightbox-8" title=""><img class="size-medium wp-image-1876 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i1.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.56.28-pm.png?resize=451%2C176" alt="Screen Shot 2014-08-23 at 12.56.28 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1876 aligncenter" src="http://i1.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-12.56.28-pm.png?resize=451%2C176" alt="Screen Shot 2014-08-23 at 12.56.28 pm" data-recalc-dims="1">
          </noscript></a></p> 
        <p>You&nbsp;can pick any of the templates and get started. I am going to choose “Use App Starter” and create an App. I am going to name it&nbsp;“helloWorldXDKApp”</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.07.08-pm.png" data-rel="lightbox-9" title=""><img class="size-medium wp-image-1879 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.07.08-pm.png?resize=451%2C133" alt="Screen Shot 2014-08-23 at 1.07.08 pm" data-recalc-dims="1">
          <noscript>
           <img class="size-medium wp-image-1879 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.07.08-pm.png?resize=451%2C133" alt="Screen Shot 2014-08-23 at 1.07.08 pm" data-recalc-dims="1">
          </noscript></a></p> 
        <p>and click on create (<em>bottom righthand corner</em>). &nbsp;Once the project is successfully created, you will be shown a tour of XDK. I recommend taking it. And then you will see&nbsp;something like</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.08.51-pm.png" data-rel="lightbox-10" title=""><img class="size-medium wp-image-1880 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.08.51-pm.png?resize=451%2C217" alt="Screen Shot 2014-08-23 at 1.08.51 pm" data-recalc-dims="1">

          </noscript></a>This window has practically everything you need to develop a Hybrid App. Once you are done with the development, you can click on the “Emulate” tab on the top.</p> 
        <p><a href="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.11.28-pm.png" data-rel="lightbox-11" title=""><img class="size-medium wp-image-1881 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.11.28-pm.png?resize=451%2C245" alt="Screen Shot 2014-08-23 at 1.11.28 pm" data-recalc-dims="1">

           <img class="size-medium wp-image-1881 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.11.28-pm.png?resize=451%2C245" alt="Screen Shot 2014-08-23 at 1.11.28 pm" data-recalc-dims="1">
          </noscript></a></p> 
        <p>As you can see, the emulation environment is pretty comprehensive. You can emulate&nbsp;your app locally and then following the instructions on “Test” tab, you can test on an actual device as well!</p> 
        <p>And when you click on “Build” tab, you issue a build to the generate the selected platform installer</p> 
        <p><a href="http://i1.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.15.50-pm.png" data-rel="lightbox-12" title=""><img class="size-medium wp-image-1882 aligncenter" src="http://i0.wp.com/thejackalofjavascript.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="http://i1.wp.com/thejackalofjavascript.com/wp-content/uploads/2014/08/Screen-Shot-2014-08-23-at-1.15.50-pm.png?resize=451%2C245" alt="Screen Shot 2014-08-23 at 1.15.50 pm" data-recalc-dims="1">

          </noscript></a>And then you can manage&nbsp;all your builds and releases right from here!! Pretty sweet right?!!</p>

              <p>Helo Shalaby, The answer to that question depends on what kind of application you are building? </p> 
              <p>From what I have seen, I would use Ionic when working with service layer heavy apps, scenarios where you are relying on network to send data and then bind them to objects, perform DOM manipulation based on that data etc. + you prefer an Angular way of building the application. </p> 
              <p>App f/w is can also do the above things, but the amount of code you need to write to achieve the same would be more. App f/w internally uses a jQuery like API with <code>$.get()</code> <code>$.post()</code> methods, which can be used to fetch data and you can use any micro templating engine to bind the received response to a template and build the UI. </p> 
              <p>I would prefer Intel’s APP f/w when dealing with less network intensive applications. I am still exploring Intel’s App f/w.</p>
