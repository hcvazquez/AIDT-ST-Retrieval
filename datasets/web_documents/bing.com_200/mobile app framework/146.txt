        <p><a href="/news/uploadImg/20110928/10-Mobile-Application-Frameworks-For-Easy-Development.jpg" rel="lightbox"><img class="aligncenter size-full wp-image-1528" title="10 Mobile Application Frameworks For Easy Development" alt="10 Mobile Application Frameworks For Easy Development " src="http://static.open-open.com/news/uploadImg/20110928/10-Mobile-Application-Frameworks-For-Easy-Development.jpg" width="500" height="496"></a><br> Since iPhone and Google Android are launched mobile application development is growing rapidly.         <p>Here at <strong>STG </strong>we explored numbers of Google Pages to collect most apporpriate Mobile Applications Frameworks which will help developers to quickly develop and design mobile applications.These frameworks are cross browser compatible, supports multiple languages such as JSON, YQL, WAML, JavaScript libraries, Jquery UI elements, HTML5 CSS3 for creating better web applications and interfaces for the <strong>touch Screen devices and smart phones. </strong></p> 
        <h2>1. RestKit: Objective-C Framework For iOS </h2> 
        <p><a href="/news/uploadImg/20110928/Restkit.jpg" rel="lightbox"><img class="aligncenter size-full wp-image-1506" title="Restkit" alt="10 Mobile Application Frameworks For Easy Development " src="http://static.open-open.com/news/uploadImg/20110928/Restkit.jpg" width="500" height="255"></a><br> Restkit is an open source objective-C framework that allows interacting with RESTful web services in objective C on iOS and Mac OS X. It combines a clean, simple HTTP request/response API with a powerful object mapping system that reduces the lengthy code which users need to written. Restkit also provides a system for modeling remote resources by mapping them from JSON (or XML) payloads back into local domain objects.<br> Restkit currently supports JSON via the SBJSON and YAJL parsers.<br> <strong><a href="http://restkit.org/" target="_new">Website</a></strong></p> 
        <h2>2. Magazine Grid: CSS –Framework For Easy Magazine Layouts </h2> 
        <p><a href="/news/uploadImg/20110928/Magazine-Grid.jpg" rel="lightbox"><img class="aligncenter size-full wp-image-1507" title="Magazine Grid" alt="10 Mobile Application Frameworks For Easy Development " src="http://static.open-open.com/news/uploadImg/20110928/Magazine-Grid.jpg" width="500" height="273"></a><br> Magazine Grid is an ultramodern CSS-Framework for iPad and iPhone which comes with common magazine design elements such as pagination, gutters and basic grid. Magazine Grid allows users to us HTML5 elements to structure magazine page for iPad. Magazine Grid is very light (lighter than 4 kb) and loads faster on mobile devices, even on older browsers like IE6 and Netscape.<br> <strong><a href="http://asidemag.com/grid/" target="_new">Website</a></strong></p> 
        <h2>3. Joshfire : Multi-Device Development Framework </h2> 
        <p><a href="/news/uploadImg/20110928/JoshFire.jpg" rel="lightbox"><img class="aligncenter size-full wp-image-1508" title="JoshFire" alt="10 Mobile Application Frameworks For Easy Development " src="http://static.open-open.com/news/uploadImg/20110928/JoshFire.jpg" width="500" height="259"></a><br> Joshfire is an open source multi device development frameworks which assists developers to develop web applications that can run on multiple devices. It uses standards like HTML5 and JavaScript and allows developers to quickly integrate native and dedicated web apps for browsers, Node.JS, desktops, smart phones, smart TVs and connected objects.<br> With the help of Joshfire applications can be controlled with keyboard, mouse, touch, tv-remote and different type of devices. Joshfire is currently supporting desktop browsers, iOS, Android, GoogleTV, Samsung TV and there is an adapter for Node.JS is also available.<br> <strong><a href="http://framework.joshfire.com/" target="_new">Website</a></strong></p> 
        <h2>4. ChocolateChip-UI : Web App Development Framework</h2> 
        <p><a href="/news/uploadImg/20110928/ChocolateChip-UI.jpg" rel="lightbox"><img class="aligncenter size-full wp-image-1510" title="ChocolateChip-UI" alt="10 Mobile Application Frameworks For Easy Development " src="http://static.open-open.com/news/uploadImg/20110928/ChocolateChip-UI.jpg" width="500" height="235"></a><br> ChocolateChip-UI is a mobile web app framework which uses HTML5, WAML, CSS and JavaScript. It is based on ChocolateChip JavaScript Library which includes two new libraries ChUI.JS and ChUI.css.<br> ChocolateChip-UI takes advantage of HTML5, CSS3 and ECMAScript5 to deliver a consistent, focused, easy to use and compact framework for creating mobile apps rivaling native iOS apps. ChocolateChip-UI provides fully functional widgets and controls which works automatically which allows users to focus on only date.
