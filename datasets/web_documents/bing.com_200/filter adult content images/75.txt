             <div class="post-text" itemprop="text"> 
              <p>There is already question like this <a href="https://stackoverflow.com/questions/9132149/how-to-convert-buffered-image-to-image-and-vice-versa">link</a> on StackOverflow and the accepted answer is "casting":</p> 
              <pre><code>Image image = ImageIO.read(new File(file));
BufferedImage buffered = (BufferedImage) image;
</code></pre> 
              <p>In my program I try:</p> 
              <pre><code>final float FACTOR  = 4f;
BufferedImage img = ImageIO.read(new File("graphic.png"));
int scaleX = (int) (img.getWidth() * FACTOR);
int scaleY = (int) (img.getHeight() * FACTOR);
Image image = img.getScaledInstance(scaleX, scaleY, Image.SCALE_SMOOTH);
BufferedImage buffered = (BufferedImage) image;
</code></pre> 
              <p>Unfortunatelly I get run time error: </p> 
              <blockquote> 
               <p>sun.awt.image.ToolkitImage cannot be cast to java.awt.image.BufferedImage</p> 
              </blockquote> 
              <p>Obviously casting does not work.<br> Question is: What is (or is there) the proper way of converting Image to BufferedImage?</p>

              <p>From a <a href="http://code.google.com/p/game-engine-for-java/source/browse/src/com/gej/util/ImageTool.java#31">Java Game Engine</a>:</p> 
              <pre><code>/**
 * Converts a given Image into a BufferedImage
 *
 * @param img The Image to be converted
 * @return The converted BufferedImage
 */
public static BufferedImage toBufferedImage(Image img)
{
    if (img instanceof BufferedImage)
    {
        return (BufferedImage) img;
    }

    // Create a buffered image with transparency
    BufferedImage bimage = new BufferedImage(img.getWidth(null), img.getHeight(null), BufferedImage.TYPE_INT_ARGB);

    // Draw the image on to the buffered image
    Graphics2D bGr = bimage.createGraphics();
    bGr.drawImage(img, 0, 0, null);
    bGr.dispose();

    // Return the buffered image
    return bimage;
}
</code></pre>

             <div class="post-text" itemprop="text"> 
              <p>One way to handle this is to create a new BufferedImage, and tell it's graphics object to draw your scaled image into the new BufferedImage:</p> 
              <pre><code>final float FACTOR  = 4f;
BufferedImage img = ImageIO.read(new File("graphic.png"));
int scaleX = (int) (img.getWidth() * FACTOR);
int scaleY = (int) (img.getHeight() * FACTOR);
Image image = img.getScaledInstance(scaleX, scaleY, Image.SCALE_SMOOTH);
BufferedImage buffered = new BufferedImage(scaleX, scaleY, TYPE);
buffered.getGraphics().drawImage(image, 0, 0 , null);
</code></pre> 
              <p>That should do the trick without casting.</p>
