                     <p>This page shows some examples on data mining with R provided at other websites. More examples on data mining with R can be found in my book "<a href="http://www.rdatamining.com/docs">R and Data Mining: Examples and Case Studies</a>", which is downloadable as a .PDF file at the link.</p>
                     <h3><a name="TOC-Clustering"></a>Clustering</h3> 
                     <ul> 
                      <li><a href="http://manuals.bioinformatics.ucr.edu/home/R_BioCondManual#R_clustering" rel="nofollow" target="_blank">Clustering in R</a>: R examples on various clustering techniques, with R codes and dataset provided. See section 4.15 in the link for <a href="http://manuals.bioinformatics.ucr.edu/home/R_BioCondManual#clustering_primer" rel="nofollow" target="_blank">exercises instructions</a>.<br></li>
