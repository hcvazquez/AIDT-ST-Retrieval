             <h2>Create Your Own Web Scraper Using node.js</h2> 
             <p>Want to make you own scraper to scrape any data form any website and return it in JSON format so you can used it anywhere you like? If yes, then you are in the right place.</p> 
             <p>In this tip, I will guide you how to scrape any website to get the desired data using node.js and to obtain the data in JSON format which can be used, e.g., make any app which will run on live data from the internet.</p> 
             <p>I will be using Windows 10 x64 and VS 2015 for this tip and will scrape from a news website, i.e.:</p> 
             <p><a href="http://4.bp.blogspot.com/-XPhdStWXmxE/VmQKlA-xrfI/AAAAAAAAAS0/apTW4v-8UZ8/s1600/node12.JPG"><img alt="Result" src="/KB/Nodejs/1062106/node12.JPG" style="width: 640px; height: 314px" title="Result"></a></p> 
             <p><img></p> 
             <ul> 
              <li>First of all, set up the IDE, go to <a href="https://nodejs.org/en/download/">https://nodejs.org/en/download/</a> and download the node.js pre build installer. For me, it will be Windows installer 64-bit. <p><a href="http://3.bp.blogspot.com/-ooBWOiDDBi8/VmQKh0qnI_I/AAAAAAAAASE/8rRnduYPThc/s1600/Node1.JPG"><img alt="Node.js" src="/KB/Nodejs/1062106/Node1.JPG" style="width: 640px; height: 365px" title="Node.js"></a></p> </li> 
              <li>After installing it, open your Visual Studio and create a new project Templates&gt;JavaScript&gt;Node.js&gt;Basic Node.js Express 4 Application. <p><a href="http://4.bp.blogspot.com/-fkED18bttfo/VmQKiNiHxuI/AAAAAAAAASQ/B4j5IsnzBEw/s1600/Node2.JPG"><img src="/KB/Nodejs/1062106/Node2.JPG" style="width: 640px; height: 392px"></a></p> </li> 
              <li>Now I have to add two packages in <em>npm</em> folder, i.e. ‘<code>Request</code>’ and ‘<code>Cheerio</code>’. <p><a href="http://3.bp.blogspot.com/-c2bMSlwT3OQ/VmQKjx47t5I/AAAAAAAAATA/uzvq-EigD-0/s1600/Node3.gif"><img src="/KB/Nodejs/1062106/Node3.gif" style="width: 640px; height: 274px"></a></p> </li> 
              <li>And uninstall ‘jade’ by doing right click as we don’t need it now and I have to host my json to Azure cloud service so jade gives an exception. If you want to consume json directly in your application or hosting using other service, then you don’t have to uninstall jade.</li> 
              <li>Now go to app.js and comment out the line numbers 14 and 15 as we are not using ‘Views’ <p><a href="http://4.bp.blogspot.com/-4EvzYw1yu0Q/VmQKlao_NSI/AAAAAAAAAS8/-H9IUXd3X1I/s1600/node4.gif"><img alt="Commenting" src="/KB/Nodejs/1062106/node4.gif" style="width: 640px; height: 342px" title="Commenting"></a></p> </li> 
              <li>Also comment out ‘<code>app.use('/', routes);</code>’</li> 
              <li>Change <code>app.use('/users', users);</code> to <code>app.use('/', users);</code></li> 
              <li>Now go to <em>users.js</em> as now we will do the main thing here.
