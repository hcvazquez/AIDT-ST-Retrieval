       <h1 class="title-x">Open Source Philosophy</h1> 
       <p><img class="img--float" style="width: 100px; margin-right: 20px;" src="/assets/images/montagejs/github.svg" alt="GitHub">At Montage Studio, open source software is at the heart of everything that we build. We created the MontageJS HTML5 framework to change the way developers and designers build software together. To help that vision reach its full potential, we extend that spirit of cooperation to the framework itself.</p> 
       <p>MontageJS is open source software, distributed under the permissive <a target="_blank" href="https://github.com/montagejs/montage/blob/master/LICENSE.md">BSD license</a>. We develop in the open, in a public <a target="_blank" href="http://github.com/montagejs/montage">repository on GitHub</a>. Great frameworks are enriched by the breadth of their ecosystems and the scale of the innovation that they foster. We hope that you will join the MontageJS project and help us shape the future of the Web.</p> 
       <a style="font-size: 18px; margin: 2em auto 0 auto; display: block; line-height: 1; padding: 1em 1.2em; border-radius: 0.4em; width: 200px; text-align: center;" target="_blank" href="http://github.com/montagejs/montage" class="button button--action">MontageJS on GitHub</a>

      <div style="padding-top: 50px" class="intro"> 
       <h1 class="title-x">Why MontageJS?</h1> 
       <p><img class="img--float" style="width: 128px; margin-right: 20px;" src="/assets/images/montagejs/logo.png" alt="MontageJS logo"> MontageJS is a modern frontend HTML5 framework that supports creating single-page applications—fast. MontageJS uses time-tested design patterns and software principles, allowing you to easily create a modular architecture for your projects and deliver a high-quality user experience. This allows designers and developers to work collaboratively and quickly. MontageJS is easy to learn because it is closely aligned with the underlying design of the Web, complementing existing standards.</p> 
      </div> 
      <div class="content"> 
       <div class="cols"> 
        <div class="col col-1-2"> 
         <h3 class="title-xxx">Reusable Components</h3> 
         <p><img class="img--montagejs" src="/assets/images/montagejs/components.svg" alt="Components"> MontageJS simplifies rich-client development by offering a modular approach to organizing code and components. Small teams can build applications faster through maximum reusability and larger teams can be more efficient by breaking complex problems into smaller units without stepping on each other during the development process.</p> 
         <h3 class="title-xxx">Component Oriented Templates</h3> 
         <p><img class="img--montagejs" src="/assets/images/montagejs/templates.svg" alt="Templates"> MontageJS templates are HTML5 documents. These HTML templates encapsulate a component's JavaScript code, as well as resources like CSS files and images, in a folder with the "reel" file extension. Developers can instantiate a component without having to worry about what JavaScript dependencies the component has, or what CSS content and other files it may rely upon. This leads to dramatic improvements in reliability and ease of use.</p> 
        </div>

         <h3 class="title-xxx">High Performance</h3> 
         <p><img class="img--montagejs" src="/assets/images/montagejs/performance.svg" alt="Performance"> MontageJS supports implicit event delegation and a managed draw cycle. This minimizes expensive layout reflows to provide a better user experience, particularly on hardware-limited mobile devices.</p> 
         <h3 class="title-xxx">Functional Reactive Bindings (FRB)</h3> 
         <p><img class="img--montagejs" src="/assets/images/montagejs/frb.svg" alt="FRB"> MontageJS supports simple and two-way data binding between components, objects, and collections, allowing you to coordinate the state between views and models, among other entangled objects. FRB go farther and can gracefully bind long property paths and the contents of collections, as well as incrementally update the results of chains of queries including maps, flattened arrays, sums, and averages. FRB makes it easy to ensure consistent state without having to write a lot of "glue" code.</p> 
        </div> 
       </div> 
       <h1 style="width: 100%; text-align: center; margin-top: 15px;" class="title-x">Visual Authoring for MontageJS</h1> 
       <p>MontageJS is the only open source HTML5 framework with a dedicated, cloud-based, visual authoring environment that is free to use for creating open source content. With <a href="/product">Montage Studio</a>, you will be able to login with your GitHub account and discover an easier, faster way to develop single page applications that exceed user expectations. Montage Studio is a great way to learn and explore the powerful capabilities of MontageJS.</p> 
       <p>If you prefer to write code by hand in your favorite text editor, you can use MontageJS on your local system. The MontageJS framework is available from npm, the node.js package manager. Read our <a href="http://docs.montagestudio.com/montagejs/montagejs-setup.html">Quick Start</a> guide to learn how to install our helpful set of MontageJS command line tools.</p> 
       <p>If you need help, visit our <a href="/community">Community</a> page to discover all the ways you can learn more about MontageJS. You can also take a look at some of our <a href="http://docs.montagestudio.com/montagejs/montagejs-examples.html">sample applications and demos</a> to see MontageJS in action.</p>
