      <p id="articleSummary">This tutorial describes JavaScript's Date object, and shows how you can use it to manipulate dates. Also included is a simple script to display today's date on your web page. Very handy!</p> 
      <p>JavaScript's <code>Date</code> object makes it easy to handle dates in JavaScript. You can use this object to store dates, convert between date formats, and retrieve the current date and time.</p> 
      <p>In this tutorial we'll take a look at using the <code>Date</code> object, and end with a simple example script that displays the current date on the web page!</p> 
      <h2>The <code>Date</code> object</h2> 
      <p>The <code>Date</code> class is used to store and retrieve dates in JavaScript. To create a <code>Date</code> object, we can use any of the following techniques:</p> 
      <h3><code>new Date ( )</code></h3> 
      <p>This creates a new <code>Date</code> object with the value of the current date and time on the browser PC. (We'll use this technique later on to display the current date on the page!) For example:</p> 
      <pre class="code"><code>
x = new Date ( );
</code></pre> 
      <h3><code>new Date ( <var>milliseconds</var> )</code></h3> 
      <p>This creates a new <code>Date</code> object with the date value represented by the number of milliseconds since midnight on Jan 1, 1970. For example:</p> 
      <pre class="code"><code>
x = new Date ( 86400000 );
</code></pre> 
      <p>creates a new date object <code>x</code> with a date value of midnight, Jan 2, 1970. (There are 86,400,000 milliseconds in a day!)</p> 
      <h3><code>new Date ( <var>dateString</var> )</code></h3> 
      <p>This creates a <code>Date</code> object with the date value represented by a date string in a format recognized by the <code>Date.parse()</code> method. (For more info on <code>Date.parse()</code>, see the "Other Date methods" section below.)</p> 
      <p>For example:</p> 
      <pre class="code"><code>
x = new Date ( "January 6, 1972" );
x = new Date ( "January 6, 1972 16:05:00" );
</code></pre> 
      <h3><code>new Date ( <var>year, month, date</var> )</code></h3> 
      <p>We can also explicitly pass the year, month and day to the <code>Date</code> constructor. Note that, in JavaScript, months run from 0 to 11, rather than 1 to 12!</p> 
      <p>For example, to create a new <code>Date</code> object with a value of January 6, 1972:</p> 
      <pre class="code"><code>
x = new Date ( 1972, 0, 6 );
</code></pre> 
      <h3><code>new Date ( <var>year, month, date, hour, minute, second</var> )</code></h3> 
      <p>It's also possible to specify the time as well as the date when creating the object. So to create a new <b>Date</b> object set to Jan 6, 1972 at 5 minutes past 4 in the afternoon:</p> 
      <pre class="code"><code>
x = new Date ( 1972, 0, 6, 16, 5, 0 );
</code></pre> 
      <h2>Manipulating dates</h2> 
      <p>Now that we know how to create <code>Date</code> objects, let's take a look at some of the operations you can perform on them.</p>

      <h4>About UTC</h4> 
      <p>UTC is a timezone-independent way of storing time values, based on milliseconds since midnight, January 1, 1970 in the GMT timezone. All dates and times are stored internally in JavaScript using UTC. When you use the above, non-UTC functions to get dates and times, JavaScript converts the time back into the local computer's time locale first. However, the UTC functions below will return times and dates in UTC format instead.</p>

      <h3>Other <code>Date</code> methods</h3> 
      <p>There is a handful of further methods that can be used on <code>Date</code> objects:</p> 
      <table class="shaded"> 
       <tbody>
        <tr> 
         <th>Method</th> 
         <th>Description</th> 
         <th>Example</th> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>parse()</code></td> 
         <td>Converts a date text string to the "milliseconds since midnight Jan 1, 1970" format. You call it using <code>Date.parse()</code> rather than <code>object.parse()</code>.</td> 
         <td style="white-space: nowrap"><code>x = Date.parse<br>("January 6, 1972")</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>UTC(<var>y, m, d, h, m, s</var>)</code></td> 
         <td>Converts the specified date to the "milliseconds since midnight Jan 1, 1970" format. You call it using <code>Date.UTC()</code> rather than <code>object.UTC()</code>.</td> 
         <td style="white-space: nowrap"><code>x = Date.UTC(1972, 0, 6)<br><br>x = Date.UTC<br>(1972, 0, 6, 16, 5, 0)</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>toString()</code></td> 
         <td>Outputs the value of the <code>Date</code> object as a date text string.</td> 
         <td style="white-space: nowrap"><code>y = x.toString()</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>toUTCString()</code></td> 
         <td>Outputs the value of the Date object as a date text string, using UTC.</td> 
         <td style="white-space: nowrap"><code>y = x.toUTCString()</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>toLocaleString()</code></td> 
         <td>Outputs the value of the Date object as a date text string, using the computer's locale conventions (e.g. D/M/Y for UK computers).</td> 
         <td style="white-space: nowrap"><code>y = x.toLocaleString()</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>toDateString()</code></td> 
         <td>Outputs the "date" portion of the Date object as a text string.</td> 
         <td style="white-space: nowrap"><code>y = x.toDateString()</code></td> 
        </tr> 
        <tr> 
         <td style="white-space: nowrap"><code>toTimeString()</code></td> 
         <td>Outputs the "time" portion of the Date object as a text string.</td> 
         <td style="white-space: nowrap"><code>y = x.toTimeString()</code></td> 
        </tr> 
       </tbody>
      </table> 
      <br> 
      <p>As you can see, there are lots of methods to help you work with dates! There are actually a few more obscure ones, but we've just listed the most useful ones here.</p> 
      <h2>A simple example</h2> 
      <p>In this example, we'll write a script that you can use to display the current date on your pages.</p> 
      <p>To try it out, view <a href="/res/File/articles/development/javascript/working-with-dates/example.html">this page</a>! (Press your browser's <strong class="menu">Back</strong> button to return here.)</p> 
      <p>The first thing we'll need to do is create a new <code>Date</code> object that stores the current date and time. You'll remember that we can do this very easily:</p> 
      <pre class="code"><code>
var current_date = new Date ( );
</code></pre> 
      <p>We could just use the <code>toDateString()</code> method to extract the date portion of <code>current_date</code>, but the results aren't that pretty!</p> 
      <pre class="samp"><samp>
Fri Nov 15 2002
</samp></pre> 
      <p>Instead, we'll use <code>getFullYear()</code>, <code>getMonth()</code>, <code>getDate()</code> and <code>getDay()</code> to extract the relevant components from the <code>Date</code> object. We can convert the results of <code>getMonth()</code> and <code>getDay()</code> from numbers to names using a couple of arrays.</p> 
      <p>Here's the whole script:</p> 
      <pre class="code"><code>
&lt;script language="JavaScript"&gt;
&lt;!--
var current_date = new Date ( );

var month_names = new Array ( );
month_names[month_names.length] = "January";
month_names[month_names.length] = "February";
month_names[month_names.length] = "March";
month_names[month_names.length] = "April";
month_names[month_names.length] = "May";
month_names[month_names.length] = "June";
month_names[month_names.length] = "July";
month_names[month_names.length] = "August";
month_names[month_names.length] = "September";
month_names[month_names.length] = "October";
month_names[month_names.length] = "November";
month_names[month_names.length] = "December";

var day_names = new Array ( );
day_names[day_names.length] = "Sunday";
day_names[day_names.length] = "Monday";
day_names[day_names.length] = "Tuesday";
day_names[day_names.length] = "Wednesday";
day_names[day_names.length] = "Thursday";
day_names[day_names.length] = "Friday";
day_names[day_names.length] = "Saturday";

document.write ( day_names[current_date.getDay()] );
document.write ( ", " );
document.write ( month_names[current_date.getMonth()] );
document.write ( " " + current_date.getDate() );
document.write ( " " );
document.write ( " " + current_date.getFullYear() );
// --&gt;
&lt;/script&gt;
</code></pre> 
      <p>Now we get a much prettier-looking string!</p> 
      <pre class="samp"><samp>
Friday, November 15 2002
</samp></pre> 
      <p>Feel free to copy and paste the above script and use it in your own pages. Just insert it in the page where you want the date to appear.</p> 
      <p>We've now looked at the <code>Date</code> object and its methods, and shown how to store, retrieve and manipulate dates. We've also included a simple example to show how it all hangs together.</p>
