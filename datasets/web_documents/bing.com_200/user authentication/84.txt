    <p>This Java tutorial is to help implement authentication in Java using Facebook OAuth Login API. We will be using Java and a JSON parser API and other than that we will not use any third-party component. Facebook is not providing any sdk for Java client. </p> 
    <p>Apart from Spring Social I couldn’t find any reputable Java SDK for Facebook integration. With a Json parser we can handle everything ourselves. In this tutorial, we will see how to use Facebook to implement an authentication in a custom Java application and get Facebook profile data like name, email and gender.</p> 
    <p><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="Login-With-Facebook" border="0" alt="Login-With-Facebook" src="http://javapapers.com/wp-content/uploads/2014/07/Login-With-Facebook.jpg" width="610" height="360"></p> 
    <h2>Facebook Web Application</h2> 
    <p>We need to create a web application in Facebook. After logging in to <a href="https://developers.facebook.com/" target="_blank">https://developers.facebook.com/</a> under Apps menu click “Create a New App” </p> 
    <p><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="Create-Facebook-Application" border="0" alt="Create-Facebook-Application" src="http://javapapers.com/wp-content/uploads/2014/07/Create-Facebook-Application.jpg" width="610" height="334"></p> 
    <h3>Facebook Application Settings</h3> 
    <p>We need to specify the application callback url in the FB settings. This will be used by the FB server on authentication to hand back control to our application.</p> 
    <p><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="Facebook-Application-Settings" border="0" alt="Facebook-Application-Settings" src="http://javapapers.com/wp-content/uploads/2014/07/Facebook-Application-Settings.jpg" width="610" height="473"></p> 
    <p>There is a lot of discussion about how Facebook will directly call our “localhost” as it will not be visible to it. Facebook will never call the application URL directly. A request will be sent back as response to the client browser with the callback url. The browser does the request. A sequence diagram given below explains the flow of control in authentication using Facebook OAuth.</p> 
    <h2>Facebook OAuth Authentication Sequence Flow</h2> 
    <p><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="Facebook-Login-Sequence-Diagram" border="0" alt="Facebook-Login-Sequence-Diagram" src="http://javapapers.com/wp-content/uploads/2014/07/Facebook-Login-Sequence-Diagram.jpg" width="610" height="360"></p> 
    <ol> 
     <li>On access of an url or in welcome page the Facebook Login button is shown. The user will click the FB button to login into the Java web application. On click of that button a Facebook URL will be invoked. </li> 
     <li>Facebook will validate the application ID and then will redirect to its login page. </li> 
    </ol> 
    <ol> 
     <li>User will enter the FB login credentials and submit the form. </li> 
     <li>Facebook will validate the credentials and then redirect back to the browser with a request to forward to the redirect_url. Redirect_url is the URL in our application which will take care of further processing. </li> 
     <li>Browser will call the redirect url. </li> 
     <li>Redirect URL page will again call the Facebook to request for access_token. </li> 
     <li>Facebook on validation success will respond back with access_token. </li> 
     <li>Redirect URL page will again call the Facebook to request for user data by sending the access_token. </li> 
     <li>Facebook on validating the access_token will respond back with user data requested. </li> 
     <li>Redirect URL page will forward to a page showing user data in the client browser. </li> 
    </ol>

    <h2>Application redirect URL</h2> 
    <p>Following is the page the Facebook issues a redirect request to the browser.</p> 
    <pre class="prettyprint lang-java">package com.javapapers.java.social.facebook;

import java.io.IOException;
import java.util.Map;

import javax.servlet.ServletException;
import javax.servlet.ServletOutputStream;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

public class MainMenu extends HttpServlet {

	private static final long serialVersionUID = 1L;
	private String code="";

	public void service(HttpServletRequest req, HttpServletResponse res)
			throws ServletException, IOException {		
		code = req.getParameter("code");
		if (code == null || code.equals("")) {
			throw new RuntimeException(
					"ERROR: Didn't get code parameter in callback.");
		}
		FBConnection fbConnection = new FBConnection();
		String accessToken = fbConnection.getAccessToken(code);

		FBGraph fbGraph = new FBGraph(accessToken);
		String graph = fbGraph.getFBGraph();
		Map&lt;String, String&gt; fbProfileData = fbGraph.getGraphData(graph);
		ServletOutputStream out = res.getOutputStream();
		out.println("&lt;h1&gt;Facebook Login using Java&lt;/h1&gt;");
		out.println("&lt;h2&gt;Application Main Menu&lt;/h2&gt;");
		out.println("&lt;div&gt;Welcome "+fbProfileData.get("first_name"));
		out.println("&lt;div&gt;Your Email: "+fbProfileData.get("email"));
		out.println("&lt;div&gt;You are "+fbProfileData.get("gender"));		
	}

}</pre> 
    <h2>Facebook Get Access Token</h2> 
    <pre class="prettyprint lang-java">package com.javapapers.java.social.facebook;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.UnsupportedEncodingException;
import java.net.MalformedURLException;
import java.net.URL;
import java.net.URLConnection;
import java.net.URLEncoder;

public class FBConnection {
	public static final String FB_APP_ID = "234231262685378";
	public static final String FB_APP_SECRET = "y2a73dede3n49uid13502f5ab8cdt390";
	public static final String REDIRECT_URI = "http://localhost:8080/Facebook_Login/fbhome";

	static String accessToken = "";

	public String getFBAuthUrl() {
		String fbLoginUrl = "";
		try {
			fbLoginUrl = "http://www.facebook.com/dialog/oauth?" + "client_id="
					+ FBConnection.FB_APP_ID + "&amp;redirect_uri="
					+ URLEncoder.encode(FBConnection.REDIRECT_URI, "UTF-8")
					+ "&amp;scope=email";
		} catch (UnsupportedEncodingException e) {
			e.printStackTrace();
		}
		return fbLoginUrl;
	}

	public String getFBGraphUrl(String code) {
		String fbGraphUrl = "";
		try {
			fbGraphUrl = "https://graph.facebook.com/oauth/access_token?"
					+ "client_id=" + FBConnection.FB_APP_ID + "&amp;redirect_uri="
					+ URLEncoder.encode(FBConnection.REDIRECT_URI, "UTF-8")
					+ "&amp;client_secret=" + FB_APP_SECRET + "&amp;code=" + code;
		} catch (UnsupportedEncodingException e) {
			e.printStackTrace();
		}
		return fbGraphUrl;
	}

	public String getAccessToken(String code) {
		if ("".equals(accessToken)) {
			URL fbGraphURL;
			try {
				fbGraphURL = new URL(getFBGraphUrl(code));
			} catch (MalformedURLException e) {
				e.printStackTrace();
				throw new RuntimeException("Invalid code received " + e);
			}
			URLConnection fbConnection;
			StringBuffer b = null;
			try {
				fbConnection = fbGraphURL.openConnection();
				BufferedReader in;
				in = new BufferedReader(new InputStreamReader(
						fbConnection.getInputStream()));
				String inputLine;
				b = new StringBuffer();
				while ((inputLine = in.readLine()) != null)
					b.append(inputLine + "\n");
				in.close();
			} catch (IOException e) {
				e.printStackTrace();
				throw new RuntimeException("Unable to connect with Facebook "
						+ e);
			}

			accessToken = b.toString();
			if (accessToken.startsWith("{")) {
				throw new RuntimeException("ERROR: Access Token Invalid: "
						+ accessToken);
			}
		}
		return accessToken;
	}
}</pre>

			BufferedReader in = new BufferedReader(new InputStreamReader(
					c.getInputStream()));
			String inputLine;
			StringBuffer b = new StringBuffer();
			while ((inputLine = in.readLine()) != null)
				b.append(inputLine + "\n");
			in.close();

		} catch (Exception e) {
			e.printStackTrace();
