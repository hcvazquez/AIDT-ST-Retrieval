                     <div class="field-items">
                      <div class="field-item even" property="content:encoded">
                       <p>The standard Java date parser doesn't recognize the yyyymmdd format.</p> 
                       <p>You can try something like</p> 
                       <p>&lt;CODE&gt;new java.util.Date(-1900+d/10000, -1+(d%10000)/100 ,d%100); &lt;/CODE&gt;</p> 
                       <p>That's a deprecated constructor but you can check the Java docs for a replacement.</p> 
                       <p>No offense, but I would suggest database redesign. Having the date stored as an integer in this format means you can't do date arithmetic (i.e., subtract two dates).</p>
                      </div>
