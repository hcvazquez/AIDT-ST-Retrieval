         <h6 id="TriggeringwebservicefromOracleDatabasewithUTL_HTTPsuppliedpackage-Scenario"><span style="color: rgb(0,0,0);">&nbsp;</span><span style="color: rgb(0,0,0);">Scenario</span></h6> 
         <p>Triggering webservice from Oracle Database with UTL_HTTP supplied package.(This package is available from Oracle 9i onwards)</p> 
         <h6 id="TriggeringwebservicefromOracleDatabasewithUTL_HTTPsuppliedpackage-Introduction"><span style="color: rgb(0,0,0);">Introduction</span></h6> 
         <p>This document is applicable to the Oracle DB version which supports the web service. This document deals with how to use the UTL_HTTP supplied package to communicate directly with the Currency conversion WebService (<a class="external-link" href="http://www.webservicex.com/CurrencyConvertor.asmx" rel="nofollow">http://www.webservicex.com/CurrencyConvertor.asmx</a> ) and send and retrieve entire SOAP messages.</p> 
         <p>The webservice will return a response for the current available conversion rate from 'USD' to 'INR' .</p> 
         <p>You can have this approach if you want to have a trigger from the Oracle DB and doesn't want to use the JDBC adapters which will be working on the polling the database day in day out.</p> 
         <div class="code panel pdl conf-macro output-block" data-hasbody="true" data-macro-name="code" style="border-width: 1px;">
          <div class="codeContent panelContent pdl"> 
           <pre class="syntaxhighlighter-pre" data-syntaxhighlighter-params="brush: java; gutter: false; theme: Default" data-theme="Default">
PL/SQL procedure Code
&nbsp;
--To Keep server output parameter on
set serveroutput on;--declaration
&nbsp; declare
&nbsp;
&nbsp; soap_request varchar2(30000);
&nbsp; soap_respond varchar2(30000);
&nbsp; http_req utl_http.req;
&nbsp; http_resp utl_http.resp;
&nbsp;
--PL/SQL procedure
&nbsp; begin--Webservice request schema
&nbsp; soap_request:= '&lt;?xml version="1.0" encoding="utf-8"?&gt;
&nbsp; &lt;soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"&nbsp;&nbsp; xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:tns="http://www.webserviceX.NET/"&gt;
&nbsp; &lt;soap:Body&gt;
&nbsp;&nbsp;&nbsp; &lt;tns:ConversionRate&gt;
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &lt;tns:FromCurrency&gt;USD&lt;/tns:FromCurrency&gt;
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &lt;tns:ToCurrency&gt;INR&lt;/tns:ToCurrency&gt;
&nbsp;&nbsp;&nbsp; &lt;/tns:ConversionRate&gt;
&nbsp;&nbsp; &lt;/soap:Body&gt;
&nbsp; &lt;/soap:Envelope&gt;';-- Set proxy details if no direct net connection.
&nbsp; --HTTP request
&nbsp; http_req:= utl_http.begin_request( 'http://209.162.186.60:80/CurrencyConvertor.asmx' , 'POST' , 'HTTP/1.1');--set authentication
&nbsp; &nbsp; utl_http.set_header(http_req, 'Content-Type', 'text/xml; charset=utf-8');
&nbsp; utl_http.set_header(http_req, 'Content-Length', length(soap_request));
&nbsp; utl_http.set_header(http_req, 'SOAPAction', 'http://www.webserviceX.NET/ConversionRate');
&nbsp; utl_http.write_text(http_req, soap_request);
&nbsp; http_resp:= utl_http.get_response(http_req);
&nbsp; utl_http.read_text(http_resp, soap_respond);
&nbsp; utl_http.end_response(http_resp);
&nbsp; dbms_output.put_line(soap_respond);&nbsp; end;
/
</pre> 
          </div>
         </div> 
         <p><strong>If the communication is through a proxy server</strong></p> 
         <p>1-add the additional code below line<br> --Set proxy details if no direct net connection.<br> --set authentication</p> 
         <div class="code panel pdl conf-macro output-block" data-hasbody="true" data-macro-name="code" style="border-width: 1px;">
          <div class="codeContent panelContent pdl"> 
           <pre class="syntaxhighlighter-pre" data-syntaxhighlighter-params="brush: java; gutter: false; theme: Default" data-theme="Default">
-- Set proxy details if no direct net connection.
&nbsp;utl_http.set_proxy('proxyserver:port');
&nbsp; -- set authentication
&nbsp; utl_http.set_authentication(http_req, 'username',&nbsp; 'password',&nbsp; 'Basic', TRUE);&nbsp;
</pre> 
          </div>
         </div> 
         <p>&nbsp;Reaponse from Webservice (<a class="external-link" href="http://www.webservicex.com/CurrencyConvertor.asmx" rel="nofollow">http://www.webservicex.com/CurrencyConvertor.asmx</a>) <span class="confluence-embedded-file-wrapper image-left-wrapper"><img class="confluence-embedded-image image-left" src="/wiki/download/attachments/66945402/clip_image002.jpg?version=1&amp;modificationDate=1228992315000&amp;api=v2" data-image-src="/wiki/download/attachments/66945402/clip_image002.jpg?version=1&amp;modificationDate=1228992315000&amp;api=v2" data-unresolved-comment-count="0" data-linked-resource-id="67567642" data-linked-resource-version="1" data-linked-resource-type="attachment" data-linked-resource-default-alias="clip_image002.jpg" data-base-url="https://wiki.scn.sap.com/wiki" data-linked-resource-content-type="image/pjpeg" data-linked-resource-container-id="66945402" data-linked-resource-container-version="8"></span><br class="atl-forced-newline"> <br class="atl-forced-newline"> The SAP XI exposed webservice can be triggered using this utl_http pakage.<br> Hope this will be useful for the cummunity.</p>
