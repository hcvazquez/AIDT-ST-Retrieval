        <h1>Java Communications API</h1> 
        <!-- content goes here --> 
        <p><br>The <a target="" href="http://download.oracle.com/docs/cd/E17802_01/products/products/javacomm/reference/api/javax/comm/package-summary.html">Java Communications 3.0 API</a> is a Java extension that facilitates developing platform-independent communications applications for technologies such as Smart Cards, embedded systems, and point-of-sale devices, financial services devices, fax, modems, display terminals, and robotic equipment.</p> 
        <p>The Java Communications API (also known as javax.comm) provides applications access to RS-232 hardware (serial ports) and limited access to IEEE-1284 (parallel ports), SPP mode.<span class="sp10">&nbsp;</span></p> 
        <p>Implementations of the API are currently available for Solaris SPARC, Solaris x86, and Linux x86. Each of Oracle's available implementations works with the <a href="/technetwork/server-storage/sunrayproducts/overview/index.html" target=""> <strong>Sun Ray</strong> </a> thin client product line, and include portmapping extensions to allow an administer to specify the locations of ports as well as their visibility, names, and in some cases annotated reference.&nbsp;</p> 
        <p><span class="sp10">&nbsp;</span><br> <strong class="dkgrey">API serial features:</strong></p>
        <div>
         <ul> 
          <li>Enumeration of ports (administrator and user configurable port mapping)</li> 
          <li>Port configuration (baud rate, speed, stop bits, parity)</li> 
          <li>Access to EIA232 standard DTR, CD, CTS, RTS and DSR signals</li> 
          <li>Transfer of data over RS-232 ports</li> 
          <li>Hardware and software flow-control options</li> 
          <li>Receive-buffer threshold control</li> 
          <li>Asynchronous event option for notification of: 
           <ul> 
            <li>Data available on an RS-232 port</li> 
            <li>Port hardware line level changes</li> 
            <li>Port ownership changes within a single JVM</li> 
           </ul></li> 
         </ul>
        </div> 
        <p><span class="sp10">&nbsp;</span></p>
        <div>
         <p><em>Java communications API was initially released in 1997, prior to the Java Community Process, was 'grandfathered in' to the JCP (<a target="_blank" href="http://www.jcp.org">http://www.jcp.org</a>) without having an associated JSR (Java Specification Request). </em><a href="http://www.oracle.com/technetwork/indexes/downloads/sdlc-decommission-333274.html"><br></a></p>
        </div>
