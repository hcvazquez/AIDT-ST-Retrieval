       <h2 id="intro" class="step-title intro-title">Introduction: Text to Speech Batch Program</h2>
       <div class="photoset">
        <img alt="Picture of Text to Speech Batch Program" src="https://cdn.instructables.com/FJO/D3WE/H9G17C56/FJOD3WEH9G17C56.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>As I promised earlier, I have made this instructable. This is part of a set of instructables I am making that focuses around one of the original features in my <a href="https://www.instructables.com/id/Creating-a-mega-menu-with-DOSbatch-files/">Mega_Menu V 2.4</a> batch program. I have one other, previously made instructable on my address book program which you can find <a href="https://www.instructables.com/id/Batch-File-Address-Book-V-1/">here</a>.&nbsp;<br><br> This program has one major function only, making your computer "speak" whatever you write in the program. This is accomplished by creating a&nbsp;separate&nbsp;(temporary) VBs file that makes SAPI speak whatever you write. This file is deleted afterwards.&nbsp;<br><br> Onto the code:<br><br> Note: I made a mistake, don't use underscores, use hyphens. I have corrected this.</p>
       </div>
      </section>
      <section class="step-container">
       <h2 id="step1" class="step-title">Step 1: </h2>
       <div class="photoset">
        <img alt="Picture of " src="https://cdn.instructables.com/FCF/SSUW/H9G17C5K/FCFSSUWH9G17C5K.MEDIUM.jpg">
       </div>
       <div class="step-body">
        <p>@echo off<br> title Text to Speech Conversion<br> color 0a<br><br> rem The user decides what to convert here<br> :input<br> cls<br> echo What do you want the computer to convert into speech?<br> echo.<br> set /p text=<br><br> rem Making the temp file<br> :num<br> set num=%random%<br> if exist temp%num%.vbs goto num<br> echo ' &gt; "temp%num%.vbs"<br> echo set speech = Wscript.CreateObject("SAPI.spVoice") &gt;&gt; "temp%num%.vbs"<br> echo speech.speak "%text%" &gt;&gt; "temp%num%.vbs"<br> start temp%num%.vbs<br> pause<br> del temp%num%.vbs<br> goto input<br><br><br> If you do not want the screen to close for no reason, I suggest you follow the tip given and don't use spaces; use hyphens (-) instead.</p>
       </div>
      </section>
