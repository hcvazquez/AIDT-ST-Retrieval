     <h2><a id="user-content-installation" class="deep-link" href="#installation">
       <svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" width="16">
        <path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
       </svg></a>Installation</h2> 
     <p>There is a hard dependency on the <a href="https://code.google.com/p/tesseract-ocr/">Tesseract project</a>. You can find installation instructions for various platforms on the project site. For Homebrew users, the installation is quick and easy.</p> 
     <pre><code>brew install tesseract --all-languages
</code></pre> 
     <p>The above will install all of the language packages available, if you don't need them all you can remove the <code>--all-languages</code> flag and install them manually, by downloading them to your local machine and then exposing the <code>TESSDATA_PREFIX</code> variable into your path:</p> 
     <pre><code>export TESSDATA_PREFIX=~/Downloads/
</code></pre> 
     <p>You can then go about installing the node-module to expose the JavaScript API:</p> 
     <pre><code>npm install node-tesseract
</code></pre>
