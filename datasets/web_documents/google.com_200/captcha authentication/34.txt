        <li><code>beforeUpload</code> (Object): Request parameters for 1st API-call (getting upload url). See <a href="https://vk.com/dev/upload_files">vk.com/dev/upload_files</a></li> 
        <li><code>afterUpload</code> (Object): Request parameters for 2nd API-call (saving file). As example, you can specify <code>artist</code> and <code>title</code> params to save audiofile with certain title and artist. (See: <a href="https://vk.com/dev/audio.save">vk.com/dev/audio.save</a>)</li> 
       </ul> </li> 
     </ul>
