   <h1 class="center">Java Algorithms and Clients</h1> 
   <p><b>Design goals.</b> Our original goal for this book was to cover the <em>50 algorithms that every programmer should know</em>. We use the word <em>programmer</em> to refer to anyone engaged in trying to accomplish something with the help of a computer, including scientists, engineers, and applications developers, not to mention college students in science, engineering, and computer science. The code is optimized for clarity, portability, and efficiency. While some of our implementations are as fast as their counterparts in <tt>java.util</tt>, our primary goal is to express the core algorithmic idea in an elegant and efficient manner. 
    <!-- For example, we implement left-leaning red–black trees (instead of classic red–black trees)
and we often use recursive implementations (instead of iterative ones). --> While we embrace some advanced Java features (such as generics and iterators), we avoid others that would interfere with the exposition (such as inheritance and concurrency). </p>
   <p><b>Algorithms and clients in the textbook.</b> The list below includes nearly 200 Java programs (some are clients, some others are basic infrastructure). Click on the program name to access the Java code; click on the description to access the <a href="javadoc">javadoc</a>; click on the data file names to access the data. You can download all of the programs as <a href="algs4.jar">algs4.jar</a> and the data as <a href="algs4-data.zip">algs4-data.zip</a>.

   <h2>Installing Java.</h2> Here are instructions for installing a Java programming environment on your operating system: 
   <a href="../mac">Mac OS X</a>, 
   <a href="../windows">Windows</a>, or 
   <a href="../linux">Linux</a>. 
   <p> </p>
   <h2><a name="classpath"></a>Installing the textbook libraries.</h2> To use the textbook libraries, download 
   <a href="algs4.jar">algs4.jar</a> and it to the classpath. Do not unjar it. Here is how to accomplish that in a variety of environments: 
   <ul> 
    <p></p>
    <li><em>Mac OS X (automatic).</em> The <a href="http://algs4.cs.princeton.edu/mac">Mac OS X installer</a> downloads <a href="algs4.jar">algs4.jar</a> to the <tt>/usr/local/algs4</tt> folder; adds it to the DrJava classpath; and provides the wrapper scripts <tt>javac-algs4</tt> and <tt>java-algs4</tt>, which classpath in <tt>algs4.jar</tt>, for use in the Terminal. <p></p></li>
    <li><em>Windows (automatic).</em> The <a href="http://algs4.cs.princeton.edu/windows">Windows installer</a> downloads <a href="algs4.jar">algs4.jar</a> to the <tt>C:\Users\username\algs4</tt> folder; adds it the DrJava classpath; and provides the wrapper scripts <tt>javac-algs4</tt> and <tt>java-algs4</tt>, which classpath in <tt>algs4.jar</tt>, for use from the Command Prompt. <p></p></li>
    <li><em>Windows Command Prompt (manual).</em> Download <a href="algs4.jar">algs4.jar</a> to a folder, say <tt>C:\Users\username\algs4</tt>. Next, add <tt>algs4.jar</tt> to the <tt>CLASSPATH</tt> environment variable. 
     <ul> 
      <p></p>
      <li> <p> Windows 7: <em>Start -&gt; Computer -&gt; System Properties -&gt; Advanced system settings -&gt; Environment Variables -&gt; User variables -&gt; CLASSPATH</em>. </p><p> Vista: <em>Start -&gt; My Computer -&gt; Properties -&gt; Advanced -&gt; Environment Variables -&gt; User variables -&gt; CLASSPATH</em>. 
        <!--
<p>
Windows XP: <em>Start -> Control Panel -> System -> Advanced -> Environment Variables -> User variables -> CLASSPATH</em>. 
--> </p><p></p></li>
      <li>Prepend the following to the <em>beginning</em> of the CLASSPATH variable: 
       <blockquote> 
        <table> 
         <tbody>
          <tr>
           <td><pre>
C:\Users\username\algs4\algs4.jar;
</pre></td>
          </tr> 
         </tbody>
        </table> 
       </blockquote> <p> The semicolons separate entries in the CLASSPATH. </p><p></p></li>
      <li>Click OK three times. </li>
     </ul> <p> If you don't see a variable named CLASSPATH, click <em>New</em> and in the popup window enter CLASSPATH for the variable name. Then, perform the instructions above. </p><p></p></li>
    <li><em>Windows Command Prompt (heavy handed).</em> Download <a href="algs4.jar">algs4.jar</a> and put it in the directory <tt>%SystemRoot%\Sun\Java\lib\ext\</tt>. <p></p></li>
    <li><em>Mac OS X Terminal (manual).</em> Download <a href="algs4.jar">algs4.jar</a> to a folder, say <tt>~/algs4</tt>. Depending on your shell, add the following line or lines to the file specified: 
     <ul> 
      <p></p>
      <li><em>Bourne-again shell (bash)</em>. Add the following line to the file <tt>~/.bash_profile</tt> (if it exists); otherwise add it to the file <tt>~/.bash_login</tt> (if it exists); otherwise, add it to the file <tt>~/.profile</tt> (if it doesn't exist, create it first): 
       <blockquote> 
        <table> 
         <tbody>
          <tr>
           <td><pre>

</pre></td>
          </tr> 
         </tbody>
        </table> 
       </blockquote> <p> The colons separate entries in the CLASSPATH. </p><p></p></li>
      <li><em>C shell (csh)</em>. Add the following line to the file <tt>~/.cshrc</tt> (if it doesn't exist, create it first):

if ( !($?CLASSPATH) ) then

endif
