       <h2 id="overview"><strong>1. Overview</strong></h2> 
       <p class="lead">In this quick tutorial, we’re going to show how to <strong>convert a <em>File</em> to an <em>InputStream</em> </strong>– first using plain Java and then Guava and the Apache Commons IO library.</p> 
       <p>This article is part of <a title="The Java Guide on IO and Collections" href="/java-tutorial">the “<strong>Java – Back to Basic</strong>” series</a> here on Baeldung.</p>

       <h2 id="java"><strong>2. Convert using Java</strong></h2> 
       <p>Let’s start with <strong>the simple Java solution</strong> – using a <em>FileInputStream</em>:</p> 
       <pre class="brush: java; gutter: true">@Test
public void givenUsingPlainJava_whenConvertingFileToInputStream_thenCorrect() 
  throws IOException {
    File initialFile = new File("src/main/resources/sample.txt");
    InputStream targetStream = new FileInputStream(initialFile);
}</pre> 
       <p>Note that we’re not closing the resulting stream in these examples.</p> 
       <h2 id="guava"><strong>3. Convert using Guava</strong></h2> 
       <p>Next – let’s see <strong>the Guava solution</strong>, using an intermediary <em>ByteSource</em>:</p> 
       <pre class="brush: java; gutter: true">@Test
public void givenUsingGuava_whenConvertingFileToInputStream_thenCorrect() 
  throws IOException {
    File initialFile = new File("src/main/resources/sample.txt");
    InputStream targetStream = Files.asByteSource(initialFile).openStream();
}</pre> 
       <h2 id="commons-io"><strong>4. Convert using Commons IO</strong></h2> 
       <p>Finally – let’s look at a solution using Apache Commons IO:</p> 
       <pre class="brush: java; gutter: true">@Test
public void givenUsingCommonsIO_whenConvertingFileToInputStream_thenCorrect() 
  throws IOException {
    File initialFile = new File("src/main/resources/sample.txt");
    InputStream targetStream = FileUtils.openInputStream(initialFile);
}</pre> 
       <p>And there you have it – <strong>3 simple and clean solutions for opening a stream from a Java file</strong>.</p>
