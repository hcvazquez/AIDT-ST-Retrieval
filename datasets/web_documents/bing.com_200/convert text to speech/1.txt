        <h2>Convert Text to Speech using FreeTTS API</h2> 
        <p>Hello everyone, I am attaching simple program about <strong>Convert Text to Speech using FreeTTS API</strong>. The programs are shard in the same post.</p> 
        <p>To convert text to speech, I am using the third party library called <strong>FreeTTS 1.2.2</strong>. Which you can download from the following link. <a href="https://sourceforge.net/projects/freetts/files/" target="_blank">Download Here</a></p> 
        <p>After downloading the <strong>FreeTTS API</strong> extract the zip file and navigate to lib folder, then copy and paste it into your eclipse project.</p>

	public static void main(String[] args) {

		Scanner inputScanner = new Scanner(System.in);
		
		TextToSpeechConvertor ttsc = new TextToSpeechConvertor();

		System.out.println("Enter the Text : (type 'exit' to terminate)");

		// Reading the text
		String inputText = inputScanner.nextLine();

		while (true) {

			if("exit".equalsIgnoreCase(inputText)) {
				
				inputText = "Bye, See you later";
				ttsc.speak(inputText);
				break;
			}
			
			ttsc.speak(inputText);
			
			System.out.println("Enter the Text : (type 'exit' to terminate)");
			inputText = inputScanner.nextLine();
		}

		inputScanner.close();
		
	}
