        <div class="at-above-post addthis_tool" data-url="https://www.javabrahman.com/java-8/java-8-java-util-function-consumer-tutorial-with-examples/"></div>
        <p><span class="jb-font-17-subtitle">Tutorial explains the in-built functional interface <code class="nl-code-17">Consumer&lt;T&gt;</code> introduced in Java 8. It uses examples to show how the <code class="nl-code-17">accept()</code> &amp; <code class="nl-code-17">andThen()</code> methods of the Consumer interface are to be used.</span></p> 
        <p><span class="jb-highlight-lightblue-bold-18">What is java.util.function.Consumer</span><br> <code class="nl-code-intext">Consumer&lt;T&gt;</code> is an in-built functional interface introduced in Java 8 in the <code class="nl-code-intext">java.util.function</code> package. Consumer can be used in all contexts where an object needs to be consumed,i.e. taken as input, and some operation is to be performed on the object without returning any result. Common example of such an operation is <em>printing</em> where an object is taken as input to the printing function and the value of the object is printed( we will expand upon the printing example in more detail below when understanding how to use Consumer interface).</p> 
        <p>Since Consumer is a functional interface, hence it can be used as the assignment target for a lambda expression or a method reference.<br> In case you are new to functional programming then you might want to go through functional interfaces and lambda expressions tutorials before proceeding ahead – <a id="internal-link" href="https://www.javabrahman.com/java-8/functional-interfaces-java-8/">functional interfaces tutorial<span id="tooltip-span">Click to Read tutorial on Functional Interfaces</span></a>&nbsp;,&nbsp;<a id="internal-link" href="https://www.javabrahman.com/java-8/lambda-expressions-java-8-explained-examples/">lambda expressions tutorial<span id="tooltip-span">Click to read Lambda Expressions tutorial</span></a>&nbsp;.</p> 
        <p><span class="jb-highlight-lightblue-bold-17">Function Descriptor of Consumer&lt;T&gt;</span>:Consumer’s Function Descriptor is <span class="jb-highlight-lightgrey"><code class="nl-code-intext">T -&gt; ()</code></span>. This means an object of type T is input to the lambda with no return value. To understand Function Descriptors in details you can refer the <a id="internal-link" href="https://www.javabrahman.com/java-8/function-descriptors-java-8-explained/">function descriptor tutorial<br> <span id="tooltip-span">Tutorial explaining function descriptors</span></a>.</p>
        <div class="code-block code-block-3" style="text-align: center; margin: -8px auto 18px;"> 
         <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
         <!-- 4th Para - Desktop - Responsive --> 
         <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-4674758778510710" data-ad-slot="1076628980" data-ad-format="auto"></ins>

        <p><span class="jb-highlight-lightblue-bold-17">Advantage of predefined java.util.function.Consumer</span>:In all scenarios where an object is to be taken as input and an operation performed on it, the in-built functional interface <code class="nl-code-intext">Consumer&lt;T&gt;</code> can be used without the need to define a new functional interface every time.</p> 
        <div class="su-box su-box-style-soft for-shortcodebox" style="border-color:#ca9536;border-radius:4px;-moz-border-radius:4px;-webkit-border-radius:4px;">
         <div class="su-box-title" style="background-color:#fcba43;color:#00000;-webkit-border-top-left-radius:3px;-webkit-border-top-right-radius:3px;-moz-border-radius-topleft:3px;-moz-border-radius-topright:3px;border-top-left-radius:3px;border-top-right-radius:3px;">
          java.util.function.Consumer source code
         </div>
         <div class="su-box-content su-clearfix"> 
          <pre class="brush: java; title: ; notranslate" title="">@FunctionalInterface
public interface Consumer&lt;T&gt; {
    void accept(T t);
    default Consumer&lt;T&gt; andThen(Consumer&lt;? super T&gt; after) {
        Objects.requireNonNull(after);
        return (T t) -&gt; { accept(t); after.accept(t); };
    }
}</pre> 
         </div>
        </div> 
        <span class="jb-highlight-lightblue-bold-17">Salient Points regarding <code class="nl-code-17">Consumer&lt;T&gt;</code>’s source code </span>:
        <p></p> 
        <ul id="code-explanation-black" style="padding-left:12px;"> 
         <li><code>Consumer </code>has been defined with the generic type <code>T</code> which is the same type which its <code>accept()</code> &amp; <code>andThen()</code> methods take as input.</li> 
         <li><code>accept()</code> method is the primary abstract method of the Consumer functional interface. Its function descriptor being <span class="jb-highlight-lightgrey"><code>T -&gt; ()</code></span>. I.e. <code>accept()</code> method takes as input the type T and returns no value. I will explain usage of <code>accept()</code> with detailed example in the next section.</li> 
         <li>All lambda definitions for Consumer must be written in accordance with accept method’s signature, and conversely all lambdas with the same signature as that of <code>accept()</code> are candidates for assignment to an instance of Consumer interface.</li> 
         <li><code>andThen()</code> is a <a id="internal-link" href="https://www.javabrahman.com/java-8/default-methods-in-java-8-with-examples/">default method<br> <span id="tooltip-span">Click to Read Detailed Tutorial on Default Methods in Java 8</span></a> in Consumer interface. Method <code>andThen()</code>, when applied on a Consumer interface, takes as input another instance of Consumer interface and returns as a result a <em>new </em>consumer interface which represents <em>aggregation </em>of both of the operations defined in the two Consumer interfaces. I will explain the usage of <code>andThen()</code> with a detailed example in coming sections. </li> 
        </ul> 
        <p><span class="jb-highlight-lightblue-bold-18">Usage of accept() method of Consumer</span>:<br> To understand the <code class="nl-code-intext">accept()</code> method lets take a look at the example below where I take a list of integers and print them using a method <code class="nl-code-intext">printList()</code>. The <code class="nl-code-intext">printList()</code> method takes 2 inputs- an instance of Consumer interface which contains the printing logic <em>and </em>the list which is to be printed. Lets have a look at the code now, post which I will explain in detail how this code works-</p>
        <div class="su-box su-box-style-soft for-shortcodebox" style="border-color:#ca9536;border-radius:4px;-moz-border-radius:4px;-webkit-border-radius:4px;">
         <div class="su-box-title" style="background-color:#fcba43;color:#00000;-webkit-border-top-left-radius:3px;-webkit-border-top-right-radius:3px;-moz-border-radius-topleft:3px;-moz-border-radius-topright:3px;border-top-left-radius:3px;border-top-right-radius:3px;">
          Code showing usage of Consumer.accept() method
         </div>
         <div class="su-box-content su-clearfix"> 
          <pre class="brush: java; title: ; notranslate" title="">package com.javabrahman.java8;
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;
import java.util.function.Consumer;
public class ConsumerFunctionExample{
  public static void main(String args[]){
    Consumer&lt;Integer&gt; consumer= i-&gt; System.out.print(" "+i);
    List&lt;Integer&gt; integerList=Arrays.asList(new Integer(1), 
                              new Integer(10), new Integer(200),
                              new Integer(101), new Integer(-10),
                              new Integer(0));
    printList(integerList, consumer);
 }
 public static void printList(List&lt;Integer&gt; listOfIntegers, Consumer&lt;Integer&gt; consumer){
  for(Integer integer:listOfIntegers){
    consumer.accept(integer);
  }
 }
}</pre> 
         </div>
        </div> 
        <strong>&nbsp;OUTPUT on executing the above code</strong>
        <div class="su-note" style="border-color:#171717;border-radius:3px;-moz-border-radius:3px;-webkit-border-radius:3px;">
         <div class="su-note-inner su-clearfix" style="background-color:#1a1a1a;border-color:#d1d1d1;color:#DAD9D7;border-radius:3px;-moz-border-radius:3px;-webkit-border-radius:3px;">
           &nbsp;1&nbsp; 10&nbsp; 200&nbsp; 101&nbsp; -10&nbsp; 0
         </div>
        </div> 
        <span class="jb-highlight-lightblue-bold-17">Explanation of above example’s Code &amp; Output</span>
        <p></p> 
        <ul id="code-explanation-black" style="padding-left:12px;"> 
         <li><code>ConsumerFunctionExample </code>is my class with 2 methods – <code>main()</code> &amp; <code>printList()</code>.</li> 
         <li>In <code>main()</code> method first an instance of Consumer&lt;Integer&gt; is defined(named <code>consumer</code>) using a <a id="internal-link" href="https://www.javabrahman.com/java-8/lambda-expressions-java-8-explained-examples/">lambda expression<span id="tooltip-span">Link to Lambda Expressions Tutorial</span></a> which takes input as an object of type Integer and contains logic to print the value of that Integer.</li> 
         <li>Next the <code>main()</code> method defines a new list of integers and passes it to the <code>printList()</code> method along with the consumer object defined earlier which contains the printing logic.</li> 
         <li>The <code>printList()</code> method iterates through the list of integers and invokes the <code>accept()</code> method of the consumer object for every integer in the list.</li> 
         <li>The <code>accept()</code> method which works as per the lambda definition assigned to the consumer interface, i.e. <span class="jb-highlight-lightgrey"><code>i-&gt; System.out.print(" "+i)</code></span> , prints out the value of each integer with a single space character prepended to it. Thus giving the output <span class="jb-highlight-lightgrey-bold"> 1 10 200 101 -10 0</span> .</li> 
        </ul> 
        <p><span class="jb-highlight-lightblue-bold-18">Usage of default method andThen() of Consumer</span><br> To understand the <code class="nl-code-intext">andThen()</code> default method of Consumer interface, I have taken the same code as that used for example of <code class="nl-code-intext">accept()</code> method above and changed it a bit to show <code class="nl-code-intext">andThen()</code> works.<br> </p>
        <div class="su-box su-box-style-soft for-shortcodebox" style="border-color:#ca9536;border-radius:4px;-moz-border-radius:4px;-webkit-border-radius:4px;">
         <div class="su-box-title" style="background-color:#fcba43;color:#00000;-webkit-border-top-left-radius:3px;-webkit-border-top-right-radius:3px;-moz-border-radius-topleft:3px;-moz-border-radius-topright:3px;border-top-left-radius:3px;border-top-right-radius:3px;">
          Code showing usage of default method Consumer.andThen()
         </div>
         <div class="su-box-content su-clearfix"> 
          <pre class="brush: java; title: ; notranslate" title="">//import statements are same as in previous example; hence skipped
public class ConsumerFunctionExample{
 public static void main(String args[]){
  Consumer&lt;Integer&gt; consumer= i-&gt; System.out.print(" "+i);
  Consumer&lt;Integer&gt; consumerWithAndThen = consumer.andThen( i-&gt; System.out.print("(printed "+i+")"));
  List&lt;Integer&gt; integerList=Arrays.asList(new Integer(1),
                            new Integer(10), new Integer(200),
                            new Integer(101), new Integer(-10),
                            new Integer(0));
  printList(integerList,consumerWithAndThen);
 }
 public static void printList(List&lt;Integer&gt; listOfIntegers, Consumer&lt;Integer&gt; consumer){
  for(Integer integer:listOfIntegers){
   consumer.accept(integer);
  }
 }
}</pre> 
         </div>
        </div> 
        <strong>&nbsp;Output on executing the above code</strong>
        <div class="su-note" style="border-color:#171717;border-radius:3px;-moz-border-radius:3px;-webkit-border-radius:3px;">
         <div class="su-note-inner su-clearfix" style="background-color:#1a1a1a;border-color:#d1d1d1;color:#DAD9D7;border-radius:3px;-moz-border-radius:3px;-webkit-border-radius:3px;">
           1(printed&nbsp; 1)&nbsp; 10(printed&nbsp; 10)&nbsp; 200(printed&nbsp; 200)&nbsp; 101(printed&nbsp; 101)&nbsp; -10(printed&nbsp; -10)&nbsp; 0(printed&nbsp; 0)
         </div>
        </div> 
        <span class="jb-highlight-lightblue-bold-17">Explanation of the code</span>
        <p></p> 
        <ul id="code-explanation-black" style="padding-left:12px;"> 
         <li>A new instance of Consumer interface is defined which is assigned a value of the previous consumer interface(which prints just the space pepended integer) <em>aggregated </em>with the new lambda (which prints <span class="jb-highlight-lightgrey-bold">(printed i)</span> where i is replaced by the value of the integer passed as input to the lambda).</li> 
         <li>We thus have two Consumer Interface based operations aggregated – first one prints the integer value and the second one prints <em>(print i)</em> where i is the value of the integer passed.</li> 
         <li>The consumer interface instance representing aggregated consumer operations is passed as parameter to the <code>printList()</code> method.</li> 
         <li>The <code>printList()</code> method is still the same as previous example i.e. it simply calls the <code>accept()</code> method on the consumer interface passed to it while iterating through the list of integers passed to it.</li> 
         <li>While in the previous example the consumer interface passed to it contained a single operation which simply printed the integer value prepended with a single space character; this time the consumer interface does two operations which have been aggregated together for each value – printing the integer value with space prepended and then printing the string (printed i) where is is the integer passed to it. The output of this example is thus –<span class="jb-highlight-lightgrey-bold"> 1(printed&nbsp; 1)&nbsp; 10(printed&nbsp; 10)&nbsp; 200(printed&nbsp; 200)&nbsp; 101(printed&nbsp; 101)&nbsp; -10(printed&nbsp; -10)&nbsp; 0(printed&nbsp; 0)</span></li> 
        </ul> 
        <p><span class="jb-highlight-lightblue-bold-18">Summary</span><br> In this tutorial we looked at what is the Consumer&lt;T&gt; in-built interface defined in Java 8 and what are its advantages. We then looked at how to use the Consumer interface using its <code class="nl-code-intext">accept()</code> method &amp; the <code class="nl-code-intext">andThen()</code> default method.<br> </p>
