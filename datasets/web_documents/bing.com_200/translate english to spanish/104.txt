      <p><b>Google Translate</b> is a free multilingual machine translation service developed by <a href="/wiki/Google" title="Google">Google</a>, to translate text, speech, images, sites, or real-time video from one language into another. It offers a <a href="/wiki/Web_application" title="Web application">website interface</a>, <a href="/wiki/Mobile_app" title="Mobile app">mobile apps</a> for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> and <a href="/wiki/IOS" title="IOS">iOS</a>, and an <a href="/wiki/Application_programming_interface" title="Application programming interface">API</a> that helps developers build <a href="/wiki/Browser_extension" title="Browser extension">browser extensions</a> and software applications. Google Translate supports over 100 languages at various levels and as of May 2013, serves over 200 million people daily.</p> 
      <p>Launched in April 2006 as a <a href="/wiki/Statistical_machine_translation" title="Statistical machine translation">statistical machine translation</a> service, it used <a href="/wiki/United_Nations" title="United Nations">United Nations</a> and <a href="/wiki/European_Parliament" title="European Parliament">European Parliament</a> transcripts to gather linguistic data. Rather than translating languages directly, it first translates text to English and then to the target language. During a translation, it looks for patterns in millions of documents to help decide on the best translation. Its accuracy has been criticized and ridiculed on several occasions. In November 2016, Google announced that Google Translate would switch to a <a href="/wiki/Neural_machine_translation" title="Neural machine translation">neural machine translation</a> engine - <a href="/wiki/Google_Neural_Machine_Translation" title="Google Neural Machine Translation">Google Neural Machine Translation</a> (GNMT) - which translates "whole sentences at a time, rather than just piece by piece. It uses this broader context to help it figure out the most relevant translation, which it then rearranges and adjusts to be more like a human speaking with proper grammar". Originally only enabled for a few languages in 2016, GNMT is gradually being used for more languages.</p>

      <h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=1" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Google Translate can translate multiple forms of text and media, including text, speech, images, sites, or real-time video, from one language to another.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> It supports over 100 languages at various levels<sup id="cite_ref-supportedlanguages_5-0" class="reference"><a href="#cite_note-supportedlanguages-5">[5]</a></sup> and as of May 2013, serves over 200 million people daily.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup> For some languages, Google Translate can pronounce translated text,<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup> highlight corresponding words and phrases in the source and target text, and act as a simple dictionary for single-word input. If "Detect language" is selected, text in an unknown language can be automatically identified. If a user enters a <a href="/wiki/Uniform_resource_locator" class="mw-redirect" title="Uniform resource locator">URL</a> in the source text, Google Translate will produce a <a href="/wiki/Hyperlink" title="Hyperlink">hyperlink</a> to a machine translation of the website.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> Users can save translations in a "phrasebook" for later use.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup> For some languages, text can be entered via an <a href="/wiki/Virtual_keyboard" title="Virtual keyboard">on-screen keyboard</a>, through <a href="/wiki/Handwriting_recognition" title="Handwriting recognition">handwriting recognition</a>, or <a href="/wiki/Speech_recognition" title="Speech recognition">speech recognition</a>.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup><sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup></p> 
      <h3><span class="mw-headline" id="Browser_integration">Browser integration</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=2" title="Edit section: Browser integration">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>Google Translate is available in some <a href="/wiki/Web_browser" title="Web browser">web browsers</a> as an optional downloadable extension that can run the translation engine.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">[12]</a></sup> In February 2010, Google Translate was integrated into the <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a> browser by default, for optional automatic webpage translation.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup></p>

      <p>The Google Translate app for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> and <a href="/wiki/IOS" title="IOS">iOS</a> supports more than 100 languages and can translate 37 languages via photo, 32 via voice in "conversation mode", and 27 via real-time video in "augmented reality mode".<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">[16]</a></sup></p> 
      <p>The Android app was released in January 2010, while an <a href="/wiki/HTML5" title="HTML5">HTML5</a> <a href="/wiki/Web_application" title="Web application">web application</a> was released for iOS users in August 2008,<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">[17]</a></sup> followed by a native app on February 8, 2011.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup></p> 
      <p>A January 2011 Android version experimented with a "Conversation Mode" that allowed users to communicate fluidly with a nearby person in another language.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">[19]</a></sup> Originally limited to English and Spanish, the feature received support for 12 new languages, still in testing, the following October.<sup id="cite_ref-20" class="reference"><a href="#cite_note-20">[20]</a></sup></p> 
      <p>In January 2015, the apps gained the ability to translate physical signs in real time using the device's camera, as a result of Google's acquisition of the <a href="/wiki/Word_Lens" title="Word Lens">Word Lens</a> app.<sup id="cite_ref-21" class="reference"><a href="#cite_note-21">[21]</a></sup><sup id="cite_ref-22" class="reference"><a href="#cite_note-22">[22]</a></sup> The original January launch only supported seven languages, but a July update added support for 20 new languages, and also enhanced the speed of Conversation Mode translations.<sup id="cite_ref-23" class="reference"><a href="#cite_note-23">[23]</a></sup><sup id="cite_ref-24" class="reference"><a href="#cite_note-24">[24]</a></sup></p> 
      <h3><span class="mw-headline" id="API">API</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=4" title="Edit section: API">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>In May 2011, Google announced that the Google Translate <a href="/wiki/Application_programming_interface" title="Application programming interface">API</a> for <a href="/wiki/Software_development" title="Software development">software developers</a> had been deprecated and would cease functioning.<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">[25]</a></sup> The Translate API page stated the reason as "substantial economic burden caused by extensive abuse" with an end date set for December 1, 2011.<sup id="cite_ref-26" class="reference"><a href="#cite_note-26">[26]</a></sup> In response to public pressure, Google announced in June 2011 that the API would continue to be available as a paid service.<sup id="cite_ref-27" class="reference"><a href="#cite_note-27">[27]</a></sup></p> 
      <p>Because the API was used in numerous third-party websites and apps, the original decision to deprecate it led some developers to criticize Google and question the viability of using Google APIs in their products.<sup id="cite_ref-28" class="reference"><a href="#cite_note-28">[28]</a></sup><sup id="cite_ref-29" class="reference"><a href="#cite_note-29">[29]</a></sup></p>

      <h2><span class="mw-headline" id="Method_of_translation">Method of translation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=6" title="Edit section: Method of translation">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>In April 2006, Google Translate launched with a statistical machine translation engine.<sup id="cite_ref-Statistical_machine_translation_1-1" class="reference"><a href="#cite_note-Statistical_machine_translation-1">[1]</a></sup></p> 
      <p>Google Translate does not apply <a href="/wiki/Grammar" title="Grammar">grammatical</a> rules, since its algorithms are based on statistical analysis rather than traditional rule-based analysis. The system's original creator, <a href="/wiki/Franz_Josef_Och" title="Franz Josef Och">Franz Josef Och</a>, has criticized the effectiveness of <a href="/wiki/Rule-based_machine_translation" title="Rule-based machine translation">rule-based</a> <a href="/wiki/Algorithm" title="Algorithm">algorithms</a> in favor of statistical approaches.<sup id="cite_ref-och_2005_30-0" class="reference"><a href="#cite_note-och_2005-30">[30]</a></sup> It is based on a method called <a href="/wiki/Statistical_machine_translation" title="Statistical machine translation">statistical machine translation</a>, and more specifically, on research by Och who won the <a href="/wiki/DARPA" title="DARPA">DARPA</a> contest for speed machine translation in 2003. Och was the head of Google's machine translation group until leaving to join Human Longevity, Inc. in July 2014.<sup id="cite_ref-31" class="reference"><a href="#cite_note-31">[31]</a></sup></p> 
      <p>According to Och, a solid base for developing a usable statistical machine translation system for a new pair of languages from scratch would consist of a bilingual <a href="/wiki/Text_corpus" title="Text corpus">text corpus</a> (or <a href="/wiki/Parallel_text" title="Parallel text">parallel collection</a>) of more than 150-200 million words, and two monolingual corpora each of more than a billion words.<sup id="cite_ref-och_2005_30-1" class="reference"><a href="#cite_note-och_2005-30">[30]</a></sup> Statistical <a href="/wiki/Mathematical_model" title="Mathematical model">models</a> from these data are then used to translate between those languages.</p> 
      <p>To acquire this huge amount of linguistic data, Google used <a href="/wiki/United_Nations" title="United Nations">United Nations</a> and <a href="/wiki/European_Parliament" title="European Parliament">European Parliament</a> transcripts.<sup id="cite_ref-32" class="reference"><a href="#cite_note-32">[32]</a></sup><sup id="cite_ref-33" class="reference"><a href="#cite_note-33">[33]</a></sup></p> 
      <p>Google Translate does not translate from one language to another (L1 ? L2). Instead, it often translates first to English and then to the target language (L1 ? EN ? L2).<sup id="cite_ref-34" class="reference"><a href="#cite_note-34">[34]</a></sup></p> 
      <p>When Google Translate generates a translation, it looks for patterns in hundreds of millions of documents to help decide on the best translation. By detecting patterns in documents that have already been translated by human translators, Google Translate makes <a href="/wiki/Artificial_intelligence" title="Artificial intelligence">intelligent guesses</a> as to what an appropriate translation should be.<sup id="cite_ref-35" class="reference"><a href="#cite_note-35">[35]</a></sup></p> 
      <p>Before October 2007, for languages other than <a href="/wiki/Arabic" title="Arabic">Arabic</a>, <a href="/wiki/Chinese_language" title="Chinese language">Chinese</a> and <a href="/wiki/Russian_language" title="Russian language">Russian</a>, Google Translate was based on <a href="/wiki/SYSTRAN" title="SYSTRAN">SYSTRAN</a>, a software engine which is still used by several other online translation services such as <a href="/wiki/Babel_Fish_(website)" title="Babel Fish (website)">Babel Fish</a> (now defunct). Since October 2007, Google Translate has used proprietary, in-house technology based on <a href="/wiki/Statistical_machine_translation" title="Statistical machine translation">statistical machine translation</a> instead.<sup id="cite_ref-36" class="reference"><a href="#cite_note-36">[36]</a></sup><sup id="cite_ref-37" class="reference"><a href="#cite_note-37">[37]</a></sup></p> 
      <h3><span class="mw-headline" id="Google_Neural_Machine_Translation">Google Neural Machine Translation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=7" title="Edit section: Google Neural Machine Translation">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>In September 2016, a research team at Google announced the development of the Google Neural Machine Translation system (GNMT) to increase fluency and accuracy in Google Translate<sup id="cite_ref-GT_Turovsky_2016_2-1" class="reference"><a href="#cite_note-GT_Turovsky_2016-2">[2]</a></sup><sup id="cite_ref-Schuster.2C_Mike_38-0" class="reference"><a href="#cite_note-Schuster.2C_Mike-38">[38]</a></sup> and in November announced that Google Translate would switch to GNMT.</p> 
      <p>Google Translate's new <a href="/wiki/Neural_machine_translation" title="Neural machine translation">neural machine translation</a> system uses a large <a href="/wiki/End-to-end_principle" title="End-to-end principle">end-to-end</a> <a href="/wiki/Artificial_neural_network" title="Artificial neural network">artificial neural network</a> capable of <a href="/wiki/Deep_learning" title="Deep learning">deep learning</a>,<sup id="cite_ref-GT_Turovsky_2016_2-2" class="reference"><a href="#cite_note-GT_Turovsky_2016-2">[2]</a></sup><sup id="cite_ref-googleblog_GNMT_2016_39-0" class="reference"><a href="#cite_note-googleblog_GNMT_2016-39">[39]</a></sup> in particular, <a href="/wiki/Long_short-term_memory" title="Long short-term memory">long short-term memory</a> networks.<sup id="cite_ref-lstm1997_40-0" class="reference"><a href="#cite_note-lstm1997-40">[40]</a></sup><sup id="cite_ref-lstm2000_41-0" class="reference"><a href="#cite_note-lstm2000-41">[41]</a></sup><sup id="cite_ref-GoogleTranslate_42-0" class="reference"><a href="#cite_note-GoogleTranslate-42">[42]</a></sup><sup id="cite_ref-WiredGoogleTranslate_43-0" class="reference"><a href="#cite_note-WiredGoogleTranslate-43">[43]</a></sup> GNMT improves the quality of translation because it uses an <a href="/wiki/Example-based_machine_translation" title="Example-based machine translation">example-based machine translation</a> (EBMT) method in which the system "learns from millions of examples."<sup id="cite_ref-googleblog_GNMT_2016_39-1" class="reference"><a href="#cite_note-googleblog_GNMT_2016-39">[39]</a></sup> It translates "whole sentences at a time, rather than just piece by piece. It uses this broader context to help it figure out the most relevant translation, which it then rearranges and adjusts to be more like a human speaking with proper grammar".<sup id="cite_ref-GT_Turovsky_2016_2-3" class="reference"><a href="#cite_note-GT_Turovsky_2016-2">[2]</a></sup> GNMT's "proposed architecture" of "system learning" was first tested on over a hundred languages supported by Google Translate.<sup id="cite_ref-googleblog_GNMT_2016_39-2" class="reference"><a href="#cite_note-googleblog_GNMT_2016-39">[39]</a></sup> With the end-to-end framework, "the system learns over time to create better, more natural translations."<sup id="cite_ref-GT_Turovsky_2016_2-4" class="reference"><a href="#cite_note-GT_Turovsky_2016-2">[2]</a></sup> The GNMT network is capable of <a href="/wiki/Interlingual_machine_translation" title="Interlingual machine translation">interlingual machine translation</a>, which encodes the "semantics of the sentence rather than simply memorizing phrase-to-phrase translations",<sup id="cite_ref-googleblog_GNMT_2016_39-3" class="reference"><a href="#cite_note-googleblog_GNMT_2016-39">[39]</a></sup><sup id="cite_ref-Biotet_44-0" class="reference"><a href="#cite_note-Biotet-44">[44]</a></sup> and the system did not invent its own universal language, but uses "the commonality found inbetween many languages".<sup id="cite_ref-45" class="reference"><a href="#cite_note-45">[45]</a></sup> GNMT was first enabled for eight languages: to and from English and Chinese, French, German, Japanese, Korean, Portuguese, Spanish and Turkish.<sup id="cite_ref-GT_Turovsky_2016_2-5" class="reference"><a href="#cite_note-GT_Turovsky_2016-2">[2]</a></sup><sup id="cite_ref-Schuster.2C_Mike_38-1" class="reference"><a href="#cite_note-Schuster.2C_Mike-38">[38]</a></sup> In March 2017, it was enabled for Hindi, Russian and Vietnamese languages,<sup id="cite_ref-46" class="reference"><a href="#cite_note-46">[46]</a></sup> followed by Indonesian, Bengali, Gujarati, Kannada, Malayalam, Marathi, Punjabi, Tamil and Telugu languages in April.<sup id="cite_ref-47" class="reference"><a href="#cite_note-47">[47]</a></sup></p> 
      <p>GNMT has the capability to translate directly from one language to another (L1 ? L2), which improves upon the previous versions of Google Translate which first translated to English and then to the target language (L1 ? EN ? L2).<sup id="cite_ref-Biotet_44-1" class="reference"><a href="#cite_note-Biotet-44">[44]</a></sup> The GNMT system is also capable of <a href="/w/index.php?title=Zero-Shot_Translation&amp;action=edit&amp;redlink=1" class="new" title="Zero-Shot Translation (page does not exist)">Zero-Shot Translation</a> - translating between a language pair (for example, Japanese to Korean) which the "system has never explicitly seen before."<sup id="cite_ref-googleblog_GNMT_2016_39-4" class="reference"><a href="#cite_note-googleblog_GNMT_2016-39">[39]</a></sup></p> 
      <h3><span class="mw-headline" id="Crowdsourcing">Crowdsourcing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=8" title="Edit section: Crowdsourcing">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>In 2014, Google launched "Translate Community", a platform aimed at improving the translation service by seeking help from volunteers.<sup id="cite_ref-48" class="reference"><a href="#cite_note-48">[48]</a></sup><sup id="cite_ref-49" class="reference"><a href="#cite_note-49">[49]</a></sup> In August 2016, a Google Crowdsource app was released for Android users, in which translation tasks are offered.<sup id="cite_ref-50" class="reference"><a href="#cite_note-50">[50]</a></sup><sup id="cite_ref-51" class="reference"><a href="#cite_note-51">[51]</a></sup></p> 
      <h2><span class="mw-headline" id="Limitations">Limitations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=9" title="Edit section: Limitations">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Some languages produce better results than others. Google Translate performs well especially when English is the target language and the source language is from the <a href="/wiki/European_Union" title="European Union">European Union</a> due to the prominence of translated EU parliament notes. A 2010 analysis indicated that French to English translation is relatively accurate.<sup id="cite_ref-tr1_52-0" class="reference"><a href="#cite_note-tr1-52">[52]</a></sup> However, if the source text is shorter, rule-based machine translations often perform better; this effect is particularly evident in Chinese to English translations. While edits of translations may be submitted, in Chinese specifically one is not able to edit sentences as a whole. Instead, one must edit sometimes arbitrary sets of characters, leading to incorrect edits.<sup id="cite_ref-tr1_52-1" class="reference"><a href="#cite_note-tr1-52">[52]</a></sup></p> 
      <h2><span class="mw-headline" id="Reviews">Reviews</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=10" title="Edit section: Reviews">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Shortly after launching the translation service, Google won an international competition for EnglishArabic and EnglishChinese machine translation.<sup id="cite_ref-53" class="reference"><a href="#cite_note-53">[53]</a></sup></p> 
      <h3><span class="mw-headline" id="Translation_mistakes_and_oddities">Translation mistakes and oddities</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=11" title="Edit section: Translation mistakes and oddities">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>Since Google Translate uses statistical matching to translate, translated text can often include apparently nonsensical and obvious errors,<sup id="cite_ref-54" class="reference"><a href="#cite_note-54">[54]</a></sup> sometimes swapping common terms for similar but nonequivalent common terms in the other language,<sup id="cite_ref-55" class="reference"><a href="#cite_note-55">[55]</a></sup> or inverting sentence meaning. Novelty websites like <a href="/wiki/Translation_Cloud#Bad_Translator" class="mw-redirect" title="Translation Cloud">Bad Translator</a> and Translation Party have utilized the service to produce humorous text by translating back and forth between multiple languages, similar to the children's game <a href="/wiki/Chinese_whispers" title="Chinese whispers">Chinese whispers</a>.<sup id="cite_ref-56" class="reference"><a href="#cite_note-56">[56]</a></sup></p> 
      <h2><span class="mw-headline" id="Court_usage">Court usage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Google_Translate&amp;action=edit&amp;section=12" title="Edit section: Court usage">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>In 2017, Google Translate was used during a court hearing when court officials at Teesside Magistrates' Court failed to book an interpreter for the Chinese defendant.<sup id="cite_ref-57" class="reference"><a href="#cite_note-57">[57]</a></sup></p>
