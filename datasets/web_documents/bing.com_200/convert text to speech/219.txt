          <hr class="hr-xs"> 
          <p style="text-align:justify;">SpeakRight is an Java framework for writing speech recognition applications in VoiceXML. Dynamic generation of VoiceXML is done using the popular StringTemplate templating framework. Although VoiceXML uses a similar web architecture as HTML, the needs of a speech app are very different. It can translate text into phoneme codes, so it could be adapted as a front end for another speech synthesis engine.</p>
