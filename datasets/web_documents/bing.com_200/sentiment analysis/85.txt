                    <span class="rendered_qtext"><p class="qtext_para">The best global package for NLP is the NLTK library. The abbreviation stands for Natural Language Tool Kit. It has what you would need to get started.</p><p class="qtext_para">It is by far NOT the only useful resource out there.</p><p class="qtext_para">However if you are just starting, it is cohesive and easier to learn than pigeoning together some of the other tools. Plus there are a number of books which use it expressly to teach the mechanics of natural language, including sentiment analysis.</p><p class="qtext_para">This book is one of them:</p><p class="qtext_para"><span class="qlink_container"><a href="http://www.geekbooks.me/book/view/mastering-natural-language-processing-with-python" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 38973784);" class="external_link" data-qt-tooltip="geekbooks.me">Mastering Natural Language Processing with Python</a></span></p><p class="qtext_para">It has some sentiment analysis in it. You can download a copy in PDF for like $5. It is definitely easier to learn if you follow and decently scoped tutorial from beginning to end.</p></span>
