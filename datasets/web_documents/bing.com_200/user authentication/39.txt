        <h1>Application Authentication with JAX-WS</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="http://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2010-12-16T19:10:03+00:00">December 16, 2010</time> | Updated : August 29, 2012 | Viewed : 274,114 times <span>+937 pv/w</span></p> 
        </div> 
        <!-- /.post-meta --> 
        <div class="post-content"> 
         <p>One of the common way to handle authentication in JAX-WS is client provides “username” and “password”, attached it in SOAP request header and send to server, server parse the SOAP document and retrieve the provided “username” and “password” from request header and do validation from database, or whatever method prefer.</p> 
         <p>In this article, we show you how to implement the above “<strong>application level authentication in JAX-WS</strong>“.</p> 
         <h2>Ideas…</h2> 
         <p>On the <strong>web service client</strong> site, just put your “username” and “password” into request header.</p> 
         <pre><code class="language-java">
    Map&lt;String, Object&gt; req_ctx = ((BindingProvider)port).getRequestContext();
    req_ctx.put(BindingProvider.ENDPOINT_ADDRESS_PROPERTY, WS_URL);

    Map&lt;String, List&lt;String&gt;&gt; headers = new HashMap&lt;String, List&lt;String&gt;&gt;();
    headers.put("Username", Collections.singletonList("mkyong"));
    headers.put("Password", Collections.singletonList("password"));
    req_ctx.put(MessageContext.HTTP_REQUEST_HEADERS, headers);
</code></pre> 
         <p>On the <strong>web service server</strong> site, get the request header parameters via <code>WebServiceContext</code>.</p> 
         <pre><code class="language-java">
    @Resource
    WebServiceContext wsctx;

    @Override
    public String method() {
		
        MessageContext mctx = wsctx.getMessageContext();
		
	//get detail from request headers
        Map http_headers = (Map) mctx.get(MessageContext.HTTP_REQUEST_HEADERS);
        List userList = (List) http_headers.get("Username");
        List passList = (List) http_headers.get("Password");

    //...
</code></pre> 
         <p>That’s all, now, your deployed JAX-WS is supported application level authentication.</p>

         <h2>1. WebService Server</h2> 
         <p>Create a simple JAX-WS hello world example to handle the authentication in application level.</p> 
         <p><em>File : HelloWorld.java</em></p> 
         <pre><code class="language-java">
package com.mkyong.ws;

import javax.jws.WebMethod;
import javax.jws.WebService;
import javax.jws.soap.SOAPBinding;
import javax.jws.soap.SOAPBinding.Style;

//Service Endpoint Interface
@WebService
@SOAPBinding(style = Style.RPC)
public interface HelloWorld{
	
	@WebMethod String getHelloWorldAsString();
	
}
</code></pre> 
         <p><em>HelloWorldImpl.java</em></p> 
         <pre><code class="language-java">
package com.mkyong.ws;

import java.util.List;
import java.util.Map;

import javax.annotation.Resource;
import javax.jws.WebService;
import javax.xml.ws.WebServiceContext;
import javax.xml.ws.handler.MessageContext;

//Service Implementation Bean
@WebService(endpointInterface = "com.mkyong.ws.HelloWorld")
public class HelloWorldImpl implements HelloWorld{

    @Resource
    WebServiceContext wsctx;

    @Override
    public String getHelloWorldAsString() {
		
	MessageContext mctx = wsctx.getMessageContext();
		
	//get detail from request headers
        Map http_headers = (Map) mctx.get(MessageContext.HTTP_REQUEST_HEADERS);
        List userList = (List) http_headers.get("Username");
        List passList = (List) http_headers.get("Password");

        String username = "";
        String password = "";
        
        if(userList!=null){
        	//get username
        	username = userList.get(0).toString();
        }
        	
        if(passList!=null){
        	//get password
        	password = passList.get(0).toString();
        }
        	
        //Should validate username and password with database
        if (username.equals("mkyong") &amp;&amp; password.equals("password")){
        	return "Hello World JAX-WS - Valid User!";
        }else{
        	return "Unknown User!";
        }
       
    }	
}
</code></pre> 
         <h2>2. EndPoint Publisher</h2> 
         <p>Create an endpoint publisher to deploy above web service at this URL : “<em>http://localhost:9999/ws/hello</em>”</p> 
         <p><em>File : HelloWorldPublisher.java</em></p> 
         <pre><code class="language-java">
package com.mkyong.endpoint;
 
import javax.xml.ws.Endpoint;
import com.mkyong.ws.HelloWorldImpl;
 
//Endpoint publisher
public class HelloWorldPublisher{
 
    public static void main(String[] args) {
	   Endpoint.publish("http://localhost:9999/ws/hello", new HelloWorldImpl());
    }
 
}
</code></pre> 
         <h2>3. WebService Client</h2> 
         <p>Create a web service client to send “username” and “password” for authentication.</p> 
         <p><em>File : HelloWorldClient.java</em></p> 
         <pre><code class="language-java">
package com.mkyong.client;

import java.net.URL;
import java.util.Collections;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

import javax.xml.namespace.QName;
import javax.xml.ws.BindingProvider;
import javax.xml.ws.Service;
import javax.xml.ws.handler.MessageContext;

import com.mkyong.ws.HelloWorld;

public class HelloWorldClient{
	
	private static final String WS_URL = "http://localhost:9999/ws/hello?wsdl";
		
	public static void main(String[] args) throws Exception {
	   
	URL url = new URL(WS_URL);
        QName qname = new QName("http://ws.mkyong.com/", "HelloWorldImplService");

        Service service = Service.create(url, qname);
        HelloWorld hello = service.getPort(HelloWorld.class);
        
        /*******************UserName &amp; Password ******************************/
        Map&lt;String, Object&gt; req_ctx = ((BindingProvider)hello).getRequestContext();
        req_ctx.put(BindingProvider.ENDPOINT_ADDRESS_PROPERTY, WS_URL);

        Map&lt;String, List&lt;String&gt;&gt; headers = new HashMap&lt;String, List&lt;String&gt;&gt;();
        headers.put("Username", Collections.singletonList("mkyong"));
        headers.put("Password", Collections.singletonList("password"));
        req_ctx.put(MessageContext.HTTP_REQUEST_HEADERS, headers);
        /**********************************************************************/
        
        System.out.println(hello.getHelloWorldAsString());
       
    }
}
</code></pre> 
         <p><em>Output</em></p> 
         <pre><code class="language-bash">
Hello World JAX-WS - Valid User!
</code></pre> 
         <h2>4. Tracing SOAP Traffic</h2> 
         <p>From top to bottom, showing how SOAP envelope flows between client and server.</p> 
         <p>1. Client send request, the username “<em>mkyong</em>” and password “<em>password</em>” are included in the SOAP envelope.</p> 
         <pre><code class="language-markup">
POST /ws/hello?wsdl HTTP/1.1
Password: password
Username: mkyong
SOAPAction: ""
Accept: text/xml, multipart/related, text/html, image/gif, image/jpeg, *; q=.2, */*; q=.2
Content-Type: text/xml; charset=utf-8
User-Agent: Java/1.6.0_13
Host: localhost:8888
Connection: keep-alive
Content-Length: 178

&lt;?xml version="1.0" ?&gt;
	&lt;S:Envelope xmlns:S="http://schemas.xmlsoap.org/soap/envelope/"&gt;
		&lt;S:Body&gt;
			&lt;ns2:getHelloWorldAsString xmlns:ns2="http://ws.mkyong.com/"/&gt;
		&lt;/S:Body&gt;
	&lt;/S:Envelope&gt;
</code></pre> 
         <p>2. Server send back a normal response.</p> 
         <pre><code class="language-markup">
HTTP/1.1 200 OK
Transfer-encoding: chunked
Content-type: text/xml; charset=utf-8

&lt;?xml version="1.0" ?&gt;
	&lt;S:Envelope xmlns:S="http://schemas.xmlsoap.org/soap/envelope/"&gt;
		&lt;S:Body&gt;
			&lt;ns2:getHelloWorldAsStringResponse xmlns:ns2="http://ws.mkyong.com/"&gt;
				&lt;return&gt;Hello World JAX-WS - Valid User!&lt;/return&gt;
			&lt;/ns2:getHelloWorldAsStringResponse&gt;
		&lt;/S:Body&gt;
	&lt;/S:Envelope&gt;
</code></pre>
