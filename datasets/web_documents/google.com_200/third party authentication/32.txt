            <div class="jive-rendered-content">
             <p>Not sure where to post this if it should be here or VSE.</p>
             <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
             <p>From the product guide:</p>
             <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
             <p>Third party software authentication McAfee Agent 5.0.1 supports third party integration, such as integration with software developed by SIA partners. For these third party software to communicate with McAfee Agent, the software should have Message Bus Certificates for mutual authentication. We have added MsgbuscertupdaterPackage.zip on SDM and other source locations which will certify third party software to communicate with McAfee Agent. The MsgbuscertupdaterPackage.zip package will be downloaded automatically at the client nodes. This default download task is also scheduled to download the package at 12 AM (local time) every day.</p>
             <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
             <p><a class="jive-link-external-small" href="https://kc.mcafee.com/resources/sites/MCAFEE/content/live/PRODUCT_DOCUMENTATION/25000/PD25735/en_US/ma_501_rn_en-us.pdf" rel="nofollow" target="_blank">https://kc.mcafee.com/resources/sites/MCAFEE/content/live/PRODUCT_DOCUMENTATION/ 25000/PD25735/en_US/ma_501_rn_en-us.pdf</a></p>
