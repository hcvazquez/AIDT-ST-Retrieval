           <p name="dc77" id="dc77" class="graf graf--p graf-after--h3"><a href="https://nodejs.org/en/" data-href="https://nodejs.org/en/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Node.js</a> is a JavaScript runtime environment and a library. It allows to run JavaScript code in the backend, outside a browser by making use of Google’s V8 VM, the same runtime environment for JavaScript as the one used by Google Chrome. It’s lightweight, efficient, and <a href="https://www.quora.com/Is-Node-js-declining-already" data-href="https://www.quora.com/Is-Node-js-declining-already" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">enjoys continuous popularity</a>.</p>
           <p name="0aa2" id="0aa2" class="graf graf--p graf-after--p">The Node.js package ecosystem, npm, is the largest ecosystem of open source libraries in the world. We have a bunch of Node-related channels on Gitter?—?check them out &amp; join the conversation!</p>
           <ul class="postList">
            <li name="4dc3" id="4dc3" class="graf graf--li graf-after--p"><a href="https://gitter.im/orgs/nodejs/rooms?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/orgs/nodejs/rooms?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">nodejs/node?</strong></a><strong class="markup--strong markup--li-strong">—?</strong>Official Node.js chat rooms.</li>
            <li name="87a2" id="87a2" class="graf graf--li graf-after--li"><a href="https://gitter.im/gitterHQ/nodejs?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/gitterHQ/nodejs?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">gitterHQ/nodejs</strong></a>?—?A general chat about Node.</li>
            <li name="df61" id="df61" class="graf graf--li graf-after--li"><a href="https://gitter.im/nwjs/nw.js?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/nwjs/nw.js?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">nwjs/nw.js</strong>?</a>—?NW.js is an app runtime based on Chromium and node.js. You can write native apps in HTML and JavaScript with NW.js. It also lets you call Node.js modules directly from the DOM and enables a new way of writing native applications with all Web technologies.</li>
            <li name="8481" id="8481" class="graf graf--li graf-after--li"><a href="https://gitter.im/linnovate/mean?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/linnovate/mean?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">linnovate/mean</strong></a>?—?MEAN is a framework for an easy starting point with <a href="https://www.mongodb.org/" data-href="https://www.mongodb.org/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">MongoDB</a>, <a href="http://www.nodejs.org/" data-href="http://www.nodejs.org/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">Node.js</a>, <a href="http://expressjs.com/" data-href="http://expressjs.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">Express</a>, and <a href="https://angularjs.org/" data-href="https://angularjs.org/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">AngularJS</a> based applications. It is designed to give you a quick and organized way to start developing MEAN based web apps with useful modules like Mongoose and Passport pre-bundled and configured.</li>
            <li name="923a" id="923a" class="graf graf--li graf-after--li"><a href="https://gitter.im/strongloop/loopback?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/strongloop/loopback?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">strongloop/loopback?</strong></a>—?LoopBack is an open source Node.js framework built on top of Express optimized for building APIs for mobile, web, and other devices. Connect to multiple data sources, write business logic in Node.js, glue on top of your existing services and data, connect using JS, iOS &amp; Android SDKs. <a href="http://loopback.io/" data-href="http://loopback.io/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">http://loopback.io</a></li>
            <li name="50fb" id="50fb" class="graf graf--li graf-after--li"><a href="https://gitter.im/aws/aws-sdk-js?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/aws/aws-sdk-js?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">aws/aws-sdk-js</strong></a>?—?AWS SDK for JavaScript in the browser and Node.js <a href="http://aws.amazon.com/javascript" data-href="http://aws.amazon.com/javascript" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">http://aws.amazon.com/javascript</a></li>
            <li name="aad9" id="aad9" class="graf graf--li graf-after--li"><a href="https://gitter.im/balderdashy/sails?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/balderdashy/sails?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">balderdashy/sails</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?Sails.js is a web framework that makes it easy to build custom, enterprise-grade Node.js apps. It is designed to resemble the MVC architecture from frameworks like Ruby on Rails, but with support for the more modern, data-oriented style of web app development. It’s especially good for building realtime features like chat.</li>
            <li name="c8a2" id="c8a2" class="graf graf--li graf-after--li"><a href="https://gitter.im/hapijs/hapi?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/hapijs/hapi?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">hapijs/hapi?</strong></a>—?hapi is a simple to use configuration-centric framework with built-in support for input validation, caching, authentication, and other essential facilities for building web and services applications. hapi enables developers to focus on writing reusable application logic in a highly modular and prescriptive approach.</li>
            <li name="2e7c" id="2e7c" class="graf graf--li graf-after--li"><a href="https://gitter.im/webdriverio/webdriverio?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/webdriverio/webdriverio?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">webdriverio/webdriverio</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?WebdriverIO is a <a href="https://w3c.github.io/webdriver/webdriver-spec.html" data-href="https://w3c.github.io/webdriver/webdriver-spec.html" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">webdriver</a> (browser automation) module for Node.JS. It makes it possible to write super easy <a href="https://en.wikipedia.org/wiki/Selenium_%28software%29" data-href="https://en.wikipedia.org/wiki/Selenium_%28software%29" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">Selenium</a> tests in your favorite BDD/TDD test framework, that will run locally or in the cloud using Sauce Labs, BrowserStack or TestingBot. WebdriverIO is agnostic with regards to the test framework you want to use. Cucumber, Jasmine and Mocha+Chai are supported by the configuration wizard, and you can use other frameworks as well?—?for example <a href="https://github.com/webdriverio/cucumber-boilerplate" data-href="https://github.com/webdriverio/cucumber-boilerplate" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">Yadda</a>.</li>
            <li name="67a2" id="67a2" class="graf graf--li graf-after--li"><a href="https://gitter.im/feross/webtorrent?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/feross/webtorrent?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">feross/webtorrent?</strong></a>—?WebTorrent is a streaming torrent client for node.js and the browser. YEP, THAT’S RIGHT. THE BROWSER. It’s written completely in JavaScript?—?the language of the web?—?so the same code works in both runtimes. In node.js, this module is a simple torrent client, using TCP and UDP to talk to other torrent clients. In the browser, WebTorrent uses WebRTC (data channels) for peer-to-peer transport. It can be used without browser plugins, extensions, or installations. It’s Just JavaScript™.</li>
            <li name="deb7" id="deb7" class="graf graf--li graf-after--li"><a href="https://gitter.im/koajs/koa?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/koajs/koa?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">koajs/koa?</strong></a>—?Koa is an expressive, light-weight HTTP framework for node.js to make web applications and APIs more enjoyable to write. Koa requests flow through middleware in a stack-like manner, allowing you to perform actions downstream then filter and manipulate the response upstream. Koa’s use of generators also greatly increases the readability and robustness of your application.</li>
            <li name="859c" id="859c" class="graf graf--li graf-after--li"><a href="https://gitter.im/nodegit/nodegit?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/nodegit/nodegit?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">nodegit/nodegit</strong>?</a>—?Native Node bindings to Git. <a href="http://www.nodegit.org/" data-href="http://www.nodegit.org/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">http://www.nodegit.org/</a></li>
            <li name="2c5a" id="2c5a" class="graf graf--li graf-after--li"><a href="https://gitter.im/keystonejs/keystone" data-href="https://gitter.im/keystonejs/keystone" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">keystonejs/keystone</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?<a href="http://keystonejs.com/" data-href="http://keystonejs.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">KeystoneJS</a> is a powerful Node.js content management system and web app framework built on <a href="http://expressjs.com/" data-href="http://expressjs.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">express</a> and <a href="http://mongoosejs.com/" data-href="http://mongoosejs.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">mongoose</a>. Keystone makes it easy to create sophisticated web sites and apps, and comes with a beautiful auto-generated Admin UI.</li>
            <li name="c4e1" id="c4e1" class="graf graf--li graf-after--li"><a href="https://gitter.im/mongodb/node-mongodb-native?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/mongodb/node-mongodb-native?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">mongodb/node-mongodb-native</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?The official <a href="https://www.mongodb.com/" data-href="https://www.mongodb.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">MongoDB</a> driver for Node.js. Provides a high-level API on top of <a href="https://www.npmjs.com/package/mongodb-core" data-href="https://www.npmjs.com/package/mongodb-core" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">mongodb-core</a> that is meant for end users.</li>
            <li name="bf19" id="bf19" class="graf graf--li graf-after--li"><a href="https://gitter.im/EmergingTechnologyAdvisors/node-serialport?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/EmergingTechnologyAdvisors/node-serialport?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">EmergingTechnologyAdvisors/node-serialport</strong></a>?—?Node.js package to access serial ports for reading and writing. Welcome your robotic JavaScript overlords. Better yet, program them!</li>
            <li name="24b3" id="24b3" class="graf graf--li graf-after--li"><a href="https://gitter.im/sass/node-sass?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/sass/node-sass?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">sass/node-sass?</strong></a>—?Node-sass is a library that provides binding for Node.js to <a href="https://github.com/sass/libsass" data-href="https://github.com/sass/libsass" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">LibSass</a>, the C version of the popular stylesheet preprocessor, Sass. It allows you to natively compile&nbsp;.scss files to css at incredible speed and automatically via a connect middleware.</li>
            <li name="c57d" id="c57d" class="graf graf--li graf-after--li"><a href="https://gitter.im/NodeBB/NodeBB?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/NodeBB/NodeBB?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">NodeBB/NodeBB</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?<a href="https://nodebb.org/" data-href="https://nodebb.org/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">NodeBB Forum Software</a> is powered by Node.js and built on either a Redis or MongoDB database. It utilizes web sockets for instant interactions and real-time notifications. NodeBB has many modern features out of the box such as social network integration and streaming discussions, while still making sure to be compatible with older browsers. Additional functionality is enabled through the use of third-party plugins.</li>
            <li name="2c32" id="2c32" class="graf graf--li graf-after--li"><a href="https://gitter.im/restify/node-restify?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/restify/node-restify?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">restify/node-restify</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?<a href="http://restify.com/" data-href="http://restify.com/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">restify</a> is a framework, utilizing <a href="https://github.com/senchalabs/connect" data-href="https://github.com/senchalabs/connect" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">connect</a> style middleware for building REST APIs.             <li name="7166" id="7166" class="graf graf--li graf-after--li"><a href="https://gitter.im/nodecg/nodecg?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" data-href="https://gitter.im/nodecg/nodecg?utm_source=blog&amp;utm_medium=content&amp;utm_campaign=node" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank"><strong class="markup--strong markup--li-strong">nodecg/nodecg</strong></a><strong class="markup--strong markup--li-strong">?</strong>—?NodeCG is a broadcast graphics framework and application. NodeCG is based on the overlay system developed for the <a href="https://www.youtube.com/watch?v=x9PzBHgN29U" data-href="https://www.youtube.com/watch?v=x9PzBHgN29U" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">Tip of the Hats 2014</a> charity event.</li>

An opinionated yet simple framework <a href="http://glazed.io/" data-href="http://glazed.io/" class="markup--anchor markup--li-anchor" rel="noopener" target="_blank">http://glazed.io</a></li>
