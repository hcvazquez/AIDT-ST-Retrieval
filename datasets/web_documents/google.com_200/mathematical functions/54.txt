              <p>Either use <code>import static processing.core.PApplet.*;</code>: <code>cos(radians(90));</code></p> 
              <p>Or w/ <code>import processing.core.PApplet;</code>, prefix them w/ <span style="text-decoration: underline;">PApplet</span>: <code>PApplet.cos(PApplet.radians(90)));</code></p> 
              <p>Keep in mind we can directly access members from Java's class <span style="text-decoration: underline;">Math</span> as well: <code>Math.cos(Math.PI);</code><br> <a href="http://docs.Oracle.com/javase/8/docs/api/java/lang/Math.html" target="_blank" rel="nofollow">http://docs.Oracle.com/javase/8/docs/api/java/lang/Math.html</a></p>
