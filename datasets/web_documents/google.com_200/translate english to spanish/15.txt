        <p>OneNote is free note-taking software from Microsoft. OneNote natively supports translation through the Microsoft Translator menu. It enables users to select text and translate it into any of the <a class="mscom-link" href="https://www.microsoft.com/en-us/translator/faq.aspx">supported languages</a>. </p>
        <p>OneNote supports the following <a class="mscom-link" href="https://www.microsoft.com/en-us/translator/features.aspx">features</a>.</p>
        <h2>How it works</h2>
        <p>To translate text in OneNote, select the text then navigate to the <strong>Review</strong> tab and select <strong>Translate</strong>.</p>
        <p><img src="https://c.s-microsoft.com/en-us/CMSImages/onenote_image1.png?version=67f2b471-37bd-d86d-4abe-e0b6fea1238e" class="mscom-image" alt="OneNote screenshot" width="461" height="160"></p>
        <p>There are two ways to translate text through this menu: directly, and by enabling the <a target="_blank" class="mscom-link" href="http://office.microsoft.com/en-us/onenote-help/see-translations-with-the-mini-translator-HA010356733.aspx">Mini Translator</a>.</p>
