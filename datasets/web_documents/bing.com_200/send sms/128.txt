              Hello,
             <br> I have a website written in Java and I want to add the SMS feature. This website is based in India. I've seen several indian websites that are having this feature.
             <br> 
             <br> When I search on Google, I find many companies that ask you to purchase certain number of sms for so much price. They even offer Java API's. But this option will work very expensive for me.
             <br> 
             <br> The other option suggested to me was using Kannel open source sms gateway. It seems even after getting through all the trouble and setting it up - u still have to configure each cell phone provider like AT&amp;T or tmobile etc in kannel so that kannel will know how to send sms to that provider. So in your java app - you will send the sms to kannel and kannel will send it to the respective provider.
             <br> 
             <br> I've also heard people mention that you need to attach a GSM modem and what not. And I'm totally confused.
             <br> 
             <br> Has anyone been able to setup kannel or any other open source gateway or for that matter any application in which they were able to successfully send and receive sms. What is involved and how do I go about it. 
             <br> 
             <br> Here I'm talking about a large ecommerce website in which 1000's of sms messages will be sent daily.
             <br> Thankyou for your help and guidance.
             <br>
