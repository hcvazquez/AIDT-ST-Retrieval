            How do I connect to a URL using Basic authentication?
           </div> 
           <div>
            Author: Deron Eriksson
           </div> 
           <div class="smaller">
            Description: This Java tutorial describes how to connect to a URL using Basic authentication.
           </div> 
           <div class="smaller">
             Tutorial created using: Windows XP || JDK 1.5.0_09 || Eclipse Web Tools Platform 2.0 (Eclipse 3.3.0)
           </div> 
          </div> 
          <hr> <br> 
          <div style="font-weight: bold">
           Page: &nbsp;&nbsp; 1 
           <a href="/tutorials/lessons/how-do-i-connect-to-a-url-using-basic-authentication.html?page=2" title="Go to page 2">2</a>&nbsp;
           <a href="/tutorials/lessons/how-do-i-connect-to-a-url-using-basic-authentication.html?page=2" title="Go to page 2">&gt;</a>
          </div><br> <p>Connecting to a web site using Basic authentication is fairly straightforward. In <a title="How do I use Basic authentication with Tomcat?" href="how-do-i-use-basic-authentication-with-tomcat.html">another tutorial</a>, we saw that Basic authentication relies on a Base64 encoded 'Authorization' header whose value consists of the word 'Basic' followed by a space followed by the Base64 encoded name:password. </p> <p>The Apache<a class="extlink2" href="http://www.apache.org/" title="Go to Site for 'Apache Software Foundation'">S</a><a class="extlink" href="http://en.wikipedia.org/wiki/Apache_Software_Foundation" title="Go to 'Apache Software Foundation' in Wikipedia">W</a> common codec project at <a href="http://commons.apache.org/codec/">http://commons.apache.org/codec/</a> contains the Apache standard for Base64 encoding/decoding, so we can add that to a project. </p> <img alt="'testing' project" title="'testing' project" src="../general-java/how-do-i-connect-to-a-url-using-basic-authentication/how-do-i-connect-to-a-url-using-basic-authentication-01.gif"> <p>The ConnectToUrlUsingBasicAuthentication class connects to a web page using Basic authentication. It takes a name and a password and concatenates them with a colon in between. It Base64 encodes the resulting string. It makes a URL connection to a web site and sets the 'Authorization' request property to be 'Basic <i>&lt;base-64-encoded-auth-string&gt;' </i>. It reads the content from the URL and displays it to standard output. </p> <h3><a href="../general-java/how-do-i-connect-to-a-url-using-basic-authentication/ConnectToUrlUsingBasicAuthentication.java">ConnectToUrlUsingBasicAuthentication.java</a></h3> <pre><span class="javaKeyword">package</span> test;

<span class="javaKeyword">import</span> java.io.IOException;
<span class="javaKeyword">import</span> java.io.InputStream;
<span class="javaKeyword">import</span> java.io.InputStreamReader;
<span class="javaKeyword">import</span> java.net.MalformedURLException;
<span class="javaKeyword">import</span> java.net.URL;
<span class="javaKeyword">import</span> java.net.URLConnection;

<span class="javaKeyword">import</span> org.apache.commons.codec.binary.Base64;

<span class="javaKeyword">public</span> <span class="javaKeyword">class</span> ConnectToUrlUsingBasicAuthentication {

	<span class="javaKeyword">public</span> <span class="javaKeyword">static</span> <span class="javaKeyword">void</span> main(String[] args) {

		<span class="javaKeyword">try</span> {
			String webPage = <span class="javaString">"http://192.168.1.1"</span>;
			String name = <span class="javaString">"admin"</span>;
			String password = <span class="javaString">"admin"</span>;

			String authString = name + <span class="javaString">":"</span> + password;
			System.out.println(<span class="javaString">"auth string: "</span> + authString);
			<span class="javaKeyword">byte</span>[] authEncBytes = Base64.encodeBase64(authString.getBytes());
			String authStringEnc = <span class="javaKeyword">new</span> String(authEncBytes);
			System.out.println(<span class="javaString">"Base64 encoded auth string: "</span> + authStringEnc);

			URL url = <span class="javaKeyword">new</span> URL(webPage);
			URLConnection urlConnection = url.openConnection();
			urlConnection.setRequestProperty(<span class="javaString">"Authorization"</span>, <span class="javaString">"Basic "</span> + authStringEnc);
			InputStream is = urlConnection.getInputStream();
			InputStreamReader isr = <span class="javaKeyword">new</span> InputStreamReader(is);

			<span class="javaKeyword">int</span> numCharsRead;
			<span class="javaKeyword">char</span>[] charArray = <span class="javaKeyword">new</span> <span class="javaKeyword">char</span>[1024];
			StringBuffer sb = <span class="javaKeyword">new</span> StringBuffer();
			<span class="javaKeyword">while</span> ((numCharsRead = isr.read(charArray)) &gt; 0) {
				sb.append(charArray, 0, numCharsRead);
			}
			String result = sb.toString();

			System.out.println(<span class="javaString">"*** BEGIN ***"</span>);
			System.out.println(result);
			System.out.println(<span class="javaString">"*** END ***"</span>);
		} <span class="javaKeyword">catch</span> (MalformedURLException e) {
			e.printStackTrace();
		} <span class="javaKeyword">catch</span> (IOException e) {
			e.printStackTrace();
		}
	}

}
</pre> <p>The execution of the ConnectToUrlUsingBasicAuthentication class is shown below. </p> <img alt="Console output" title="Console output" src="../general-java/how-do-i-connect-to-a-url-using-basic-authentication/how-do-i-connect-to-a-url-using-basic-authentication-02.gif"> <p class="continue">(Continued on page 2)</p>
