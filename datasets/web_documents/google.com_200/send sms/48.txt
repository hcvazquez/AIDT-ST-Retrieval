        <h3> <a href="https://www.nexmo.com/blog/2016/10/19/how-to-send-sms-messages-with-node-js-and-express-dr/" title="How to Send SMS Messages with Node.js and Express">How to Send SMS Messages with Node.js and Express</a> </h3> 
        <!-- /post title --> 
        <!-- post details --> 
        <span class="date">October 19, 2016</span> 
        <span class="author">Published by <a href="https://www.nexmo.com/blog/author/tomomi/" title="Posts by Tomomi Imura" rel="author">Tomomi Imura</a></span> 
        <!-- /post details --> 
        <div class="at-above-post addthis_tool" data-url="https://www.nexmo.com/blog/2016/10/19/how-to-send-sms-messages-with-node-js-and-express-dr/"></div>
        <p><em>This is the first article in a series of “Getting Started with Nexmo and Node.js” tutorials.</em></p> 
        <p>The <a href="https://docs.nexmo.com/messaging/sms-api">Nexmo SMS API</a> allows you to send and receive a high volume of SMS anywhere in the world. Once you get your virtual phone number, you can use the API to manage outbound messages (“sending”) and inbound messages (“receiving”). In this article, you will learn how to send SMS messages with Node.js and express.</p> 
        <p>All the step-by-step articles I am going to post in this <em>Getting Started</em> series are written from my experiences as a new employee at Nexmo! Whenever I try a new thing, technical or not, I tend to write down how I did it whether I succeed or fail. Working with Nexmo APIs is not an exception – I have been writing down every step I took to work with each API from scratch. Now I am posting my notes with a bunch of screenshots to share with you, so I hope you find them helpful. So let’s walk through with me!</p> 
        <p><img style="width: 32px;height: 32px" src="https://www.nexmo.com/wp-content/uploads/2016/10/GitHub-Mark-64px.png" alt="GitHub icon"> <strong>View</strong> <strong><a href="https://github.com/nexmo-community/nexmo-node-quickstart/blob/master/sms/send-express.js">the source code on GitHub</a></strong></p> 
        <h3>Prerequisites</h3> 
        <p>Before starting this tutorial, make sure you have:</p> 
        <ul> 
         <li>the basic understanding of JavaScript and Node.js</li>

        <p></p> 
        <p>Now, try sending an SMS to any mobile phone number (including Google Voice numbers) using your app.</p> 
        <p>In this tutorial, we are not going to create an HTML with the form UI where a user can fill out a phone number and message (I will write a full tutorial including a front-end code sometimes!), so let’s pretend we are sending data from a web interface by using <a href="https://www.getpostman.com/">Postman</a> to make requests to your app. Postman is a good tool to have when you develop apps with REST APIs!</p> 
        <p>1. Launch Postman and Select <strong>POST</strong>, and enter <em>http://localhost:3000/send</em>.</p> 
        <p>2. At <strong>Headers</strong>, Set <em>Content-Type: application/json</em></p> 
        <p><img src="https://www.nexmo.com/wp-content/uploads/2016/10/postman-headers.png" alt="Send a post request to your app using postman"></p> 
        <p>3. At <strong>Body</strong>, type a valid JSON with “toNumber” and its value (use your mobile phone number! To receive an SMS message!), also “message” and its value.<br> <img src="https://www.nexmo.com/wp-content/uploads/2016/10/postman-body.png" alt="Send a post request to your app using postman"></p> 
        <p>4. Press the blue <strong>Send</strong> button</p> 
        <p>Once you made a successfully POST to your app, you will get a text message to your phone from the virtual number!</p> 
        <p><img src="https://www.nexmo.com/wp-content/uploads/2016/10/screencast-sms.gif" alt="SMS on Android"></p> 
        <p>Also, you will see the response printed on your terminal.</p> 
        <p><img src="https://www.nexmo.com/wp-content/uploads/2016/10/terminal-response.png" alt="Response from Nexmo SMS API"></p> 
        <p>You can view <a href="https://github.com/nexmo-community/nexmo-node-quickstart/blob/master/sms/send-express.js">the code sample</a> used in this tutorial on GitHub.</p> 
        <p>In the next tutorial, you will learn how to receive SMS messages to your virtual number. Stay tuned!</p>
