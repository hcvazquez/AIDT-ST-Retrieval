           This section of our 1000+ Java MCQs focuses on text formatting in Java Programming Language. 
         </div> 
         <p>1. Which of these package is used for text formatting in Java programming language?<br> a) java.text<br> b) java.awt<br> c) java.awt.text<br> d) java.io<br> <span class="collapseomatic " id="id2823" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id2823" class="collapseomatic_content ">
           Answer: a
          <br> Explanation: java.text allows formatting, searching and manipulating text.
          <br> 
         </div> 
         <p>2. Which of this class can be used to format dates and times?<br> a) Date<br> b) SimpleDate<br> c) DateFormat<br> d) textFormat<br> <span class="collapseomatic " id="id9082" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id9082" class="collapseomatic_content ">
           Answer: c
          <br> Explanation: DateFormat is an abstract class that provides the ability to format and parse dates and times.
          <br> 
         </div> 
         <p>3. Which of these method returns an instance of DateFormat that can format time information?<br> a) getTime()<br> b) getTimeInstance()<br> c) getTimeDateinstance()<br> d) getDateFormatinstance()<br> <span class="collapseomatic " id="id7271" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id7271" class="collapseomatic_content ">
           Answer: b
          <br> Explanation: getTimeInstance() method returns an instance of DateFormat that can format time information.
          <br> 
         </div> 
         <p>4. Which of these class allows us to define our own formatting pattern for dates and time?<br> a) DefinedDateFormat<br> b) SimpleDateFormat<br> c) ComplexDateFormat<br> d) UsersDateFormat<br> <span class="collapseomatic " id="id7009" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id7009" class="collapseomatic_content ">
           Answer: b
          <br> Explanation: The DateFormat is a concrete subclass of DateFormat. It allows you to define your own formatting patterns that are used to display date and time information.
          <br> 
         </div> 
         <p>5. Which of these formatting strings of SimpleDateFormat class is used to print AM or PM in time?<br> a) a<br> b) b<br> c) c<br> d) d<br> <span class="collapseomatic " id="id4185" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id4185" class="collapseomatic_content ">
           Answer: a
          <br> Explanation: By using format string “a” we can print AM/PM in time.
          <br> 
         </div> 
         <p>6. Which of these formatting strings of SimpleDateFormat class is used to print week of the year?<br> a) w<br> b) W<br> c) s<br> d) S<br> <span class="collapseomatic " id="id5772" tabindex="0" title="View Answer">View Answer</span></p>
         <div id="target-id5772" class="collapseomatic_content ">
           Answer: a
          <br> Explanation: By using format string “w” we can print week in a year whereas by using ‘W’ we can print week of month.
