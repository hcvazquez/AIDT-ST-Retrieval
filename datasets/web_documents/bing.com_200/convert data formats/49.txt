        <h1 class="entry-title">Java 8 – How to convert String to LocalDate and LocalDate to String in specific format</h1> 
        <div class="entry-meta"> 
         <span class="entry-date"><a href="https://www.javabrahman.com/quick-tips/java-8-how-to-convert-string-to-localdate-and-localdate-to-string-in-specific-format/" rel="bookmark"><time class="entry-date" datetime="2016-09-28T22:50:56+00:00">September 28, 2016</time></a></span>

        <p>This Java 8 code tip first shows <strong>how to convert</strong> <code class="nl-code-intext">String</code> <strong>date to</strong> <code class="nl-code-intext">java.time.LocalDate</code> instance using the <code class="nl-code-intext">parse()</code> method of <code class="nl-code-intext">java.time.DateTimeFormatter</code> class. It then shows how to use the <code class="nl-code-intext">format()</code> method of <code class="nl-code-intext">LocalDate</code> class to <strong>convert <code class="nl-code-intext">LocalDate</code> instance to a <code class="nl-code-intext">String</code></strong> in the specified format.</p> 
        <div class="su-box su-box-style-soft for-shortcodebox" style="border-color:#ca9536;border-radius:4px;-moz-border-radius:4px;-webkit-border-radius:4px;">
         <div class="su-box-title" style="background-color:#fcba43;color:#00000;-webkit-border-top-left-radius:3px;-webkit-border-top-right-radius:3px;-moz-border-radius-topleft:3px;-moz-border-radius-topright:3px;border-top-left-radius:3px;border-top-right-radius:3px;">
          Java 8 code to convert String to LocalDate
         </div>
         <div class="su-box-content su-clearfix"> 
          <pre class="brush: java; title: ; notranslate" title="">package com.javabrahman.java8.time;
import java.time.LocalDate;
import java.time.format.DateTimeFormatter;
public class StringToLocalDate {
  public static void main(String args[]){
   //Converting String in format 'dd-MMM-yyyy' to LocalDate
   String dateStr_1="28-Sep-2016";
   DateTimeFormatter formatter_1=DateTimeFormatter.ofPattern("dd-MMM-yyyy");
   LocalDate localDate_1= LocalDate.parse(dateStr_1,formatter_1);
   System.out.println("Input String with value: "+dateStr_1);
   System.out.println("Converted Date in default ISO format: "+localDate_1+"\n");

   //Converting String in format 'EEEE, MMM d yyyy' to LocalDate
   String dateStr_2="Wednesday, Sep 28 2016";
   DateTimeFormatter formatter_2=DateTimeFormatter.ofPattern("EEEE, MMM d yyyy");
   LocalDate localDate_2= LocalDate.parse(dateStr_2,formatter_2);
   System.out.println("Input String with value: "+dateStr_2);
   System.out.println("Converted Date in default ISO format: "+localDate_2+"\n");

   //Converting String in format 'dd/MM/YY' to LocalDate
   String dateStr_3="28/09/16";
   DateTimeFormatter formatter_3=DateTimeFormatter.ofPattern("dd/MM/yy");
   LocalDate localDate_3= LocalDate.parse(dateStr_3,formatter_3);
   System.out.println("Input String with value: "+dateStr_3);
   System.out.println("Converted Date in default ISO format: "+localDate_3);
 }
}</pre>

        <span class="jb-highlight-lightblue-bold-17">Explanation of the code</span>
        <p></p>
        <div class="code-block code-block-3" style="text-align: center; margin: -8px auto 18px;"> 
         <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> 
         <!-- 4th Para - Desktop - Responsive --> 
         <ins class="adsbygoogle" style="display:block" data-ad-client="ca-pub-4674758778510710" data-ad-slot="1076628980" data-ad-format="auto"></ins>

        <ul id="code-explanation" style="padding-left:12px;"> 
         <li>We start by creating a <code>String</code> with value <code>28-Sep-2016</code>, named <code>dateStr_1</code>, which we then convert to a <code>LocalDate</code>.</li> 
         <li>We need to convey to the Java 8 Date-Time API the format in which the date is being given to it inside <code>dateStr_1</code>. This format is specified using the <code>DateTimeFormatter</code> class. We pass the format of the date <code>dd-MMM-yyyy</code> to <code>DateTimeFormatter</code>’s static method named <code>ofPattern()</code>. </li> 
         <li><code>ofPattern()</code> method returns an instance of <code>DateTimeFormatter</code>, named <code>formatter_1</code>, with the required date format set in it.</li> 
         <li>We then pass the <code>dateStr_1</code> and <code>formatter_1</code> as inputs to the <code>LocalDate.parse()</code> method which creates an instance of <code>LocalDate</code>, named <code>localDate_1</code>, with the value as <strong>28th of September 2016</strong>. </li> 
         <li>We then print <code>localDate_1</code> which prints the date as <code>2016-09-28</code> which is in the Standard ISO Format for dates.</li> 
         <li>Similarly, two other date formats are shown as examples to show how the date formats work resulting in dates <code>localDate_2</code> and <code>localDate_3</code>. These are also printed in the default ISO format to show that the <code>String</code> has been converted properly to its equivalent <code>LocalDate</code>.</li> 
         <li>To print dates in formats other than the default ISO format, we will need to use the <code>LocalDate.format()</code> method. <em>The next section of this article shows how to format a <code>LocalDate</code> into formats other than the default ISO format</em>.</li>

import java.time.LocalDate;

public class LocalDateToString {

   //We will use current date from the system clock
   LocalDate today=LocalDate.now();
   System.out.println("Current date using default toString(same as ISO Standard Format): "+today);

   //Converting Date to a user specific format 1 - dd-MMM-yyyy
   DateTimeFormatter formatter_1 = DateTimeFormatter.ofPattern("dd-MMM-yyyy");
   String format_1=(today).format(formatter_1);
