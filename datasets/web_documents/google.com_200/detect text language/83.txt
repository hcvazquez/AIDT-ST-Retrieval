       <h1 itemprop="name" class="devsite-page-title"> Detecting Languages </h1> 
       <nav class="devsite-page-nav-embedded devsite-nav"></nav> 
       <div class="devsite-article-body clearfix
            " itemprop="articleBody"> 
        <p>This document describes how to use the Google Translation API to detect the language of a string.</p> 
        <h2 id="detecting_the_language_of_a_text_string">Detecting the language of a text string</h2> 
        <p>You can detect the language of a text string by sending an HTTP request using a URL of the following format:</p> 
        <pre class="prettyprint"><code>https://translation.googleapis.com/language/translate/v2/detect
</code></pre> 
        <p>Two query parameters are required with each detection request:</p> 
        <ul> 
         <li><strong>Source text string:</strong> Use the <code>q</code> parameter to specify the text to translate.</li> 
         <li><strong><a href="https://support.google.com/cloud/answer/6158862">API key:</a></strong> Use the <code>key</code> parameter to identify your application. If you are using OAuth 2.0 service account credentials (recommended), do not supply this parameter.</li> 
        </ul> 
        <p>All other <a href="https://cloud.google.com/translate/docs/reference/detect#body.QUERY_PARAMETERS">query parameters</a> are optional. The URL for the <code>GET</code> request, including parameters, must be less than 2K characters.</p> 
        <h3 id="detecting_the_language_of_a_single_string">Detecting the language of a single string</h3> 
        <div id="translate-detect-language-code-sample" class="ds-selector-tabs" data-ds-scope="code-sample"> 
         <section> 
          <h3 id="translate-detect-language-protocol" track-type="sampleTabNav" track-name="protocol" track-metadata-position="translate-detect-language">Protocol</h3> 
          <p>Here is an example HTTP <code>POST</code> request that detects the language of the sentence "Google Translate Rocks":</p> 
          <pre class="notranslate devsite-click-to-copy" suppresswarning="suppresswarning">POST https://translation.googleapis.com/language/translate/v2/detect?key=YOUR_API_KEY</pre> 
          <pre class="notranslate prettyprint" suppresswarning="suppresswarning">
{
  'q': 'Google Translate Rocks',
}
</pre> 
          <p>If the request is successful, the server returns a <code>200 OK</code> HTTP status code and the response in JSON format:</p> 
          <pre class="notranslate" suppresswarning="suppresswarning">200 OK</pre> 
          <pre class="notranslate prettyprint" suppresswarning="suppresswarning">
{
  "data": {
    "detections": [
      [
        {
          "confidence": 0.24697750806808472,
          "isReliable": false,
          "language": "en"
        }
      ]
    ]
  }
}
</pre> 
          <p>In this example, <code>language</code> is the detected language code. The <code>isReliable</code> member has been deprecated. The <code>confidence</code> field is an optional floating point value between 0 and 1. The closer this value is to 1, the higher the confidence level for the language detection. The <code>confidence</code> field is not always available.</p>

func detectLanguage(text string) (*translate.Detection, error) {
	ctx := context.Background()
	client, err := translate.NewClient(ctx)
	if err != nil {
		return nil, err
	}
	defer client.Close()

	lang, err := client.DetectLanguage(ctx, []string{text})
	if err != nil {
		return nil, err
	}
	return &amp;lang[0][0], nil
}</pre>

        <div id="translate-detect-language-multiple-code-sample" class="ds-selector-tabs" data-ds-scope="code-sample"> 
         <section> 
          <h3 id="translate-detect-language-multiple-protocol" track-type="sampleTabNav" track-name="protocol" track-metadata-position="translate-detect-language-multiple">Protocol</h3> 
          <p>To detect language for more than one string, use the <code>q</code> parameter to specify each string. This example passes two separate strings for detection:</p> 
          <pre class="notranslate devsite-click-to-copy" suppresswarning="suppresswarning">POST https://translation.googleapis.com/language/translate/v2/detect?key=YOUR_API_KEY</pre> 
          <pre class="notranslate prettyprint" suppresswarning="suppresswarning">
{
  'q': 'Hello world',
  'q': '???????'
}
</pre> 
          <p>If the request is successful, the server returns a <code>200 OK</code> HTTP status code and the response in JSON format:</p> 
          <pre class="notranslate prettyprint" suppresswarning="suppresswarning">
{
  "data": {
    "detections": [
      [
        {
          "confidence": 0.71052581071853638,
          "isReliable": false,
          "language": "en"
        }
      ],
      [
        {
          "confidence": 0.87668424844741821,
          "isReliable": false,
          "language": "zh-TW"
        }
      ]
    ]
  }
}
</pre> 
          <p>Here, the response contains two detections, in the same order as the corresponding source strings were provided in the request.</p>
