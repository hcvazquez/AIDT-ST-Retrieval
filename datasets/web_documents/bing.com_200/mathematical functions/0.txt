      <p>The <code>java.lang.Math</code> class allows the use of many common mathematical functions that can be used while creating programs.</p>

      <h3><span class="mw-headline" id="Exponential_methods">Exponential methods</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=5" title="Edit section: Exponential methods">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

      <p>There are several methods in the <code>Math</code> class that deal with exponential functions.</p> 
      <h4><span class="mw-headline" id="Exponentiation"><a href="/w/index.php?title=Exponentiation&amp;action=edit&amp;redlink=1" class="new" title="Exponentiation (does not exist)">Exponentiation</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=6" title="Edit section: Exponentiation">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <p>The power method, <code>double Math.<b>pow</b>(double, double)</code>, returns the first parameter to the power of the second parameter. For example, a call to <code>Math.pow(2, 10)</code> will return a value of 1024 (2<sup>10</sup>).</p> 
      <p>The <code>Math.<b>exp</b>(double)</code> method, a special case of <code>pow</code>, returns <i>e</i> to the power of the parameter. In addition, <code>double Math.<b>expm1</b>(double)</code> returns (<i>e</i><sup>x</sup> - 1). Both of these methods are more accurate and convenient in these special cases.</p> 
      <p>Java also provides special cases of the pow function for square roots and cube roots of doubles, <code>double Math.<b>sqrt</b>(double)</code> and <code>double Math.<b>cbrt</b>(double)</code>.</p> 
      <h3><span class="mw-headline" id="Logarithms">Logarithms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=7" title="Edit section: Logarithms">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>Java has no general logarithm function; when needed this can be simulated using the change-of-base theorem.</p> 
      <p><code><code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code> Math.<b>log</b>(<code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code>)</code> returns the natural logarithm of the parameter (<b>not the common logarithm</b>, as its name suggests!).</p> 
      <p><code><code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code> Math.<b>log10</b>(<code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code>)</code> returns the common (base-10) logarithm of the parameter.</p> 
      <p><code><code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code> Math.<b>log1p</b>(<code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code>)</code> returns ln(parameter+1). It is recommended for small values.</p> 
      <h3><span class="mw-headline" id="Trigonometric_and_hyperbolic_methods">Trigonometric and hyperbolic methods</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=8" title="Edit section: Trigonometric and hyperbolic methods">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>The trigonometric methods of the <code>Math</code> class allow users to easily deal with trigonometric functions in programs. All accept only <code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code>s. Please note that all values using these methods are initially passed and returned in <b>radians</b>, <i>not degrees</i>. However, conversions are possible.</p> 
      <h4><span class="mw-headline" id="Trigonometric_functions">Trigonometric functions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=9" title="Edit section: Trigonometric functions">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <p>The three main trigonometric methods are <code>Math.<b>sin</b>(x)</code>, <code>Math.<b>cos</b>(x)</code>, and <code>Math.<b>tan</b>(x)</code>, which are used to find the sine, cosine, and tangent, respectively, of any given number. So, for example, a call to <code>Math.sin(Math.PI/2)</code> would return a value of about 1. Although methods for finding the cosecant, secant, and cotangent are not available, these values can be found by taking the reciprocal of the sine, cosine, and tangent, respectively. For example, the cosecant of pi/2 could be found using <code>1/Math.sin(Math.PI/2)</code>.</p> 
      <h4><span class="mw-headline" id="Inverse_trigonometric_functions">Inverse trigonometric functions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=10" title="Edit section: Inverse trigonometric functions">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <p>Java provides inverse counterparts to the trigonometric functions: <code>Math.<b>asin</b>(x)</code>, and <code>Math.<b>acos</b>(x)</code>, <code>Math.<b>atan</b>(x)</code>.</p> 
      <h4><span class="mw-headline" id="Hyperbolic_functions">Hyperbolic functions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=11" title="Edit section: Hyperbolic functions">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <p>In addition, hyperbolic functions are available: <code>Math.<b>sinh</b>(x)</code>, <code>Math.<b>cosh</b>(x)</code>, and <code>Math.<b>tanh</b>(x)</code>.</p> 
      <h4><span class="mw-headline" id="Radian.2Fdegree_conversion">Radian/degree conversion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=12" title="Edit section: Radian/degree conversion">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
      <p>To convert between degree and radian measures of angles, two methods are available, <code>Math.<b>toRadians</b>(x)</code> and <code>Math.<b>toDegrees</b>(x)</code>. While using <code>Math.toRadians(x)</code>, a degrees value must be passed in, and that value in radians (the degree value multiplied by pi/180) will be returned. The <code>Math.toDegrees(x)</code> method takes in a value in radians and the value in degrees (the radian value multiplied by 180/pi) is returned.</p> 
      <h3><span class="mw-headline" id="Absolute_value:_Math.abs">Absolute value: <code>Math.abs</code></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=13" title="Edit section: Absolute value: Math.abs">edit</a><span class="mw-editsection-bracket">]</span></span></h3>

      <h3><span class="mw-headline" id="Maximum_and_minimum_values">Maximum and minimum values</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=14" title="Edit section: Maximum and minimum values">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>These methods are very simple comparing functions. Instead of using <code><a href="/wiki/Java_Programming/Keywords/if" title="Java Programming/Keywords/if"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">if</font></a></code>...<code><a href="/wiki/Java_Programming/Keywords/else" title="Java Programming/Keywords/else"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">else</font></a></code> statements, one can use the <code>Math.max(x1, x2)</code> and <code>Math.min(x1, x2)</code> methods. The <code>Math.max(x1, x2)</code> simply returns the greater of the two values, while the <code>Math.min(x1, x2)</code> returns the lesser of the two. Acceptable types for these methods include <code><a href="/wiki/Java_Programming/Keywords/int" title="Java Programming/Keywords/int"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">int</font></a></code>, <code><a href="/wiki/Java_Programming/Keywords/long" title="Java Programming/Keywords/long"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">long</font></a></code>, <code><a href="/wiki/Java_Programming/Keywords/float" title="Java Programming/Keywords/float"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">float</font></a></code>, and <code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a></code>.</p> 
      <h2><span class="mw-headline" id="Functions_dealing_with_floating-point_representation">Functions dealing with floating-point representation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Java_Programming/Mathematical_functions&amp;action=edit&amp;section=15" title="Edit section: Functions dealing with floating-point representation">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>Java 1.5 and 1.6 introduced several non-mathematical functions specific to the computer floating-point representation of numbers.</p> 
      <p><code>Math.<b>ulp</b>(</code><a href="/wiki/Java_Programming/Keywords/double" title="Java Programming/Keywords/double"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">double</font></a>) and <code>Math.<b>ulp</b>(</code><a href="/wiki/Java_Programming/Keywords/float" title="Java Programming/Keywords/float"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">float</font></a>) return an ulp, the smallest value which, when added to the argument, would be recognized as larger than the argument.</p> 
      <p><code>Math.copySign</code> returns the value of the first argument with the sign of the second argument. It can be used to determine the sign of a zero value.</p> 
      <p><code>Math.getExponent</code> returns (as an <code><a href="/wiki/Java_Programming/Keywords/int" title="Java Programming/Keywords/int"><font style="color:DodgerBlue4; text-decoration: none; font-weight: bold;">int</font></a></code>) the exponent used to scale the floating-point argument in computer representation.</p>
