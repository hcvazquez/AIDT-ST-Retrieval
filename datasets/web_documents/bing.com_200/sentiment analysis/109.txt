     <h2><strong>Sentiment Analysis API</strong></h2> 
     <p>TheySay’s real-time Sentiment Analysis API gives you access to a state-of-the-art sentiment analysis algorithm through a scalable and secure RESTful API service.</p> 
     <p>Our analysis is powered by a hybrid <a href="http://www.theysay.io/natural-language-processing/">Natural Language Processing (NLP)</a> engine that runs highly sophisticated linguistic algorithms and Machine Learning classifiers. The engine is wrapped in a platform-agnostic REST API service that enables your software applications, workflows, and services to receive rich TheySay JSON metadata with minimal integration work.</p> 
     <p>Our API service is called <a href="http://www.theysay.io/product/preceive/">PreCeive</a> and is available for a 30-day free trial.</p> 
     <p><a class="shortcode-btn wpb_button_a" title="Sign-Up For Free Trial" href="http://www.theysay.io/text-analytics-api-signup/"><span class="gform_button button">Sign-Up For Free Trial</span></a></p> 
     <p>Please visit our API demo site at <a href="http://apidemo.theysay.io/">http://apidemo.theysay.io/</a> to see how our text analytics API analyses your text or send HTTP GET and POST requests to the service using one of our Open Source clients listed below.</p> 
     <h2><strong>Free API Demo</strong></h2> 
     <p>Try out our free text analytics api at <a href="http://apidemo.theysay.io/">http://apidemo.theysay.io/</a></p> 
     <h2><strong>Open Source API Clients</strong></h2> 
     <p>We offer a number of Open Source API clients which include:</p> 
     <p>•&nbsp;<a href="https://github.com/theysay/affectr-csharp">C#</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-java-2">Java</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-nodejs">NodeJS</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-php">PHP</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-python">Python</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-ruby">Ruby</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-r">R</a><br> •&nbsp;<a href="https://github.com/theysay/affectr-scala">Scala</a></p> 
     <p>You’ll find these and more in our GitHub repository at&nbsp;<a title="https://github.com/theysay" href="https://github.com/theysay">https://github.com/theysay</a>.</p> 
     <h2><strong>PreCeive Batch Client</strong></h2> 
     <p>We also released an <a href="http://www.theysay.io/developer/hit-the-road-running-with-the-preceive-batch-client/">Open Source batch client</a> to allow you to analyse your text from an Excel, OpenOffice or OpenDocument spreadsheet or just a bunch of Twitter data</p> 
     <h2><strong>API Documentation</strong></h2>
