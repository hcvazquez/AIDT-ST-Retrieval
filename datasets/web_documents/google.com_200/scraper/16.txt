       <header class="entry-header"> 
        <h1 class="entry-title">Create Scraper in Javascript for Loklak Scraper JS</h1> 
       </header> 
       <div class="entry-content"> 
        <p><a href="https://github.com/fossasia/loklak_scraper_js/">Loklak Scraper JS</a> is the latest repository in Loklak project. It is one of the interesting projects because of expected benefits of Javascript in web scraping. It has a <a href="https://nodejs.org/en/about/">Node Javascript engine</a> and is used in <a href="https://github.com/fossasia/loklak_wok_android">Loklak Wok project</a> as bundled package. It has potential to be used in different repositories and enhance them.</p> 
        <p>Scraping in Python is easy (at least for Pythonistas) as one needs to just import <a href="http://docs.python-requests.org/"><i>Request library</i></a> and <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/"><i>BeautifulSoup</i></a> library (<a href="https://stackoverflow.com/questions/4967103/beautifulsoup-and-lxml-html-what-to-prefer"><i>lxml as better option</i></a>), write some lines of code using Request library to get webpage and some lines of <a href="https://stackoverflow.com/questions/29283510/what-is-the-difference-beautifulsoup-and-bs4"><i>bs4</i></a> to walk through html and scrape data. This sums up to about less than a hundred lines of coding, where as Javascript coding isn’t easily readable (at least to me) as compared to Python. <i>But it has an advantage</i>, it can easily deal with Javascript in the pages we are scraping. This is one of the motive, Loklak Scraper JS repository was created and we contributed and worked on it.</p> 
        <p>I recently coded a Javascript scraper in <a href="https://github.com/fossasia/loklak_scraper_js/"><i>loklak_scraper_js repository</i></a>. While coding, I found it’s libraries similar to the libraries, I use to code in Python. Therefore, this blog is for Pythonistas how they can start scraping in Javascript as they finish reading and also contribute to Loklak Scraper JS.</p> 
        <p><b>First</b>, replace Python interpreter, Request and Beautifulsoup library with Node JS interpreter, Request and Cheerio JS library.</p> 
        <p>1) <b><i>Node JS Interpreter</i></b>: Node JS Interpreter is used to interpret Javascript files. This is different from Python as it deals with the project instead of a module in case of Python. The most compatible Node for most of the libraries is 6.0.0 , where as latest version available(as I checked) is 8.0.0</p> 
        <p>TIP: use `–save` with npm like <a href="https://stackoverflow.com/questions/19578796/what-is-the-save-option-for-npm-install"><i>here</i></a> while installing a library.</p> 
        <p>2) <b><i>Request Library</i></b> :- This is used to load webpage to be processed. Similar to one in Python.</p> 
        <p><a href="https://www.npmjs.com/package/request-promise"><b><i>Request-promise</i></b></a> library, a wrapper around Request with implementation of <a href="http://bluebirdjs.com/docs/getting-started.html"><i>Bluebird</i></a> library, improves readability and makes code cleaner (<a href="http://bluebirdjs.com/docs/why-promises.html">how?</a>).</p> 
        <p>&nbsp;</p> 
        <p>3) <b><i>Cheerio Library</i></b>:- A Pythonista (a rookie one) can call it twin of BeautifulSoup Library. But this is faster and is Javascript. It’s selector implementation is nearly identical to jQuery’s.</p> 
        <p><b>Let us</b> code a basic Javascript scraper. I will take TimeAndDate scraper from loklak_scraper_js as example <a href="https://github.com/fossasia/loklak_scraper_js/commit/0e616983b4bc98df76b5370e59e1d2db6363d975"><i>here</i></a>. It inputs place and outputs its local time.</p> 
        <p><b>Step#1</b>: fetching HTML from webpage with the help of Request library.</p> 
        <p>We input url to Request function to fetch the webpage and is saved to `html` variable. This scrapeTimeAndDate() function scrapes data from html</p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #d0d0d0">url</span> <span style="color: #d0d0d0">=</span> <span style="color: #ed9d13">"http://www.timeanddate.com/worldclock/results.html?query=London"</span><span style="color: #d0d0d0">;</span>

<span style="color: #d0d0d0">request(url,</span> <span style="color: #6ab825;font-weight: bold">function</span><span style="color: #d0d0d0">(error,</span> <span style="color: #d0d0d0">response,</span> <span style="color: #d0d0d0">body)</span> <span style="color: #d0d0d0">{</span>

 <span style="color: #6ab825;font-weight: bold">if</span><span style="color: #d0d0d0">(error)</span> <span style="color: #d0d0d0">{</span>

    <span style="color: #d0d0d0">console.log(</span><span style="color: #ed9d13">"Error: "</span> <span style="color: #d0d0d0">+</span> <span style="color: #d0d0d0">error);</span>

    <span style="color: #d0d0d0">process.exit(-</span><span style="color: #3677a9">1</span><span style="color: #d0d0d0">);</span>

 <span style="color: #d0d0d0">}</span>

 <span style="color: #d0d0d0">html</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">body;</span>

 <span style="color: #d0d0d0">scrapeTimeAndDate()</span>

<span style="color: #d0d0d0">});</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <p><b>Step#2</b>: To scrape important data from html using <a href="https://cheerio.js.org/"><i>Cheerio JS</i></a></p> 
        <p>list of date and time of locations is embedded in table tag, So we will iterate through &lt;td&gt; and extract text.</p> 
        <ol> 
         <li>a) Load html to Cheerio as we do in beautifulsoup</li> 
        </ol> 
        <p><em>In Python</em></p> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #d0d0d0">soup = BeautifulSoup(html,</span><span style="color: #ed9d13">'html5lib'</span><span style="color: #d0d0d0">)</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <p><em>In Cheerio JS</em></p> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #d0d0d0">$</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">cheerio.load(html);</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <ol> 
         <li>b) This line finds first tr tag in table tag.</li> 
        </ol> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #6ab825;font-weight: bold">var</span> <span style="color: #d0d0d0">htmlTime</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">$(</span><span style="color: #ed9d13">"table"</span><span style="color: #d0d0d0">).find(</span><span style="color: #ed9d13">'tr'</span><span style="color: #d0d0d0">);</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <ol> 
         <li>c) Iterate through td tags data by using each() function. This function acts as loop (in Python) iterating through list of elements in which data will be extracted.</li> 
        </ol> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #d0d0d0">htmlTime.each(</span><span style="color: #6ab825;font-weight: bold">function</span> <span style="color: #d0d0d0">(index,</span> <span style="color: #d0d0d0">element)</span> <span style="color: #d0d0d0">{</span>      

  <span style="color: #999999;font-style: italic">// in python, we will use loop, `for element from elements:`</span>

  <span style="color: #d0d0d0">tag</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">$(element).find(</span><span style="color: #ed9d13">"td"</span><span style="color: #d0d0d0">);</span>    <span style="color: #999999;font-style: italic">// in python, `tag = soup.find_all('td')`</span>

  <span style="color: #6ab825;font-weight: bold">if</span><span style="color: #d0d0d0">(</span> <span style="color: #d0d0d0">tag.text()</span> <span style="color: #d0d0d0">!=</span> <span style="color: #ed9d13">""</span><span style="color: #d0d0d0">)</span> <span style="color: #d0d0d0">{</span>

    <span style="color: #d0d0d0">.</span>

    <span style="color: #d0d0d0">.</span>

    <span style="color: #999999;font-style: italic">//EXTRACT DATA</span>

    <span style="color: #d0d0d0">.</span>

    <span style="color: #d0d0d0">.</span>

  <span style="color: #d0d0d0">}</span> <span style="color: #6ab825;font-weight: bold">else</span> <span style="color: #d0d0d0">{</span>

    <span style="color: #999999;font-style: italic">//go to next td tag</span>

    <span style="color: #d0d0d0">tag</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">tag.next();</span>

  <span style="color: #d0d0d0">}</span>

<span style="color: #d0d0d0">}</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <ol> 
         <li>d) To extract data</li> 
        </ol> 
        <p>Cheerio JS loads html and uses DOM model traverse through. DOM model considers html is tree. So, go to the tag, and scrape data you want.</p> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #999999;font-style: italic">//extract location(text) enclosed in tag</span>

<span style="color: #d0d0d0">location</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">tag.text();</span>

<span style="color: #999999;font-style: italic">//go to next tag</span>

<span style="color: #d0d0d0">tag</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">tag.next();</span>

<span style="color: #999999;font-style: italic">//extract time(text) enclosed in tag</span>

<span style="color: #d0d0d0">time</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">tag.text();</span>

<span style="color: #999999;font-style: italic">//save in dictionary like in python</span>

<span style="color: #d0d0d0">loc_list[</span><span style="color: #ed9d13">"location"</span><span style="color: #d0d0d0">]</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">location;</span>

<span style="color: #d0d0d0">loc_list[</span><span style="color: #ed9d13">"time"</span><span style="color: #d0d0d0">]</span> <span style="color: #d0d0d0">=</span> <span style="color: #d0d0d0">time;</span>
</pre> 
        </div> 
        <p>&nbsp;</p> 
        <p>Some other useful functions:-</p> 
        <p>1) <strong><em>$(selector, [context], [root])</em></strong></p> 
        <p>returns object of selector(any tag) with class or id inside root</p> 
        <p>2) <em><strong>$(“table”).attr(name, value)</strong></em></p> 
        <p>To get tag object having attribute having `value`</p> 
        <p>3) <em><strong>obj.html()</strong></em></p> 
        <p>To get html enclosed in tags</p> 
        <p>For more just drop in <a href="https://cheerio.js.org/"><i>here</i></a></p> 
        <p><b>Step#3</b>: Execute scraper using command</p> 
        <p></p> 
        <div style="background: #202020;overflow: auto;width: auto;border: solid black;border-width: .1em .1em .1em .8em;padding: .2em .6em"> 
         <pre style="margin: 0;line-height: 125%"><span style="color: #cccccc">node &lt;scrapername&gt;.js</span>
</pre> 
        </div>
