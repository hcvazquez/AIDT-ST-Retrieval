  <h1>Contributed Packages</h1> 
  <h3 id="pkgs">Available Packages</h3> 
  <p> Currently, the CRAN package repository features 11442 available packages. </p> 
  <p> <a href="available_packages_by_date.html"> Table of available packages, sorted by date of publication </a> </p> 
  <p> <a href="available_packages_by_name.html"> Table of available packages, sorted by name </a> </p> 
  <h3 id="install">Installation of Packages</h3> 
  <p> Please type <kbd>help("INSTALL")</kbd> or <kbd>help("install.packages")</kbd> in R for information on how to install packages from this repository. The manual <a href="../../manuals.html#R-admin"> R Installation and Administration</a> (also contained in the R base sources) explains the process in detail. </p> 
  <p> <a href="../views/">CRAN Task Views</a> allow you to browse packages by topic and provide tools to automatically install all packages for special areas of interest. Currently, 35 views are available. </p> 
  <h3 id="check">Package Check Results</h3> 
  <p> All packages are tested regularly on machines running <a href="http://www.debian.org/">Debian GNU/Linux</a>, <a href="http://www.fedoraproject.org/">Fedora</a>, OS X, Solaris and Windows. </p> 
  <p> The results are summarized in the <a href="../../web/checks/check_summary.html">check summary</a> (some <a href="../../web/checks/check_timings.html">timings</a> are also available). Additional details for Windows checking and building can be found in the <a href="../../bin/windows/contrib/checkSummaryWin.html">Windows check summary</a>. </p> 
  <h3 id="write">Writing Your Own Packages</h3> 
  <p> The manual <a href="../../manuals.html#R-exts">Writing R Extensions</a> (also contained in the R base sources) explains how to write new packages and how to contribute them to CRAN. </p> 
  <h3 id="policies">Repository Policies</h3> 
  <p> The manual <a href="../../web/packages/policies.html">CRAN Repository Policy</a> <a href="../../web/packages/policies.pdf">[PDF]</a> describes the policies in place for the CRAN package repository. </p>
