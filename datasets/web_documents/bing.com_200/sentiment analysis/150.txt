        <h1 class="codeTitle word-wrap">Sentiment analysis using R</h1> 
       </div> 
      </div> 
      <div class="panel-body word-wrap question"> 
       <p>Are there any R packages that focus on sentiment analysis? I have a small survey where users can write a comment about their experience of using a web-tool. I ask for a numerical ranking, and there is the option of including a comment. <br> <br> I am wondering what the best way of assessing the positiveness or negativeness of the comment is. I would like to be able to compare it to the numerical ranking that the user provides, using R.<br> </p>

       <div class="answer" style="background-color: #fafafa;padding: 20px;">
        <p>And there is <a target="_blank" rel="nofollow" href="http://cran.r-project.org/web/packages/sentiment/index.html">this package</a>:</p> 
        <p><code class="language-r">sentiment: Tools for Sentiment Analysis</code></p> 
        <p>sentiment is an R package with tools for sentiment analysis including bayesian classifiers for positivity/negativity and emotion classification.</p> 
        <p>Update 14 Dec 2012: it has been removed to the <a target="_blank" rel="nofollow" href="http://cran.r-project.org/src/contrib/Archive/sentiment/">archive</a>...</p> 
        <p>Update 15 Mar 2013: the <a target="_blank" rel="nofollow" href="http://cran.us.r-project.org/web/packages/qdap/index.html">qdap</a> package has a <code class="language-r">polarity</code> function, based on Jeffery Breen's work</p>
