         <div class="description__title">
           Description 
         </div> 
         <p>Learn all of the different aspects of full stack JavaScript development using the MEAN stack. We're not talking about any generators or MEAN frameworks here, we're talking about a full understanding of MongoDB, Express, AngularJS and Node<em>.</em>js. Throughout this course we'll show you how to use each of these technologies, and how to use them together.</p> 
         <p><strong class="redactor-inline-converted">Build Great JavaScript Applications using MongoDB, Express, AngularJS 1.x and Node<em>.</em>js</strong></p> 
         <p>The overall aim of the course is to enable to you confidently build all different types of application using the MEAN stack.</p> 
         <p>To do this, the course is divided into four sections, each section focusing on a different goal. The four sections all work together building a full application, with an overall outcome of showing how to <strong class="redactor-inline-converted">architect and build complete MEAN applications</strong>.</p> 
         <p>The breakdown of sections looks like this:</p> 
         <ol>
          <li>By the end of section one you will be able to <strong class="redactor-inline-converted">set up a web-server using Node.js and Express</strong>, to listen for requests and return responds.</li>
          <li>By the end of section 2 you will be able to <strong class="redactor-inline-converted">design NoSQL databases and work with MongoDB</strong> from the command line and from Node and Express.</li>
          <li>After section 3 you will be able to design and build <strong class="redactor-inline-converted">robust REST APIs using Node<em>.</em>js</strong>, Express and MongoDB, following industry best practices.</li>
          <li>By the end of section four you will be able to build high quality <strong class="redactor-inline-converted">AngularJS 1.x single page applications (SPAs)</strong>, following industry best practices.</li>
          <li>When you have finished with section five you will able to <strong>add authentication to the MEAN stack</strong>, enabling users to log in and manage sessions.</li>
         </ol> 
         <p>Along the way there are various activities, so you can be has hands on as you like. You'll get the most out of the course if you follow along and code as you go, but if you want to speed through it the source code is supplied with each video (where relevant).</p> 
         <div class="audience" data-purpose="course-audience"> 
          <div class="audience__title">
            Who is the target audience? 
          </div> 
          <ul class="audience__list"> 
           <li>This course is meant for anyone who wants to start building full stack JavaScript applications in Node.js, AngularJS, Express and MongoDB. It starts from the basic concepts of each technology, so user's experienced in a particular area will be able to speed through these sections. This course assumes you have some JavaScript knowledge, and does not teach JavaScript itself.</li> 
          </ul>
