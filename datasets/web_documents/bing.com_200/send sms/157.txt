           <p class="themeta"><span class="thetime">September 19, 2010</span><span class="thecategories"><a href="http://devlup.com/category/programming/" rel="nofollow">Programming</a></span><span class="thecomments"><a href="http://devlup.com/programming/java-application-send-sms-free/950/#comments">30 Comments</a></span></p>

        <div class="format_text entry-content"> 
         <p>The problem with sending text to mobiles from your java program is that you’ve to spend few bucks buying GSM Modem, buy SIM with free SMS option and then connect it with Java Communication API. So here is a simple API that uses the help of <a rel="nofollow" href="http://wwwe.way2sms.com" target="_blank">way2sms</a> service to send free SMS.</p> 
         <p>Credits to <a rel="nofollow" href="http://www.aswinanand.com" target="_blank">Aswin Anand</a> and <a rel="nofollow" href="http://shaileshsoliwal.blogspot.com" target="_blank">Sailesh Soliwal</a> for developing the API</p> 
         <p>Steps:</p> 
         <p>1) Download the jar file from <a rel="nofollow" href="http://www.mediafire.com/?ztwzzwmfmmz" target="_blank">http://www.mediafire.com/?ztwzzwmfmmz</a></p> 
         <p>2) Add the jar file to library in your Netbeans or Eclipse IDE.</p> 
         <p>3) To send SMS call the Send() method of smsclient class.</p> 
         <p>4) Here is a sample implementation.</p> 
         <p style="padding-left: 30px;"><em>import com.way2sms.SMS;<br> class TestSMS {</em></p> 
         <p><em>public static void main(String[] args) {</em></p> 
         <p style="padding-left: 30px;"><em>SMS smsClient=new SMS();<br> smsClient.send( <span style="color: #000080;">Username, Password, Number, Message,Proxy</span>);<br> }<br> }</em></p> 
         <p>Where,</p> 
         <ul> 
          <li>Username &amp; Password are Your way2sms username and password</li> 
          <li>Number is recipient’s mobile number</li> 
          <li>Message is the text to be sent</li> 
          <li>and Proxy is the address of the proxy server. If you are using at home leave the proxy field as “” or null</li> 
         </ul> 
         <p>If you are using it in your college lab then you’ll have to connect internet with the help of proxy. But in my lab I couldn’t connect since my proxy server required authentication. If you can solve it please leave as comment.</p> 
         <p>The author <a rel="nofollow" href="http://shaileshsoliwal.blogspot.com/" target="_blank">Sailesh Soliwal</a> has made few applications with this API. You can check it <a rel="nofollow" href="http://shaileshsoliwal.blogspot.com/2009/10/send-free-sms-using-way2sms.html" target="_blank">here</a>.</p> 
         <p>If you are a web developer and require a similar API, don’t worry. You can use the same JAR file if your server has Java virtual machine or else <a rel="nofollow" href="http://www.aswinanand.com/" target="_blank">Aswin Anand</a> has given you a free php script. Check it <a rel="nofollow" href="http://www.aswinanand.com/2008/07/send-free-sms-web-service/" target="_blank">here</a>.</p>
