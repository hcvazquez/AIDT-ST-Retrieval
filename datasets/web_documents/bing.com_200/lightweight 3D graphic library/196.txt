     <div class="ipWidget ipWidget-Heading ipSkin-default"> 
      <h2 class="_title"> Key features </h2> 
     </div> 
    </div> 
   </div> 
   <div class="row"> 
    <div class="col-md-8"> 
     <div id="ipBlock-home3" data-revisionid="77" data-languageid="0" class="ipBlock">
      <div class="ipWidget ipWidget-Text ipSkin-default"> 
       <div class="ipsContainer"> 
        <h3></h3> 
        <ul> 
         <li><span><b>Different plotting types</b></span><br>Line Chart, Bar Chart and Points Chart<br>and they can be plotted together as a <b>combination</b>.&nbsp;<a href="http://www.android-graphview.org/mix-up-chart-types/" title="Mix up chart types">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b>Draw multiple series of data<br></b><span>Let the diagram show more that one series in a graph. You can set a color and a description for every series.</span>&nbsp;<a href="http://www.android-graphview.org/mix-up-chart-types/" title="mix up chart types">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b>Realtime / Live Chart<br></b>Append new data live or reset the whole data.&nbsp;<a href="http://www.android-graphview.org/realtime-chart/" title="Realtime chart">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b>Secondary Scale</b>&nbsp;<a href="http://www.android-graphview.org/secondary-scale-axis/" title="secondary scale axis">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b>Tap Listener</b><br>Handle tap events on specific data points.&nbsp;<a href="http://www.android-graphview.org/tap-listener-on-data/" title="tap listener on chart">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b><span>Show legend</span><br></b>A legend can be displayed inline the chart. You can set the width and the vertical align (top, middle, bottom).&nbsp;<a href="http://www.android-graphview.org/legend/" title="Legend">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b><span>Custom label formatter</span><br></b>The labels for the x- and y-axis are generated automatically. But you can set your own labels, Strings are possible.&nbsp;<a href="http://www.android-graphview.org/labels-and-label-formatter/" title="Labels">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b><span>Handle incomplete data</span><br></b>It's possible to give the data in different frequency.</li> 
        </ul> 
        <ul> 
         <li><span><b>Viewport</b></span><br>You can limit the viewport so that only a part of the data will be displayed.&nbsp;<a href="http://www.android-graphview.org/basics-of-graphview/" title="Basics of graph view">learn more</a></li> 
        </ul> 
        <ul> 
         <li><span><b>Scrolling and Scaling / Zooming</b></span><br>You can scroll with a finger touch move gesture.<br><span>With two-fingers touch scale gesture (Multi-touch), the viewport can be changed.</span>&nbsp;<a href="http://www.android-graphview.org/zooming-and-scrolling/" title="zooming and scrolling graph">learn more</a></li> 
        </ul> 
        <ul> 
         <li><span><b>XML Integration</b>&nbsp;<a href="http://www.android-graphview.org/create-graph-via-xml/">learn more</a><span></span></span></li> 
         <li><b>Optional Axis Titles</b><br>Set vertical and horizontal axis titles.&nbsp;<a href="http://www.android-graphview.org/style-options/">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b><span>Many Styles<br></span></b><span>change the color and thickness, label font size/color and more</span>&nbsp;<a href="http://www.android-graphview.org/style-options/">learn more</a></li> 
        </ul> 
        <ul> 
         <li><b>Very customizable</b><br>There are many hooks to use in order to do custom rendering or changing the paint objects.&nbsp;&nbsp;<a href="http://www.android-graphview.org/style-options/">learn more</a></li> 
        </ul> 
        <ul> 
         <li><span>.. and many more</span></li> 
         <li>Take a look at the <a href="https://play.google.com/store/apps/details?id=com.jjoe64.graphview_demos" target="_blank" title="">GraphView-Demo</a> project.</li> 
        </ul>
