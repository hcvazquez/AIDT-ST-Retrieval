        <p>#What is Laravel Authentication Acl?</p> 
        <p>Laravel Authentication ACL is a Laravel4 package (based on Sentry) for user registration, authentication and ACL. Also allow custom user profile info and fields and a configurable admin panel.</p> 
        <p>##Features:</p> 
        <ul> 
         <li>User authentication and signup</li> 
         <li>You can decide wether to use email confirmation or not</li> 
         <li>Can create groups and permissions and associate permissions to user or group</li> 
         <li>Any user can have multiple groups and permissions</li> 
         <li>Handles login throttling and password recovery</li> 
         <li>Configurable captcha integration</li> 
         <li>Password strength check</li> 
         <li>User banning</li> 
         <li>Dashboard</li> 
         <li>Infinite custom profile fields!</li> 
         <li>Can create custom menu items with configurable permissions</li> 
         <li>Have two login forms: admin area and user area</li> 
         <li>The code is fully tested with Phpunit</li> 
         <li>Many usable hasing algorithms sha256, md5 etc...</li> 
         <li>100% object oriented</li> 
         <li>Laravel4 based and easy to integrate in any Laravel application</li> 
         <li>Have an Api that integrates with your application</li> 
         <li>Bootstrap 3 and responsive design (mobile first)</li> 
         <li>Fully customizable and easy to extend</li> 
         <li>Easy install script from command line</li> 
        </ul> 
        <p>##Links:</p> 
        <ul> 
         <li> <a href="https://github.com/intrip/laravel-authentication-acl">Github</a> </li> 
         <li> <a href="https://packagist.org/packages/jacopo/laravel-authentication-acl">Packagist</a> </li> 
         <li> <a href="https://github.com/intrip/laravel-authentication-acl/blob/master/docs/index.md">Docs and screenshots</a> </li> 
        </ul> 
        <p>##Requirements:</p> 
        <ul> 
         <li>PHP 5.4+</li> 
         <li>Laravel 4</li> 
        </ul>
