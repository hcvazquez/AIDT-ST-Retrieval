       <h1 class="post-title" itemprop="name">Convert and Loop through JSON with PHP and JavaScript Arrays/Objects</h1> 
       <p class="post-meta"> <time content="2014-09-30" datetime="2014-09-30" class="post-meta-time" itemprop="datePublished">September 30, 2014</time> <span class="post-meta-author"> by <a href="/about/" class="post-meta-author-link" rel="author"><span itemprop="author" itemscope itemtype="http://schema.org/Person"><span itemprop="name">Jonathan Suh</span></span></a></span> </p> 
      </header> 
      <section class="post-content" itemprop="articleBody"> 
       <p>If you’re working with JSON (JavaScript Object Notation) and either need to convert a JSON string to array or object and loop through it or vice-versa, take an array or object and convert it to a JSON string to return, both can be done in PHP or JavaScript.</p> 
       <!--more--> 
       <p>I broke up this post into three sections:</p> 
       <ul> 
        <li><a href="#php">Working with PHP</a></li> 
        <li><a href="#javascript">Working with JavaScript</a></li> 
        <li><a href="#php-javascript">Working with both PHP and JavaScript</a></li> 
       </ul> 
       <p><a id="php"></a></p> 
       <h2 id="convert-json-string-to-php-array-or-object">Convert JSON String to PHP Array or Object</h2> 
       <p>PHP &gt;= 5.2.0 features a function, <code class="highlighter-rouge">json_decode</code>, that decodes a JSON string into a PHP variable. By default it returns an object. The second parameter accepts a boolean that when set as <code class="highlighter-rouge">true</code>, tells it to return the objects as associative arrays. You can learn more about the <code class="highlighter-rouge">json_decode</code> function from <a href="http://php.net/manual/en/function.json-decode.php" target="_blank">PHP’s documentation</a>.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="cp">&lt;?php</span>
  <span class="c1">// JSON string </span>  <span class="nv">$someJSON</span> <span class="o">=</span> <span class="s1">'[{"name":"Jonathan Suh","gender":"male"},{"name":"William Philbin","gender":"male"},{"name":"Allison McKinnery","gender":"female"}]'</span><span class="p">;</span>

  <span class="c1">// Convert JSON string to Array </span>  <span class="nv">$someArray</span> <span class="o">=</span> <span class="nb">json_decode</span><span class="p">(</span><span class="nv">$someJSON</span><span class="p">,</span> <span class="kc">true</span><span class="p">);</span>
  <span class="nb">print_r</span><span class="p">(</span><span class="nv">$someArray</span><span class="p">);</span>        <span class="c1">// Dump all data of the Array </span>  <span class="k">echo</span> <span class="nv">$someArray</span><span class="p">[</span><span class="mi">0</span><span class="p">][</span><span class="s2">"name"</span><span class="p">];</span> <span class="c1">// Access Array data </span>
  <span class="c1">// Convert JSON string to Object </span>  <span class="nv">$someObject</span> <span class="o">=</span> <span class="nb">json_decode</span><span class="p">(</span><span class="nv">$someJSON</span><span class="p">);</span>
  <span class="nb">print_r</span><span class="p">(</span><span class="nv">$someObject</span><span class="p">);</span>      <span class="c1">// Dump all data of the Object </span>  <span class="k">echo</span> <span class="nv">$someObject</span><span class="p">[</span><span class="mi">0</span><span class="p">]</span><span class="o">-&gt;</span><span class="na">name</span><span class="p">;</span> <span class="c1">// Access Object data </span><span class="cp">?&gt;</span>
</code></pre> 
       </div> 
       <h2 id="loop-through-php-array-or-object">Loop through PHP Array or Object</h2> 
       <p>Loop through a PHP array or object with a <code class="highlighter-rouge">foreach</code> loop.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="cp">&lt;?php</span>
  <span class="c1">// Loop through Array </span>  <span class="nv">$someArray</span> <span class="o">=</span> <span class="o">...</span><span class="p">;</span> <span class="c1">// Replace ... with your PHP Array </span>  <span class="k">foreach</span> <span class="p">(</span><span class="nv">$someArray</span> <span class="k">as</span> <span class="nv">$key</span> <span class="o">=&gt;</span> <span class="nv">$value</span><span class="p">)</span> <span class="p">{</span>
    <span class="k">echo</span> <span class="nv">$value</span><span class="p">[</span><span class="s2">"name"</span><span class="p">]</span> <span class="o">.</span> <span class="s2">", "</span> <span class="o">.</span> <span class="nv">$value</span><span class="p">[</span><span class="s2">"gender"</span><span class="p">]</span> <span class="o">.</span> <span class="s2">"&lt;br&gt;"</span><span class="p">;</span>
  <span class="p">}</span>

  <span class="c1">// Loop through Object </span>  <span class="nv">$someObject</span> <span class="o">=</span> <span class="o">...</span><span class="p">;</span> <span class="c1">// Replace ... with your PHP Object </span>  <span class="k">foreach</span><span class="p">(</span><span class="nv">$someObject</span> <span class="k">as</span> <span class="nv">$key</span> <span class="o">=&gt;</span> <span class="nv">$value</span><span class="p">)</span> <span class="p">{</span>
    <span class="k">echo</span> <span class="nv">$value</span><span class="o">-&gt;</span><span class="na">name</span> <span class="o">.</span> <span class="s2">", "</span> <span class="o">.</span> <span class="nv">$value</span><span class="o">-&gt;</span><span class="na">gender</span> <span class="o">.</span> <span class="s2">"&lt;br&gt;"</span><span class="p">;</span>
  <span class="p">}</span>
<span class="cp">?&gt;</span>
</code></pre> 
       </div> 
       <p>Note the differences in accessing the values of an array vs an object.</p> 
       <h2 id="convert-php-array-or-object-to-json-string">Convert PHP Array or Object to JSON String</h2> 
       <p>PHP also features a <code class="highlighter-rouge">json_encode</code> function to convert an array or object into a string. Read more about the <code class="highlighter-rouge">json_encode</code> function from <a href="http://php.net/manual/en/function.json-encode.php" target="_blank">PHP’s documentation</a>.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="cp">&lt;?php</span>
  <span class="c1">// Array </span>  <span class="nv">$someArray</span> <span class="o">=</span> <span class="p">[</span>
    <span class="p">[</span>
      <span class="s2">"name"</span>   <span class="o">=&gt;</span> <span class="s2">"Jonathan Suh"</span><span class="p">,</span>
      <span class="s2">"gender"</span> <span class="o">=&gt;</span> <span class="s2">"male"</span>
    <span class="p">],</span>
    <span class="p">[</span>
      <span class="s2">"name"</span>   <span class="o">=&gt;</span> <span class="s2">"William Philbin"</span><span class="p">,</span>
      <span class="s2">"gender"</span> <span class="o">=&gt;</span> <span class="s2">"male"</span>
    <span class="p">],</span>
    <span class="p">[</span>
      <span class="s2">"name"</span>   <span class="o">=&gt;</span> <span class="s2">"Allison McKinnery"</span><span class="p">,</span>
      <span class="s2">"gender"</span> <span class="o">=&gt;</span> <span class="s2">"female"</span>
    <span class="p">]</span>
  <span class="p">];</span>

  <span class="c1">// Convert Array to JSON String </span>  <span class="nv">$someJSON</span> <span class="o">=</span> <span class="nb">json_encode</span><span class="p">(</span><span class="nv">$someArray</span><span class="p">);</span>
  <span class="k">echo</span> <span class="nv">$someJSON</span><span class="p">;</span>
<span class="cp">?&gt;</span>
</code></pre> 
       </div> 
       <p>Note that I’m using the short array syntax that’s featured in PHP 5.4+.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="cp">&lt;?php</span>
  <span class="nv">$array</span> <span class="o">=</span> <span class="k">array</span><span class="p">(</span>
    <span class="s2">"foo"</span> <span class="o">=&gt;</span> <span class="s2">"bar"</span><span class="p">,</span>
    <span class="s2">"bar"</span> <span class="o">=&gt;</span> <span class="s2">"foo"</span>
  <span class="p">);</span>

  <span class="c1">// as of PHP 5.4 </span>  <span class="nv">$array</span> <span class="o">=</span> <span class="p">[</span>
    <span class="s2">"foo"</span> <span class="o">=&gt;</span> <span class="s2">"bar"</span><span class="p">,</span>
    <span class="s2">"bar"</span> <span class="o">=&gt;</span> <span class="s2">"foo"</span>
  <span class="p">];</span>
<span class="cp">?&gt;</span>
</code></pre> 
       </div> 
       <hr> 
       <p><a id="javascript"></a></p> 
       <h2 id="convert-json-string-to-javascript-object">Convert JSON String to JavaScript Object</h2> 
       <p>JavaScript has a built-in <code class="highlighter-rouge">JSON.parse()</code> method that parses a JSON string and returns an object.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="o">&lt;</span><span class="nx">script</span><span class="o">&gt;</span>
  <span class="c1">// Convert JSON String to JavaScript Object</span>
  <span class="kd">var</span> <span class="nx">JSONString</span> <span class="o">=</span> <span class="s1">'[{"name":"Jonathan Suh","gender":"male"},{"name":"William Philbin","gender":"male"},{"name":"Allison McKinnery","gender":"female"}]'</span><span class="p">;</span>

  <span class="kd">var</span> <span class="nx">JSONObject</span> <span class="o">=</span> <span class="nx">JSON</span><span class="p">.</span><span class="nx">parse</span><span class="p">(</span><span class="nx">JSONString</span><span class="p">);</span>
  <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">);</span>      <span class="c1">// Dump all data of the Object in the console</span>
  <span class="nx">alert</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">[</span><span class="mi">0</span><span class="p">][</span><span class="s2">"name"</span><span class="p">]);</span> <span class="c1">// Access Object data</span>
<span class="o">&lt;</span><span class="sr">/script</span><span class="err">&gt; </span></code></pre> 
       </div> 
       <p><code class="highlighter-rouge">JSON.parse()</code> is very well-supported, but there are browsers that do not support it (i.e. &lt;= IE 7. More information at <a href="http://caniuse.com/#feat=json" target="_blank">caniuse.com</a>).</p> 
       <p>jQuery 1.x has a <code class="highlighter-rouge">$.parseJSON()</code> method that should fill in the gaps for those browsers if you’re needing to support them. You can also use the <a href="https://github.com/douglascrockford/JSON-js" target="_blank">JSON-js</a> library as a polyfill.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="o">&lt;</span><span class="nx">script</span><span class="o">&gt;</span>
  <span class="c1">// Convert JSON String to JavaScript Object with jQuery</span>
  <span class="kd">var</span> <span class="nx">JSONString</span> <span class="o">=</span> <span class="s2">"..."</span><span class="p">;</span> <span class="c1">// Replace ... with your JSON String</span>

  <span class="kd">var</span> <span class="nx">JSONObject</span> <span class="o">=</span> <span class="nx">$</span><span class="p">.</span><span class="nx">parseJSON</span><span class="p">(</span><span class="nx">JSONString</span><span class="p">);</span>
  <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">);</span>      <span class="c1">// Dump all data of the Object in the console</span>
  <span class="nx">alert</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">[</span><span class="mi">0</span><span class="p">][</span><span class="s2">"name"</span><span class="p">]);</span> <span class="c1">// Access Object data</span>
<span class="o">&lt;</span><span class="sr">/script</span><span class="err">&gt; </span></code></pre> 
       </div> 
       <h2 id="loop-through-javascript-object">Loop through JavaScript Object</h2> 
       <p>You can then loop through a JavaScript object using a <code class="highlighter-rouge">for in</code> loop.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="o">&lt;</span><span class="nx">script</span><span class="o">&gt;</span>
  <span class="c1">// Loop through Object</span>
  <span class="kd">var</span> <span class="nx">JSONObject</span> <span class="o">=</span> <span class="p">...;</span> <span class="c1">// Replace ... with your JavaScript Object</span>

  <span class="k">for</span> <span class="p">(</span><span class="kd">var</span> <span class="nx">key</span> <span class="k">in</span> <span class="nx">JSONObject</span><span class="p">)</span> <span class="p">{</span>
    <span class="k">if</span> <span class="p">(</span><span class="nx">JSONObject</span><span class="p">.</span><span class="nx">hasOwnProperty</span><span class="p">(</span><span class="nx">key</span><span class="p">))</span> <span class="p">{</span>
      <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">[</span><span class="nx">key</span><span class="p">][</span><span class="s2">"name"</span><span class="p">]</span> <span class="o">+</span> <span class="s2">", "</span> <span class="o">+</span> <span class="nx">JSONObject</span><span class="p">[</span><span class="nx">key</span><span class="p">][</span><span class="s2">"gender"</span><span class="p">]);</span>
    <span class="p">}</span>
  <span class="p">}</span>
<span class="o">&lt;</span><span class="sr">/script</span><span class="err">&gt; </span></code></pre> 
       </div> 
       <h2 id="convert-javascript-object-to-json-string">Convert JavaScript Object to JSON String</h2> 
       <p>JavaScript has a <code class="highlighter-rouge">JSON.stringify</code> method to convert a value into a JSON string.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="o">&lt;</span><span class="nx">script</span><span class="o">&gt;</span>
  <span class="kd">var</span> <span class="nx">JSONObject</span> <span class="o">=</span> <span class="p">[</span>
    <span class="p">{</span>
      <span class="s2">"name"</span><span class="p">:</span> <span class="s2">"Jonathan Suh"</span><span class="p">,</span>
      <span class="s2">"gender"</span><span class="p">:</span> <span class="s2">"male"</span>
    <span class="p">},</span>
    <span class="p">{</span>
      <span class="s2">"name"</span><span class="p">:</span> <span class="s2">"William Philbin"</span><span class="p">,</span>
      <span class="s2">"gender"</span><span class="p">:</span> <span class="s2">"male"</span>
    <span class="p">},</span>
    <span class="p">{</span>
      <span class="s2">"name"</span><span class="p">:</span> <span class="s2">"Allison McKinnery"</span><span class="p">,</span>
      <span class="s2">"gender"</span><span class="p">:</span> <span class="s2">"female"</span>
    <span class="p">}</span>
  <span class="p">];</span>

  <span class="kd">var</span> <span class="nx">JSONString</span> <span class="o">=</span> <span class="nx">JSON</span><span class="p">.</span><span class="nx">stringify</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">);</span>
  <span class="nx">alert</span><span class="p">(</span><span class="nx">JSONString</span><span class="p">);</span>
<span class="o">&lt;</span><span class="sr">/script</span><span class="err">&gt; </span></code></pre> 
       </div> 
       <p>Like <code class="highlighter-rouge">JSON.parse</code>, <code class="highlighter-rouge">JSON.stringify</code> is not supported in dinosaur browsers like &lt;= IE 7. You can use the <a href="https://github.com/douglascrockford/JSON-js" target="_blank">JSON-js library</a> to polyfill <code class="highlighter-rouge">JSON.stringify</code> as well.</p> 
       <hr> 
       <p><a id="php-javascript"></a></p> 
       <p>You can combine the methods above to create powerful, dynamic implementations on your website or application.</p> 
       <p>Let’s say you want to get information from a database, safely return the data as JSON, and loop through it dynamically, you can do so with a bit of PHP and JavaScript with Ajax.</p> 
       <h2 id="dynamically-get-json-via-ajax-and-loop-through-json">Dynamically Get JSON via Ajax and Loop Through JSON</h2> 
       <p>Let’s assume your database structure looks like the following:</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code>Table: people
?????????????????????????????????????
| id | name               | gender  |
?????????????????????????????????????
| 0  | Jonathan Suh       | male    |
| 1  | William Philbin    | male    |
| 2  | Allison McKinnery  | female  |
| 3  | Becky Borgster     | female  |
| 4  | Victoria Einsteen  | female  |
?????????????????????????????????????
</code></pre> 
       </div> 
       <p>And you want to dynamically get a list of people from the database based on gender, like this:</p> 
       <p><img src="/assets/images/blog/2014/convert-loop-through-json-php-javascript-arrays-objects/dynamic-select.gif" style="max-width: 18.750em"></p> 
       <p>Let’s start with the front-end file <code class="highlighter-rouge">index.html</code> that’ll have a select dropdown with genders to select from, a table to display the results, and the script to handle the Ajax. The JavaScript is written in jQuery.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="nt">&lt;select</span> <span class="na">id=</span><span class="s">"gender"</span> <span class="na">name=</span><span class="s">"gender"</span><span class="nt">&gt;</span>
  <span class="nt">&lt;option</span> <span class="na">value=</span><span class="s">"male"</span><span class="nt">&gt;</span>Male<span class="nt">&lt;/option&gt;</span>
  <span class="nt">&lt;option</span> <span class="na">value=</span><span class="s">"female"</span><span class="nt">&gt;</span>Female<span class="nt">&lt;/option&gt;</span>
<span class="nt">&lt;/select&gt;</span>

<span class="nt">&lt;table</span> <span class="na">id=</span><span class="s">"people"</span> <span class="na">border=</span><span class="s">"1"</span><span class="nt">&gt;</span>
  <span class="nt">&lt;thead&gt;</span>
    <span class="nt">&lt;th&gt;</span>Name<span class="nt">&lt;/th&gt;</span>
    <span class="nt">&lt;th&gt;</span>Gender<span class="nt">&lt;/th&gt;</span>
  <span class="nt">&lt;/thead&gt;</span>
  <span class="nt">&lt;tbody&gt;</span>

  <span class="nt">&lt;/tbody&gt;</span>
<span class="nt">&lt;/table&gt;</span>

<span class="nt">&lt;script </span><span class="na">src=</span><span class="s">"http://code.jquery.com/jquery-1.11.1.min.js"</span><span class="nt">&gt;&lt;/script&gt;</span>
<span class="nt">&lt;script&gt;</span>
<span class="nx">$</span><span class="p">(</span><span class="s2">"#gender"</span><span class="p">).</span><span class="nx">on</span><span class="p">(</span><span class="s2">"change"</span><span class="p">,</span> <span class="kd">function</span><span class="p">()</span> <span class="p">{</span>
  <span class="nx">$</span><span class="p">.</span><span class="nx">ajax</span><span class="p">({</span>
    <span class="na">type</span><span class="p">:</span> <span class="s2">"POST"</span><span class="p">,</span>
    <span class="na">data</span><span class="p">:</span> <span class="p">{</span>
      <span class="s2">"gender"</span><span class="p">:</span> <span class="nx">$</span><span class="p">(</span><span class="s2">"#gender"</span><span class="p">).</span><span class="nx">val</span><span class="p">()</span>
    <span class="p">},</span>
    <span class="na">url</span><span class="p">:</span> <span class="s2">"response.php"</span><span class="p">,</span>
    <span class="na">dataType</span><span class="p">:</span> <span class="s2">"json"</span><span class="p">,</span>
    <span class="na">success</span><span class="p">:</span> <span class="kd">function</span><span class="p">(</span><span class="nx">JSONObject</span><span class="p">)</span> <span class="p">{</span>
      <span class="kd">var</span> <span class="nx">peopleHTML</span> <span class="o">=</span> <span class="s2">""</span><span class="p">;</span>

      <span class="c1">// Loop through Object and create peopleHTML</span>
      <span class="k">for</span> <span class="p">(</span><span class="kd">var</span> <span class="nx">key</span> <span class="k">in</span> <span class="nx">JSONObject</span><span class="p">)</span> <span class="p">{</span>
        <span class="k">if</span> <span class="p">(</span><span class="nx">JSONObject</span><span class="p">.</span><span class="nx">hasOwnProperty</span><span class="p">(</span><span class="nx">key</span><span class="p">))</span> <span class="p">{</span>
          <span class="nx">peopleHTML</span> <span class="o">+=</span> <span class="s2">"&lt;tr&gt;"</span><span class="p">;</span>
            <span class="nx">peopleHTML</span> <span class="o">+=</span> <span class="s2">"&lt;td&gt;"</span> <span class="o">+</span> <span class="nx">JSONObject</span><span class="p">[</span><span class="nx">key</span><span class="p">][</span><span class="s2">"name"</span><span class="p">]</span> <span class="o">+</span> <span class="s2">"&lt;/td&gt;"</span><span class="p">;</span>
            <span class="nx">peopleHTML</span> <span class="o">+=</span> <span class="s2">"&lt;td&gt;"</span> <span class="o">+</span> <span class="nx">JSONObject</span><span class="p">[</span><span class="nx">key</span><span class="p">][</span><span class="s2">"gender"</span><span class="p">]</span> <span class="o">+</span> <span class="s2">"&lt;/td&gt;"</span><span class="p">;</span>
          <span class="nx">peopleHTML</span> <span class="o">+=</span> <span class="s2">"&lt;/tr&gt;"</span><span class="p">;</span>
        <span class="p">}</span>
      <span class="p">}</span>

      <span class="c1">// Replace table’s tbody html with peopleHTML</span>
      <span class="nx">$</span><span class="p">(</span><span class="s2">"#people tbody"</span><span class="p">).</span><span class="nx">html</span><span class="p">(</span><span class="nx">peopleHTML</span><span class="p">);</span>
    <span class="p">}</span>
  <span class="p">});</span>
<span class="p">});</span>
<span class="nt">&lt;/script&gt;</span>
</code></pre> 
       </div> 
       <p>Now let’s create a <code class="highlighter-rouge">response.php</code> file to handle the back-end logic of getting the information from the database and returning the results as a JSON string.</p> 
       <div class="highlighter-rouge">
        <pre class="highlight"><code><span class="cp">&lt;?php</span>
  <span class="c1">// File: response.php </span>
  <span class="c1">// Get POST gender value </span>  <span class="nv">$gender</span> <span class="o">=</span> <span class="nv">$_POST</span><span class="p">[</span><span class="s2">"gender"</span><span class="p">];</span>

  <span class="c1">// Connect to the database </span>  <span class="c1">// replace the parameters with your proper credentials </span>  <span class="nv">$connection</span> <span class="o">=</span> <span class="nb">mysqli_connect</span><span class="p">(</span><span class="s2">"localhost"</span><span class="p">,</span> <span class="s2">"username"</span><span class="p">,</span> <span class="s2">"password"</span><span class="p">,</span> <span class="s2">"database_name"</span><span class="p">);</span>

  <span class="c1">// Query to run </span>  <span class="nv">$query</span> <span class="o">=</span> <span class="nx">mysqli_query</span><span class="p">(</span><span class="nv">$connection</span><span class="p">,</span>
           <span class="s2">"SELECT * FROM people WHERE gender = '"</span> <span class="o">.</span> <span class="nv">$gender</span> <span class="o">.</span> <span class="s2">"'"</span><span class="p">);</span>

  <span class="c1">// Create empty array to hold query results </span>  <span class="nv">$someArray</span> <span class="o">=</span> <span class="p">[];</span>

  <span class="c1">// Loop through query and push results into $someArray; </span>  <span class="k">while</span> <span class="p">(</span><span class="nv">$row</span> <span class="o">=</span> <span class="nx">mysqli_fetch_assoc</span><span class="p">(</span><span class="nv">$query</span><span class="p">))</span> <span class="p">{</span>
    <span class="nb">array_push</span><span class="p">(</span><span class="nv">$someArray</span><span class="p">,</span> <span class="p">[</span>
      <span class="s1">'name'</span>   <span class="o">=&gt;</span> <span class="nv">$row</span><span class="p">[</span><span class="s1">'name'</span><span class="p">],</span>
      <span class="s1">'gender'</span> <span class="o">=&gt;</span> <span class="nv">$row</span><span class="p">[</span><span class="s1">'gender'</span><span class="p">]</span>
    <span class="p">]);</span>
  <span class="p">}</span>

  <span class="c1">// Convert the Array to a JSON String and echo it </span>  <span class="nv">$someJSON</span> <span class="o">=</span> <span class="nb">json_encode</span><span class="p">(</span><span class="nv">$someArray</span><span class="p">);</span>
  <span class="k">echo</span> <span class="nv">$someJSON</span><span class="p">;</span>
<span class="cp">?&gt;</span>
</code></pre> 
       </div> 
       <p>To get a more in-depth and better example of PHP-JSON-JavaScript/jQuery-Ajax interaction, read my <a href="/blog/jquery-ajax-call-to-php-script-with-json-return/">jQuery Ajax Call to PHP Script with JSON Return</a> post.</p>
