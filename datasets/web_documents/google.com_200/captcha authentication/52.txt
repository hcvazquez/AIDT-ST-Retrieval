     <div class="headline">
      <h2>Latest Open Source Project</h2>
     </div> 
     <h3>Laravel 5.3 Social and Email Multi-Authentication</h3> 
     <p>This project aims to simplify common task in Laravel web application development and that is building user authentication system.</p> 
     <p>Almost every web app requires login via email and now many require login via oAuth providers. This project covers it all including Facebook, Twitter, Google+, GitHub and many more. System is using Socialite Laravel package to cover OAuth part, and it also posses email activation feature, it handles denied OAuth requests, User roles, Google ReCaptcha, Parsley.js validation and many more features are coming. </p> 
     <p>Entire project is covered with series of <a href="https://tuts.codingo.me/laravel-multi-auth-app-social-logins">tutorials</a> if you are curious to see how it looks live we deployed it <a href="https://demo1.codingo.me/">here</a>.</p> 
     <p><a href="https://github.com/codingo-me/laravel-social-email-authentication"><i class="fa fa-github"></i> codingo-me/laravel-social-email-authentication</a></p>
