       <h2>QuickStart</h2> 
       <h3>Step 1: Include angular-fusioncharts.js</h3> 
       <p>In your HTML, include <code>angular-fusioncharts.js</code> after all other scripts:</p> 
       <pre class="language-markup"><code>&lt;script type="text/javascript" src="/path/to/fusioncharts.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="/path/to/angular.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="/path/to/angular-fusioncharts.js"&gt;&lt;/script&gt;</code></pre> 
       <h3>Step 2: Include <code>ng-fusioncharts</code> in your module</h3> 
       <p>In the app, include <code>ng-fusioncharts</code> as a dependency. If you looking for where to add the dependency, look for the call to <code>angular.module</code> in your code.</p> 
       <pre class="language-javascript"><code>angular.module("myApp", ["ng-fusioncharts"])</code></pre> 
       <h3>Step 3: Add the <code>fusioncharts</code> directive</h3> 
       <p>In your HTML, find the section where you wish to add the chart and add a <code>&lt;div&gt;</code> with the <code>fc-chart</code> directive. We are assuming it's inside a controller called <code>MyController</code> which would change based on your usage.</p> 
       <pre class="language-markup"><code class="ng-binding">&lt;div ng-controller="MyController"&gt;
    &lt;div fusioncharts
       width="600"
       height="400"
       type="column2d"
       dataSource="" &gt;
    &lt;/div&gt;
&lt;/div&gt;</code></pre> 
       <p>Now this is bound to a datasource with the <code>myDataSource</code> scope object.</p> 
       <h3>Step 4: Populate required variables in controller</h3> 
       <p>In the previous code, we are binding to a scope variable <code>myDataSource</code>, but that hasn't been defined yet.</p> 
       <p>In your controller, set the DataSource as you would for a regular FusionCharts JSON Format DataSource (see <a href="http://docs.fusioncharts.com/tutorial-getting-started-your-first-charts-building-your-first-chart.html">this tutorial</a> for a general introduction to this format).</p> 
       <pre class="language-javascript"><code>app.controller('MyController', function ($scope) {
              $scope.myDataSource = {
                chart: {
                    caption: "Harry's SuperMart",
                    subCaption: "Top 5 stores in last month by revenue",
                },
                data:[{
                    label: "Bakersfield Central",
                    value: "880000"
                },
                {
                    label: "Garden Groove harbour",
                    value: "730000"
                },
                {
                    label: "Los Angeles Topanga",
                    value: "590000"
                },
                {
                    label: "Compton-Rancho Dom",
                    value: "520000"
                },
                {
                    label: "Daly City Serramonte",
                    value: "330000"
                }]
              };
            });</code></pre> 
       <p>And your chart should display when you load the page. </p>
