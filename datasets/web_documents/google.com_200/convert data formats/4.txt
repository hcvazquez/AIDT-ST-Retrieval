  -t, --text &lt;string&gt;             Input text

  -s, --output-style &lt;option&gt;     Ouput scheme. A combination of --output-format json and --output-style citation-* is considered invalid. Options: csl (Citation Style Lanugage JSON), bibtex, citation-* (where * is any formatting style)

         <p>The value passed to the append/prepend options is either</p> 
         <ol> 
          <li>a constant string or</li> 
          <li>a callback taking a parameter <code>entry</code>, a CSL-JSON object, and returning HTML or plain text</li> 
         </ol> 
         <p>that should either be appended or prepended to the corresponding entry in the outputted bibliography. Example:</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-k">const</span> <span class="pl-c1">Cite</span> <span class="pl-k">=</span> <span class="pl-c1">require</span>(<span class="pl-s"><span class="pl-pds">'</span>citation-js<span class="pl-pds">'</span></span>)
<span class="pl-k">const</span> <span class="pl-c1">data</span> <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-en">Cite</span>(<span class="pl-s"><span class="pl-pds">'</span>Q30000000<span class="pl-pds">'</span></span>)

<span class="pl-k">const</span> <span class="pl-c1">date</span> <span class="pl-k">=</span> (<span class="pl-k">new</span> <span class="pl-en">Date</span>()).<span class="pl-en">toLocaleDateString</span>()

<span class="pl-smi">data</span>.<span class="pl-c1">get</span>({
  type<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>html<span class="pl-pds">'</span></span>,
  style<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>citation-apa<span class="pl-pds">'</span></span>,
  <span class="pl-en">prepend</span> ({id}) {

         <p><code>Cite</code> instances have some more functions:</p> 
         <ul> 
          <li><code>Cite#options(&lt;options&gt;)</code>: Change default options</li> 
          <li><code>Cite#set(&lt;data&gt;)</code>: Replace all data with new data</li> 
          <li><code>Cite#add(&lt;data&gt;)</code>: Add data</li> 
          <li><code>Cite#reset()</code>: Remove all data and options</li> 
          <li><code>Cite#currentVersion()</code>: Get current version number</li> 
          <li><code>Cite#retrieveVersion(&lt;version number&gt;)</code>: Retrieve a certain version of the object</li> 
          <li><code>Cite#retrieveLastVersion()</code>: Retrieve the last saved version of the object</li> 
          <li><code>Cite#undo(&lt;number&gt;)</code>: Restore the n to last version (default: <code>1</code>)</li> 
          <li><code>Cite#save()</code>: Save the current object</li> 
          <li><code>Cite#sort()</code>: Sort all entries on basis of their BibTeX label</li> 
         </ul> 
         <p><code>Cite#set()</code> and <code>Cite#get()</code> also have async variants (<code>Cite#setAsync()</code> and <code>Cite#addAsync()</code>), which return Promises.</p>
