       <h1 itemprop="name" class="devsite-page-title"> Java on Google App Engine </h1> 
       <nav class="devsite-page-nav-embedded devsite-nav"></nav> 
       <div class="devsite-article-body clearfix
            " itemprop="articleBody"> 
        <div class="docs-landing-page-main"> 
         <section class="article"> 
          <article id="cloud-site" class="grid-fluid l-fixed-width"> 
           <section class="intro" style="padding-top: 20px;">
             App Engine offers you a choice between two environments for Java applications: flexible environment and standard environment, with Java 7 and Java 8 beta runtimes for standard. Both environments have the same code-centric developer workflow, scale quickly and efficiently to handle increasing demand, and enable you to use Googles proven serving technology to build your web, mobile, and IoT applications quickly and with minimal operational overhead. While the available environments have a lot in common, they differ in a few important ways. 
           </section> 
           <div> 
            <h2>Choose your preferred environment</h2> 
           </div> 
           <div class="l-clearfix"> 
            <div id="environments" class="l-clearfix flex"> 
             <div class="c6 card l-pad-3"> 
              <div class="inner-card"> 
               <h3 class="hide-from-toc"> Standard environment Java 8 </h3> 
               <ul> 
                <li> <p>Has all the benefits of Java 7 runtime but with upgrades and enhancements.</p> </li> 
                <li> <p>OpenJDK 8-based Java runtime supports the standard public Java library (no whitelist).</p> </li> 
                <li> <p>Doesn't impose a security manager--your code won't be restricted by Java permissions issues.</p> </li> 
                <li> <p><a href="https://cloud.google.com/appengine/docs/standard/java/runtime-java8">Java 8 runtime documentation</a></p> </li> 
               </ul> 
              </div> 
              <div class="learn-more"> 
               <a href="https://cloud.google.com/appengine/docs/standard/java/runtime-java8" track-type="docsLandingPage" track-name="standardDocs"> View Docs </a> 
              </div> 
             </div> 
             <div class="c6 card l-pad-3"> 
              <div class="inner-card"> 
               <h3 class="hide-from-toc"> Standard environment Java 7 </h3> 
               <ul> 
                <li> <p>Lightweight Java 7 runtime is optimized to scale nearly instantaneously to handle huge traffic spikes.</p> </li> 
                <li> <p>The performance optimized programming model limits the use of native code, access to the file system, and some network activities. <a href="https://cloud.google.com/appengine/docs/java/runtime">Learn more</a></p> </li> 
                <li> <p>Uses proprietary APIs to simplify common tasks like database access, queuing, and in-memory caching.</p> </li> 
                <li> <p>Most cost-effective for applications that have significant periods where they are not serving traffic.</p> </li> 
                <li> <p><a href="https://cloud.google.com/appengine/docs/standard/java/runtime">Java 7 runtime documentation</a></p> </li> 
               </ul> 
              </div> 
              <div class="learn-more"> 
               <a href="https://cloud.google.com/appengine/docs/standard/java/" track-type="docsLandingPage" track-name="standardDocs"> View Docs </a> 
              </div> 
             </div> 
             <div class="c6 card l-pad-3"> 
              <div class="inner-card"> 
               <h3 class="hide-from-toc">Flexible environment</h3> 
               <ul> 
                <li> <p>Full <a href="http://github.com/GoogleCloudPlatform/">open source</a> language runtimes.</p> </li> 
                <li> <p>Use any framework, library, or binary of your choice.</p> </li> 
                <li> <p>Code is portable to anywhere that supports Docker containers.</p> </li> 
                <li> <p>Most cost-effective for applications that serve traffic continuously.</p> </li> 
                <li> <p><a href="https://cloud.google.com/appengine/docs/flexible/java/dev-jetty9">Java 8 runtime documentation</a></p> </li> 
               </ul> 
              </div> 
              <div class="learn-more"> 
               <a href="https://cloud.google.com/appengine/docs/flexible/java/" track-type="docsLandingPage" track-name="FlexibleDocs"> View Docs </a> 
              </div> 
             </div> 
            </div> 
           </div> 
           <div id="choosing" class="l-clearfix gray l-pad-7"> 
            <div class="c3 flex-3 align-items-center first justify-right l-pad-5"> 
             <h2>Choosing an environment</h2> 
            </div> 
            <div class="choosing-text c9 l-pad-5"> 
             <div class="inner-text l-pad-bottom-5">
               You can run an App Engine application in two environments, the standard environment and the flexible environment. See a summary of differences between the two environments. 
             </div> 
             <a href="https://cloud.google.com/appengine/docs/the-appengine-environments" class="button" track-type="docsLandingPage" track-name="choosingEnvironment"> Learn more </a> 
            </div> 
           </div>
