    <h1 id="GUI_Programming_in_Python">GUI Programming in Python</h1> 
    <span class="anchor" id="line-3"></span>
    <p class="line862">Python has a huge number of GUI frameworks (or toolkits) available for it, from <a href="/moin/TkInter">TkInter</a> (traditionally bundled with Python, using Tk) to a number of other cross-platform solutions, as well as bindings to platform-specific (also known as "native") technologies. <span class="anchor" id="line-4"></span><span class="anchor" id="line-5"></span></p>
    <p class="line867"><span class="anchor" id="line-6"></span><span class="anchor" id="line-7"></span></p>
    <div class="note">
     <span class="anchor" id="line-1-1"></span>
     <p class="line867"><a href="/moin/GUI%20Programming%20in%20Python">GUI Programming in Python</a> is a similar page whose content could arguably complement this page with some editing. </p>
    </div>

An up-to-date book covering <a href="/moin/PyQt4">PyQt4</a>, <a class="http" href="http://www.qtrac.eu/pyqtbook.html">Rapid GUI Programming with Python and Qt</a> by Mark Summerfield, is now available in print and as an online "rough cut".     <h2 id="Editorial_Notes">Editorial Notes</h2> 
    <span class="anchor" id="line-124"></span>
    <p class="line874">The above lists should be arranged in ascending alphabetical order - please respect this when adding new frameworks or tools. <span class="anchor" id="line-125"></span><span class="anchor" id="line-126"></span></p>
    <p class="line862">Currently, this section is rather incomplete. Don't be too disappointed if you can't find the information you need, but please add any information you can contribute. It might be useful to have information about the GUIs and development tools in a fixed format to make them more easily comparable (see <a href="/moin/WebProgrammingTemplate">WebProgrammingTemplate</a> and <a href="/moin/WebProgrammingTemplateHelp">WebProgrammingTemplateHelp</a>). Some categories: directly supported widgets, add-on widgets, layout managers, etc. Maybe it could answer the questions in <a href="/moin/ChoosingGuiToolkits">ChoosingGuiToolkits</a> for each toolkit. <span class="anchor" id="line-127"></span><span class="anchor" id="line-128"></span></p>
