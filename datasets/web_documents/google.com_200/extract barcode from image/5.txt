                <div class="jive-rendered-content">
                 <p>There are two ways to create a barcode on a report.</p>
                 <p>- You can use a barcode font.</p>
                 <p>- Use a Java package that creates the barcode as an image.</p>
                 <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                 <p>It depends on your reporting tool what you can use. Here is an example for a Java package:</p>
                 <p><a class="jive-link-external-small" href="http://www.idautomation.com/barcode-components/java/" rel="nofollow">http://www.idautomation.com/barcode-components/java/</a></p>
                 <p>(barcode fonts can be found on the same website).</p>
                 <p style="min-height: 8pt; padding: 0px;">&nbsp;</p>
                 <p>There is (should be) nothing special about scanning a barcode. The only thing a barcode scanner does is translating a barcode into a readable code. So, if your cursor is in some text field, and you point the scanner to a barcode, the code will be displayed in the text field in readable text. It's just like typing the code via your keyboard.</p>
