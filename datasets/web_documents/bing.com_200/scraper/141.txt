       <h1 title="Nuspec reference: <title>Java Runtime (JRE)</title>">Java Runtime (JRE)</h1> 
       <h2 title="Nuspec reference: <version>8.0.144</version>">8.0.144</h2> 
       <h2> <a href="https://gist.github.com/ac7fd5db72df5594659fd2bcd4ff4201"><img src="/content/images/green_ball_48.png" title="Package test results are passing. Follow the link for more information." alt="Package test results are passing."></a> </h2> 
      </div> 
      <p class="message approved">This package was approved as <a href="https://chocolatey.org/faq#what-is-a-trusted-package">a trusted package</a> on 8/11/2017.</p> 
      <a name="description" id="description" title="Nuspec reference: description"></a> 
      <p>Java allows you to play online games, chat with people around the world, calculate your mortgage interest, and view images in 3D, just to name a few. It's also integral to the intranet applications and other e-business solutions that are the foundation of corporate computing.</p> 
      <h3>Note</h3> 
      <p>This is just a metapackage for the latest JRE (currenly JRE 8). Have a look at the <a href="/packages/jre8">jre8 package</a> for more details.</p> 
      <p>To install Java Runtime (JRE), run the following command from the command line or from PowerShell: </p> 
      <div class="nuget-badge"> 
       <p><code><span>C:\&gt; </span>choco install javaruntime </code></p> 
      </div> 
      <p>To upgrade Java Runtime (JRE), run the following command from the command line or from PowerShell: </p> 
      <div class="nuget-badge"> 
       <p><code><span>C:\&gt; </span>choco upgrade javaruntime </code></p> 
      </div>
