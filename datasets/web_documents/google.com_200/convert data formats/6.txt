    <h1 id="units">Units <a href="#units" title="Permalink">#</a></h1> 
    <p>Math.js supports units. Units can be used to do calculations and to perform conversions.</p> 
    <h2 id="usage">Usage <a href="#usage" title="Permalink">#</a></h2> 
    <p>Units can be created using the function <code class="highlighter-rouge">math.unit</code>. This function accepts either a single string argument containing a value and unit, or two arguments, the first being a numeric value and the second a string containing a unit. Most units support prefixes like <code class="highlighter-rouge">k</code> or <code class="highlighter-rouge">kilo</code>, and many units have both a full name and an abbreviation. The returned object is a <code class="highlighter-rouge">Unit</code>.</p> 
    <p>Syntax:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="nx">value</span><span class="err">:</span> <span class="nx">number</span><span class="p">,</span> <span class="nx">name</span><span class="err">:</span> <span class="nx">string</span><span class="p">)</span> <span class="err">:</span> <span class="nx">Unit</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="nx">unit</span><span class="err">:</span> <span class="nx">string</span><span class="p">)</span> <span class="err">:</span> <span class="nx">Unit</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="nx">unit</span><span class="err">:</span> <span class="nx">Unit</span><span class="p">)</span> <span class="err">:</span> <span class="nx">Unit</span>
</code></pre> 
    </div> 
    <p>Example usage:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="kd">var</span> <span class="nx">a</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="mi">45</span><span class="p">,</span> <span class="s1">'cm'</span><span class="p">);</span>            <span class="c1">// Unit 450 mm</span>
<span class="kd">var</span> <span class="nx">b</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'0.1 kilogram'</span><span class="p">);</span>      <span class="c1">// Unit 100 gram</span>
<span class="kd">var</span> <span class="nx">c</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'2 inch'</span><span class="p">);</span>            <span class="c1">// Unit 2 inch</span>
<span class="kd">var</span> <span class="nx">d</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'90 km/h'</span><span class="p">);</span>           <span class="c1">// Unit 90 km/h</span>
<span class="kd">var</span> <span class="nx">e</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'101325 kg/(m s^2)'</span><span class="p">);</span> <span class="c1">// Unit 101325 kg / (m s^2)</span>
</code></pre> 
    </div> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="kd">var</span> <span class="nx">a</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="mi">55</span><span class="p">,</span> <span class="s1">'cm'</span><span class="p">);</span>        <span class="c1">// Unit 550 mm</span>
<span class="kd">var</span> <span class="nx">b</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'0.1 kilogram'</span><span class="p">);</span>  <span class="c1">// Unit 100 gram</span>
<span class="kd">var</span> <span class="nx">c</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'2 inch'</span><span class="p">);</span>        <span class="c1">// Unit 100 millimeter</span>

<span class="kd">var</span> <span class="nx">d</span> <span class="o">=</span> <span class="nx">c</span><span class="p">.</span><span class="nx">to</span><span class="p">(</span><span class="s1">'cm'</span><span class="p">);</span>                 <span class="c1">// Unit 5.08 cm</span>
<span class="nx">b</span><span class="p">.</span><span class="nx">toNumber</span><span class="p">(</span><span class="s1">'gram'</span><span class="p">);</span>                 <span class="c1">// Number 100</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">number</span><span class="p">(</span><span class="nx">b</span><span class="p">,</span> <span class="s1">'gram'</span><span class="p">);</span>             <span class="c1">// Number 100</span>

<span class="nx">c</span><span class="p">.</span><span class="nx">equals</span><span class="p">(</span><span class="nx">a</span><span class="p">);</span>                        <span class="c1">// false</span>
<span class="nx">c</span><span class="p">.</span><span class="nx">equals</span><span class="p">(</span><span class="nx">d</span><span class="p">);</span>                        <span class="c1">// true</span>
<span class="nx">c</span><span class="p">.</span><span class="nx">equalBase</span><span class="p">(</span><span class="nx">a</span><span class="p">);</span>                     <span class="c1">// true</span>
<span class="nx">c</span><span class="p">.</span><span class="nx">equalBase</span><span class="p">(</span><span class="nx">b</span><span class="p">);</span>                     <span class="c1">// false</span>

<span class="nx">d</span><span class="p">.</span><span class="nx">toString</span><span class="p">();</span>                       <span class="c1">// String "5.08 cm"</span>
</code></pre> 
    </div> 
    <p>Use care when creating a unit with multiple terms in the denominator. Implicit multiplication has the same operator precedence as explicit multiplication and division, which means these three expressions are identical:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="c1">// These three are identical</span>
<span class="kd">var</span> <span class="nx">correct1</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'8.314 m^3 Pa / mol / K'</span><span class="p">);</span>         <span class="c1">// Unit 8.314 (m^3 Pa) / (mol K)</span>
<span class="kd">var</span> <span class="nx">correct2</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'8.314 (m^3 Pa) / (mol K)'</span><span class="p">);</span>       <span class="c1">// Unit 8.314 (m^3 Pa) / (mol K)</span>
<span class="kd">var</span> <span class="nx">correct3</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'8.314 (m^3 * Pa) / (mol * K)'</span><span class="p">);</span>   <span class="c1">// Unit 8.314 (m^3 Pa) / (mol K)</span>
</code></pre> 
    </div> 
    <p>But this expression, which omits the second <code class="highlighter-rouge">/</code> between <code class="highlighter-rouge">mol</code> and <code class="highlighter-rouge">K</code>, results in the wrong value:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="c1">// Missing the second '/' between 'mol' and 'K'</span>
<span class="kd">var</span> <span class="nx">incorrect</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'8.314 m^3 Pa / mol K'</span><span class="p">);</span>          <span class="c1">// Unit 8.314 (m^3 Pa K) / mol</span>
</code></pre> 
    </div> 
    <h2 id="calculations">Calculations <a href="#calculations" title="Permalink">#</a></h2> 
    <p>The operations that support units are <code class="highlighter-rouge">add</code>, <code class="highlighter-rouge">subtract</code>, <code class="highlighter-rouge">multiply</code>, <code class="highlighter-rouge">divide</code>, <code class="highlighter-rouge">pow</code>, <code class="highlighter-rouge">abs</code>, <code class="highlighter-rouge">sqrt</code>, <code class="highlighter-rouge">square</code>, <code class="highlighter-rouge">cube</code>, and <code class="highlighter-rouge">sign</code>. Trigonometric functions like <code class="highlighter-rouge">cos</code> are also supported when the argument is an angle.</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="kd">var</span> <span class="nx">a</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="mi">45</span><span class="p">,</span> <span class="s1">'cm'</span><span class="p">);</span>        <span class="c1">// Unit 450 mm</span>
<span class="kd">var</span> <span class="nx">b</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'0.1m'</span><span class="p">);</span>          <span class="c1">// Unit 100 mm</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">add</span><span class="p">(</span><span class="nx">a</span><span class="p">,</span> <span class="nx">b</span><span class="p">);</span>                     <span class="c1">// Unit 0.65 m</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">multiply</span><span class="p">(</span><span class="nx">b</span><span class="p">,</span> <span class="mi">2</span><span class="p">);</span>                <span class="c1">// Unit 200 mm</span>

<span class="kd">var</span> <span class="nx">c</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="mi">45</span><span class="p">,</span> <span class="s1">'deg'</span><span class="p">);</span>       <span class="c1">// Unit 45 deg</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">cos</span><span class="p">(</span><span class="nx">c</span><span class="p">);</span>                        <span class="c1">// Number 0.7071067811865476</span>

<span class="c1">// Kinetic energy of average sedan on highway</span>
<span class="kd">var</span> <span class="nx">d</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'80 mi/h'</span><span class="p">)</span>        <span class="c1">// Unit 80 mi/h</span>
<span class="kd">var</span> <span class="nx">e</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'2 tonne'</span><span class="p">)</span>        <span class="c1">// Unit 2 tonne</span>
<span class="kd">var</span> <span class="nx">f</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">multiply</span><span class="p">(</span><span class="mf">0.5</span><span class="p">,</span> <span class="nx">math</span><span class="p">.</span><span class="nx">multipy</span><span class="p">(</span><span class="nx">math</span><span class="p">.</span><span class="nx">pow</span><span class="p">(</span><span class="nx">d</span><span class="p">,</span> <span class="mi">2</span><span class="p">),</span> <span class="nx">e</span><span class="p">));</span>
                                    <span class="c1">// 1.2790064742399996 MJ</span>
</code></pre> 
    </div> 
    <p>Operations with arrays are supported too:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="c1">// Force on a charged particle moving through a magnetic field</span>
<span class="kd">var</span> <span class="nx">B</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'[1, 0, 0] T'</span><span class="p">);</span>            <span class="c1">// [1 T, 0 T, 0 T]</span>
<span class="kd">var</span> <span class="nx">v</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'[0, 1, 0] m/s'</span><span class="p">);</span>          <span class="c1">// [0 m / s, 1 m / s, 0 m / s]</span>
<span class="kd">var</span> <span class="nx">q</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'1 C'</span><span class="p">);</span>                    <span class="c1">// 1 C</span>

<span class="kd">var</span> <span class="nx">F</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">multiply</span><span class="p">(</span><span class="nx">q</span><span class="p">,</span> <span class="nx">math</span><span class="p">.</span><span class="nx">cross</span><span class="p">(</span><span class="nx">v</span><span class="p">,</span> <span class="nx">B</span><span class="p">));</span>  <span class="c1">// [0 N, 0 N, -1 N]</span>
</code></pre> 
    </div> 
    <p>All arithmetic operators act on the value of the unit as it is represented in SI units. This may lead to surprising behavior when working with temperature scales like <code class="highlighter-rouge">celsius</code> (or <code class="highlighter-rouge">degC</code>) and <code class="highlighter-rouge">fahrenheit</code> (or <code class="highlighter-rouge">degF</code>). In general you should avoid calculations using <code class="highlighter-rouge">celsius</code> and <code class="highlighter-rouge">fahrenheit</code>. Rather, use <code class="highlighter-rouge">kelvin</code> (or <code class="highlighter-rouge">K</code>) and <code class="highlighter-rouge">rankine</code> (or <code class="highlighter-rouge">R</code>) instead. This example highlights some problems when using <code class="highlighter-rouge">celsius</code> and <code class="highlighter-rouge">fahrenheit</code> in calculations:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="kd">var</span> <span class="nx">T_14F</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="s1">'14 degF'</span><span class="p">);</span>          <span class="c1">// Unit 14 degF (263.15 K)</span>
<span class="kd">var</span> <span class="nx">T_28F</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">multiply</span><span class="p">(</span><span class="nx">T1</span><span class="p">,</span> <span class="mi">2</span><span class="p">);</span>          <span class="c1">// Unit 487.67 degF (526.3 K), not 28 degF</span>

<span class="kd">var</span> <span class="nx">Tnegative</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="o">-</span><span class="mi">13</span><span class="p">,</span> <span class="s1">'degF'</span><span class="p">);</span>    <span class="c1">// Unit -13 degF (248.15 K)</span>
<span class="kd">var</span> <span class="nx">Tpositive</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">abs</span><span class="p">(</span><span class="nx">T1</span><span class="p">);</span>              <span class="c1">// Unit -13 degF (248.15 K), not 13 degF</span>

<span class="kd">var</span> <span class="nx">Trate1</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'5 (degC/hour)'</span><span class="p">);</span>   <span class="c1">// Unit 5 degC/hour</span>
<span class="kd">var</span> <span class="nx">Trate2</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'(5 degC)/hour'</span><span class="p">);</span>   <span class="c1">// Unit 278.15 degC/hour</span>
</code></pre> 
    </div> 
    <p>The expression parser supports units too. This is described in the section about units on the page <a href="../expressions/syntax.html#units">Syntax</a>.</p> 
    <h2 id="userdefined-units">User-Defined Units <a href="#userdefined-units" title="Permalink">#</a></h2> 
    <p>You can add your own units to Math.js using the <code class="highlighter-rouge">math.createUnit</code> function. The following example defines a new unit <code class="highlighter-rouge">furlong</code>, then uses the user-defined unit in a calculation:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="nx">math</span><span class="p">.</span><span class="nx">createUnit</span><span class="p">(</span><span class="s1">'furlong'</span><span class="p">,</span> <span class="s1">'220 yards'</span><span class="p">);</span>
<span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'1 mile to furlong'</span><span class="p">);</span>            <span class="c1">// 8 furlong</span>
</code></pre> 
    </div> 
    <p>If you cannot express the new unit in terms of any existing unit, then the second argument can be omitted. In this case, a new base unit is created:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="c1">// A 'foo' cannot be expressed in terms of any other unit.</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">createUnit</span><span class="p">(</span><span class="s1">'foo'</span><span class="p">);</span>
<span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'8 foo * 4 feet'</span><span class="p">);</span>               <span class="c1">// 32 foo feet</span>
</code></pre> 
    </div> 
    <p>The second argument to <code class="highlighter-rouge">createUnit</code> can also be a configuration object consisting of the following properties:</p> 
    <ul> 
     <li><strong>definition</strong> A <code class="highlighter-rouge">string</code> or <code class="highlighter-rouge">Unit</code> which defines the user-defined unit in terms of existing built-in or user-defined units. If omitted, a new base unit is created.</li> 
     <li><strong>prefixes</strong> A <code class="highlighter-rouge">string</code> indicating which prefixes math.js should use with the new unit. Possible values are <code class="highlighter-rouge">'none'</code>, <code class="highlighter-rouge">'short'</code>, <code class="highlighter-rouge">'long'</code>, <code class="highlighter-rouge">'binary_short'</code>, or <code class="highlighter-rouge">'binary_long'</code>. Default is <code class="highlighter-rouge">'none'</code>.</li> 
     <li><strong>offset</strong> A value applied when converting to the unit. This is very helpful for temperature scales that do not share a zero with the absolute temperature scale. For example, if we were defining fahrenheit for the first time, we would use: <code class="highlighter-rouge">math.createUnit('fahrenheit', {definition: '0.555556 kelvin', offset: 459.67})</code></li> 
     <li><strong>aliases</strong> An array of strings to alias the new unit. Example: <code class="highlighter-rouge">math.createUnit('knot', {definition: '0.514444 m/s', aliases: ['knots', 'kt', 'kts']})</code></li> 
    </ul> 
    <p>An optional <code class="highlighter-rouge">options</code> object can also be supplied as the last argument to <code class="highlighter-rouge">createUnits</code>. Currently only the <code class="highlighter-rouge">override</code> option is supported:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="c1">// Redefine the mile (would not be the first time in history)</span>
<span class="nx">math</span><span class="p">.</span><span class="nx">createUnit</span><span class="p">(</span><span class="s1">'mile'</span><span class="p">,</span> <span class="s1">'1609.347218694'</span><span class="p">,</span> <span class="p">{</span><span class="na">override</span><span class="p">:</span> <span class="kc">true</span><span class="p">}});</span>
</code></pre> 
    </div> 
    <p>Base units created without specifying a definition cannot be overridden.</p> 
    <p>Multiple units can defined using a single call to <code class="highlighter-rouge">createUnit</code> by passing an object map as the first argument, where each key in the object is the name of a new unit and the value is either a string defining the unit, or an object with the configuration properties listed above. If the value is an empty string or an object lacking a definition property, a new base unit is created.</p> 
    <p>For example:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="nx">math</span><span class="p">.</span><span class="nx">createUnit</span><span class="p">(</span> <span class="p">{</span>
  <span class="na">foo</span><span class="p">:</span> <span class="p">{</span>
    <span class="na">prefixes</span><span class="p">:</span> <span class="s1">'long'</span>
  <span class="p">},</span>
  <span class="na">bar</span><span class="p">:</span> <span class="s1">'40 foo'</span><span class="p">,</span>
  <span class="na">baz</span><span class="p">:</span> <span class="p">{</span>
    <span class="na">definition</span><span class="p">:</span> <span class="s1">'1 bar/hour'</span><span class="p">,</span>
    <span class="na">prefixes</span><span class="p">:</span> <span class="s1">'long'</span>
  <span class="p">}</span>
<span class="p">},</span>
<span class="p">{</span>
  <span class="na">override</span><span class="p">:</span> <span class="kc">true</span>
<span class="p">});</span>
<span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'50000 kilofoo/s'</span><span class="p">);</span>   <span class="c1">// 4.5 gigabaz</span>
</code></pre> 
    </div> 
    <h3 id="return-value">Return Value <a href="#return-value" title="Permalink">#</a></h3> 
    <p><code class="highlighter-rouge">createUnit</code> returns the created unit, or, when multiple units are created, the last unit created. Since <code class="highlighter-rouge">createUnit</code> is also compatible with the expression parser, this allows you to do things like this:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="nx">math</span><span class="p">.</span><span class="nb">eval</span><span class="p">(</span><span class="s1">'45 mile/hour to createUnit("knot", "0.514444m/s")'</span><span class="p">)</span>
<span class="c1">// 39.103964668651976 knot</span>
</code></pre> 
    </div> 
    <h2 id="api">API <a href="#api" title="Permalink">#</a></h2> 
    <p>A <code class="highlighter-rouge">Unit</code> object contains the following functions:</p> 
    <h3 id="unitclone">unit.clone() <a href="#unitclone" title="Permalink">#</a></h3> 
    <p>Clone the unit, returns a new unit with the same parameters.</p> 
    <h3 id="unitequalbaseunit">unit.equalBase(unit) <a href="#unitequalbaseunit" title="Permalink">#</a></h3> 
    <p>Test whether a unit has the same base as an other unit: length, mass, etc.</p> 
    <h3 id="unitequalsunit">unit.equals(unit) <a href="#unitequalsunit" title="Permalink">#</a></h3> 
    <p>Test whether a unit equals an other unit. Units are equal when they have the same base and same value when normalized to SI units.</p> 
    <h3 id="unitformatoptions">unit.format([options]) <a href="#unitformatoptions" title="Permalink">#</a></h3> 
    <p>Get a string representation of the unit. The function will determine the best fitting prefix for the unit. See the <a href="../functions/format.html">Format</a> page for available options.</p> 
    <h3 id="unitfromjsonjson">unit.fromJSON(json) <a href="#unitfromjsonjson" title="Permalink">#</a></h3> 
    <p>Revive a unit from a JSON object. Accepts An object <code class="highlighter-rouge"><span class="p">{</span><span class="err">mathjs:</span><span class="w"> </span><span class="err">'Unit',</span><span class="w"> </span><span class="err">value:</span><span class="w"> </span><span class="err">number,</span><span class="w"> </span><span class="err">unit:</span><span class="w"> </span><span class="err">string,</span><span class="w"> </span><span class="err">fixPrefix:</span><span class="w"> </span><span class="err">boolean</span><span class="p">}</span></code>, where the property <code class="highlighter-rouge">mathjs</code> and <code class="highlighter-rouge">fixPrefix</code> are optional. Used when deserializing a unit, see <a href="../core/serialization.html">Serialization</a>.</p> 
    <h3 id="unitsplitunitparts">unit.splitUnit(parts) <a href="#unitsplitunitparts" title="Permalink">#</a></h3> 
    <p>Split a unit into the specified parts. For example:</p> 
    <div class="language-js highlighter-rouge">
     <pre class="highlight"><code><span class="kd">var</span> <span class="nx">u</span> <span class="o">=</span> <span class="nx">math</span><span class="p">.</span><span class="nx">unit</span><span class="p">(</span><span class="mi">1</span><span class="p">,</span> <span class="s1">'m'</span><span class="p">);</span>
<span class="nx">u</span><span class="p">.</span><span class="nx">splitUnit</span><span class="p">([</span><span class="s1">'ft'</span><span class="p">,</span> <span class="s1">'in'</span><span class="p">]);</span>    <span class="c1">// 3 feet,3.3700787401574765 inch</span>
</code></pre> 
    </div> 
    <h3 id="unittounitname">unit.to(unitName) <a href="#unittounitname" title="Permalink">#</a></h3> 
    <p>Convert the unit to a specific unit name. Returns a clone of the unit with a fixed prefix and unit.</p> 
    <h3 id="unittojson">unit.toJSON() <a href="#unittojson" title="Permalink">#</a></h3> 
    <p>Returns a JSON representation of the unit, with signature <code class="highlighter-rouge"><span class="p">{</span><span class="err">mathjs:</span><span class="w"> </span><span class="err">'Unit',</span><span class="w"> </span><span class="err">value:</span><span class="w"> </span><span class="err">number,</span><span class="w"> </span><span class="err">unit:</span><span class="w"> </span><span class="err">string,</span><span class="w"> </span><span class="err">fixPrefix:</span><span class="w"> </span><span class="err">boolean</span><span class="p">}</span></code>. Used when serializing a unit, see <a href="../core/serialization.html">Serialization</a>.</p> 
    <h3 id="unittonumberunitname">unit.toNumber(unitName) <a href="#unittonumberunitname" title="Permalink">#</a></h3> 
    <p>Get the value of a unit when converted to the specified unit (a unit with optional prefix but without value). The type of the returned value is always <code class="highlighter-rouge">number</code>.</p> 
    <h3 id="unittonumericunitname">unit.toNumeric(unitName) <a href="#unittonumericunitname" title="Permalink">#</a></h3> 
    <p>Get the value of a unit when converted to the specified unit (a unit with optional prefix but without value). The type of the returned value depends on how the unit was created and can be <code class="highlighter-rouge">number</code>, <code class="highlighter-rouge">Fraction</code>, or <code class="highlighter-rouge">BigNumber</code>.</p> 
    <h3 id="unittosi">unit.toSI() <a href="#unittosi" title="Permalink">#</a></h3> 
    <p>Returns a clone of a unit represented in SI units. Works with units with or without a value.</p> 
    <h3 id="unittostring">unit.toString() <a href="#unittostring" title="Permalink">#</a></h3> 
    <p>Get a string representation of the unit. The function will determine the best fitting prefix for the unit.</p>
