             <h3>Package Description</h3> 
             <p class="package-description-body">Tools for Java developers creating Java EE and Web applications, including a Java IDE, tools for Java EE, JPA, JSF, Mylyn, EGit and others.</p> 
             <div class="package-includes"> 
              <h4>This package includes:</h4>

                <li><span class="title" title="Data Tools Platform Intro Extension">Data Tools Platform</span></li>

                <li><span class="title" title="Eclipse Java development tools (binary runtime and user documentation).">Eclipse Java Development Tools</span></li>
                <li><span class="title" title="Eclipse Java EE Developer Tools">Eclipse Java EE Developer Tools</span></li>

                <li><span class="title" title="Provides the Task List for task and time management.">Mylyn Task List</span></li>
                <li><span class="title" title="Eclipse plug-in development environment.">Eclipse Plug-in Development Environment</span></li>
                <li><span class="title" title="An integrated framework and toolkit for seamless working on remote systems through SSH, FTP or dstore protocols.">Remote System Explorer</span></li>
                <li><span class="title" title="Eclipse Code Recommenders for Java Developers makes you more productive when working with unfamiliar APIs. Install this feature if you want to use its intelligent code completion and data\u2010mined API documentation that tell you exactly how others have solved your problem before.">Code Recommenders Tools for Java Developers</span></li>
