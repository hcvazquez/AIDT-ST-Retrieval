align="left"><b>Using the <span style="font-weight: 400"> <font face="Courier New" color="#0000ff">Math</font></span></b> class</h2> 
          <hr color="#cbd1dc" noshade size="1" width="50%" align="left"> <h2><strong>Using pre-written Java code</strong> </h2> 
          <ul> 
           <li> <p style="MARGIN-BOTTOM: 18px">The Java compiler makes it easy for programmers to use "packages" of existing code. Most packages contain dozens of related classes with each having a number of useful methods. Among the most frequently used packages are: </p></li>
          </ul> 
          <blockquote> 
           <blockquote> 
            <table cellspacing="0" cellpadding="0" width="75%" border="1"> 
             <tbody> 
              <tr> 
               <td width="25%" bgcolor="#c0c0c0"> <p align="center"><strong>Package</strong></p></td> 
               <td width="75%" bgcolor="#c0c0c0"> <p align="center"><b>Contents</b> </p></td>
              </tr> 
              <tr> 
               <td width="25%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">java.applet</font></p></td> 
               <td valign="center" width="75%"> <p style="MARGIN-LEFT: 5px">Supports Java applets (small programs that may be embedded inside other applications - such as Web pages)</p></td>
              </tr> 
              <tr> 
               <td width="25%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">java.awt</font></p></td> 
               <td width="75%"> <p style="MARGIN-LEFT: 5px">The "abstract windows toolkit" supports graphical, Windows programming</p></td>
              </tr> 
              <tr> 
               <td width="25%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">java.lang</font></p></td> 
               <td width="75%"> <p style="MARGIN-LEFT: 5px">Supports Java language extensions and mathematical tools</p></td>
              </tr> 
              <tr> 
               <td width="25%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">java.text</font></p></td> 
               <td width="75%"> <p style="MARGIN-LEFT: 5px">Supports text formatting and manipulation</p></td>
              </tr> 
              <tr> 
               <td width="25%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">java.util</font></p></td> 
               <td width="75%"> <p style="MARGIN-LEFT: 5px">A set of general purpose utility classes</p></td>
              </tr>
             </tbody>
            </table>
           </blockquote>Beginning Java programmers will be most interested in the last three of these packages. 
          </blockquote> 
          <ul> 
           <li>The <font face="Courier New" color="#0000ff" size="2">import</font> reserved word is used in Java to make external classes of other packages available to the current source file through use of an abbreviated name. For example, the following statement makes it possible to reference all classes in the <font face="Courier New" color="#0000ff" size="2">java.util</font> package: </li>
          </ul> 
          <blockquote> 
           <blockquote> 
            <p align="left"><font face="Courier New" color="#0000ff" size="2">import java.util.*;</font></p>
           </blockquote> 
           <p>Notes:</p> 
           <ol> 
            <li> <p style="MARGIN-BOTTOM: 18px">Code <font face="Courier New" color="#0000ff" size="2">import</font> statements at the beginning of the Java source file prior to the class header. </p> </li>
            <li> <p style="MARGIN-BOTTOM: 18px">The only package that doesn't require an <font face="Courier New" color="#0000ff" size="2">import</font> statement is <font face="Courier New" color="#0000ff" size="2">java.lang</font>. Its contents are so frequently used that it is automatically available. </p></li>
           </ol> 
           <ol start="3"> 
            <li>Optionally, only a selected class within a particular package might be specified, such as </li>
           </ol> 
           <blockquote> 
            <blockquote> 
             <p align="left"><font face="Courier New" color="#0000ff" size="2">import java.util.TimeZone;</font></p>
            </blockquote> 
            <p align="left">which has features to work with a time zone and determine daylight savings. </p>
           </blockquote> 
           <ol start="4"> 
            <li>Without an <font face="Courier New" color="#0000ff" size="2">import</font> statement pre-written code can still be accessed, but every time you reference a class or a class feature (such as a method) you would need to use its package name as a qualifier. For example, </li>
           </ol> 
           <blockquote> 
            <blockquote> 
             <p align="left"><font face="Courier New" color="#0000ff" size="2">java.util.TimeZone.getDefault()</font></p>
            </blockquote> 
            <p style="MARGIN-BOTTOM: 18px; MARGIN-LEFT: 0px">instead of </p>
            <blockquote> 
             <p align="left"><font face="Courier New" color="#0000ff" size="2">TimeZone.getDefault()</font></p>
            </blockquote> 
            <p style="MARGIN-BOTTOM: 18px; MARGIN-LEFT: 0px">would be required to retrieve the default time zone of the platform. </p>
           </blockquote>
          </blockquote> 
          <ul> 
           <li> <p style="MARGIN-BOTTOM: 18px; MARGIN-LEFT: 0px">Use the help facility to learn about Java packages, their classes, and the features of their classes. In this lesson, we will explore the use of the <font face="Courier New" color="#0000ff" size="2">Math</font> class in the <font face="Courier New" color="#0000ff" size="2">java.lang</font> package. </p></li>

          <ul> 
           <li> <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Is part of the <font face="Courier New" color="#0000ff" size="2">java.lang</font> package </p></li>
          </ul> 
          <ul> 
           <li> <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Contains class methods for performing basic numeric operations such as random number generation, rounding, absolute value, exponentiation, square root, logarithmic functions, and trigonometric functions </p> </li>
           <li> <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Contains two frequently used public field. These are: </p></li>
          </ul> 
          <blockquote> 
           <blockquote> 
            <div align="left"> 
             <table cellspacing="0" cellpadding="0" width="70%" border="1">

               </tr> 
               <tr> 
                <td width="31%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">E</font></p></td> 
                <td width="69%"> <p style="MARGIN-LEFT: 4px" align="left">The <font face="Courier New" color="#0000ff" size="2">double</font> value that is closer than any other to <font face="Courier New" size="2">e</font>, the base of the natural logarithms</p></td>
               </tr> 
               <tr> 
                <td width="31%"> <p align="center"><font face="Courier New" color="#0000ff" size="2">PI</font></p></td> 
                <td width="69%"> <p style="MARGIN-LEFT: 4px" align="left">The <font face="Courier New" color="#0000ff" size="2">double</font> value that is closer than any other to <font face="Courier New" size="2">pi</font>, the ratio of the circumference of a circle to its diameter</p></td>
               </tr>
              </tbody>
             </table>
            </div>
           </blockquote> 
           <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">To display the value of <font face="Courier New" size="2">e</font> and the value of <font face="Courier New" size="2">pi</font>, you may run the following program: </p>
           <blockquote> 
            <p class="MsoNormal" style="MARGIN-BOTTOM: 18px"><font face="Courier New" color="#0000ff" size="2">public class App {<br \> public static void main(String[] args) {<br \> System.out.println("e = " + Math.E);<br \> System.out.println("pi = " + Math.PI);<br \><br \> }<br \>}</font> </p>
            <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Note: Because <font face="Courier New" color="#0000ff" size="2">E</font> and <font face="Courier New" color="#0000ff" size="2">PI</font> are fields of the <font face="Courier New" color="#0000ff" size="2">Math</font> class and not methods of the <font face="Courier New" color="#0000ff" size="2">Math</font> class, no parenthesis are coded after their name. Parenthesis are only used to indicate a method call. </p>
           </blockquote>
          </blockquote> 
          <ul> 
           <li> <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Has a number of useful class methods. Among them are the following (with detailed parameter requirements and the returned data type omitted for brevity): </p></li>
          </ul> 
          <blockquote> 
           <blockquote> 
            <div align="left">

          </blockquote> <h3><b>Sample programs</b></h3> <p>The following small samples will help you understand how easy it is to use the class methods of the <font face="Courier New" color="#0000ff" size="2">Math</font> class:</p> 
          <blockquote> 
           <h4>Example 1: Performing some miscellaneous mathematical calculations on a number</h4> 
           <blockquote> 
            <p><font face="Courier New" color="#0000ff" size="2">public class App {<br \> public static void main(String[] args) {<br \><br \> double x;<br \> char again;<br \><br \> do {<br \> Utility.separator(50, '~');<br \> System.out.print("Enter a number: ");<br \> x = Keyboard.readDouble();<br \> Utility.skip();<br \> System.out.println(" Ceiling = " + Math.ceil(x));<br \> System.out.println(" Floor = " + Math.floor(x));<br \> System.out.println(" Rounded = " + Math.round(x));<br \> System.out.println(" Squared = " + Math.pow(x, 2));<br \> System.out.println(" Square root = " + Math.sqrt(x));<br \> Utility.skip();<br \> System.out.print("Again? (Y/N): ");<br \> again = Keyboard.readChar();<br \> } while (again == 'Y' || again == 'y');<br \> }<br \>}</font></p> 
            <p class="MsoNormal" style="MARGIN-BOTTOM: 18px">Note: Try several different values when testing this program to see what the various methods do (including a negative number). </p>
           </blockquote> 
           <h4>Example 2: Simulating 100 rolls of a single die to generate an integer value from 1 to 6</h4> 
           <blockquote> 
            <p><font face="Courier New" color="#0000ff" size="2">public class App {<br \> public static void main(String[] args) {<br \><br \> int value;<br \><br \> for (int i = 1; i &lt;= 100; i++) {<br \> value = (int) (((Math.random() * 1000) % 6) + 1);<br \> System.out.println("Roll " + i + ": " + value);<br \> }<br \> }<br \>}</font></p> 
            <p>Note: The <font face="Courier New" color="#0000ff" size="2">random()</font> method returns a <font face="Courier New" color="#0000ff" size="2">double</font> that is less than 1.0 and greater than or equal to zero. Multiplying by 1000 provides extra digits on the left of the decimal point so that modulo 6 can produce a fairly random value from 0 to 5. By adding 1 and casting to an <font face="Courier New" color="#0000ff" size="2">int</font>, the die's value can be stored in an <font face="Courier New" color="#0000ff" size="2">int</font> variable. </p>
           </blockquote> 
           <h4>Example 3: Calculating the area of a circle</h4> 
           <blockquote> 
            <p><font face="Courier New" color="#0000ff" size="2">public class App {<br \> public static void main(String[] args) {<br \><br \> double radius;<br \> double area;<br \> char again;<br \><br \> do {<br \> Utility.separator(50, '~');<br \> System.out.print("Enter the radius: ");<br \> radius = Keyboard.readDouble();<br \> Utility.skip();<br \> if (radius &lt;= 0) {<br \> System.out.println(" Invalid radius");<br \> }<br \> else {<br \> area = Math.PI * Math.pow(radius, 2);<br \> System.out.println(" Area = " + area);<br \> }<br \> Utility.skip();<br \> System.out.print("Again? (Y/N): ");<br \> again = Keyboard.readChar();<br \> } while (again == 'Y' || again == 'y');<br \> }<br \>}</font></p> 
            <p>Note: The <font face="Courier New" color="#0000ff" size="2">Math.pow()</font> method is used to square the radius of the circle. </p>
           </blockquote>
