      <p style="font-style: italic">A captcha library that generates audio and image CAPTCHAs.</p> 
      <p> <strong>Latest Version: </strong><a title="Latest Version URL" href="https://pypi.python.org/pypi/captcha/0.2.4"> <span>0.2.4</span> </a> </p> 
      <p>A captcha library that generates audio and image CAPTCHAs.</p> 
      <a href="https://travis-ci.org/lepture/captcha" rel="nofollow"><img src="https://travis-ci.org/lepture/captcha.png?branch=master"></a> 
      <a href="https://ci.appveyor.com/project/lepture/captcha" rel="nofollow"><img src="https://ci.appveyor.com/api/projects/status/amm21f13lx4wuura"></a>

      <div id="features"> 
       <h2>Features</h2> 
       <ol> 
        <li>Audio CAPTCHAs <a href="https://github.com/lepture/captcha/releases/download/v0.1-beta/out.wav" rel="nofollow">DEMO</a></li> 
        <li>Image CAPTCHAs</li> 
       </ol>

      <div id="installation"> 
       <h2>Installation</h2> 
       <p>Install captcha with pip:</p> 
       <pre>$ pip install captcha
</pre> 
      </div> 
      <div id="usage"> 
       <h2>Usage</h2> 
       <p>Audio and Image CAPTCHAs are in seprated modules:</p> 
       <pre>from io import BytesIO
from captcha.audio import AudioCaptcha
from captcha.image import ImageCaptcha

audio = AudioCaptcha(voicedir='/path/to/voices')
image = ImageCaptcha(fonts=['/path/A.ttf', '/path/B.ttf'])

data = audio.generate('1234')
assert isinstance(data, bytearray)
audio.write('1234', 'out.wav')

data = image.generate('1234')
assert isinstance(data, BytesIO)
image.write('1234', 'out.png')
</pre> 
       <p>This is the APIs for your daily works. We do have built-in voice data and font data. But it is suggested that you use your own voice and font data.</p> 
      </div> 
      <div id="contribution"> 
       <h2>Contribution</h2> 
       <p>We need voice wav files. The voice wav file should be in 8-bit, please keep it as small as possible. Name your voice file as:</p> 
       <pre>{{language}}-{{character}}-{{username}}.wav
# exmaple: zh-1-lepture.wav
</pre> 
       <p>TODO: we need a place to upload voice files.</p>
