    <div> 
     <h2 class="text-uppercase">Features</h2> 
     <div class="alert alert-info" style="font-weight:bold;" data-pgc-define="pgweb.notupdated" data-pgc-define-name="Not updated for 3"> 
      <p>Please note that this website is not yet updated for Pinegrow 3.</p> 
      <p>Check out the <a href="pinegrow3.html">Quick Guide to Pinegrow 3</a> to see all that's new and improved.</p> 
     </div> 
     <p class="lead">Are you working with HTML pages, WordPress themes, Bootstrap or Foundation?<br>See why you should have Pinegrow in your toolbelt:</p> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Open HTML files on your desktop</h3> 
     <p>Pinegrow lets you work with HTML files on your computer. It doesn't add any frameworks, layouts or styles of its own to your code. There is nothing to upload, import or export. Simply open, edit and save your HTML files.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="5000" src="https://d33wubrfki0l68.cloudfront.net/4639f1ea1cd37b671d72efe0550ee421ffcb3b1b/ce591/images/open.html.good.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Speed up working with HTML</h3> 
     <p>Quickly build the layout of your webpage with Pinegrow's powerful visual tools for adding, editing, moving, cloning and deleting HTML elements.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="12000" src="https://d33wubrfki0l68.cloudfront.net/685f6da617acf82c3d61d4c0a3170074a910a1f2/aa35b/images/build.html.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Live editing</h3> 
     <p>Everything you do in Pinegrow is live. There are no previews. You can edit and test your page at the same time - even if you're using dynamic JavaScript elements.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" src="https://d33wubrfki0l68.cloudfront.net/27db3118f1901163161a7eadd1aa27509711aa2b/1d47a/images/live.edit.dither.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Bootstrap, Foundation, AngularJS or HTML</h3> 
     <p> Pinegrow has full support for Bootstrap and Foundation. Drag components to the page and customize them through UI properties. Pinegrow automatically sets attributes, adds classes or even adjusts the component's HTML code. AngularJS, 960 Grid and plain HTML are also supported.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="10000" src="https://d33wubrfki0l68.cloudfront.net/b3758f083474f4a115cd249b2e0c2d06ac240b43/91e17/images/bs.build.page.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>WordPress theme builder</h3> 
     <p>Open or create a HTML page in Pinegrow Web Editor. Add WordPress actions to HTML elements and set their parameters. Export the WordPress theme. Pinegrow generates PHP code and splits the page into PHP theme files. <a href="/wordpress-theme-builder.html" target="_blank">And much more!</a></p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="10000" src="https://d33wubrfki0l68.cloudfront.net/1c79e30642b7732ceb78b3008ceaaf101e6972c7/78e51/images/wp/first.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Multi-page editing</h3> 
     <p>Edit multiple pages at the same time makes responsive website design a breeze. Duplicate and mirror pages (changes in the source page are automatically reflected in a mirrored page), set different zoom levels and device sizes.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" src="https://d33wubrfki0l68.cloudfront.net/a373c11c6672e9c793ebf0967f65a008ea325e49/e19b8/images/multipage.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Pinegrow loves the code</h3> 
     <p> Pinegrow doesn't hide the code from you. Edit pages visually and through code - at the same time. Changes made through visual UI are immediately reflected in the code view. Code edits - even if made in external code editor - are immediately visible on the page and in visual UI. (BTW, this makes Pinegrow the perfect tool for learning about HTML &amp; CSS).</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" src="https://d33wubrfki0l68.cloudfront.net/4b9159207f0ed96198bb020c5d4be761db1deaf7/1265c/images/edit.code.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Powerful CSS editor</h3> 
     <p> Edit CSS rules visually or through code. Add variables and expressions to create easily customizable themes. Use Stylesheet manager to clone, attach and remove stylesheets. Changes made to a stylesheet will be immediately visible on all pages that use that stylesheet.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" src="https://d33wubrfki0l68.cloudfront.net/7909a7991e1588ce11572222e803408e23fb7705/eb3dd/images/edit.css.gif" alt="Pinegrow works with other frameworks"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Make your pages responsive</h3> 
     <p> Easily design responsive websites with Media query helper tool. Add custom breakpoints or let PG detect them by analyzing stylesheets. Use multi-page editing to edit the page at multiple device sizes.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="8000" src="https://d33wubrfki0l68.cloudfront.net/0f546629611c2875c2859c0a2a481d9b7651eeee/f28ad/images/responsive.gif" alt="Edit"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Quickly build websites with Blocks</h3> 
     <p> Use the collection of <a href="http://docs.pinegrow.com/bootstrap-blocks">ready-made website blocks</a> to quickly build great looking websites. Drag blocks to the page, customise the content, style CSS rules if you want - and your website is done. You can even create fully customisable WordPress themes with blocks.&nbsp;</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="8000" src="https://d33wubrfki0l68.cloudfront.net/bf66b60bd7f10d467fce212e2575a7b485c7b24b/bbe53/images/blocks.gif" alt="Edit"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Edit any page from the web</h3> 
     <p> Pinegrow also lets you open any page from the web. Just enter the URL and start editing the remote page: change layout, edit text and images, modify CSS rules... This is great for editing CSS of server-generated pages and for trying out templates. Then save modified HTML and CSS files to your computer.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="8000" src="https://d33wubrfki0l68.cloudfront.net/5999d5510150f178771989273f8ab9b7312a8886/20022/images/openurl.gif" alt="Editing multiple pages"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Use Pinegrow alongside your favorite code editor</h3> 
     <p>Pinegrow fits right into your workflow. Use it alongside your favorite code editor (with smart auto-refresh), source control system, package manager and deployment tools. Use plugin for Atom to <a href="http://pinegrow.com/pinegrow-atom.html">live sync edits with Pinegrow</a>.</p> 
     <div class="gif-wrap"> 
      <img class="img-responsive gif" data-len="12000" src="https://d33wubrfki0l68.cloudfront.net/e7eabb12e07329bb485203bb3575cd5db1b0c798/527ec/images/external.edit.gif" alt="Editing multiple pages"> 
     </div> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Edit PHP, ASP and ERB HTML templates</h3> 
     <p> Visually edit HTML layouts with dynamic server-side code tags. Double-click on any element that contains PHP, ASP or Ruby on Rails blocks to edit the code. <a href="http://docs.pinegrow.com/editing/editing-php-html-templates">Read more&nbsp;»</a></p> 
    </div> 
    <div class=" col-center feature-anim col-md-10 col-md-offset-1"> 
     <h3>Projects, Master pages, Components, Partials and Libraries</h3> 
     <p><a href="http://docs.pinegrow.com/misc/pinegrow-pro"> Pinegrow PRO</a> is an edition of Pinegrow Web Editor with powerful tools like Projects, Master pages, Partials and Smart components, perfect for building and managing multi-page websites.<br><br></p> 
    </div>

      <h3>Work with Bootstrap and Foundation</h3> 
      <p>Pinegrow knows how to rearrange HTML code of an element if you change its properties. For example, changing form layout from normal to horizontal requires just one click.</p> 
      <p><a href="#video" data-toggle="modal" data-target="#videoModal2">Watch the video »</a></p>

     </div> 
     <div class="col-sm-6"> 
      <h3>Use CSS variables and functions</h3> 
      <p>Use variables instead of hard coding CSS values. Use math expressions and functions like lighten and darken to create designs that can be customised by changing just a couple of variables - all in real time.&nbsp;</p> 
      <p><a href="#video" data-toggle="modal" data-target="#videoModal4">Watch the video »</a></p>

      <h3>Edit multiple pages at the same time</h3> 
      <p>Open multiple pages that share the same stylesheet. Style changes are reflected on all pages in real time. View them at different screen sizes. Mirror content from one page to another.</p> 
      <p><a href="#video" data-toggle="modal" data-target="#videoModal5">Watch the video »</a></p> 
     </div> 
     <div class="col-sm-6"> 
      <h3>Experiment with different designs</h3> 
      <p>Take a page and open a mirrored copy. Use Stylesheet manager to duplicate and attach / detach stylesheets. Play with different designs in real time. Bonus points if you use different screen sizes and CSS variables.</p> 
     </div>

      <h3>Edit content of a page</h3> 
      <p>Use Pinegrow as a CMS for your static Html pages. Move elements around and double click on any element to edit its text content.</p>

     <div class="col-sm-6"> 
      <h3>Try templates online</h3> 
      <p>Just open a template demo from the internet and start customizing it with your images and content. Immediately see if the template works for you or not.</p> 
      <p><a href="#video" data-toggle="modal" data-target="#videoModal7">Watch the video »</a></p>

    <h2 class="text-center">Frequently Asked Questions</h2> 
    <div class="row text-left"> 
     <div class="col-sm-6"> 
      <h3>How is Pinegrow different from Dreamweaver, Macaw, Jetstrap, Webflow?</h3> 
      <p>Pinegrow is a desktop app that works with regular HTML files. We believe that a standalone app fits into a web development workflow much better than cloud solutions.</p> 
      <p>Pinegrow has unique features like editing multiple pages at the same time, complete range of Bootstrap and Foundation components, CSS styling with LESS support, Stylesheet manager... And more useful features are on the way.</p> 
      <p>Pinegrow doesn't add any abstraction on top of HTML and CSS. It simply helps you to work with HTML and CSS more efficiently, either visually or through code.</p> 
     </div> 
     <div class="col-sm-6"> 
      <h3>What are the license terms?</h3> 
      <p>A separate license is needed for each individual user who is using the app. Exception is made for your family members - it makes no sense requiring your kids to buy a separate license if they want to play with Pinegrow on your computer.</p> 
      <p>You can install your copy of Pinegrow on up to 3 of your computers (at home and at work, Windows, Mac or Linux) as long as you are the one using the app.&nbsp;</p> 
      <p>If you are buying Pinegrow for a company, you need the company license. You need one company license for every person in your company who will be using Pinegrow.</p> 
     </div> 
     <div class="clear-columns clearfix hidden-xs"></div> 
     <div class="col-sm-6"> 
      <h3>How does the 1 year of free updates work?</h3> 
      <p>Each product purchase gives you a license to use the current version of&nbsp;the program forever and includes one full year of free updates to&nbsp;your edition of Pinegrow.</p> 
      <p>After the one year of free updates expires you may wish to renew your&nbsp;license - for just half the price of a new license - so as to obtain an&nbsp;additional year of free updates.</p> 
      <p>Your existing version will continue to work, but we're confident that&nbsp;you'll want to take full advantage of our latest enhancements at a&nbsp;substantial discount.</p> 
     </div> 
     <div class="col-sm-6"> 
      <h3>Why the name "Pinegrow"?</h3> 
      <p>We love pine trees. They smell amazing and it’s fun to chill in their shadow in the summer. Pine needles let through the perfect amount of sunlight. And as trees normally do, they grow. So, Pinegrow!</p>
