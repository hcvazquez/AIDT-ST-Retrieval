    <h2 id="javanetwork_overview"><a class="link" href="#javanetwork_overview">1. Java and HTTP access</a></h2> 
    <div class="sectionbody"> 
     <div class="paragraph"> 
      <p> Java provides a HTTP client API to access resources via the HTTP or HTTPS protocol. The main classes to access the Internet are the <code>java.net.URL</code> class and the <code>java.net.HttpURLConnection</code> class.</p> 
     </div> 
     <div class="paragraph"> 
      <p>The <code>URL</code> class can be used to define a pointer to a web resource while the <code>HttpURLConnection</code> class can be used to access a web resource.</p> 
     </div> 
     <div class="paragraph"> 
      <p><code>HttpURLConnection</code> allows you to create an <code>InputStream</code>.</p> 
     </div> 
     <div class="paragraph"> 
      <p>Once you have accessed an <code>InputStream</code> you can read it similarly to an <code>InputStream</code> from a local file.</p> 
     </div> 
     <div class="paragraph"> 
      <p><code>HttpURLConnection</code> supports the transparent response compression (via the header <code>Accept-Encoding: gzip</code>, Server Name Indication (extension of SSL and TLS) and a response cache.</p> 
     </div> 
     <div class="paragraph"> 
      <p>The API is relatively straight forward. For example to retrieve the webpage www.vogella.com you can use the following example.</p> 
     </div> 
     <div class="listingblock"> 
      <div class="content"> 
       <pre class="CodeRay highlight"><code data-lang="text">package com.vogella.java.introduction;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStream;
import java.io.InputStreamReader;
import java.net.HttpURLConnection;
import java.net.URL;

public class DownloadWebpageExample {
    private static final String newLine  = System.getProperty("line.separator");
    public static void main(String[] args) {
        try  {
            URL url = new URL("http://www.vogella.com");
            HttpURLConnection con = (HttpURLConnection) url.openConnection();
            String readStream = readStream(con.getInputStream());
            // Give output for the command line
            System.out.println(readStream);
        } catch (Exception e) {
            e.printStackTrace();
        }

    }

    private static String readStream(InputStream in) {
        StringBuilder sb = new StringBuilder();
        try (BufferedReader reader = new BufferedReader(new InputStreamReader(in));) {
            String nextLine = "";
            while ((nextLine = reader.readLine()) != null) {
                sb.append(nextLine + newLine);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
        return sb.toString();
    }
