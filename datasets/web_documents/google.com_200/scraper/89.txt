              <article id="post-1321" class="post-1321 post type-post status-publish format-standard hentry category-blog tag-javascript tag-jquery tag-nodejs tag-scraping"> 
               <header class="entry-header"> 
                <h1 class="entry-title"><a href="https://www.webniraj.com/2015/03/16/nodejs-scraping-websites-using-request-and-cheerio/" title="NodeJS: Scraping Websites Using Request and Cheerio">NodeJS: Scraping Websites Using Request and Cheerio</a></h1> 
                <div class="comment-bubble"> 
                 <a href="https://www.webniraj.com/2015/03/16/nodejs-scraping-websites-using-request-and-cheerio/#comments" class="comments-link">2</a> 
                </div> 
               </header>
               <!-- .entry-header --> 
               <div class="entry-content"> 
                <p>I’ve recently been using NodeJS&nbsp;build website scrapers quickly, and usually in less than 100 lines of code. This tutorial shows how you can easily create your own scraper using two NodeJs modules: <code>request</code> and <code>cheerio</code>. The same code can easily be adapted to perform complex tasks like completing and submitting a form.</p> 
                <p><span id="more-1321"></span></p> 
                <h2>Node Modules</h2> 
                <p>Assuming you already have NodeJS&nbsp;installed, we require two additional modules to be installed:</p> 
                <ul> 
                 <li>request – The simplest way possible to make <code>http</code> and <code>https</code> calls</li> 
                 <li>cheerio – A fast, flexible, and lean implementation of core jQuery designed specifically for the server</li> 
                </ul> 
                <p>You can install them by using the <code>npm</code> (Node Package Manager) command-line tool:</p> 
                <div class="oembed-gist">
                 <script src="https://gist.github.com/niraj-shah/747f999d63de70281c9b.js"></script>
                 <noscript>
                  View the code on 
                  <a href="https://gist.github.com/niraj-shah/747f999d63de70281c9b">Gist</a>.
                 </noscript>
                </div> 
                <h2>The Scraper</h2> 
                <p>Once installed, we’re ready to create our scraper. In my tutorial, we use the request module to fetch a webpage, and use cheerio to select the elements we need. Since cheerio is an implementation of jQuery, we can use the same selectors to select and extract information from the page we just scraped.</p> 
                <div class="oembed-gist">
                 <script src="https://gist.github.com/niraj-shah/39454eb66296ac737d6e.js"></script>
                 <noscript>
                  View the code on 
                  <a href="https://gist.github.com/niraj-shah/39454eb66296ac737d6e">Gist</a>.
                 </noscript>
                </div> 
                <p>On lines 6-10, we set the default settings we want to use. Since most pages will drop cookies, it’s a good idea to keep <code>jar</code> set to <code>true</code>. This will mean that any cookies the page sets will be passed to subsequent requests.</p> 
                <p>On line 14, we set the URL we want to scrape. Lines 15-17 show an example of custom headers we can pass. Here, you can set&nbsp;anything from authentication headers to content-type, cookies and user-agent.</p> 
                <p>On lines 19-27, we load the content of the page into cheerio and then select the elements we’re interested in. If it works correctly, the script will return your IP address, host-name and user-agent. Here’s an example:</p> 
                <p><a href="https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example.png" rel="themeblvd_lightbox" title="getIP.js Example"><img class="aligncenter wp-image-1322 size-full" title="getIP.js Example" src="https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example.png" alt="NodeJS-Scraping-Example" width="1252" height="548" srcset="https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example.png 1252w, https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example-275x120.png 275w, https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example-500x219.png 500w, https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example-940x411.png 940w, https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example-620x271.png 620w, https://www.webniraj.com/wp-content/uploads/2015/03/NodeJS-Scraping-Example-195x85.png 195w" sizes="(max-width: 1252px) 100vw, 1252px"></a></p> 
                <h2>POST Requests</h2> 
                <p>The above is an example of a <code>GET</code> request, but <code>POST</code> requests are possible too. You can make POST requests by calling <code>req.post</code> and pass in form data using the <code>form</code> variable in the POST function. Example:</p> 
                <div class="oembed-gist">
                 <script src="https://gist.github.com/niraj-shah/e3ae239194d982e938f2.js"></script>
                 <noscript>
                  View the code on 
                  <a href="https://gist.github.com/niraj-shah/e3ae239194d982e938f2">Gist</a>.
                 </noscript>
                </div>
