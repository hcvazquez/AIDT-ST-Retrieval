         <h2 id="convert_slide_notes_to_speech_or_closed_captioned_text"> Convert slide notes to speech </h2> 
        </div> 
        <!-- for Cascading Toc --> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_2"></a> 
        <div class="text">
         <p>You can convert text in the slide notes into an audio file using text-to-speech software from NeoSpeech™. The following voices from NeoSpeech are supported in Adobe Captivate:</p> 
         <ul> 
          <li>Bridget</li> 
          <li>Chloe</li> 
          <li>James</li> 
          <li>Julie</li> 
          <li>Kate</li> 
          <li>Paul</li> 
          <li>Yumi</li> 
         </ul> 
         <p>You can install NeoSpeech using the Adobe Captivate installation CD or download it from the Adobe website. For detailed instructions on how to install NeoSpeech, see <a href="slide-notes.html#install_text_to_speech_converters">Install text to speech converters</a>.</p> 
        </div> 
       </div> 
       <div class="note parbase section"> 
        <div class="help-note"> 
         <p> <span class="help-note-title">Note:</span> </p>
         <p>You can also use third-party voices or the voices installed with your operating system. See <a href="slide-notes.html#using_third_party_voices_for_speech_text">Using third-party voices for speech text</a>.</p> 
         <p></p> 
        </div> 
       </div> 
       <div class="text parbase section"> 
        <a name="main-pars_text_3"></a> 
        <div class="text">
         <p>When you use NeoSpeech voices, you can modify different aspects of the narration using the native NeoSpeech tags. See <a href="http://www.adobe.com/go/learn_neospeechtags_en">www.adobe.com/go/learn_neospeechtags_en</a> to know how you can tweak the speed, pitch, and volume of NeoSpeech voices.</p> 
         <p>When you convert text to speech, the audio file is inserted into the corresponding slide. Multiple slide notes are merged into a single audio file and appear on the Timeline as a single object. You can also convert slide notes in the panel to closed captions in a single click.</p> 
         <p>The audio file pauses at locations where you have inserted commas or periods in the slide notes.</p> 
         <p>When you play the movie:</p> 
         <ul> 
          <li><p>The slide notes are read aloud to the user.</p> </li> 
          <li><p>The closed caption text on the slide is displayed to the user.</p> </li> 
         </ul>
