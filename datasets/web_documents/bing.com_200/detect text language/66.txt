       <h2>Text Analytics API</h2> 
       <p></p>
       <p>The Text Analytics API is a suite of text analytics web services built with best-in-class Microsoft machine learning algorithms. The API can be used to analyze unstructured text for tasks such as sentiment analysis, key phrase extraction and language detection. No training data is needed to use this API; just bring your text data. This API uses advanced natural language processing techniques to deliver best in class predictions.</p> 
       <p>Further documentation can be found in <a href="https://docs.microsoft.com/en-us/azure/cognitive-services/text-analytics/overview">Overview</a>.</p> 
       <p></p> 
       <div class="panel"> 
        <h3>Detect Language</h3> 
        <p></p>
        <p>The API returns the detected language and a numeric score between 0 and 1. Scores close to 1 indicate 100% certainty that the identified language is true. A total of 120 languages are supported.</p>
