      <p class="lead">Anything that doesnt fall inside posts, pages or taxonomy goes into String Translation. This includes the sites tagline, general texts in Admin screens, widget titles and many other texts.</p> 
      <h2>Activating the module</h2> 
      <p>WPMLs String Translation module is part of the <strong>Multilingual CMS</strong> package. To enable it, you first need to download and install it from your WPML.org account, under the <strong>Downloads</strong> section.</p> 
      <p>Then, go to the&nbsp;<strong>WPML </strong>-&gt;&nbsp;<strong>String Translation&nbsp;</strong>page in your site.</p> 
      <div id="attachment_661911" style="width: 196px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation.png"><img class="size-medium wp-image-661911" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation-186x300.png" alt="String translation" width="186" height="300" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation-186x300.png 186w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation-93x150.png 93w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation-636x1024.png 636w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/string-translation.png 1175w" sizes="(max-width: 186px) 100vw, 186px"></a>
       <p class="wp-caption-text">String Translation admin screen</p>
      </div> 
      <h2>Choosing Which Strings to Display</h2> 
      <p>A complex site may contain many strings to translate. Use the display filter, at the top of the String Translation page, to determine which strings to display.</p> 
      <ul> 
       <li><strong>Which strings to display</strong>  choose the translation status (everything, translation complete, translation needed or waiting for translator)</li> 
       <li><strong>Strings domain</strong>  WPML organizes strings into different domains, making it easier to find a specific string. If, for example, you are looking for a widget string, select the <strong>Widgets</strong>&nbsp;domain</li> 
       <li><strong>Search</strong>  you can enter a search value to narrow down specific strings</li> 
      </ul> 
      <h2>Translating Strings</h2> 
      <p>Click on the <strong>translations</strong> link to open the translation editor.</p> 
      <div id="attachment_661914" style="width: 310px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings.png"><img class="size-medium wp-image-661914" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings-300x90.png" alt="Translating text strings" width="300" height="90" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings-300x90.png 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings-150x45.png 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings-1024x308.png 1024w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/translating_strings.png 1173w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Translating a string</p>
      </div> 
      <p>Be sure to click on <strong>translation is complete</strong>&nbsp;after you translate. Incomplete translations will not appear on the site.</p> 
      <p>If you are using WPMLs Translation Management module, you can also send strings to translation by your translators.</p> 
      <p>Click on the checkboxes next to strings, select the language from the list (in the<strong> Translation options</strong> box) and choose to send to translation. Please note that you cannot choose a specific translator. All translators within a language pair would be able to translate the strings youve chosen.</p> 
      <div id="attachment_661915" style="width: 310px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation.png"><img class="size-medium wp-image-661915" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation-300x96.png" alt="Send strings to translation" width="300" height="96" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation-300x96.png 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation-150x48.png 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation-1024x327.png 1024w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/send_strings_to_translation.png 1185w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Sending strings to translation</p>
      </div> 
      <h3 id="auto-register-fe">Deleting strings</h3> 
      <p>When you no longer use a certain theme or a plugin you might wish to remove the strings associated with it. Simply select the strings you want to remove and click <strong>Delete selected strings</strong>.</p> 
      <div id="attachment_1205005" style="width: 310px" class="wp-caption aligncenter">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/wpml-string-translation-deleting-strings.png"><img class="size-medium wp-image-1205005" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/wpml-string-translation-deleting-strings-300x225.png" alt="Deleting selected string" width="300" height="225" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/wpml-string-translation-deleting-strings-300x225.png 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/wpml-string-translation-deleting-strings-150x113.png 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/wpml-string-translation-deleting-strings.png 561w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Deleting selected string</p>
      </div> 
      <h2>Original Language of Strings</h2> 
      <p>You can set the source language of specific strings or all strings that belong to a domain (context).</p> 
      <p>To change the language of specific strings, select them and click on the new <strong>Change language of selected strings</strong> button. Then, select the new language.</p> 
      <div id="attachment_720441" style="width: 310px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/change-the-language-of-selected-strings.jpg"><img class="size-medium wp-image-720441" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/change-the-language-of-selected-strings-300x148.jpg" alt="" width="300" height="148" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/change-the-language-of-selected-strings-300x148.jpg 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/change-the-language-of-selected-strings-150x74.jpg 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/change-the-language-of-selected-strings.jpg 661w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Change the language of selected strings</p>
      </div> 
      <p>You can also change the language of entire contexts. This is useful when you get a theme or a plugin, thats coded in a language other than English. Click on the new link, at the top of the String Translation page, <strong>Language of domains</strong>.</p> 
      <div id="attachment_720444" style="width: 310px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains.jpg"><img class="size-medium wp-image-720444" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-300x242.jpg" alt="" width="300" height="242" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-300x242.jpg 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-150x121.jpg 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains.jpg 567w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Click on the link to Languages of domains</p>
      </div> 
      <p>Then, choose the new language.</p> 
      <div id="attachment_720448" style="width: 310px" class="wp-caption alignnone">
       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-dialog.jpg"><img class="size-medium wp-image-720448" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-dialog-300x234.jpg" alt="" width="300" height="234" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-dialog-300x234.jpg 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-dialog-150x117.jpg 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2015/10/languages-of-domains-dialog.jpg 528w" sizes="(max-width: 300px) 100vw, 300px"></a>

      <p>The dialog will show you a table of the strings in that domain. You can choose the new language and also make it the default language for new strings that will be added to that domain.</p>

      <p>How to use:</p> 
      <ol> 
       <li>Enable the <strong>Track where strings appear on the site&nbsp;</strong>option.</li> 
       <li>Visit the sites front-end pages.</li> 
       <li>Switch to a language other than the sites default language.</li> 
       <li>WPML will register any string that appears on the page you are viewing. Move from page to page to cover all the strings you need.</li> 
      </ol> 
      <p>Please note that this is a database-intense operation. You should only enable it when building the site. When you are done registering strings, dont forget to return to the String Translation admin page and disable Track where strings appear on the site. Your sites speed will be significantly lower if you keep this option running in production mode.</p> 
      <h3>Auto register strings for translation</h3> 
      <p>WPML uses static code analysis to locate strings that require translation in the theme and plugins. In some cases, the static code-scan cannot reliably find all strings. This often happens when strings are generated dynamically using code (not really static/hard-coded).</p> 
      <p>By enabling auto-registration, WPML will register strings that need translation at display-time. The conservative option is to register strings when logged in as an administrator. This means that when a logged-in admin browses the site, WPML registers for translation any strings that are displayed.</p> 
      <p>This operation may increase an amount of database queries while strings are registering for translation.</p> 
      <h3>Import / export .po</h3> 
      <p>You can import translations from other sources and export your current translations using .po files. If you choose to export strings without their translations, the file extension will be .pot (Portable Object Template). If you choose to include translations, the extension would be .po.</p> 
      <p>When you import translations, you must choose the language to import to.</p> 
      <h3 id="admin_texts">Translating strings in the wp_options table</h3> 
      <p>Many themes and plugins store texts in the <em>wp_options</em> table. This is also the case with Widgets. Widgets also store their setting fields in the wp-options table. You can tell WPML to register these strings to translation.</p> 
      <p>To do this, click on <strong>Translate texts in admin screens</strong>. This will open a dialog showing the strings that WPML finds in your site.</p>

       <a href="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/export_selected-strings_as_a_wpml_config_file.png"><img class="size-medium wp-image-558811" src="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/export_selected-strings_as_a_wpml_config_file-300x234.png" alt="Export strings as a wpml config file" width="300" height="234" srcset="https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/export_selected-strings_as_a_wpml_config_file-300x234.png 300w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/export_selected-strings_as_a_wpml_config_file-150x117.png 150w, https://d2salfytceyqoe.cloudfront.net/wp-content/uploads/2011/08/export_selected-strings_as_a_wpml_config_file.png 793w" sizes="(max-width: 300px) 100vw, 300px"></a>
       <p class="wp-caption-text">Selecting strings in the wp_options table for translation</p>
      </div> 
      <p><strong>Developers note</strong>: Theme and plugin authors are encouraged to include a <a href="https://wpml.org/documentation/support/language-configuration-files/">language configuration file</a> in their code, which tells WPML which strings from the wp_options table to register for translation. This can save a lot of time for end-users configuring multilingual sites. You can easily export the strings you want by using the <span id="id_l.I.ic.icr.it.issSum" class="issue-summary issue-summary_fsi fsi__summary">Export selected strings as a WPML config file that can be added to themes or plugins </span> button (see image above).</p> 
      <p>If your option includes a lot of sub-options, you can use the <strong>asterisk (*) </strong>as a wildcard&nbsp;character in your configuration file. You can find more information and examples on our <a href="https://wpml.org/documentation/support/language-configuration-files/">page about creating language configuration files</a>.</p>
