             <div class="post-text" itemprop="text"> 
              <p>Altered piece of code I often use:</p> 
              <pre><code>Date.prototype.yyyymmdd = function() {
  var mm = this.getMonth() + 1; // getMonth() is zero-based
  var dd = this.getDate();

  return [this.getFullYear(),
          (mm&gt;9 ? '' : '0') + mm,
          (dd&gt;9 ? '' : '0') + dd
         ].join('');
};

var date = new Date();
date.yyyymmdd();
</code></pre>

              <p>In addition to o-o's answer I'd like to recommend separating logic operations from the return and put them as ternaries in the variables instead. </p> 
              <p>Also, use <code>concat()</code> to ensure safe concatenation of variables</p> 
              <pre><code> Date.prototype.yyyymmdd = function() {
   var yyyy = this.getFullYear();
   var mm = this.getMonth() &lt; 9 ? "0" + (this.getMonth() + 1) : (this.getMonth() + 1); // getMonth() is zero-based
   var dd  = this.getDate() &lt; 10 ? "0" + this.getDate() : this.getDate();
   return "".concat(yyyy).concat(mm).concat(dd);
  };

 Date.prototype.yyyymmddhhmm = function() {
   var yyyy = this.getFullYear();
   var mm = this.getMonth() &lt; 9 ? "0" + (this.getMonth() + 1) : (this.getMonth() + 1); // getMonth() is zero-based
   var dd  = this.getDate() &lt; 10 ? "0" + this.getDate() : this.getDate();
   var hh = this.getHours() &lt; 10 ? "0" + this.getHours() : this.getHours();
   var min = this.getMinutes() &lt; 10 ? "0" + this.getMinutes() : this.getMinutes();
   return "".concat(yyyy).concat(mm).concat(dd).concat(hh).concat(min);
  };

 Date.prototype.yyyymmddhhmmss = function() {
   var yyyy = this.getFullYear();
   var mm = this.getMonth() &lt; 9 ? "0" + (this.getMonth() + 1) : (this.getMonth() + 1); // getMonth() is zero-based
   var dd  = this.getDate() &lt; 10 ? "0" + this.getDate() : this.getDate();
   var hh = this.getHours() &lt; 10 ? "0" + this.getHours() : this.getHours();
   var min = this.getMinutes() &lt; 10 ? "0" + this.getMinutes() : this.getMinutes();
   var ss = this.getSeconds() &lt; 10 ? "0" + this.getSeconds() : this.getSeconds();
   return "".concat(yyyy).concat(mm).concat(dd).concat(hh).concat(min).concat(ss);
  };

var d = new Date();
d.yyyymmdd();
d.yyyymmddhhmm();
d.yyyymmddhhmmss();
</code></pre>

              <pre><code>function toYYYYMMDD(d) {
    var yyyy = d.getFullYear().toString();
    var mm = (d.getMonth() + 101).toString().slice(-2);
    var dd = (d.getDate() + 100).toString().slice(-2);
    return yyyy + mm + dd;
}

             <div class="post-text" itemprop="text"> 
              <p>Another way is to use <a href="https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Global_Objects/Date/toLocaleDateString" rel="nofollow"><code>toLocaleDateString</code></a> with a locale that has a <a href="https://en.wikipedia.org/wiki/Calendar_date#Gregorian.2C_year-month-day_.28YMD.29" rel="nofollow">big-endian date format standard</a>, such as Sweden, Lithuania, Hungary, South Korea, ...:</p> 
              <pre><code>date.toLocaleDateString('se')
</code></pre> 
              <p>To remove the delimiters (<code>-</code>) is just a matter of replacing the non-digits:</p> 
              <p></p>
              <div class="snippet" data-lang="js" data-hide="false" data-console="true" data-babel="false"> 
               <div class="snippet-code"> 
                <pre class="snippet-code-js lang-js prettyprint-override"><code>console.log( new Date().toLocaleDateString('se').replace(/\D/g, '') );</code></pre> 
               </div> 
              </div> 
              <p></p> 
              <p>This does not have the potential error you can get with UTC date formats: the UTC date may be one day off compared to the date in the local time zone.</p>

              <p>Working from @o-o's answer this will give you back the string of the date according to a format string. You can easily add a 2 digit year regex for the year &amp; milliseconds and the such if you need them.</p> 
              <pre><code>Date.prototype.getFromFormat = function(format) {
    var yyyy = this.getFullYear().toString();
    format = format.replace(/yyyy/g, yyyy)
    var mm = (this.getMonth()+1).toString(); 
    format = format.replace(/mm/g, (mm[1]?mm:"0"+mm[0]));
    var dd  = this.getDate().toString();
    format = format.replace(/dd/g, (dd[1]?dd:"0"+dd[0]));
    var hh = this.getHours().toString();
    format = format.replace(/hh/g, (hh[1]?hh:"0"+hh[0]));
    var ii = this.getMinutes().toString();
    format = format.replace(/ii/g, (ii[1]?ii:"0"+ii[0]));
    var ss  = this.getSeconds().toString();
    format = format.replace(/ss/g, (ss[1]?ss:"0"+ss[0]));
    return format;
};

d = new Date();
var date = d.getFromFormat('yyyy-mm-dd hh:ii:ss');
alert(date);
</code></pre> 
              <p>I don't know how efficient that is however, especially perf wise because it uses a lot of regex. It could probably use some work I do not master pure js.</p>

              <p>I usually use the code below when I need to do this.</p> 
              <pre><code>var date = new Date($.now());
var dateString = (date.getFullYear() + '-'
    + ('0' + (date.getMonth() + 1)).slice(-2)
    + '-' + ('0' + (date.getDate())).slice(-2));
console.log(dateString); //Will print "2015-09-18" when this comment was written
</code></pre> 
              <p>To explain, .slice(-2) gives us the last two characters of the string.</p> 
              <p>So no matter what, we can add "0" to the day or month, and just ask for the last two since those are always the two we want.</p> 
              <p>So if the MyDate.getMonth() returns 9, it will be:</p> 
              <pre><code>("0" + "9") // Giving us "09"
</code></pre> 
              <p>so adding .slice(-2) on that gives us the last two characters which is:</p> 
              <pre><code>("0" + "9").slice(-2)

"09"
</code></pre> 
              <p>But if date.getMonth() returns 10, it will be:</p> 
              <pre><code>("0" + "10") // Giving us "010"
</code></pre> 
              <p>so adding .slice(-2) gives us the last two characters, or:</p> 
              <pre><code>("0" + "10").slice(-2)

"10"
</code></pre>

              <p>and then just format your stuff:</p> 
              <p><code>var myYYYYmmddDate = dateformat(new Date(), 'yyyy-mm-dd');</code></p>

              <pre><code>Date.prototype.ddmmyyyy = function(delimiter) {
    var yyyy = this.getFullYear().toString();
    var mm = (this.getMonth()+1).toString(); // getMonth() is zero-based
    var dd  = this.getDate().toString();
    return (dd[1]?dd:"0"+dd[0]) + delimiter + (mm[1]?mm:"0"+mm[0]) + delimiter +yyyy  ; // padding
};
</code></pre>

              <p>this post helped me to write this helper, so I share it in case some one is looking for this solution, it supports all variations of yyyy, mm, dd</p> 
              <pre><code>Date.prototype.formattedDate = function (pattern) {
    formattedDate = pattern.replace('yyyy', this.getFullYear().toString());
    var mm = (this.getMonth() + 1).toString(); // getMonth() is zero-based
    mm = mm.length &gt; 1 ? mm : '0' + mm;
    formattedDate = formattedDate.replace('mm', mm);
    var dd = this.getDate().toString();
    dd = dd.length &gt; 1 ? dd : '0' + dd;
    formattedDate = formattedDate.replace('dd', dd);
    return formattedDate;
};

d = new Date();
pattern = 'yyyymmdd';  // 20150813
d.formattedDate(pattern);

pattern = 'yyyy-mm-dd';
d.formattedDate(pattern); // 2015-08-13
</code></pre>

             <div class="post-text" itemprop="text"> 
              <p>Here is a more generic approach which allows both <em>date</em> and <em>time</em> components and is identically sortable as either number or string.</p> 
              <p>Based on the number order of Date ISO format, convert to a local timezone and remove non-digits. i.e.:</p> 
              <pre><code>// monkey patch version
Date.prototype.IsoNum = function (n) {
    var tzoffset = this.getTimezoneOffset() * 60000; //offset in milliseconds
    var localISOTime = (new Date(this - tzoffset)).toISOString().slice(0,-1);
    return localISOTime.replace(/[-T:\.Z]/g, '').substring(0,n || 20); // YYYYMMDD
}
</code></pre> 
              <p><strong>Usage</strong></p> 
              <pre><code>var d = new Date();
// Tue Jul 28 2015 15:02:53 GMT+0200 (W. Europe Daylight Time)
console.log(d.IsoNum(8));  // "20150728"
console.log(d.IsoNum(12)); // "201507281502"
console.log(d.IsoNum());   // "20150728150253272"
</code></pre>
