          <h1>Node Express Passport Facebook Twitter Google GitHub Login</h1> 
          <span class="list-date">by Didin J. on Aug 30, 2017</span> 
          <img src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-300817093943.png" alt="Node Express Passport Facebook Twitter Google GitHub Login" class="img-responsive padding-top"> 
          <blockquote>
           <h2>Step by step tutorial of Node, Express, Mongoose, Passport.js, Facebook, Twitter, Google, and GitHub login or authentication.</h2>
          </blockquote> 
          <div class="article-content">
           <p>This tutorial shows you how to create multiple social media (Facebook, Twitter, Google and GitHub) authentication (login) using Node, Express, Mongoose and `Passport.js`. This authentication is using OAuth provider which has the different name and authentication keys between Facebook, Twitter, Google and GitHub. If you like to use form based authentication, we already publish authentication or login tutorial using <a href="https://www.djamware.com/post/58bd823080aca7585c808ebf/nodejs-expressjs-mongoosejs-and-passportjs-authentication">Node.js, Express.js, </a><a href="https://www.djamware.com/post/58bd823080aca7585c808ebf/nodejs-expressjs-mongoosejs-and-passportjs-authentication">Passport.js</a><a href="https://www.djamware.com/post/58bd823080aca7585c808ebf/nodejs-expressjs-mongoosejs-and-passportjs-authentication"> and Mongoose.js</a>.</p> 
           <p> <span id="ezoic-pub-ad-placeholder-100"></span><span style="display:inline-block;float:none !important;margin-bottom:2px !important;margin-left:0px !important;margin-right:0px !important;margin-top:2px !important;min-height:110px;min-width:728px;" class="ezoic-ad medrectangle-3 adtester-container adtester-container-100" data-ez-name="djamware_com-medrectangle-3"><span id="div-gpt-ad-djamware_com-medrectangle-3-0" ezaw="728" ezah="90" style="position:relative;z-index:0;display:inline-block;" class="ezoic-ad"><script data-cfasync="false" type="text/javascript" style="display:none;">eval(ez_write_tag([[728,90],'djamware_com-medrectangle-3','ezslot_4']));</script></span></span> </p> 
           <p>The following tools and library are required for this tutorial:</p> 
           <p>- <a href="https://nodejs.org/">Node.js</a><br> - <a href="https://expressjs.com/en/starter/generator.html">Express.js</a> Application Generator<br> - <a href="http://passportjs.org/">Passport.js</a>, passport-facebook, passport-twitter, passport-google, passport-github Library<br> - Terminal or Node Command Line<br> - Text Editor or IDE (<a href="https://atom.io/">Atom</a>, <a href="https://netbeans.org/">Netbeans</a>, etc)</p> 
           <p>The first thing before starting the tutorial is making sure that `Node.js` and it's command line installed and working properly. After `Node.js` installed, open terminal or Node command line then type this command.</p> 
           <pre><code>node -v</code></pre> 
           <p>You will see the version like this.</p> 
           <pre><code>v6.9.4</code></pre> 
           <p>Now, install Express generator for generating new Express application by this command.</p> 
           <pre><code>sudo npm install express-generator -g</code></pre> 
           <h3><br> <strong>1. Get Facebook </strong><strong>APP ID</strong><strong> and Secret</strong></h3> 
           <p>Facebook using APPID and App Secret that use for OAuth provider requirements for authentication (login). Open your browser then go to <a href="https://developers.facebook.com/apps/">Facebook Developer Dashboard</a>. Log in using your Facebook developer account or your standard user account.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Facebook Developers Dashboard" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-200817205624.png" style="border-style:solid; border-width:1px; height:301px; width:700px"><span id="ezoic-pub-ad-placeholder-" null" class="ezoic-adpicker-ad"></span></p> 
           <p>Click Add a New App button then fill Display Name (ours using `MEANSocial`) and email address.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Facebook Create New App ID" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-200817205950.png" style="border-style:solid; border-width:1px; height:175px; width:400px"></p> 
           <p>Click Create App ID button, enter captcha then click Submit button. It will redirect to Facebook App dashboard.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Facebook App Settings" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-200817210331.png" style="border-style:solid; border-width:1px; height:356px; width:700px"></p> 
           <p>Click Settings menu on the left menu. Click Add Platform then choose the website. Fill site URL with the callback URL for OAuth authentication callback URL.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Facebook App ID and Secret" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-200817211004.png" style="border-style:solid; border-width:1px; height:519px; width:700px"></p> 
           <p>Don't forget to write down the App ID and App Secret to your notepad.</p> 
           <h3><br> <strong>2. Get Twitter Consumer Key (API Key) and Consumer Secret (API Secret)</strong></h3> 
           <p>To get Twitter Consumer Key (API Key) and Consumer Secret (API Secret), go to <a href="https://apps.twitter.com/">Twitter Apps console</a>. Log in using your existing Twitter account.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Twitter Apps Dashboard" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817094825.png" style="border-style:solid; border-width:1px; height:346px; width:700px"></p> 
           <p>Click Create new app button then fill all required fields. Fill callback URL with your MEAN application URL, ours using "http://127.0.0.1:3000/auth/twitter/callback". Check developer agreement then clicks Create Your Twitter Application button. It will redirect to your new application dashboard.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Twitter Application Management" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817095443.png" style="border-style:solid; border-width:1px; height:473px; width:700px"></p> 
           <p>Click on Keys and Access Tokens tab.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Twitter Application Settings" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817095705.png" style="border-style:solid; border-width:1px; height:223px; width:400px"></p> 
           <p>Now, write down the Consumer Key (API Key) and Consumer Secret (API Secret) to your notepad.</p> 
           <h3><br> <strong>3. Get Google Client ID and Client Secret</strong></h3> 
           <p>To get the Google Client ID and Client Secret, go to <a href="https://console.developers.google.com">Google Developer Console</a>. Login using your Gmail account. On the selected project drop down&nbsp;click "+" button to create the new project then give it a name.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Google New Project" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817100519.png" style="border-style:solid; border-width:1px; height:331px; width:400px"></p> 
           <p>Click Create button then it will redirect to Project API dashboard.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Google Project Dashboard" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817100733.png" style="border-style:solid; border-width:1px; height:198px; width:400px"></p> 
           <p>Click Credentials menu from left menu then click OAuth consent screen tabs to create product name, just fill product name and email then click save button.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Google Credentials" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817102845.png" style="border-style:solid; border-width:1px; height:400px; width:251px"></p> 
           <p>Back to Credentials tab then click Create Credentials and choose OAuth Client ID. Choose Web Application then fill the Authorized redirect URI with MEAN application URL like previously then click create.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Google Create Client ID" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817103900.png" style="border-style:solid; border-width:1px; height:352px; width:400px"></p> 
           <p>Now, you get Client ID and Secret. Write it down on your notepad.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Google Client ID for Web Application" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817104339.png" style="border-style:solid; border-width:1px; height:375px; width:500px"></p> 
           <h3><br> <strong>4. Get GitHub Client ID and Client Secret</strong></h3> 
           <p>To get GitHub client ID and client secret, go to <a href="https://developer.github.com/">Github developers</a>. Log in using your GitHub account.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Github Developer Console" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817104805.png" style="border-style:solid; border-width:1px; height:253px; width:700px"></p> 
           <p>Click Register New OAuth Application button. Fill all required fields, use your MEAN application URL for Authorization callback URL then click Register Application button.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - GitHub Register OAuth Application" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817105311.png" style="border-style:solid; border-width:1px; height:400px; width:400px"></p> 
           <p>Now, you get the Client ID and Client Secret. Write down to your notepad.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - GitHub Client ID and Secret" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-210817105545.png" style="border-style:solid; border-width:1px; height:335px; width:600px"></p> 
           <h3><br> <strong>5. Generate Express Application</strong></h3> 
           <p>Now, we will create Node, Express and Mongoose Web Application. Open terminal od Node command line then types this command.</p> 
           <pre><code>express --view=pug node-passport-social</code></pre> 
           <p>That command creates new Express web application with using `pug` view template, this different than the previous tutorial that uses `ejs` or `jade`. Here's the output from that command.&nbsp; &nbsp;</p> 
           <pre><code>create : node-passport-social

           <p>Now, do as terminal output command. Type or paste this command.</p> 
           <pre><code>cd node-passport-social &amp;&amp; npm install</code></pre> 
           <p>Next, check if the Express web application working by type this command.</p> 
           <pre><code>nodemon</code></pre> 
           <p>or</p> 
           <pre><code>npm start</code></pre> 
           <p>Open the browser then go to `http://localhost:3000`, you will see this page if everything is working properly.</p> 
           <p><img alt="Node Express Passport Facebook Twitter Google GitHub Login - Express Run Application" src="https://s3-ap-southeast-1.amazonaws.com/djamblog/article-190217090105.png" style="border-style:solid; border-width:1px; height:267px; width:400px"></p> 
           <h3><br> <strong>6. Add Mongoose as MongoDB and Node ORM</strong></h3> 
           <p>Now, we will add Mongoose module and dependencies. Stop the app by press `CTRL+C` key. Type this command to install Mongoose the modules and dependencies.</p> 
           <p> <span id="ezoic-pub-ad-placeholder-113"></span><span style="display:block !important;float:none !important;margin-bottom:2px !important;margin-left:0px !important;margin-right:0px !important;margin-top:2px !important;min-height:270px;min-width:300px;text-align:center !important;" class="ezoic-ad banner-1 adtester-container adtester-container-113" data-ez-name="djamware_com-banner-1"><span id="div-gpt-ad-djamware_com-banner-1-0" ezaw="300" ezah="250" style="position:relative;z-index:0;display:inline-block;" class="ezoic-ad"><script data-cfasync="false" type="text/javascript" style="display:none;">eval(ez_write_tag([[300,250],'djamware_com-banner-1','ezslot_1']));</script></span></span> </p> 
           <pre><code>npm install mongoose --save</code></pre> 
           <p>Also, install this dependency to make easy query find or create data.</p> 
           <pre><code>npm install find-or-create --save</code></pre> 
           <p>Open and edit app.js on the root of the project. Add require line for Mongoose.</p> 
           <pre><code>var mongoose = require('mongoose');</code></pre>
