       <h1>Product Overview</h1> 
       <p> Version 2 of the Big Faceless Graph Library builds on everything we were asked for in version 1 - we believe it's the most comprehensive Graph library on the market. Key features include: </p> 
       <ul> 
        <li>XML and JSP Tag Library interface, for extremely easy integration with web projects</li> 
        <li>SOAP and XML Web Services interface for graph creation</li> 
        <li>Very comprehensive API and Tag documentation with dozens of examples</li> 
        <li>Embed JavaScript in your tags to easily create powerful rollovers</li> 
        <li>Full 3D engine with adjustable light source produces the most realistic graphs on the market </li> 
        <li>Output to PNG, SVG, Macromedia Flash, PDF or <code>java.awt.Image</code> for use in Applets </li> 
        <li>Graph Dates, Currencies, Percentages, Numbers and Discrete data out of the box, easily extended for total control over the axes </li> 
        <li>Mix Bars, Lines, Candles, Bubbles, all on the same graph</li> 
        <li>Create <a href="/blog/2009/03/25/funnel_charts/">funnel charts</a></li> 
        <li>Add text, lines or custom markers anywhere on the graph, including axes</li> 
        <li>Use up to four axes on the one graph, including logarithmic axes</li> 
        <li>Plot mathematical functions, data samples and spline curves fitted to data, all on the same graph. </li> 
        <li>It's fast - plot several 3D graphs a <i>second</i> to PDF or SVG </li> 
       </ul> 
       <h2>Product Benefits</h2> 
       <p> There are many interesting features of the Graph Library, but we suspect the most useful is the Tag Library interface. This makes embedding graphs in web pages almost trivial - as easy as creating an HTML table, for instance. Most people will embed the graph as a PNG, which means JavaScript can be used to create rollovers and actions in exactly the same way as an image map. If you prefer a vector image, SVG and Flash can be created by changing one word of your XML. </p> 
       <p> The most popular request we received with version 1 was mixed lines and bars on the same graph. We've gone one better and in version 2 any number of data sets can be mixed on the graph, overlaid on each other as if they were panes of glass. You can use up to four axes (one on each side) to mix data measuring different scales on the one graph and have full control over the scaling. </p> 
       <p>For further reading <a href="/blog/categories/graph">vist the Graph Library section of our blog</a>. </p>
