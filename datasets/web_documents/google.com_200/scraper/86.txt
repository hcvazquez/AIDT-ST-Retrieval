         <p>Ok, I seemed to have my Arcade games scraped...What was interesting was that I had to play with the default utility settings.</p> 
         <p>I logged into the pi via SSH (you could do this local as well) and ran the following to start the utility...</p> 
         <p>sudo ~/RetroPie-Setup/retropie_setup.sh</p> 
         <p>Selected option for Configuration/Tools...</p> 
         <p>Selected 828 Scraper...</p> 
         <p>Made sure of the following settings...</p> 
         <p>arcade source (MameDB)<br> console source (thegamesdb)<br> gamelist (Overwrite)</p> 
         <p>This worked for all my Mame Based games...</p> 
         <p>It is still not working for my Genesis, NES etc....on those I am getting a hashing error. Stay tuned.....</p>
