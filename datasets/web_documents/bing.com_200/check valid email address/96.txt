        <h1>How to validate IP address with regular expression</h1> 
        <div class="post-meta"> 
         <p class="post-meta">By <a href="https://www.mkyong.com/author/mkyong/" title="mkyong" rel="author">mkyong</a> | <time datetime="2009-11-07T08:24:44+00:00">November 7, 2009</time> | Updated : November 13, 2012 | Viewed : 253,727 times <span>+714 pv/w</span></p> 
        </div> 
        <!-- /.post-meta --> 
        <div class="post-content"> 
         <p>IP Address Regular Expression Pattern</p> 
         <pre><code class="language-java">
^([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.
([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.([01]?\\d\\d?|2[0-4]\\d|25[0-5])$
</code></pre> 
         <p>Description</p> 
         <pre><code class="language-java">
^		#start of the line
 (		#  start of group #1
   [01]?\\d\\d? #    Can be one or two digits. If three digits appear, it must start either 0 or 1
		#    e.g ([0-9], [0-9][0-9],[0-1][0-9][0-9])
    |		#    ...or
   2[0-4]\\d	#    start with 2, follow by 0-4 and end with any digit (2[0-4][0-9]) 
    |           #    ...or
   25[0-5]      #    start with 2, follow by 5 and ends with 0-5 (25[0-5]) 
 )		#  end of group #2
  \.            #  follow by a dot "."
....            # repeat with 3 times (3x)
$		#end of the line
</code></pre> 
         <p>Whole combination means, digit from 0 to 255 and follow by a dot “.”, repeat 4 time and ending with no dot “.” Valid IP address format is “0-255.0-255.0-255.0-255”.</p> 
         <h2>1. Java Regular Expression Example</h2> 
         <div class="filename">
          IPAddressValidator.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.regex;

import java.util.regex.Matcher;
import java.util.regex.Pattern;
 
public class IPAddressValidator{
	
    private Pattern pattern;
    private Matcher matcher;
 
    private static final String IPADDRESS_PATTERN = 
		"^([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\." +
		"([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\." +
		"([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\." +
		"([01]?\\d\\d?|2[0-4]\\d|25[0-5])$";
	  
    public IPAddressValidator(){
	  pattern = Pattern.compile(IPADDRESS_PATTERN);
    }
	  
   /**
    * Validate ip address with regular expression
    * @param ip ip address for validation
    * @return true valid ip address, false invalid ip address
    */
    public boolean validate(final String ip){		  
	  matcher = pattern.matcher(ip);
	  return matcher.matches();	    	    
    }
}
</code></pre>

         <h2>3. IP address that doesn’t match :</h2> 
         <p>1. “10.10.10” – must have 4 “.”<br> 2. “10.10” – must have 4 “.”<br> 3. “10” – must have 4 “.”<br> 4. “a.a.a.a” – only digit has allowed<br> 5. “10.0.0.a” – only digit has allowed<br> 6. “10.10.10.256” – digit must between [0-255]<br> 7. “222.222.2.999” – digit must between [0-255]<br> 8. “999.10.10.20” – digit must between [0-255]<br> 9. “2222.22.22.22” – digit must between [0-255]<br> 10. “22.2222.22.2” – digit must between [0-255]</p> 
         <h2>4. Unit Test</h2> 
         <div class="filename">
          IPAddressValidatorTest.java
         </div> 
         <pre><code class="language-java">
package com.mkyong.regex;

import org.testng.Assert;
import org.testng.annotations.*;
 
/**
 * IPAddress validator Testing
 * @author mkyong
 *
 */
public class IPAddressValidatorTest {
 
	private IPAddressValidator ipAddressValidator;
    
	@BeforeClass
        public void initData(){
		ipAddressValidator = new IPAddressValidator();
        }
    
	@DataProvider
	public Object[][] ValidIPAddressProvider() {
		return new Object[][]{
		   new Object[] {"1.1.1.1"},new Object[] {"255.255.255.255"},
                   new Object[] {"192.168.1.1"},new Object[] {"10.10.1.1"},
                   new Object[] {"132.254.111.10"},new Object[] {"26.10.2.10"},
		   new Object[] {"127.0.0.1"}
		};
	}
	
	@DataProvider
	public Object[][] InvalidIPAddressProvider() {
		return new Object[][]{
		   new Object[] {"10.10.10"},new Object[] {"10.10"},
                   new Object[] {"10"},new Object[] {"a.a.a.a"},
                   new Object[] {"10.0.0.a"},new Object[] {"10.10.10.256"},
		   new Object[] {"222.222.2.999"},new Object[] {"999.10.10.20"},
                   new Object[] {"2222.22.22.22"},new Object[] {"22.2222.22.2"},
                   new Object[] {"10.10.10"},new Object[] {"10.10.10"},	
		};
	}
	
	@Test(dataProvider = "ValidIPAddressProvider")
	public void ValidIPAddressTest(String ip) {
		   boolean valid = ipAddressValidator.validate(ip);
		   System.out.println("IPAddress is valid : " + ip + " , " + valid);
		   Assert.assertEquals(true, valid);
	}
	
	@Test(dataProvider = "InvalidIPAddressProvider",
                 dependsOnMethods="ValidIPAddressTest")
	public void InValidIPAddressTest(String ip) {
		   boolean valid = ipAddressValidator.validate(ip);
		   System.out.println("IPAddress is valid : " + ip + " , " + valid);
		   Assert.assertEquals(false, valid); 
	}	
}
