        <span id="ctl00_ContentLeft_LabelBlogDetail"><p>Quicksort is a divide-and-conquer sorting algorithm in which division is dynamically carried out (as opposed to static division in Mergesort).<br> <br> The three steps of Quicksort are as follows:<br> <br> <strong>Divide</strong>: Rearrange the elements and split the array into two subarrays and an element in between such that so that each element in the left subarray is less than or equal the middle element and each element in the right subarray is greater than the middle element.<br> <br> <strong>Conquer</strong>: Recursively sort the two subarrays.<br> <br> <strong>Combine</strong>: None.<br> <br> <strong>Algorithm<br> </strong><br> <strong># choose pivot<br> <br> </strong>swap a[1,rand(1,n)]<br> <br> <strong># 2-way partition<br> </strong><br> k = 1<br> for i = 2:n, if a[i] &lt; a[1], swap a[++k,i]<br> swap a[1,k]<br> ? invariant: a[1..k-1] &lt; a[k] &lt;= a[k+1..n]<br> <br> <strong># recursive sorts</strong><br> <br> sort a[1..k-1]<br> sort a[k+1,n]</p> <p style="line-height: normal;
