        <p><strong>Number:</strong> It can be a number or a valid numerical expression.</p> 
        <ul> 
         <li>If the <em>number</em> argument is positive or negative double or float value, Math.getExponent function will return the unbiased exponent.</li> 
         <li>If the <em>number</em> argument is not a number(<em><strong>NaN</strong></em>) or Infinite, Math.getExponent function will return <em><strong>Float.MAX_EXPONENT + 1</strong></em>.</li> 
         <li>If the <em>number</em> argument is Zero, Math.getExponent function will return <em><strong>Float.MIN_EXPONENT – 1</strong></em>.</li> 
        </ul> 
        <p>The mathematical formula behind this function is 2<sup>exp</sup> &lt;= number (Where exp is the output or result). For example, We want to find the result of Math.getExponent(24.50)</p> 
        <p>2<sup>exp</sup> &lt;= 24.50</p> 
        <p>2<sup>4</sup> &lt;= 24.50 so answer will be 4. (If our value is greater than 32 then output will be 5 because 2<sup>5</sup> = 32)</p> 
        <p>Java Programming provides two different functions to find the closest exponent of the specified value. Following function will accept positive or negative float value as argument and returns the closest exponent value of type Int.</p>
