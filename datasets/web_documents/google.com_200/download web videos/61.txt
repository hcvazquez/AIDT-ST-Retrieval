              <div class="field-items">
               <div class="field-item even">
                <p>Video.js is a HTML5-based video player with a built-in Flash fallback for older browsers. This means that videos can be played on nearly all devices and operating systems, provided the right codecs are used.</p> 
                <p>This module is a support module for Video.js. It doesn't contain Video.js itself, but integrates it with the File, Link and Video modules after you've installed it.</p> 
                <!--break-->
                <h3>New in version 8.x-1.x</h3> 
                <ul>
                 <li>Support for Video.js 5.0.x</li> 
                 <li>Support for subtitle tracks.</li> 
                 <li>Support for looping, hiding player controls and changing preload behavior.</li> 
                 <li>Setup above settings for each field and view individually, as well as width and height of the player.</li> 
                 <li>Load the Video.js files from the Video.js CDN - downloading and installing the player is no longer necessary!</li> 
                 <li>Locate the Video.js files using the <a href="http://drupal.org/project/libraries" rel="nofollow">Libraries API</a>.</li> 
                </ul>
                <p><span class="project-issue-issue-link project-issue-status-info project-issue-status-7"><a href="/node/2654404" title="Status: Closed (fixed)">#2654404: Drupal 8 Roadmap</a></span></p>
