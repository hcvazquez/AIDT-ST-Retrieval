     <h2>Release Notes</h2> 
     <p>9.2.0.1716 (August 1, 2017)<br>===========================<br>+ Added methods to decode barcodes from image file in memory (stream or byte array).<br>- Fixed rare crashes when Heuristic Mode enabled.<br>- Fixed ReadFromHBitmap() method crashing on x86 platform.<br>= Improved PDF documents rendering.<br>= WIAImageScanner class now able to aquire multiple images from still imaging devices.<br>+ New DPM DataMatrix decoder (BETA).<br>= Other minor improvements and bug fixes.<br><br>9.1.0.1688 (June 1, 2017)<br>=========================<br>= Performance improvements.<br>= Removed obsolete ClientProfile assemblies.<br>= `SearchNegative` and `OptimizedProcessingOfLargeImages` properties got obsolete <br> &nbsp;and replaced with `ImagePreprocessingFilters`.<br>= Median filter replaced with better and faster implementation.<br>+ Added gamma correction filter.<br>- Fixed rare crash on MICR barcodes.<br>- Fixed OMR multi-threading routine.<br>+ Barcode Reader Freeware: Added Image Adjustments tab to Options dialog and Results wizard.<br>+ Added Preview for image preprocessing.<br>= Other minor improvements and bug fixes.<br><br>9.0.0.1676 (April 12, 2017)<br>===========================<br>+ The decoding is multi-threaded now.<br>= Performance improvements.<br>= Linear barcodes are scanned in two direction by default now: from left to right and from right to left.<br>- Minor fix in PDF417 decoding.<br><br>8.90.0.1651 (March 09, 2017)<br>============================<br>= Improved decoding of 180° rotated DataMatrix.<br>+ Added new API for image preprocessing filters.<br>= Old preprocessing properties made obsolete: RemoveNoize, NoiseRemovalFilterSize, NoiseRemovalFilterType, Contrast.<br>+ Added new image preprocessing filters: Scale, Auto-cropping.<br>+ Added enhanced color conversion mode for better decoding of barcodes from photo images.<br>= Updated VideoCameraModule.<br>- Fixed rare error in Code39 decoder.<br><br>8.80.0.1621 (January 11, 2017)<br>==============================<br>- Fixed malfunctioning MaxNumberOfBarcodesPerPage property when different (1D/2D/OMR) barcode types enabled.<br>+ Added Reader.DecoderSpecificOptions for fine tuning of specific decoders.<br>+ Added ScanStep property defining row scan interval for linear barcodes;<br>- Code 128: Fixed zero-length false positives.<br>= Improved DataMatrix decoding.<br>- Intelligent Mail: Fixed zero-padding of value components.<br>+ Added support for profiles - quick way to apply multiple settings at once.<br>= Improved Heuristic mode.<br>- EXIF rotation tag in photo images is correctly handled now.<br>- Improved handling of TIFF images with 204x98 DPI resoultion.<br>= Other minor improvements and bug fixes.<br><br>...</p>
