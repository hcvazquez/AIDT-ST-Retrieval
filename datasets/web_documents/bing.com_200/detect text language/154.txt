         <p class="fpara "></p>
         <p>In <strong>Google Translate API</strong> Tutorial, I have explained how to convert text from one language to another using Google Translate API in PHP / JavaScript / Java.<br> Translate API has daily limit of 2 Million characters per day. You can increase the limit in the cloud console up to 50 Millions characters per day.They charge 20$ for 1 Million characters. Check this URL for <a href="https://developers.google.com/translate/v2/pricing" target="__blank">Pricing Details</a>.</p>
         <p>Topics covered in this article<br> <a href="#api-key">1) Get Google Translate API Key</a><br> <a href="#api-list">2) List of Google Translate API</a><br> <a href="#php-example">3) Google Translate API PHP Example.</a><br> <a href="#java-example">4) Google Translate API Java Example.</a><br> <a href="#javascript-example">5) Google Translate API Javascript Example.</a></p>
         <p>&nbsp;</p>
         <h3 id="api-key">1). How to Get Google Translate API Key.</h3>
         <p>To use the API you need get a Google API key;<br> a). Go to <a href="https://cloud.google.com/console" target="__blank">Cloud Console</a><br> b). Create a Project and select it.<br> c). On the left side bar, Go to “APIs &amp; auth” Tab =&gt; APIs Tab<br> d). Enable “Translate API”<br> e). Go to “Credentials” Tab<br> f). Click on “CREATE NEW KEY”<br> g). Create “Server Key” for PHP / Java / Ruby / Python. Create “Browser Key” for JavaScript.</p>
         <p><a href="http://cdn.hayageek.com/wp-content/uploads/2013/12/NewProject.png"><img class="alignnone size-thumbnail wp-image-957" alt="NewProject" src="http://cdn.hayageek.com/wp-content/uploads/2013/12/NewProject-150x150.png" width="150" height="150"></a> &nbsp; <a href="http://cdn.hayageek.com/wp-content/uploads/2013/12/Enable-Google-Translate-API.png"><img class="alignnone size-thumbnail wp-image-954" alt="Enable Google Translate API" src="http://cdn.hayageek.com/wp-content/uploads/2013/12/Enable-Google-Translate-API-150x150.png" width="150" height="150"></a> <a href="http://cdn.hayageek.com/wp-content/uploads/2013/12/Credentials.png"><img class="alignnone size-thumbnail wp-image-956" alt="Credentials" src="http://cdn.hayageek.com/wp-content/uploads/2013/12/Credentials-150x150.png" width="150" height="150"></a> <a href="http://cdn.hayageek.com/wp-content/uploads/2013/12/Create-New-Key.png"><img class="alignnone size-thumbnail wp-image-952" alt="Create New Key" src="http://cdn.hayageek.com/wp-content/uploads/2013/12/Create-New-Key-150x150.png" width="150" height="150"></a></p>
         <p>Check this page, how to get <a href="http://hayageek.com/google-api-key/" target="__blank">Google API key</a></p>
         <p>&nbsp;</p>
         <h3 id="api-list">2). List of Google Translate API</h3>
         <p>Google Provided REST API, so you can use them in any language.</p>
         <p>You can pass below parameters to Translate API.</p>
         <p>a).<strong>callback</strong>&nbsp; -&gt; this is for JavaScript. callback is invoked the when the translation is successful.<br> b).<strong>key</strong>&nbsp; -&gt; Google API key, without this you are not allowed to use this API.<br> c).<strong>source</strong> -&gt; Source language<br> d).<strong>target</strong>&nbsp; -&gt; Destination language.<br> e).<strong>prettyprint</strong>&nbsp; -&gt; If it set to true, response will be readable format. Default is <strong>true</strong><br> f).<strong>q</strong> -&gt; Text to be translated.<br> Below is the list of API.<br> <strong>2.1) translate</strong> – You can use this API&nbsp; to translate text from source language to destination language</p>
         <p>You need to send GET request in the below format.</p>
         <pre class="brush: html; gutter: true">https://www.googleapis.com/language/translate/v2?key={YOUR_API_KEY}&amp;source=en&amp;target=de&amp;q=Hello%20Ravi</pre>
         <p><strong>Response from API :</strong></p>
         <pre class="brush: javascript; gutter: true">{
 "data": {
  "translations": [
   {
    "translatedText": "Hallo Ravi"
   }
  ]
 }
}</pre>
         <p>You can request multiple translations in single request. You can do this by sending multiple ‘q’ parameters.<br> <strong> Sample GET request:</strong></p>
         <pre class="brush: html; gutter: true">https://www.googleapis.com/language/translate/v2?key={YOUR_API_KEY}&amp;source=en&amp;target=de&amp;q=Hello%20Ravi&amp;q=How%20are%20you&amp;q=I%20am%20fine</pre>
         <p><strong>Response format:</strong></p>
         <pre class="brush: javascript; gutter: true">{
 "data": {
  "translations": [
   {
    "translatedText": "Hallo Ravi"
   },
   {
    "translatedText": "Wie geht es Ihnen"
   },
   {
    "translatedText": "Mir geht es gut"
   }
  ]
 }
}</pre>
         <p><strong>2.2) detect</strong> – You can use this API to detect the language of given text.</p>
         <p><strong>Sample GET Request:</strong></p>
         <pre class="brush: html; gutter: true">https://www.googleapis.com/language/translate/v2/detect?key={YOUR_API_KEY}&amp;q=Wie%20geht%20es%20Ihnen</pre>
         <p><strong>Response:</strong></p>
         <pre class="brush: javascript; gutter: true">{
 "data": {
  "detections": [
   [
    {
     "language": "de",
     "isReliable": false,
     "confidence": 1.0
    }
   ]
  ]
 }
}</pre>
         <p><strong>2.3). languages</strong> – This API Lists the Source and Destination languages supported by the translate method</p>
         <p><strong>Sample GET Request:</strong></p>
         <pre class="brush: html; gutter: true">https://www.googleapis.com/language/translate/v2/languages?key={YOUR_API_KEY}&amp;target=de</pre>
         <p>&nbsp;</p>
         <p><strong>How to handle Errors ? :</strong></p>
         <p>If there is any error in the URL request, you will get the error in the below formats.<br> <strong>ERROR 1</strong>:&nbsp; Response when the key is invalid</p>
         <pre class="brush: javascript; gutter: true">{
 "error": {
  "errors": [
   {
    "domain": "usageLimits",
    "reason": "keyInvalid",
    "message": "Bad Request"
   }
  ],
  "code": 400,
  "message": "Bad Request"
 }
}</pre>
         <p><strong>ERROR 2</strong>: Response when target parameter is missing.</p>
         <pre class="brush: javascript; gutter: true">{
 "error": {
  "errors": [
   {
    "domain": "global",
    "reason": "required",
    "message": "Required parameter: target",
    "locationType": "parameter",
    "location": "target"
   }
  ],
  "code": 400,
  "message": "Required parameter: target"
 }
}</pre>
         <p>&nbsp;</p>
         <h3 id="php-example">3) Google Translate API PHP Example.</h3>
         <p>To use Google Translate API in PHP, you need to get Google API <strong>“Server Key”</strong>. All the translate API are called using GET requests. So we can use <strong>file_get_contents()</strong> or <a href="http://php.net/curl">PHP CURL</a> Library.To encode the parameter, <strong>rawurlencode()</strong> function is used. To decode JSON text as object, <strong>json_decode()</strong> function is used.</p>
         <p><a title="Google Translate API in PHP" href="http://hayageek.com/examples/google/translate/translate.php?text=I am Ravi. How are you" target="__blank"><img alt="Google Translate API in PHP" src="https://s3.amazonaws.com/hayageek/images/demo.jpg"></a><br> I have given two examples, you can use any of these two examples.</p>
         <p><strong>Using CURL library:</strong></p>
         <pre class="brush: php; gutter: true">&lt;?php
$api_key = 'PUT_YOUR_SERVER_KEY_HERE';
$text = 'How are you';
$source="en";
$target="fr";

$obj = translate($api_key,$text,$target,$source);
if($obj != null)
{
	if(isset($obj['error']))
	{
		echo "Error is : ".$obj['error']['message'];
	}
	else
	{
		echo "Translsated Text: ".$obj['data']['translations'][0]['translatedText']."n";
		if(isset($obj['data']['translations'][0]['detectedSourceLanguage'])) //this is set if only source is not available.
			echo "Detecte Source Languge : ".$obj['data']['translations'][0]['detectedSourceLanguage']."n";		
	}
}
else
	echo "UNKNOW ERROR";

function translate($api_key,$text,$target,$source=false)
{
	$url = 'https://www.googleapis.com/language/translate/v2?key=' . $api_key . '&amp;q=' . rawurlencode($text);
	$url .= '&amp;target='.$target;
	if($source)
	 $url .= '&amp;source='.$source;

	$ch = curl_init($url);
	curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
	$response = curl_exec($ch);                 
	curl_close($ch);

	$obj =json_decode($response,true); //true converts stdClass to associative array.
	return $obj;
}	

?&gt;</pre>
         <p><strong>PHP Example Using file_get_contents()</strong></p>
         <pre class="brush: bash; gutter: true">&lt;?php
$api_key = 'PUT_YOUR_SERVER_KEY_HERE';
$text = 'How are you';
$source="en";
$target="fr";

$url = 'https://www.googleapis.com/language/translate/v2?key=' . $api_key . '&amp;q=' . rawurlencode($text);
$url .= '&amp;target='.$target;

         <h3 id="java-example">4) Google Translate API Java Example.</h3>
         <p>To use Translate API in Java, you need to get Google API <strong>“Server Key”</strong>.&nbsp;&nbsp; <a href="https://code.google.com/p/google-gson/" target="__blank">GSON</a> Library is used to parse JSON text.</p>
         <p>You can use the below Java code to translate text.</p>
         <pre class="brush: java; gutter: true">import com.google.gson.JsonElement;
import com.google.gson.JsonObject;
import com.google.gson.JsonParser;
import com.google.gson.JsonSyntaxException;
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStream;
import java.io.InputStreamReader;
import java.net.URL;
import java.net.URLEncoder;
import javax.net.ssl.HttpsURLConnection;
/**
 *
 * @author Ravishanker Kusuma
 */
public class GoogleTranslate {

    private String key;

    public GoogleTranslate(String apiKey) {
        key = apiKey;
    }

    String translte(String text, String from, String to) {
        StringBuilder result = new StringBuilder();
        try {
            String encodedText = URLEncoder.encode(text, "UTF-8");
            String urlStr = "https://www.googleapis.com/language/translate/v2?key=" + key + "&amp;q=" + encodedText + "&amp;target=" + to + "&amp;source=" + from;

            URL url = new URL(urlStr);

            HttpsURLConnection conn = (HttpsURLConnection) url.openConnection();
            InputStream stream;
            if (conn.getResponseCode() == 200) //success
            {
                stream = conn.getInputStream();
            } else
                stream = conn.getErrorStream();

            BufferedReader reader = new BufferedReader(new InputStreamReader(stream));
            String line;
            while ((line = reader.readLine()) != null) {
                result.append(line);
            }

            JsonParser parser = new JsonParser();

            JsonElement element = parser.parse(result.toString());

            if (element.isJsonObject()) {
                JsonObject obj = element.getAsJsonObject();
                if (obj.get("error") == null) {
                    String translatedText = obj.get("data").getAsJsonObject().
                    get("translations").getAsJsonArray().
                    get(0).getAsJsonObject().
                    get("translatedText").getAsString();
                    return translatedText;

                }
            }

            if (conn.getResponseCode() != 200) {
                System.err.println(result);
            }

        } catch (IOException | JsonSyntaxException ex) {
            System.err.println(ex.getMessage());
        }

        return null;
    }
    public static void main(String[] args) {

        GoogleTranslate translator = new GoogleTranslate("PUT_YOUR_KEY_HERE");
        String text = translator.translte("How are you. I am fine", "en", "fr");
        System.out.println(text);
    }
}</pre>
