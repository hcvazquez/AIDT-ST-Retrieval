         <section class="Abstract" id="Abs1" tabindex="-1" lang="en">
          <h2 class="Heading">Abstract</h2>
          <p class="Para">We study fixed parameter algorithms for three problems: Kemeny rank aggregation, feedback arc set tournament, and betweenness tournament. For Kemeny rank aggregation we give an algorithm with runtime <span class="InlineEquation" id="IEq1">\(O^*(2^{O(\sqrt{OPT})})\)</span>, where <em class="EmphasisTypeItalic ">n</em> is the number of candidates, <span class="InlineEquation" id="IEq2">\(OPT \le \binom{n}{2}\)</span> is the cost of the optimal ranking, and <em class="EmphasisTypeItalic ">O</em><sup>*</sup>(·) hides polynomial factors. This is a dramatic improvement on the previously best known runtime of <em class="EmphasisTypeItalic ">O</em><sup>*</sup>(2<sup><em class="EmphasisTypeItalic ">O</em>(<em class="EmphasisTypeItalic ">OPT</em>)</sup>). For feedback arc set tournament we give an algorithm with runtime <span class="InlineEquation" id="IEq3">\(O^*(2^{O(\sqrt{OPT})})\)</span>, an improvement on the previously best known <span class="InlineEquation" id="IEq4">\(O^*(OPT^{O(\sqrt{OPT})})\)</span> [4]. For betweenness tournament we give an algorithm with runtime <span class="InlineEquation" id="IEq5">\(O^*(2^{O(\sqrt{OPT/n})})\)</span>, where <em class="EmphasisTypeItalic ">n</em> is the number of vertices and <span class="InlineEquation" id="IEq6">\(OPT \le \binom{n}{3}\)</span> is the optimal cost. This improves on the previously known <span class="InlineEquation" id="IEq7">\(O^*(OPT^{O(OPT^{1/3})})\)</span> [28], especially when <em class="EmphasisTypeItalic ">OPT</em> is small. Unusually we can solve instances with <em class="EmphasisTypeItalic ">OPT</em> as large as <em class="EmphasisTypeItalic ">n</em> (log<em class="EmphasisTypeItalic ">n</em>)<sup>2</sup> in polynomial time!</p>
         </section>
