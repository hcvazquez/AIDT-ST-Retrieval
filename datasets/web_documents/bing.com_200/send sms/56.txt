         <h2 class="h4 title-bold ">Installation Instructions</h2> 
         <h3 class="h5 title-bold ">Quick Installation</h3> 
         <p>The easiest way to install the TextMagic Java wrapper is from Maven. You can add the following dependency to your existing project, specifying the latest version in the version tag:</p> 
         <pre class="code brush:php">
          <dependency>
    
           <groupid>
            com.textmagic.sdk
           </groupid>
    
           <artifactid>
            textmagic-java-sdk
           </artifactid>
    
           <version>
            1.2.0
           </version>

          </dependency>
</pre> 
         <h3 class="h5 title-bold ">Manual Installation</h3> 
         <p>You can also install the TextMagic Java wrapper from the&nbsp;<a href="https://github.com/textmagic/textmagic-rest-java" target="_blank">GitHub repository</a> using git. Run the following commands:</p> 
         <pre class="code brush:php">git clone git://github.com/textmagic/textmagic-rest-java.git
cd textmagic-rest-java
mvn install
</pre>
