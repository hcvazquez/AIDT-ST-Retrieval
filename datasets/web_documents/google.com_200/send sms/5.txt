            <!-- start snippet smsblaster web --> <h3 class="TitleFont2">Node.js&nbsp;&nbsp;Interface for aspsms.com</h3> <p> Node.js is a server-side software system designed for writing scalable Internet applications, notably web servers.[1] Programs are written on the server side in JavaScript, using event-driven, asynchronous I/O to minimize overhead and maximize scalability. <a href="http://en.wikipedia.org/wiki/Node.js" target="_Blank">Node.js on Wikipedia</a></p> <h4>mod-aspsms</h4> <p>SMS interface for ASPSMS by tuxle:<br> <i>"I released this code because I believe<br>everyone should be able to send texts with node."</i> <br> <a href="https://npmjs.org/package/mod-aspsms">https://npmjs.org/package/mod-aspsms</a> <a href="https://github.com/dns2utf8/mod-aspsms" target="_Blank"><img src="/img/solutions/indielab/github.jpg" align="right" border="0"></a> </p><h4>Project on Github</h4> <p><a href="https://github.com/dns2utf8/mod-aspsms">https://github.com/dns2utf8/mod-aspsms</a></p> <h4>Example</h4> <pre style="font-size:10px;background:#efefef;padding:10px">
var config = {
  "disable": true,
  "userkey": "YOUR_USERKEY",
  "password": "YOUR_PASSWORD",
  "originator": "YOUR_NAME",
  // Optional:
  "deliveryNotificationUrl": "http://example.com/success",
  "nonDeliveryNotificationUrl": "http://example.com/failure"
};
