         <h1><a href="#onsen-ui---cross-platform-html5-hybrid-app-framework" aria-hidden="true" class="anchor" id="user-content-onsen-ui---cross-platform-html5-hybrid-app-framework">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a><a href="https://onsen.io/">Onsen UI</a> - Cross-Platform HTML5 Hybrid App Framework</h1> 
         <p>Onsen UI is a library of UI components to help you create beautiful hybrid and mobile web apps for both Android and iOS using Javascript.</p> 
         <p align="center"><a href="https://camo.githubusercontent.com/7645060259afb4904e589e5ad9e0cbb94182859f/68747470733a2f2f6f6e73656e75692e6769746875622e696f2f6172742f73686f77636173652f73686f77636173652d6f6e73656e75692e676966" target="_blank"><img height="500" src="https://camo.githubusercontent.com/7645060259afb4904e589e5ad9e0cbb94182859f/68747470733a2f2f6f6e73656e75692e6769746875622e696f2f6172742f73686f77636173652f73686f77636173652d6f6e73656e75692e676966" alt="Onsen UI Todo App" data-canonical-src="https://onsenui.github.io/art/showcase/showcase-onsenui.gif" style="max-width:100%;"></a></p> 
         <p>It is <strong>open source</strong> and designed to deliver native look and feel. Apps are made out of simple HTML tags, thus providing a gentle learning curve.</p> 
         <p>The core library is written in <strong>pure Javascript</strong> (on top of <a href="http://webcomponents.org/">Web Components</a>) and is <strong>framework agnostic</strong>, which means you can use it with your favorite framework and its tools. Currently, we provide some extra binding packages in order to <strong>tailor Onsen UI's API</strong> for specific frameworks:</p>

         <p>Some other frameworks are supported by <strong>community</strong> packages (i.e. not tested or implemented by the core team): <a href="https://www.npmjs.com/package/aurelia-onsenui">Aurelia</a>, <a href="https://www.npmjs.com/package/ember-onsenui">EmberJS</a>.</p> 
         <p>Both <strong>iOS flat and Android's Material design</strong> are included. The components are optionally auto-styled based on the platform, which makes it possible to support both iOS and Android with the <strong>same source code</strong>.</p> 
         <h2><a href="#getting-started" aria-hidden="true" class="anchor" id="user-content-getting-started">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>Getting started</h2> 
         <p>We have several resources to help you get started creating hybrid apps with Onsen UI:</p> 
         <ul> 
          <li><strong>The official docs</strong>: we provide guides and references for the <a href="https://onsen.io/v2/docs/js.html">core library (vanilla JS or jQuery)</a> as well as the <a href="https://onsen.io/v2/docs/guide/vue/index.html">Vue</a>, <a href="https://onsen.io/v2/docs/guide/react/index.html">React</a>, <a href="https://onsen.io/v2/docs/guide/angular2/">Angular2</a> and <a href="https://onsen.io/v2/docs/guide/angular1/">AngularJS 1.x</a> components.</li> 
          <li><strong>Components overview</strong>: a <a href="https://onsen.io/v2/docs/css.html">list of included CSS components</a> in both flat and Material Design. Note that these components are just pure and performant CSS without JavaScript behavior. Some extra details (such as dragging or ripple effect) are added by Onsen UI custom elements.</li> 
          <li><strong>Playground</strong>: an <a href="https://onsen.io/tutorial/">interactive Onsen UI tutorial</a> is also available where you can learn how to use the library and play around with the components.</li> 
          <li><strong>Blog</strong>: there are lots of great tutorials and guides published in our <a href="https://onsen.io/blog/categories/tutorial.html">official Onsen UI blog</a> and we are adding new content regularly.</li> 
          <li><strong>Support</strong>: if you are having trouble using some component the best place to get help is the <a href="https://community.onsen.io/">Onsen UI Forum</a> or <a href="https://gitter.im/OnsenUI/OnsenUI">Gitter chat</a>. We are also available to answer short questions on Twitter at <a href="https://twitter.com/Onsen_UI">@Onsen_UI</a>.</li> 
         </ul> 
         <h2><a href="#onsen-ui-ecosystem" aria-hidden="true" class="anchor" id="user-content-onsen-ui-ecosystem">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>Onsen UI ecosystem</h2> 
         <p>Because sometimes a UI framework may not be enough to make hybrid app development easily, Onsen UI comes with a <strong>complete ecosystem</strong> of well integrated tools. Meet <a href="https://monaca.io/"><strong>Monaca</strong></a>.</p> 
         <p align="center"><a href="https://monaca.io"><img width="300" src="https://camo.githubusercontent.com/edba328da6d751c3bcc28d130348441d65ed56ec/68747470733a2f2f6f6e73656e75692e6769746875622e696f2f6172742f6c6f676f732f6d6f6e6163612d6c6f676f2d322e706e67" data-canonical-src="https://onsenui.github.io/art/logos/monaca-logo-2.png" style="max-width:100%;"></a></p> 
         <p>Developed by the same team, <strong>Monaca Platform</strong> is a toolkit that makes hybrid mobile app development with <strong>PhoneGap / Cordova</strong> simple and easy: Onsen UI Cordova-like templates, debugging suite, push notifications, remote build, back-end solutions, encryption, version control, continuous integration and more. Furthermore, it provides multiple development environments with everything already configured and ready to go:</p> 
         <p align="center"> <a href="https://monaca.io/cloud.html"><strong>Cloud IDE</strong></a> - <a href="https://monaca.io/cli.html"><strong>Command Line Interface</strong></a> - <a href="https://monaca.io/localkit.html"><strong>Localkit GUI</strong></a> </p> 
         <p>Example with <strong>CLI</strong>:</p> 
         <pre><code>$ [sudo] npm -g install monaca
$ monaca create helloworld # And choose the starter template
$ monaca preview # Preview on the browser
$ monaca debug # Preview on a real device
$ monaca remote-build # Production build on the cloud
</code></pre> 
         <p>See the <a href="http://onsen.io/v2/guide/">Onsen UI Getting Started Page</a> for more information.</p>
