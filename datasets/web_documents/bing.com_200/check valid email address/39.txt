     <h1 class="text-center singlepost_title_singleview ">List of 25+ Email Validation APIs</h1> 
     <h2 class="text-center">by <span class="singlepost_meta_highlight"><a href="http://blog.mashape.com?s=&amp;author_name=chrisismael">Chris Ismael</a></span> on December 4, 2013</h2>

    <p><a href="http://i1.wp.com/blog.mashape.com/wp-content/uploads/2013/12/tumblr_inline_mxaw93yEjg1rghs7f.jpg"><img class=" size-medium wp-image-1354 alignleft" src="http://i1.wp.com/blog.mashape.com/wp-content/uploads/2013/12/tumblr_inline_mxaw93yEjg1rghs7f.jpg?resize=300%2C168" alt="Email Validation API" srcset="http://i1.wp.com/blog.mashape.com/wp-content/uploads/2013/12/tumblr_inline_mxaw93yEjg1rghs7f.jpg?resize=300%2C168 300w, http://blog.mashape.com/wp-content/uploads/2013/12/tumblr_inline_mxaw93yEjg1rghs7f.jpg 400w" sizes="(max-width: 300px) 100vw, 300px" data-recalc-dims="1"></a></p>

    <p>Here’s a list of useful APIs to help you determine if that email address is real ?</p>

      <tr> 
       <th scope="row">1</th> 
       <td>Verify Email</td> 
       <td class="asd">simple api that checks if an email address if valid and exists. It checks the format of the address then tries to contact the server to make sure the address exists.<br> <a href="https://www.mashape.com/fetch/verify-email#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">2</th> 
       <td>Social Media Email Search</td> 
       <td class="asd">This API allows searching the popular social networks (facebook, linkedin, twitter, tumblr, myspace) for an email address. If found, the site ID will be returned.<br> <a href="https://www.mashape.com/vondrakk/social-media-email-search#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">3</th> 
       <td>Fluxifi Tools</td> 
       <td class="asd">Miscellaneous tools for getting page rank, page speed, validating domains &amp; emails<br> <a href="https://www.mashape.com/fluxifi/fluxifi-tools#!endpoint-Validate-Email"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">4</th> 
       <td>Email Validator</td> 
       <td class="asd">Checks for fake DNS as well as uses regex functions to check the email for the right length and accepted characters.<br> <a href="https://www.mashape.com/pozzad/email-validator-1#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">5</th> 
       <td>Email Search by EmailSherlock</td> 
       <td class="asd">A reverse email search conducted at EmailSherlock.com can help determine the identity of the owner of an unknown address that shows up in your inbox.<br> <a href="https://www.mashape.com/infopay/email-search-by-emailsherlock#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">6</th> 
       <td>FullContact Person</td> 
       <td class="asd">The FullContact Person API takes a piece of contact information, such as an email address, and adds information such as name, job title, and social profile data from more than 120 different social networks including Facebook, Twitter, LinkedIn, and more.<br> <a href="https://www.mashape.com/fullcontact/fullcontact-person#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">7</th> 
       <td>Block Disposable Email</td> 
       <td class="asd">The bdea.cc api helps to detect domains of disposable email address services such as mailinator.com, spamgourmet. In other words: if you need to keep your userbase clean you should avoid throw-away, one-time, instant, temporary email addresses.<br> <a href="https://www.mashape.com/bdea-cc/block-disposable-email#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">8</th> 
       <td>Kickbox.io</td> 
       <td class="asd">Kickbox.io determines if an email address is not only valid, but associated with an actual user.<br> <a href="https://www.mashape.com/kickboxio/email-verification#!documentation"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr> 
      <tr> 
       <th scope="row">9</th> 
       <td>QuickEmailVerficiation</td> 
       <td class="asd">This API help you improve your mail deliverability rates and protect your IP reputation by checking email addresses based on syntax, mx records, disposable and wildcard emails and deep smtp checks!<br> <a href="https://www.mashape.com/quickemailverification/quick-email-verification"><img src="https://d1g84eaw0qjo7s.cloudfront.net/images/badges/badge-icon-light-9e8eba63.png" alt="Mashape" width="143" height="38"></a></td> 
      </tr>
