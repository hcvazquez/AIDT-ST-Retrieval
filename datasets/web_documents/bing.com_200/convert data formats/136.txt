             <div class="cd">
              <p>This object can export HTML table data to Excel format.<br> <br> It takes a given HTML table on the current page and extract their contents to put on a template in the Excel XML format. A data URL is returned with the whole table data in the Excel format.<br> <br> If the browser is Internet Explorer it creates an ActiveX Excel.Application object to open the Excel program.
