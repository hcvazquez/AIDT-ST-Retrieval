       <div id="html2"> 
        <p> Lazy loading slider image will not load until the slider starts showing its previous image. It is preloaded in the background and be cached by the browser once loaded. This allows the slider to start right away on page load without waiting for all images to be fully loaded. </p> 
        <h3>How do I enable Lazy Loading Image?</h3> For each image (usually it is not necessary for the first slide image),
        <br>instead of: 
        <code><del>&lt;img src="image.jpg" alt="image desc" /&gt;</del></code> You can: 
        <code>&lt;a class="lazyImage" href="image.jpg" title="image desc"&gt; . . . &lt;/a&gt;</code> 
        <p> If your image has already been wrapped with a link, it is illegal to nest a link with another link: <span class="cn" style="text-decoration:line-through;">&lt;a&gt;&lt;a&gt;&lt;/a&gt;&lt;/a&gt;</span>. Fortunately there is a solution for this scenario as shown below: </p> 
        <code>&lt;a href="(link adress)"&gt; &lt;b data-src="image.jpg" data-alt="image desc"&gt; . . . &lt;/b&gt; &lt;/a&gt;</code> 
        <p>Please refer to the download package where you can find the souce HTML code of the lazy loaded images. For example, the 2nd, 3rd and 4th slide image in demo3.html are all lazy loading images.</p> 
       </div> 
      </div> 
      <br> 
      <ul> 
       <li> <strong>How do I add caption or text on top of a slide image?</strong> <p> Caption is referred to the text content shown with the image. Caption can be defined through each slide image's <span class="cn red">alt</span> attribute: <code>&lt;img src="..." <span class="red">alt="..."</span> /&gt;</code> </p> <p> If the image is marked up in the lazy loaded image format, its caption can be defined by the <span class="cn red">title</span> or <span class="cn red">data-alt</span> attribute: </p> <code>&lt;a class="lazyImage" href="..." <span class="red">title="..."</span>&gt; ... &lt;/a&gt; <span class="green">or:</span> &lt;b data-src="..." <span class="red">data-alt="..."</span>&gt;...&lt;/b&gt;</code> <p><span class="tooltip" onmouseover="tooltip.pop(this, '#tipHtmlCap');">How do I add HTML caption to a slide image?</span></p> </li> 
       <li> <strong>How to Link an Image?</strong> <p>Just wrap the image with the &lt;a&gt; tag:</p> <code>&lt;a href="(url)"&gt;&lt;img src=... /&gt;&lt;/a&gt;</code> </li> 
      </ul>
