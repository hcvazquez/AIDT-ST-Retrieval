              <h2 class="post-title entry-title"> <a href="http://www.code2learn.com/2013/02/quick-sort-algorithm-tutorial.html">QuickSort Algorithm Tutorial</a> </h2> 
             </header> 
             <div class="post-header-line-1"></div> 
             <div class="post-body entry-content"> 
              <div style="float:right;padding:10px 10px 10px 10px; "> 
               <script type="text/javascript">

              <div dir="ltr" style="text-align: left;" trbidi="on"> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">We have already done <a href="http://www.code2learn.com/2011/07/merge-sort-using-java.html">tutorial on Merge Sort</a> and <a href="http://www.code2learn.com/2011/09/heapsort-array-based-implementation-in.html">a tutorial on Heap Sort (Array Based)</a> with both having a time complexity of O(n*log n). Here is a</span>
                <span style="font-family: Verdana, sans-serif;">nother algorithm which has a time complexity of O(n*log n) and it's called QuickSort.</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">QuickSort as we all know has a similar approach to Merge Sort i.e. it uses Divide-and-Conquer recursive algorithm to sort the values. The difference being is it's an </span>
                <b style="font-family: Verdana, sans-serif;">in-place </b>
                <span style="font-family: Verdana, sans-serif;">sorting algorithm.</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">Basically an in-place algorithm is one which transforms the input using a data structure with a small, constant amount of extra storage space.</span>
               </div> 
               <a name="more"></a>
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <span style="font-family: Verdana, sans-serif;"></span>
               <br> 
               <h1 style="text-align: justify;"> <span style="font-family: Verdana, sans-serif;">Basic Idea of QuickSort</span></h1> 
               <br> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">1. </b>
                <span style="font-family: Verdana, sans-serif;">Pick an element in the array as the pivot element.</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><b>2. </b>Make a pass to the array, called the PARTITION step, which rearranges the elements in the array:</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">&nbsp; &nbsp;<b>a.</b>&nbsp;The pivot element is in the proper place</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><b>&nbsp; &nbsp; b. </b>The elements less than pivot element are on the left of it</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><b>&nbsp; &nbsp; c.&nbsp;</b></span>
                <span style="font-family: Verdana, sans-serif;">The elements greater than pivot element are on the right of it</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><b>3. </b>Recursively apply the above process to the left and right part of the pivot element.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span> 
                <span style="font-family: Verdana, sans-serif;"></span>
                <br> 
                <table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;">
                 <tbody>

                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <h1 style="text-align: justify;"> <span style="font-family: Verdana, sans-serif;">Algorithm</span></h1> 
               <br> 
               <span style="font-family: Verdana, sans-serif;"></span>
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><b>Step 1. </b>&nbsp;Choosing the Pivot Element</span>
               </div> 
               <span style="font-family: Verdana, sans-serif;"> </span> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">Choosing the pivot element can determine the complexity of the algorithm i.e. whether it will be n*logn or quadratic time:</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">a.</b>
                <span style="font-family: Verdana, sans-serif;"> Normally we choose the first, last or the middle element as pivot. This can harm us badly as the pivot might end up to be the smallest or the largest element, thus leaving one of the partitions empty.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">b. </b>
                <span style="font-family: Verdana, sans-serif;">We should choose the Median of the first, last and middle elements. If there are N elements, then the ceiling of N/2 is taken as the pivot element.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">Example:</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">8, 3, 25, 6, 10, 17, 1, 2, 18, 5</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">first element: 8</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">middle element: 10</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">last element: 5</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">Therefore the median on [8,10,5] is 8.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">Step 2. </b>
                <span style="font-family: Verdana, sans-serif;">&nbsp;Partitioning&nbsp;</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">a. </b>
                <span style="font-family: Verdana, sans-serif;">First thing is to get the pivot out of the way and swapping it with the last number.&nbsp;</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">Example: (shown using the above array elements)</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">5,</b>
                <span style="font-family: Verdana, sans-serif;">&nbsp;3, 25, 6, 10, 17, 1, 2, 18, &nbsp;</span>
                <b style="font-family: Verdana, sans-serif;">8</b>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif; font-weight: bold;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">b. </b>
                <span style="font-family: Verdana, sans-serif;">Now we want the elements greater than pivot to be on the right side of it and&nbsp;similarly&nbsp;the elements less than pivot to be on the left side of it.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">For this we define 2 pointers, namely i and j. i being at the first index and j being and the last index of the array.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">&nbsp; &nbsp;* While </span>
                <b style="font-family: Verdana, sans-serif;">i</b>
                <span style="font-family: Verdana, sans-serif;">&nbsp;is less than </span>
                <b style="font-family: Verdana, sans-serif;">j </b>
                <span style="font-family: Verdana, sans-serif;">&nbsp;we keep in incrementing </span>
                <b style="font-family: Verdana, sans-serif;">i</b>
                <span style="font-family: Verdana, sans-serif;">&nbsp;until we find an element &nbsp; &nbsp;greater than pivot.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">&nbsp; &nbsp;* Similarly, while <b>j</b>&nbsp;is greater then <b>i</b>&nbsp;keep decrementing <b>j </b>&nbsp;until we find an element less than pivot.</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">&nbsp; &nbsp;* After both <b>i </b>and <b>j </b>stop we swap the elements at the indexes of <b>i </b>and <b>j</b>&nbsp;respectively.</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">c. </b>
                <span style="font-family: Verdana, sans-serif;">Restoring the pivot</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">When the above steps are done correctly we will get this as our output:</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;">[5, 3, 2, 6, 1] [8] [10, 25, 18, 17]</span>
               </div> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <div style="text-align: justify;"> 
                <b style="font-family: Verdana, sans-serif;">Step 3. </b>
                <span style="font-family: Verdana, sans-serif;">Recursively Sort the left and right part of the pivot.</span>
               </div> 
               <br> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span> 
                <span style="font-family: Verdana, sans-serif;"></span>
               </div> 
               <h1 style="text-align: justify;"> <span style="font-family: Verdana, sans-serif;">Java Code</span></h1> 
               <div style="text-align: justify;"> 
                <span style="font-family: Verdana, sans-serif;"><br></span>
               </div> 
               <pre style="background: #211e1e; color: #dadada;">public class QuickSortAlgorithm {

    protected static int A[];

    public void sort(int[] A) {
        <span style="color: #6969fa; font-weight: 700;">if</span> (A <span style="color: #47b8d6;">==</span> <span style="color: #de8e30; font-weight: 700;">null</span> <span style="color: #47b8d6;">||</span> A<span style="color: #47b8d6;">.</span>length <span style="color: #47b8d6;">==</span> <span style="color: #cccccc;">0</span>)
            <span style="color: #6969fa; font-weight: 700;">return</span>;
        quicksort(A, <span style="color: #cccccc;">0</span>, A<span style="color: #47b8d6;">.</span>length <span style="color: #47b8d6;">-</span> <span style="color: #cccccc;">1</span>);
    }

    public void quicksort(int[] A, int <span style="color: #c1c144;">left</span>, int <span style="color: #c1c144;">right</span>) {
        int pivot <span style="color: #47b8d6;">=</span> A[left <span style="color: #47b8d6;">+</span> (right <span style="color: #47b8d6;">-</span> left) <span style="color: #47b8d6;">/</span> <span style="color: #cccccc;">2</span>];
        int i <span style="color: #47b8d6;">=</span> left;
        int j <span style="color: #47b8d6;">=</span> right;
        <span style="color: #6969fa; font-weight: 700;">while</span> (i <span style="color: #47b8d6;">&lt;=</span> j) {
            <span style="color: #6969fa; font-weight: 700;">while</span> (A[i] <span style="color: #47b8d6;">&lt;</span> pivot) {
                i<span style="color: #47b8d6;">++</span>;
            }
            <span style="color: #6969fa; font-weight: 700;">while</span> (A[j] <span style="color: #47b8d6;">&gt;</span> pivot) {
                j<span style="color: #47b8d6;">--</span>;
            }
            <span style="color: #6969fa; font-weight: 700;">if</span> (i <span style="color: #47b8d6;">&lt;=</span> j) {
                exchange(i, j);
                i<span style="color: #47b8d6;">++</span>;
                j<span style="color: #47b8d6;">--</span>;
            }
        }
        
        <span style="color: #6969fa; font-weight: 700;">if</span>(left <span style="color: #47b8d6;">&lt;</span> j)
            quicksort(A,left,j);
        <span style="color: #6969fa; font-weight: 700;">if</span>(i <span style="color: #47b8d6;">&lt;</span> right)
            quicksort(A,i,right);
    }

    public void exchange(int <span style="color: #c1c144;">i</span>, int <span style="color: #c1c144;">j</span>){
        int temp<span style="color: #47b8d6;">=</span>A[i];
        A[i]<span style="color: #47b8d6;">=</span>A[j];
        A[j]<span style="color: #47b8d6;">=</span>temp;
    }
    
    public String toString() {
        String s <span style="color: #47b8d6;">=</span> <span style="color: #ad9361;">""</span>;
        s <span style="color: #47b8d6;">+=</span> <span style="color: #ad9361;">"["</span> <span style="color: #47b8d6;">+</span> A[<span style="color: #cccccc;">0</span>];
        <span style="color: #6969fa; font-weight: 700;">for</span> (int i <span style="color: #47b8d6;">=</span> <span style="color: #cccccc;">1</span>; i <span style="color: #47b8d6;">&lt;</span> A<span style="color: #47b8d6;">.</span>length; i<span style="color: #47b8d6;">++</span>) {
            s <span style="color: #47b8d6;">+=</span> <span style="color: #ad9361;">", "</span> <span style="color: #47b8d6;">+</span> A[i];
        }
        s <span style="color: #47b8d6;">+=</span> <span style="color: #ad9361;">"]"</span>;
        <span style="color: #6969fa; font-weight: 700;">return</span> s;
    }

}

                <h1 style="text-align: justify;"> <span style="font-family: Verdana, sans-serif;">Complexity of QuickSort</span></h1> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b>Worst Case : O(N^2)</b></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b><br></b></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">This happens when the pivot is the smallest or the largest element. Then one of the partition is empty and we repeat the recursion for N-1 elements</span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b>Best Case: O(NlogN)</b></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b><br></b></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">This is when the pivot is the median of the array and the left and right part are the of the same size.</span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">There are logN partitions and to compare we do N comparisions</span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
                 </div> 
                </div> 
                <div> 
                 <span style="font-family: Verdana, sans-serif;"></span>
                 <br> 
                 <h1 style="text-align: justify;"> <span style="font-family: Verdana, sans-serif;">Conclusion :</span></h1> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b><br></b></span>
                 </div> 
                </div> 
                <div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b>Advantages</b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b><br></b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">* One of the fastest algorithm on average.</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">* Doesn't need additional memory i.e. it's an in-place sorting algorithm.</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b>Disadvantages</b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><b><br></b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">Worst Case complexity is O(N^2).</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif; font-size: large;"><b>Comparision to Merge Sort</b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif; font-size: large;"><b><br></b></span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">* Merge Sort guarantee O(NlogN) time, however it requires additional memory with size N.</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">* Quick Sort doesn't require additional memory but the running time is not guaranteed.</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;">* Usually Merge Sort is not used for Memory Sorting. Its used only for external memory sorting.</span>
                 </div> 
                 <div style="text-align: justify;"> 
                  <span style="font-family: Verdana, sans-serif;"><br></span>
