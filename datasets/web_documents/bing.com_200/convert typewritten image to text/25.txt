   <div class="sect1"> 
    <h2 id="itext"><a class="link" href="#itext">1. Overview</a></h2> 
    <div class="sectionbody"> 
     <div class="paragraph"> 
      <p>iText is a Java library originally created by Bruno Lowagie which allows to create PDF, read PDF and manipulate them. The following tutorial will show how to create PDF files with iText. This tutorial assumes that you have basis <a href="http://www.vogella.com/tutorials/JavaIntroduction/article.html">Java</a> and <a href="http://www.vogella.com/tutorials/Eclipse/article.html">Eclipse</a> knowledge.</p> 
     </div> 
     <div class="paragraph"> 
      <p>iText has a hierarchical structure. The smallest text unit is a "Chunk" which is a String with a pre-defined font. A "Phrase" combines several Chunks and allows to define line spacing. "Paragraph" is a subclass of "Phrase" and allows to define more layout attributes, e.g. margins. The class "Anchor" is a subclass of "Paragraph" and serves as the basis for hyperlinks in the generated PDF.</p> 
     </div> 
    </div> 
   </div>

    <h2 id="installation"><a class="link" href="#installation">2. Installation</a></h2>

     </div>

    <h2 id="createpdf"><a class="link" href="#createpdf">3. Create a PDF</a></h2> 
    <div class="sectionbody"> 
     <div class="paragraph"> 
      <p>Create a new Java project "de.vogella.itext.write" with the package "de.vogella.itext.write". Create a folder "lib" and put the iText library (jar file) into this folder. <a href="http://www.vogella.com/tutorials/Eclipse/article.html#classpath">Add the jar to your classpath</a>.</p> 
     </div> 
     <div class="paragraph">

    <h2 id="readpdf"><a class="link" href="#readpdf">5. Read an existing pdf</a></h2> 
    <div class="sectionbody"> 
     <div class="paragraph"> 
      <p>iText allows to read existing pdf’s and include them into your own pdf. The following example will create page 2 of the previous example and create a new document with this page.</p> 
     </div> 
     <div class="paragraph"> 
      <p>Create a new Java project "de.vogella.itext.readpdf" with the package "de.vogella.itext.read". Create a folder "lib" and put the iText library (jar file) into this folder. <a href="http://www.vogella.com/tutorials/Eclipse/article.html#classpath">Add the jar to your classpath</a>. Create the following class "ReadAndUsePdf.java".</p> 
     </div> 
     <div class="listingblock"> 
      <div class="content"> 
       <pre class="CodeRay highlight"><code data-lang="java"><span class="keyword">package</span> <span class="namespace">de.vogella.itext.read</span>;

<span class="keyword">import</span> <span class="include">java.io.IOException</span>;

<span class="keyword">import</span> <span class="include">com.itextpdf.text.Document</span>;
<span class="keyword">import</span> <span class="include">com.itextpdf.text.DocumentException</span>;
<span class="keyword">import</span> <span class="include">com.itextpdf.text.Image</span>;
<span class="keyword">import</span> <span class="include">com.itextpdf.text.pdf.PdfImportedPage</span>;
<span class="keyword">import</span> <span class="include">com.itextpdf.text.pdf.PdfReader</span>;
<span class="keyword">import</span> <span class="include">com.itextpdf.text.pdf.PdfWriter</span>;

<span class="directive">public</span> <span class="type">class</span> <span class="class">ReadAndUsePdf</span> {
    <span class="directive">private</span> <span class="directive">static</span> <span class="predefined-type">String</span> INPUTFILE = <span class="string"><span class="delimiter">"</span><span class="content">c:/temp/FirstPdf.pdf</span><span class="delimiter">"</span></span>;
    <span class="directive">private</span> <span class="directive">static</span> <span class="predefined-type">String</span> OUTPUTFILE = <span class="string"><span class="delimiter">"</span><span class="content">c:/temp/ReadPdf.pdf</span><span class="delimiter">"</span></span>;

    <span class="directive">public</span> <span class="directive">static</span> <span class="type">void</span> main(<span class="predefined-type">String</span><span class="type">[]</span> args) <span class="directive">throws</span> DocumentException,
            <span class="exception">IOException</span> {
        <span class="predefined-type">Document</span> document = <span class="keyword">new</span> <span class="predefined-type">Document</span>();

        PdfWriter writer = PdfWriter.getInstance(document,
                <span class="keyword">new</span> <span class="predefined-type">FileOutputStream</span>(OUTPUTFILE));
        document.open();
        PdfReader reader = <span class="keyword">new</span> PdfReader(INPUTFILE);
        <span class="type">int</span> n = reader.getNumberOfPages();

                <span class="predefined-type">Image</span> instance = <span class="predefined-type">Image</span>.getInstance(page);
                document.add(instance);
            }
        }
