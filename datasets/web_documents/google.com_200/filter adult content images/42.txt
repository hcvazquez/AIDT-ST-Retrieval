Using these policies, you can block or allow specific URLs or an entire category of websites.</span>&nbsp;</p> <p><span style="font-family: Arial; font-size: 10pt">Cyberoam’s webcat database contains millions of websites grouped into categories like “Games”, “DatingAndMatrimonials”, “Cricket”, “CrimeandSuicide”, etc. Allowing or blocking any particular category allows/blocks all websites falling under that particular category. This reduces the effort on part of the administrator to create policies for each individual website.</span>&nbsp;</p> <p><span style="font-family: Arial; font-size: 10pt">This article describes how you can block access to websites belonging to a particular category.</span>&nbsp;</p> <p><span style="font-family: Arial; font-size: 12pt"><strong>Prerequisite</strong></span></p> <p><span style="font-family: Arial; font-size: 10pt">Web and Application Filter Module Subscribed.</span>&nbsp;</p> <p><span style="font-family: Arial; font-size: 12pt"><strong>Scenario</strong></span></p> <p><span style="font-family: Arial; font-size: 10pt">Create a Web Filter Policy such that it denies access to all the websites falling under “Games” web category.</span>&nbsp;</p> <p><span style="font-family: Arial; font-size: 12pt"><strong>Configuration</strong></span></p> <p><span style="font-family: Arial; font-size: 10pt"></span><span style="font-family: Arial; font-size: 10pt"></span></p> <p style="margin: 0cm 0cm 0pt" class="MsoBodyText"><span style="font-family: Arial; font-size: 10pt" lang="EN-GB">You must be logged on to the Web Admin Console as an administrator with Read-Write permission for relevant feature(s).</span><span lang="EN-GB">

               <span style="font-family: Arial; font-size: 10pt"><strong>Step 3: Apply Policy to Firewall Rule or User/User Group</strong></span>
              </div> <p><strong style="font-family: Arial; font-size: 10pt"><u>Firewall Rule</u></strong></p> <p><span style="font-family: Arial; font-size: 10pt">You can apply the policy through a Firewall Rule such that it is applied on all traffic that hits on that rule. To create a Firewall Rule, go to </span><strong style="font-family: Arial; font-size: 10pt">Firewall &gt; Rule &gt; Rule </strong><span style="font-family: Arial; font-size: 10pt">and click </span><strong style="font-family: Arial; font-size: 10pt">Add. </strong><span style="font-family: Arial; font-size: 10pt">As shown below, apply the Policy created in step 1.</span></p> <p> </p>
              <div align="center">
               <img style="font-size: 10pt" border="0" alt="" src="/admin/virtual/imgs/2013/Web Filter/Configure Web Filter Policy/Firewall Rule NG.png">
              </div> <p>&nbsp;</p> <p><span style="font-family: Arial; font-size: 10pt">Click </span><strong style="font-family: Arial; font-size: 10pt">OK </strong><span style="font-family: Arial; font-size: 10pt">to create the Firewall Rule.</span><strong><u><span></span></u></strong></p> <p><strong style="font-family: Arial; font-size: 10pt"><u>User/User Group</u></strong></p> <p><span style="font-family: Arial; font-size: 10pt">You can apply the rule to individual users or user groups. Here, as an example we have applied the rule on a user named John Smith. To apply the policy on an individual user, go to </span><strong style="font-family: Arial; font-size: 10pt">Identity &gt; Users &gt; Users </strong><span style="font-family: Arial; font-size: 10pt">and select the user on whom policy is to be applied. As shown below, apply the Policy created in step 1.</span></p> <p> </p>
              <div align="center">
               <img border="0" alt="" src="/admin/virtual/imgs/2013/Web Filter/Configure Web Filter Policy/User Configuration NG.png" width="739" height="739">&nbsp;
              </div> <p>&nbsp;</p> <p><span style="font-family: Arial; font-size: 10pt">Click </span><strong style="font-family: Arial; font-size: 10pt">OK </strong><span style="font-family: Arial; font-size: 10pt">to apply policy on the user.</span>&nbsp;</p> 
              <div>
               <span style="font-family: Arial; font-size: 10pt"> <p>Similarly, you can apply the policy on a Group. To apply the policy on a group, Go to <strong>Identity &gt; Groups &gt; Groups </strong>and select the group on which the policy is to be applied. As an example, we have applied the policy on the group ‘Sales and Marketing’.&nbsp;</p>
