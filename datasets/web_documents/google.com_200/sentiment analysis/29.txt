           <p>IBM’s Bluemix platform is home to an ever growing number of cognitive computing services that, as developers, we can use in our applications to learn more about our data. Earlier this year <a href="https://www.twilio.com/blog/2015/02/answering-health-questions-via-sms-with-ibm-watson-and-twilio.html" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.twilio.com/blog/2015/02/answering-health-questions-via-sms-with-ibm-watson-and-twilio.html', 'Ricky showed us how combining Watson’s question and answer API, Twilio and Node.js could get us answers to health questions');">Ricky showed us how combining Watson’s question and answer API, Twilio and Node.js could get us answers to health questions</a>. Also living under the Watson banner, <a href="http://www.alchemyapi.com/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://www.alchemyapi.com/', 'AlchemyAPI');">AlchemyAPI</a>&nbsp;is a set of services for understanding content and context within text and images.</p> 
           <p>This is where my lame headline joke kicks in. I’m not talking about getting nostalgic about sending abbreviated SMS messages from your Nokia 3310. Today we’re going to look into analysing the sentiment of text messages using one of the AlchemyAPI services hosted on Bluemix. We’ll build up an application that receives SMS messages and deals with them differently based on the sentiment in the message.</p> 
           <p>If you’d rather watch a video of this tutorial you’re in luck as earlier this year <a href="https://www.youtube.com/watch?v=-y0W9k_CP6Y&amp;feature=youtu.be" onclick="__gaTracker('send', 'event', 'outbound-article', 'https://www.youtube.com/watch?v=-y0W9k_CP6Y&amp;feature=youtu.be', 'I live coded the application as part of an IBM Bluemix webinar');" data-rel="lightbox-video-0">I live coded the application as part of an IBM Bluemix webinar</a>.</p> 
           <p>If you’d like to see what the app does right now you can send a text to my customer support line on 07903596047, if you’re in the UK like me, or (415) 214-9793 in the US.&nbsp;Feel free to praise or complain about the service and see how it responds.</p> 
           <p><img alt="Sending messages with different sentiments to the service results in different replies" src="https://www.twilio.com/blog/wp-content/uploads/2015/11/Vgw0FfpFUvxytbxD301tY5MnYsbkPJkT59s6eBUAx7pLRbC8xt7NcE69uUNBacah1HC86X_iUqRSrj5Z1UMXPAmsI2q9igIc0688Dks2_NrmpjRpV9DTeTOA5Lq4mviKBHV-Fdg.png" class="aligncenter"></p> 
           <h3>Tools for today</h3> 
           <p>Before we start&nbsp;you’re going to need to set up some bits and pieces. You’ll need:</p>
