        <h1 class="disp-none">Ionic Framework, Angularjs &amp; PhoneGap</h1> 
        <p><em>This is a getting started post on Ionic Framework &amp; deploying the starter template to PhoneGap Build.</em></p> 
        <p><em>A more advance post on this topic can be found <a title="Ionic Restify MongoDB – An End to End Hybrid App" href="http://thejackalofjavascript.com/an-end-to-end-hybrid-app/" target="_blank">here</a>.</em></p> 
        <p>When I read <a href="http://apachecordova.blogspot.in/2012/11/who-is-murdering-phonegap-its-jquery.html" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://apachecordova.blogspot.in/2012/11/who-is-murdering-phonegap-its-jquery.html', 'this');" title="Murder!!" target="_blank">this</a> article, I was pretty convinced that jQuery mobile is really killing PhoneGap &amp; its abilities. jQuery mobile is actually awesome with its cross browser support &amp; loading pages via Ajax, yada yada yada. But yes this is an overkill for a mobile app. We know what we are dealing with when we develop a hybrid app for a mobile. So why not use a user interface framework that will have only patches wrt mobile browsers!?</p> 
        <p>Yes, thats where Ionic fits in! Ionic is a</p> 
        <blockquote>
         <p>beautiful, open source front-end framework for developing hybrid mobile apps with HTML5.</p>
        </blockquote> 
        <p>The only reason I continued exploring Ionic was</p> 
        <p>1. Its Performance obsessed<br> 2. Its has a built in support for AngularJS (- The mother of all SPA frameworks)<br> 3. Most importantly its Native focused</p> 
        <p>I was sold!! And then I started diving in. And this post as mentioned earlier is a quick hands on.</p>

        <h2 id="intro">Introductions</h2> 
        <p>This post will be oriented more towards Ionic. So let me introduce you to its counterparts.</p> 
        <p>First its&nbsp;<a href="http://angularjs.org/" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://angularjs.org/', 'Angularjs');" title="Angular" target="_blank">Angularjs</a>. Angularjs is a</p> 
        <blockquote>
         <p>structural framework for dynamic web apps. It lets you use HTML as your template language and lets you extend HTML’s syntax to express your application’s components clearly and succinctly.</p> 
         <p>Angular’s data binding and dependency injection eliminate much of the code you currently have to write. And it all happens within the browser, making it an ideal partner with any server technology.</p>
        </blockquote> 
        <p>You can read more <a href="http://docs.angularjs.org/guide/introduction" onclick="__gaTracker('send', 'event', 'outbound-article', 'http://docs.angularjs.org/guide/introduction', 'here');" title="Angular Intro" target="_blank">here</a>.</p> 
        <p>Next, we have PhoneGap build. PhoneGap build provides</p> 
        <blockquote>
         <p>a way for users to create mobile applications using technologies such as HTML, CSS, and Javascript.</p>
        </blockquote> 
        <p>You can read <a title="PhoneGap Quick Start" href="http://thejackalofjavascript.com/phonegap-quick-start/" target="_blank">this</a> to know more about PhoneGap, Cordova &amp; PhoneGap Build.</p> 
        <p>Now, lets see where they fit in the big picture.</p>

And also Ionic CLI (command line tools) have a few scaffolding templates too as well as a subset of Phonegap/Cordova tools for native app development.</p>
