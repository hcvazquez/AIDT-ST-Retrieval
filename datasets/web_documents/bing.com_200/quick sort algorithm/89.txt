                <h3 style="font-size: 20px;"> <span style="color: #0b5394;">Sort Linked List using Merge Sort.</span></h3> 
                <hr> 
                <span style="color: #0b5394;">Given a Linked list, Sort it using Merge Sort Algorithm.</span>
                <br> 
                <br> 
                <span style="color: #0b5394;"><b>Merge sort is preferred algorithm for sorting a linked list, lets see why, </b></span> 
                <br> 
                <span style="color: #0b5394;">The way linked list is structured which doesn't allow random access makes some other algorithms like Quicksort perform poorly, So Merge sort is often preferred algorithm for sorting linked list.</span>
                <br> 
                <span style="color: #0b5394;">&nbsp;</span>
                <br> 
                <span style="color: #0b5394;">Lets understand what is the input and the expected output.</span>
                <br> 
                <br> 
                <div class="separator" style="clear: both; text-align: center;"> 
                 <a href="http://1.bp.blogspot.com/-vh9OYMRSyhU/VoK6XmXDPxI/AAAAAAAAA1A/fMmiJtfp8RA/s1600/merge-sort-linked-list.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="273" src="https://1.bp.blogspot.com/-vh9OYMRSyhU/VoK6XmXDPxI/AAAAAAAAA1A/fMmiJtfp8RA/s640/merge-sort-linked-list.png" width="640"></a>
                </div> 
                <br> 
                <h3 style="font-size: 18px;"> <span style="color: #0b5394;"><b><a name="more"></a></b></span><span style="color: #0b5394;"><b>Algorithm (We will use Merge Sort for sorting Linked list.)</b></span></h3> 
                <hr> 
                <!-- adsense --> 
                <span style="color: #0b5394;">For sorting Linked list, we can use any algorithm but the most suitable algorithm is Merge Sort.</span>
                <br> 
                <br> 
                <b><span style="color: #0b5394;">Why is merge sort preferred over quick sort for sorting linked lists?</span></b>
                <br> 
                <span style="color: #0b5394;">&nbsp;</span>
                <span style="color: #0b5394;">&nbsp;</span>
                <br> 
                <span style="color: #0b5394;">As Singly Linked list can be accessed in only one direction and </span>
                <span style="color: #0b5394;"><span style="color: #0b5394;">cannot be accessed randomly</span>, Quick sort will not be well suitable here.</span>
                <br> 
                <br> 
                <span style="color: #0b5394;"><b>Quick sort requires access to elements in both direction for swapping and doing such operation in Linked list is not as easy as working with Arrays.</b></span> 
                <br> 
                <span style="color: #0b5394;">Starting from the end and moving backward is usually expensive operation in Singly linked list.</span>
                <br> 
                <span style="color: #0b5394;">So Quick sort is well suited for arrays and not linked list.</span>
                <br> 
                <br> 
                <span style="color: #0b5394;">Merge sort is a divide and conquer algorithm in which need of Random access to elements is less.</span>
                <br> 
                <span style="color: #0b5394;">So Merge Sort can be used for sorting Linked list.</span>
                <br> 
                <br> 
                <br> 
                <span style="color: #0b5394;"><b>How Merge Sort works</b></span> 
                <br> 
                <ol style="text-align: left;"> 
                 <li><span style="color: #0b5394;">Merge Sort works by breaking the linked list(or Array) into 2 equal parts say Left half and Right half.</span></li> 
                 <li><span style="color: #0b5394;">Again break 2 list<b> </b>that we got in Step 1 in two equal parts each. </span><span style="color: #0b5394;">&nbsp;</span></li> 
                 <li><span style="color: #0b5394;">Repeat above steps until only 1 element remains in Linked list (or Array) because list with only 1 element is always sorted.</span><span style="color: #0b5394;">&nbsp;</span></li> 
                 <li><span style="color: #0b5394;">So in each step we are breaking the list in Left half and Right half.</span> <span style="color: #0b5394;">&nbsp;</span></li> 
                 <li><span style="color: #0b5394;">When complete list is divided and contains only Single element in Left and Right half each,</span><span style="color: #0b5394;"><b> </b>Start comparing and sorting each Left and Right half, So that portion of Linked list will be sorted.</span></li> 
                 <li><span style="color: #0b5394;">Repeat Step 5 for all the remaining Left and Right half and complete linked list will be sorted. </span></li> 
                </ol> 
                <blockquote class="tr_bq"> 
                 <span style="color: #0b5394;"><b><span style="color: #0b5394;">It works on divide and conquer technique. Time complexity is O(N log N).</span></b> </span>
                </blockquote> 
                <b><span style="color: #0b5394;">Lets understand with the help of below example.</span></b>
                <br> 
                <br> 
                <div class="separator" style="clear: both; text-align: center;"> 
                 <a href="http://1.bp.blogspot.com/-sx73LLcWZks/VoLTZRUJxeI/AAAAAAAAA1Q/ycR-g6YNmsU/s1600/merge-sort-linked-list-recursive-stack-trace.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="322" src="https://1.bp.blogspot.com/-sx73LLcWZks/VoLTZRUJxeI/AAAAAAAAA1Q/ycR-g6YNmsU/s640/merge-sort-linked-list-recursive-stack-trace.png" width="640"></a>
                </div> 
                <br> 
                <h3 style="font-size: 20px;"> <span style="color: #0b5394;">Sorting Linked list using Merge sort in Java. (Recursive Approach)</span></h3> 
                <hr> 
                <pre class="brush: java">package linkedlist.singly;

public class SortLinkedList {

 Node startNode;
 
 public static void main(String[] args) {
  new SortLinkedList();
 }

 public SortLinkedList() {
  Node node1 = new Node(10);
  Node node2 = new Node(1);
  Node node3 = new Node(-2);
  Node node4 = new Node(8);
  Node node5 = new Node(9);
  Node node6 = new Node(10);
  Node node7 = new Node(1);

  node1.setNext(node2);
  node2.setNext(node3);
  node3.setNext(node4);
  node4.setNext(node5);

                <pre class="brush: java">package linkedlist.singly;

public class SortLinkedList {

 Node startNode;

 public static void main(String[] args) {
  new SortLinkedList();
 }

 public SortLinkedList() {
  Node node1 = new Node(10);
  Node node2 = new Node(1);
  Node node3 = new Node(-2);
  Node node4 = new Node(8);
  Node node5 = new Node(9);
  Node node6 = new Node(10);
  Node node7 = new Node(1);

  node1.setNext(node2);
  node2.setNext(node3);
  node3.setNext(node4);
  node4.setNext(node5);
  node5.setNext(node6);
  node6.setNext(node7);

  startNode = node1;

  Node sortedStartNode = mergeSortLinkList(startNode);
  printLinkList(sortedStartNode);
 }

 private Node mergeSortLinkList(Node startNode){

  //Break the list until list is null or only 1 element is present in List.
  if(startNode==null || startNode.getNext()==null){
   return startNode;
  }

  //Break the linklist into 2 list.
  //Finding Middle node and then breaking the Linled list in 2 parts.
  //Now 2 list are, 1st list from start to middle and 2nd list from middle+1 to last.

  Node middle = getMiddle(startNode);
  Node nextOfMiddle = middle.getNext();
  middle.setNext(null);

  //Again breaking the List until there is only 1 element in each list.
  Node left = mergeSortLinkList(startNode);
  Node right = mergeSortLinkList(nextOfMiddle);

  //Once complete list is divided and contains only single element, 
  //Start merging left and right half by sorting them and passing Sorted list further. 
  Node sortedList = mergeTwoListIterative(left, right);

  return sortedList;
 }

 //Iterative Approach for Merging Two Sorted List
 private Node mergeTwoListIterative(Node leftStart, Node rightStart) {

  Node merged=null;
  Node temp=null;

  //To keep track of last element, so that we don't need to iterate for adding the element at last of 
  //list when either LeftStart or rightStart is NULL.
  Node lastAddedNode = null;

  while(leftStart!=null &amp;&amp; rightStart!=null){

   if(leftStart.getData()&gt;rightStart.getData()){
    temp = new Node(rightStart.getData());
    rightStart=rightStart.getNext();

   }else{
    temp = new Node(leftStart.getData());
    leftStart=leftStart.getNext();
   }

   if(merged==null){
    merged=temp;
   }else{
    lastAddedNode.setNext(temp);     
   }
   lastAddedNode=temp;
  }

  if(leftStart!=null){
   lastAddedNode.setNext(leftStart);
  }else{
   lastAddedNode.setNext(rightStart);
  }
  
  return merged;
 }

 private Node getMiddle(Node startNode) {
  if(startNode==null){
   return startNode;
  }

  Node pointer1=startNode;
  Node pointer2=startNode;

  while(pointer2!=null &amp;&amp; pointer2.getNext()!=null &amp;&amp; pointer2.getNext().getNext()!=null){
   pointer1 = pointer1.getNext();
   pointer2 = pointer2.getNext().getNext();

  }
  return pointer1;
 }

 private void printLinkList(Node startNode) {
  Node temp = startNode;
  while(temp!=null){
   System.out.print(temp.getData() + " ");
   temp = temp.getNext();
  }
 }
