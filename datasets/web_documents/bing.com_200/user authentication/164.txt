                 <a href="https://2.bp.blogspot.com/-ETk20q392rs/WAwNzQ3bi9I/AAAAAAAAAMU/bzbQ4N5SujcgZUphUofVFQWOxIcWo3jygCLcB/s1600/FB-BANNER.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="Firebase User authentication" border="0" height="312" src="https://2.bp.blogspot.com/-ETk20q392rs/WAwNzQ3bi9I/AAAAAAAAAMU/bzbQ4N5SujcgZUphUofVFQWOxIcWo3jygCLcB/s640/FB-BANNER.png" title="Firebase User authentication" width="640"></a>
                </div> Firebase provides more features as described in 
                <a href="https://androidmads.blogspot.in/2016/10/android-getting-started-with-firebase.html">previous post</a>. One of the very useful feature is User Authentication. Different types of User Authentication are, 
                <br> 
                <ol> 
                 <li>Email and Password Authentication</li> 
                 <li>Google Plus Authentication</li> 
                 <li>Facebook Authentication</li> 
                 <li>Github Authentication</li> 
                </ol> 
                <div style="text-align: justify;">
                  In this post, I will explain the way to implement Firebase Email &amp; Password authentication. To demonstrate how simplified and easy to use Firebase is, we will build a simple login / register (Firebase Authentication) demo. This separates sensitive user credentials from your application data, and lets you focus on the user interface and experience for your Application. It is suitable for Simple, Easy and Perfect way of handling Login, Registration, Forget Password and so on.
                 <br> 
                 <br>
                </div> 
                <div style="text-align: justify;">
                  The Feature and Settings up of Application in Application is Explained in 
                 <a href="https://androidmads.blogspot.in/2016/10/android-getting-started-with-firebase.html">Previous post</a>. After creating the project, Enable Firebase Email &amp; Password authentication by selecting Authentication in Left Pane in Firebase Console and Go to Sign-in-Method as in the following Image.
                </div> 
                <br> 
                <div class="separator" style="clear: both; text-align: center;"> 
                 <a href="https://1.bp.blogspot.com/-GQmzYbWmq9s/WAwOB-04NQI/AAAAAAAAAMY/0Hn0k_hSvtcspcwhlCp-nsR4WtrNSqpEACLcB/s1600/enable_user_auth.png" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img alt="Enable Authentication" border="0" height="267" src="https://1.bp.blogspot.com/-GQmzYbWmq9s/WAwOB-04NQI/AAAAAAAAAMY/0Hn0k_hSvtcspcwhlCp-nsR4WtrNSqpEACLcB/s400/enable_user_auth.png" title="Enable Authentication" width="400"></a>
                </div> 
                <br> 
                <h2 id="customh2"> <b>I. Creating and Setting up Android Project</b></h2> 1. Create a new project in Android Studio from File ? New Project. When it prompts you to select the default activity, select Blank Activity and proceed. While filling the project details, use the same package name which you gave in Firebase console.
                <br> 
                <br> 2. Open AndroidManifest.xml and add the INTERNET permission as we need to make network calls.
                <br> 
                <pre class="prettyprint notranslate" style="background: rgb(247, 247, 247); box-sizing: inherit; color: #37474f; font-family: &quot;roboto mono&quot;, monospace; font-size: 14px; font-stretch: normal; line-height: 20px; margin-bottom: 16px; margin-top: 16px; overflow-x: auto; padding: 8px; position: relative;" translate="no"><code>&lt;uses-permission android:name="android.permission.INTERNET" /&gt;</code></pre> 
                <br> 3. Paste the 
                <b>google-services.json</b> file to your projects app folder. This step is very important as your project wont build without this file.
                <br> 
                <br> 4. Now open the build.gradle located in projects home directory and add firebase dependency.
                <br> 
                <pre class="prettyprint notranslate" style="background: rgb(247, 247, 247); box-sizing: inherit; color: #37474f; font-family: &quot;roboto mono&quot;, monospace; font-size: 14px; font-stretch: normal; line-height: 20px; margin-bottom: 16px; margin-top: 16px; overflow-x: auto; padding: 8px; position: relative;" translate="no"><code><u class="file_name">build.gradle</u>

}</code></pre> 
               </div> 5. Open 
               <b>app/build.gradle</b> and add firebase auth dependency. At the very bottom of the file, add 
               <b>apply plugin: com.google.gms.google-services</b>
               <br> 
               <pre class="prettyprint notranslate" style="background: rgb(247, 247, 247); box-sizing: inherit; color: #37474f; font-family: &quot;roboto mono&quot;, monospace; font-size: 14px; font-stretch: normal; line-height: 20px; margin-bottom: 16px; margin-top: 16px; overflow-x: auto; padding: 8px; position: relative;" translate="no"><code><u class="file_name">app/build.gradle</u>
