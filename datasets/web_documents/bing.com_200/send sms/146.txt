      <h1 class="title"> Sending and Receiving Messages (JavaScript Library) </h1> 
      <div id="mainSection"> 
       <div id="mainBody"> 
        <div class="alert">
         <table summary="table">
          <tbody>
           <tr>
            <th scope="col"><img id="note" alt="Ff750612.note(en-us,MSDN.10).gif" src="https://i-msdn.sec.s-msft.com/areas/global/content/clear.gif" title="Ff750612.note(en-us,MSDN.10).gif" xmlns="" class="cl_IC101471">Note: </th>

            <td> Current information about Live Connect is now available in the <a href="http://go.microsoft.com/fwlink/?LinkID=191078" target="_blank">Windows Live Developer Center</a>. The information in the following sections is provided for legacy purposes only. </td>

        <span id="changeHistory"> </span> 
        <p>You can give users the ability to send messages to any member of their Windows Live Messenger contact lists. To do this, you need to incorporate the following functionality in your application.</p> 
        <ul>
         <li> Get a <a href="https://msdn.microsoft.com/en-us/library/ff749649.aspx">Messenger Context</a> instance. See <a href="https://msdn.microsoft.com/en-us/library/ff749551.aspx">Signing In to Messenger (JavaScript Library)</a> for information about how to do this.<br> </li>
         <li> Create a function to send messages.<br> </li>
         <li> Create a function to handle incoming messages.<br> </li>
        </ul> 
        <div>
         <h2 class="LW_CollapsibleArea_TitleDiv">
          <div>
           <a class="LW_CollapsibleArea_TitleAhref" title="Collapse" role="heading"><span class="cl_CollapsibleArea_expanding LW_CollapsibleArea_Img"></span><span class="LW_CollapsibleArea_Title">Types of Messages</span></a>
           <div id="Anchor_0" class="LW_CollapsibleArea_Anchor_Div">
            <a href="/en-us/library/ff750612.aspx#Anchor_0" class="LW_CollapsibleArea_Anchor_Img" title="Right-click to copy and share the link for this section"></a>
           </div>
           <div class="LW_CollapsibleArea_HrDiv">
            <hr class="LW_CollapsibleArea_Hr">
           </div>
          </div></h2>
         <div class="sectionblock">
          <a id="sectionToggle0"></a> 
          <p>There are three types of messages that you can send, as defined by the <a href="https://msdn.microsoft.com/en-us/library/ff751774.aspx">Microsoft.Live.Messenger.MessageType</a> enumerator:</p> 
          <ul>
           <li> Application message - An application-specific message.<br> </li>
           <li> Nudge message - Used to send a "nudge" message.<br> </li>
           <li> Text message - Used to send a traditional Messenger text message.<br> </li>
          </ul> 
          <p>This topic covers sending text and nudge messages. For more information about application messages, see <a href="https://msdn.microsoft.com/en-us/library/ff751471.aspx">Sending and Receiving Application Messages (JavaScript Library)</a>.</p> 
         </div>
        </div>
        <div>
         <h2 class="LW_CollapsibleArea_TitleDiv">
          <div>
           <a class="LW_CollapsibleArea_TitleAhref" title="Collapse" role="heading"><span class="cl_CollapsibleArea_expanding LW_CollapsibleArea_Img"></span><span class="LW_CollapsibleArea_Title">Procedures</span></a>
           <div id="Anchor_1" class="LW_CollapsibleArea_Anchor_Div">
            <a href="/en-us/library/ff750612.aspx#Anchor_1" class="LW_CollapsibleArea_Anchor_Img" title="Right-click to copy and share the link for this section"></a>
           </div>
           <div class="LW_CollapsibleArea_HrDiv">
            <hr class="LW_CollapsibleArea_Hr">
