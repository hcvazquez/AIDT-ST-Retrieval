     <td align="left" width="100%" valign="top"> <h1>ASM</h1> 
      <table border="0" cellpadding="5" cellspacing="0" summary=""> 
       <tbody>
        <tr> 
         <td rowspan="1" colspan="1" valign="top" align="left"> <p> <img src="images/asm2.png" border="0" width="200" height="125" alt="ASM logo" align="middle"> </p> <p> <strong>ASM</strong> is an all purpose Java bytecode manipulation and analysis framework. It can be used to modify existing classes or dynamically generate classes, directly in binary form. Provided common transformations and analysis algorithms allow to easily assemble custom complex transformations and code analysis tools.</p> <p>ASM offer similar functionality as other bytecode frameworks, but it is focused on simplicity of use and performance. Because it was designed and implemented to be as small and as fast as possible, it makes it very attractive for using in dynamic systems*.</p> <p>(*) ASM can of course be used in a static way too.</p> <p> <a href="current/asm-eng.pdf">"ASM: a code manipulation tool to implement adaptable systems"</a>, E. Bruneton, R. Lenglet and T. Coupaye, <a href="http://arcad.essi.fr/2002-10-composants.html">Adaptable and extensible component systems</a>, November 2002, Grenoble, France. Also available in <a href="current/asm-fr.pdf">french</a>.</p> <p> <a href="current/asm-transformations.pdf">"Using ASM framework to implement common bytecode transformation patterns"</a>, E. Kuleshov, <a href="http://aosd.net/2007/">AOSD.07</a>, March 2007, Vancouver, Canada.</p> 
          <hr noshade size="1"> <p>ASM is already <a href="users.html">used</a> in many Java products. Please <a href="http://mail.ow2.org/wws/info/asm">let us know</a> if your project is using ASM. You can also add the "Powered by ASM" button below to your project page.</p> <a href="http://asm.objectweb.org/"><img src="images/poweredbyasm.gif" border="0" width="88" height="31" alt="Powered by ASM" align="middle"></a> </td>
         <td rowspan="1" colspan="1" valign="top" align="left"> <p> <strong>Documentation</strong> </p> 
          <hr noshade size="1"> <p> <strong>The best way to learn to use ASM is to write a Java source file that is equivalent to what you want to generate and then use the ASMifier mode of the <a href="eclipse/index.html">Bytecode Outline</a> plugin for Eclipse (or the <a href="doc/faq.html#Q10">ASMifier</a> tool) to see the equivalent ASM code. If you want to implement a class transformer, write two Java source files (before and after transformation) and use the compare view of the plugin in ASMifier mode to compare the equivalent ASM code.</strong> </p> <p>The following document provides an in-depth introduction to ASM 4.0, going from the basic features to advanced topics such as bytecode analysis.</p> <p> <a href="http://download.forge.objectweb.org/asm/asm4-guide.pdf">ASM 4.0 A Java bytecode engineering library</a>.</p> <p> <a href="doc/tutorial-asm-2.0.html">Tutorial for ASM 2.0</a>.</p> <p> <a href="doc/tutorial.html">Tutorial for ASM 1.5.x</a>.</p> <p> <a href="doc/tutorial-annotations.html">Tutorial for using J2SE 5.0 Annotations with ASM 1.5.x</a>.</p> <p> <a href="doc/faq.html">Frequently Asked Questions</a>.</p> <p> <br> <strong>External articles</strong> </p> 
          <hr noshade size="1"> <p> <a href="http://www-128.ibm.com/developerworks/java/library/j-cwt05125/index.html">ASM 2.0 Classworking</a>.</p> <p> <a href="http://www-128.ibm.com/developerworks/java/library/j-cwt06075/">Annotations with ASM 2.0</a>.</p> <p> <a href="http://www-128.ibm.com/developerworks/java/library/j-cwt02076.html">Generics with ASM 2.0</a>.</p> <p> <br> <strong>API</strong> </p> 
          <hr noshade size="1"> <p> <a href="asm60/javadoc/user/index.html">ASM 6.0 API</a> (<a href="asm60/javadoc/user/overview-summary.html">noframes</a>). </p> <p> <a href="asm50/javadoc/user/index.html">ASM 5.0 API</a> (<a href="asm50/javadoc/user/overview-summary.html">noframes</a>). </p> <p> <a href="asm40/javadoc/user/index.html">ASM 4.0 API</a> (<a href="asm40/javadoc/user/overview-summary.html">noframes</a>). </p> </td>
