           <h1 data-reactid="67">How to Write a Web Scraper in Node.js</h1>
           <div class="article__header__section space-between" data-reactid="68">
            <div class="article__header__timestamps" data-reactid="69">
             <small class="publish-date" data-reactid="70">
              <!-- react-text: 71 -->Published 
              <!-- /react-text -->
              <!-- react-text: 72 -->Apr 29, 2016
              <!-- /react-text --></small>
             <small class="updated-at" data-reactid="73">
              <!-- react-text: 74 -->Last updated 
              <!-- /react-text -->
              <!-- react-text: 75 -->Jan 18, 2017
              <!-- /react-text --></small>
            </div>
           </div>
          </div>
          <div class="lazy-image article__cover-photo" data-reactid="76">
           <div class="lazy-image__progressive" data-reactid="77">
            <img class="lazy-image__progressive__preview" alt="How to Write a Web Scraper in Node.js" src="https://process.filestackapi.com/cache=expiry:max/resize=width:20/V03cIeyITfE1mfVUMM0N" data-reactid="78">
            <div style="height:200px;" class="lazyload-placeholder" data-reactid="79"></div>
           </div>
          </div>
          <div data-reactid="80">
           <div class="article__body" data-reactid="81">
            <div class="article__content" data-reactid="82">
             <p>Sometimes we need to collect information from different web pages automagically. Obviously, a human is not needed for that. A smart script can do the job pretty good, especially if it's something repetitive.</p> 
             <p>When there is no web based API to share the data with our app, and we still want to extract some data from that website, we have to fallback to scraping.</p> 
             <p>This means:</p> 
             <ol> 
              <li>We load the page (a <code>GET</code> request is often enough)</li> 
              <li>Parse the HTML result</li> 
              <li>Extract the needed data</li> 
             </ol> 
             <p>In Node.js, all these three steps are quite easy because the functionality is already made for us in different modules, by different developers.</p> 
             <p>Because I often scrape random websites, I created yet another scraper: <a href="https://github.com/IonicaBizau/scrape-it" target="_blank" rel="nofollow"><code>scrape-it</code></a> – <em>a Node.js scraper for humans</em>. It's designed to be really simple to use and still is quite minimalist.</p> 
             <p>Here is how I did it:</p> 
             <h3 id="1-load-the-page"><a class="header-anchor" href="#1-load-the-page" aria-hidden="true"><i class="fa fa-link" aria-hidden="true"></i></a> 1. Load the page</h3> 
             <p>To load the web page, we need to use a library that makes HTTP(s) requests. There are a lot of modules doing that that. Like always, I recommend choosing simple/small modules - I wrote a tiny package that does it: <a href="https://github.com/IonicaBizau/tinyreq" target="_blank" rel="nofollow"><code>tinyreq</code></a>.</p> 
             <p>Using this module, you can easily get the HTML rendered by the server from a web page:</p> 
             <pre><code class="language-js"><span class="hljs-keyword">const</span> request = <span class="hljs-built_in">require</span>(<span class="hljs-string">"tinyreq"</span>);

request(<span class="hljs-string">"http://ionicabizau.net/"</span>, <span class="hljs-function"><span class="hljs-keyword">function</span> (<span class="hljs-params">err, body</span>) </span>{
    <span class="hljs-built_in">console</span>.log(err || body); <span class="hljs-comment">// Print out the HTML</span>
});
</code></pre> 
             <p><code>tinyreq</code> is actually a friendlier wrapper around the native <a href="https://nodejs.org/api/http.html#http_http_request_options_callback" target="_blank" rel="nofollow"><code>http.request</code></a> built-in solution.</p> 
             <p>Once we have a piece of HTML, we need to parse it. How to do that? <img class="emoji" draggable="false" alt="?" src="https://twemoji.maxcdn.com/2/72x72/1f4ad.png"></p> 
             <h3 id="2-parsing-the-html"><a class="header-anchor" href="#2-parsing-the-html" aria-hidden="true"><i class="fa fa-link" aria-hidden="true"></i></a> 2. Parsing the HTML</h3> 
             <p>Well, that's a complex thing. Other people did that for us already. I like very much the <a href="https://github.com/cheeriojs/cheerio" target="_blank" rel="nofollow"><code>cheerio</code></a> module. It provides a jQuery-like interface to interact with a piece of HTML you already have.</p> 
             <pre><code class="language-js"><span class="hljs-keyword">const</span> cheerio = <span class="hljs-built_in">require</span>(<span class="hljs-string">"cheerio"</span>);

<span class="hljs-comment">// Parse the HTML </span>
<span class="hljs-keyword">let</span> $ = cheerio.load(<span class="hljs-string">"&lt;h2 class="</span>title<span class="hljs-string">"&gt;Hello world&lt;/h2&gt;"</span>);

<span class="hljs-comment">// Take the h2.title element and show the text</span>
<span class="hljs-built_in">console</span>.log($(<span class="hljs-string">"h2.title"</span>).text());
<span class="hljs-comment">// =&gt; Hello world</span>
</code></pre> 
             <p>Because I like to modularize all the things, I created <a href="https://github.com/IonicaBizau/cheerio-req" target="_blank" rel="nofollow"><code>cheerio-req</code></a> which is basically <code>tinyreq</code> combined with <code>cheerio</code> (basically the previous two steps put together):</p> 
             <pre><code class="language-js"><span class="hljs-keyword">const</span> cheerioReq = <span class="hljs-built_in">require</span>(<span class="hljs-string">"cheerio-req"</span>);

cheerioReq(<span class="hljs-string">"http://ionicabizau.net"</span>, (err, $) =&gt; {
    <span class="hljs-built_in">console</span>.log($(<span class="hljs-string">".header h1"</span>).text());
    <span class="hljs-comment">// =&gt; Ionic? Biz?u</span>
});
</code></pre> 
             <p>Since we already know how to parse the HTML, the next step is to build a nice public interface we can export into a module.</p> 
             <h3 id="3-extract-the-needed-data"><a class="header-anchor" href="#3-extract-the-needed-data" aria-hidden="true"><i class="fa fa-link" aria-hidden="true"></i></a> 3. Extract the needed data</h3> 
             <p>Putting the previous steps together, we have this (follow the inline comments):</p> 
             <pre><code class="language-js"><span class="hljs-meta">"use strict"</span>

<span class="hljs-comment">// Import the dependencies</span>
<span class="hljs-keyword">const</span> cheerio = <span class="hljs-built_in">require</span>(<span class="hljs-string">"cheerio"</span>)
    , req = <span class="hljs-built_in">require</span>(<span class="hljs-string">"tinyreq"</span>)
    ;

<span class="hljs-comment">// Define the scrape function</span>
<span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">scrape</span>(<span class="hljs-params">url, data, cb</span>) </span>{
    <span class="hljs-comment">// 1. Create the request</span>
    req(url, (err, body) =&gt; {
        <span class="hljs-keyword">if</span> (err) { <span class="hljs-keyword">return</span> cb(err); }

        <span class="hljs-comment">// 2. Parse the HTML</span>
        <span class="hljs-keyword">let</span> $ = cheerio.load(body)
          , pageData = {}
          ;

        <span class="hljs-comment">// 3. Extract the data</span>
        <span class="hljs-built_in">Object</span>.keys(data).forEach(<span class="hljs-function"><span class="hljs-params">k</span> =&gt;</span> {
            pageData[k] = $(data[k]).text();
        });

        <span class="hljs-comment">// Send the data in the callback</span>
        cb(<span class="hljs-literal">null</span>, pageData);
    });
}

<span class="hljs-comment">// Extract some data from my website</span>
scrape(<span class="hljs-string">"http://ionicabizau.net"</span>, {
    <span class="hljs-comment">// Get the website title (from the top header)</span>
    title: <span class="hljs-string">".header h1"</span>
    <span class="hljs-comment">// ...and the description</span>
  , <span class="hljs-attr">description</span>: <span class="hljs-string">".header h2"</span>
}, (err, data) =&gt; {
    <span class="hljs-built_in">console</span>.log(err || data);
});
</code></pre> 
             <p>When running this code, we get the following output in the console:</p> 
             <pre><code class="language-js">{ <span class="hljs-attr">title</span>: <span class="hljs-string">'Ionic? Biz?u'</span>,
  <span class="hljs-attr">description</span>: <span class="hljs-string">'Web Developer, Linux geek and Musician'</span> }
</code></pre> 
             <p>Hey! That's my website information, so it's working. We now have a small function that can get the text from any element on the page.</p> 
             <hr> 
             <p>In the <a href="https://github.com/IonicaBizau/scrape-it" target="_blank" rel="nofollow">module I have written</a> I made it possible to scrape lists of things (e.g. articles, pages etc).</p> 
             <p>So, basically to get the latest 3 articles on my blog, you can do:</p> 
             <pre><code class="language-js"><span class="hljs-keyword">const</span> scrapeIt = <span class="hljs-built_in">require</span>(<span class="hljs-string">"scrape-it"</span>);

<span class="hljs-comment">// Fetch the articles on the page (list)</span>
scrapeIt(<span class="hljs-string">"http://ionicabizau.net"</span>, {
    <span class="hljs-attr">listItem</span>: <span class="hljs-string">".article"</span>
  , <span class="hljs-attr">name</span>: <span class="hljs-string">"articles"</span>
  , <span class="hljs-attr">data</span>: {
        <span class="hljs-attr">createdAt</span>: {
            <span class="hljs-attr">selector</span>: <span class="hljs-string">".date"</span>
          , <span class="hljs-attr">convert</span>: <span class="hljs-function"><span class="hljs-params">x</span> =&gt;</span> <span class="hljs-keyword">new</span> <span class="hljs-built_in">Date</span>(x)
        }
      , <span class="hljs-attr">title</span>: <span class="hljs-string">"a.article-title"</span>
      , <span class="hljs-attr">tags</span>: {
            <span class="hljs-attr">selector</span>: <span class="hljs-string">".tags"</span>
          , <span class="hljs-attr">convert</span>: <span class="hljs-function"><span class="hljs-params">x</span> =&gt;</span> x.split(<span class="hljs-string">"|"</span>).map(<span class="hljs-function"><span class="hljs-params">c</span> =&gt;</span> c.trim()).slice(<span class="hljs-number">1</span>)
        }
      , <span class="hljs-attr">content</span>: {
            <span class="hljs-attr">selector</span>: <span class="hljs-string">".article-content"</span>
          , <span class="hljs-attr">how</span>: <span class="hljs-string">"html"</span>
        }
    }
}, (err, page) =&gt; {
    <span class="hljs-built_in">console</span>.log(err || page);
});
<span class="hljs-comment">// { articles:</span>
<span class="hljs-comment">// [ { createdAt: Mon Mar 14 2016 00:00:00 GMT+0200 (EET),</span>
<span class="hljs-comment">// title: 'Pi Day, Raspberry Pi and Command Line',</span>
<span class="hljs-comment">// tags: [Object],</span>
<span class="hljs-comment">// content: '&lt;p&gt;Everyone knows (or should know)...a" alt=""&gt;&lt;/p&gt;\n' },</span>
<span class="hljs-comment">// { createdAt: Thu Feb 18 2016 00:00:00 GMT+0200 (EET),</span>
<span class="hljs-comment">// title: 'How I ported Memory Blocks to modern web',</span>
<span class="hljs-comment">// tags: [Object],</span>
<span class="hljs-comment">// content: '&lt;p&gt;Playing computer games is a lot of fun. ...' },</span>
<span class="hljs-comment">// { createdAt: Mon Nov 02 2015 00:00:00 GMT+0200 (EET),</span>
<span class="hljs-comment">// title: 'How to convert JSON to Markdown using json2md',</span>
<span class="hljs-comment">// tags: [Object],</span>
<span class="hljs-comment">// content: '&lt;p&gt;I love and ...' } ] }</span>
</code></pre> 
             <p><img class="b-lazy" data-src="http://i.imgur.com/vU1ve3s.png" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" alt=""></p> 
             <p>Happy scraping!</p>

I like very much the [`cheerio`](https://github.com/cheeriojs/cheerio) module. It provides a jQuery-like interface to interact with a piece of HTML you already have.\r\n\r\n```js\r\nconst cheerio = require(\"cheerio\");\r\n\r\n// Parse the HTML \r\nlet $ = cheerio.load(\"\u003ch2 class=\"title\"\u003eHello world\u003c/h2\u003e\");\r\n\r\n// Take the h2.title element and show the text\r\nconsole.log($(\"h2.title\").text());\r\n// =\u003e Hello world\r\n```\r\n\r\nBecause I like to modularize all the things, I created [`cheerio-req`](https://github.com/IonicaBizau/cheerio-req) which is basically `tinyreq` combined with `cheerio` (basically the previous two steps put together):\r\n\r\n```js\r\nconst cheerioReq = require(\"cheerio-req\");\r\n\r\ncheerioReq(\"http://ionicabizau.net\", (err, $) =\u003e {\r\n    console.log($(\".header h1\").text());\r\n    // =\u003e Ionic? Biz?u\r\n});\r\n```\r\n\r\nSince we already know how to parse the HTML, the next step is to build a nice public interface we can export into a module.\r\n\r\n### 3. Extract the needed data\r\n\r\nPutting the previous steps together, we have this (follow the inline comments):\r\n\r\n```js\r\n\"use strict\"\r\n\r\n// Import the dependencies\r\nconst cheerio = require(\"cheerio\")\r\n    , req = require(\"tinyreq\")\r\n    ;\r\n\r\n// Define the scrape function\r\nfunction scrape(url, data, cb) {\r\n    // 1. Create the request\r\n    req(url, (err, body) =\u003e {\r\n        if (err) { return cb(err); }\r\n\r\n        // 2. Parse the HTML\r\n        let $ = cheerio.load(body)\r\n          , pageData = {}\r\n          ;\r\n\r\n        // 3. Extract the data\r\n        Object.keys(data).forEach(k =\u003e {\r\n            pageData[k] = $(data[k]).text();\r\n        });\r\n\r\n        // Send the data in the callback\r\n        cb(null, pageData);\r\n    });\r\n}\r\n\r\n// Extract some data from my website\r\nscrape(\"http://ionicabizau.net\", {\r\n    // Get the website title (from the top header)\r\n    title: \".header h1\"\r\n    // ...and the description\r\n  , description: \".header h2\"\r\n}, (err, data) =\u003e {\r\n    console.log(err || data);\r\n});\r\n```\r\n\r\nWhen running this code, we get the following output in the console:\r\n\r\n```js\r\n{ title: 'Ionic? Biz?u',\r\n  description: 'Web Developer,  Linux geek and  Musician' }\r\n```\r\n\r\nHey! That's my website information, so it's working. We now have a small function that can get the text from any element on the page.\r\n\r\n---\r\n\r\nIn the [module I have written](https://github.com/IonicaBizau/scrape-it) I made it possible to scrape
