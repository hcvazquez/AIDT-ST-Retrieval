   <h1><a class="u" name="NAME">NAME <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <p>WebService::Google::Language - Perl interface to the Google AJAX Language API</p> 
   <h1><a class="u" name="DEPRECATION_WARNING">DEPRECATION WARNING <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <p>Please take note of this statement by Google (see links): "With the exception of the Translate v2 API, which is now available as a paid service, the APIs included in this family have all been deprecated."</p> 
   <p><b>The v1 API, which this module is based on, WILL BE SHUT OFF COMPLETELY on Dec 1, 2011</b> (see <a href="http://code.google.com/apis/language/translate/overview.html" class="podlinkurl">http://code.google.com/apis/language/translate/overview.html</a>).</p> 
   <p>The live tests of this module already started failing sporadically on Oct 29 and are now disabled by default. To check if the API is still working try:</p> 
   <pre class="sh_perl">  export WGL_FORCE_LIVE_TESTS=1 &amp;&amp; perl -Ilib -T t/05-methods.t</pre> 
   <h1><a class="u" name="SYNOPSIS">SYNOPSIS <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <pre class="sh_perl">  use WebService::Google::Language;

  my $service = WebService::Google::Language-&gt;new(
    referer =&gt; '<a href="http://example.com/" class="podlinkurl">http://example.com/</a>',
    src     =&gt; '',
    dest    =&gt; 'en',
  );

  my $result = $service-&gt;translate('Hallo Welt');
  if ($result-&gt;error) {
    printf "Error code: %s\n", $result-&gt;code;
    printf "Message:    %s\n", $result-&gt;message;
  }
  else {
    printf "Detected language: %s\n", $result-&gt;language;
    printf "Translation:       %s\n", $result-&gt;translation;
  }

  $result = $service-&gt;detect('Bonjour tout le monde');
  printf "Detected language: %s\n", $result-&gt;language;
  printf "Is reliable:       %s\n", $result-&gt;is_reliable ? 'yes' : 'no';
  printf "Confidence:        %s\n", $result-&gt;confidence;</pre> 
   <h1><a class="u" name="DESCRIPTION">DESCRIPTION <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <p>WebService::Google::Language is an object-oriented interface to the Google AJAX Language API (<a href="http://code.google.com/apis/ajaxlanguage/" class="podlinkurl">http://code.google.com/apis/ajaxlanguage/</a>).</p> 
   <p>The AJAX Language API is a web service to translate and detect the language of blocks of text.</p> 
   <h1><a class="u" name="CONSTRUCTOR">CONSTRUCTOR <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <dl> 
    <dt>
     <a name="service"></a>
     <a name="$service_=_WebService::Google::Language->new(%options);">$service = WebService::Google::Language-&gt;new(%options);</a>
    </dt> 
    <dd> 
     <p>Creates a new <code>WebService::Google::Language</code> object and returns it.</p> 
     <p>Key/value pair arguments set up the initial state:</p> 
     <pre class="sh_perl">  Key       Usage         Expected value
  ---------------------------------------------------------
  referer   mandatory     HTTP referer
  src       optional      default source language
  dest      optional      default destination language
  key       recommended   application's key
  ua        optional      an LWP::UserAgent object for reuse
  json      optional      a JSON object for reuse</pre> 
     <p>Since Google demands a "valid and accurate http referer header" in requests to their service, a non-empty referer string must be passed to the constructor. Otherwise the constructor will fail.</p> 
     <p>Unless the key 'ua' contains an instance of <code>LWP::UserAgent</code>, any additional entries in the <code>%options</code> hash will be passed unmodified to the constructor of <code>LWP::UserAgent</code>, which is used for performing the requests.</p> 
     <p>E.g. you can set your own user agent identification and specify a timeout this way:</p> 
     <pre class="sh_perl">  $service = WebService::Google::Language-&gt;new(
    referer =&gt; '<a href="http://example.com/" class="podlinkurl">http://example.com/</a>',
    agent   =&gt; 'My Application 2.0',
    timeout =&gt; 5,
  );</pre> 
     <p>Or reuse existing instances of <code>LWP::UserAgent</code> and <code>JSON</code> respectively:</p> 
     <pre class="sh_perl">  $service = WebService::Google::Language-&gt;new(
    referer =&gt; '<a href="http://example.com/" class="podlinkurl">http://example.com/</a>',
    ua      =&gt; $my_ua_obj,
    json    =&gt; $my_json_obj,
  );</pre> 
    </dd>
    <dt>
     <a name="service"></a>
     <a name="$service_=_WebService::Google::Language->new($referer);">$service = WebService::Google::Language-&gt;new($referer);</a>
    </dt> 
    <dd> 
    </dd>
    <dt>
     <a name="service"></a>
     <a name="$service_=_WebService::Google::Language->new($referer,_%options);">$service = WebService::Google::Language-&gt;new($referer, %options);</a>
    </dt> 
    <dd> 
     <p>Since the referer is the only mandatory parameter, the constructor can alternatively be called with an uneven parameter list. The first element will then be taken as the referer, e.g.:</p> 
     <pre class="sh_perl">  $service = WebService::Google::Language-&gt;new('my-valid-referer');</pre> 
    </dd> 
   </dl> 
   <h1><a class="u" name="METHODS">METHODS <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <dl> 
    <dt>
     <a name="result"></a>
     <a name="$result_=_$service->translate($text,_%args);">$result = $service-&gt;translate($text, %args);</a>
    </dt> 
    <dd> 
    </dd>
    <dt>
     <a name="result"></a>
     <a name="$result_=_$service->translate(%args);">$result = $service-&gt;translate(%args);</a>
    </dt> 
    <dd> 
     <p>The <code>translate</code> method will request the translation of a given text.</p> 
     <p>Either place the <code>$text</code> as the first parameter to this method or store it into the arguments hash using the key 'text'.</p> 
     <p>The source and the destination language can be specified as values of the keys 'src' and 'dest'. If these parameters are missing, the default values specified on construction of the object will be used.</p> 
     <p>If the object has been constructed without default values, the translate request will default to an empty string for the source language - i.e. Google will attempt to identify the language of the given text automatically. The destination language will be set to English (en).</p> 
     <p>Examples:</p> 
     <pre class="sh_perl">  # initialize without custom language defaults
  $service = WebService::Google::Language-&gt;new('<a href="http://example.com/" class="podlinkurl">http://example.com/</a>');

  # auto-detect source language and translate to English
  # (internal defaults)
  $result = $service-&gt;translate('Hallo Welt');

  # auto-detect source language and translate to French (fr)
  $result = $service-&gt;translate( 'Hallo Welt', dest =&gt; 'fr' );

  # set source to English and destination to German (de)
  %args = (
    text =&gt; 'Hello world',
    src  =&gt; 'en',
    dest =&gt; 'de',
  );
  $result = $service-&gt;translate(%args);</pre> 
     <p>See Google's documentation for supported languages, language codes and valid language translation pairs.</p> 
    </dd>
    <dt>
     <a name="result"></a>
     <a name="$result_=_$service->detect($text);">$result = $service-&gt;detect($text);</a>
    </dt> 
    <dd> 
    </dd>
    <dt>
     <a name="result"></a>
     <a name="$result_=_$service->detect(_text_=>_$text_);">$result = $service-&gt;detect( text =&gt; $text );</a>
    </dt> 
    <dd> 
     <p>The <code>detect</code> method will request the detection of the language of a given text. <code>$text</code> is the single parameter and can be passed directly or as key 'text' of a hash.</p> 
    </dd>
    <dt>
     <a name="result"></a>
     <a name="$result_=_$service->detect_language($text);">$result = $service-&gt;detect_language($text);</a>
    </dt> 
    <dd> 
     <p>If <code>detect</code> as a method name is just not descriptive enough, there is an alias <code>detect_language</code> available.</p> 
     <p>Examples:</p> 
     <pre class="sh_perl">  # detect language
  $result = $service-&gt;detect('Hallo Welt');
  # using the more verbose alias
  $result = $service-&gt;detect_language('Hallo Welt');</pre> 
    </dd>
    <dt>
     <a name="boolean"></a>
     <a name="$boolean_=_$service->ping;">$boolean = $service-&gt;ping;</a>
    </dt> 
    <dd> 
     <p>Checks if internet access to Google's service is available.</p> 
    </dd>
    <dt>
     <a name="json"></a>
     <a name="$json_=_$service->json;">$json = $service-&gt;json;</a>
    </dt> 
    <dd> 
     <p>Returns the <code>JSON</code> object used by this instance.</p> 
    </dd>
    <dt>
     <a name="service"></a>
     <a name="$service_=_$service->json($json);">$service = $service-&gt;json($json);</a>
    </dt> 
    <dd> 
     <p>Sets the <code>JSON</code> object to be used by this instance. Setters return their instance and can be chained.</p> 
    </dd>
    <dt>
     <a name="ua"></a>
     <a name="$ua_=_$service->ua;">$ua = $service-&gt;ua;</a>

    <dd> 
     <p>Returns/sets the <code>LWP::UserAgent</code> object.</p> 
    </dd>

     <a name="referer"></a>

   <h1><a class="u" name="RESULT_ACCESSOR_METHODS">RESULT ACCESSOR METHODS <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <p>Google returns the result encoded as a JSON object which will be automatically turned into a Perl hash with identically named keys. See the description of the JSON response at Google's page for the meaning of the JavaScript properties, which is identical to the Perl hash keys.</p> 
   <p>To provide some convenience accessor methods to the result, the hash will be blessed into the package <code>WebService::Google::Language::Result</code>. The method names are derived from Google's JavaScript class reference of the AJAX Language API.</p> 
   <p>The accessors marked as 'no' in the following table will always return <code>undef</code> for a result from <code>translate</code> or <code>detect</code> respectively.</p> 
   <pre class="sh_perl">  Accessor   translate  detect  description
  ---------------------------------------------------------------------
  error         yes      yes    a hash with code and message on error
  code          yes      yes    HTTP-style status code
  message       yes      yes    human readable error message
  translation   yes      no     translated text
  language      yes      yes    detected source language
  is_reliable   no       yes    reliability of detected language
  confidence    no       yes    confidence level, ranging from 0 to 1.0</pre> 
   <p>The <a href="#SYNOPSIS" class="podlinkpod">"SYNOPSIS"</a> of this module includes a complete example of using the accessor methods.</p> 
   <h1><a class="u" name="LIMITATIONS">LIMITATIONS <img alt="^" src="http://st.pimg.net/tucs/img/up.gif"></a></h1> 
   <p>Google does not allow submission of text exceeding 5000 characters in length to their service (see Terms of Use). This module will check the length of text passed to its methods and will fail if text is too long (without sending a request to Google).</p>
