      <h1 class="title" id="page-title">Mathematical Functions</h1> 
      <article class="node-3373 node node-book view-mode-full view-mode-full view-mode-full clearfix" about="/mathematical-functions" typeof="sioc:Item foaf:Document"> 
       <header class="clearfix"> 
        <span property="dc:title" content="Mathematical Functions" class="rdf-meta element-hidden"></span>
        <span property="sioc:num_replies" content="0" datatype="xsd:integer" class="rdf-meta element-hidden"></span> 
       </header> 
       <div class="field field-name-body field-type-text-with-summary field-label-hidden">
        <div class="field-items">
         <div class="field-item even" property="content:encoded">
          <p>The mathematical functions are present in both <a href="http://root.cern.ch/drupal/content/mathcore-library">MathCore</a> and <a href="http://root.cern.ch/drupal/content/mathmore-library">MathMore</a> libraries. All mathematical functions are implemented as free functions in the namespace ROOT::Math or TMath. The most used functions are in the MathCore library while the others are in the MathMore library. The functions in MathMore are all using the implementation of the <a target="_top" rel="nofollow" href="http://www.gnu.org/software/gsl/">GSL.</a><br> The naming of the ROOT::Math special functions is the same defined in the C++ Technical Report on Standard Library extensions. These special functions are defined in the header file Math/SpecFunc.h.<br> Statistical functions include the probability density functions, cumulative distribution functions and their inverses (quantiles) for various statistical distributions. They are defined in the header file Math/DistFunc.h</p> 
          <p>Some extra documentation on the mathematical functions provided by ROOT can be found in:</p> 
          <ul>
           <li><a href="http://www.cern.ch/mathlibs/sw/html/group__SpecFunc.html">Special functions</a>.</li> 
           <li><a href="http://www.cern.ch/mathlibs/sw/html/group__StatFunc.html">Statistical functions</a>.</li> 
           <li><a href="http://root.cern.ch/drupal/content/tmath">TMath</a>.</li> 
          </ul>
         </div>
