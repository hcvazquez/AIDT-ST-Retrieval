       <h2 class="leader-0">Build 3D web apps!</h2>
       <div>
        The 4.x series of the ArcGIS API for JavaScript is Esri's next-generation JavaScript API that integrates 2D and 3D into a single, easy-to-use, powerful API. Version 4.5 lets you build full-featured 3D applications powered by 
        <a href="latest/sample-code/webscene-basic/index.html">web scenes</a> that can include rich information layers such as 
        <a href="latest/sample-code/scene-toggle-elevation/index.html">terrain</a>, 
        <a href="latest/sample-code/intro-widgets/index.html">basemaps</a>, 
        <a href="latest/sample-code/layers-imagery-popup/index.html">imagery</a>, 
        <a href="latest/sample-code/visualization-vv-size/index.html">features</a>, 
        <a href="latest/sample-code/layers-integratedmeshlayer/index.html">integrated mesh layers</a>, and 
        <a href="latest/sample-code/layers-scenelayer-vv-color/index.html">3D objects</a>.
       </div>
