        <h2 id="description-header">Description</h2> 
        <p><a href="https://noorsplugin.com/easy-media-download-plugin-for-wordpress/" rel="nofollow">Easy Media Download</a> is a free download manager for WordPress. It gives your users the ability to download digital media files from your website. The plugin is lightweight and makes downloading easier. You can create beautiful download button which instantly starts the download once clicked.</p> 
        <h4>Features</h4> 
        <ul> 
         <li>Create quick downloads</li> 
         <li>Insert download buttons into your landing page</li> 
         <li>Allow users to directly download in all file formats</li> 
         <li>Allow large file download (No limit on the file size)</li> 
         <li>Accept donations for your free digital downloads using PayPal</li> 
         <li>Free downloads for music or audio files e.g. mp3, aac, m4a, wav</li> 
         <li>Free downloads for video files e.g. mp4, mov, qt (QuickTime), mpg, mpeg, flv, m4v, swf, flash, avi, wma, ogg, webm</li> 
         <li>Free downloads for document file e.g. doc, docx (OpenOffice, Microsoft Word), odt (OpenDocument), htm, html, ps, tex, xml, txt, csv</li> 
         <li>Free downloads for Office file e.g. xlsx (Microsoft Excel), pptx (Microsoft PowerPoint) </li> 
         <li>Free downloads for script e.g. js, css, php</li> 
         <li>Free downloads for archive or compressed file e.g. tar, rar, zip, 7z, gz, z</li> 
         <li>Free downloads for image file e.g. png, gif, jpg, jpeg, bmp</li> 
         <li>Free downloads for e-book e.g. pdf, epub</li> 
         <li>Track/Monitor your downloads</li> 
         <li>Ability to force a digital file download</li> 
        </ul> 
        <p>For more information please check the <a href="https://noorsplugin.com/easy-media-download-plugin-for-wordpress/" rel="nofollow">Easy Media Download</a> documentation page</p> 
        <h4>Embedding Shortcodes for the Downloads</h4> 
        <p>Create a new post/page and use the following shortcode to create a download button for your digital media file:</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip"]
</code></pre> 
        <p>here, url is a shortcode parameter that you need to replace with the actual URL to your digital file</p> 
        <p>You can also specify additional optional parameters in the shortcode to customize the free download button.</p> 
        <p><strong>Button Text</strong></p> 
        <p>By default the text for the download button is “Download Now”. But you can choose to modify the text using the “text” parameter.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" text="Free Download"]
</code></pre> 
        <p><strong>Button Width</strong></p> 
        <p>By default the width for the download button is “153”. But you can choose to modify it using the “width” parameter.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" text="Free Download" width="100"]
</code></pre> 
        <p><strong>Button Height</strong></p> 
        <p>By default the height for the download button is “41”. But you can choose to modify it using the “height” parameter.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" text="Free Download" width="100" height="50"]
</code></pre> 
        <p><strong>Download in a new window</strong></p> 
        <p>By default the download starts in the same window. But you can choose to modify it using the “target” parameter.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" target="_blank"]  (opens the download in a new window)
</code></pre> 
        <h4>Force File Download</h4> 
        <p>Usually when a digital file is recognised by the browser it does not prompt for download. The file just opens in the browser instead. You can choose to modify this behaviour using the <code>force_dl</code> parameter in the shortcode.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.mp3" force_dl="1"] 
</code></pre> 
        <h4>Download Button Templates</h4> 
        <p>Easy Media Download comes with 21 different download button templates. All you need to do is specify the style and the button will look just like it. For example:</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" color="red_darker"]
</code></pre> 
        <ul> 
         <li>grey</li> 
         <li>grey_light</li> 
         <li>red</li> 
         <li>green_dark</li> 
         <li>green_light</li> 
         <li>green</li> 
         <li>blue</li> 
         <li>blue_two</li> 
         <li>blue_three</li> 
         <li>blue_four</li> 
         <li>orange</li> 
         <li>orange_two</li> 
         <li>orange_light</li> 
         <li>orange_dark</li> 
         <li>purple</li> 
         <li>purple_dark</li> 
         <li>purple_light</li> 
         <li>yellow_red</li> 
         <li>hot_pink</li> 
         <li>pink</li> 
        </ul> 
        <p>If you want to fully customize the download button you can use your own button image instead.</p> 
        <h4>Custom CSS Class</h4> 
        <p>In addition to various button templates, If you want to apply custom styling to a button you can do so by using the <code>class</code> parameter.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" class="myclass"]
</code></pre> 
        <p>You can specify multiple CSS classes in the shortcode.</p> 
        <pre><code>[easy_media_download url="http://example.com/wp-content/uploads/myfile.zip" class="myclass1 myclass2 myclass3"]
</code></pre> 
        <h4>Embedding a Donation Button for Free Download</h4> 
        <p>You can accept donations from users visiting your site for your free downloads. Simply use the following shortcode with your PayPal email address in it.</p> 
        <pre><code>[emd_donation email="abc123@hotmail.com"]
</code></pre> 
        <p><strong>Donation Currency</strong></p> 
        <p>The default currency is “USD”. But you can also accept payments in your preferred currency using the “currency” parameter.</p> 
        <pre><code>[emd_donation email="abc123@hotmail.com" currency="CAD"] (This will set the currency to Canadian Dollar)
</code></pre> 
        <p><strong>Payment Page Language</strong></p> 
        <p>The default locale for the payment page is “US”. But you can set it to your preferred language using the “locale” parameter.</p> 
        <pre><code>[emd_donation email="abc123@hotmail.com" locale="BE"] (This will set the locale to Belgium)
</code></pre> 
        <p><strong>Donation Button Image</strong></p> 
        <p>You can customize the donation button by specifying your preferred image in the shortcode.</p> 
        <pre><code>[emd_donation email="abc123@hotmail.com" image="http://example.com/wp-content/uploads/images/donate.jpg"]
</code></pre> 
        <h4>Digital Download Tracking</h4> 
        <p>Easy Media Download is integrated with the <a href="https://wordpress.org/plugins/simple-universal-google-analytics/">Google Analytics</a> plugin. It allows you to see how many times a particular file has been downloaded in real time from your Google Analytics account.</p> 
        <h4>Languages</h4> 
        <p>Easy Media Download can be translated into your languages. Currently it supports the following languages:</p> 
        <ul> 
         <li>French</li> 
         <li>Spanish</li> 
         <li>German</li> 
         <li>Italian</li> 
         <li>Dutch</li> 
         <li>Turkish</li> 
        </ul> 
        <p>If you are a non-English speaker please help <a href="https://translate.wordpress.org/projects/wp-plugins/easy-media-download" rel="nofollow">translate Easy Media Download</a> into your language.</p>
