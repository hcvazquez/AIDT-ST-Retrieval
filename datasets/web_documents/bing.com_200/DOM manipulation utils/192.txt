    <tr> 
     <td colspan="5" width="696"> <p align="LEFT"><font face="Verdana"><br><br>Here are some ports of common <a href="http://www.fsf.org/">GNU</a> utilities to native Win32. In this context, native means the executables do only depend on the Microsoft C-runtime (msvcrt.dll) and not an emulation layer like that provided by <a href="http://www.cygwin.com/">Cygwin</a> tools.</font></p> <p align="LEFT"><strong><font face="Verdana">Download:</font></strong></p> 
      <blockquote style="text-align: left">
       <font face="Verdana">by http: <a href="UnxUtils.zip">UnxUtils.zip</a><br><br>by ftp: no FTP mirror at the moment</font>
      </blockquote> <p align="LEFT"><font face="Verdana">latest updates (after 14-04-03):&nbsp; <a href="UnxUpdates.zip">UnxUpdates.zip</a> </font> </p> <p><strong><font face="Verdana">Source code:</font></strong></p> 
      <blockquote style="text-align: left">
       <font face="Verdana">I have started an Open source project at <a href="http://sourceforge.net/projects/unxutils">http://unxutils.sourceforge.net</a>. There is a &nbsp; <a href="#cvs">CVS</a> repository of all sources, even if the changes to the original GNU code are trivial. The repository can be accessed via anonymous CVS with the command<br></font>
       <font size="2"><font face="Courier New">cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/unxutils co unxutils<br>Note: The repository is not up-to-date at the moment, but will recover soon.</font></font>
      </blockquote> 
      <hr> <p align="LEFT"><font face="Verdana"><font size="2">The following program versions are covered:</font></font></p> 
      <blockquote style="text-align: left">
       <font face="Verdana"><font size="1">bc-1.05<br>bison-1.28<br>bzip2-1.0.2<br>diffutils-2.7<br>fileutils-3.16<br>findutils-4.1<br>flex-2.5.4<br>gawk-3.1.0<br>grep-2.4.2<br>gsar110<br>gzip-1.2.4<br>indent-2.2.9<br>jwhois-2.4.1<br>less-340<br>m4-1.4<br>make-3.78.1<br>patch-2.5<br>recode-3.6<br>rman-3.0.7<br>sed-3.02<br>shellutils-1.9.4<br>tar-1.12<br>textutils-2.1</font><br><font size="1">unrar-3.00<br>wget-1.8.2<br>which-2.4</font></font>
