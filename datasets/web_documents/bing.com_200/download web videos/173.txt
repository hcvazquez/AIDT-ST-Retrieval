             <div>
              <span style="color: rgb(51, 51, 51); font-size: 30px">Introduction</span>
             </div> 
             <p>The project in this, how to use OpenCV with Java under Eclipse IDE platform. This will help developers who love Java and OpenCV environment. The application is totally made in Eclipse INDIGO version environment. The application shows how to use the OpenCV with Eclipse IDE.</p> 
             <p>In this application, we see how to capture video from webcam. This application is totally a demonstration for how to create applications in Eclipse INDIGO. In this, I explain how to configure the Eclipse INDIGO IDE in order to execute OpenCV applications.</p> 
             <h2>Eclipse</h2> 
             <p>Eclipse is free and an Open source software. Eclipse is multi-language software development environment. Eclipse is IDE, i.e., Integrated Development Environment. Eclipse can be used to develop an application in java. Eclipse is mostly used for JAVA. It contains workspace &amp; plug-ins in system for customizing the environment. By means of various plug-ins, other programming languages including Ada, C, C++, COBOL, Perl, PHP, Python, Ruby (including Ruby on Rails framework), Scala, and Scheme. The IDE is often called Eclipse ADT for Ada, Eclipse CDT for C/C++, Eclipse JDT for Java, and Eclipse PDT for PHP.</p> 
             <h2>Eclipse: How To Start Work?</h2> 
             <h4>Video </h4> 
             <iframe style="width:560px; height:315px" src="https://www.youtube.com/embed/jVyWevgrb9k" allowfullscreen="true"></iframe> 
             <p>Start with Eclipse INDIGO IDE: Configuring</p> 
             <p>The following steps show how to configure the Eclipse INDIGO version with OpenCV 2.4.8 version. First of all, create the project. Then select the properties from Project Menu. Then follow the steps listed below.</p> 
             <h3>Step 1: Add External JARs File</h3> 
             <p>Select Java Build Path, Select Libraries, Click on "Add External JARs". find <em>opencv-248.jar</em> file in "<em>C:\opencv\build\java</em>", select file, click open.</p> 
             <p>Properties&gt;&gt;Select Java Build Path&gt;&gt; Select Libraries &gt;&gt; Click Add External JARs &gt;&gt; Select File "C:\opencv\build\java\opencv-248.jar" &gt;&gt; Click Ok.</p> 
             <p style="text-align: center"><img height="325" src="/KB/java/719878/Step1.png" width="470"></p> 
             <h3>Step 2: Add Native Library</h3> 
             <p>Select "Native Library Location" then click on "Edit" Button. put Location path "<em>C:\opencv\build\java\x86</em>"</p> 
             <p>Click OK.</p> 
             <p>Select Native Library Location &gt;&gt; Edit &gt;&gt; Location path ="C:\opencv\build\java\x86" &gt;&gt; OK.</p> 
             <p style="text-align: center"><img height="325" src="/KB/java/719878/Step2.png" width="470"></p> 
             <h3>Step 3: Finish</h3> 
             <p>Finally, click on OK button to add in project.</p> 
             <p style="text-align: center"><img height="325" src="/KB/java/719878/Step3.png" width="470"></p> 
             <h3>Package Explorer</h3> 
             <p>Finally all details are seen in Package Explorer. Default Package contains the <em>VideoCap.java</em> file, JRE System Library containing the all JavaSE packages and Reference libraries containing the opencv-2.4.8 package.</p> 
             <p style="text-align: center"><img height="473" src="/KB/java/719878/Program.png" style="text-align: center" width="263"></p> 
             <h2>Using the Code</h2> 
             <p>The following sample code is for web camera video capture. Below, I will be explaining the program with details, that how the program is executing.</p> 
             <pre lang="C++"><span class="code-comment">//</span><span class="code-comment"></span>
<span class="code-comment">//</span><span class="code-comment"> sample program to run in eclipse.</span>
<span class="code-comment">//</span><span class="code-comment"></span></pre> 
             <pre lang="C++">import org.opencv.core.*;
import org.opencv.highgui.Highgui;        
import org.opencv.highgui.VideoCapture;        
        
<span class="code-keyword">public</span> <span class="code-keyword">class</span> VideoCap {
    <span class="code-keyword">public</span> <span class="code-keyword">static</span> <span class="code-keyword">void</span> main (<span class="code-sdkkeyword">String</span> args[]){
    	System.loadLibrary(Core.NATIVE_LIBRARY_NAME);
    	VideoCapture camera = <span class="code-keyword">new</span> VideoCapture(<span class="code-digit">0</span>);
    	
    	<span class="code-keyword">if</span>(!camera.isOpened()){
    		System.out.println(<span class="code-string">"</span><span class="code-string">Error"</span>);
    	}
    	<span class="code-keyword">else</span> {
    		Mat frame = <span class="code-keyword">new</span> Mat();
    	    <span class="code-keyword">while</span>(<span class="code-keyword">true</span>){
    	    	<span class="code-keyword">if</span> (camera.read(frame)){
    	    		System.out.println(<span class="code-string">"</span><span class="code-string">Frame Obtained"</span>);
    	    		System.out.println(<span class="code-string">"</span><span class="code-string">Captured Frame Width "</span> + 
    	    		frame.width() + <span class="code-string">"</span><span class="code-string"> Height "</span> + frame.height());
    	    		Highgui.imwrite(<span class="code-string">"</span><span class="code-string">camera.jpg"</span>, frame);
    	    		System.out.println(<span class="code-string">"</span><span class="code-string">OK"</span>);
    	    		<span class="code-keyword">break</span>;
    	    	}
    	    }	
    	}
    	camera.release();
    }
}   </pre> 
             <p>First of all, import the respective package in application. Following are the basic packages for implementing video capture from webcam or video file.</p> 
             <ul> 
              <li><code><strong>Core</strong></code>: <code>Core </code>package includes having basic data structures, including multi dimensional array Mat and basic function used by modules.</li> 
              <li><code><strong>Highgui</strong></code>: Easy to use interface to images, videocapture, video codec, and simple GUI capabilities.</li> 
              <li><code><strong>VideoCapture</strong></code>: <code>VideoCapture </code>subpackage of <code>Highgui</code>. <code>VideoCapture </code>is responsible for reading video from webcam or video file. It is also responsible for setting or getting video properties like frame height, frame width.</li> 
             </ul> 
             <pre lang="C++">import org.opencv.core.*;
import org.opencv.highgui.Highgui;        
import org.opencv.highgui.VideoCapture;  </pre> 
             <p>Secondly, we need to load the native library. This command must be executed once per Java process to using native <code>OpenCV </code>methods. If it does not get used, it throws an exception like "<code>UnsatisfiedLink </code>errors."</p> 
             <pre lang="C++">System.loadLibrary(Core.NATIVE_LIBRARY_NAME); </pre> 
             <p>The following statement states that the camera variable is initialized with VideoCapture with 0 "Zero" parameter. It means that zero is first device ID. It accesses the first web camera. Video Camera is also initialized with video file, just pass video file path.</p> 
             <pre lang="C++">VideoCapture camera = <span class="code-keyword">new</span> VideoCapture(<span class="code-digit">0</span>); </pre> 
             <p style="text-align: center">OR</p> 
             <pre lang="C++">VideoCapture camera = <span class="code-keyword">new</span> VideoCapture(<span class="code-string">"</span><span class="code-string">F\\video.avi"</span>); </pre> 
             <p><code>isOpned </code>function is used to check if the binding of the class to a video source that means device index or video source file path was successful or not to use.</p> 
             <pre lang="C++"><span class="code-keyword">if</span>(!camera.isOpened()){
    		System.out.println(<span class="code-string">"</span><span class="code-string">Error"</span>);
    	} </pre> 
             <p><code>Mat </code>class with two data parts, first the <strong>matrix header</strong> which contains information such as the size of the matrix and second a <strong>pointer to the matrix</strong> contains the pixel values. The matrix header size is constant.</p> 
             <pre lang="C++">Mat frame = <span class="code-keyword">new</span> Mat();</pre> 
             <p>Following is an infinite loop till the frame is not obtained. If frame has been obtained, it breaks the loop. The following loop prints the frame height and width, it also writes the frame to file and break the loop.</p> 
             <pre lang="C++"><span class="code-keyword">while</span>(<span class="code-keyword">true</span>){
            <span class="code-keyword">if</span> (camera.read(frame)){
                System.out.println(<span class="code-string">"</span><span class="code-string">Frame Obtained"</span>);
                System.out.println(<span class="code-string">"</span><span class="code-string">Captured Frame Width "</span> +
                frame.width() + <span class="code-string">"</span><span class="code-string"> Height "</span> + frame.height());
                Highgui.imwrite(<span class="code-string">"</span><span class="code-string">camera.jpg"</span>, frame);
                System.out.println(<span class="code-string">"</span><span class="code-string">OK"</span>);
                <span class="code-keyword">break</span>;
            }
        }
</pre> 
             <p><code>imwrite </code>is a <code>static </code>method from the <code>Highgui </code>class. This method is used to write an image to a file. In the following frame variable is written to <em>camera.jpg</em> file.</p> 
             <p>&nbsp;</p> 
             <pre lang="C++">Highgui.imwrite(<span class="code-string">"</span><span class="code-string">camera.jpg"</span>, frame); </pre> 
             <p>For closing the video, we need to call its <code>release </code>function.</p> 
             <pre lang="C++">camera.release();</pre>
