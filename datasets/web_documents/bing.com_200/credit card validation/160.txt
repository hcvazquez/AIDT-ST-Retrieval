        <h2>Step 1: Create Dynamic Web project</h2> 
        <p> Open Eclipse and go to<span style="color: #000080;"> File -&gt; New -&gt; Project </span> and select<span style="color: #000080;"> Dynamic Web Project </span>in the New Project wizard screen. </p> 
        <p> <img class="img center" width="450" height="430" title="JSF Example" alt="JSF Example" src="../../images/jsf/jsf_app_in_eclipse.png"> </p> 
        <br> 
        <p> Provide the name of the project as<span style="color: #000080;"> JSFINBUILTVALIDATION </span>. Once this is done, select the target runtime environment as <span style="color: #000080;"> Apache Tomcat v6.0</span>. This is to run the project inside Eclipse environment. In configuration select<span style="color: #000080;"> JavaServer Faces v1.2 Project </span>and press<span style="color: #000080;"> Next -&gt; Next -&gt; Next</span>. </p> 
        <p> <img class="img center" title="JSF validation Example" alt="JSF validation Example" src="../../images/jsf/jsf_app_in_eclipse1.png"> </p> 
        <p></p> 
        <p> In next screen select Disable Library Configuration from drop down and click Finish. </p> 
        <p></p> 
        <p> <img class="img center" title="JSF validation Example" alt="JSF validation Example" src="../../images/jsf/jsf_app_in_eclipse2.png"> </p> 
        <br> 
        <h2>Step 2: Add jar files to build path of the project</h2> 
        <p> Copy all the required jar files shown bellow to the lib folder of the project directory and add all jar files to the build path of the project. </p> 
        <p> <img class="img center" title="JSF validation Example" alt="JSF validation Example" src="../../images/jsf/jsf_database_example1.png"> </p> 
        <br> 
        <h2>Step 3: Create JSP files</h2> 
        <p> Create two JSP files:<span style="color: #000080;"> index.jsp </span> and <span style="color: #000080;"> result.jsp </span> in WebContent folder.The login.jsp prompts the user to enter his/her credit card details.After successful validation user will be redirected to<span style="color: #000080;"> result.jsp </span> page.In case of any data validation error message will be shown in the red color and it will ask to correct the data for form submission. </p> 
        <h3>index.jsp</h3> 
        <div class="code"> 
         <pre class="brush: java;">
&lt;%@ page contentType="text/html"%&gt;
&lt;%@ taglib uri="http://java.sun.com/jsf/core" prefix="f"%&gt;
&lt;%@ taglib uri="http://java.sun.com/jsf/html" prefix="h"%&gt;

&lt;f:view&gt;
	&lt;html&gt;
&lt;head&gt;
&lt;link href="css/style.css" rel="stylesheet" type="text/css" /&gt;
&lt;title&gt;&lt;h:outputText value="#{msg.title}" /&gt;&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
	&lt;h:form&gt;
		&lt;h1&gt;
			&lt;h:outputText value="#{msg.card_details}" /&gt;
		&lt;/h1&gt;
		&lt;h:panelGrid columns="3"&gt;
			&lt;h:outputText value="#{msg.amount}" /&gt;
			&lt;h:inputText id="amount" label="#{msg.amount}"
				value="#{payment.amount}" required="true"&gt;
				&lt;f:convertNumber minFractionDigits="2" /&gt;
				&lt;f:validateDoubleRange minimum="100" maximum="50000" /&gt;
			&lt;/h:inputText&gt;
			&lt;h:message for="amount" styleClass="errormsg" /&gt;

			&lt;h:outputText value="#{msg.creditCard}" /&gt;
			&lt;h:inputText id="card" label="#{msg.creditCard}"
				value="#{payment.card}" required="true"&gt;
				&lt;f:validateLength minimum="16" /&gt;
				&lt;f:attribute name="requiredMessage" value="#{msg.card_required}" /&gt;
			&lt;/h:inputText&gt;
			&lt;h:message for="card" styleClass="errormsg" /&gt;

			&lt;h:outputText value="#{msg.expiry_date}" /&gt;
			&lt;h:inputText id="date" label="#{msg.expiry_date}"
				value="#{payment.date}" required="true"&gt;
				&lt;f:convertDateTime pattern="MM/yyyy" /&gt;
			&lt;/h:inputText&gt;
			&lt;h:message for="date" styleClass="errormsg" /&gt;

			&lt;h:outputText value="#{msg.cvv_no}" /&gt;
			&lt;h:inputText id="cvv" label="#{msg.cvv_no}" value="#{payment.cvvNo}"
				required="true"&gt;
				&lt;f:validateLength minimum="3" /&gt;

        <h2>Step 5 : Create CSS File :</h2>

        <div class="code">

.errormsg {
   color:red; 
}

        <h2>Step 6 : Create JSF Managed Bean</h2> 
        <p> A Managed Bean is a regular Java Bean class, registered with JSF. In other words, Managed Bean is a java bean, managed by the JSF framework.From JSF 2.0 and onwards, we can declare a managed bean, just by using the annotation @ManagedBean. </p> 
        <p> Create a java class<span style="color: #000080;"> PaymentBean </span> inside<span style="color: #000080;"> "com.javawebtutor.jsf.bean" </span> package and add bellow line of code into this class. </p> 
        <h3>EmployeeBean.java</h3> 
        <div class="code"> 
         <pre class="brush: java;">
package com.javawebtutor.jsf.bean;

import java.util.Date;

public class PaymentBean {
	private double amount;
	private String card;
	private Date date = new Date();
	private String cvvNo;

	public double getAmount() {
		return amount;
	}

	public void setAmount(double amount) {
		this.amount = amount;
	}

	public String getCard() {
		return card;
	}

	public void setCard(String card) {
		this.card = card;
	}

	public Date getDate() {
		return date;
	}

	public void setDate(Date date) {
		this.date = date;
	}

	public String getCvvNo() {
		return cvvNo;
	}

	public void setCvvNo(String cvvNo) {
		this.cvvNo = cvvNo;
	}

}

</pre> 
        </div> 
        <br> 
        <h2>Step 7 : Update Configuration File(faces-config.xml)</h2> 
        <p> Jsf looks for<span style="color: #000080;"> faces-config.xml </span> in classpath and loads the configuration defined from here. Here we have defined managed bean (the backing bean), and navigation rule.Open<span style="color: #000080;"> faces-config.xml </span> from <span style="color: #000080;"> WebContent -&gt; WEB-INF </span>folder and following code into this file. </p> 
        <h3>faces-config.xml</h3> 
        <div class="code"> 
         <pre class="brush: xml;">
&lt;?xml version="1.0"?&gt;
&lt;faces-config xmlns="http://java.sun.com/xml/ns/javaee"
	xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
	xsi:schemaLocation="http://java.sun.com/xml/ns/javaee 
        http://java.sun.com/xml/ns/javaee/web-facesconfig_1_2.xsd"
	version="1.2"&gt;
	&lt;navigation-rule&gt;
		&lt;from-view-id&gt;/index.jsp&lt;/from-view-id&gt;
		&lt;navigation-case&gt;
			&lt;from-outcome&gt;submit&lt;/from-outcome&gt;
			&lt;to-view-id&gt;/result.jsp&lt;/to-view-id&gt;
		&lt;/navigation-case&gt;

	&lt;/navigation-rule&gt;

	&lt;navigation-rule&gt;
		&lt;navigation-case&gt;
			&lt;from-outcome&gt;back&lt;/from-outcome&gt;
			&lt;to-view-id&gt;/index.jsp&lt;/to-view-id&gt;
		&lt;/navigation-case&gt;
	&lt;/navigation-rule&gt;

	&lt;managed-bean&gt;
		&lt;managed-bean-name&gt;payment&lt;/managed-bean-name&gt;
		&lt;managed-bean-class&gt;com.javawebtutor.jsf.bean.PaymentBean&lt;/managed-bean-class&gt;
		&lt;managed-bean-scope&gt;session&lt;/managed-bean-scope&gt;
	&lt;/managed-bean&gt;

	&lt;application&gt;
		&lt;resource-bundle&gt;
			&lt;base-name&gt;com.javawebtutor.jsf.properties.messages&lt;/base-name&gt;
			&lt;var&gt;msg&lt;/var&gt;
		&lt;/resource-bundle&gt;
	&lt;/application&gt;
&lt;/faces-config&gt;

</pre> 
        </div>
