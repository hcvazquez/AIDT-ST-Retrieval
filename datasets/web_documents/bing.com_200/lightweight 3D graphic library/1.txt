            <h1>Voxel</h1> 
            <p>The goal of this project is to provide a lightweight 3D CSS library with very simple implementation.</p> 
            <p><a href="https://github.com/HunterLarco/voxel.css"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-1.jpg" dummy" class="aligncenter" alt=""></a></p> 
            <h1>Three.js</h1> 
            <p>The aim of the project is to create a lightweight 3D library with a very low level of complexity — in other words, for dummies. The library provides &lt;canvas&gt;, &lt;svg&gt;, CSS3D and WebGL renderers.</p> 
            <p><a href="http://threejs.org/"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-2.jpg" dummy" class="aligncenter" alt=""></a></p> 
            <h1>Photon</h1> 
            <p>Photon is a JavaScript library that adds simple lighting effects to DOM elements in 3D space. It’s rather processor-intensive, so please use responsibly.</p> 
            <p><a href="http://photon.attasi.com/"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-3.jpg" dummy" class="aligncenter" alt=""></a></p> 
            <h1>svg-mesh-3d</h1> 
            <p>A high-level module to convert a SVG &lt;path&gt; string into a 3D triangulated mesh. Best suited for silhouettes, like font icon SVGs.</p> 
            <p><a href="https://github.com/mattdesl/svg-mesh-3d"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-4.jpg" dummy" class="aligncenter" alt="svg-mesh-3d"></a></p> 
            <h1>Sprite3D.js</h1> 
            <p>Sprite3D.js makes it easy to manipulate HTML elements in 3D space through the use of CSS 3D transforms. It allows to control the position, rotation and scale of the element using simple, chainable methods. A Sprite3D object is an HTML element, so you can freely use native browser events and properties, and control their appearance through CSS directives.</p> 
            <p><a href="https://github.com/boblemarin/Sprite3D.js"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-5.jpg" dummy" class="aligncenter" alt="Sprite3D.js"></a></p> 
            <h1>interactive_3d</h1> 
            <p>jQuery Interactive 3D let you create a 3D interactive object using several images as frames allowing the user to drag around and manipulate the image. This plugin is perfect for showcasing your product on the web.</p> 
            <p><a href="https://github.com/peachananr/interactive_3d"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-6.jpg" dummy" class="aligncenter" alt="interactive_3d"></a></p> 
            <h1>Phoria.js</h1> 
            <p>JavaScript library for simple 3D graphics and visualisation on a HTML5 canvas 2D renderer. It does not use WebGL. Works on all HTML5 browsers, including desktop, iOS and Android.</p> 
            <p><a href="https://github.com/kevinroast/phoria.js"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-7.jpg" dummy" class="aligncenter" alt="Phoria.js"></a></p> 
            <h1>DivSugar</h1> 
            <p>DivSugar is a CSS-based 3D graphics library, which provides a 3D scene graph, an animation system, and geometry classes. Each node of the scene graph is an extended div element, so it is very easy to integrate 3D animations into existing web pages.</p> 
            <p><a href="https://github.com/kitao/divsugar"><img src="http://bashooka.com/wp-content/uploads/2016/01/javascript-3d-libraries-2016-8.jpg" dummy" class="aligncenter" alt="DivSugar"></a></p>
