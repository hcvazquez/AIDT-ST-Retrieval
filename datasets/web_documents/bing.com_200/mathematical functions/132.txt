    <h1>Java random numbers</h1> 
    <p> Random numbers from very low to very high ranges can be generated with Java. </p> 
    <!--consider revising intro paragraph--> 
    <p class="tutSpec"> This tutorial focuses on: </p> 
    <ul class="tutOutline"> 
     <li><span class="outline">The random() method</span></li> 
     <li><span class="outline">The Random class</span></li> 
     <li><span class="outline">The psuedorandom process</span></li> 
     <li><span class="outline">Setting a seed</span></li> 
    </ul> 
    <h2>The random() method</h2> 
    <p> Generate random numbers using the random() method of the Math class. Random numbers returned will be of data type double from 0.0(inclusive) to 1.0(exclusive) - meaning that the random number generated will be greater than or equal to 0.0 and less than 1.0. </p> 
    <div class="code">
     <span class="example">Generating a random number:</span>
     <div class="theCode">
      double randNumber = Math.random(); System.out.print(randNumber);
     </div>
    </div> 
    <p> Why limit yourself to such a low range as between 0.0 and 1.0? You can actually return random numbers in any range. To do this multiply the returned value from a random() function by a number that when added to the lowest number in your range will produce the highest range. If this sounds confusing, not to worry, here is an example: </p> 
    <div class="justCode">
     //will return a random number between 40.0(inclusive) and 90.0(exclusive) 
     <span class="highlight">double randNumber = Math.random() * 50 + 40;</span> /* The random() method naturally returns a random number between 0.0 and 1.0. That value is then multiplied by 50. The difference between the lower and upper limit now ranges from 0.0. to 50.0. When 40 is added to that value, the range becomes 40.0 to 90.0. */
    </div> 
    <p> The random() method returns decimal numbers, but you can also return whole numbers. This is accomplished by data type casting the value returned by the Math.random() method to data type int. </p> 
    <div class="code">
     <span class="example">Example:</span>
     <div class="theCode">
      //return a random number between 10.0(inclusive) and 60.0(exclusive) //and cast it to a whole number int aNumber = 
      <span class="highlight">(int)</span> (Math.random() * 50 + 10);
     </div>
    </div> 
    <p> Placing the word int like that in parentheses before the random() function and also placing the random() function in parentheses will convert the value returned by the random() function from a double to a whole number. This process is known as <b>casting</b>. </p> 
    <h2>Using the Random class to generate random numbers</h2> 
    <p> The Random class is located in the <b>java.util</b> package. It can be used generate random numbers of different data types (double, float, int, long) as well as random booleans. </p> 
    <p class="eText"> Methods of the Random class: </p> 
    <ul> 
     <li> <span class="data">nextInt()</span> - returns a random number of data type int. </li> 
     <li> <span class="data">nextInt(int n)</span> - returns a random number of data type int ranging from 0(inclusive) to n(exclusive) </li> 
     <li> <span class="data">nextDouble()</span> - returns a random double ranging from 0.0(inclusive) to 1.0(exclusive) </li> 
     <li> <span class="data">nextBoolean()</span> - returns a random boolean (true/false value) </li> 
    </ul> 
    <p class="note"> <b>NOTE:</b> When you use Math.random() to generate a random double, it actually creates a Random object and calls the nextDouble() method. </p> 
    <div class="code">
     <span class="example">A bunch of randomness using the Random class:</span>
     <div class="theCode">
      import java.util.Random; class GenerateRandomValues{ public static void main(String[] args){ Random aRandom = new Random(); System.out.println(aRandom.nextInt()); //print a random number between 1 and 10 System.out.println(aRandom.nextInt(10) + 1); System.out.println(aRandom.nextDouble()); System.out.println(aRandom.nextBoolean()); } }
     </div>
    </div> 
    <div class="output">
     <span class="head">Output:</span>
     <div class="theOutput">
      235659484 9 0.8259070568528002 true
     </div>
    </div> 
    <h2>The psuedorandom process</h2> 
    <p> It seems that these random numbers (and other random values) are really random, but they are not. The Random class actually generates pseudorandom numbers - numbers generated in a non random way, but in a way that makes them look random. This is accomplished by the Random class taking an initial value (called a seed) and through a specific algorithm generating another value based on the seed. </p> 
    <p> By default, the seed given to a Random object is the value of the system clock in milliseconds (which is your computer's interpretation of the current time). Since this is constantly changing, the seed is always changing and therefore a different "random" number is always being generated. </p> 
    <p> If you set the seed yourself to a fixed number, then the "random" numbers generated will always be the same like in this example below. </p> 
    <div class="code">
     <span class="example">Setting the seed to a fixed number:</span>
     <div class="theCode">
      import java.util.Random; class GenerateRandomNumbers{ public static void main(String[] args){ Random aRandom = new Random(); //set the seed of aRandom to 1015 
      <span class="highlight">aRandom.setSeed(1015);</span> //the number generated will not change //since the seed is always the same System.out.println(aRandom.nextInt()); } }
     </div>
