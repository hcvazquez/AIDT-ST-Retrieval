       <p>In a recent <a href="https://channel9.msdn.com/Shows/Visual-Studio-Toolbox/Cordova-Updates-in-Visual-Studio">Visual Studio Toolbox episode</a>, I highlighted some new Ionic 2 templates for use with the Visual Studio 2015 Tools for Apache Cordova (TACO). These <a href="https://marketplace.visualstudio.com/vsgallery/b21470f5-d29f-44a1-8f4b-8a3f96ac5860">new Ionic 2 RC templates</a> are now available for you to try out and in this post I’ll talk you through what’s new.</p> 
       <p>&gt; <a href="https://marketplace.visualstudio.com/vsgallery/b21470f5-d29f-44a1-8f4b-8a3f96ac5860">Download Ionic 2 templates on the Visual Studio Extension Gallery</a>.</p> 
       <h1>What is Ionic?</h1> 
       <p>Ionic is a JavaScript framework that helps you build apps for Android, iOS, and Windows 10 devices using the design patterns of the native operating system. Ionic builds on top of <a href="http://cordova.io">Apache Cordova</a>, an open source framework that helps you build mobile apps using HTML, JavaScript, and CSS. These mobile apps have access to native device capabilities, like the camera or accelerometer, and can be published through public or private app stores just like any native mobile app.</p> 
       <p>Ionic provides HTML-based controls that go beyond the primitive UI of HTML; giving you a set of ready-made UI controls that are optimized for mobile devices. If you need to work with tabs, lists, navigation bars – Ionic has those for you. If you want micro-interactions, animations and buttery smooth scrolling – Ionic has those for you, too.</p> 
       <p><a href="https://msdnshared.blob.core.windows.net/media/2016/12/Ionic-2-UI-Overview1.gif"><img width="640" height="377" class="alignnone wp-image-9555 size-full" alt="ionic-2-ui-overview" src="https://msdnshared.blob.core.windows.net/media/2016/12/Ionic-2-UI-Overview1.gif"></a></p> 
       <p>Examples of how the Ionic UI controls adapt to each device. Taken from the <a href="https://github.com/microsoft/ionic-azure-conference-app">Ionic Azure Conference app</a> demo application.</p> 
       <p>Ionic has been around since 2012 and over 3,000,000 apps have been built using it. Ionic 1 was built using Angular 1 and released in 2014. The Ionic team is now nearing the first release of Ionic 2, built on top of Angular 2.</p> 
       <h1>How do I build apps with Ionic?</h1> 
       <p>When you build an app with Ionic 2, you’re building an app using <a href="http://angular.io">Angular 2</a>. This means that you’ll use <a href="http://typescriptlang.org">TypeScript</a> as your programming language of choice, which brings several benefits. You be able to:</p> 
       <ul> 
        <li><b>Use future JavaScript features (like classes and modules) today</b> instead of waiting for your target mobile platforms to support them. TypeScript compiles to simple, human-readable, JavaScript.</li> 
        <li><b>Commit code changes with higher confidence</b>, thanks to type checking and refactoring tools that make sure your code is working the way you expect before and after a change.</li> 
        <li><b>Investigate bugs, or new code bases, more quickly</b> thanks to high-confidence navigation tools like Find References or Peek.</li> 
       </ul> 
       <p>You’ll also work with <a href="http://sass-lang.com/">Sass</a>, a language to define styling for your apps. Sass provides valuable features for CSS development, like support for variables, and it compiles down to simple CSS files.</p> 
       <p>When building your Ionic 2 application, you focus on building self-contained components which combine HTML &amp; CSS templates for presentation with JavaScript/TypeScript for component logic. These components are then combined into pages – i.e. the individual screens of your application.</p> 
       <p>For example, if you look at this default page for the Ionic 2 Blank template:</p> 
       <p><a href="https://msdnshared.blob.core.windows.net/media/2016/12/ionic2blankTemplate.png"><img width="162" height="300" class="alignnone wp-image-9545 size-medium" style="border: solid 1px black" alt="ionic2blanktemplate" src="https://msdnshared.blob.core.windows.net/media/2016/12/ionic2blankTemplate-162x300.png"></a></p>

       <p>When you build your project, Ionic takes care of compiling the TypeScript (.ts) files to JavaScript (.js) files, and Sass (.scss) files to CSS. It also places all the compiled files in the right places for Cordova to build your application and run it on a mobile device.</p> 
       <p>Try out the <a href="http://taco.visualstudio.com/en-us/docs/tutorial-ionic2/">Ionic 2 tutorial</a> on the TACO documentation site, to experience it for yourself.</p> 
       <h1>What’s new for Ionic 2 &amp; TACO?</h1> 
       <p>Our <a href="http://go.microsoft.com/fwlink/?LinkId=398477">Tools for Apache Cordova</a> (TACO) team, has been working closely with the Ionic team for years to make sure that Visual Studio developers have a first-class experience when using the Ionic framework and the TypeScript language,. We previously released templates for beta versions of Ionic 2, and now we’ve refreshed these for the latest Ionic 2 RC 4 release.</p> 
       <p>The new templates mirror the standard starter templates included in Ionic:</p> 
       <ul> 
        <li>Blank</li> 
        <li>SideMenu</li> 
        <li>Tabs</li> 
       </ul> 
       <p>Each of these templates adapts to match the look and feel of the device you’re using, as you can see in this following comparison of the SideMenu template:</p> 
       <p><a href="https://msdnshared.blob.core.windows.net/media/2016/12/ionicStarterTemplatePlatformComparison.png"><img width="879" height="523" class="alignnone wp-image-9535 size-large" alt="ionicstartertemplateplatformcomparison" src="https://msdnshared.blob.core.windows.net/media/2016/12/ionicStarterTemplatePlatformComparison-1024x609.png"></a></p> 
       <h1>Getting started with Ionic 2 RC templates</h1> 
       <p>You can use these templates with Visual Studio 2015; Visual Studio 2017 RC isn’t supported yet, but we’re working on it. To use these with Visual Studio 2015, there are a few additional extensions you need to install:</p> 
       <ul> 
        <li><a href="https://visualstudiogallery.msdn.microsoft.com/8879fc17-5651-4d30-be02-20138eb38ae2">TypeScript 2.0.6 editor</a> – Adds support for version 2 of the TypeScript language, used by Ionic.</li> 
        <li><a href="https://visualstudiogallery.msdn.microsoft.com/c94a02e9-f2e9-4bad-a952-a63a967e3935">Microsoft ASP.NET and Web Tools</a> – Among other changes for ASP.NET &amp; web development, this adds support for new versions of NPM and the task runner explorer, used by the Ionic templates.</li> 
        <li><a href="https://visualstudiogallery.msdn.microsoft.com/8f2f2cbc-4da5-43ba-9de2-c9d08ade4941">NPM Task Runner</a> – Adds NPM script support to the Visual Studio task runner explorer, used to perform builds in Ionic.</li> 
       </ul> 
       <p>Finally, you’ll install the new <a href="https://marketplace.visualstudio.com/vsgallery/b21470f5-d29f-44a1-8f4b-8a3f96ac5860">Ionic 2 RC template extension</a> and can follow along with the <a href="http://taco.visualstudio.com/en-us/docs/tutorial-ionic2/">Ionic 2 tutorial</a> in the TACO documentation.</p> 
       <h1>Join our Insiders and share your thoughts</h1> 
       <p>After you <a href="https://marketplace.visualstudio.com/vsgallery/b21470f5-d29f-44a1-8f4b-8a3f96ac5860">try out the Ionic 2 RC templates</a>, please let us know what you think! For early access to future Ionic &amp; Cordova related updates, <a href="https://aka.ms/qfggm3">join our TACO insiders</a>.</p> 
       <p>You can also share feedback via the <a href="https://docs.microsoft.com/en-us/visualstudio/ide/how-to-report-a-problem-with-visual-studio-2015">Report a Problem</a> option in the upper right corner of the Visual Studio IDE itself and track your feedback on the <a href="https://developercommunity.visualstudio.com/index.html">developer community portal</a>. If you have a question about Ionic development, check out the <a href="https://forum.ionicframework.com/">Ionic Forum</a>.</p>
