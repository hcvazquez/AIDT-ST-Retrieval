                       <br> &lt;bo***********@gmail.comwrote in message

                       <blockquote class="post_quotes">
                         Dear all
                        <br>
                        <br> In my ASP page, user can enter date in dd/mm/yyyy format. How can I use
                        <br> Javascript to convert dd/mm/yyyy to yyyy-mm-dd hh:mm:ss.
                        <br>
                        <br> Please give me some advices. 
                       </blockquote> Assuming that the date format as dd/mm/yyyy you can easily convert it to a
                       <br> date by splitting it into date parts:
                       <br> 
                       <br> myDateParts = myDate.split("/");
                       <br> 
                       <br> You can then build a new date from the parts like so:
                       <br> 
                       <br> myJSDate = new Date(myDateParts[2], myDateParts[1], myDateParts[0]);
                       <br> 
                       <br> It should be clear that you could also just output the date as you want it
                       <br> using the same array but I always prefer generating a real date. Note that
                       <br> you're not specifying a time so time will always be "00:00:00".
                       <br> 
                       <br> Once you have a date you can output it however you like. To make that
                       <br> easier I've a date extensions library that might help:
                       <br> 
                       <br> 
                       <a rel="nofollow" href="http://www.depressedpress.com/Content/Development/JavaScript/Extensions/DP_DateExtensions/Index.cfm" target="_blank">http://www.depressedpress.com/Conten...ions/Index.cfm</a>
                       <br> 
                       <br> Part of this is are "dateFormat()" and "timeFormat()" methods. You can use
                       <br> them to output the format you want like so:
                       <br> 
                       <br> myJSDate.formatDate("yyyy-mm-dd") + " " + myJSDate.timeFormat("hh:mm:ss")
                       <br> 
                       <br> The library is large and if you're just looking for a one-off then it's
                       <br> almost definately overkill, but if you want the freedom to change the format
                       <br> at whim then it makes things very easy.
                       <br> 
                       <br> Jim Davis
