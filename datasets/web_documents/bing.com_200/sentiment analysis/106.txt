          <p>With the volume of social media traffic that Twitter has each day, it has become a fantastic store of information. Utilizing Rstudio&nbsp;and a few handy R packages one can easily import tweets with specific keywords and conduct various analyzes on them.</p>

Head to&nbsp;<a href="http://apps.twitter.com">apps.twitter.com&nbsp;</a>&nbsp;and create a new application. Once done, be sure to create an access token for said application. You will then utilize the information for your Twitter app within R with the following code:</p>

          <p>Below is code that will create a word cloud based on the words showing up in the tweets with your keyword&nbsp;with the greatest frequency. I have annotated the code to give an explanation as to what various lines are doing. The tm library is used with various commands (i.e.,&nbsp;myCorpus &lt;- tm_map(myCorpus, removePunctuation)) to clean up the tweets and remove extraneous information such as URLs or punctuation.</p> 
          <p>More information on the tm and wordcloud packages can be found here:&nbsp;<a href="https://cran.r-project.org/web/packages/wordcloud/wordcloud.pdf" target="_blank">wordcloud package</a>&nbsp;<a href="https://mran.microsoft.com/web/packages/tm/tm.pdf" target="_blank">tm package</a></p> 
          <p>Now for the code:</p>

            <textarea wrap="soft" class="crayon-plain print-no" data-settings="dblclick" readonly style="-moz-tab-size:4; -o-tab-size:4; -webkit-tab-size:4; tab-size:4; font-size: 12px !important; line-height: 15px !important;">
library(twitteR)
library(wordcloud)
library(tm)

# retrieve the first 200 tweets (or all tweets if fewer than 200) from the
# user timeline of @rdatammining
tweets &lt;- userTimeline("weather", n=400)
nDocs &lt;- length(tweets)

##convert tweets to data frame
df &lt;- do.call("rbind", lapply(tweets, as.data.frame))
dim(df)

myCorpus &lt;- Corpus(VectorSource(df$text))
myCorpus &lt;- tm_map(myCorpus, removePunctuation)
myCorpus &lt;- tm_map(myCorpus, content_transformer(tolower))
myCorpus &lt;- tm_map(myCorpus, removeWords, stopwords("english")) #removes common prepositions and conjunctions 
myCorpus &lt;- tm_map(myCorpus, removeWords, c("example"))
removeURL &lt;- function(x) gsub("http[[:alnum:]]*", "", x)
myCorpus &lt;- tm_map(myCorpus, removeURL)
myCorpus &lt;- tm_map(myCorpus, stripWhitespace)
corpus_clean &lt;- tm_map(myCorpus, PlainTextDocument) ##this ensures the corpus transformations final output is a PTD
##optional, remove word stems (cleaning, cleaned, cleaner all would become clean): 
##wordCorpus &lt;- tm_map(wordCorpus, stemDocument)


#create term document matrix for analysis
myTdm &lt;- TermDocumentMatrix(corpus_clean, control=list(wordLengths=c(1,Inf)))
myTdm


##or create a word cloud from the corpus_clean data 
pal &lt;- brewer.pal(9,"YlGnBu")
pal &lt;- pal[-(1:4)]
set.seed(123)
wordcloud(words = corpus_clean, scale=c(4,0.5), max.words=50, random.order=FALSE, 
          rot.per=0.35, use.r.layout=TRUE, colors=pal)</textarea>

          <p>After a bit of manipulation of the data can be plotted using ggplot:</p> 
          <p><a href="http://www.andrewbonneau.com/wp-content/uploads/2016/03/weather_sentiment-analysis.png"><img class="alignnone size-medium wp-image-210" src="http://www.andrewbonneau.com/wp-content/uploads/2016/03/weather_sentiment-analysis-300x258.png" alt="weather sentiment analysis" width="300" height="258"></a>(click to enlarge)</p> 
          <p>Unsurprisingly, social media has a knack for producing generally negative and fearful posts regarding weather.</p> 
          <p>Taking these concepts further, one can break down twitter data into months, years, or even hours. For instance, you could look at twitter data regarding the weather during tornado season and compare it with the weather keyword during other months. Or, you could see how sentiment toward certain political candidates has changed over time.</p>
