           <h1 class="entry-title">15 Best JavaScript Animation Libraries for Developers</h1> 
           <a class="entry-comments" href="https://codegeekz.com/15-best-javascript-animation-libraries-for-developers/#comments"><i class="icon-comments-alt"></i><span class="dsq-postid" data-dsqidentifier="4531 http://codegeekz.com/?p=4531">4</span></a> 
           <ul class="entry-meta group"> 
            <li class="entry-author vcard author">by <span class="fn" style="display:inline"><a href="https://codegeekz.com/author/gavin/" title="Posts by Gavin" rel="author">Gavin</a></span></li> 
            <li class="category">In <a href="https://codegeekz.com/category/web-development-2/" rel="category tag">DEVELOPMENT</a> · <a href="https://codegeekz.com/category/javascript/" rel="category tag">Javascript</a> · <a href="https://codegeekz.com/category/jquery/" rel="category tag">JQuery</a> · <a href="https://codegeekz.com/category/resources/" rel="category tag">RESOURCES</a> · <a href="https://codegeekz.com/category/tuts/" rel="category tag">TUTORIALS</a></li>

           <p>The possibilities when it comes to performing various effects with jQuery are limitless and can be utilized to perform just about anything imaginable – and even unimaginable ( depending on one’s creativity and imagination ) with its simple and intuitive <a href="http://api.jquery.com/" target="_blank" rel="nofollow">API. </a></p> 
           <p>The innovative uses of jQuery can enable a plethora of animations within web design elements that can enhance the end user visual experience- and certainly go onto create stunning web effects which will have your end users wanting to come back for more of the magic which was once only created with Flash.</p> 
           <p>For this round-up, we would like to introduce some of these innovative uses and benefits of jQuery when it comes to animation effects for your projects, websites and apps.</p> 
           <h3>Enjoy !</h3> 
           <h2>1. Tween JS</h2> 
           <p><img class="alignnone size-full wp-image-4532" alt="tweenjs" src="http://codegeekz.com/wp-content/uploads/tweenjs.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/tweenjs.jpeg 625w, https://codegeekz.com/wp-content/uploads/tweenjs-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>TweenJS is a simple tweening library for use in Javascript. It was developed to integrate well with the EaselJS library, but is not dependent on or specific to it. It supports tweening of both numeric object properties &amp; CSS style properties. The API is simple but very powerful, making it easy to create complex tweens by chaining commands.</p> 
           <span class="button-wrap"><a href="http://www.createjs.com/#!/TweenJS" class="button medium" target="_blank">Tween JS</a></span> &nbsp;
           <p></p> 
           <h2>2. Jsanim</h2> 
           <p><img class="alignnone size-full wp-image-4533" alt="jsAnim" src="http://codegeekz.com/wp-content/uploads/jsAnim.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/jsAnim.jpeg 625w, https://codegeekz.com/wp-content/uploads/jsAnim-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>jsAnim was a clumbsy and poorly implemented library for adding mediocre animations to websites, sacrificing standards and accessibility. Weighing in at a whopping 25 kilobytes.</p> 
           <span class="button-wrap"><a href="http://jsanim.com/" class="button medium" target="_blank">Jsanim</a></span> &nbsp;
           <p></p> 
           <h2>3. Animo.js</h2> 
           <p><img class="alignnone size-full wp-image-4534" alt="animojs" src="http://codegeekz.com/wp-content/uploads/animojs.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/animojs.jpeg 625w, https://codegeekz.com/wp-content/uploads/animojs-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Animo.js is a powerful little tool for managing CSS animations. It includes features like Stack animations, create cross-browser blurring, set callbacks on animation completion.</p> 
           <span class="button-wrap"><a href="http://labs.bigroomstudios.com/libraries/animo-js" class="button medium" target="_blank"> Animo.js</a></span> &nbsp;
           <p></p> 
           <h2>4. Move.js</h2> 
           <p><img class="alignnone size-full wp-image-4535" alt="movejs" src="http://codegeekz.com/wp-content/uploads/movejs.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/movejs.jpeg 625w, https://codegeekz.com/wp-content/uploads/movejs-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Move.js is a small JavaScript library making CSS3 backed animation extremely simple and elegant.</p> 
           <span class="button-wrap"><a href="http://visionmedia.github.io/move.js/" class="button medium" target="_blank">Move.js</a></span> &nbsp;
           <p></p> 
           <h2>5. Collie</h2> 
           <p><img class="alignnone size-full wp-image-4536" alt="collie" src="http://codegeekz.com/wp-content/uploads/collie.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/collie.jpeg 625w, https://codegeekz.com/wp-content/uploads/collie-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Collie is a Javascript library that helps to create highly optimized animations and games using HTML 5. Collie runs on both PC and mobile using HTML 5 canvas and DOM. Collie can stably process multiple objects using rendering pipelines and supports useful features including sprite animation and user events. Stably supports iOS and Android, and renders with an optimized method for each platform. Also easily responds to retina display.</p> 
           <span class="button-wrap"><a href="http://jindo.dev.naver.com/collie/" class="button medium" target="_blank">Collie</a></span> &nbsp;
           <p></p> 
           <h2>6. Minified.js</h2> 
           <p><img class="alignnone size-full wp-image-4537" alt="minified" src="http://codegeekz.com/wp-content/uploads/minified1.jpg" width="625" height="317" srcset="https://codegeekz.com/wp-content/uploads/minified1.jpg 625w, https://codegeekz.com/wp-content/uploads/minified1-425x215.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Minified.js is a client-side JavaScript library with low footprint (&lt;8kB) and a large feature set. It offers jQuery-like features (DOM manipulation, animation, events, HTTP requests) and utility functions (collections, date&amp;number formatting, date arithmetic, templates) with a simple, coherent API.</p> 
           <span class="button-wrap"><a href="http://minifiedjs.com/" class="button medium" target="_blank">Minified.js</a></span> &nbsp;
           <p></p> 
           <h2>7. Rekapi</h2> 
           <p><img class="alignnone size-full wp-image-4538" alt="rekapi" src="http://codegeekz.com/wp-content/uploads/rekapi.jpeg" width="625" height="331" srcset="https://codegeekz.com/wp-content/uploads/rekapi.jpeg 625w, https://codegeekz.com/wp-content/uploads/rekapi-425x225.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Rekapi is a keyframe animation library for JavaScript. It gives you an API for Defining keyframe-based animations and controlling animation playback. Rekapi is renderer-agnostic. At its core, Rekapi does not perform any rendering. However, it does expose an API for defining renderers, and the standard distribution comes bundled with renderers for the HTML DOM and HTML5 2D canvas.</p> 
           <span class="button-wrap"><a href="http://rekapi.com/" class="button medium" target="_blank">Rekapi</a></span> &nbsp;
           <p></p> 
           <h2>8. Snap.svg</h2> 
           <p><img class="alignnone size-full wp-image-4539" alt="snapsvg" src="http://codegeekz.com/wp-content/uploads/snapsvg1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/snapsvg1.jpg 625w, https://codegeekz.com/wp-content/uploads/snapsvg1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>SVG is an excellent way to create interactive, resolution-independent vector graphics that will look great on any size screen. And the Snap.svg JavaScript library makes working with your SVG assets as easy as jQuery makes working with the DOM. Snap.svg is designed for modern browsers and therefore supports the newest SVG features like masking, clipping, patterns, full gradients, groups, and more.</p> 
           <span class="button-wrap"><a href="http://snapsvg.io/" class="button medium" target="_blank">Snap.svg</a></span> &nbsp;
           <p></p> 
           <h2>9. Favico</h2> 
           <p><img class="alignnone size-full wp-image-4540" alt="favico" src="http://codegeekz.com/wp-content/uploads/favico.jpg" width="625" height="342" srcset="https://codegeekz.com/wp-content/uploads/favico.jpg 625w, https://codegeekz.com/wp-content/uploads/favico-425x232.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Want to add badges, images, or even video to your favicon? Favico.js makes it simple to add animated badges for your favicon, or create a favicon on the fly from an image, video, or even a live image from your visitor’s webcam.</p> 
           <span class="button-wrap"><a href="http://lab.ejci.net/favico.js/" class="button medium" target="_blank">Favico</a></span> &nbsp;
           <p></p> 
           <h2>10. Textillate.js</h2> 
           <p><img class="alignnone size-full wp-image-4546" alt="textilliate" src="http://codegeekz.com/wp-content/uploads/textilliate.jpg" width="625" height="310" srcset="https://codegeekz.com/wp-content/uploads/textilliate.jpg 625w, https://codegeekz.com/wp-content/uploads/textilliate-425x210.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Textillate.js is a simple plugin for CSS3 text animations. Textillate.js combines some awesome libraries to provide an easy-to-use plugin for applying CSS3 animations to any text.</p> 
           <span class="button-wrap"><a href="http://jschr.github.io/textillate/" class="button medium" target="_blank">Textillate.js</a></span> &nbsp;
           <p></p> 
           <h2>11. Firmin</h2> 
           <p><img class="alignnone size-full wp-image-4547" alt="firmin" src="http://codegeekz.com/wp-content/uploads/firmin.jpeg" width="625" height="310" srcset="https://codegeekz.com/wp-content/uploads/firmin.jpeg 625w, https://codegeekz.com/wp-content/uploads/firmin-425x210.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Firmin is a JavaScript animation library that uses CSS transforms and transitions to create smooth, hardware-accelerated animations.</p> 
           <span class="button-wrap"><a href="http://extralogical.net/projects/firmin/" class="button medium" target="_blank"> Firmin</a></span> &nbsp;
           <p></p> 
           <h2>12. AliceJS</h2> 
           <p><img class="alignnone size-full wp-image-4548" alt="alicejs" src="http://codegeekz.com/wp-content/uploads/alicejs.jpeg" width="625" height="310" srcset="https://codegeekz.com/wp-content/uploads/alicejs.jpeg 625w, https://codegeekz.com/wp-content/uploads/alicejs-425x210.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>AliceJS – (A Lightweight Independent CSS Engine) is a micro JavaScript library focused on using hardware-accelerated capabilities (in particular CSS3 features) in modern browsers for generating high-quality, high-end visual effects.</p> 
           <span class="button-wrap"><a href="http://blackberry.github.io/Alice/" class="button medium" target="_blank">Alice.js</a></span> &nbsp;
           <p></p> 
           <h2>13. SVG.js</h2> 
           <p><img class="alignnone size-full wp-image-4549" alt="svg" src="http://codegeekz.com/wp-content/uploads/svg1.jpg" width="625" height="321" srcset="https://codegeekz.com/wp-content/uploads/svg1.jpg 625w, https://codegeekz.com/wp-content/uploads/svg1-425x218.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>SVG.js is a lightweight JavaScript library for animating and manipulating SVG. It has an uncluttered syntax, is only 7k gzipped, has various useful plugins, and much more.</p> 
           <span class="button-wrap"><a href="http://svgjs.com/" class="button medium" target="_blank">SVG.js</a></span> &nbsp;
           <p></p> 
           <h2>14. Motio</h2> 
           <p><img class="alignnone size-full wp-image-4550" alt="Motio" src="http://codegeekz.com/wp-content/uploads/Motio1.jpeg" width="625" height="310" srcset="https://codegeekz.com/wp-content/uploads/Motio1.jpeg 625w, https://codegeekz.com/wp-content/uploads/Motio1-425x210.jpeg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Motio is a small JavaScript library for simple but powerful sprite based animations and panning. It takes an element, and changes the background position to create an animation effect. Motio has no dependencies, but there is an optional Motio jQuery plugin version available.</p> 
           <span class="button-wrap"><a href="http://darsa.in/motio/" class="button medium" target="_blank">Motio</a></span> &nbsp;
           <p></p> 
           <h2>15. Anima.js</h2> 
           <p><img class="alignnone size-full wp-image-4551" alt="animajs" src="http://codegeekz.com/wp-content/uploads/animajs.jpg" width="625" height="318" srcset="https://codegeekz.com/wp-content/uploads/animajs.jpg 625w, https://codegeekz.com/wp-content/uploads/animajs-425x216.jpg 425w" sizes="(max-width: 625px) 100vw, 625px"></p> 
           <p>Anima.js makes it easy to animate a hundred or more objects at a time with CSS and JavaScript. It also gives more control over your animations, and can even create CSS-only animations (though those offer less control and the feature is only experimental at the moment).</p> 
           <span class="button-wrap"><a href="http://lvivski.com/anima/" class="button medium" target="_blank">Anima.js</a></span>
