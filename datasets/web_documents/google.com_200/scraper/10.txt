     <h1 class="content-title Tutorial-header"> <button type="button" class="Button--reset Tutorial-headerButton " data-toggle="tooltip" title="Mark as Complete" data-js="markAsComplete" data-tutorialinteraction data-tutorialsingle>

       </svg> </button> How To Use node.js, request and cheerio to Set Up Simple Web-Scraping </h1> 
     <span class="meta-section timestamp"><span class="tutorial-date-text">Posted</span><span class="tutorial-date">September 16, 2013</span></span> 
     <span class="meta-section pageviews"><span class="icon icon-eye v-mid"></span><span class="views-count v-mid">164.3k</span><span class="sr-only"> views</span></span> 
     <span class="meta-section tags"> <a class="tag" href="/community/tags/node-js?type=tutorials">Node.js</a> </span> 
    </div> 
    <div class="content-body tutorial-content tutorial-content-legacy" data-growable-markdown>

      <h3>Introduction:</h3> 
      <p>In this tutorial, we will scrape the front page of <a href="http://news.ycombinator.com" target="_blank">Hacker News</a> to get all the top ranking links as well as their metadata - such as the title, URL and the number of points/comments it received. This is one of many techniques to extract data from web pages using node.js and mainly uses a module called <a href="https://github.com/MatthewMueller/cheerio" target="_blank">cheerio</a> by Matthew Mueller which implements a subset of jQuery specifically designed for server use.</p> 
      <p>Cheerio is lightweight, fast, flexible and easy to use, if you're already accustomed to working with jQuery. We will also make use of Mikael Rogers' excellent <a href="https://github.com/mikeal/request" target="_blank">request</a> module as a simplified HTTP client.</p> 
      <h2>Requirements:</h2> 
      <p>I will assume that you're already familiar with node.js, jQuery and basic Linux administrative tasks like connecting to your VPS using SSH.</p> 
      <p>If you're unfamiliar with node.js or if you haven't installed it yet, please refer to the Articles &amp; Tutorials section above to find installation instructions for your operating system.</p> 
      <h2>Code:</h2> 
      <p>To install the required modules using NPM, simply type the following command:</p> 
      <p><code>npm install request cheerio</code></p> 
      <p>This will install the modules in your current working directory only.</p> 
      <p>To install the modules globally run: <code>npm install -g request cheerio</code></p> 
      <p>Create a file called <code>scrape.js</code> and add the following lines:</p> 
      <pre>
var request = require('request');
var cheerio = require('cheerio');
</pre> 
      <p>This will load all of our module dependencies.</p> 
      <p>Now we'll load the front page of Hacker News with a simple request and display the HTML code of the page, if no error occurs and the HTTP status code equals 200.</p> 
      <p>Append these lines to the file:</p> 
      <pre>
request('https://news.ycombinator.com', function (error, response, html) {
  if (!error &amp;&amp; response.statusCode == 200) {
    console.log(html);
  }
});
</pre> 
      <p>Try running the script using <code>node scrape.js</code> and you should see the HTML code being logged in your terminal window.</p> 
      <p>In order to know how to extract our desired meta-data, we need to know how the elements are structured within the HTML code. A preferred way is to use the web-developer tools built into Google Chrome to inspect the desired target element on the web page simply by right clicking on it and selecting "Inspect element".</p> 
      <p>In this example, I've opened Hacker News in Chrome, right-clicked and inspected the title of the top ranking story:</p> 
      <img src="https://assets.digitalocean.com/tutorial_images/rVbJWSt.png" alt="Chrome Web Developer
Tools"> 
      <p>After a quick look at the web-developer console, I came to the conclusion that we could simply select each "span" element which has a class added to it named "comhead" and select the "a" element above it by using the jQuery API function <a href="http://api.jquery.com/prev" target="_blank">prev()</a> to easily parse all 30 top ranking links and their titles.</p> 
      <p>We can test my assumption by changing our request code to this:</p> 
      <pre>
request('https://news.ycombinator.com', function (error, response, html) {
  if (!error &amp;&amp; response.statusCode == 200) {
    var $ = cheerio.load(html);
    $('span.comhead').each(function(i, element){
      var a = $(this).prev();
      console.log(a.text());
    });
  }
});
</pre> 
      <p>As expected, by running the code we get a list of 30 titles. Let's modify it a bit more to parse the remaining metadata:</p> 
      <pre>
request('https://news.ycombinator.com', function (error, response, html) {
  if (!error &amp;&amp; response.statusCode == 200) {
    var $ = cheerio.load(html);
    $('span.comhead').each(function(i, element){
      var a = $(this).prev();
      var rank = a.parent().parent().text();
      var title = a.text();
      var url = a.attr('href');
      var subtext = a.parent().parent().next().children('.subtext').children();
      var points = $(subtext).eq(0).text();
      var username = $(subtext).eq(1).text();
      var comments = $(subtext).eq(2).text();
      // Our parsed meta data object
      var metadata = {
        rank: parseInt(rank),
        title: title,
        url: url,
        points: parseInt(points),
        username: username,
        comments: parseInt(comments)
      };
      console.log(metadata);
    });
  }
});
</pre> 
      <h3>Here is an Overview of what the Added Code Does:</h3> 
      <p>Select the previous element:</p> 
      <pre>
var a = $(this).prev();
</pre> 
      <p>Get the rank by parsing the element two levels above the "a" element:</p> 
      <pre>
var rank = a.parent().parent().text();
</pre> 
      <p>Parse the link title:</p> 
      <pre>
var title = a.text();
</pre> 
      <p>Parse the href attribute from the "a" element:</p> 
      <pre>
var url = a.attr('href');
</pre> 
      <p>Get the subtext children from the next row in the HTML table:</p> 
      <pre>
var subtext = a.parent().parent().next().children('.subtext').children();
</pre> 
      <p>Extract the relevant data from the children:</p> 
      <pre>
var points = $(subtext).eq(0).text();
var username = $(subtext).eq(1).text();
var comments = $(subtext).eq(2).text();
</pre> 
      <p>Running the modified script should output an array of objects like this:</p> 
      <pre>
[ { rank: 1,
    title: 'The Meteoric Rise of DigitalOcean ',
    url: 'http://news.netcraft.com/archives/2013/06/13/the-meteoric-rise-of-digitalocean.html',
    points: 240,
    username: 'beigeotter',
    comments: 163 },
  { rank: 2,
    title: 'Introducing Private Networking',
    url: 'https://www.digitalocean.com/blog_posts/introducing-private-networking',
    points: 172,
    username: 'Goranek',
    comments: 75 },
...
</pre> 
      <p><strong>That's it!</strong> You can now store the the extracted data in a database like MongoDB or Redis to further process it. Here is the full source code of our scrape.js file:</p> 
      <pre>
var request = require('request');
var cheerio = require('cheerio');

request('https://news.ycombinator.com', function (error, response, html) {
  if (!error &amp;&amp; response.statusCode == 200) {
    var $ = cheerio.load(html);
    var parsedResults = [];
    $('span.comhead').each(function(i, element){
      // Select the previous element
      var a = $(this).prev();
      // Get the rank by parsing the element two levels above the "a" element
      var rank = a.parent().parent().text();
      // Parse the link title
      var title = a.text();
      // Parse the href attribute from the "a" element
      var url = a.attr('href');
      // Get the subtext children from the next row in the HTML table.
      var subtext = a.parent().parent().next().children('.subtext').children();
      // Extract the relevant data from the children
      var points = $(subtext).eq(0).text();
      var username = $(subtext).eq(1).text();
      var comments = $(subtext).eq(2).text();
      // Our parsed meta data object
      var metadata = {
        rank: parseInt(rank),
        title: title,
        url: url,
        points: parseInt(points),
        username: username,
        comments: parseInt(comments)
      };
      // Push meta-data into parsedResults array
      parsedResults.push(metadata);
    });
    // Log our finished parse results in the terminal
    console.log(parsedResults);
  }
});
</pre>
