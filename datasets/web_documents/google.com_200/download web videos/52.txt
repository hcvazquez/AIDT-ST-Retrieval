     <h1>What is WebTorrent?</h1> 
     <p><strong>WebTorrent</strong> is a streaming torrent client for the <strong>web browser</strong> and the <strong>desktop</strong>.</p> 
     <p>WebTorrent is written completely in JavaScript – the language of the web – and uses <strong>WebRTC</strong> for peer-to-peer transport whenever possible. No browser plugins, extensions, or installation is required to use WebTorrent in your browser.</p> 
     <h2>Why?</h2> 
     <p>Imagine a peer-to-peer YouTube where viewers help to host the site's content. By making BitTorrent easier to use, we can get more people to participate and take the first steps to re-decentralizing the Internet.</p> 
     <h2>How do I try it out?</h2> 
     <p>See WebTorrent in action on the Web at <a href="https://instant.io">Instant.io</a>. Send a file to a friend using BitTorrent!</p> 
     <p>Or, install <a href="/desktop">WebTorrent Desktop</a> for Mac, Windows, and Linux. Try out a fast, lightweight, new torrent client today!</p>
