     <p>User authentication and authorization module for AngularJS.</p> 
     <h2><a id="user-content-features" class="deep-link" href="#features" aria-hidden="true">
       <svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" viewbox="0 0 16 16" width="16">
        <path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
       </svg></a>Features</h2> 
     <ul> 
      <li>Transparently handles authentication and authorization for your token based REST API</li> 
      <li>Authenticates with a backend, stores the token in the local storage of the browser and adds the <code>Authorization</code> HTTP header to every request</li> 
      <li>Redirects the user to the login page when the token is no longer valid</li> 
      <li>Expects to get a list of user permissions from the backend that is then used for the client side authorization</li> 
      <li>Integrates with <code>angular-ui-router</code> and adds route based authorization</li> 
      <li>Provides a directive that can be used for element based authorization</li> 
     </ul>

     <p>Follow these steps to add the ng-user-auth module to your application:</p> 
     <ul> 
      <li>Clone the repo and add the file <code>dist/ng-user-auth.min.js</code> to your HTML header</li> 
      <li><strong>OR</strong> add NPM package to your application: <code>npm install --save ng-user-auth</code></li> 
      <li><strong>OR</strong> add yarn package to your application: <code>yarn add ng-user-auth</code></li> 
      <li>Configure the module</li>
