                Quick sort is based on one principle 
               <i>divide and conquer &nbsp;. </i>Quick sort first
               <i>&nbsp;</i>divides the large lists into two sub smaller lists ,the low elements and the high elements .Quick sort can then recursively sort the sublists.
               <br> 
               <br> 
               <i><b>Read Also :</b></i>&nbsp;&nbsp; 
               <a href="http://javahungry.blogspot.com/2013/06/java-sorting-program-code-merge-sort.html" target="_blank">Merge Sort Java Code with Example</a>
               <br> 
               <br> * Choose an element and called it as 
               <i>pivot ,</i>&nbsp;in the given list .
               <br> 
               <br> *&nbsp;
               <span style="background-color: white; line-height: 19.1875px;">Reorder the list so that all elements with values less than the pivot come before the pivot, while all elements</span>
               <br> 
               <span style="background-color: white; line-height: 19.1875px;">&nbsp; &nbsp;</span>
               <span style="background-color: white; line-height: 19.1875px;">with values greater than the pivot come after it (equal values can go either way). After this partitioning pivot&nbsp; is in &nbsp;</span>
               <span style="font-family: inherit;"><span style="background-color: white; line-height: 19.1875px;">its final position. This is called the&nbsp;</span><span style="background-color: white; line-height: 19.1875px;">partition</span><span style="background-color: white; line-height: 19.1875px;">&nbsp;operation.</span></span>
               <span style="background-color: white; line-height: 19.1875px;">&nbsp;&nbsp;</span>
               <br> 
               <br> 
               <span style="background-color: white; line-height: 19.1875px;">* Recursively apply the above steps to the sublists of the elements with smaller values and separately the&nbsp;&nbsp; </span>
               <span style="background-color: white; line-height: 19.1875px;">sublists of elements with &nbsp;greater values.</span>
               <br> 
               <br> 
               <a name="more"></a>
               <br> 
               <br> It is also known as partition exchange sort .Quicksort's sequential and localized memory references work with a cache .
