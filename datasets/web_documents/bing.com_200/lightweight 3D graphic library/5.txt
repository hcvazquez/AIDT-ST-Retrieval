   <h1 id="firstHeading" class="firstHeading" lang="en">List of 3D graphics libraries</h1> 
   <div id="bodyContent" class="mw-body-content"> 
    <div id="siteSub" class="noprint">
     From Wikipedia, the free encyclopedia
    </div> 
    <div id="contentSub"></div> 
    <div id="jump-to-nav" class="mw-jump">
      Jump to: 
     <a href="#mw-head">navigation</a>, 
     <a href="#p-search">search</a> 
    </div> 
    <div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr">
     <div class="mw-parser-output">
      <p>3D graphics have become so popular, particularly in <a href="/wiki/Video_game" title="Video game">video games</a>, that specialized <a href="/wiki/Application_programming_interface" title="Application programming interface">APIs</a> (application programming interfaces) have been created to ease the processes in all stages of computer graphics generation. These APIs have also proved vital to computer graphics hardware manufacturers, as they provide a way for <a href="/wiki/Programmer" title="Programmer">programmers</a> to access the hardware in an abstract way, while still taking advantage of the special hardware of any specific <a href="/wiki/Graphics_card" class="mw-redirect" title="Graphics card">graphics card</a>.</p> 
      <p>The first 3D graphics framework was probably <a href="/wiki/3D_Core_Graphics_System" title="3D Core Graphics System">Core</a>, published by the ACM in 1977.</p> 
      <p></p>

      <h2><span class="mw-headline" id="Low-level_3D_API">Low-level 3D API</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_3D_graphics_libraries&amp;action=edit&amp;section=1" title="Edit section: Low-level 3D API">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>These APIs for 3D computer graphics are particularly popular:</p> 
      <ul> 
       <li><a href="/wiki/Direct3D" title="Direct3D">Direct3D</a> (a subset of <a href="/wiki/DirectX" title="DirectX">DirectX</a>)</li> 
       <li><a href="/wiki/Glide_(API)" title="Glide (API)">Glide</a></li> 
       <li><a href="/wiki/Mantle_(API)" title="Mantle (API)">Mantle</a> developed by AMD.</li> 
       <li><a href="/wiki/Metal_(API)" title="Metal (API)">Metal</a> developed by Apple.</li> 
       <li><a href="/wiki/OpenGL" title="OpenGL">OpenGL</a> and the <a href="/wiki/OpenGL_Shading_Language" title="OpenGL Shading Language">OpenGL Shading Language</a></li> 
       <li><a href="/wiki/OpenGL_ES" title="OpenGL ES">OpenGL ES</a> 3D API for embedded devices</li> 
       <li><a href="/wiki/QuickDraw_3D" title="QuickDraw 3D">QuickDraw 3D</a> developed by Apple Computer starting in 1995, abandoned in 1998</li> 
       <li><a href="/wiki/RenderMan_Interface_Specification" title="RenderMan Interface Specification">RenderMan</a></li> 
       <li><a href="/wiki/RenderWare" title="RenderWare">RenderWare</a></li> 
       <li><a href="/wiki/Vulkan_(API)" title="Vulkan (API)">Vulkan</a></li> 
      </ul> 
      <h3><span class="mw-headline" id="Web-based_API">Web-based API</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_3D_graphics_libraries&amp;action=edit&amp;section=2" title="Edit section: Web-based API">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <ul> 
       <li><a href="/wiki/Stage3D" title="Stage3D">Stage3D</a> in the low-level 3D library in Flash version 11 and thereafter, somewhat similar to WebGL</li> 
       <li><a href="/wiki/WebGL" title="WebGL">WebGL</a> is a JavaScript interface for OpenGL-ES-2.x API, promoted by <a href="/wiki/Khronos_Group" title="Khronos Group">Khronos</a>. This is gaining more interest recently, as this enables applications to use native graphics.</li> 
      </ul> 
      <h2><span class="mw-headline" id="High-level_3D_API">High-level 3D API</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_3D_graphics_libraries&amp;action=edit&amp;section=3" title="Edit section: High-level 3D API">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <p>There are also higher-level 3D <a href="/wiki/Scene_graph" title="Scene graph">scene-graph</a> APIs which provide additional functionality on top of the lower-level rendering API. Such libraries under active development include:</p> 
      <ul> 
       <li><a href="/wiki/ClanLib" title="ClanLib">ClanLib</a></li> 
       <li><a href="/wiki/Crystal_Space" title="Crystal Space">Crystal Space</a></li> 
       <li><a href="/wiki/HOOPS_3D_Graphics_System" title="HOOPS 3D Graphics System">HOOPS 3D Graphics System</a></li> 
       <li><a href="/wiki/Horde3D_(game_engine)" class="mw-redirect" title="Horde3D (game engine)">Horde3D</a></li> 
       <li><a href="/wiki/Irrlicht_Engine" title="Irrlicht Engine">Irrlicht Engine</a></li> 
       <li><a href="/wiki/Java_3D" title="Java 3D">Java 3D</a></li> 
       <li><a href="/wiki/Java_FX" class="mw-redirect" title="Java FX">Java FX</a></li> 
       <li><a href="/wiki/JMonkey_Engine" class="mw-redirect" title="JMonkey Engine">JMonkey Engine</a></li> 
       <li><a href="/wiki/JT_(visualization_format)" title="JT (visualization format)">JT Open</a> from <a href="/wiki/Siemens_PLM_Software" title="Siemens PLM Software">Siemens PLM Software</a></li> 
       <li><a href="/wiki/Mobile_3D_Graphics_API" title="Mobile 3D Graphics API">Mobile 3D Graphics API</a> (M3G; JSR-184)</li> 
       <li><a href="/wiki/OGRE" title="OGRE">OGRE</a></li> 
       <li><a href="/wiki/OpenGL_Performer" title="OpenGL Performer">OpenGL Performer</a></li> 
       <li><a rel="nofollow" class="external text" href="http://www.openinventor.com">Open Inventor</a></li> 
       <li><a href="/wiki/OpenSceneGraph" title="OpenSceneGraph">OpenSceneGraph</a></li> 
       <li><a href="/wiki/OpenSG" title="OpenSG">OpenSG</a></li> 
       <li><a href="/wiki/QSDK" title="QSDK">QSDK</a></li> 
       <li><a href="/wiki/Vega_Prime" class="mw-redirect" title="Vega Prime">Vega Prime</a> by <a href="/wiki/Presagis" title="Presagis">Presagis</a></li> 
       <li><a href="/wiki/VTK" title="VTK">VTK</a></li> 
      </ul> 
      <h3><span class="mw-headline" id="JavaScript-based_engines">JavaScript-based engines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_3D_graphics_libraries&amp;action=edit&amp;section=4" title="Edit section: JavaScript-based engines">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>There is more interest in <a href="/wiki/Web_browser" title="Web browser">web browser</a> based high-level API for 3D graphics engines. Some are:</p> 
      <ul> 
       <li><a href="/wiki/A-Frame_(VR)" class="mw-redirect" title="A-Frame (VR)">A-Frame</a></li> 
       <li><a href="/wiki/Blend4Web" title="Blend4Web">Blend4Web</a></li> 
       <li><a href="/wiki/CopperLicht" title="CopperLicht">CopperLicht</a></li> 
       <li><a href="/wiki/O3D" title="O3D">O3D</a></li> 
       <li><a href="/wiki/StormEngineC" title="StormEngineC">StormEngineC</a></li> 
       <li><a href="/wiki/Three.js" title="Three.js">Three.js</a></li> 
       <li><a href="/wiki/X3DOM" class="mw-redirect" title="X3DOM">X3DOM</a></li> 
      </ul> 
      <h3><span class="mw-headline" id="Flash-based_engines">Flash-based engines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_3D_graphics_libraries&amp;action=edit&amp;section=5" title="Edit section: Flash-based engines">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
      <p>3D engines have been developed on the <a href="/wiki/Adobe_Flash" title="Adobe Flash">Adobe Flash</a> platform to run in web browsers.</p> 
      <ul> 
       <li><a href="/wiki/Papervision3D" title="Papervision3D">Papervision3D</a> and its fork <a href="/wiki/Away3D" title="Away3D">Away3D</a> for Flash <a href="/wiki/SWF" title="SWF">SWF</a></li> 
      </ul>
