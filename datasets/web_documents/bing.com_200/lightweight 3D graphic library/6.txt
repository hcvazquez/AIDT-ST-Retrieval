    <h1 class="sectionedit1"><a name="javascript-graphics-library-jsgl" id="javascript-graphics-library-jsgl">JavaScript Graphics Library (JSGL)</a></h1> 
    <div class="level1"> 
     <p> JSGL is an <strong>Open-Source, browser independent 2D vector graphics library for JavaScript</strong>. </p> 
     <p> It is designed for, and well tested on most major web browsers: </p> 
     <p> </p>
     <table> 
      <tbody>
       <tr>
        <td valign="middle"> <img src="/lib/exe/fetch.php?w=16&amp;media=wiki:msie6.png" class="media" alt="" width="16"><img src="/lib/exe/fetch.php?w=16&amp;media=wiki:msie7.png" class="media" alt="" width="16"> 
         <div style="float:right">
          Internet Explorer 
          <strong>6.0+</strong>,
         </div></td> 
        <td valign="top"> <img src="/lib/exe/fetch.php?w=16&amp;media=wiki:firefox.png" class="media" alt="" width="16"> 
         <div style="float:right">
          Firefox 3.0+,
         </div></td> 
        <td valign="top"> <img src="/lib/exe/fetch.php?w=16&amp;media=wiki:opera.png" class="media" alt="" width="16"> 
         <div style="float:right">
          Opera 9.0+,
         </div></td> 
        <td valign="top"> <img src="/lib/exe/fetch.php?w=16&amp;media=wiki:safari.png" class="media" alt="" width="16"> 
         <div style="float:right">
          Safari 3.0+,
         </div></td> 
        <td valign="top"> <img src="/lib/exe/fetch.php?w=16&amp;media=wiki:chrome.png" class="media" alt="" width="16"> 
         <div style="float:right">
          Google Chrome
         </div></td> 
       </tr> 
      </tbody>
     </table> 
     <p></p> 
     <p> It provides a well-designed, <strong>object-oriented <acronym title="Application Programming Interface">API</acronym></strong> for creating <strong>interactive graphics applications</strong> in JavaScript. </p> 
     <p> In integrates VML/<acronym title="Scalable Vector Graphics">SVG</acronym> vector markup languages. However, the <acronym title="Application Programming Interface">API</acronym> provided is <strong>cross-browser</strong>, leaving all the hard work to JSGL. </p> 
     <p> <strong>Drawing interactive graphics in JavaScript has never been easier</strong> <img src="/lib/images/smileys/icon_biggrin.gif" class="middle" alt=":-D"> </p> 
    </div> 
    <h2 class="sectionedit2"><a name="jsgl-20-released" id="jsgl-20-released">JSGL 2.0 Released!</a></h2> 
    <div class="level2"> 
     <p> </p>
     <div id="introholder" style="width: 150px; height: 150px; border: 1px solid black; float: left; margin-right: 32px"></div> 
     <script type="text/javascript">
var introPanel = new jsgl.Panel(document.getElementById('introholder'));

var introCircle = introPanel.createCircle();
introPanel.addElement(introCircle);
introCircle.setRadius(10);
with(introCircle.getStroke()) {
  setWeight(5);
  setColor('#900');
}
introCircle.getFill().setColor('#fed');

var introRect = introPanel.createRectangle();
introPanel.addElement(introRect);
introRect.setSizeWH(20,50);
introRect.setHorizontalAnchor(jsgl.HorizontalAnchor.CENTER);
introRect.setVerticalAnchor(jsgl.VerticalAnchor.MIDDLE);
introRect.setX(75);
with(introRect.getStroke()) {
  setWeight(5);
  setColor('#090');
}
introRect.getFill().setColor('#efd');

var introAnimator = new jsgl.util.Animator();
introAnimator.setStartValue(0);
introAnimator.setEndValue(2*Math.PI);
introAnimator.setDuration(2000);
introAnimator.setRepeating(true);

introAnimator.addStepListener(function(t) {
    introCircle.setCenterX(75+40*Math.cos(t));
    introCircle.setCenterY(75+40*Math.sin(t));
  });
introAnimator.addStepListener(function(t) {
    introRect.setY(75+40*Math.sin(t+Math.PI));
  });

introAnimator.init();

introPanel.setCursor(jsgl.Cursor.POINTER);
introPanel.addClickListener(function(eventArgs) {
    window.location = '?id=documentation';
  });
introAnimator.play();

introPanel.addMouseOverListener(function(eventArgs) {
    introAnimator.pause();
  });
introPanel.addMouseOutListener(function(eventArgs) {
    introAnimator.play();
  });

</script> 
     <div style="float:left"> 
      <p></p> 
      <p> <big>What JSGL 2.0 offers?</big> </p> 
      <ul> 
       <li class="level1">
        <div class="li"> 
         <strong>excellent <a href="/doku.php?id=documentation" class="wikilink1" title="documentation">documentation</a></strong> with huge amounts of 
         <strong>examples</strong>,
        </div> </li> 
      </ul> 
      <ul> 
       <li class="level1">
        <div class="li">
          well-designed, 
         <strong>browser-independent <acronym title="Application Programming Interface">API</acronym></strong> for working with 
         <strong>dynamically updatable graphic shapes</strong>, such as
        </div> 
        <ul> 
         <li class="level2">
          <div class="li"> 
           <strong>line</strong>, 
           <strong>polyline</strong>, 
           <strong>Bezièr curve</strong>,
          </div> </li> 
         <li class="level2">
          <div class="li"> 
           <strong>rectangle</strong>, 
           <strong>circle</strong>, 
           <strong>ellipse</strong>, 
           <strong>polygon</strong>,
          </div> </li> 
         <li class="level2">
          <div class="li"> 
           <strong>raster image</strong>, 
           <strong>text label</strong>, 
           <strong>group</strong>,
          </div> </li> 
         <li class="level2">
          <div class="li">
            general 
           <strong><acronym title="Scalable Vector Graphics">SVG</acronym> path</strong>,
          </div> </li> 
        </ul> </li> 
      </ul> 
      <ul> 
       <li class="level1">
        <div class="li">
          support for 
         <strong>mouse events</strong>, allowing you to create 
         <strong>interactive Web 2.0</strong> applications,
        </div> </li> 
      </ul> 
      <ul> 
       <li class="level1">
        <div class="li">
          tools for 
         <strong>animation</strong> of shapes, and other 
         <strong>utilities</strong>,
        </div> </li> 
      </ul> 
      <ul> 
       <li class="level1">
        <div class="li">
          JSGL is 
         <strong>FREE of charge</strong> and can also be used in commercial applications under the terms of 
         <strong>LGPLv3 license</strong>.
        </div> </li> 
      </ul>
