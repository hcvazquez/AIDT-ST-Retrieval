            <p>Classes that we will need to perform this task:</p> 
            <ul> 
             <li> MediaMetaDataRetriever </li> 
             <li> Bitmap</li> 
             <li> BitmapFactory</li> 
            </ul> 
            <p>Knowledgeable Facts: </p> 
            <ul> 
             <li> Understanding of MediaMetadataRetriever class.</li> 
             <li> Extraction of Album art, Album name, Artist name, Genre from mp3 file.</li> 
             <li> Conversion of Byte array to Bitmap for showing Album art.</li> 
            </ul> 
            <p>Introduction to Classes used:</p> 
            <p>MediaMetadataRetriever: This class is used to extract metadata from media file. It comes from “android.media.MediaMetadataRetriever” package. It has many of available metadata types. A developer can use this class as per need. Here is the below some metadata types that we can extract from a media file in android.</p> 
            <ul> 
             <li> METADATA_KEY_ALBUM : This key of metadata is used to extract the information about the Album associated with media file.</li> 
             <li> METADATA_KEY_ALBUMARTIST : This key of metadata is used to extract the information about the Artist or Performer associated with media file.</li> 
             <li> METADATA_KEY_ARTIST : This key of metadata is used to extract the information about the Artist associated with media file.</li> 
             <li> METADATA_KEY_AUTHOR : This key of metadata is used to extract the information about the author associated with media file.</li> 
             <li> METADATA_KEY_COMPOSER : This key of metadata is used to extract the information about the Composer associated with media file.</li> 
             <li>METADATA_KEY_BITRATE : This key of metadata is used to extract the information about the Bitrate associated with media file.</li> 
             <li>METADATA_KEY_GENRE: This key of metadata is used to extract the information about the Genre associated with media file.</li> 
            </ul> 
            <p> Methods available in MediaMetadataRetriever class:</p> 
            <ol> 
             <li>void setDataSource(String path) : This method is used to set the media file source path in string format.</li> 
             <li>string extractMetadata(int keyCode) : This method is used to extract metadata as per keys selected that are mentioned above.</li> 
             <li>byte[] getEmbeddedPicture() : This method is used to extract album art associated with media file.</li> 
            </ol> 
            <p><strong>Bitmap Class</strong>: Bitmap class is basically used to work on Images. Bitmap class has the following useful method.</p> 
            <p>createScaledBitmap(Bitmap src, int dstWidth, int dstHeight, boolean filter) : This method is used to create scaled bitmap from existing bitmap with specified width, height.</p> 
            <p>Here src defines the Bitmap source, dstWidth defines that width specified, dstHeight defines that height specified and filter defines whether source is filtered or not.</p> 
            <p><strong>BitmapFactory Class</strong> : This class is very useful to work with decoding of byte array, files. It contains the following useful method.</p> 
            <p>decodeByteArray(byte[] data, int offset, int length) : This method is used to decode the byte array and convert it into Bitmap.</p> 
            <p>Here data defines the object of byte array, offset defines the starting point from where decoding has to start and length indicates the total length till array has to decode.</p>

            <h3>Step 3: </h3> 
            <p><strong>Listing 3</strong>: Paste the following code into your MetaExtractActivity.java</p> 
            <div class="div_listagem">
             <pre class="pre_listagem">package com.nkm.metaextract;

import android.app.Activity;
import android.graphics.Bitmap;
import android.graphics.BitmapFactory;
import android.graphics.Color;
import android.media.MediaMetadataRetriever;
import android.os.Bundle;
import android.widget.ImageView;
import android.widget.TextView;

public class MetaExtractActivity extends Activity {
	ImageView album_art;
	TextView album, artist, genre;

	MediaMetadataRetriever metaRetriver;
	byte[] art;

	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.main);
		getInit();

		// Ablum_art reterival code //

		metaRetriver = new MediaMetadataRetriever();
		metaRetriver.setDataSource("/sdcard/audio.mp3");
		try {
			art = metaRetriver.getEmbeddedPicture();
			Bitmap songImage = BitmapFactory
					.decodeByteArray(art, 0, art.length);
			album_art.setImageBitmap(songImage);
			album.setText(metaRetriver
					.extractMetadata(MediaMetadataRetriever.METADATA_KEY_ALBUM));
			artist.setText(metaRetriver
					.extractMetadata(MediaMetadataRetriever.METADATA_KEY_ARTIST));
			genre.setText(metaRetriver
					.extractMetadata(MediaMetadataRetriever.METADATA_KEY_GENRE));
		} catch (Exception e) {
			album_art.setBackgroundColor(Color.GRAY);
			album.setText("Unknown Album");
			artist.setText("Unknown Artist");
			genre.setText("Unknown Genre");
		}

	}

	// Fetch Id's form xml 

	public void getInit() {

		album_art = (ImageView) findViewById(R.id.album_art);
		album = (TextView) findViewById(R.id.Album);
		artist = (TextView) findViewById(R.id.artist_name);
		genre = (TextView) findViewById(R.id.genre);

	}
}
</pre>
