             <h2 style="text-align:center;color:#000000;line-height:normal;font-family:'Times New Roman';"> <a name="Create_your_own_web_scrapper_using_node_js"></a></h2> 
             <div style="color:#000000;font-family:'Times New Roman';font-size:medium;"></div> 
             <div style="color:#000000;font-family:'Times New Roman';font-size:medium;"> 
              <p style="margin:0px;">Want to make you own scraper to scrape any data form any website and return it in JSON format so you can used it anywhere you like? If yes, then you are at right place.<br> <br> </p> 
             </div> 
             <div style="color:#000000;font-family:'Times New Roman';font-size:medium;"> 
              <p style="margin:0px;">In this article I will guide you how to scrape any website to get desired data using Node.js and to obtain the data in JSON format which can be used e.g. make any app which will run on live data from the internet.<br> <br> </p> 
             </div> 
             <div style="color:#000000;font-family:'Times New Roman';font-size:medium;"> 
              <p style="margin:0px;">I will be using Windows 10 x64 and VS 2015 for this article and will scrape from a news website i.e.</p> 
             </div> 
             <div style="color:#000000;font-family:'Times New Roman';font-size:medium;"></div> 
             <ul style="color:#000000;font-family:'Times New Roman';font-size:medium;"> 
              <li><span style="text-indent:-0.25in;">First of all set up the IDE, go to&nbsp;</span><a href="https://nodejs.org/en/download/" style="text-indent:-0.25in;">https://nodejs.org/en/download/</a><span style="text-indent:-0.25in;">&nbsp;and download the Nnode.js pre build installer.               <li><span style="text-indent:-0.25in;">After installing it, open your Visual Studio and create a new project Templates&gt;JavaScript&gt;Node.js&gt;Basic Node.js Express 4 Application <p class="separator" style="margin:0px;text-align:center;clear:both;"><a href="http://4.bp.blogspot.com/-fkED18bttfo/VmQKiNiHxuI/AAAAAAAAASQ/B4j5IsnzBEw/s1600/Node2.JPG" style="margin-right:1em;margin-left:1em;"><img width="640" height="392" alt=" " src="https://4.bp.blogspot.com/-fkED18bttfo/VmQKiNiHxuI/AAAAAAAAASQ/B4j5IsnzBEw/s640/Node2.JPG" style="border-width:0px;border-style:solid;"></a></p> </span></li>
              <li><span style="text-indent:-0.25in;font-family:'times new roman';font-size:7pt;">&nbsp;</span> <span style="text-indent:-0.25in;">Now I have to add two packages in npm folder, i.e. ‘Request’ and ‘Cheerio’. <p class="separator" style="margin:0px;text-align:center;clear:both;"><a href="http://3.bp.blogspot.com/-c2bMSlwT3OQ/VmQKjx47t5I/AAAAAAAAATA/uzvq-EigD-0/s1600/Node3.gif" style="margin-right:1em;margin-left:1em;"><img width="640" height="274" alt=" " src="https://3.bp.blogspot.com/-c2bMSlwT3OQ/VmQKjx47t5I/AAAAAAAAATA/uzvq-EigD-0/s640/Node3.gif" style="border-width:0px;border-style:solid;"></a></p> </span></li>
              <li><span style="text-indent:-0.25in;">And uninstall ‘Jade’ by doing a right click as we don’t need it now and I have to host my JSON to Azure cloud service so Jade gives an exception. If you want to consume JSON directly in your application or hosting using other service than you don’t have to uninstall Jade.</span> </li>
              <li><span style="text-indent:-0.25in;">Now go to app.js and comment out the line numbers 14 and 15 as we are not using ‘Views’ <p class="separator" style="margin:0px;text-align:center;clear:both;"><a href="http://4.bp.blogspot.com/-4EvzYw1yu0Q/VmQKlao_NSI/AAAAAAAAAS8/-H9IUXd3X1I/s1600/node4.gif" style="margin-right:1em;margin-left:1em;"><img width="640" height="342" title="Commenting" alt="Commenting" src="https://4.bp.blogspot.com/-4EvzYw1yu0Q/VmQKlao_NSI/AAAAAAAAAS8/-H9IUXd3X1I/s640/node4.gif" style="border-width:0px;border-style:solid;"></a></p> </span></li>
              <li><span style="text-indent:-0.25in;">Also comment out</span><span style="text-indent:-0.25in;">&nbsp;&nbsp;</span><span style="text-indent:-0.25in;">‘</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">app.use(</span><span style="background:white;color:#a31515;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">'/'</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">, routes);</span><span style="line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">’</span> </li>
              <li><span style="text-indent:-0.25in;">Change</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">&nbsp;app.use(</span><span style="background:white;color:#a31515;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">'/users'</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">, users);</span><span style="line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">&nbsp;</span><span style="text-indent:-0.25in;">to</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">&nbsp;app.use(</span><span style="background:white;color:#a31515;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">'/'</span><span style="background:white;line-height:13.55px;text-indent:-0.25in;font-family:consolas;font-size:9.5pt;">, users);</span> </li>
              <li><span style="text-indent:-0.25in;">Now go to users.js as now we will do the main thing here. First of all add the files ‘cheerio’ and ‘request’. <p class="separator" style="margin:0px;text-align:center;clear:both;"><a href="http://4.bp.blogspot.com/-LU96r6AX05k/VmQKllKTE9I/AAAAAAAAATE/RrOl8u_M4Ho/s1600/node5.gif" style="margin-right:1em;margin-left:1em;"><img width="640" height="294" alt=" " src="https://4.bp.blogspot.com/-LU96r6AX05k/VmQKllKTE9I/AAAAAAAAATE/RrOl8u_M4Ho/s640/node5.gif" style="border-width:0px;border-style:solid;"></a></p> </span></li>
              <li><span style="text-indent:-0.25in;">Create a variable to save the URL of the link:</span> </li>
             </ul> 
             <p style="margin:0px;color:#000000;font-family:'Times New Roman';font-size:medium;"> <span style="background:white;color:#0000ff;line-height:13.55px;font-family:consolas;font-size:9.5pt;"></span></p> 
             <div class="reCodeBlock" style="border:1px solid #7f9db9;overflow-y:auto;"> 
              <div style="background-color:#ffffff;">
               <span><code style="color:#006699;font-weight:bold;">var</code> <code style="color:#000000;">url = </code><code style="color:blue;">"<a href="http://www.thenews.com.pk/CitySubIndex.aspx?ID=14">http://www.thenews.com.pk/CitySubIndex.aspx?ID=14</a>"</code><code style="color:#000000;">;</code></span>

                 <span><code style="color:#000000;">router.get (</code><code style="color:blue;">'/'</code><code style="color:#000000;">, </code><code style="color:#006699;font-weight:bold;">function</code> <code style="color:#000000;"> (req, res) {</code></span>
                </div> 
                <div style="background-color:#f8f8f8;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#000000;">request (url, </code><code style="color:#006699;font-weight:bold;">function</code> <code style="color:#000000;"> (error, response, body) {</code></span></span>
                </div> 
                <div style="background-color:#ffffff;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#006699;font-weight:bold;">if</code> <code style="color:#000000;">(!error &amp;&amp; response.statusCode === 200) {</code></span></span>
                </div> 
                <div style="background-color:#f8f8f8;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#006699;font-weight:bold;">var</code> <code style="color:#000000;">data = scrapeDataFromHtml(body);</code></span></span>
                </div> 
                <div style="background-color:#ffffff;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#000000;">res.send(data);</code></span></span>
                </div> 
                <div style="background-color:#f8f8f8;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#000000;">}</code></span></span>
                </div> 
                <div style="background-color:#ffffff;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#006699;font-weight:bold;">return</code> <code style="color:#000000;">console.log(error);</code></span></span>

                <div style="background-color:#f8f8f8;">
                 <span><code>&nbsp;&nbsp;&nbsp;&nbsp;</code><span><code style="color:#000000;">});</code></span></span>

                 <span><code style="color:#000000;">});</code></span>

             <p style="margin:0px;color:#000000;font-family:'Times New Roman';font-size:medium;"> </p>

             <ul style="color:#000000;font-family:'Times New Roman';font-size:medium;"> 
              <li><span style="text-indent:-0.25in;">Here comes the main and difficult part. Now we have to write the main logic of scraping our website.
