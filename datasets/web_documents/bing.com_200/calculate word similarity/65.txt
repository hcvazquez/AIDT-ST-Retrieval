      <h1 itemprop="itemreviewed">How to use basic calculations in PDF forms using Acrobat X or XI</h1> 
      <h2>Learn how to calculate fillable PDF forms using Adobe Acrobat X or XI.</h2> 
      <div class="auc-tutorial-author"> 
       <p>By <a rel="author" href="http://blogs.adobe.com/documentcloud/acrobat-expert/donna-baker/">Donna Baker</a> ? <em>February 6, 2012</em></p> 
       <p> <a href="http://blogs.adobe.com/documentcloud/acrobat-expert/donna-baker/"><img src="/assets/images/template/btn_auc_expert.jpg"></a> &nbsp; <a href="http://twitter.com/acrofacts" target="_blank"><img src="/assets/images/template/btn_twitter_follow.jpg"></a> </p> 
       <img class="photo" src="https://acrobatusers.com/assets/uploads/author/1637/dbaker15.jpg"> 
       <a class="googleprofile" target="_blank" rel="author" href="https://plus.google.com/112583456778229924467">Donna Baker</a> 
      </div> 
      <iframe title="AdobeTV Video Player" width="680" height="391" src="https://tv.adobe.com/embed/629/12266/" frameborder="0" allowfullscreen scrolling="no"></iframe> 
      <br>
      <br> 
      <p>In this tutorial, learn how to calculate fillable <a href="https://acrobat.adobe.com/us/en/how-to/create-fillable-pdf-forms-creator.html?sdid=KSAJO">PDF forms</a> using Adobe Acrobat X or XI. Follow these steps to set calculations in text fields and drop-down boxes that appear in your fillable PDF forms:</p> 
      <h3>To add text fields</h3> 
      <ol> 
       <li>Double-click the field where you want to display the result to open the Properties dialog box.</li> 
       <li>Click the Calculate tab.</li> 
       <li>To add the values entered into fields, click the Value Is The radio button.</li> 
       <li>Pick Sum from the popup menu. </li> 
       <li>Click Pick to open a list of the fields in your form, select the fields you want to add, and click OK to list the fields in the dialog box.</li> 
      </ol> 
      <h3> To multiply values</h3> 
      <ol> 
       <li>Double-click the field where you want to display the result to open the Properties dialog box.</li> 
       <li>Click the Calculate tab.</li> 
       <li>To multiply the values entered into fields, click the Value Is The radio button.</li> 
       <li>Pick Product from the popup menu. </li> 
       <li>Click Pick to open a list of the fields in your form, select the fields you want to multiply, and click OK to list the fields in the dialog box.</li> 
      </ol> 
      <p>Subtraction and division are a bit harder since you don't have a simple option to pick.</p> 
      <h3>To subtract one field from another</h3> 
      <ol> 
       <li>Open the Properties for the results field.</li> 
       <li>Click the Calculate tab.</li> 
       <li>Now click the Simplified Field Notation radio button.</li> 
       <li> click Edit to open the Javascript Editor.</li> 
       <li>Type the expression in the editor, such as Field1-Field2. Make sure not to add any spaces.</li> 
       <li>Click OK and you'll see the expression show on the dialog box.</li> 
      </ol> 
      <h3>To divide two fields</h3> 
      <ol> 
       <li> In the Calculate tab for the results field, choose Simplified Field Notation, and click Edit. </li> 
       <li>In the JavaScript Editor, type FieldB/FieldA. Again, make sure you don't have any spaces. Click OK and you'll see the expression on the Properties dialog box.</li> 
       <li>Click Preview to toggle the Preview mode to test your work. You won't have any surprises with adding, subtracting, or multiplying, but a division calculation can cause display problems.</li> 
      </ol> 
      <p>If you leave the default settings for the field as unformatted content, the field shows NaN—which means Not a Number. If you open the Format tab and choose Number from the Format category, you'll see an error message. The only way to guarantee a seamless division operation is to use a JavaScript IF statement to define conditions for the field when it's blank. Unless you write the JavaScript, leave the field unformatted as Acrobat will still calculate the proper value.</p> 
      <p>So now your form not only looks great, it also gives you and your users immediate information.</p>
