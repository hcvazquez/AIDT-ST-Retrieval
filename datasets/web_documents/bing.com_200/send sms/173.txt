           <h1 name="title" id="title" class="graf graf--h2 graf-after--figure graf--title">Send an email using only JavaScript</h1>
           <h2 name="subtitle" id="subtitle" class="graf graf--h4 graf-after--h2 graf--subtitle">No server-side languages involved.</h2>
           <p name="d801" id="d801" class="graf graf--p graf-after--h4">I recently ran into the need of sending an email without any server-side technology. Luckily, this is a piece of cake with the awesome Mandrill App. (Disclaimer: I know nothing about Mandrill except for the fact that the first 12,000 emails per month are free.)</p>
           <p name="98ac" id="98ac" class="graf graf--p graf-after--p">Here’s the skinny:</p>
           <pre name="b69f" id="b69f" class="graf graf--pre graf-after--p">1. register for Mandrill to get an API key<br>2. load jQuery<br>3. use $.ajax to send an email</pre>

When you’re done, click on Get API Keys:</p>
