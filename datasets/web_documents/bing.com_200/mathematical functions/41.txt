   <h1 id="firstHeading" class="firstHeading" lang="en">List of numerical libraries</h1> 
   <div id="bodyContent" class="mw-body-content"> 
    <div id="siteSub" class="noprint">

    </div>

    <div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr">
     <div class="mw-parser-output">
      <p>This is a list of <a href="/wiki/Wikipedia:N" class="mw-redirect" title="Wikipedia:N">notable</a> <b>numerical libraries</b>, which are <a href="/wiki/Library_(computing)" title="Library (computing)">libraries</a> used in <a href="/wiki/Software_development" title="Software development">software development</a> for performing <a href="/wiki/Numerical_analysis" title="Numerical analysis">numerical</a> calculations. It is not a complete listing but is instead a list of numerical libraries with articles on Wikipedia, with few exceptions.</p> 
      <p>The choice of a typical library depends on a diverse range of requirements such as: desired features (for e.g.: large dimensional linear algebra, parallel computation, partial differential equations), commercial/opensource nature, readability of API, portability or platform/compiler dependence (for e.g.: Linux, Windows, Visual C++, GCC), performance in speed, ease-of-use, continued support from developers, standard compliance, specialized optimization in code for specific application scenarios or even the size of the code-base to be installed.</p> 
      <p>As we find comprehensive surveys rarely available, there is almost always (at least initially) a difficult choice among a number of possible libraries. Often it tends to be at the discretion of the user based on his own taste and comforts, only due to the lack of proper information.</p>

      <h2><span class="mw-headline" id="Multi-language">Multi-language</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=1" title="Edit section: Multi-language">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/ALGLIB" title="ALGLIB">ALGLIB</a> is an open source numerical analysis library which may be used from C++, C#, FreePascal, Delphi, VBA.</li> 
       <li><a href="/wiki/IMSL_Numerical_Libraries" title="IMSL Numerical Libraries">IMSL Numerical Libraries</a> are libraries of numerical analysis functionality implemented in standard programming languages like C, Java, C# .NET, Fortran, and Python.</li> 
       <li>The <a href="/wiki/NAG_Numerical_Library" title="NAG Numerical Library">NAG Library</a> is a collection of mathematical and statistical routines for multiple programming languages (C, C++, Fortran, Visual Basic, Java and C#) and packages (MATLAB, Excel, R, LabVIEW).</li> 
       <li><a href="/wiki/GNU_Octave" title="GNU Octave">GNU Octave</a> is an open source high level programming language and library, including a command line interface, analogous to commercial alternatives such as Maple, MatLab, Mathematica, etc. APIs, functions and libraries can be called from many platforms, including high level engineering programs, where functions are, in many cases, seamlessly interpreted and integrated in similar fashion to MatLab. It also can be used with batch orientation.</li> 
       <li><a href="/wiki/Librsb" title="Librsb">librsb</a> is an open source library for high performance sparse matrix computations providing multi-threaded primitives to build iterative solvers (implements also the Sparse <a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a> standard). It can be used from C, C++, Fortran, and a dedicated <a href="/wiki/GNU_Octave" title="GNU Octave">GNU Octave</a> package.</li> 
      </ul> 
      <h2><span class="mw-headline" id="C"><a href="/wiki/C_(programming_language)" title="C (programming language)">C</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=2" title="Edit section: C">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/BLOPEX" class="mw-redirect" title="BLOPEX">BLOPEX</a> (Block Locally Optimal Preconditioned Eigenvalue Xolvers) is an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> <a href="/wiki/Library_(computing)" title="Library (computing)">library</a> for the <a href="/wiki/Scalability" title="Scalability">scalable</a> (<a href="/wiki/Parallel_computing" title="Parallel computing">parallel</a>) solution of eigenvalue problems.</li> 
       <li><a href="/wiki/FFTW" title="FFTW">FFTW</a> (Fastest Fourier Transform in the West) is a software library for computing Fourier and related transforms.</li> 
       <li><a href="/wiki/GNU_Scientific_Library" title="GNU Scientific Library">GNU Scientific Library</a>, a popular, free numerical analysis library implemented in <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a>.</li> 
       <li><a href="/wiki/GNU_Multi-Precision_Library" class="mw-redirect" title="GNU Multi-Precision Library">GNU Multi-Precision Library</a> is a library for doing <a href="/wiki/Arbitrary-precision_arithmetic" title="Arbitrary-precision arithmetic">arbitrary-precision arithmetic</a>.</li> 
       <li><a href="/wiki/Hypre" title="Hypre">hypre</a> (High Performance Preconditioners) is an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> <a href="/wiki/Library_(computing)" title="Library (computing)">library</a> of <a href="/wiki/Computer_code" class="mw-redirect" title="Computer code">routines</a> for <a href="/wiki/Scalability" title="Scalability">scalable</a> (<a href="/wiki/Parallel_computing" title="Parallel computing">parallel</a>) solution of linear systems and preconditioning.</li> 
       <li><a href="/wiki/LabWindows/CVI" title="LabWindows/CVI">LabWindows/CVI</a> is an ANSI C <a href="/wiki/Integrated_Development_Environment" class="mw-redirect" title="Integrated Development Environment">IDE</a> that includes built-in libraries for analysis of raw measurement data, signal generation, windowing, filter functions, signal processing, linear algebra, array and complex operations, curve fitting and statistics.</li> 
       <li><a href="/wiki/Lis_(linear_algebra_library)" title="Lis (linear algebra library)">Lis</a> is a scalable parallel library for solving systems of linear equations and eigenvalue problems using iterative methods.</li> 
       <li><a href="/wiki/Intel_MKL" class="mw-redirect" title="Intel MKL">Intel MKL</a>, Intel Math Kernel Library (in C), a library of optimized math routines for science, engineering, and financial applications, written in C/C++ and Fortran. Core math functions include BLAS, LAPACK, ScaLAPACK, sparse solvers, fast Fourier transforms, and vector math.</li> 
       <li><a href="/wiki/Portable,_Extensible_Toolkit_for_Scientific_Computation" title="Portable, Extensible Toolkit for Scientific Computation">Portable, Extensible Toolkit for Scientific Computation</a> (<b>PETSc</b>), is a suite of <a href="/wiki/Data_structures" class="mw-redirect" title="Data structures">data structures</a> and <a href="/wiki/Computer_code" class="mw-redirect" title="Computer code">routines</a> for the <a href="/wiki/Scalability" title="Scalability">scalable</a> (<a href="/wiki/Parallel_computing" title="Parallel computing">parallel</a>) solution of <a href="/wiki/Scientific_computing" class="mw-redirect" title="Scientific computing">scientific applications</a> modeled by <a href="/wiki/Partial_differential_equations" class="mw-redirect" title="Partial differential equations">partial differential equations</a>.</li> 
       <li><a href="/wiki/SLEPc" title="SLEPc">SLEPc</a> <b>Scalable Library for Eigenvalue Problem Computations</b> is a <a href="/wiki/PETSc" class="mw-redirect" title="PETSc">PETSc</a>-based <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> <a href="/wiki/Library_(computing)" title="Library (computing)">library</a> for the <a href="/wiki/Scalability" title="Scalability">scalable</a> (<a href="/wiki/Parallel_computing" title="Parallel computing">parallel</a>) solution of eigenvalue problems.</li> 
      </ul> 
      <h2><span id="C++"></span><span class="mw-headline" id="C.2B.2B"><a href="/wiki/Boost_(C%2B%2B_libraries)" title="Boost (C++ libraries)">C++</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=3" title="Edit section: C++">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/Advanced_Simulation_Library" title="Advanced Simulation Library">Advanced Simulation Library</a> is free and open source hardware accelerated multiphysics simulation software with an <a href="/wiki/OpenCL" title="OpenCL">OpenCL</a>-based internal computational engine.</li> 
       <li><a href="/wiki/Armadillo_(C%2B%2B_library)" title="Armadillo (C++ library)">Armadillo</a> is a C++ linear algebra library (matrix and vector maths), aiming towards a good balance between speed and ease of use. It employs template classes, and has optional links to BLAS and LAPACK. The syntax (<a href="/wiki/Application_Programming_Interface" class="mw-redirect" title="Application Programming Interface">API</a>) is similar to <a href="/wiki/MATLAB" title="MATLAB">MATLAB</a>.</li> 
       <li><a href="/wiki/Blitz%2B%2B" title="Blitz++">Blitz++</a> is a high-performance vector mathematics library written in C++.</li> 
       <li><a href="/wiki/Boost_(C%2B%2B_libraries)" title="Boost (C++ libraries)">Boost.uBLAS</a> C++ libraries for numerical computation</li> 
       <li><a href="/wiki/Deal.II" title="Deal.II">deal.II</a> is a library supporting all the finite element solution of partial differential equations.</li> 
       <li><a href="/wiki/Dlib" title="Dlib">Dlib</a> is a modern C++ library with easy to use linear algebra and optimization tools which benefit from optimized BLAS and LAPACK libraries.</li> 
       <li><a href="/wiki/Eigen_(C%2B%2B_library)" title="Eigen (C++ library)">Eigen</a> is a vector mathematics library with performance comparable with Intel's <a href="/wiki/Math_Kernel_Library" title="Math Kernel Library">Math Kernel Library</a></li> 
       <li><a href="/wiki/Hermes_Project" title="Hermes Project">Hermes Project</a>: C++/Python library for rapid prototyping of space- and space-time adaptive hp-FEM solvers.</li> 
       <li><a href="/wiki/IML%2B%2B" title="IML++">IML++</a> is a C++ library for solving linear systems of equations, capable of dealing with dense, sparse, and distributed matrices.</li> 
       <li><a href="/wiki/IT%2B%2B" title="IT++">IT++</a> is a C++ library for linear algebra (matrices and vectors), signal processing and communications. Functionality similar to MATLAB and Octave.</li> 
       <li><a href="/wiki/LAPACK%2B%2B" title="LAPACK++">LAPACK++</a>, a C++ wrapper library for <a href="/wiki/LAPACK" title="LAPACK">LAPACK</a> and <a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a></li> 
       <li><a href="/wiki/Intel_MKL" class="mw-redirect" title="Intel MKL">Intel MKL</a>, Intel Math Kernel Library (in C++), a library of optimized math routines for science, engineering, and financial applications, written in C/C++ and Fortran. Core math functions include BLAS, LAPACK, ScaLAPACK, sparse solvers, fast Fourier transforms, and vector math.</li> 
       <li><a href="/wiki/MLPACK_(C%2B%2B_library)" title="MLPACK (C++ library)">MLPACK</a> is an open-source library for machine learning, exploiting C++ language features to provide maximum performance and flexibility while providing a simple and consistent API</li> 
       <li><a href="/wiki/Matrix_Template_Library" title="Matrix Template Library">MTL4</a> is a generic <a href="/wiki/C%2B%2B" title="C++">C++</a> template library providing sparse and dense BLAS functionality. MTL4 establishes an intuitive interface (similar to <a href="/wiki/MATLAB" title="MATLAB">MATLAB</a>) and broad applicability thanks to <a href="/wiki/Generic_programming" title="Generic programming">Generic programming</a>.</li> 
       <li><a href="/wiki/Number_Theory_Library" title="Number Theory Library">NTL</a> is a C++ library for number theory.</li> 
       <li><a href="/wiki/Trilinos" title="Trilinos">Trilinos</a> is an effort to develop algorithms and enabling technologies for the solution of large-scale, complex multi-physics engineering and scientific problems. It is a collection of <i>packages</i>.</li> 
       <li><a href="/w/index.php?title=VISPACK&amp;action=edit&amp;redlink=1" class="new" title="VISPACK (page does not exist)">VISPACK</a> is a standard visualization software library for manipulating level-set surface models</li> 
      </ul> 
      <h2><span class="mw-headline" id="Delphi"><a href="/wiki/Delphi_(programming_language)" title="Delphi (programming language)">Delphi</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=4" title="Edit section: Delphi">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/ALGLIB" title="ALGLIB">ALGLIB</a> - an open source numerical analysis library.</li> 
      </ul> 
      <h2><span id=".NET_Framework_languages_C#,_F#,_VB.NET_and_PowerShell"></span><span class="mw-headline" id=".NET_Framework_languages_C.23.2C_F.23.2C_VB.NET_and_PowerShell">.NET Framework languages <a href="/wiki/C_Sharp_(programming_language)" title="C Sharp (programming language)">C#</a>, <a href="/wiki/F_Sharp_(programming_language)" title="F Sharp (programming language)">F#</a>, <a href="/wiki/Visual_Basic_.NET" title="Visual Basic .NET">VB.NET</a> and <a href="/wiki/Windows_PowerShell" class="mw-redirect" title="Windows PowerShell">PowerShell</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=5" title="Edit section: .NET Framework languages C#, F#, VB.NET and PowerShell">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/AForge.NET" title="AForge.NET">AForge.NET</a> is a computer vision and artificial intelligence library. It implements a number of genetic, fuzzy logic and machine learning algorithms with several architectures of artificial neural networks with corresponding training algorithms.</li> 
       <li><a href="/wiki/Accord.NET" title="Accord.NET">Accord.NET</a> is a collection of libraries for scientific computing, including numerical linear algebra, optimization, statistics, artificial neural networks, machine learning, signal processing and computer vision.</li> 
       <li><a href="/wiki/ILNumerics.Net" class="mw-redirect" title="ILNumerics.Net">ILNumerics.Net</a> Commercial high performance, typesafe numerical array classes and functions for general math, FFT and linear algebra, aims .NET/mono, 32&amp;64 bit, script-like syntax in C#, 2D &amp; 3D plot controls, efficient memory management</li> 
       <li><a href="/wiki/Measurement_Studio" title="Measurement Studio">Measurement Studio</a> is a commercial integrated suite UI controls and class libraries for use in developing test and measurement applications. The analysis class libraries provide various digital signal processing, signal filtering, signal generation, peak detection, and other general mathematical functionality.</li> 
       <li>NeuronDotNet is a <a href="/wiki/GPL" class="mw-redirect" title="GPL">GPL</a>-licensed artificial neural network library entirely written in C#. Because it only depends on the core .NET assemblies, it is easily portable across platforms.</li> 
       <li><a href="/wiki/NMath" title="NMath">NMath</a> by <a href="/wiki/CenterSpace_Software" title="CenterSpace Software">CenterSpace Software</a>: Commercial numerical component libraries for the .NET platform, including signal processing (FFT) classes, a linear algebra (LAPACK &amp; BLAS) framework, and a statistics package.</li> 
       <li><a href="/wiki/Math.NET_Numerics" title="Math.NET Numerics">Math.NET Numerics</a> aims to provide methods and algorithms for numerical computations in science, engineering and every day use. Covered topics include special functions, linear algebra, probability models, random numbers, interpolation, integral transforms and more. MIT/X11 license.</li> 
      </ul> 
      <h2><span class="mw-headline" id="Fortran"><a href="/wiki/Fortran" title="Fortran">Fortran</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=6" title="Edit section: Fortran">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a> (Basic Linear Algebra Subprograms) is a de facto <a href="/wiki/Application_programming_interface" title="Application programming interface">application programming interface</a> standard for publishing libraries to perform basic <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a> operations such as <a href="/wiki/Vector_space" title="Vector space">vector</a> and <a href="/wiki/Matrix_multiplication" title="Matrix multiplication">matrix multiplication</a>.</li> 
       <li><a href="/wiki/CERN_Program_Library" title="CERN Program Library">CERNLIB</a> is a collection of <a href="/wiki/FORTRAN_77" class="mw-redirect" title="FORTRAN 77">FORTRAN 77</a> libraries and modules.</li> 
       <li><a href="/wiki/EISPACK" title="EISPACK">EISPACK</a> is a <a href="/wiki/Software_library" class="mw-redirect" title="Software library">software library</a> for <a href="/wiki/Numerical_computation" class="mw-redirect" title="Numerical computation">numerical computation</a> of <a href="/wiki/Eigenvalues" class="mw-redirect" title="Eigenvalues">eigenvalues</a> and <a href="/wiki/Eigenvectors" class="mw-redirect" title="Eigenvectors">eigenvectors</a> of matrices, written in <a href="/wiki/FORTRAN" class="mw-redirect" title="FORTRAN">FORTRAN</a>. It contains subroutines for calculating the eigenvalues of nine classes of <a href="/wiki/Matrix_(Mathematics)" class="mw-redirect" title="Matrix (Mathematics)">matrices</a>: complex general, complex <a href="/wiki/Hermitian" class="mw-redirect" title="Hermitian">Hermitian</a>, real general, real symmetric, real symmetric <a href="/wiki/Band_matrix" title="Band matrix">banded</a>, real symmetric tridiagonal, special real tridiagonal, generalized real, and generalized real symmetric matices.</li> 
       <li><a href="/wiki/IMSL_Numerical_Libraries" title="IMSL Numerical Libraries">IMSL Numerical Libraries</a> are cross-platform libraries containing a comprehensive set of mathematical and statistical functions that can be embedded in a users application.</li> 
       <li><a href="/wiki/Harwell_Subroutine_Library" class="mw-redirect" title="Harwell Subroutine Library">Harwell Subroutine Library</a> is a collection of <a href="/wiki/Fortran_77" class="mw-redirect" title="Fortran 77">Fortran 77</a> and 95 codes that address core problems in numerical analysis.</li> 
       <li><a href="/wiki/LAPACK" title="LAPACK">LAPACK</a>, the <b>Linear Algebra PACKage</b>, is a <a href="/wiki/Software_library" class="mw-redirect" title="Software library">software library</a> for <a href="/wiki/Numerical_computation" class="mw-redirect" title="Numerical computation">numerical computing</a> originally written in <a href="/wiki/Fortran" title="Fortran">FORTRAN 77</a> and now written in <a href="/wiki/Fortran" title="Fortran">Fortran 90</a>.</li> 
       <li><a href="/wiki/LINPACK" title="LINPACK">LINPACK</a> is a software <a href="/wiki/Library_(computer_science)" class="mw-redirect" title="Library (computer science)">library</a> for performing numerical <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a> on digital computers. It was written in <a href="/wiki/Fortran" title="Fortran">Fortran</a> by <a href="/wiki/Jack_Dongarra" title="Jack Dongarra">Jack Dongarra</a>, Jim Bunch, <a href="/wiki/Cleve_Moler" title="Cleve Moler">Cleve Moler</a>, and Pete Stewart, and was intended for use on <a href="/wiki/Supercomputer" title="Supercomputer">supercomputers</a> in the 1970s and early 1980s. It has been largely superseded by <a href="/wiki/LAPACK" title="LAPACK">LAPACK</a>, which will run more efficiently on modern architectures.</li> 
       <li><a href="/wiki/Lis_(linear_algebra_library)" title="Lis (linear algebra library)">Lis</a> is a scalable parallel library for solving systems of linear equations and eigenvalue problems using iterative methods.</li> 
       <li><a href="/wiki/MINPACK" title="MINPACK">MINPACK</a> is a library of <a href="/wiki/FORTRAN" class="mw-redirect" title="FORTRAN">FORTRAN</a> subroutines for the solving of systems of nonlinear equations, or the <a href="/wiki/Least_squares" title="Least squares">least squares</a> minimization of the residual of a set of <a href="/wiki/System_of_linear_equations" title="System of linear equations">linear</a> or nonlinear equations.</li> 
       <li>The <a href="/wiki/NAG_Numerical_Library" title="NAG Numerical Library">NAG Fortran Library</a> is a collection of mathematical and statistical routines for Fortran.</li> 
       <li><a href="/wiki/Naval_Observatory_Vector_Astrometry_Subroutines" title="Naval Observatory Vector Astrometry Subroutines">NOVAS</a> is a software library for astrometry-related numerical computations. Both <a href="/wiki/Fortran" title="Fortran">Fortran</a> and <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> versions are available.</li> 
       <li><a href="/wiki/Netlib" title="Netlib">Netlib</a> is a repository of scientific computing software which contains a large number of separate programs and libraries including <a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a>, <a href="/wiki/EISPACK" title="EISPACK">EISPACK</a>, <a href="/wiki/LAPACK" title="LAPACK">LAPACK</a> and others.</li> 
       <li><a href="/wiki/Physics_Analysis_Workstation" title="Physics Analysis Workstation">PAW</a> is a free <a href="/wiki/Data_analysis" title="Data analysis">data analysis</a> package developed at <a href="/wiki/CERN" title="CERN">CERN</a>.</li> 
       <li><a href="/wiki/Portable,_Extensible_Toolkit_for_Scientific_Computation" title="Portable, Extensible Toolkit for Scientific Computation">Portable, Extensible Toolkit for Scientific Computation</a> (<b>PETSc</b>), is a suite of <a href="/wiki/Data_structures" class="mw-redirect" title="Data structures">data structures</a> and <a href="/wiki/Computer_code" class="mw-redirect" title="Computer code">routines</a> for the <a href="/wiki/Scalability" title="Scalability">scalable</a> (<a href="/wiki/Parallel_computing" title="Parallel computing">parallel</a>) solution of <a href="/wiki/Scientific_computing" class="mw-redirect" title="Scientific computing">scientific applications</a> modeled by <a href="/wiki/Partial_differential_equations" class="mw-redirect" title="Partial differential equations">partial differential equations</a>.</li> 
       <li><a href="/wiki/QUADPACK" title="QUADPACK">QUADPACK</a> is a <a href="/wiki/FORTRAN_77" class="mw-redirect" title="FORTRAN 77">FORTRAN 77</a> library for <a href="/wiki/Numerical_integration" title="Numerical integration">numerical integration</a> of one-dimensional functions</li> 
       <li><a href="/wiki/SLATEC" title="SLATEC">SLATEC</a> is a <a href="/wiki/FORTRAN_77" class="mw-redirect" title="FORTRAN 77">FORTRAN 77</a> library of over 1400 general purpose mathematical and statistical routines.</li> 
       <li><a href="/wiki/SOFA_(Astronomy)" class="mw-redirect" title="SOFA (Astronomy)">SOFA</a> is a collection of subroutines that implement official <a href="/wiki/International_Astronomical_Union" title="International Astronomical Union">IAU</a> <a href="/wiki/Algorithms" class="mw-redirect" title="Algorithms">algorithms</a> for <a href="/wiki/Astronomy" title="Astronomy">astronomical</a> computations. Both <a href="/wiki/Fortran" title="Fortran">Fortran</a> and <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> versions are available.</li> 
       <li><a href="/w/index.php?title=SPARSKIT&amp;action=edit&amp;redlink=1" class="new" title="SPARSKIT (page does not exist)">SPARSKIT</a> is a tool package for working with <a href="/wiki/Sparse_matrix" title="Sparse matrix">sparse matrices</a>.</li> 
       <li><a href="/wiki/ARPACK" title="ARPACK">ARPACK</a> is a collection of Fortran77 subroutines designed to solve large scale eigenvalue problems.</li> 
      </ul> 
      <h2><span class="mw-headline" id="Java"><a href="/wiki/Java_programming_language" class="mw-redirect" title="Java programming language">Java</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=7" title="Edit section: Java">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/ND4J_(software)" title="ND4J (software)">ND4J</a> is an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> library that supports n-dimensional (ND) arrays, similar to <a href="/wiki/NumPy" title="NumPy">NumPy</a>. It runs on distributed <a href="/wiki/Graphics_processing_unit" title="Graphics processing unit">GPUs</a> or CPUs cross-platform, and leverages Spark for parallel computation. It supports the <a href="/wiki/Deep_learning" title="Deep learning">deep learning</a> library <a href="/wiki/Deeplearning4j" title="Deeplearning4j">Deeplearning4j</a>.</li> 
       <li><a href="/wiki/Apache_Commons" title="Apache Commons">Apache Commons</a>, is an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> for creating reusable Java components. It has numerical packages for linear algebra and non-linear optimization.</li> 
       <li><a href="/wiki/Colt_(libraries)" title="Colt (libraries)">Colt</a> provides a set of Open Source Libraries for High Performance Scientific and Technical Computing.</li> 
       <li><a href="/wiki/Efficient_Java_Matrix_Library" title="Efficient Java Matrix Library">Efficient Java Matrix Library (EJML)</a> is an open-source linear algebra library for manipulating dense matrices.</li> 
       <li><a href="/wiki/JAMA_(numerical_linear_algebra_library)" title="JAMA (numerical linear algebra library)">JAMA</a>, a numerical <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a> toolkit for the <a href="/wiki/Java_programming_language" class="mw-redirect" title="Java programming language">Java programming language</a>. No active development has taken place since 2005, but it still one of the more popular linear algebra packages in Java.</li> 
       <li><a href="/wiki/Jblas:_Linear_Algebra_for_Java" title="Jblas: Linear Algebra for Java">Jblas: Linear Algebra for Java</a>, a linear algebra library which is an easy to use wrapper around <a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a> and <a href="/wiki/LAPACK" title="LAPACK">LAPACK</a>.</li> 
       <li><a href="/wiki/Parallel_Colt" title="Parallel Colt">Parallel Colt</a> is an open source library for scientific computing. A parallel extension of <a href="/wiki/Colt_(libraries)" title="Colt (libraries)">Colt</a>.</li> 
       <li><a href="/wiki/DataMelt" title="DataMelt">DataMelt</a>, an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> Java libraries for numerical calculations, data I/O and visualization of scientific results. Used together with <a href="/wiki/Jython" title="Jython">Jython</a>.</li> 
       <li><a href="/wiki/Matrix_Toolkit_Java" title="Matrix Toolkit Java">Matrix Toolkit Java</a> is a linear algebra library based on <a href="/wiki/BLAS" class="mw-redirect" title="BLAS">BLAS</a> and <a href="/wiki/LAPACK" title="LAPACK">LAPACK</a>.</li> 
       <li><a href="/wiki/OjAlgo" title="OjAlgo">OjAlgo</a> is an open source Java library for mathematics, linear algebra and optimisation.</li> 
       <li><a href="/wiki/Exp4j" title="Exp4j">exp4j</a> is a small <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a> library for evaluation of mathematical expressions.</li> 
      </ul> 
      <h2><span class="mw-headline" id="Scala"><a href="/wiki/Scala_programming_language" class="mw-redirect" title="Scala programming language">Scala</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=8" title="Edit section: Scala">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/ND4S" title="ND4S">ND4S</a> is a scientific computing library for Scala that includes support for n-dimensional arrays on the JVM.</li> 
      </ul> 
      <h2><span class="mw-headline" id="Perl"><a href="/wiki/Perl_programming_language" class="mw-redirect" title="Perl programming language">Perl</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=9" title="Edit section: Perl">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/Perl_Data_Language" title="Perl Data Language">Perl Data Language</a> gives standard Perl the ability to compactly store and speedily manipulate the large <i>N</i>-dimensional data arrays.</li> 
      </ul> 
      <h2><span class="mw-headline" id="Python"><a href="/wiki/Python_programming_language" class="mw-redirect" title="Python programming language">Python</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=10" title="Edit section: Python">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/DataMelt" title="DataMelt">DataMelt</a> scientific libraries based on <a href="/wiki/Jython" title="Jython">Jython</a>, Java implementation of the Python language.</li> 
       <li><a href="/wiki/NumPy" title="NumPy">NumPy</a>, a library that adds support for the manipulation of large, multi-dimensional arrays and matrices; it also includes a large collection of high-level mathematical functions. NumPy serves as the backbone for a number of other numerical libraries, notably <a href="/wiki/SciPy" title="SciPy">SciPy</a>.</li> 
       <li><a href="/wiki/SageMath" title="SageMath">SageMath</a> is a large mathematical software application which integrates the work of nearly 100 <a href="/wiki/Free_software" title="Free software">free software</a> projects and supports linear algebra, combinatorics, numerical mathematics, calculus, and more.</li> 
       <li><a href="/wiki/SciPy" title="SciPy">SciPy</a>, a large library of scientific tools.</li> 
       <li><a href="/wiki/ScientificPython" title="ScientificPython">ScientificPython</a>, a library with a different set of scientific tools</li> 
      </ul> 
      <h2><span class="mw-headline" id="Groovy"><a href="/wiki/Groovy_(programming_language)" title="Groovy (programming language)">Groovy</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_numerical_libraries&amp;action=edit&amp;section=11" title="Edit section: Groovy">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <ul> 
       <li><a href="/wiki/DataMelt" title="DataMelt">DataMelt</a>, an <a href="/wiki/Open_source" class="mw-redirect" title="Open source">open-source</a> Java libraries for numerical calculations, data I/O and visualization of scientific results. Used together with <a href="/wiki/Groovy_(programming_language)" title="Groovy (programming language)">Groovy</a>.</li> 
      </ul>
