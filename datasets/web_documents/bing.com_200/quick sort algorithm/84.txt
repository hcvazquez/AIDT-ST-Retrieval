                <h3 style="text-align: left;"> <strong><span style="font-family: Arial, Helvetica, sans-serif;">What is Quick Sort algorithm? How to implement Quick Sort in C++?</span></strong></h3> 
                <ul> 
                 <li><span style="font-family: Arial, Helvetica, sans-serif;">Quick Sort is a sorting algorithm. It is also referred as partition exchange sort.</span></li> 
                 <li><span style="font-family: Arial, Helvetica, sans-serif;">In the best/ average case it gives a time complexity of O(nlogn) and worst case time complexity of O(n*n).</span></li> 
                 <li><span style="font-family: Arial, Helvetica, sans-serif;">Can be implemented as in-place sorting without need for additional space.</span></li> 
                 <li><span style="font-family: Arial, Helvetica, sans-serif;">Quick Sorting involves these steps:</span></li> 
                 <ul> 
                  <li><span style="font-family: Arial, Helvetica, sans-serif;">Pick a pivot element from the input list.</span></li> 
                  <li><span style="font-family: Arial, Helvetica, sans-serif;">Perform an input partition operation based on the pivot. Move all elements less than pivot before the pivot element in the list. Move all elements greater than the pivot after the pivot element in the list.</span></li> 
                  <li><span style="font-family: Arial, Helvetica, sans-serif;">Recursively sort the left and right sub-lists of the pivot.</span></li> 
                 </ul> 
                </ul> 
                <h3 style="text-align: left;"> <span style="font-family: Arial, Helvetica, sans-serif;">A simple demonstration of quick sort</span></h3> 
                <pre class="prettyprint"><code>#include &lt;iostream&gt;
using namespace std;

const int INPUT_SIZE = 10;

// A simple print function
void print(int *input)
{
    for ( int i = 0; i &lt; INPUT_SIZE; i++ )
        cout &lt;&lt; input[i] &lt;&lt; " ";
    cout &lt;&lt; endl;
}

// The partition function
int partition(int* input, int p, int r)
{
    int pivot = input[r];

    while ( p &lt; r )
    {
        while ( input[p] &lt; pivot )
            p++;

        while ( input[r] &gt; pivot )
            r--;

        if ( input[p] == input[r] )
            p++;
        else if ( p &lt; r )
        {
            int tmp = input[p];
            input[p] = input[r];
            input[r] = tmp;
        }
    }

    return r;
}

// The quicksort recursive function
void quicksort(int* input, int p, int r)
{
    if ( p &lt; r )
    {
        int j = partition(input, p, r);        
        quicksort(input, p, j-1);
        quicksort(input, j+1, r);
    }
}

int main()
{
    int input[INPUT_SIZE] = {500, 700, 800, 100, 300, 200, 900, 400, 1000, 600};
    cout &lt;&lt; "Input: ";
    print(input);
    quicksort(input, 0, 9);
    cout &lt;&lt; "Output: ";
    print(input);
    return 0;
}
