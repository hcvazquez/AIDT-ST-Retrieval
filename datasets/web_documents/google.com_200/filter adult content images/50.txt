           <p>Images that can be viewed on the Internet are easy to re-use without even asking for permission. It is as easy as saving the image to your computer and reuploading it to your website. Placing javascript code to disable right click on the webpage containing the images cannot protect against advanced users who know how to use third party plugins such as Firebug to <a href="https://www.raymond.cc/blog/urlsnooper-displays-full-path-of-hidden-remote-url/">monitor HTTP traffic</a> to get the direct link of the image.</p>
           <p>A common method used to deter against image hotlinking or reusing the image that you’ve created is by watermarking. A visible watermark in the form of a logo or text will be stamped to show the owner of the image. Stamping the watermark at the corner of an image can be easily bypassed by simply <a href="https://www.raymond.cc/blog/resize-image-or-pictures-online/">cropping it away</a> while placing it in the center can annoy the viewer.</p>
           <p>Another solution that can be used to protect your image is to sign it with an invisible watermark. The invisible watermark does not annoy the viewer while it can be proven that you are the owner of the image which can greatly help in a take down action. Here we have 5 software tools that can add proof of ownership to your digital images.
            <!-- blog-content-middle-new --> <ins class="adsbygoogle adslot_2" style="display:block" data-ad-client="ca-pub-8102232298595506" data-ad-slot="8037657894" data-ad-format="auto"></ins> <script>
		(adsbygoogle = window.adsbygoogle || []).push({});
		</script> 1. <strong>SignMyImage</strong></p>
           <p>SignMyImage is a shareware tool that cost 14.99€ and comes with unlimited free upgrades. It allows you to sign your image with an invisible signature so that it can be used as a proof of ownership. There is also a batch signing and detection feature but it’s disabled in the trial version. To speed up the process of signing and detection, the program has been made to automatically use multiple cores of your computer’s CPU.</p>
           <p><img width="600" height="287" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/signmyimage.png" alt="signmyimage"></p>
           <p>Signing an image with SignMyImage is very easy. First, run SignMyImage and load the image to the program. Click on <em>Menu</em> &gt; <em>Help</em> &gt; <em>Open Shorter URL</em>. Enter the URL of your website and click the Generate button. Copy the generated code.</p>
           <p><img width="582" height="248" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/signmyimage-shorter-url.png" alt="signmyimage shortener url"></p>
           <p>Go back to SignMyImage, click the “Sign Current Image” icon and paste the code that is generated from the URL shortener. Click the Test button and make sure that your default web browser will run and forward you to your website. If that is working, click the OK button to sign the image with the code.</p>
           <p><img width="457" height="255" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/signmyimage-sign-image.png" alt="signmyimage sign image"></p>
           <p>You’ll need to save after signing the image. To detect the signature on a signed image, you’ll need to load the image into SignMyImage program, click on the “Detect Sign On Current Image” button. If a sign is found, it will show the 10 character sign and clicking the “Get More Info” button will bring you to the owner’s website.</p>
           <p><img width="435" height="155" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/signmyimage-sign-found.png" alt="signmyimage sign found"></p>
           <p>Signed images are able to survive cropping, compression and scaling. Registered users of SignMyImage gets access to their image spider where their web crawler will notify you if they found any signed image on a third party website. Do take note that only 24-bit images can be signed.</p>
           <p><strong><a rel="nofollow" href="http://www.adptools.com/en/signmyimage-trynow.html" target="_blank">Download SignMyImage</a></strong></p>
           <hr>
           <p>2. <strong>Icemark</strong></p>
           <p>Icemark is a newer shareware product on the market that can embed an invisible watermark to digital images and it cost $49.95 for a single license. Basically you will sign the image with a private key that is known only to you. There are 3 watermark types in Icemark which is Identification, Image Id or Text and they are meant for your reference. For example, using the identification will only watermark the image while the Image Id allows you to assign an ID together with 3 indications which is “Restricted Use”, “Do Not Copy”, and “Adult Content”. If you prefer to embed a custom message to the image, use the Text type.</p>
           <p><img width="593" height="354" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/icemark.png" alt="icemark"></p>
           <p>To detect the invisible watermark signed by Icemark, load the image into the program, click the magnifying glass icon and it will tell you if the watermark is detected or not. Make sure the correct private key is entered before detecting the watermark. Batch processing is available in File menu where you can automatically process multiple embedding or detection.</p>
           <p>Images that are watermarked with Icemark are resistant to common image editing operations such as compression, distortions, cropping and etc.</p>
           <p><strong><a rel="nofollow" href="http://www.phibit.com/icemark/" target="_blank">Download Icemark</a></strong></p>
           <hr>
           <p>3. <strong>OpenStego</strong></p>
           <p>OpenStego is an open source application that can watermark images using an invisible signature. Using OpenStego is very straight forward. The first step is to create a signature file based on passphrase. Then use the signature file to watermark the images.</p>
           <p><img width="600" height="260" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/openstego.png" alt="openstego"></p>
           <p>Do note that OpenStego runs on Java and the digital watermarking feature is still in beta. We’ve got very weird results after embedding the invisible watermark using the signature file. First of all, the watermarked file size is much smaller than the original and there is a noticeable quality loss. Moreover, we are only seeing a 67% strength for the unedited watermarked file which may not be convincing enough to proof that you’re the owner of the image.</p>
           <p><img width="509" height="319" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/openstego-watermark-strength.png" alt="openstego watermark strength"></p>
           <p><strong><a rel="nofollow" href="http://sourceforge.net/projects/openstego/files/latest/download" target="_blank">Download OpenStego</a></strong></p>
           <hr>
           <p>4. <strong>Steganography</strong></p>
           <p>The meaning of Steganography is hiding information within another data. By using this technique, you can actually embed a text file to an image file. The image file with an embedded hidden text acts like a normal image file and can be viewed using any image viewing software. However the hidden text can be revealed when the image is opened using a special software together with the correct password supplied to it.</p>
           <p>There are quite a number of tools that can do this and we’ve already covered some in an article with <a href="https://www.raymond.cc/blog/hide-your-secret-files-by-embedding-it-to-another-safe-source/">5 different ways to secretly hide a file or message inside another file</a>.</p>
           <p><img width="600" height="443" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2012/10/hide-files-oursecret.png" alt="Our Secret"></p>
           <p>Although this method works, it will only work if the image is untouched. If the image has been edited in any way, the hidden embedded text will be removed and you won’t be able to prove that you are the owner that originally signed the file.</p>
           <hr>
           <p>5. <strong>Digimarc Guardian for Images</strong></p>
           <p>You can find the Digimarc plugin bundled together with Adobe Photoshop and it requires a valid subscription to work. To watermark your images using Digimarc, run Adobe Photoshop, go to Filter &gt; Digimarc &gt; Embed Watermark. If you’ve purchased a subscription, click on the Personalize button to enter your ID and PIN.</p>
           <p><img width="600" height="409" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/digimarc-embed-watermark.png" alt="digimarc embed watermark"></p>
           <p>You can now give more information about the image such as the copyright year, image ID or transaction ID, then followed by selecting the image attributes which is similar to the one that you see in Icemark. There are 4 levels of durability for the watermark whereby a more visible watermark is more durable to changes on the image after an edit. To verify the watermark, load the image into Photoshop, go to Filter &gt; Digimarc &gt; Read Watermark.</p>
           <p><img width="600" height="394" class="aligncenter" src="https://img.raymond.cc/blog/wp-content/uploads/2015/01/digimarc-watermark-information.png" alt="digimarc watermark information"></p>
           <p>If you’ve watermarked under your account, it will show your ID instead of the Demo and the Web Lookup button will connect the viewer to the author of the image.</p>
           <p>The basic package of Digimarc cost $49 per year and it allows up to 1000 digital watermarkings but it does not come with the Digimarc Search Service that shows you where your images are found on the Internet. Do note that the image requires at least 256×256 pixels to retain a reliable watermark.</p>
           <p><strong><a rel="nofollow" href="https://www.digimarc.com/products/guardian/images/photoshop-plug-in" target="_blank">Visit Digimarc Guardian for Images Photoshop Plug-in</a></strong></p>
           <p><strong><em>Additional Information</em></strong>: <a rel="nofollow" href="http://www.signumtech.com/template3.asp?pageID=4&amp;prodID=19" target="_blank">SureSign plug-in</a> for Adobe Photoshop by Signum Technologies can also embed and read watermarks. The cheapest option cost US$220 per annum per workstation for individual users and evaluation copy is only available upon request.</p>
