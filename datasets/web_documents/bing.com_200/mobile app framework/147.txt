             <h3>Package Description</h3> 
             <p class="package-description-body"><a href="http://www.eclipse.org/scout">Eclipse Scout</a> is a Java/HTML5 framework to develop business applications that run on the desktop, on tablets and mobile devices. This package includes Eclipse IDE support for Scout developers and source code.</p> 
             <div class="package-includes"> 
              <h4>This package includes:</h4> 
              <div class="package-includes-list"> 
               <ul>
                <li><span class="title" title="Versioning with Git, integration with Gerrit, GitHub, Gitflow, and Task repositories">Git integration for Eclipse</span></li>
                <li><span class="title" title="Eclipse Java development tools (binary runtime and user documentation).">Eclipse Java Development Tools</span></li>
                <li><span class="title" title="m2e is a Maven integration in Eclipse. Launching Maven from Eclipse, dependency management and search.">Maven Integration for Eclipse</span></li>
                <li><span class="title" title="Provides the Task List for task and time management.">Mylyn Task List</span></li>
                <li><span class="title" title="XML, DTD and XML Schema Editors, validators, and XML Catalog support">Eclipse XML Editors and Tools</span></li>
               </ul> 
              </div>
