           <div id="post_message_96644">
             I've done this in the past using different techniques and tools but none were exactly what you are asking for. The same text in a different place is a difference as well as a similarity. So what you want is to highlight the same text wherever it is. I haven't seen any program that does this for free.
            <br> 
            <br> The best I found from a quick search is 
            <a rel="nofollow" href="http://www.duplichecker.com/comparison" target="_blank">DupliChecker.com</a>. It identifies different phrases easily but only seems to identify similar phrases if they are at least 10 words long which is often too long.
            <br> 
            <br> The easiest to use are the 
            <b>plagiarism checkers</b>. The best do reports like what you want. Some also handle similarities that are not exact ie where one word is changed in a sentence. The problem with these is that:
            <ul>
             <li>they are usually not free</li> 
             <li>they mainly compare one document with a wider range of sources than just one document. Your documents add to their library and you can often limit the comparison to a subset of documents such as a assignments from one class/course.</li> 
            </ul>
            <br> I have used 
            <b>text editor and word processor document comparisons</b> that are close to what you want but the last was in 1992 so I can't remember any of the products now. You might check for suitable plug-ins for Notepad++ which is a free text editor.
            <br> 
            <br> 
            <b>Word/phrase frequency analysis </b>is another option. But, again, the most useful products are not free. You can try it out on a single body of text at a website like 
            <a rel="nofollow" href="http://www.writewords.org.uk/phrase_count.asp" target="_blank">WriteWords</a>. There are probably better ones available. Notice that you have to specify the phrase length.
            <br> 
            <br> The rest of my answer just explains why such software is usually not free:
            <ul>
             <li>Document comparison is usually about comparing versions of the same documents. In its simplest form it is easy to do because any difference is visible by checking the next character in each document. However most comparisons will highlight more complex differences that are also the sort of similarities you want to find: where text has been moved but only for several or more words the same. In many word processors these are identified as a "move" so what you want is a combination of no difference plus moves. The problem is usually that you can't just look at only the moves.</li> 
             <li>Identifying similarities is a lot more complex than finding differences. Similarities have to be discovered using more complex algorithms that involve comparing longer text strings. Then some judgements have to be made about when a difference is actually a similarity. To do it properly some analysis of the grammar should be done to match similar structures.</li> 
            </ul>
