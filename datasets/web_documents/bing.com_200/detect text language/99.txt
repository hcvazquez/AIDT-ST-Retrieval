              <p>I created a JavaScript library to handle your requirement (returns the culture name from a decimal lcid value)</p> 
              <p>The mapping comes from this page: <a href="http://msdn.microsoft.com/en-us/goglobal/bb896001.aspx" rel="nofollow">National Language Support (NLS) API Reference</a> (Windows 7)</p> 
              <p>Practically inside the library there is JSON object with the values and two methods to get them.</p> 
              <p>The Culture name is the full one (for example en-US for 1033) you can slice the string if you need only the first two letters.</p> 
              <p>You can find more information on my blog:</p> 
              <p><a href="http://www.crmanswers.net/2014/06/lcid-javascript-helper-library.html" rel="nofollow">LCID JavaScript Helper Library </a></p>

             <div class="post-text" itemprop="text"> 
              <p>In SharePoint 2013 JSOM API you could utilize <a href="https://msdn.microsoft.com/en-us/library/office/jj850983.aspx" rel="nofollow">SP.ServerSettings.getGlobalInstalledLanguages Method</a> to retrieve all the installed languages, like this:</p> 
              <pre><code>function getInstalledLanguages(success,error) 
{
    var ctx = new SP.ClientContext.get_current();
    var languages = SP.ServerSettings.getGlobalInstalledLanguages(ctx,15);
    ctx.executeQueryAsync(
       function()
       { 
          success(languages);     
       },
       error);    
}
</code></pre> 
              <p>and then find the corresponding language by <code>lcid</code>:</p> 
              <pre><code>getInstalledLanguages(
     function(languages)
     { 
         var lcid = parseInt(SP.Res.lcid);  //get current locale
         var language = languages.filter(function(l) {
                          return l.get_lcid() === lcid; 
                       })[0];
         var langTag =  language.get_languageTag();   
         var langCode = langTag.split('-')[0];          
         console.log(langCode);     
     },
     function(sender,args)
     {
         console.log(args.get_message());
     });
</code></pre>
