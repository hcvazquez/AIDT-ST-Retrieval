        <p align="justify"></p>
        <p><strong>Introduction</strong><br> <br> In my previous article, we saw an overview of <a href="http://www.c-sharpcorner.com/UploadFile/ff2f08/token-based-authentication-using-Asp-Net-web-api-owin-and-i/"> Token based authentication using ASP.net web API and OWIN</a>. In this post, I will explain how to use Token based authentication in AngularJS.<br> <br> In this post, I will build sample Single Page Application (SPA) using AngularJS and this application will do the following:</p> 
        <ul> 
         <li>Allow user to login using "Admin" and "Jignesh" user ID</li> 
         <li>Token keep alive 30 minutes</li> 
         <li>Authenticated user will access the certain views.</li> 
        </ul> 
        <p><strong>Prerequisites<br> </strong><br> Before reading this article, you must have some basic knowledge about AngularJS and Token base authentication using OWIN. <br> <br> The following are the steps to create AngularJS Token Authentication using ASP.NET Web API 2 and OWIN<br> <br> <strong>Step 1</strong><br> <br> Include 3rd party libraries</p>
        <div>
          To get started, we required to include the following libraries:
         <br> 
         <br> 
         <strong>AngularJS</strong>
         <br> 
         <br> We can download the latest AngularJS version using NuGet package manager.
         <br> 
         <em><br> PM&gt; Install-Package angularjs</em>
         <br> 
         <br> Preceding command includes all available AngularJS libraries including minified version. So delete script files which are not required.
         <br> 
         <br> 
         <strong>UI Bootstrap</strong>
         <br> 
         <br> We can download the latest Bootstrap version using NuGet package manager.
         <br> 
         <br> 
         <em>PM&gt; Install-Package bootstrap -Version 3.3.5</em>
         <br> 
         <br> 
         <strong>Step 2<br> <br> Organize Project Structure</strong>
         <br> 
         <br> We can use any IDE to build the web application because this web app totally decouples with backend API and it develops using HTML, AngularJS, and CSS. Here I am using Visual Studio 2013. I have created project using empty project template.
         <br> 
         <br> In this project structure, I have created a folder named "Modules", this contains all AngularJS application files and resources files and "Asset" folder contains the asset of this project i.e. AngularJS libraries file, CSS files, etc.
         <br> 
         <br>
         <img src="http://csharpcorner.mindcrackerinc.netdna-cdn.com/UploadFile/ff2f08/token-based-authentication-using-Asp-Net-web-api-in-angularj/Images/Picture1.png" alt="">
         <br> 
         <br> 
         <strong>Step 3</strong> 
        </div> 
        <p><strong>Boot Strapping Angular Application</strong><br> <br> Boot Strapping Angular Application means creating angular application and modules (modules are nothing but collection of services, directives, filters which are used by the application). Each module has configuration block and applied to the application during this process. To do this, I have added a file called "app.js" in the root folder. This file also contains the route and interceptor.<br> <br> Interceptor is a regular service and it allows us to capture every XHR request and we can also manipulate it before sending it to server end point (web API). It also captures all response and response error.<br></p>
