                      <h1>SSH Factory</h1> 
                      <h2>Java SSH and Java Telnet Components</h2> 
                      <p>SSH Factory is a set of <strong>Java SSH</strong> and <strong>Java telnet</strong> client components for communicating with SSH and telnet servers. Including both SSH (Secure Shell) and telnet components, developers will appreciate the easy-to-use API making it possible to communicate with a remote server using just a few lines of code.</p> 
                      <p><a href="/download-ssh-factory/" target="_self"><img alt="download java ssh components" border="0" src="http://www.jscape.com/hs-fs/hub/26878/file-13610408-gif/images/download.gif?t=1506914196394" style=""></a> <a href="http://store.jscape.com/index.php?main_page=index&amp;cPath=5_6_12" target="_self"><img alt="purchase java ssh components" border="0" id="img-1348097896095" src="http://www.jscape.com/hs-fs/hub/26878/file-13614448-gif/images/order.gif?t=1506914196394" style=""></a></p> 
                      <p><strong>Example Uses</strong></p> 
                      <ul> 
                       <li>Improve efficiency and reduce network down time by automating router configuration tasks</li> 
                       <li>Save time by creating scripts to automate user creation, deletion and password management tasks</li> 
                       <li>Simplify complex remote procedures using task automation</li> 
                      </ul> 
                      <p><strong>Feature Summary</strong></p> 
                      <ul> 
                       <li>Includes both SSH and telnet components</li> 
                       <li>Comprehensive API for building and automating tasks</li> 
                       <li>Easy to use language for scripting tasks in plain text files without having to compile Java code</li> 
                       <li>Support for authenticating with servers using public key authentication</li> 
                       <li>Data compression for increased network performance (SSH)</li> 
                       <li>Define connection timeout limit for detecting unavailable servers</li> 
                       <li>Secure data exchange protected using highly secure encryption algorithms (SSH)</li> 
                       <li>Define complex task prompts using regular expressions</li> 
                       <li>Event listener classes for listening for and responding to events</li> 
                       <li>Stream data exchanged between the client and server to a file for debug purposes</li> 
                       <li>Written using 100% Java for use on any platform</li> 
                       <li>Redistribute the included components as part of your application without runtime or royalty fees (see license for restrictions)</li> 
                       <li>Free priority technical support and upgrades for 1 year after purchase</li> 
                      </ul> 
                      <p><strong>Testimonial</strong></p>
