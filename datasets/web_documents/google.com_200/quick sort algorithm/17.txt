                    <span class="rendered_qtext"><p class="qtext_para">First off, why are you not using the built-in <code class="prettyprint inline">sort()</code> method? It can take a function as a parameter which lets you sort however you see fit:</p><pre class="prettyprint linenums">var measurements = [
    [1, 5.6, 9, 78.4],
    [4, 3],
    [987, 45, 34, 23.3, 45.2, 104.5],
    [6, 3.3333, Math.PI]
];

// Sort the arrays in measurements by length
measurements.sort(function (a, b) {
    return a.length - b.length;
});
</pre><p class="qtext_para">Read <span class="qlink_container"><a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort" rel="noopener" target="_blank" onclick="return Q.openUrl(this, 27595931);" class="external_link" data-qt-tooltip="mozilla.org">the documentation for Array.prototype.sort</a></span> for more information.</p><p class="qtext_para">If you really need to avoid the built-in <code class="prettyprint inline">sort</code> method, use <span class="qlink_container"><a href="https://en.wikipedia.org/wiki/Quicksort" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 27595931);" class="external_link" data-qt-tooltip="wikipedia.org">quicksort</a></span> as its usually the most performant (bubble sort is terrible in this regard). Heres the implementation I use for my <span class="qlink_container"><a href="http://stackmerge.andyf.me/" rel="noopener nofollow" target="_blank" onclick="return Q.openUrl(this, 27595931);" class="external_link" data-qt-tooltip="andyf.me">image processing app</a></span>:</p><pre class="prettyprint linenums">function swap(array, i, j) {
    var temp = array[i];
    array[i] = array[j];
    array[j] = temp;
}

function partition(array, left, right) {
    var cmp = array[right - 1];
    var minEnd = left;
    var maxEnd;

    for (maxEnd = left; maxEnd &lt; right - 1; maxEnd += 1) {
        if (array[maxEnd] &lt;= cmp) {
            swap(array, maxEnd, minEnd);
            minEnd += 1;
        }
    }
    swap(array, minEnd, right - 1);
    return minEnd;
}

function quickSort(array, left, right) {
    var p;
    if (left &lt; right) {
        p = partition(array, left, right);
        quickSort(array, left, p);
        quickSort(array, p + 1, right);
    }
}
</pre><p class="qtext_para">The initial values for <code class="prettyprint inline">left</code> and <code class="prettyprint inline">right</code> should be 0 and <code class="prettyprint inline">yourarray.length - 1</code> respectively. Note that like the built-in sort method, this sorts the array <b>in place</b>.</p><p class="qtext_para">Before you go ahead with this, though, be sure you actually benefit from the performance enough to outweigh the additional complexity and maintenance difficulties.</p></span>
