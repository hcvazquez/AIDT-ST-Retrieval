         <p>Very fast and Simple Barcode decoder in Salesforce1 using quaggaJS javascript library</p> 
        </div>
        <!-- .entry-summary --> 
        <div class="entry-content"> 
         <p style="text-align: justify;">Recently there was a need for <a href="https://en.wikipedia.org/wiki/Barcode">Barcode</a>&nbsp;decoder &nbsp;component in Salesforce1 using Javascript. I tried few&nbsp;libraries like <a href="https://github.com/EddieLa/JOB">JOB </a>(Javascript Only Barcodereader), <a href="http://bobbuzzard.blogspot.com/2014/03/reading-barcodes-in-salesforce1.html">this post</a> from Bob Buzzard (it uses JOB library) and&nbsp;<a href="http://serratus.github.io/quaggaJS/">QuaggaJS</a>. And after comparing all these, decided to go with quaggajs library because of its speed. Bob Buzzard already <a href="http://bobbuzzard.blogspot.com/2014/03/reading-barcodes-in-salesforce1.html">wrote very good article on JOB library</a> and I would suggest to try that code as well. <a href="http://www.jitendrazaa.com/blog/salesforce/barcode-scanner-component-in-salesforce-lightning-video/">You can find same code rewritten for lightning component here</a>.</p> 
         <p><a href="http://serratus.github.io/quaggaJS/">Quaggajs </a>supports <a href="https://en.wikipedia.org/wiki/Code_128">Code 128</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Code_39">Code 39</a>,&nbsp;Code 39 VIN,&nbsp;EAN,&nbsp;<a href="https://en.wikipedia.org/wiki/EAN-8">EAN-8</a>,&nbsp;<a href="https://en.wikipedia.org/wiki/Universal_Product_Code">UPC</a>,&nbsp;UPC-E and&nbsp;<a href="https://en.wikipedia.org/wiki/Codabar">Codabar</a> format of Barcode.</p> 
         <p>To run this code you will need 3 static resources in your visualforce page :</p> 
         <ol> 
          <li>JQuery</li> 
          <li>Bootstrap (totally optional and can be safely ignored) and</li> 
          <li><a href="http://serratus.github.io/quaggaJS/examples/">QuaggaJs</a></li> 
         </ol> 
         <p style="text-align: justify;">If you want, below&nbsp;images can be used to try this code. Also, please note – If you are using mobile camera to capture barcode, make sure image is white else this library will not work.</p>

         <p style="text-align: justify;"><span id="more-4667"></span>Create a Visualforce page with below code,&nbsp;To keep code very simple, I am only displaying extracted code from Barcode on Visualforce page.</p> 
         <pre class="brush: xml; title: ; notranslate" title="">
&lt;apex:page showheader="false" sidebar="false" standardStylesheets="false" &gt;
    &lt;apex:includeScript value="{!$Resource.JQuery}"/&gt;
    &lt;apex:includeScript value="{!$Resource.Barcode_quaggaJS}"/&gt;
    &lt;apex:stylesheet value="{!URLFOR($Resource.BootStrap3, 'bootstrap-3.1.1-dist/css/bootstrap.min.css')}"/&gt;

    &lt;div class="container"&gt;
         &lt;img width="320" height="240" src="" id="picturePreview" style="border:1px solid black" /&gt; &lt;br /&gt;
        &lt;input id="barcodeFileInput" type="file" accept="image/*;capture=camera" /&gt; &lt;br /&gt;
        &lt;button id="startDecode" class="btn btn-primary"&gt;Decode&lt;/button&gt;
        &lt;div id="codeResult"&gt; &lt;/div&gt;
    &lt;/div&gt;

&lt;script type="text/javascript"&gt;
    $(function() {
        var App = {
            init: function() {
                App.attachListeners();
            } ,
            debugApp : function(msg){
                    if(true)
                    {
                     debugCon = $("#codeResult");
                     debugCon.text(debugCon.text()+" , "+msg) ;
                    }
                },
            attachListeners: function() {
                var self = this; 

                $("#startDecode").on("click", function(e) {
                    var input = $("#barcodeFileInput");
                    if (input[0].files &amp;&amp; input[0].files.length) {
                        var tmpImgURL = URL.createObjectURL(input[0].files[0]) ;
                        App.decode(tmpImgURL);
                    }
                }); 

                $("#barcodeFileInput").on("change",function(e){
                    $("#codeResult").text(" ");
                    var files = e.target.files ;
                        if (files &amp;&amp; files.length &gt; 0) {
                            file = files[0];
                            try {
                                var URL = window.URL || window.webkitURL;
                                var imgURL = URL.createObjectURL(file);
                                $("#picturePreview").attr("src",imgURL);
                            }
                            catch (e) {
                                try {
                                    var fileReader = new FileReader();
                                    fileReader.onload = function (event) {
                                     $("#picturePreview").attr("src",event.target.result);
                                    };
                                    fileReader.readAsDataURL(file);
                                }
                                catch (e) {
                                    $("#codeResult").text("Fail !!! createObjectURL and FileReader are not supported by browser");
                                }
                            }
                        }
                });

            },
            detachListeners: function() {
                $("#startDecode").off("click");
            },

            decode: function(src) {
                Quagga.decodeSingle(
                    {
                      inputStream: {
                        size: 640,
                        singleChannel: false
                      },
                      locator: {
                        patchSize: "large",
                        halfSample: false
                      },
                      decoder: {
                        readers: ["upc_reader", "code_128_reader", "code_39_reader", "code_39_vin_reader", "ean_8_reader", "ean_reader", "upc_e_reader", "codabar_reader"]
                    },
                      locate: true,
                      src: src
                    },
                 function(result){
                      if(result &amp;&amp; result.codeResult &amp;&amp; result.codeResult.code)
                      {
                        $("#codeResult").text('Code - '+result.codeResult.code);
                      }else{
                        $("#codeResult").text("unable to read");
                      }
                    });
            }
        };
        App.init();
    });
&lt;/script&gt;

         <p style="text-align: justify;">Make sure this visualforce is marked to use in Salesforce1 mobile application. After this, Create a Visualforce tab and in Mobile Navigation, enable it for Salesforce 1.</p>
