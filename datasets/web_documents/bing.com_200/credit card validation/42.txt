        <h1 class="entry-title" itemprop="headline">Credit Card Check Digit Generator Java Program</h1>
        <p class="entry-meta"><time class="entry-modified-time" itemprop="dateModified" datetime="2017-08-22T11:42:44+00:00">August 22, 2017</time> by <span class="entry-author" itemprop="author" itemscope itemtype="https://schema.org/Person"><a href="https://www.journaldev.com/author/pankaj" class="entry-author-link" itemprop="url" rel="author"><span class="entry-author-name" itemprop="name">Pankaj</span></a></span> <span class="entry-comments-link"><a href="https://www.journaldev.com/1449/credit-card-check-digit-generator-java-program#respond">Leave a Comment</a></span></p>
       </header>
       <div class="entry-content" itemprop="text">
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">

        <p>In the last post, we saw that <strong>Credit Card</strong> numbers are not random and it can be validated using <a href="https://en.wikipedia.org/wiki/Luhn_algorithm">Luhn Algorithm</a> and I wrote a java program for <a href="https://www.journaldev.com/1443/java-credit-card-validation-luhn-algorithm-java"><strong>credit card number validation</strong></a>.</p>
        <p>A credit card number last digit is called <strong>Check Digit</strong> and its appended to a partial credit card number to generate the complete valid credit card number. For example, if we have a partial card number of 15 digits as 123456789012345 then using Luhn algorithm, we find that check digit should be 2, so the <strong>valid credit card number</strong> will be 1234567890123452.</p>
        <p>Here I am providing a method in java to generate the check digit for a partial credit card number.</p>
        <div style="float:none;margin:10px 0 10px 0;text-align:center;">

        <pre class="brush: java; title: GenerateCheckDigit.java; notranslate" title="GenerateCheckDigit.java">
package com.journaldev.design.test;

public class GenerateCheckDigit {

	public static void main(String[] args) {
		long l = 123456789012345L;
		int cd = generateCheckDigit(l);
		System.out.println(&amp;quot;Valid card number=&amp;quot;+l+cd);
	}


	private static int generateCheckDigit(long l) {
		String str = Long.toString(l);
		int[] ints = new int[str.length()];
		for(int i = 0;i&amp;lt; str.length(); i++){
			ints[i] = Integer.parseInt(str.substring(i, i+1));
		}
		for(int i = ints.length-2; i&amp;gt;=0; i=i-2){
			int j = ints[i];
			j = j*2;
			if(j&amp;gt;9){
				j = j%10 + 1;
			}
			ints[i]=j;
		}
		int sum=0;
		for(int i = 0;i&amp;lt; ints.length; i++){
			sum+=ints[i];
		}
		if(sum%10==0){
			return 0;
		}else return 10-(sum%10);
	}

}
</pre>
        <p>Once the valid account number is generated by adding check digit to the partial credit card number, you can validate it using my last program to <a href="https://www.journaldev.com/1443/java-credit-card-validation-luhn-algorithm-java" title="Java Credit Card Validation Program using Luhn Algorithm">validate credit card number</a>.</p>
