            <p><a rel="nofollow" href="http://vozag.com/uploads/blog/machine_learning_in_js.png" target="_blank"><img src="http://vozag.com/uploads/blog/machine_learning_in_js.png?width=500" width="500" class="align-full"></a></p> 
            <p></p> 
            <p>One of the beauties of running Javascript related applications is you don’t need to install any client side software, optimize servers and spend tons of time on the core infrastructure. Javascript just work outs of the core browser. In that spirit, there is a lot of increasing momentum on building Machine Learning in Javascript. We have collected a list of resources on Javascript that will be helpful if you are building machine learning applications in Javascript.</p> 
            <p><span><span>&nbsp;</span></span></p> 
            <p dir="ltr"><span>A great starting point for learning about</span> <a rel="nofollow" href="http://harthur.github.io/txjs-slides/"><span>Javascript in Machine learning is to go through these slides by Heather Arthur</span></a><span>. The library goes through examples of doing image recognition using Javascript.</span></p> 
            <p><span><span>&nbsp;</span></span></p> 
            <p dir="ltr">Here is a compilation of machine learning libraries in Javascript:</p> 
            <ol> 
             <li dir="ltr"><span><a rel="nofollow" href="https://github.com/encog/encog-javascript">Encog’s machine learning framework in Javascript</a>&nbsp;: Encog is a machine learning framework available for Java, .Net, and C++. Encog supports different learning algorithms such as Bayesian Networks, Hidden Markov Models and Support Vector Machines. However, its main strength lay in its neural network algorithms. Encog contains classes to create a wide variety of networks, as well as support classes to normalize and process data for these neural networks. Encog trains using many different techniques. Multithreading is used to allow optimal training performance on multicore machines. The C++ version of Encog can offload some processing to an OpenCL compatible GPU for further performance gains. Here is a great case study &amp; example using&nbsp;<a rel="nofollow" href="http://www.codeproject.com/Articles/477689/JavaScript-Machine-Learning-and-Neural-Networks-wi">Encog’s machine learning in Javascript</a>.</span></li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="http://cs.stanford.edu/people/karpathy/convnetjs/"><span>Deep learning with Java script</span></a> <span>: The library allows you to formulate and solve Neural Networks in Javascript, and was originally written by Andrej Karpathy (a PhD student at Stanford). However, the library has since been extended by contributions from the community.</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="https://github.com/harthur/brain"><span>Brain.js</span></a> <span>&nbsp;A neural network library</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="https://github.com/karpathy/svmjs"><span>svmjs</span></a> <span>SVM library</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="https://github.com/karpathy/forestjs"><span>Forest.js</span></a> <span>Javascript library implementing Random trees</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="http://www.numericjs.com/"><span>Numeric.js</span></a> <span>Performs sophisticated numerical computations.</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="https://github.com/NaturalNode/node-sylvester"><span>Node-sylvester</span></a> <span>Linear algebra library</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><a rel="nofollow" href="https://github.com/lastlegion/linearReg.js"><span>Lineareg.js</span></a> <span>Linear regression with Javascript.</span></p> </li> 
             <li dir="ltr"><p dir="ltr"><span>Bayesian classifiers including</span> <a rel="nofollow" href="https://github.com/NaturalNode/natural"><span>NaturalNode/natural</span></a> <span>&nbsp;&amp;</span> <a rel="nofollow" href="https://github.com/harthur/classifier"><span>harthur/classifier</span></a></p> </li> 
             <li dir="ltr"><p dir="ltr"><span>Clustering:</span> <a rel="nofollow" href="https://github.com/harthur/clusterfck"><span>harthur/clusterfck</span></a></p> </li> 
            </ol>
