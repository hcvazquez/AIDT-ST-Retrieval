            <div class="topic_text">
              Scope of variable is nothing but the statements or expressions in which the variable can be used. Any variable is in scope ( valid ) in the statements below its declaration. It will not be in scope or it will be invalid above its declaration. 
            </div> 
            <br> 
            <div class="topic_text"> 
             <div class="exampleBox">
              <div class="exampleTopSection">
               <span class="zt_ex_name">Scope in same block</span>
               <a href="/example_programs/11/download" class="zl_ex_options"><img style="width:30px;" src="/assets/down_1-88be5e30e9a180aa53510e4bd6a82a69.png"></a>
              </div>
              <div class="example_code">
               <div class="code_try_it">
                <div class="main_title_code">
                 <div class="title_code text-left">
                  CODE
                 </div>
                 <div class="text-right">
                  <a href="/java_compiler/run?example_program_id=11" target="_blank" class="zl_ex_options_try"> Try it Online</a>
                 </div>
                </div>
               </div>
               <div class="prettyprint_zt_code">
                <pre class="prettyprint_zt_pre">class ScopeInSameBlock<br>{<br>&nbsp;&nbsp;&nbsp;&nbsp;public static void main(String arg[])<br>&nbsp;&nbsp;&nbsp;&nbsp;{<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;int x = 15;&nbsp;&nbsp;&nbsp;&nbsp;// LINE A<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;int y = x;&nbsp;&nbsp;&nbsp;&nbsp;// LINE B<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;int z = y + 20;&nbsp;&nbsp;&nbsp;&nbsp;// LINE C<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println("x = " + x);&nbsp;&nbsp;&nbsp;&nbsp;// LINE D<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println("y = " + y);&nbsp;&nbsp;&nbsp;&nbsp;// LINE E<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println("z = " + z);&nbsp;&nbsp;&nbsp;&nbsp;// LINE F<br>&nbsp;&nbsp;&nbsp;&nbsp;<br>&nbsp;&nbsp;&nbsp;&nbsp;}<br>}</pre>
               </div>
              </div>
              <div class="example_output">
               <span class="title">OUTPUT</span>
               <p class="exampleOutput">x = 15<br>y = 15<br>z = 35</p>
              </div>
              <div class="example_description">
               <span class="title">DESCRIPTION</span>
               <div class="des_div_description">
                <p class="exampleDescription">Any variable will be valid in the statements below its declaration. It will not be valid before it is declared. Here <code class="code">x</code> is in scope in all lines below <code class="code">LINE A</code>, until the end of the block. Where as <code class="code">y</code> is valid in lines below <code class="code">LINE B</code> and <code class="code">z</code> is valid (or in scope) after <code class="code">LINE C</code>. Using any variable not in scope will throw an 'undefined variable' compilation error.</p>
               </div>
              </div>
              <div class="example_things_to_try">
               <div class="title_things_to_try">
                THINGS TO TRY
               </div>
               <div class="things_to_try_div">
                <ul>
                 <li class="bullet_color">Change the order of lines - put <code class="code">LINE B</code> above <code class="code">LINE A</code>.</li>
                </ul> 
                <ul>
                 <li class="bullet_color">Move <code class="code">LINE D</code> above <code class="code">LINE B</code> or try moving <code class="code">LINE F</code> above <code class="code">LINE B</code></li>
                </ul>
