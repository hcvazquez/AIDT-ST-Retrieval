             <div class="post-text" itemprop="text"> 
              <p>Sentiment library in R is now obsolete. Even if you get that package from the CRAN archive, it would look hassle free until you deploy the code in Shiny. If you are about to use Shiny too in your work, I recommend you to do the following:</p> 
              <ol> 
               <li><p>Unzip the sentiment package file</p></li> 
               <li><p>Get whatever R script available there and deploy the code to the file you are working.</p></li> 
               <li><p>Place the dependent files (like .CSV in the sentiment package) to your present working directory.</p></li> 
              </ol> 
              <p>Now your code would be hassle free.</p> 
              <p>I also suggest you to use ‘e1071’ package for sentiment computation. This package is more inclined to strong ML concepts than being generic like the ‘sentiment’ package as it only uses 'Naive Bayes Classifier' which won't produce more accuracy.</p> 
              <p>A thumps up if you liked my answer! :) good luck, welcome to the new field!</p>
