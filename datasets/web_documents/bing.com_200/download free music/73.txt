              <p>I am making a card game, and it's almost done. The last thing I want to do is play some background music. Now if I copy the file into the default package and make a single jar file of the game, will the music play on all computers? Currently, on my PC it's running without any problem by giving a specific path for the file like <code>"C:\\samp.wav";</code>. But I am worried that if I make a jar file and run it on another PC it won't work properly. I think there will be a <code>FileNotFoundException</code>. Am I right or wrong?</p>

              <pre><code>jLabel1.setIcon(new javax.swing.ImageIcon(getClass().getResource("/1.jpg")));

            <td class="answercell"> 
             <div class="post-text" itemprop="text"> 
              <p>You should include the wav file inside your application jar. This way you won't have to manage the copy of the file in the user's file system (keep in mind that, for example, in UNIX, Mac, etc. you can't access to the hard drive through <code>C:/...</code>).</p> 
              <p>For example, if you place the wav file in the root of the app jar (<code>app.jar/samp.wav</code>):</p> 
              <pre><code>InputStream is = getClass().getClassLoader().getResourceAsStream("samp.wav");
</code></pre> 
              <p>or, if you had a "sounds" directory in the app jar root (<code>app.jar/sounds/samp.wav</code>):</p> 
              <pre><code>InputStream is = getClass().getClassLoader().getResourceAsStream("sounds/samp.wav");
</code></pre> 
              <p>Check <a href="https://stackoverflow.com/questions/2416935/how-to-play-wav-files-with-java">this post</a> for extra information about playing wav files with Java (though for your question, I think you have already solved this problem). Consider as well playing it in a separate thread. Check also <a href="http://www.jsresources.org/examples/audio_playing_recording.html" rel="nofollow noreferrer">this web for examples</a> about managing media files in Java.</p>

AudioPlayer MGP = AudioPlayer.player;
AudioStream BGM;

ContinuousAudioDataStream loop = null;
try{
BGM = new AudioStream(new FileInputStream("C:\\test\\ha.wav"));
MD = BGM.getData();
loop = new ContinuousAudioDataStream(MD);
}catch(IOException error){
System.out.print("file not found");
}

}
