           </svg></a>Why?</h2> 
         <p>If you're writing a frontend library, it's best to avoid a dependency on <a href="http://jquery.com">jQuery</a>. This means having to write your own DOM manipulation code, though. To speed you along, I've packaged all that typical DOM manipulation code into many single-use JS files.</p>
