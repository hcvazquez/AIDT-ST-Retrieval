         <h1><a href="#trackjs" aria-hidden="true" class="anchor" id="user-content-trackjs">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>TrackJS</h1> 
         <p>The JavaScript Browser Agent for JavaScript Browser Error Monitoring from TrackJS.</p> 
         <p>The agent wraps the Browser's API to record context about your application, the network, and the visitor leading up to errors, allowing you to recreate and debug errors fast.</p> 
         <p><strong><a href="https://trackjs.com/signup?utm_source=repository">Signup for TrackJS</a></strong>.</p> 
         <p><strong><a href="http://docs.trackjs.com/">Documentation</a></strong></p> 
         <h2><a href="#quickstart" aria-hidden="true" class="anchor" id="user-content-quickstart">
           <svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewbox="0 0 16 16" width="16">
            <path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path>
           </svg></a>Quickstart</h2> 
         <ol> 
          <li> <p><a href="https://trackjs.com/signup?utm_source=repository">Signup for TrackJS</a> and get your token.</p> </li> 
          <li> <p>Create a <code>_trackJs</code> initialization object in your app:</p> 
           <div class="highlight highlight-source-js">
            <pre><span class="pl-c1">window</span>.<span class="pl-smi">_trackJs</span> <span class="pl-k">=</span> { <span class="pl-c"><span class="pl-c">//</span> Complete reference at http://docs.trackjs.com/</span>
  token<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>YOUR_TOKEN_HERE<span class="pl-pds">'</span></span>
};</pre>
           </div> </li> 
          <li> <p>Include the <code>tracker.js</code> script in your app, after the initialization object. You can either reference directly, or bundle it into your other sources. We recommend that <code>tracker.js</code> be the first script to execute so that it can catch errors from everything else.</p> </li> 
          <li> <p>Test it out by tracking an error, either in your code or from the console: <code>trackJs.track('testing!')</code></p> </li> 
          <li> <p>See the new error in your <a href="https://my.trackjs.com/recent">TrackJS Recent Errors</a> within a few seconds.</p> </li> 
         </ol> 
         <p>If you run into any trouble, let us know right away at <code>hello@trackjs.com</code></p>
