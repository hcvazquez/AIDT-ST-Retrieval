        <div id="mission">
         Java-ML in a nutshell: 
         <ul> 
          <li>A collection of machine learning algorithms</li> 
          <li>Common interface for each type of algorithms</li> 
          <li>Library aimed at software engineers and programmers, so no GUI, but clear interfaces</li> 
          <li>Reference implementations for algorithms described in the scientific literature.</li> 
          <li>Well documented source code.</li> 
          <li>Plenty of code samples and tutorials.</li> 
         </ul>

          <h2><a href="/content/java-ml-website" title="Java-ML website">Java-ML website</a></h2> 
          <span class="submitted">Wed, 08/06/2008 - 21:41 — Thomas Abeel</span> 
          <div class="content clear-block"> 
           <p>We are proud to announce that we moved the website for Java-ML from my personal hosting to <a href="http://sourceforge.net">Sourceforge</a>. </p> 
           <p>All data is hosted on the same platform and on the same servers. The website is based on <a href="http://drupal.org/">Drupal</a>.<br> Main features of this new website:</p> 
           <ul> 
            <li>Documentation in Wiki-style</li> 
            <li><a href="/api">API documentation</a> as generated by javadoc</li> 
            <li>Issue tracking for bugs and feature requests</li> 
            <li>Support to incorporate sub-projects on particular algorithms or features</li> 
           </ul>
