          <h2 class="text-title"> Dynamically translate between thousands of available language pairs </h2>

         <dl class="c5 a1"> 
          <dt>
           Programmatic Access
          </dt> 
          <dd>
            Translation API is accessible via a standard Google REST API. See 
           <a href="https://cloud.google.com/translate/docs/samples">sample code</a> for different programming languages including Python, Objective C and Ruby.

          <dt>
           Text Translation
          </dt> 
          <dd>
            Translation API supports 
           <a href="https://cloud.google.com/translate/docs/languages"> more than 100 languages </a> and thousands of language pairs. You can send in HTML and receive HTML with translated text back. You don't need to extract your source text or reassemble the translated content. 
          </dd> 
          <dt>
           Language Detection
          </dt> 
          <dd>
            Detect a document’s language and translate it using 
           <a href="https://cloud.google.com/translate/docs/reference/rest"> RESTful API</a>. 
          </dd> 
         </dl> 
         <dl class="c5"> 
          <dt>
           Continuous Updates
          </dt> 
          <dd>

          </dd>

           Adjustable Quota
          </dt> 
          <dd>
            Easily increase your quota from 2M characters per day to 50M per day or request a higher quota. 
          </dd>

           Affordable, Easy Pricing
          </dt> 
          <dd>
