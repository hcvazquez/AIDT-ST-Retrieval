         <p>A general purpose library for building credit card forms, validating inputs and formatting numbers.</p>

           </svg></a>Project status</h2> 
         <p><strong>jquery.payment is deprecated. We recommend that you use either <a href="https://stripe.com/docs/checkout">Stripe Checkout</a> or <a href="https://stripe.com/docs/elements">Stripe Elements</a> to collect card information.</strong></p> 
         <p>We will patch jquery.payment for major critical/security issues, but we won't be adding new features.</p> 
         <h2><a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true">

           </svg></a>Usage</h2> 
         <p>You can make an input act like a credit card field (with number formatting and length restriction):</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-en">$</span>(<span class="pl-s"><span class="pl-pds">'</span>input.cc-num<span class="pl-pds">'</span></span>).<span class="pl-en">payment</span>(<span class="pl-s"><span class="pl-pds">'</span>formatCardNumber<span class="pl-pds">'</span></span>);</pre>
         </div> 
         <p>Then, when the payment form is submitted, you can validate the card number on the client-side:</p> 
         <div class="highlight highlight-source-js">
          <pre><span class="pl-k">var</span> valid <span class="pl-k">=</span> <span class="pl-smi">$</span>.<span class="pl-smi">payment</span>.<span class="pl-en">validateCardNumber</span>(<span class="pl-en">$</span>(<span class="pl-s"><span class="pl-pds">'</span>input.cc-num<span class="pl-pds">'</span></span>).<span class="pl-en">val</span>());

<span class="pl-k">if</span> (<span class="pl-k">!</span>valid) {
  <span class="pl-en">alert</span>(<span class="pl-s"><span class="pl-pds">'</span>Your card is not valid!<span class="pl-pds">'</span></span>);
  <span class="pl-k">return</span> <span class="pl-c1">false</span>;
}</pre>
         </div> 
         <p>You can find a full <a href="http://stripe.github.io/jquery.payment/example">demo here</a>.</p>

          <li>Visa</li> 
          <li>MasterCard</li> 
          <li>American Express</li> 
          <li>Diners Club</li>

          <li>JCB</li> 
          <li>Maestro</li>

         <div class="highlight highlight-source-js">

      <span class="pl-c1">22</span>, <span class="pl-c1">23</span>, <span class="pl-c1">24</span>, <span class="pl-c1">25</span>, <span class="pl-c1">26</span>, <span class="pl-c1">27</span>
  ],
  <span class="pl-c"><span class="pl-c">//</span> Array of valid card number lengths.</span>

  <span class="pl-c"><span class="pl-c">//</span> Array of valid card CVC lengths.</span>
  cvcLength<span class="pl-k">:</span> [<span class="pl-c1">3</span>],

  <span class="pl-c"><span class="pl-c">//</span> Regex used to format the card number. Each match is joined with a space.</span>
