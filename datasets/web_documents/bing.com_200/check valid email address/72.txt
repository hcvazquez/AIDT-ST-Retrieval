       <div style="clear: both; padding-top:6px;"> 
        <p class="firstpara">We can validate email address at client side and server side. To validate email address on client side, we can use java script with regular expression. Java script can check the regular expression pattern for valid email address. We have different regular expression patterns for validating email address. In this article, I am sharing some useful cross browsers regular expression patterns for validating email address with java script and jquery.</p> 
        <h2>Simple regular expression to validate email address</h2> 
        <p>Below is simple regular expression to validate an email address. It will accept email address in upper case also.</p> 
        <pre class="prettyprint lang-html linenums"> &lt;script type="text/javascript"&gt;
function validateEmail(email)
{
 var reg = /^\w+([-+.']\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/
 if (reg.test(email)){
 return true; }
 else{
 return false;
 }
} 
&lt;/script&gt; </pre> 
        <h2>Regular expression to validate case-sensitive email address</h2> 
        <p>Below is regular expression to validate an email address with lower case chars.</p> 
        <pre class="prettyprint lang-html linenums"> &lt;script type="text/javascript"&gt;
function validateCaseSensitiveEmail(email) 
{ 
 var reg = /^[_a-z0-9-]+(\.[_a-z0-9-]+)*@[a-z0-9-]+(\.[a-z0-9-]+)*(\.[a-z]{2,4})$/;
 if (reg.test(email)){
 return true; 
}
 else{
 return false;
 } 
} 
&lt;/script&gt; </pre> 
        <h2>Regular expression to validate free/domain specific email address</h2> 
        <p>Below is regular expression to validate domain specific email address.</p> 
        <pre class="prettyprint lang-html linenums"> &lt;script type="text/javascript"&gt;
function validateFreeEmail(email) 
{
 var reg = /^([\w-\.]+@(?!gmail.com)(?!yahoo.com)(?!hotmail.com)([\w-]+\.)+[\w-]{2,4})?$/
 if (reg.test(email)){
 return true;
 }
 else{
 return false;
 }
} 
&lt;/script&gt; </pre>

    function SubmitComments() {
        flag = true;
        if ($.trim($("#Comment").val()) == "") {
            $("#Comment").addClass("input-validation-error");
            flag = false;

        }
        else {
            $("#Comment").removeClass("input-validation-error");
        }

        if (flag) {
            var obj = {
                Contents: $("#Comment").val(),
                ReferralID: $("#ReferralID").val(),
                IsBook: $("#IsBook").val(),
            };

            $.ajax({
                type: "POST",
                contentType: "application/json; charset=utf-8",
                url: '/Comment/SaveComment',
                data: JSON.stringify(obj),
                dataType: "json",
                beforeSend: function () {
                    // setting a timeout
                    $('#btnwaitcm').show();
                    $('#btndowncm').hide();
                },
                success: function (data) {
                    if (data.msg == "success") {

                        $("#Comment").val("");
                       location.reload();
                    }

                },
                error: function (result) {
                    alert('Service call failed: ' + result.status + ' Type :' + result.statusText);
                },
                complete: function () {
                    setTimeout(function () {
                        $('#btnwaitcm').hide();
                        $('#btndowncm').show();

                    }, 2000);
                },
            });
        }
    }

    function Subscribe() {
        flag = true;
        if ($.trim($("#Email").val()) == "") {

            $("#Email").addClass("input-validation-error");
            flag = false;
        }
        else {
            $("#Email").removeClass("input-validation-error");
        }

        if ($.trim($("#Email").val()) != "") {
            var email = $.trim($("#Email").val());
            if (!validEmail(email)) {
                $("#Email").addClass("input-validation-error");
                flag = false;
            }
            else {
                $("#Email").removeClass("input-validation-error");
            }
        }

        if (flag) {
            var obj = {
                EmailID: $("#Email").val(),
            };

            $.ajax({
                type: "POST",
                contentType: "application/json; charset=utf-8",
                url: '/Home/Subscription',
                data: JSON.stringify(obj),
                dataType: "json",
                success: function (data) {
                    if (data.msg == "success") {
                        $("#alertModel").modal('show');
                        $('.modal-body > p > span').html('');
                        $('.modal-body > p > span').html('You have been subscribed successfully. You will receive an email shortly. Please verify your email.');

                        $("#Email").val("");
                    }
                    else if (data.msg == "alreadyexists") {
                        $("#alertModel").modal('show');
                        $('.modal-body > p > span').html('');
                        $('.modal-body > p > span').html('This email-id already exists.');
                    }
                    else if (data.msg == "emailsentfail") {
                        $("#alertModel").modal('show');
                        $('.modal-body > p > span').html('');
                        $('.modal-body > p > span').html('You have been subscribed successfully.');
                    }
                    else {
                        $("#alertModel").modal('show');
                    }
                },
                error: function (result) {
                    alert('Service call failed: ' + result.status + ' Type :' + result.statusText);
                }
            });
        }
    }
